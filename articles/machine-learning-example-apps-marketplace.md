# Azure 마켓플레이스의 기계 학습 예제 앱

데이터 과학자는 Azure 기계 학습을 통해 스마트폰 앱 개발자처럼 자신이 보유한 노하우와 창의성을 수익화하는 새로운 채널을 확보할 수 있습니다. 기계 학습 서비스를 Azure 마켓플레이스에 게시하여 전 세계 모든 사용자가 데이터 과학자가 새로 게시한 클라우드 API를 결제 및 구독하여 자신의 앱에 사용하도록 할 수 있습니다.

그러나 Azure 마켓플레이스는 기본 분석 API 이상을 제공할 수 있습니다. 예를 들어 특정 비즈니스 요구 사항을 해결하기 위해 모두 Azure 기계 학습을 사용하여 완성된 일련의 종단 간 기계 학습 응용 프로그램을 작성했습니다. 이러한 기계 학습 앱을 사용하기 위해 데이터 과학자나 기계 학습 전문 지식이 필요하지는 않습니다. 과학은 이미 솔루션에 반영되어 있습니다. 사용자는 고유한 데이터를 가져와 바로 사용할 수 있습니다. 

이러한 예는 다음과 같습니다.

* [추천](http://datamarket.azure.com/dataset/amla/recommendations) - 셀프 서비스이지만 매우 강력한 추천 엔진으로서, 웹 사이트나 메일 캠페인에 개인 설정된 제품 추천을 쉽게 추가할 수 있게 해줍니다. 
  
* [자주 함께 구매됨](https://datamarket.azure.com/dataset/amla/mba) - 고객이 카탈로그에서 자주 함께 구매되는 항목을 검색하는 데 도움을 주는 API입니다. 고객 구매 내역을 사용하여 "자주 함께 구매됨" 추천을 웹 사이트에 추가함으로써 디지털 스토어의 구매 전환율을 향상시킬 수 있습니다.

* [비정상 감지](https://datamarket.azure.com/dataset/aml_labs/anomalydetection) - 시계열 데이터(예: 서버 원격 분석, 센서 기반 데이터)에서 비정상적인 이벤트를 감지할 수 있는 API입니다.

이러한 API를 사용하면 데이터 과학자가 되거나 스스로 API를 개발하는 데 시간을 들일 필요 없이 Azure 기계 학습에 빌드된 강력한 기계 학습 기반 서비스를 사용할 수 있습니다.
