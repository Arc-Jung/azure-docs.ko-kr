# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Azure Maps 계정의 가격 책정 계층을 관리하는 방법

Azure Portal을 통해 Azure Maps 계정의 가격 책정 계층을 관리할 수 있습니다. [계정](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)을 만들면 계정의 가격 책정 계층을 보거나 변경할 수 있습니다.

올바른 가격 책정 계층을 선택하는 방법에 대한 자세한 내용은 [올바른 가격 책정 계층 선택](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier)을 참조하세요.

## <a name="view-pricing-tier"></a>가격 책정 계층 보기

선택한 가격 책정 계층을 보려면 설정 메뉴에서 **가격 책정 계층** 옵션으로 이동합니다.

![선택한 가격 책정 계층 보기](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>가격 책정 계층 변경

Azure Maps 계정을 만든 경우 Azure Maps 계정에 대한 가격 책정 계층을 업그레이드하거나 다운그레이드할 수 있습니다. 업그레이드 또는 다운그레이드하려면 설정 메뉴에서 **가격 책정 계층** 옵션으로 이동한 다음 강조 표시가 해제된 가격 책정 계층을 선택합니다. 저장 단추를 클릭하여 선택한 가격 책정 계층 옵션을 저장합니다.

![가격 책정 계층 변경](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> Azure Maps 계정에 대한 가격 책정 계층을 업그레이드 또는 다운그레이드하는 경우 새 구독 키를 생성할 필요가 없습니다.
