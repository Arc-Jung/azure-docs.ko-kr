# <a name="azure-maps-routing-coverage"></a>Azure Maps 라우팅 적용 범위

이 문서에서는 Azure Maps 라우팅에 대한 적용 범위 정보를 제공합니다. 검색 쿼리시 Azure Maps는 위치 A에서 위치 B로의 최적의 경로를 반환합니다. 정확한 여행 시간, 여행 정보의 실시간 업데이트 및 경로 지침이 제공됩니다. 현재 트래픽, 차량 유형 및 피해야 할 조건과 같은 추가 검색 매개변수를 추가할 수도 있습니다. 경로의 최적화는 지역에 따라 다릅니다. Azure Maps에는 다양한 지역에 대한 다양한 수준의 정보와 정확도가 있기 때문입니다. 다음 표에는 지역 및 요청할 수 있는 정보의 종류가 나열되어 있습니다.
 
[**지오코딩**](geocoding-coverage.md)에 대한 검사를 확인해 보세요.       
[**트래픽**](traffic-coverage.md)에 대한 검사를 확인해 보세요.  
[**렌더링**](render-coverage.md)에 대한 검사를 확인해 보세요.
<br>  
**범례**

| 기호             | 의미                                |
|--------------------|----------------------------------------|
| ✓                  | 자세한 데이터와 함께 제공되는 국가/지역.   |
| Ø                  | 간소화된 데이터와 함께 제공되는 국가/지역. |
| 국가가 누락되었습니다. | 국가/지역 데이터는 제공되지 않습니다.          |
