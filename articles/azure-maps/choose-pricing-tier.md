# <a name="choosing-the-right-pricing-tier-in-azure-maps"></a>Azure Maps에서 적절한 가격 책정 계층 선택

Azure Maps는 두 가지 가격 책정 계층을 제공합니다. 이 문서의 목적은 요구 사항에 적합한 가격 책정 계층을 선택하도록 도와주는 것입니다. 적절한 가격 책정 계층을 선택하려면 다음 두 질문에 대답합니다.

## <a name="what-geospatial-capabilities-do-i-plan-to-use"></a>어떤 지리 공간적 기능을 사용할 계획입니까?
핵심 지리 공간적 API가 서비스 요구 사항을 충족한다고 생각되면 S0 가격 책정 계층이 적합합니다. areal+하이브리드 이미지, 경로 범위 가져오기, 지오코딩 일괄 처리 같은 고급 기능이 애플리케이션에 필요한 경우 S1 가격 책정 계층을 고려해 보아야 합니다. **가격 책정 계층 기능**이 나열된 아래 표는 애플리케이션의 요구 사항을 보다 정확하게 이해하고 애플리케이션에 가장 적합한 가격 책정 계층을 선택하도록 도와줍니다.

## <a name="how-many-concurrent-users-do-i-plan-to-support"></a>동시 사용자를 몇 명이나 지원할 계획입니까? 
S0 및 S1 가격 책정 계층이 처리할 수 있는 데이터 처리량은 서로 다릅니다. Azure Maps 가격 책정 계층을 선택하기 전에, 동시 사용자를 몇 명이나 지원할 것인지 생각해 보아야 합니다. S0 가격 책정 계층은 **초당 최대 50개 쿼리**를 처리할 수 있고 S1 가격 책정 계층은 **초당 50개가 넘는 쿼리**를 처리할 수 있습니다.

### <a name="pricing-tier-capabilities"></a>가격 책정 계층 기능

| 기능                              |        S0           |  S1      |
|-----------------------------------------|:-------------------:|:--------:|
| 검색                                  |        ✓           |     ✓    |
| 라우팅                                 |        ✓           |     ✓    |
| 렌더링                                  |        ✓           |     ✓    |
| 트래픽                                 |        ✓           |     ✓    |
| 표준 시간대                              |        ✓           |     ✓    |
| *이미지 + 하이브리드 이미지(미리 보기)     |        ✓           |     ✓    |
| *경로 범위(미리 보기)                  |        ✓           |     ✓    |
| *IP 2 위치(미리 보기)                |        ✓           |     ✓    |
| *검색의 다각형(미리 보기)         |        ✓           |     ✓    |
| *일괄 처리 지오코딩(미리 보기)              |        ✓           |     ✓    |
| *일괄 처리 라우팅(미리 보기)                |        ✓           |     ✓    |
| *매트릭스 라우팅(미리 보기)               |        ✓           |     ✓    |

> [!Note]
> 2019년 2월 4일 이후에는 S0 가격 책정 계층에서 이러한 기능에 액세스할 수 없습니다.

갖고 있는 엔터프라이즈의 유형 또는 빌드 중인 애플리케이션의 중요도 등의 추가 데이터 요소를 생각해 볼 필요가 있습니다.

**가격 책정 계층 대상 고객**이 포함된 표를 참조하여 S0 및 S1 가격 책정 계층에 대해 자세히 알아보세요. Azure Maps 가격 책정에 대한 자세한 내용은 [Azure Maps 가격 책정](https://azure.microsoft.com/pricing/details/azure-maps/)을 참조하세요. 

### <a name="pricing-tier-targeted-customers"></a>가격 책정 계층 대상 고객

| 가격 책정 계층   |        대상 고객                                                                |
|---------------|:-----------------------------------------------------------------------------------------|
| S0            |    <p>S0 가격 책정 계층의 대상 고객은 소규모 또는 중간 규모 기업입니다. 예상하는 동시 사용자 수가 많지 않고 위의 표에 표시된 것처럼 핵심 지리 공간적 API가 서비스 요구 사항을 충족하는 경우에 적합한 가격 책정 계층입니다. 이 계층은 일반 공급되며 개념 증명 개발 및 초기 단계 테스트부터 애플리케이션 프로덕션 및 배포까지 모든 프로덕션 단계에 적용됩니다.<p>|
| S1            |    <p>S1 가격 책정 계층은 대기업, 중요 업무용 애플리케이션, 많은 동시 사용자를 지원해야 하거나 고급 지리 공간적 서비스가 필요한 고객에게 적합합니다.</p>|
