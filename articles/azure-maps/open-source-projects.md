# <a name="azure-maps-community---open-source-projects"></a>Azure Maps 커뮤니티 - 오픈 소스 프로젝트

이러한 오픈 소스, 커뮤니티 기반 이니셔티브는 Azure Maps 팀에서 만들고 유지 관리합니다. 표준 제품 또는 서비스 제품의 일부가 아닙니다.

다음은 가장 인기 있는 Azure Maps 오픈 소스 프로젝트 및 샘플 중 일부를 나열합니다.

| 프로젝트 이름 | 설명 |
|-|-|
| [Azure 지도 코드 샘플](https://github.com/Azure-Samples/AzureMapsCodeSamples) | 웹 기반 앱에서 Azure Maps를 사용하기 위한 코드 샘플 의 컬렉션입니다. |
| [Azure 지도 Gov 클라우드 코드 샘플](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Azure 정부 클라우드를 통해 Azure 지도를 사용하기 위한 코드 샘플의 컬렉션입니다. |
| [Azure & Azure 활성 디렉터리 샘플](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | Azure 맵에서 Azure Active Directory를 사용하는 방법을 보여 주는 샘플 컬렉션입니다. | 
| [Azure Maps를 사용하여 IoT 공간 분석 구현](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | 공간과 시간에서 발생하는 관련 이벤트를 추적하고 캡처하는 것은 일반적인 IoT 시나리오입니다. |
| [Azure 지도 주피터 노트북 샘플](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | Azure Maps REST 서비스를 사용하는 파이썬 샘플 컬렉션입니다. |
| [봇 프레임워크 - 관심 분야 기술](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | 관심 지점 기술은 Azure 봇 서비스 및 봇 프레임워크를 사용하여 Azure Maps를 사용하는 가상 도우미에 POI 관련 기능을 제공합니다. |

커뮤니티에서 만든 오픈 소스 프로젝트를 포함하는 더 긴 목록은 [여기에서](https://github.com/microsoft/Maps/blob/master/AzureMaps.md) 확인할 수 있습니다.

## <a name="supportability-of-open-source-projects"></a>오픈 소스 프로젝트의 지원 가능성

다음 문은 생성되고 유지 관리되는 모든 Azure Maps 프로젝트 및 샘플에 적용됩니다.

- Azure Maps 오픈 소스 프로젝트 및 샘플은 Microsoft 및 커뮤니티에서 생성합니다.
- Azure Maps 오픈 소스 프로젝트 및 샘플은 Microsoft 및 커뮤니티에서 유지 관리합니다.
- Azure Maps 오픈 소스 프로젝트 및 샘플은 지원되는 권장 기술을 사용합니다.
- Azure Maps 오픈 소스 프로젝트 및 샘플은 커뮤니티 이니셔티브로, 다른 사람의 이익을 위해 이니셔티브에 참여하고 정상적인 하루 일자리를 가진 사람들입니다.
- Azure Maps 오픈 소스 프로젝트 및 샘플은 제품이 아니므로 프리미어 지원 또는 기타 공식 지원 채널에서 지원되지 않습니다.
- Azure Maps 오픈 소스 프로젝트 및 샘플은 커뮤니티에서 커뮤니티의 지원을 받아 Microsoft에서 수행한 다른 오픈 소스 프로젝트와 유사한 방식으로 지원됩니다.
