# <a name="interaction-types-and-keyboard-shortcuts-in-the-drawing-tools-module"></a>그리기 도구 모듈의 상호 작용 유형 및 키보드 단축키

이 문서에서는 마우스, 터치 스크린 또는 키보드 단축키를 사용하여 셰이프를 그리고 편집하는 다양한 방법을 간략하게 설명합니다.

드로잉 관리자는 맵과 상호 작용하는 세 가지 방법을 지원하여 셰이프를 그립니다.

* `click`- 좌표는 마우스 또는 터치를 클릭 할 때 추가됩니다.
* `freehand `- 좌표는 마우스 또는 터치가지도에 드래그 할 때 추가됩니다.
* `hybrid`- 좌표는 마우스 또는 터치를 클릭하거나 드래그 할 때 추가됩니다.

## <a name="how-to-draw-shapes"></a>셰이프 그리는 방법

 셰이프를 그리기 전에 도면 관리자의 `drawingMode` 옵션을 지원되는 도면 설정으로 설정합니다. 이 설정은 도구 모음의 그리기 단추 중 하나를 눌러 프로그래밍하거나 호출할 수 있습니다. 도형이 그려진 후에도 드로잉 모드가 활성화되어 있으므로 동일한 유형의 추가 셰이프를 쉽게 그릴 수 있습니다. 프로그래밍 방식으로 도면 모드를 유휴 상태로 설정합니다. 또는 도구 모음의 현재 그리기 모드 단추를 클릭하여 유휴 상태로 전환합니다.

다음 섹션에서는 맵에서 셰이프를 그릴 수 있는 다양한 방법을 간략하게 설명합니다.

### <a name="how-to-draw-a-point"></a>점을 그리는 방법

드로잉 관리자가 드로잉 `draw-point` 모드에 있을 때 다음 작업을 수행하여 맵에서 점을 그릴 수 있습니다. 이러한 메서드는 모든 상호 작용 모드에서 작동합니다.

**그리기 시작**
 - 왼쪽 마우스 버튼을 클릭하거나 맵을 터치하여 맵에 점을 추가합니다. 
 - 마우스가 맵 위에 있으면 키를 `F` 누르면 마우스 포인터의 좌표에 점이 추가됩니다. 이 메서드는 맵에 점을 추가하기 위한 정확도를 높입니다. 왼쪽 마우스 버튼을 누르면 마우스의 움직임이 줄어듭니다.
 - 계속 클릭, 터치 `F` 또는 길게 눌러 맵에 더 많은 포인트를 추가합니다.
 
**그리기 완료**
 - 그리기 도구 모음에서 단추를 클릭합니다. 
 - 프로그래밍 방식으로 도면 모드를 설정합니다. 
 - 키를 `C` 누릅니다.

**도면 취소**
 - 키를 `Escape` 누릅니다.

### <a name="how-to-draw-a-line"></a>선을 그리는 방법

드로잉 관리자가 모드에 `draw-line` 있을 때 상호 작용 모드에 따라 맵에서 포인트를 그리기 위해 다음 작업을 수행할 수 있습니다.

**그리기 시작**
 - 클릭 모드
   * 왼쪽 마우스 버튼을 클릭하거나 맵을 터치하여 맵에 각 선의 각 점을 추가합니다. 각 클릭 또는 터치에 대한 좌표가 줄에 추가됩니다. 
   * 마우스가 맵 위에 있으면 키를 `F` 누르면 마우스 포인터의 좌표에 점이 추가됩니다. 이 메서드는 맵에 점을 추가하기 위한 정확도를 높입니다. 왼쪽 마우스 버튼을 누르면 마우스의 움직임이 줄어듭니다.
   * 원하는 모든 점이 줄에 추가될 때까지 계속 클릭합니다.
 - 프리핸드 모드
   * 왼쪽 마우스 버튼을 누르거나 맵을 터치다운한 후 마우스또는 터치 포인트를 끕습니다. 마우스 또는 터치 포인트가 맵 주위를 이동할 때 좌표가 선에 추가됩니다. 마우스 또는 터치업 이벤트가 트리거되는 즉시 드로잉이 완료됩니다. 좌표가 추가되는 빈도는 도면 관리자 `freehandInterval` 옵션에 의해 정의됩니다.
 - 혼합 모드
   * 한 줄을 그리면서 원하는 대로 클릭 메서드와 자유형 메서드를 번갈아 가며 진행합니다. 예를 들어 몇 개의 점을 클릭한 다음 마우스를 길게 드래그하여 여러 포인트를 추가한 다음 몇 점을 더 클릭합니다. 

**그리기 완료**
 - 하이브리드/클릭 모드
   * 마지막 지점에서 맵을 두 번 클릭합니다. 
   * 그리기 도구 모음에서 단추를 클릭합니다. 
   * 프로그래밍 방식으로 도면 모드를 설정합니다. 
 - 프리핸드 모드
   * 마우스 버튼 또는 터치 포인트를 놓습니다.
 - 키를 `C` 누릅니다.

**도면 취소**
 - 키를 `Escape` 누릅니다.

**그리기 완료**
 - 마우스 버튼 또는 터치 포인트를 놓습니다.
 - 프로그래밍 방식으로 도면 모드를 설정합니다.

**그리기 완료**

## <a name="keyboard-shortcuts"></a>바로 가기 키

그리기 도구는 키보드 단축키를 지원합니다. 이러한 키보드 단축키는 맵에 포커스가 있을 때 작동합니다.

| Key      | 작업                            |
|----------|-----------------------------------|
| `C` | 진행 중인 드로잉을 완료하고 드로잉 모드를 유휴 상태로 설정합니다. 포커스가 최상위 맵 요소로 이동합니다.  |
| `Escape` | 진행 중인 도면을 취소하고 드로잉 모드를 유휴 상태로 설정합니다. 포커스가 최상위 맵 요소로 이동합니다.  |
| `F` | 마우스가 맵 위에 있는 경우 점, 선 또는 다각형에 좌표를 추가합니다. 클릭 또는 하이브리드 모드에서 맵을 클릭하는 것과 동등한 동작입니다. 이 바로 가기를 사용하면 보다 정확하고 빠른 도면을 사용할 수 있습니다. 한 손으로 마우스를 배치하고 다른 한 손을 사용하여 마우스를 누르는 제스처에서 마우스를 이동하지 않고 버튼을 누를 수 있습니다. |
