# <a name="interact-with-the-map---mouse-events"></a>맵과 상호 작용 - 마우스 이벤트

이 문서에서는 [addEventListener](https://docs.microsoft.com/javascript/api/azure-maps-control/atlas.map?view=azure-iot-typescript-latest#addeventlistener)를 사용하여 대화형 맵을 만드는 방법을 보여 줍니다.
