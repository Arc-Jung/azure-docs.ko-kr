#Azure 지원은 어떻게 구입하나요?
Azure는 클라우드에서 이제 막 시작한 개발자에서부터 사업에 결정적인 응용 프로그램을 배포하는 기업체까지 모든 사용자에게 유연한 지원 옵션을 제공합니다. 청구 지원 및 구독 관리 문제는 무료로 제공되지만 Azure 기술 지원을 받으려면 Azure 지원 구독을 구입해야 합니다.

구입 가능한 Azure 구독에 대한 자세한 정보는 [여기](https://azure.microsoft.com/support/legal/offer-details/)를 클릭하세요. 기술 지원을 구매하려면 [Azure 지원 계획 페이지](https://azure.microsoft.com/support/plans/)를 방문하세요.

자가 진단 리소스 또한 사용 가능합니다. 여기에서 Azure 지원 [FAQ](https://azure.microsoft.com/support/faq/)를 볼 수 있고 모든 Azure.com 페이지의 문서 탭에서 지원 기사에 액세스할 수 있습니다.
