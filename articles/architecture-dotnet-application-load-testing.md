# Azure의 부하 테스트

클라우드 기반 부하 테스트 서비스는 웹 응용 프로그램의 부하를 테스트할 수 있습니다. Visual Studio에서 부하 테스트를 만들어 웹 응용 프로그램의 용량과 성능을 확인합니다. 고유한 Azure VM을 사용할 필요가 없습니다. 대신, [클라우드 기반 부하 테스트 서비스][]에서 자동으로 VM을 만들도록 할 수 있습니다.
