# <a name="what-is-datadog"></a>Datadog 란?

## <a name="overview"></a>개요

Datadog는 대규모 응용 프로그램에 대 한 모니터링 및 분석 플랫폼입니다. 인프라 모니터링, 응용 프로그램 성능 모니터링, 로그 관리 및 사용자 환경 모니터링이 포함 됩니다. Datadog는 전체 스택에 데이터를 집계 하 여 문제 해결, 경고 및 그래프를 위한 400 개 이상의 통합을 포함 합니다. 이는 문제 해결, 성능 최적화 및 팀 간 공동 작업을 위한 단일 소스로 사용할 수 있습니다.

Azure Marketplace에서 제공 하는 Datadog의 기능을 사용 하면 Azure 콘솔의 Datadog를 통합 서비스로 관리할 수 있습니다. 이 가용성은 간소화 된 워크플로를 통해 클라우드 워크 로드에 대 한 모니터링 솔루션으로 Datadog을 구현할 수 있습니다. 워크플로는 조달부터 구성까지 모든 것을 포함 합니다. 온 보 딩 환경은 전적으로 Azure에 기반 하거나 하이브리드 또는 다중 클라우드 환경에 걸쳐 있든 상관 없이 응용 프로그램의 상태 및 성능 모니터링을 시작 하는 방법을 간소화 합니다.

이라는 리소스 공급자를 통해 Datadog 리소스를 프로 비전 합니다 `Microsoft.Datadog` . [Azure Portal](https://portal.azure.com/)를 통해 Datadog 조직 리소스를 만들고 프로 비전 하 고 관리할 수 있습니다. Datadog는 조직 및 API 키를 비롯 한 SaaS (software as a service) 응용 프로그램을 소유 하 고 실행 합니다.

## <a name="capabilities"></a>기능

Datadog를 Azure와 통합 하면 다음과 같은 기능을 제공 합니다.

- **통합 온 보 딩** -Datadog는 Azure의 통합 서비스입니다. Datadog를 프로 비전 하 고 Azure Portal를 통해 통합을 관리할 수 있습니다.
- **통합 청구** -Datadog 비용은 Azure 월간 청구를 통해 보고 됩니다.
- **Datadog에 대 한 Single sign-on** -Datadog 포털에 대 한 별도의 인증이 필요 하지 않습니다.
- **로그 전달자** -Datadog에 구독 활동 및 리소스 로그를 자동으로 전달할 수 있습니다.
- **메트릭 수집** -모든 Azure 리소스 메트릭을 자동으로 Datadog에 보냅니다.
- **Datadog 에이전트 배포** -Datadog 에이전트의 통합 관리 환경을 제공 합니다. Datadog 에이전트를 Virtual Machines 및 Azure 앱 서비스에 대 한 확장으로 설치 하 고 제거 합니다.
