# <a name="azure-ad-domain-services---contact-us"></a>Azure AD Domain Services - 문의

## <a name="feedback"></a>사용자 의견
서비스 또는 비기술적 질문에 대한 피드백이 있는 경우 Azure AD 도메인 서비스 피드백: Azure [AD 도메인 서비스](mailto:aaddsfb@microsoft.com)피드백으로 이메일을 보내 요청을 Azure AD 도메인 서비스 제품 그룹과 공유할 수 있습니다. 이 주소로 보낸 이메일은 검토됩니다. 제품 그룹의 구성원은 추가 정보 요청에 응답할 **수 있습니다.** 이 이메일 주소에 대한 기술 지원 요청은 응답되지 않습니다.

## <a name="technical-assistance"></a>기술 지원
관리된 도메인과 관련된 문제가 발생하는 경우 이 [문제 해결 가이드](troubleshoot.md) 에 설명된 단계가 문제를 해결하는지 확인해 주세요. 그래도 문제가 있는 경우 Azure 포털에 로그인합니다. Azure **AD 도메인 서비스** 홈 페이지를 열고 **지원 + 문제 해결**아래의 작업 창에서 새 지원 **요청을** 선택합니다.

![새 지원 요청](./media/contact-us/supportRequest.png) 

지원 요청의 각 섹션을 가능한 한 정확하게 작성합니다. 서비스를 선택할 때 Azure **Active Directory 도메인 서비스(VM - 도메인 컨트롤러)를** 선택하여 요청을 적절한 지원 전문가에게 라우팅해야 합니다.

> [!IMPORTANT]
> Azure Active Directory를 사용 하 고 도메인 유효성 검사에 대 한 도움말을 위해 서비스 목록에서 **Azure Active 디렉터리, 도메인 및 개체를** 선택 합니다.
>
> 

## <a name="feature-requests"></a>기능 요청
Azure Active Directory 도메인 서비스를 개선하는 방법에 대한 아이디어가 있습니까? 우리는 그것에 대해 듣고 싶습니다. Azure 피드백 포럼으로 이동하여 [도메인 서비스](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) 범주로 이동하여 아이디어를 공유합니다. 제품 그룹의 구성원은 이러한 요청을 검토합니다.
