# 10분 내 Azure 백업 시도
Azure 백업 설치는 매우 쉬우며, 다음과 같은 방법 동영상도 시작하는 데 도움이 됩니다.

## 1단계: Azure 구독하기
- Azure 구독이 없는 경우 모든 Azure 서비스에 액세스할 수 있게 해주는 무료 평가판으로 시작할 수 있습니다. 자세한 내용은 다음 링크 [http://azure.microsoft.com/pricing/free-trial/](http://azure.microsoft.com/pricing/free-trial)를 참조하세요.
- Azure 구독이 있는 경우 이 단계를 건너뛸 수 있습니다.

## 2단계: Azure 백업 서비스 구성
[Azure 백업 시작하기 1/3- Azure에 백업 자격 증명 모음 설정](http://azure.microsoft.com/documentation/videos/getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure/) 비디오에 설명한 단계를 진행하려면 다음을 수행하세요.

- 백업 자격 증명 모음 만들기
- "저장소 자격 증명" 다운로드
- Azure 백업 에이전트 다운로드

> [AZURE.VIDEO getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure]

## 3단계: 온-프레미스 컴퓨터 준비
[Azure 백업 시작하기 2/3- Azure 백업을 위한 프로덕션 서버 준비](http://azure.microsoft.com/documentation/videos/getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup/) 비디오에 설명한 단계를 진행하려면 다음을 수행하세요.

- Windows Server/클라이언트 또는 DPM 서버에 Azure 백업 에이전트 설치
- 백업 자격 증명 모음에 대해 컴퓨터 등록

> [AZURE.VIDEO getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup]

## 4단계: 데이터 백업 및 복원
[Azure 백업 시작하기 3/3- 프로덕션 서버 준비 백업 시작](http://azure.microsoft.com/documentation/videos/getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server/) 비디오에 설명한 단계를 진행하려면 다음을 수행하세요.

- 서버에서 데이터 백업
- Azure에서 백업된 데이터 복원

> [AZURE.VIDEO getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server]
