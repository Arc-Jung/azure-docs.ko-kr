# <a name="upgrade-a-backup-vault-to-a-recovery-services-vault"></a>Recovery Services 자격 증명 모음으로 백업 자격 증명 모음 업그레이드

이 문서에서는 Recovery Services 자격 증명 모음으로 백업 자격 증명 모음을 업그레이드하는 방법을 설명합니다. 업그레이드 프로세스는 실행 중인 백업 작업에 영향을 주지 않고 백업 데이터는 손실되지 않습니다. Recovery Services 자격 증명 모음으로 백업 자격 증명 모음을 업그레이드하는 주요 목적은 다음과 같습니다.
- Recovery Services 자격 증명 모음에 백업 자격 증명 모음의 기능이 모두 유지됩니다.
- Recovery Services 자격 증명 모음은 보안 향상, 모니터링 통합, 빠른 복원 및 항목 수준의 복원, 향상되고 간소화된 포털에서 백업 항목 관리를 포함하여 백업 자격 증명 모음보다 더 많은 기능을 제공합니다.
-  Recovery Services 자격 증명 모음에 대한 새로운 기능만이 릴리스됩니다.

## <a name="how-to-upgrade"></a>업그레이드 방법

백업 자격 증명 모음을 Recovery Services 자격 증명 모음으로 업그레이드해야 한다는 상당한 요구가 있습니다. 이 프로세스를 간소화하기 위해 일괄 처리로 업그레이드 큐를 출시합니다. 업그레이드 큐의 항목을 등록하려면 해당 링크를 사용합니다.
- [백업 자격 증명 모음 업그레이드](https://www.surveymonkey.com/r/Y57BJQX)
- [Site Recovery 자격 증명 모음 업그레이드](https://www.surveymonkey.com/r/5HHPZQN)

## <a name="what-happens-after-i-sign-up"></a>등록 후에는 어떻게 되나요?
구독이 업그레이드의 허용 목록에 추가되면 Microsoft에서 사용자에게 연락하여 업그레이드를 진행합니다. 업그레이드 프로세스에 대한 자세한 내용은 Azure Backup [블로그](http://azure.microsoft.com/blog/upgrade-classic-backup-and-siterecovery-vault-to-arm-recovery-services-vault)를 참조하세요.
