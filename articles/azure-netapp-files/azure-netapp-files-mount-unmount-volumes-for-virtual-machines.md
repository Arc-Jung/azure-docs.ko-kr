# <a name="mount-or-unmount-a-volume-for-windows-or-linux-virtual-machines"></a>Windows 또는 Linux 가상 머신에 대한 볼륨 탑재 또는 탑재 해제 

필요에 따라 Windows 또는 Linux 가상 머신에 대한 볼륨을 탑재 또는 탑재 해제할 수 있습니다.  Linux 가상 머신에 대한 탑재 지침은 Azure NetApp Files에서 사용할 수 있습니다.  

1. 클릭 합니다 **볼륨** 블레이드에서 다음 탑재 하려는 볼륨을 선택 합니다. 
2. 클릭 **탑재 지침** 선택한 볼륨에서 볼륨을 탑재 하려면 지침을 따릅니다. 

    ![NFS 탑재 지침](../media/azure-netapp-files/azure-netapp-files-mount-instructions-nfs.png)

    ![SMB 탑재 지침](../media/azure-netapp-files/azure-netapp-files-mount-instructions-smb.png)
