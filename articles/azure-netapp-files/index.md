# <a name="azure-netapp-files-documentation"></a>Azure NetApp Files 설명서

Azure NetApp Files를 사용하여 NetApp 계정, 용량 풀 및 볼륨을 만들고 관리하며, 내보내기 정책을 구성하고, 가상 머신에 대한 볼륨을 탑재하고, 스냅숏을 관리하는 방법을 알아봅니다.

> [!IMPORTANT] 
> 이 설명서에 설명된 모든 작업을 수행하려면 사용자가 공개 미리 보기 프로그램에 속해 있고 Microsoft.NetApp Azure 리소스 공급자에 액세스하는 허용 목록에 지정되어야 합니다. 공개 미리 보기 프로그램에 조인하도록 요청하려면 [NetApp Azure Files에 등록](azure-netapp-files-register.md)을 참조하세요.
