# <a name="azure-data-catalog-developer-samples"></a>Azure Data Catalog 개발자 샘플

데이터 카탈로그 REST API를 사용하여 Azure 데이터 카탈로그 앱 개발을 시작했습니다. 데이터 카탈로그 REST API는 데이터 카탈로그 리소스에 대한 등록, 주석 달기 및 데이터 자산을 프로그래밍 방식으로 검색하기 위해 프로그래밍 방식으로 액세스를 제공하는 REST 기반 API입니다.

## <a name="samples-available-on-githubcom"></a>GitHub.com 샘플 제공

* [Azure Azure Data Catalog 시작](https://github.com/Azure-Samples/data-catalog-dotnet-get-started/)
  
   시작 하기 샘플에서는 Azure AD를 사용하여 데이터 카탈로그 REST API를 사용하여 데이터 자산을 등록, 검색 및 삭제하는 방법을 보여 줍니다.
   
* [서비스 주체를 사용하여 Azure 데이터 카탈로그 시작](https://github.com/Azure-Samples/data-catalog-dotnet-service-principal-get-started/)

   이 샘플에서는 데이터 카탈로그 REST API를 사용하여 데이터 자산을 등록, 검색 및 삭제하는 방법을 보여 주며, 이 샘플에서는 데이터 자산을 등록, 검색 및 삭제하는 방법을 보여 주며, 이 샘플에서는 서비스 주체 인증을 사용합니다.

* [Azure 데이터 카탈로그에 대한 가져오기/내보내기 도구](https://github.com/Azure-Samples/data-catalog-dotnet-import-export/)

   이 샘플에서는 데이터 카탈로그 REST API를 사용하여 Azure 데이터 카탈로그에서 자산을 가져오고 파일로 직렬화하는 방법을 보여 주는 이 샘플입니다. 또한 JSON으로 직렬화된 자산 집합을 가져오고 카탈로그에 게시하는 방법을 보여 줍니다. 검색 쿼리를 사용하여 카탈로그의 하위 집합 내보내기를 지원합니다.

* [Azure 데이터 카탈로그에 대량 등록 및 추가](https://github.com/Azure-Samples/data-catalog-dotnet-excel-register-data-assets/)
  
   이 샘플에서는 데이터 카탈로그 REST API 및 Open XML을 사용하여 Excel 통합 문서의 데이터 자산을 일괄 등록하는 방법을 보여 주는 이 샘플입니다.
  
* [Azure 데이터 카탈로그에 용어집 용어가져오기](https://github.com/Azure-Samples/data-catalog-bulk-import-glossary/)

   이 샘플에서는 CSV 파일에서 ADC 용어집으로 용어를 가져오는 방법을 보여 줍니다.

* [Azure 데이터 카탈로그로 관계를 대량 가져오기](https://github.com/Azure-Samples/data-catalog-bulk-import-relationship/)

   이 샘플에서는 CSV 파일에서 데이터 카탈로그로 관계 정보를 프로그래밍 방식으로 가져오는 방법을 보여 주며, 이 샘플에서는 관계 정보를 프로그래밍 방식으로 가져올 수 있습니다.

* [Azure 데이터 카탈로그에 관계 게시](https://github.com/Azure-Samples/data-catalog-dotnet-publish-relationships/)

   이 샘플에서는 관계 정보를 데이터 카탈로그에 프로그래밍 방식으로 게시하는 방법을 보여 주며 이 샘플을 보여 주실 수 있습니다.
