# Azure 데이터 카탈로그에 대한 키보드 바로 가기

## 일반 키보드 바로 가기

다음은 Azure 데이터 카탈로그에 대한 키보드 바로 가기입니다.

|원하는 작업|누르기
|---|---
|페이지의 각 컨트롤에 대한 탭|탭
|옵션을 선택합니다.|입력 또는 스페이스바
|페이지 최대화|WIN+위쪽 화살표
|페이지 복원/최소화 | WIN+아래쪽 화살표
|페이지 메뉴 열기| ALT+스페이스바


## 인증 페이지

|원하는 작업|누르기
|---|---
|서비스에 로그인합니다.|ALT+S

## 데이터 원본 선택 페이지

|원하는 작업|누르기
|---|---
|(설정된 경우)다음 페이지로 이동합니다.|ALT+N
|타일에 포커스가 있을 때 선택한 형식을 변경합니다.|왼쪽, 위쪽, 오른쪽 또는 아래쪽 화살표

## 데이터 원본 연결 페이지

|원하는 작업|누르기
|---|---
|(설정된 경우)이전 페이지로 이동합니다.|ALT+P
|연결 단추를 선택합니다.| ALT+C

## 연결 컨텍스트 페이지

|원하는 작업|누르기
|---|---
|등록 단추(설정된 경우).| ALT+R
|미리 보기 확인란을 포함합니다(사용 가능한 경우).|ALT+P
|데이터 프로필 확인란을 포함합니다(사용 가능한 경우).|ALT+D
|각 목록 항목으로 이동합니다.|위쪽 또는 아래쪽 화살표
| 서버 계층 구조, 트리 보기의 노드를 확장합니다. |오른쪽 화살표
| "사용 가능한 개체" 또는 "개체 등록"에 선택된 항목을 이동합니다. | 스페이스바

## 진행률 페이지 게시

|원하는 작업|누르기
|---|---
|취소 단추|ALT+C
|추가 개체 등록 단추| ALT+R
|포털 보기 단추 | ALT + V

## 진행률 페이지 게시

|원하는 작업|누르기
|---|---
|저장 단추(설정된 경우)| ALT+S
|취소 단추|ALT+C
