# <a name="connect-one-identity-safeguard"></a>하나의 ID 보호 장치 연결
One IDENTITY Safeguard 데이터 커넥터는 권한 있는 세션별 대시보드에 대한 보호 기능을 사용하여 표준 CEF(일반 이벤트 형식) 커넥터를 향상시킵니다. 이 커넥터를 사용하여 시각화, 경고, 조사 등을 위해 장치에서 생성된 이벤트를 사용하여 쉽게 시작할 수 있습니다.


## <a name="prerequisites"></a>사전 요구 사항

- Azure Sentinel 작업 영역에 대한 사용 권한을 읽고 씁니다.

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>Azure Sentinel을 하나의 ID 보호 장치에 연결

1. Azure 포털에서 Azure **Sentinel** > 데이터 커넥터로 이동한 다음 **One ID 보호** 커넥터를**선택합니다.**

2. **연결선 열기 페이지를**선택합니다.

3. **One ID 보호** 페이지의 지침을 따르십시오.
