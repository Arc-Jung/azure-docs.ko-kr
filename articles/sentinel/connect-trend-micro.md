# <a name="connect-trend-micro-deep-security"></a>트렌드 마이크로 딥 시큐리티 연결

Trend Micro Deep Security 커넥터를 사용하면 Azure Sentinel에 딥 보안 로그를 쉽게 연결하여 대시보드를 보고, 사용자 지정 경고를 만들고, 조사를 개선할 수 있습니다. 이를 통해 조직의 네트워크 및 시스템에 대한 더 많은 통찰력을 얻고 보안 운영 기능을 개선할 수 있습니다.


## <a name="prerequisites"></a>사전 요구 사항

- Azure Sentinel 작업 영역에 대한 사용 권한을 읽고 씁니다.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Azure 센티넬을 트렌드 마이크로 심층 보안에 연결

1. Azure 포털에서 Azure **Sentinel** > **데이터 커넥터로** 이동한 다음 **추세 마이크로 딥 보안** 커넥터를 선택합니다.

2. **연결선 열기 페이지를**선택합니다.

3. 지침및 다음 단계를 따르십시오.
