# <a name="connect-citrix-analytics-security"></a>Citrix Analytics 연결(보안) 

Azure Sentinel과의 Citrix Analytics(보안) 통합을 사용하면 Citrix Analytics(보안)에서 위험한 이벤트에 대해 분석된 데이터를 Azure Sentinel으로 내보낼 수 있습니다. 사용자 지정 대시보드를 만들고, Citrix Analytics(보안)의 데이터와 함께 다른 소스의 데이터를 분석하고, 논리 앱을 사용하여 보안 이벤트를 모니터링하고 완화하는 사용자 지정 워크플로를 만들 수 있습니다. 



## <a name="prerequisites"></a>사전 요구 사항

- Azure Sentinel 작업 영역에 대한 사용 권한을 읽고 씁니다.

- Citrix 클라우드에서 Citrix 보안 분석에 대한 자격.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Azure 센티넬을 Citrix 분석(보안)에 연결

1. Azure 포털에서 Azure **Sentinel** > 데이터 커넥터로 이동한 다음 **Citrix 분석(보안)** 커넥터를**선택합니다.**

2. **연결선 열기 페이지를**선택합니다.

3. **Citrix 분석(보안)** 페이지의 지침을 따르십시오.
