# 클라우드 및 프레미스로 확장되는 하이브리드 응용 프로그램 사용

하이브리드 클라우드/온-프레미스 응용 프로그램은 현대 IT 환경의 현실입니다. 서비스 버스는 기존의 온-프레미스 자산을 활용하고 현재 인프라를 확대하고 확장하는 안정적인 보안 솔루션을 만들도록 도와줍니다.
