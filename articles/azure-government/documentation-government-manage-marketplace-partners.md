# <a name="azure-government-marketplace"></a>Azure Government Marketplace
Azure Marketplace for Government에 제품을 게시하려는 파트너는 아래 세부 정보를 참조하세요.

## <a name="publishing"></a>게시
> [!NOTE]
> 기존 Azure Certified Marketplace 파트너가 아닌 경우 계속하기 전에 [이](../marketplace-publishing/marketplace-publishing-getting-started.md) 단계를 완료하십시오.
> 
> 

### <a name="step-1"></a>1단계
[https://publish.windowsazure.com](https://publish.windowsazure.com)에서 로그온합니다.

### <a name="step-2"></a>2단계
게시하려는 제품을 클릭합니다.

### <a name="step-3"></a>3단계
**SKUS**를 클릭하고 Azure Government Cloud 상자를 클릭합니다.

> [!NOTE]
> BYOL(사용자 라이선스 필요) SKU만 지원됩니다.  이 옵션은 PayG(종량제) SKU로 사용할 수 없습니다.
> 
> 

![대체 텍스트](./media/government-manage-marketplace-partner-1.png)

### <a name="step-4"></a>4단계
+ 인증서 추가 링크를 클릭하여 제품 인증에 대한 링크를 추가합니다.

![대체 텍스트](./media/government-manage-marketplace-partner-2.png)

### <a name="step-5"></a>5단계
게시 포털 [https://azuregov.microsoft.com/trial/azuregovtrial](https://azuregov.microsoft.com/trial/azuregovtrial)에서 이미지를 테스트할 수 있도록 Microsoft Azure Government 클라우드의 평가판 계정을 요청합니다.

미국 연방 정부, 주정부, 지방 정부 또는 소규모 조직을 지원하는 파트너 자격을 확인한 후 전자 메일을 통해 확인서를 발급합니다.  평가판 계정은 영업일 기준 3-5일 이내 사용할 수 있습니다.

### <a name="step-6"></a>6단계
게시를 클릭하고 스테이징으로 푸시를 클릭합니다. 

![대체 텍스트](./media/government-manage-marketplace-partner-3.png)

스테이징 제품에 대한 액세스 권한이 있는 허용 목록 구독을 입력하라는 메시지가 표시됩니다. 새로 얻은 Azure Government 평가판 계정에서 구독 ID를 입력합니다.

![대체 텍스트](./media/government-manage-marketplace-partner-4.png)

### <a name="step-7"></a>7단계
제품이 성공적으로 스테이징되면 Azure Government 평가판 계정을 사용하여 [https://portal.azure.us](https://portal.azure.us)에 로그인하고 이미지를 테스트할 수 있습니다.

### <a name="step-8"></a>8단계
평가판 구독을 사용하여 이미지를 검증한 후에는 게시를 클릭하고 프로덕션 승인을 요청하여 제품을 공급할 수 있습니다. 

![대체 텍스트](./media/government-manage-marketplace-partner-5.png)

## <a name="next-steps"></a>다음 단계
