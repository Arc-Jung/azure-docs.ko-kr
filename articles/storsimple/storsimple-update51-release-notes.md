# <a name="storsimple-8000-series-update-51-release-notes"></a>StorSimple 8000 시리즈 업데이트 5.1 릴리스 정보

## <a name="overview"></a>개요

다음 릴리스 노트에서는 새로운 기능에 대해 설명하고 StorSimple 8000 시리즈 업데이트 5.1의 중요한 공개 문제를 식별합니다. 또한 이 릴리스에 포함된 StorSimple 소프트웨어 업데이트 목록을 포함합니다.

업데이트 5.1은 업데이트 5를 실행하는 StorSimple 장치에 적용할 수 있습니다. 5보다 낮은 버전을 사용하는 경우 먼저 업데이트 5를 적용한 다음 5.1을 적용합니다. 업데이트 5.1과 관련된 장치 버전은 6.3.9600.17885입니다.

StorSimple 솔루션에 업데이트를 배포하기 전에 릴리스 정보에 포함된 정보를 검토하세요.

> [!IMPORTANT]
>
> * 업데이트 5.1은 필수 업데이트이며 즉시 설치해야 합니다. 자세한 내용은 [5.1 업데이트를 적용하는](storsimple-8000-install-update-51.md)방법을 참조하세요.
> * 업데이트 5.1에는 보안 업데이트만 있습니다. 이 업데이트를 설치하는 데 약 30분이 걸립니다. 장치의 작동을 보장하기 위해 업데이트 5.1을 적용하는 것이 좋습니다.
> * 새 릴리스의 경우, 업데이트의 단계적 롤아웃을 수행하기 때문에 즉시 업데이트를 볼 수는 없습니다. 이러한 업데이트가 곧 제공될 예정이니 몇 일 후에 업데이트를 다시 검색하세요.

## <a name="whats-new-in-update-51"></a>5.1 업데이트의 새로운 내용

업데이트 5.1에서 다음과 같은 주요 개선 사항 및 버그 수정이 이루어졌습니다.

* **TLS 1.2** - 이 StorSimple 업데이트는 모든 클라이언트에서 TLS 1.2를 적용합니다. 이것은 모든 StorSimple 8000 시리즈 장치에 대한 필수 업데이트입니다.

## <a name="known-issues-in-update-51-from-previous-releases"></a>이전 릴리스의 업데이트 5.1에서 알려진 문제

업데이트 5.1에는 알려진 새로운 문제가 없습니다. 이전 릴리스의 5.1 업데이트로 이월된 문제 목록은 [업데이트 3 릴리스 정보로](storsimple-update3-release-notes.md#known-issues-in-update-3)이동하십시오.

## <a name="storsimple-cloud-appliance-updates-in-update-51"></a>5.1 업데이트에서 StorSimple 클라우드 어플라이언스 업데이트

이 업데이트는 StorSimple 클라우드 어플라이언스에 적용할 수 없습니다(가상 디바이스라고도 함). 업데이트 5.1 이미지를 사용하여 새 클라우드 어플라이언스를 만들어야 합니다. StorSimple Cloud Appliance를 만드는 방법에 대한 자세한 내용은 [StorSimple Cloud Appliance 배포 및 관리](storsimple-8000-cloud-appliance-u2.md)로 이동합니다.
