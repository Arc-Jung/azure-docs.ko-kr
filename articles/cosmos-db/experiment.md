# <a name="azure-cosmos-db-documentation-experiment"></a>Azure Cosmos DB 설명서 실험

Azure Cosmos DB는 전 세계에 배포된 다중 모델 데이터베이스입니다. 그래프 만들기부터 문서 데이터베이스 만들기 또는 MongoDB, Cassandra, 또는 Azure Table Storage에서 데이터 마이그레이션까지 Azure Cosmos DB로 많은 것을 할 수 있기 때문에 설명서를 조직하는 것은 도전일 수 있습니다. 따라서 도움이 필요합니다.

다음 이미지에 표시된 것처럼 Azure Cosmos DB 목차 및 [방문 페이지](https://docs.microsoft.com/en-us/azure/cosmos-db/) 조직을 테스트하고 이러한 제안된 변경에 대해 사용자 피드백을 가져오려고 합니다.

## <a name="to-load-both-views"></a>두 보기를 로드하려면

1. 두 보기를 나란히 열려면 즐겨 찾는 웹 브라우저에서 [방문 페이지](https://docs.microsoft.com/en-us/azure/cosmos-db/)로 이동합니다. 
2. 위의 이미지를 사용하여 원본 또는 실험본 중 어떤 버전인지 확인합니다. 
3. inprivate 또는 incognito 브라우저 창을 열고 새 브라우저 창에서 [방문 페이지](https://docs.microsoft.com/en-us/azure/cosmos-db/)를 로드하고 다시 이미지를 사용하여 어떤 버전인지 확인합니다. 

   두 보기가 존재하지 않는 경우 새 inprivate 또는 incognito 창을 닫았다가 다시 열고 [방문 페이지](https://docs.microsoft.com/en-us/azure/cosmos-db/) 다시 로드합니다. 별도 창에 로드된 두 버전을 가져오려면 몇 번 시도해야 할 수 있습니다. Incognito 창을 새로 고치기하면 다른 버전을 로드하지 않아 새 창을 열어야 합니다. 

4. 창을 크기 조정하여 비교할 보기 사이를 전환하거나 나란히 수정할 수 있습니다. 

5. 실험적 경험의 레이아웃 및 조직과 친숙해질 기회를 가진 후 아래의 **실험 설문 조사** 단추를 클릭하고 피드백을 제공합니다. 피드백을 검토하고 통합할 수 있도록 2018년 4월 4일 자정 이전에 양식을 제출합니다.
