## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Linkurious 로고](./media/graph-visualization/linkurious-logo.jpg)

Linkurious 엔터프라이즈 그래프 기술 및 데이터 시각화를 사용 하 여 대화형 시각적 네트워크에 복잡 한 데이터 집합 설정. 플랫폼에는 데이터 원본에 연결 하 고 원활 하 게 수십억 개의 엔터티 및 관계를 탐색할 수 investigators를 활성화 합니다. 결과 쿼리 또는 테이블을 사용 하 여 저글링 없이 의심 스러운 관계를 검색 하는 새 기능입니다.

Linkurious Enterprise의 대화형 시각화 인터페이스는 복잡 한 연결 된 데이터를 조사 하는 쉬운 방법을 제공 합니다. 숨겨진된 관계를 파악 하 고 복잡 한 네트워크 분리 원하는 레이아웃을 적용 하는 연결 확장을 특정 엔터티를 검색 합니다. Linkurious는 Azure Cosmos DB Gremlin API와 호환 됩니다. 엔드-투-엔드 그래프 시각화 시나리오에 적합 하 고 읽기 및 쓰기 기능 사용자 인터페이스에서 지원 합니다.
