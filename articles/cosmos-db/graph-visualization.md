# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Azure Cosmos DB Gremlin API에 저장된 그래프 데이터 시각화 데이터 시각화 솔루션

다양한 데이터 시각화 솔루션을 사용하여 Azure Cosmos DB Gremlin API에 저장된 데이터를 시각화할 수 있습니다. 그래프 데이터 시각화를 위해 [아파치 Tinkerpop 커뮤니티에서](https://tinkerpop.apache.org/#poweredby) 다음 솔루션을 권장합니다.

## <a name="linkurious-enterprise"></a>링커러스 엔터프라이즈

![링커러스 로고](./media/graph-visualization/linkurious-logo.jpg)

[Linkurious Enterprise는](https://linkurio.us/product/) 그래프 기술과 데이터 시각화를 사용하여 복잡한 데이터 세트를 대화형 시각적 네트워크로 전환합니다. 이 플랫폼은 데이터 원본에 연결되며 조사관이 수십억 개의 엔터티와 관계를 원활하게 탐색할 수 있도록 합니다. 그 결과 쿼리 나 테이블을 저글링하지 않고도 의심스러운 관계를 감지 할 수있는 새로운 기능이 생겼습니다.

Linkurious Enterprise의 대화형 인터페이스는 복잡한 데이터를 쉽게 조사할 수 있는 방법을 제공합니다. 특정 엔터티를 검색하고, 연결을 확장하여 숨겨진 관계를 발견하고, 원하는 레이아웃을 적용하여 복잡한 네트워크를 풀 수 있습니다. 링커러스 엔터프라이즈는 이제 Azure 코스모스 DB 그렘린 API와 호환됩니다. 종단 간 그래프 시각화 시나리오에 적합하며 사용자 인터페이스의 읽기 및 쓰기 기능을 지원합니다. [Azure 코스모스 DB와 링큐어스의 데모를](https://linkurio.us/contact/) 요청할 수 있습니다.

![링커러스 데모](./media/graph-visualization/linkurious-demo.gif)

## <a name="cambridge-intelligence"></a>케임브리지 인텔리전스

![케임브리지 인텔리전스 로고](./media/graph-visualization/ci-logo.png)

[캠브리지 인텔리전스의](https://cambridge-intelligence.com/products/) 그래프 시각화 도구 키트는 이제 Azure 코스모스 DB를 지원합니다. Azure Cosmos DB에서 다음 두 시각화 도구 키트를 지원합니다.

- [자바 스크립트 개발자를위한 키 라인](https://cambridge-intelligence.com/keylines/)

- [반응 개발자를 위한 다시 그래프](https://cambridge-intelligence.com/regraph/)

![케임브리지 인텔리전스 데모](./media/graph-visualization/ci-demo-2.gif)

이러한 툴킷을 사용하면 사용 사례에 맞는 고성능 그래프 시각화 및 분석 애플리케이션을 설계할 수 있습니다. 강력한 웹 그래픽 라이브러리(WebGL) 렌더링과 신중하게 제작된 코드를 활용하여 사용자에게 빠르고 통찰력 있는 시각화 환경을 제공합니다. 이러한 도구는 모든 브라우저, 장치, 서버 또는 데이터베이스와 호환되며 단계별 자습서, 완전히 문서화된 API 및 대화형 데모와 함께 제공됩니다.

![케임브리지 인텔리전스 데모](./media/graph-visualization/ci-demo-1.gif)
