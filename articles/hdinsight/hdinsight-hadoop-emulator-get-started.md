# Hadoop 샌드박스를 사용하여 Hadoop 에코 시스템에서 시작

Hadoop 에코 시스템에 대해 알아보기 위해 가상 컴퓨터에서 Hadoop 샌드박스(Hortonworks의)를 설정할 수 있습니다. 이 샌드박스는 Hadoop, HDFS(Hadoop Distributed File System), 작업 제출 등을 이해할 수 있는 로컬 개발 환경을 제공합니다. Hortonworks Hadoop 샌드박스를 시작하려면 [Hortonworks 샌드박스](http://hortonworks.com/downloads/#sandbox)를 참조하고 **VM에서 Hortonworks 샌드박스 ** 섹션을 살펴보세요. HDFS, 작업이 클러스터에 제출되는 방법, 클러스터에서 실행되는 작업을 추적하는 방법 등을 이해하려면 샌드박스와 함께 제공되는 자습서를 실행하는 것이 좋습니다.

Hadoop에 익숙해졌으면 HDInsight 클러스터를 만들어 Azure에서 Hadoop 사용을 시작할 수 있습니다. 시작 방법에 대한 자세한 내용은 [HDInsight에서 Hadoop 시작](hdinsight-hadoop-linux-tutorial-get-started.md)을 참조하세요.
