# Azure 포털에서 Azure Multi-Factor Authentication 설정 관리

Azure와 함께 다단계 인증을 사용하는 경우 Azure 포털을 통해 추가 보안 확인 설정을 관리할 수 있습니다.

## Azure 포털에서 추가 보안 확인 설정에 액세스하려면

<ol>
<li>Azure 포털에 로그온합니다. <li>Azure 포털의 위쪽에서 사용자 이름을 클릭합니다. 그러면 드롭다운 상자가 표시됩니다. <li>드롭다운 상자에서 추가 보안 확인을 선택합니다. 그러면 추가 보안 확인 섹션이 열립니다.

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/changesettings.png)</center>

<li>이제 검사 페이지로 이동되며 설정을 변경할 수 있습니다.</li>
