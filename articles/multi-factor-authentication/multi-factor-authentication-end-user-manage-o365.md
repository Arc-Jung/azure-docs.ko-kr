# Office 365 포털에서 Azure Multi-Factor Authentication 설정 관리


Office 365와 함께 다단계 인증을 사용하는 경우 Office 365 포털을 통해 추가 보안 확인 설정을 관리할 수 있습니다.

## Office 365 포털에서 추가 보안 확인 설정에 액세스하려면

<ol>

<li>[Office 365 포털](https://login.microsoftonline.com/)에 로그온합니다.</li>
<li>위쪽에서 작은 톱니바퀴 모양의 아이콘을 클릭합니다. Office 365 설정을 나타내는 드롭다운이 표시됩니다.</li>


<center>![Setup](./media/multi-factor-authentication-end-user-manage-o365/o365a.png)</center>

<li>Office 365, 설정을 클릭합니다. 그러면 설정 페이지가 열립니다.</li>
<li>왼쪽에서 추가 보안 확인을 클릭합니다.</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-o365/o365.png)</center>

<li>오른쪽의 Update my phone numbers used for account security(계정 보안에 사용되는 전화 번호 업데이트) 링크를 클릭합니다. 새 페이지로 리디렉션됩니다. 휴대폰 또는 모바일 앱 사용 방식을 다시 확인해야 합니다.</li>
<li>이 작업이 완료되면 검사 페이지가 표시되며 설정을 변경할 수 있습니다.</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-manage-o365/addsec.png)</center>
