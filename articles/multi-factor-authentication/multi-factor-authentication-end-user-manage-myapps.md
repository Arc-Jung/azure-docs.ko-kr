# Myapps 포털에서 Azure Multi-Factor Authentication 설정 관리


Office 365와 함께 다단계 인증을 사용하는 경우 Office 365 포털을 통해 추가 보안 확인 설정을 관리할 수 있습니다.

## Myapps 포털에서 추가 보안 확인 설정에 액세스하려면
<ol>
<li>[https://myapps.microsoft.com](https://myapps.microsoft.com)에 로그인합니다.</li>
<li>위쪽에서 프로필을 선택합니다.</li>
<li>사용자 이름을 클릭하고 추가 보안 확인을 선택합니다.</li>

<li>이제 검사 페이지로 이동되며 설정을 변경할 수 있습니다.</li>
