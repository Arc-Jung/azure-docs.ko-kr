# Azure Multi-Factor Authentication에 대해 Office 365 포털에서 앱 암호 만들기

Office 365에서 다단계 인증을 사용하는 경우 Office 365 포털을 통해 앱 암호를 만들 수 있습니다.

## Office 365 포털에서 앱 암호를 만들려면
--------------------------------------------------------------------------------

<ol>
<li>Office 365 포털에 로그인</li>
<li>오른쪽 위 구석에서 설정 위젯을 선택합니다.</li>
<li>왼쪽에서 추가 보안 확인을 선택합니다.</li>

![클라우드](./media/multi-factor-authentication-end-user-app-passwords-create-o365/security.png)

<li>오른쪽의 **Update my phone numbers used for account security(계정 보안에 사용되는 전화 번호 업데이트)**를 선택합니다.</li>
<li>검사 페이지 위쪽에서 앱 암호를 선택합니다.</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-app-passwords-create-o365/addsec.png)</center>

<li>**만들기**를 클릭합니다.</li>


<center>![Cloud](./media/multi-factor-authentication-end-user-app-passwords-create-o365/apppass.png)</center>

<li>앱 암호의 이름을 입력하고 **다음**을 클릭합니다.</li>
<li>앱 암호를 클립보드에 복사하고 앱에 붙여 넣습니다.</li>
