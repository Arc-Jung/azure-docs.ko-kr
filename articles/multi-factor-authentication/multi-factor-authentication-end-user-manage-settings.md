# Azure Multi-Factor Authentication 설정 변경
다음 정보는 다단계 인증 설정을 관리하는 데 도움을 주기 위한 것입니다.

## 설정 변경 방식은 다단계 인증을 사용하는 방법에 따라 달라집니다.
다단계 인증 설정을 변경할 수 있는 여러 위치가 있습니다. 다단계 인증을 사용하는 방법에 따라 이동 위치가 달라집니다. 본인에게 가장 적합한 위치를 선택하세요.

다단계 인증을 사용하는 방법|설명
:------------- | :------------- | 
[Office 365에서 사용](multi-factor-authentication-end-user-manage-o365.md)| Office 365 포털을 통해 설정을 변경할 수 있습니다.
[Microsoft Azure에서 사용](multi-factor-authentication-end-user-manage-azure.md)| Azure 포털을 통해 설정을 변경할 수 있습니다.
[잘 모름](multi-factor-authentication-end-user-manage-myapps.md)|즉, [http://myapps.microsoft.com](http://myapps.microsoft.com)에 로그인한 후 여기에서 설정을 변경할 수 있습니다.
