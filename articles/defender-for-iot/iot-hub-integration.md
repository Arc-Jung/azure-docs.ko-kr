# <a name="get-started-with-built-in-iot-hub-integration"></a>기본 제공 IoT Hub 통합 시작

이 옵션을 사용 하면 IoT 보안 에이전트에 대해 Defender를 사용 하지 않고 서비스를 사용할 수 있습니다.

## <a name="enable-built-in-iot-hub-integration"></a>기본 제공 IoT Hub 통합 사용

장치 id 관리, 장치-클라우드 및 클라우드-장치 통신 패턴의 모니터링을 사용 하도록 설정 하려면 다음을 수행 하 여 서비스를 시작 합니다.

1. **IoT Hub**를 엽니다.
1. **보안 개요** 메뉴를 선택 합니다.
1. **IoT 솔루션 보안** 을 클릭 하 고 온 보 딩 폼을 완료 합니다.

축하합니다! IoT Hub에서 IoT 서비스용 Defender를 사용 하도록 설정 했습니다.
