# <a name="security-posture--cis-benchmark"></a>보안 상태 – CIS 벤치 마크 

IoT 마이크로 에이전트의 Defender를 사용 하면 조직이 보안 규정 준수 및 상태를 향상할 수 있습니다. CIS 벤치 마크 기반 OS 기준 권장 사항은 장치 보안 문제를 식별 하 고 보안 강화를 위해 변경의 우선 순위를 지정 하는 데 도움이 됩니다.  

## <a name="best-practices-for-secure-configuration"></a>보안 구성에 대 한 모범 사례

CIS 벤치 마크는 대상 시스템의 구성에 대 한 보안을 유지 하기 위한 모범 사례입니다. CIS 벤치 마크는 전 세계에 사이버 보안 전문가와 실무 전문가로 구성 된 고유한 합의 기반 프로세스를 통해 개발 됩니다. 

CIS 벤치 마크는 정부, 비즈니스, 산업 및 학계에서 개발 하 고 승인한 유일한 합의 기반 보안 구성 가이드입니다.

IoT 마이크로 에이전트 용 Azure Defender를 사용 하면 OS 보안 상태에서 기존 weak 링크를 일정 하 게 식별 하는 것과 함께 CI 모범 사례 구성을 제공 하 여 조직의 장치 보안 및 방어 기능을 신속 하 게 향상 시킬 수 있습니다.
