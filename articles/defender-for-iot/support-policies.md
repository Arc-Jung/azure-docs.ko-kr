# <a name="versioning-and-support-for-azure-defender-for-iot"></a>IoT 용 Azure Defender에 대 한 버전 관리 및 지원 

이 문서에서는 지원, IoT 용 Defender의 주요 변경 정책 및 현재 사용할 수 있는 IoT 용 Azure Defender의 버전을 설명 합니다. 

## <a name="servicing-information-and-timelines"></a>서비스 정보 및 타임라인 

Microsoft는 분기별로 한 번 이상, IoT 용 Azure defender 업데이트를 릴리스할 계획입니다. IoT 용 Azure Defender 센서의 각각의 GA (일반 공급) 버전 및 온-프레미스 관리 콘솔의 IoT 용 Azure Defender는 릴리스 후 최대 9 개월 동안 지원 됩니다. 수정 사항 및 새 기능은 현재 지원 되 고 있는 현재 GA 버전에 적용 되며 이전 GA 버전에는 적용 되지 않습니다.

## <a name="versions-and-support-dates"></a>버전 및 지원 날짜

| 버전 | 출시 날짜 | 종료 지원 날짜 |
|--|--|--|
| 10.0 | 01/2021 | 10/2021 |
