# <a name="introducing-azure-defender-for-iot-preview"></a>IoT 미리 보기용 Azure Defender 소개

> [!IMPORTANT]
> IoT 용 Azure Defender는 현재 공개 미리 보기로 제공 됩니다.
> 이 미리 보기 버전은 서비스 수준 계약 없이 제공되며 프로덕션 워크로드에는 사용하지 않는 것이 좋습니다. 특정 기능이 지원되지 않거나 기능이 제한될 수 있습니다. 자세한 내용은 [Microsoft Azure Preview에 대한 추가 사용 약관](https://azure.microsoft.com/support/legal/preview-supplemental-terms/)을 참조하세요.

기업에서 효율성과 생산성을 높이기 위해 디지털 전환을 구현 하는 경우 보드 및 관리 팀은 많은 수의 IoT (관리 사물 인터넷) 및 IoT (운영 기술) 장치를 배포 하 여 발생 하는 책임과 재무 위험에 대해 점점 우려 하 고 있습니다.
이 확장 된 공격 노출 영역을 대상으로 하는 위협 행위자는 안전 및 환경 인시던트, 비용이 많이 드는 프로덕션 가동 중지 시간, 중요 한 지적 재산 도난 등의 많은 회사 영향을 초래할 수 있습니다

레거시 IoT 및 OT 장치는 에이전트를 지원 하지 않으며 종종 패치가 적용 되지 않고 잘못 구성 되며 IT 팀에 표시 되지 않는 것으로, 회사 네트워크를 더 심층적으로 보고 하는 위협 행위자의 소프트 목표를 만듭니다. 

회사 IT 네트워크용으로 개발 된 기존 네트워크 보안 모니터링 도구는 IoT 및 OT 환경에서 발견 된 특수 프로토콜, 장치 및 컴퓨터 간 (M2M) 동작을 깊이 이해 하지 못하기 때문에 이러한 환경을 처리할 수 없습니다.

IoT 용 Azure Defender는 Azure IoT Hub 또는 레거시 관리 되지 않는 장치를 통해 관리 되는 최신 장치 든 상관 없이 모든 IoT 및 OT 장치에서 IoT 및 OT 위협, 위험 및 취약성을 지속적으로 검색, 모니터링 및 관리 하는 전체적인 솔루션입니다. IoT 용 Azure Defender는 인시던트 응답을 가속화 하 고, 운영 문제에 대 한 통찰력을 제공 하며, 통합 IoT와 OT의 가시성과 제어를 제공 하 여 하이브리드 워크 로드 보호를 간소화 합니다

이 전체적인 솔루션은 다음을 포함 하는 주요 사용 사례를 다룹니다.
- 모든 IoT 및 OT 장치에 대 한 자동 자산 검색
- 연속 IoT 및 기타 위험 및 취약성 관리
- IoT 및 OT 위협 구하기 및 인시던트 응답
- 운영 효율성 

IoT 용 Azure Defender는 모든 장치에서 IoT 및 OT 보안을 위한 통합 솔루션입니다. 또한 Azure IoT Hub를 통해 관리 되는 새 IoT 및 장치에 보안을 구축 하는 도구 뿐만 아니라 관리 되지 않는 IoT 및 OT 자산을 보호 하는 데 필요한 에이전트 없는 네트워크 계층 모니터링 기능을 제공 합니다.

## <a name="unified-visibility-and-control"></a>통합된 표시 유형 및 제어

Azure IoT 솔루션을 포함 하 여 모든 온-프레미스 및 클라우드 워크 로드에서 통합 된 보안 보기를 가져옵니다. 새 디바이스를 온보딩하고, 워크로드(리프 디바이스, Microsoft Edge 디바이스, IoT Hub)에 보안 정책을 적용하여 보안 표준 준수 및 향상된 보안 상태를 보장합니다.

## <a name="adaptive-threat-prevention"></a>적응형 위협 방지

IoT 용 Defender를 사용 하 여 컴퓨터, 네트워크 및 Azure 서비스의 보안을 지속적으로 모니터링 합니다. 수백 개의 기본 제공 보안 평가 중 하나를 선택 하거나 중앙 Defender for IoT Hub 대시보드에 직접 만드세요. 가상 컴퓨터, 네트워크, 앱 및 데이터에 대 한 조치 가능한 권장 사항을 사용 하 여 보안 설정을 최적화 하 고 보안 점수를 높입니다. 새로 추가 된 IoT 기능을 사용 하면 Azure IoT 솔루션에 대 한 공격 노출 영역을 줄이고 문제를 해결할 수 있습니다.

## <a name="intelligent-threat-detection-and-response"></a>지능형 위협 검색 및 대응

고급 분석 및 Microsoft Intelligent Security Graph를 사용 하 여 진화 하는 사이버 공격에 대 한 경계를 가져옵니다. 기본 제공 되는 동작 분석과 machine learning은 공격 및 제로 일 익스플로잇을 식별 합니다. IoT 솔루션에서 들어오는 공격 및 위반 후 활동을 모니터링 합니다. 대화형 도구 및 상황에 맞는 위협 인텔리전스를 사용 하 여 장치 조사 및 업데이트를 간소화 합니다.
