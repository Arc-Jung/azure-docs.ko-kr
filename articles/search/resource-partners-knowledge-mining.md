# <a name="partner-spotlight"></a>파트너 스포트라이트

Azure Cognitive Search를 포함 하는 종단 간 솔루션을 구축 하는 Microsoft 파트너의 전문 도움을 받으세요. 다음 파트너는 다양 한 비즈니스 및 기술 시나리오에서 전체 텍스트 검색 및 AI 보강을 활용 하는 뛰어난 환경을 제공 합니다.
