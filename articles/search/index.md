# <a name="azure-search-documentation"></a>Azure Search 설명서

Azure Search는 서버 및 인프라 관리를 Microsoft에 의한 클라우드 SaaS(Search-as-a-Service) 솔루션으로, 사용자는 데이터를 채운 다음 웹 또는 모바일 응용 프로그램에 검색을 추가하는 데 사용할 수 있는 즉시 사용 가능한 서비스입니다. Azure Search를 사용하면 검색 인프라를 관리할 필요가 없고 검색 전문가가 아니라도 간단한 REST API 또는 .NET SDK를 사용하여 응용 프로그램에 강력한 검색 환경을 쉽게 추가할 수 있습니다.
