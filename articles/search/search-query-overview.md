# Azure 검색의 쿼리
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Fiddler](search-fiddler.md)
- [Postman](search-chrome-postman.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

내장된 검색 환경에 대해 Azure 검색을 사용하는 사용자 지정 솔루션은 검색 쿼리를 생성하여 이를 처리하도록 검색 서비스에 전송하는 코드를 포함해야 합니다.

쿼리 실행을 위한 메서드를 작성하기 위해 .NET SDK 또는 REST API를 사용할 수 있습니다. 예비 테스트 또는 탐색을 위해 Fiddler 또는 Chrome Postman과 같은 도구를 사용하여 쿼리를 서비스로 보낼 수 있습니다.
