# <a name="azure-sql-database-connect-and-query-quickstarts"></a>Azure SQL 데이터베이스 연결 및 쿼리 빠른 시작

다음 표에는 Azure SQL 데이터베이스에 연결하고 쿼리하는 방법을 보여 주는 Azure 예제에 대한 링크가 나와 있습니다.

| |  |
|---|---|
|[SQL Server Management Studio](sql-database-connect-query-ssms.md)|SSMS를 사용하여 Azure SQL 데이터베이스에 연결한 다음, Transact-SQL 문을 사용하여 데이터베이스에서 데이터를 쿼리, 삽입, 업데이트 및 삭제하는 방법을 보여 줍니다.|
|[SQL Operations Studio](https://docs.microsoft.com/sql/sql-operations-studio/quickstart-sql-database?toc=/azure/sql-database/toc.json)|SQL Operations Studio(미리 보기)를 사용하여 Azure SQL 데이터베이스에 연결한 다음, T-SQL(Transact-SQL) 문을 사용하여 SQL Operations Studio(미리 보기) 자습서에 사용되는 TutorialDB를 만드는 방법을 보여 줍니다.|
|[Azure Portal](sql-database-connect-query-portal.md)|쿼리 편집기를 사용하여 SQL 데이터베이스에 연결한 다음, Transact-SQL 문을 사용하여 데이터베이스에서 데이터를 쿼리, 삽입, 업데이트 및 삭제하는 방법을 보여 줍니다.|
|[Visual Studio Code](sql-database-connect-query-vscode.md)|Visual Studio Code를 사용하여 Azure SQL 데이터베이스에 연결한 다음, Transact-SQL 문을 사용하여 데이터베이스에서 데이터를 쿼리, 삽입, 업데이트 및 삭제하는 방법을 보여 줍니다.|
|[Visual Studio에서 .NET 사용](sql-database-connect-query-dotnet-visual-studio.md)|Visual Studio에서 .NET Framework를 통해 Azure SQL 데이터베이스에 연결하고, Transact-SQL 문을 사용하여 데이터를 쿼리하는 C# 프로그램을 만드는 방법을 보여 줍니다.|
|[.NET Core](sql-database-connect-query-dotnet-core.md)|Windows/Linux/macOS에서 .NET Core를 통해 Azure SQL 데이터베이스에 연결하고, Transact-SQL 문을 사용하여 데이터를 쿼리하는 C# 프로그램을 만드는 방법을 보여 줍니다.|
|[Go](sql-database-connect-query-go.md)|Go를 사용하여 Azure SQL 데이터베이스에 연결하는 방법을 보여 줍니다. 데이터 쿼리 및 수정을 위한 Transact SQL 문도 보여 줍니다.|
|[Java](sql-database-connect-query-java.md)|Java를 사용하여 Azure SQL 데이터베이스에 연결한 다음, Transact-SQL 문을 사용하여 데이터를 쿼리하는 방법을 보여 줍니다.|
|[Node.js](sql-database-connect-query-nodejs.md)|Node.js를 사용하여 Azure SQL 데이터베이스에 연결한 다음, Transact-SQL 문을 사용하여 데이터를 쿼리하는 프로그램을 만드는 방법을 보여 줍니다.|
|[PHP](sql-database-connect-query-php.md)|PHP를 사용하여 Azure SQL 데이터베이스에 연결한 다음, Transact-SQL 문을 사용하여 데이터를 쿼리하는 프로그램을 만드는 방법을 보여 줍니다.|
|[Python](sql-database-connect-query-python.md)|Python을 사용하여 Azure SQL 데이터베이스에 연결한 다음, Transact-SQL 문을 사용하여 데이터를 쿼리하는 방법을 보여 줍니다. |
|[Ruby](sql-database-connect-query-ruby.md)|Ruby를 사용하여 Azure SQL 데이터베이스에 연결한 다음, Transact-SQL 문을 사용하여 데이터를 쿼리하는 프로그램을 만드는 방법을 보여 줍니다.|
