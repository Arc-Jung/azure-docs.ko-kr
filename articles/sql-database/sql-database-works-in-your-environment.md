# 사용자 환경에서 작동하는 Azure SQL 데이터베이스
## 이미 보유한 지식과 연결
사용자는 데이터베이스를 사용하여 연결하기 위해 새로운 내용을 배우지 않기 원합니다. Azure SQL 데이터베이스를 사용하면 다양한 언어 및 플랫폼을 사용하는 데 쉽게 익숙해질 수 있습니다.

![works-in-your-environment](./media/sql-database-works-in-your-environment/sql-database-works-in-your-environment.png)

데이터베이스를 선택할 때 새 플랫폼으로 이동하거나 새 언어를 배울 필요가 없어야 합니다. SQL 데이터베이스에는 어디에서든지 시작하는 데 도움이 되는 코드 샘플이 포함되어 있습니다.

[SQL 데이터베이스 개발에 대해 자세히 알아보기](sql-database-develop-overview.md)

## 작업에 대한 적절 한 도구
SQL 데이터베이스를 시작하기 위해 완전히 새로운 도구 집합을 배울 필요가 없습니다. [cheetah](https://github.com/wunderlist/cheetah), [sql-cli](https://www.npmjs.com/package/sql-cli), [VS Code](https://code.visualstudio.com/)와 같은 오픈 소스 도구를 활용할 수 있습니다. 또한 Azure SQL 데이터베이스는 [Visual Studio](https://www.visualstudio.com/visual-studio-homepage-vs.aspx) 및 [SQL Server Management Studio](https://msdn.microsoft.com/library/ms174173.aspx)와 같은 Microsoft 도구로 작동합니다. Azure 관리 포털, PowerShell 및 REST API를 사용하면 추가 생산성을 얻을 수도 있습니다.

## 신속하게 시작하기
SQL 데이터베이스와 관련된 질문에 대한 올바른 답변과 첫 번째 SQL 데이터베이스를 만들고 쿼리 문제를 해결하는 데 대한 자습서 및 빠른 시작 코드 샘플을 쉽게 찾을 수 있습니다.

[SQL 데이터베이스에 대해 자세히 알아보기](sql-database-technical-overview.md)
