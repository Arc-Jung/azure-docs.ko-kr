# Azure SQL 데이터베이스 탄력적 데이터베이스 도구 항목

Azure SQL DB는 탄력적 데이터베이스 기능은 데이터 계층 개발 및 SaaS(서비스) 개발자로서 특히 소프트웨어에 대한 관리를 간소화하도록 설계되었습니다. 많은 수의 데이터베이스는 동적 최종 고객 기반을 지원하는 데 사용되는 위치입니다.

**탄력적 데이터베이스 도구** 구축하고 여러 데이터베이스에 걸쳐 확장 분할 패턴을 활용하는 응용 프로그램을 관리할 수 있습니다. 탄력적 데이터베이스 클라이언트 라이브러리 지원 분할 관리, 데이터 종속 라우팅 및 ADO.Net에 대한 분할된 데이터베이스 간 쿼리 기능이 포함되어 있습니다. 또한 하나의 분할에서 다른 분할(병합)까지 데이터를 이동(분할)할 수 있도록 하여 분할된 관리를 단순화하는 분할 병합 도구도 포함되어 있습니다. 이러한 도구를 개수에 관계 없이 데이터베이스에 대한 보장된 성능 특징을 가진 가격을 예측할 수 있게 하는 탄력적 데이터베이스 풀을 보완하십시오. 연결된 작업 기능을 사용하면 업그레이드 및 T-SQL 스크립트를 통해 풀의 모든 데이터베이스를 관리하기 위한 간편하고 신뢰할 수 있는 도구입니다.
