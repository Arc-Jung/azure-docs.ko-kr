# <a name="view-and-update-an-azure-sql-database-and-its-settings"></a>Azure SQL Database와 해당 설정 보기 및 업데이트

[Azure Portal](sql-database-manage-portal.md), [PowerShell](sql-database-manage-powershell.md), [SQL Server Management Studio](sql-database-manage-azure-ssms.md), REST API 또는 C#을 사용하여 Azure SQL Database 및 해당 설정을 보고 업데이트할 수 있습니다. 

## <a name="view-and-update-sql-database-settings-using-the-azure-portal"></a>Azure Portal을 사용하여 SQL Database 설정 보기 및 업데이트

1. [Azure Portal](https://portal.azure.com/)에서 **SQL Database** 블레이드를 엽니다. 

    ![SQL 데이터베이스](./media/sql-database-get-started/sql-databases.png)

2. 작업하려는 데이터베이스를 클릭한 다음 SQL Database 블레이드에서 원하는 설정을 클릭합니다.

    ![새 샘플 데이터베이스 블레이드](./media/sql-database-get-started/new-sample-db-blade.png)

> [!TIP]
> Azure Portal을 사용하여 자습서 시작하기는 [Azure Portal 및 SQL Server Management Studio를 사용하여 Azure SQL Database 서버, 데이터베이스 및 방화벽 규칙 시작](sql-database-get-started.md)을 참조하세요.
>
