# <a name="view-and-update-an-azure-sql-database-server-and-its-settings"></a>Azure SQL Database 서버와 해당 설정 보기 및 업데이트

[Azure portal]((sql-database-manage-portal.md) [PowerShell](sql-database-manage-powershell.md), REST API 또는 C#를 사용하여 Azure SQL Database 서버와 해당 설정을 보고 업데이트할 수 있습니다. 

## <a name="view-and-update-an-azure-sql-server-settings-using-the-azure-portal"></a>Azure Portal을 사용하여 Azure SQL 서버 설정 보기 및 업데이트
1. [Azure Portal](https://portal.azure.com/)에서 **SQL 서버** 블레이드를 엽니다. 

    ![새 SQL 서버](./media/sql-database-get-started/new-sql-server.png)

2. 작업할 서버를 클릭한 다음 SQL 서버 블레이드에서 원하는 설정을 클릭합니다. 

    ![SQL 서버 블레이드](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Azure Portal을 사용하여 자습서 시작하기는 [Azure Portal 및 SQL Server Management Studio를 사용하여 Azure SQL Database 서버, 데이터베이스 및 방화벽 규칙 시작](sql-database-get-started.md)을 참조하세요.
>
