# Azure SQL 데이터베이스 학습 및 적응
## 응용 프로그램을 사용하여 Azure SQL 데이터베이스 학습, 적응 및 성장 방법
Azure SQL 데이터베이스는 하루에 수많은 트랜잭션 및 수많은 데이터베이스를 관리합니다. 또한 주요 기능 중 하나는 SQL 데이터베이스에서 앱을 사용하여 배우고 적응하는 것입니다. 이런 방식으로 사용자가 성능, 안정성 및 데이터 보안을 동적으로 쉽게 최대화할 수 있습니다.

![learn-and-adapt](./media/sql-database-learn-and-adapt/sql-database-learn-and-adapt.png)

## 위협 탐지 및 경고
위협 및 비정상 검색 기능과 함께 SQL 데이터베이스는 기본 제공된 동작 분석, 실시간 경고, 구성 가능한 위협 정책, 감사 로그 및 이상한 패턴을 감지 및 수정하는 지능형 방법을 사용합니다.

[설명서 읽기](sql-database-threat-detection-get-started.md)

## 자동 조정
SQL 데이터베이스는 앱이 최상의 성능으로 실행되도록 설치되었습니다. 사용자가 원하는 대로 어떤 작업을 수행하지 않고도 지속적으로 응용 앱의 패턴을 학습하고 해당 성능을 적절하게 자동 조정하며 자동으로 구체화할 수 있습니다.

[설명서 읽기](http://go.microsoft.com/fwlink/?LinkID=787566)

## 필요할 경우 유용한 정보
SQL 데이터베이스는 각 쿼리 및 해당 기간, 빈도 및 리소스 사용률을 추적할 수 있습니다. 이 원격 분석에 따라 자동 알고리즘은 사용자 데이터베이스를 쿼리에 정확하게 최적으로 조정합니다. 또한 SQL 데이터베이스는 쿼리를 조정하고 성능 문제를 해결하는 시간을 최소화하기 위해 유용한 정보를 제공합니다. 즉, 자세한 내용은 리소스 사용, 최고 성능 쿼리 및 드릴 다운 기능에 대한 유용할 정보를 직접 파악합니다.

[설명서 읽기](http://go.microsoft.com/fwlink/?LinkID=787567)

## 필요한 관리 없음
SQL 데이터베이스는 코딩을 가장 잘 수행하기 위해 돌아갈 수 있도록 앱에서 필요한 자동 관리 및 데이터 보호를 제공합니다. 자동으로 백업, 재해 복구 장애 조치, 인프라 유지 관리, 보안 및 소프트웨어 패치 및 기능 업데이트가 가능합니다. SQL 데이터베이스가 내부적으로 작동하여 시간과 비용을 절약할 뿐만 아니라 골치 아픈 앱 가동 중지 시간을 걱정할 필요도 없습니다.

[설명서 읽기](http://go.microsoft.com/fwlink/?LinkID=787568)

SQL 데이터베이스에서 제공해야 하는 다른 항목을 보려면 홈 페이지로 이동합니다. [확인해 보세요.](https://azure.microsoft.com/services/sql-database/)
