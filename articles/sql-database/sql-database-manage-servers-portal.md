# <a name="create-and-manage-azure-sql-database-servers-with-the-azure-portal"></a>Azure Portal을 사용하여 Azure SQL Database 서버 만들기 및 관리

[Azure Portal](https://portal.azure.com/), PowerShell, REST API 또는 C#을 사용하여 Azure SQL Database 서버를 만들고 관리할 수 있습니다. 이 항목은 Azure Portal 사용에 대해 설명합니다. PowerShell의 경우 [Powershell을 사용하여 서버 만들기 및 관리](sql-database-manage-servers-powershell.md)를 참조하세요.

## <a name="create-an-azure-sql-database-server-using-the-azure-portal"></a>Azure Portal을 사용하여 Azure SQL Database 서버 만들기

1. [Azure Portal](https://portal.azure.com/)에서 **SQL Servers** 블레이드를 엽니다. 

    ![SQL Server](./media/sql-database-get-started/new-sql-server.png)

2. **추가**를 클릭하여 SQL Server 만들기

    ![새 SQL Server 추가](./media/sql-database-get-started/new-sql-server-add.png)

> [!TIP]
> Azure Portal 및 SQL Server Management Studio를 사용하여 자습서 시작하기는 [Azure Portal 및 SQL Server Management Studio를 사용하여 Azure SQL Database 서버, 데이터베이스 및 방화벽 규칙 시작](sql-database-get-started.md)을 참조하세요.
>

## <a name="view-and-update-azure-sql-server-settings-using-the-azure-portal"></a>Azure Portal을 사용하여 Azure SQL Server 설정 보기 및 업데이트
1. [Azure Portal](https://portal.azure.com/)에서 **SQL 서버** 블레이드를 엽니다. 

    ![새 SQL 서버](./media/sql-database-get-started/new-sql-server.png)

2. 작업할 서버를 클릭한 다음 SQL 서버 블레이드에서 원하는 설정을 클릭합니다. 

    ![SQL 서버 블레이드](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Azure Portal을 사용하여 서버를 만드는 시작 자습서를 보려면 [데이터베이스 만들기 - Azure Portal](sql-database-get-started.md)을 참조하세요.
>
