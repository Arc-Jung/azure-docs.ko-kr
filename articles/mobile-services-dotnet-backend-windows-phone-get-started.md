# <a name="getting-started"> </a>모바일 서비스 시작

이 항목은 더 이상 지원되지 않습니다. Windows Phone 8.0 퀵 스타트 프로젝트는 더 이상 포털에서 다운로드할 수 없습니다. 기존 Windows Phone 8.0 또는 Windows Phone 8.1 Silverlight 앱에 모바일 서비스를 추가하려면 [기존 앱에 모바일 서비스 추가](mobile-services-dotnet-backend-windows-phone-get-started-data.md) 항목을 참조하세요. 

Windows Phone Store 8.1 앱용 퀵 스타트 프로젝트 다운로드는 범용 Windows 프로젝트의 일부로 다운로드할 수 있습니다. 자세한 내용은 [모바일 서비스 시작](mobile-services-dotnet-backend-windows-store-dotnet-get-started.md)을 참조하세요.
