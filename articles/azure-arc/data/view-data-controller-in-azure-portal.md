첫 번째 [메트릭 또는 로그를 azure에 업로드](upload-metrics-and-logs-to-azure-monitor.md) 하거나 [사용을 업로드](view-billing-data-in-azure.md)한 후에는 Azure에서 만든 azure Arc data controller 리소스가 Azure Portal에 표시 될 수 있습니다.
