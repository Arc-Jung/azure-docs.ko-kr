# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Azure Portal에서 인스턴스의 인벤토리 보기

[메트릭, 로그](upload-metrics-and-logs-to-azure-monitor.md)또는 [사용 현황을](view-billing-data-in-azure.md)업로드 한 후 Azure Portal에서 Azure 리소스로 인스턴스를 볼 수 있습니다. Azure Portal에서 리소스를 보려면 특수 URL로 이동 하 여 [https://portal.azure.com](https://portal.azure.com) 다음 단계를 완료 합니다.

1. **모든 서비스**로 이동 합니다.
1. 데이터베이스 인스턴스 유형을 검색 합니다.
1. 즐겨찾기에 형식을 추가 합니다.
1. 왼쪽 메뉴에서 인스턴스 유형을 선택 합니다.
1. 다른 Azure SQL 또는 Azure PostgreSQL Hyperscale 리소스와 동일한 보기에서 인스턴스를 봅니다 (세부적인 보기에 필터 사용).
