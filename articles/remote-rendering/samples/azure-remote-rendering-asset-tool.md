# <a name="azure-remote-rendering-asset-tool-arrt"></a>ARRT(Azure Remote Rendering 자산 도구)

ARRT(Azure Remote Rendering 자산 도구)는 C++/Qt에서 개발한 오픈 소스 데스크톱 애플리케이션입니다.

![ARRT](./media/azure-remote-rendering-asset-tool.png "ARRT 스크린샷")

## <a name="features"></a>기능

애플리케이션을 사용하여 다음 작업을 수행할 수 있습니다.

* 3D 모델 업로드
* 모델 변환 제어
* 원격 렌더링 세션 만들기 및 관리
* 3D 모델 로드
* 3D 모델 미리 보기
* 재질 수정

3D 모델 변환을 관리하기 위해 Azure Storage 클라이언트 라이브러리를 사용하여 Azure Remote Rendering C++ SDK의 프런트 엔드를 구현하는 방법을 알아볼 수 있는 오픈 소스 샘플로 사용할 수도 있습니다.
