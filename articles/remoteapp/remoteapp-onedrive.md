# 비즈니스용 OneDrive 및 Azure RemoteApp

비즈니스용 OneDrive는 Azure RemoteApp에서 지원되지 않습니다.

이 문서는 이전에 Azure RemoteApp과 함께 비즈니스용 OneDrive를 사용하기 위한 다른 구현 옵션을 설명했습니다. 이러한 구현 옵션은 더 이상 지원되지 않습니다.
