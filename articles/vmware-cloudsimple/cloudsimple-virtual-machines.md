# <a name="cloudsimple-virtual-machines-overview"></a>클라우드간단한 가상 머신 개요

CloudSimple을 사용하면 Azure 포털에서 VMware 가상 시스템(VM)을 관리할 수 있습니다.  vSphere 클러스터의 클러스터 또는 리소스 풀은 구독에 매핑하여 Azure를 통해 관리됩니다.

Azure에서 CloudSimple VM을 만들려면 사설 클라우드 vCenter에 VM 템플릿이 있어야 합니다.  템플릿은 운영 체제 및 응용 프로그램을 사용자 지정하는 데 사용됩니다.  템플릿 VM을 강화하여 엔터프라이즈 보안 정책을 충족할 수 있습니다.  템플릿을 사용하여 VM을 만든 다음 셀프 서비스 모델을 사용하여 Azure 포털에서 VM을 사용할 수 있습니다.

## <a name="benefits"></a>이점

Azure 포털의 CloudSimple 가상 컴퓨터는 사용자가 VMware 가상 컴퓨터를 만들고 관리할 수 있는 셀프 서비스 메커니즘을 제공합니다.

* 프라이빗 클라우드 vCenter에서 클라우드 단순 VM 만들기
* VM 속성 관리
  * 디스크 추가/제거
  * NIC 추가/제거
* 클라우드 단순 VM의 전원 운영
  * 전원 켜기 및 전원 끄기
  * VM 다시 설정
* VM 삭제
