# <a name="cloudsimple-node-quota-limits"></a>CloudSimple 노드 할당량 한도

4 개의 노드가 구매할 수 있는 기본 수량 때 CloudSimple 서비스에 대 한 구독 사용 가능 합니다.  모든 구입할 수 있습니다 [노드이므로](cloudsimple-node.md) Azure portal에서 합니다.  동일한 SKU의 세 개 노드의 최소 사설 클라우드를 만드는 데 필요한 합니다.  노드를 구입한 경우 추가 노드를 구입 하려고 할 때 오류가 표시 될 수 있습니다.

## <a name="quota-increase"></a>할당량 증가

지원 요청을 제출 하 여 노드 할당량을 늘릴 수 있습니다. 서비스 운영 팀은 요청을 평가 하 고 노드 할당량을 늘릴 수 있습니다.  새 티켓을 열 때 다음 옵션을 선택 합니다.

* 문제 유형: **기술**
* 구독: **구독 ID**
* 서비스 유형: **CloudSimple VMware 솔루션**
* 문제 유형: **전용된 노드 할당량**
* 문제가 하위 유형: **전용된 노드 할당량을 늘리려면**
* 주체: **할당량 증가**

필요한 수의 노드 및 노드 SKU에 지원 티켓의 세부 정보를 제공 합니다.

Microsoft 계정 담당자에 게 문의할 수 있습니다 [ azurevmwaresales@microsoft.com ](mailto:azurevmwaresales@microsoft.com) 구독 노드 할당량을 늘려야 합니다.  제공 해야 합니다.

* 구독 ID
* 노드 SKU
* 할당량 증가 요청 하는 추가 노드 수
