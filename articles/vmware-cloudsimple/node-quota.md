# <a name="cloudsimple-node-quota-limits"></a>클라우드 단순 노드 할당량 제한

4개의 노드는 CloudSimple 서비스에 대한 구독을 사용하도록 설정하면 구입할 수 있는 기본 수량입니다.  Azure 포털에서 모든 [노드 유형을](cloudsimple-node.md) 구입할 수 있습니다.  프라이빗 클라우드를 만들려면 동일한 SKU의 노드가 세 개 이상 필요합니다.  노드를 구입한 경우 추가 노드를 구입하려고 할 때 오류가 표시될 수 있습니다.

## <a name="quota-increase"></a>할당량 증가

지원 요청을 제출하여 노드 할당량을 늘릴 수 있습니다. 서비스 운영 팀은 요청을 평가하고 노드 할당량을 늘리기 위해 고객과 함께 작업합니다.  새 항공권을 열 때 다음 옵션을 선택합니다.

* 문제 유형: **기술**
* 구독: **구독 ID**
* 서비스 유형: **클라우드간이에 의한 VMware 솔루션**
* 문제 유형: **전용 노드 할당량**
* 문제 하위 유형: **전용 노드의 할당량 증가**
* 제목: **할당량 증가**

지원 티켓의 세부 정보에서 필요한 수의 노드 및 노드 SKU를 제공합니다.

* 노드 SKU
* 할당량 증가를 요청하는 추가 노드 수
