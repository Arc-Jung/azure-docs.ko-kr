# <a name="what-is-vmware-solution-on-azure-by-cloudsimple"></a>CloudSimple 하 여 Azure에서 VMware 솔루션 이란

**CloudSimple 하 여 Azure에서 VMware 솔루션** 는 Azure에서 VMware 플랫폼을 실행할 수 있는 완전히 관리 되는 서비스입니다. 이 솔루션 vSphere, vCenter, vSAN, NSX-T 및 해당 도구를 포함합니다.
VMware 환경을 Azure 클라우드 위치에서 Azure 운영 체제 미 설치 인프라에서 기본적으로 실행 됩니다. 서비스를 효율적이 고 안전 하 게 VMware 플랫폼을 사용 하는 데 필요한 모든 기능을 포함 합니다.

![CloudSimple 개요는 Azure에서 VMware 솔루션](media/azure-vmware-solution-by-cloudsimple.png)

## <a name="features"></a>기능

* 주문형 셀프 서비스 VMware 클라우드 환경의 프로 비전 합니다. 추가 주문형 용량을 제거 하는 기능
* VMware 플랫폼 배포, 업그레이드, 관리 평면 백업, 상태/용량 모니터링, 경고, 문제 해결 및 수정 합니다.
* L2/L3 서비스 및 방화벽 규칙 관리를 포함 하 여 VMware를 사용 하도록 설정 하는 데 필요한 네트워킹 서비스 언더레이 합니다.
* 가장자리 유형을 네트워킹 서비스, VPN, 공용 IP 및 인터넷 게이트웨이 포함 합니다. 이러한 지 서비스 Azure에서 실행 하 고 해당 하는 보안 및 Azure의 DDoS 보호 합니다.
* 용량 예약 비용을 줄일 수 있습니다.
* Azure 및 온-프레미스에 대 한 고속, 지연율이 낮은 연결 합니다.
* 통합 하는 방식으로 Azure 서비스를 사용 하는 고객에 대 한 솔루션 아키텍처를 사용 하면이 고유한 "공용 클라우드에 VMware 클라우드" 아키텍처의 활용 합니다. 이러한 Azure 서비스에는 Azure AD, storage, application gateway 등.
* 인프라를 완벽 하 게 전용 이며 다른 고객의 인프라와에서 물리적으로 격리 됩니다.
* 이러한 작업 관리, 사용, 청구/계량 및 사용자 관리 기능을 관리 합니다.
* 연중 무휴 고객 지원 합니다.

## <a name="benefits"></a>이점

* **작업 연속성:** CloudSimple는 VMware 플랫폼에 기본 액세스를 제공합니다. 기존 호환 CloudSimple 아키텍처가 같습니다.
  * 애플리케이션
  * 작업
  * 네트워킹
  * 보안
  * Backup
  * 재해 복구
  * 감사
  * 규정 준수 도구
  * 프로세스
* **재 학습:** VMware 플랫폼 호환성을 사용 하면 기존 기술과 지식을 사용할 수 있습니다.
* **또한 인프라 기민 성이:** 더 이상 모든 용량 요구를 예측할 수 있고 끝나는 불필요 하 게 용량 또는 인프라 부족. 클라우드 서비스로 CloudSimple 배달 되 고 추가 하거나 언제 든 지 용량을 줄일 수 있습니다.
* **보안:** Azure 통해 CloudSimple 환경에 대 한 액세스는 기본 DDoS 보호 및 보안 모니터링을 제공합니다.
* **저렴 한 비용:** CloudSimple 플랫폼은 항상 설계 하 고 높은 수준의 자동화, 운영 효율성 및 규모의 경제를 제공 합니다. 또한 CloudSimple 비용 절감을 공용 클라우드에 VMware의 존재를 활용 하는 솔루션 아키텍처를 게시 합니다. 예제에서는 Azure AD에서 Azure storage, application gateway에 대 한 백업, 부하 분산 장치 및 다른 사용자가 있습니다.
* **새 하이브리드 플랫폼:** 서비스에는 Azure의 나머지 부분에 고속, 낮은 대기 시간 액세스할을 수 있습니다. 또한 CloudSimple 관리 VMware Virtual Machines의 통합된 관리 및 UI와 API를 사용 하 여 Azure의 나머지 부분에 있습니다. 개발 팀에 통합 하 고 일관 된 방식으로 공용 및 개인 플랫폼 활용을 걸릴 수 있습니다.
* **인프라 모니터링, 문제 해결 및 지원:** CloudSimple 서비스로 기본 인프라를 작동합니다. 실패할 경우 자동으로 하드웨어를 교체 합니다. CloudSimple 환경을 원활 하 게 실행 되도록 하는 동안 소비에 집중할 수 있습니다.
* **정책 호환성:** VMware 기반 도구, 보안 절차, 감사 사례 및 규정 준수 인증을 유지 합니다.

## <a name="scenarios"></a>시나리오

* **데이터 센터 사용 중지 또는 마이그레이션:** 기존 데이터 센터의 한계에 도달 하거나 하드웨어를 새로 고칠 때 추가 용량을 가져옵니다. 클라우드에서 필요한 용량을 추가 하 고 하드웨어 새로 고침 관리는 골치 아픈 문제를 제거 하는 것이 쉽습니다. 시간이 많이 걸리는 변환 또는 아키텍처에 비해 클라우드 마이그레이션 비용과 위험을 줄입니다. 익숙한 VMware 도구 및 기술을 사용 하 여 클라우드 마이그레이션을 가속화 하 합니다. 한 번 클라우드에서 속도로 응용 프로그램 현대화를 Azure 서비스를 사용 합니다.
* **주문형 확장 합니다.** 새 개발 환경을 계절 용량 급증 등의 예기치 않은 요구 사항에 맞게 클라우드로 확장 합니다. 쉽게 요청 시 새 용량을 만들 하 고 유지 시간 만큼 필요할 수 있습니다. 사전 투자를 줄이고, 프로 비전의 속도 가속화 온-프레미스 및 클라우드에서 모두 동일한 아키텍처 및 정책을 사용 하 여 복잡성을 줄입니다.
* **재해 복구 및 Azure 클라우드에서 가상 데스크톱:** 데이터, 앱 및 Azure 클라우드에서 데스크톱에 대 한 원격 액세스를 설정 합니다. 높은 대역폭 연결을 사용 하면 업로드 다운로드 / 데이터 보안 문제 로부터 복구 신속 하 게 합니다. 지연율이 낮은 네트워크 지정 응답을 빠른 데스크톱 앱에서 사용자가 필요로 하는 시간입니다. CloudSimple를 사용 하 여 모든 정책 및 네트워킹 CloudSimple 포털 및 익숙한 VMware 도구를 사용 하 여 클라우드에서 복제 하기 쉽습니다. 복구 및 복제 편의성 노력 및 만들기 및 관리 DR 및 VDI 구현의 위험을 크게 줄여 줍니다.
* **고성능 응용 프로그램 및 데이터베이스의 경우:** CloudSimple 가장 까다로운 VMware 워크 로드를 실행 하도록 설계 하는 하이퍼 수렴 형 아키텍처를 제공 합니다. Oracle, Microsoft SQL server, 미들웨어 시스템 및 성능이 뛰어난 비 SQL 데이터베이스를 실행 합니다. 온-프레미스에서 Azure 및 Azure에 VMware에 걸쳐 있는 하이브리드 앱을 실행할 수 있도록 하는 고속 25-Gbps 네트워크 연결을 사용 하 여 사용자 고유의 데이터 센터 클라우드 환경을 개인 워크 로드 성능 저하 없이 합니다.
* **진정한 하이브리드:** VMware와 Azure에서 DevOps를 통합 합니다. Azure 서비스 및 모든 작업에 걸쳐 적용할 수 있는 솔루션에 대 한 VMware 관리를 최적화 합니다. 데이터 센터를 확장 하 고 응용 프로그램 아키텍처를 변경 하지 않고도 공용 클라우드 서비스에 액세스 합니다. 액세스 제어 정책, 로깅 및 모니터링 하 고 Azure에서 VMware 응용 프로그램 id를 중앙 집중화 합니다.
