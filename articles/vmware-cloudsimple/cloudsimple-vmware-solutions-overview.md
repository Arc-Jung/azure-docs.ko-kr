# <a name="what-is-azure-vmware-solution-by-cloudsimple"></a>클라우드에 의한 Azure VMware 솔루션이란 무엇입니까?

**CloudSimple에 의한 Azure VMware 솔루션은** Azure에서 VMware 플랫폼을 실행할 수 있는 완전히 관리되는 서비스입니다. 이 솔루션에는 vSphere, vCenter, vSAN, NSX-T 및 해당 도구가 포함됩니다. VMware 환경은 Azure 클라우드 위치의 Azure 베어 메탈 인프라에서 기본적으로 실행됩니다. 이 서비스에는 VMware 플랫폼을 효율적이고 안전하게 사용하는 데 필요한 모든 기능이 포함되어 있습니다.

![클라우드로 Azure에 VM웨어 솔루션간단한 개요](media/azure-vmware-solution-by-cloudsimple.png)

## <a name="features"></a>기능

* VMware 클라우드 환경의 온디맨드 셀프 서비스 프로비저닝. 필요에 따라 용량을 추가하고 제거할 수 있는 기능입니다.
* VMware 플랫폼 배포, 업그레이드, 관리 평면 백업, 상태/용량 모니터링, 경고, 문제 해결 및 수정.
* L2/L3 서비스 및 방화벽 규칙 관리를 포함하여 VMware를 활성화하는 데 필요한 언더레이 네트워킹 서비스입니다.
* VPN, 공용 IP 및 인터넷 게이트웨이를 포함한 에지 유형 네트워킹 서비스. 이러한 서비스는 Azure에서 실행되며 Azure의 보안 및 DDoS 보호를 수행합니다.
* 용량을 예약하여 비용을 절감할 수 있습니다.
* Azure 및 온-프레미스에 대한 고속, 대기 시간 단축 연결
* 고객이 통합된 방식으로 Azure 서비스를 사용하고 이 고유한 "공용 클라우드의 VMware 클라우드" 아키텍처를 활용할 수 있는 솔루션 아키텍처입니다. Azure 서비스에는 Azure AD, 저장소, 응용 프로그램 게이트웨이 등이 포함됩니다.
* 전적으로 전념하고 다른 고객의 인프라와 물리적으로 격리된 인프라입니다.
* 관리 기능은 활동 관리, 사용량, 청구/계량 및 사용자 관리를 특징으로 합니다.
* 24x7 고객 지원.

## <a name="benefits"></a>이점

* **운영 연속성**. CloudSimple은 VMware 플랫폼에 대한 기본 액세스를 제공합니다. CloudSimple 아키텍처는 기존 아키텍처와 호환됩니다.
    * 애플리케이션
    * 작업
    * 보안
    * Backup
    * 재해 복구
    * 감사
    * 컴플라이언스 도구
    * 프로세스
* **재교육 없음**. VMware 플랫폼 호환성을 사용하면 기존 기술과 지식을 사용할 수 있습니다.
* **인프라 민첩성.** 더 이상 모든 용량 요구 사항을 예측할 필요가 없으며 결국 용량 이나 인프라 부족으로 끝납니다. CloudSimple은 클라우드 서비스로 제공되며 언제든지 용량을 추가하거나 줄일 수 있습니다.
* **보안**. Azure를 통한 CloudSimple 환경에 대한 액세스는 기본 제공 DDoS 보호 및 보안 모니터링을 제공합니다.
* **낮은 비용**. CloudSimple 플랫폼은 고도로 엔지니어링되어 있으며 높은 수준의 자동화, 운영 효율성 및 규모의 경제를 제공합니다. 또한 CloudSimple은 퍼블릭 클라우드에 VMware의 존재를 활용하여 비용을 절감하는 솔루션 아키텍처를 게시합니다. 예를 들어 Azure AD, Azure 저장소에 대한 백업, 응용 프로그램 게이트웨이, 로드 밸러터 등이 있습니다.
* **새로운 하이브리드 플랫폼.** 이 서비스를 사용하면 나머지 Azure에 대한 고속의 짧은 대기 시간 액세스를 사용할 수 있습니다. 또한 CloudSimple 관리를 사용하면 동일한 UI 및 API를 사용하여 VMware 가상 컴퓨터와 Azure의 나머지 부분을 통합관리할 수 있습니다. 개발 팀은 통합되고 일관된 방식으로 공용 및 비공개 플랫폼을 모두 활용할 수 있습니다.
* **인프라 모니터링, 문제 해결 및 지원.** CloudSimple은 기본 인프라를 서비스로 운영합니다. 장애 하드웨어가 자동으로 교체됩니다. CloudSimple은 환경이 원활하게 실행되도록 하는 동안 소비에 집중할 수 있습니다.
* **정책 호환성**. VMware 기반 도구, 보안 절차, 감사 관행 및 규정 준수 인증을 유지하십시오.

## <a name="scenarios"></a>시나리오

* **데이터 센터 사용 중지 또는 마이그레이션.** 기존 데이터 센터 또는 하드웨어 새로 고침의 한계에 도달하면 추가 용량을 얻을 수 있습니다. 클라우드에서 필요한 용량을 추가하고 하드웨어 새로 고침 관리의 번유를 쉽게 제거할 수 있습니다. 시간이 많이 소요되는 변환 또는 아키텍처에 비해 클라우드 마이그레이션의 위험과 비용을 줄입니다. 익숙한 VMware 도구와 기술을 사용하여 클라우드 마이그레이션을 가속화합니다. 클라우드에서 Azure 서비스를 사용하여 사용자의 속도에 맞춰 응용 프로그램을 현대화합니다.
* **온디맨드 확장.** 새로운 개발 환경이나 계절적 용량 버스트와 같은 예기치 않은 요구 사항을 충족하도록 클라우드로 확장합니다. 필요에 따라 새 용량을 쉽게 생성하고 필요한 기간 동안만 유지할 수 있습니다. 온-프레미스와 클라우드 모두에서 동일한 아키텍처 및 정책으로 선행 투자를 줄이고, 프로비저닝 속도를 가속화하며, 복잡성을 줄입니다.
* **Azure 클라우드의 재해 복구 및 가상 데스크톱.** Azure 클라우드에서 데이터, 앱 및 데스크톱에 대한 원격 액세스를 설정합니다. 고대역폭 연결을 사용하면 데이터를 빠르게 업로드하고 다운로드하여 인시던트에서 복구할 수 있습니다. 대기 시간이 짧은 네트워크는 사용자가 데스크톱 앱에서 기대하는 빠른 응답 시간을 제공합니다. CloudSimple을 사용하면 CloudSimple 포털 및 친숙한 VMware 도구를 사용하여 클라우드에서 모든 정책과 네트워킹을 쉽게 복제할 수 있습니다. 복구 및 복제의 용이성은 DR 및 VDI 구현을 만들고 관리하는 노력과 위험을 크게 줄여줍니다.
* **고성능 응용 프로그램 및 데이터베이스.** CloudSimple은 가장 까다로운 VMware 워크로드를 실행하도록 설계된 하이퍼컨버지드 아키텍처를 제공합니다. 오라클, Microsoft SQL 서버, 미들웨어 시스템 및 고성능 노SQL 데이터베이스를 실행합니다. 온-프레미스, Azure의 VMware 및 Azure 개인 워크로드에 걸쳐 있는 하이브리드 앱을 성능 저하 없이 실행할 수 있는 고속 25Gbps 네트워크 연결을 통해 클라우드를 자체 데이터 센터로 경험하십시오.
* **진정한 하이브리드**. VM웨어 및 Azure에서 데브옵스를 통합합니다. 모든 워크로드에 적용할 수 있는 Azure 서비스 및 솔루션에 대해 VMware 관리를 최적화합니다. 데이터 센터를 확장하거나 애플리케이션을 다시 설계할 필요 없이 퍼블릭 클라우드 서비스에 액세스할 수 있습니다. Azure에서 Id를 중앙 집중화하고, 제어 정책에 액세스하고, VMware 응용 프로그램에 대한 로깅 및 모니터링을 합니다.
