# <a name="azure-vmware-solution-by-cloudsimple"></a>CloudSimple의 Azure VMware 솔루션

CloudSimple의 Azure VMware 솔루션을 시작합니다. CloudSimple을 사용하는 방법과 서비스에서 솔루션을 사용하는 방법을 자세히 알아볼 것입니다. 문서를 검색하거나 목차를 찾아보세요.

- [개요](cloudsimple-vmware-solutions-overview.md)를 살펴보고 CloudSimple의 Azure VMware 솔루션에 대해 자세히 알아보세요.
- [CloudSimple 사설 클라우드 개요](cloudsimple-private-cloud.md)를 선택하여 CloudSimple 사설 클라우드 솔루션에 대해 자세히 알아보세요.
- [노드](cloudsimple-node.md)를 선택하여 CloudSimple 노드에 대해 자세히 알아보세요.
- [용량 초기화 및 구매](quickstart-create-cloudsimple-service.md)를 선택하여 CloudSimple의 Azure VMware 솔루션을 시작하세요.
