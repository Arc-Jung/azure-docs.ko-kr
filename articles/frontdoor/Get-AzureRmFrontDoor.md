# Get-AzureRmFrontDoor
Front Door 구성 가져오기

## 구문

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명
**Get-AzureRmFrontDoor** cmdletGet은 제공된 정보와 일치하는 현재 구독에서 기존의 모든 Front Doors를 가져옵니다.
