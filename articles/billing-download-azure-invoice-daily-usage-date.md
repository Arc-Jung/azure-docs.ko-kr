# Azure 청구 송장 및 일간 사용 현황 데이터를 다운로드 하는 방법

Azure 계정 관리자는 Azure 계정 센터에서 현재 청구서를 볼 수 있고 이전 6개의 청구 기간에 대한 명세서를 다운로드할 수 있습니다. 방법을 알아보겠습니다.

1. Microsoft 계정 ID, 직장 계정 ID 또는 학교 계정 ID를 사용하여 [사용량 및 대금 청구 포털](https://account.windowsazure.com/subscriptions)에 로그인 합니다.
2. 원하는 송장 및 사용 정보에 대한 구독을 선택합니다.
3. **청구 내역**을 선택합니다. </br>![청구 내역](./media/billing-download-azure-invoice-daily-usage-date/Billinghisotry.png)
4. 이전 6개의 청구 기간 및 현재 미결제한 기간의 명세서를 볼 수 있습니다. </br>![청구 합계](./media/billing-download-azure-invoice-daily-usage-date/billingSum.png)</br>
5. **현재 명세서 보기**를 선택하여 예상 요금을 볼 수 있습니다. 이 정보는 매일 업데이트만 될 뿐 모든 사용량을 포함하지 않습니다. 월별 청구서는 이 예상 요금과 다를 수 있습니다.</br>![청구 합계2](./media/billing-download-azure-invoice-daily-usage-date/billingSum2.png)</br>![청구 합계3](./media/billing-download-azure-invoice-daily-usage-date/billingSum3.png)</br>
6. **송장 다운로드**를 선택하여 마지막 청구서의 복사본을 볼 수 있습니다. </br>![DLInvoice](./media/billing-download-azure-invoice-daily-usage-date/DLInvoice1.png)
7. **사용량 다운로드**를 선택하여 일간 사용 데이터를 CSV 파일로 다운로드 할 수 있습니다.</br>![DLusage](./media/billing-download-azure-invoice-daily-usage-date/DLusage.png)
