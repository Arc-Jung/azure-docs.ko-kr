# <a name="supported-browsers-and-devices"></a>지원되는 브라우저 및 장치
Microsoft Azure 포털은 모든 최신 데스크톱 및 태블릿 장치와 브라우저에서 실행됩니다.

### <a name="supported-devices"></a>지원되는 장치
포털은 최신 PC, Mac 및 태블릿에서 효율적으로 실행됩니다. 휴대폰은 현재 지원되지 않습니다.

### <a name="supported-browsers"></a>지원되는 브라우저
다음과 같은 최신 버전 브라우저가 지원됩니다.

* Edge(최신)
* Internet Explorer 11
* Safari(최신, Mac 전용)
* Chrome(최신)
* Firefox(최신)
