# 구입 및 Azure 구독에 등록하는 방법

## Azure 무료 평가판 구독 등록
Azure 무료 평가판 구독에 등록하려면 전화 번호, 신용 카드, [Microsoft 계정](https://www.microsoft.com/ko-KR/account/faq.aspx)(이전 Windows Live ID) 또는 학교/회사 계정(즉, 조직 ID)을 지정해야 합니다. 전화번호 및 신용카드는 오직 신원 확인을 위해 사용되고, 결제 용도로 쓰이지 않습니다.

>[AZURE.NOTE]결제 카드가 유효한 카드인지 확인하기 위해 미국에서는 1달러, 그 밖에 지역에서는 그에 해당하는 돈을 배치하고, 금융 기관의 정책에 따라 3-5일 내에 해지됩니다. 카드로 결제하지 않습니다.

Azure 무료 평가판 구독을 등록하기

1. [Azure 무료 평가판 페이지](https://azure.microsoft.com/ko-KR/pricing/free-trial/)를 열고, **지금 평가판 사용**을 선택합니다.

2. Microsoft 계정 또는 학교/직장 계정에 로그인 합니다.

3. **사용자 정보** 섹션에 개인 정보를 입력합니다. 국가는 일단 지정하면 변경할 수 없습니다.![개인정보의 screenshow](./media/billing-buy-sign-up-azure-subscription/AboutYou.png)

4. 전화 통화 또는 SMS를 통해 휴대폰으로 전송되는 코드로 인증합니다. 받은 코드를 입력하면, 사용자의 신원을 확인하는데 도움이 됩니다. </br></br> **전화로 인증** 섹션에서, 사용자 국가의 국제 지역 번호를 선택하고, 전화 번호(내부 지역 번호를 입력하지 않음)를 입력한 후 **텍스트 메시지 보내기** 또는 **전화 걸기**를 선택하세요. 인증 코드를 받으면 **코드 확인** 상자에 입력한 후 **코드 확인** 단추를 선택하여 전화 번호를 확인하세요. VOIP 전화 번호로는 휴대폰을 인증할 수 없습니다.</br> ![휴대폰 인증 스크린샷](./media/billing-buy-sign-up-azure-subscription/PhoneVerify.png)</br>문자 메시지 또는 전화를 받지 못한 경우, 다른 전화 번호를 사용하거나 [Microsoft 지원](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)에 문의하여 도움을 받으세요.</br>
5. 휴대폰 인증이 완료되면, 카드 인증 섹션으로 이동합니다.![카드인증하기](./media/billing-buy-sign-up-azure-subscription/VardVerify.png)</br>
6. 신용 카드 인증이 완료되면, **계약** 옵션을 선택한 후 **등록**을 선택합니다. ![등록](./media/billing-buy-sign-up-azure-subscription/Signup.png)</br>구독을 설정하는 데 몇 분 정도 걸릴 수 있습니다. 15분 이내에 구독 설정이 완료되지 않으면 [Microsoft 지원](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)에 도움을 요청하세요. </br>오류 메시지가 표시될 경우 문제 해결을 위해 메시지의 단계를 따르세요. 일반적인 오류 메시지는 다음과 같습니다.

**죄송합니다. 작업을 완료할 수 없습니다. 나중에 다시 시도하거나 웹 페이지를 다시 로드하세요.** </br>

이 경우, 다른 브라우저를 사용하여 등록 프로세스를 완료하십시오. 문제가 계속 발생하면 [Microsoft 지원](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)에 도움을 요청하세요.

![등록오류1](./media/billing-buy-sign-up-azure-subscription/signuperror1.png)</br>7. 구독이 준비되면 **내 서비스 관리 시작**을 선택하세요.![서비스시작](./media/billing-buy-sign-up-azure-subscription/startservice.png)

## Azure 무료 평가판 구독을 종량제 구독으로 업그레이드

1. 무료 평가판을 등록하는 데 사용한 Microsoft 계정 또는 학교/회사 계정을 사용하여 [Azure 계정 포털](https://account.windowsazure.com/subscriptions)에 로그온합니다.
2. 구독을 선택한 후 **업그레이드**를 선택합니다.</br>![결제페이지](./media/billing-buy-sign-up-azure-subscription/billpage.png)
3. **예**를 선택한 후 **지금 업그레이드**를 선택합니다.</br>![업그레이드](./media/billing-buy-sign-up-azure-subscription/Upgrade.png)

종량제 구독으로 업그레이드한 이후에도 사용자는 30일/200$ 둘 중 하나가 소진될 때까지 평가판을 계속 사용할 수 있습니다.

## 온라인 서비스 활성화 키 (5 x 5 키)를 사용하여 Azure 구독을 활성화

재판매 통신 사업자에게 Azure in Open 크레딧을 구매한 경우, 다음 단계를 수행하여 구독을 활성화 하세요.

1. [오픈 라이선스의 Azure 키](http://azure.microsoft.com/offers/ms-azr-0111p/) 페이지를 열고 새 구독 활성화 또는 기존 구독을 선택합니다.
2. 제품 키를 입력하고 첫 번째 계약 옵션을 선택하여(특별 제안을 받으려는 경우 두 번째 옵션도 선택) 구독을 활성화합니다.</br>![OSA 키](./media/billing-buy-sign-up-azure-subscription/OSAkey.png)
3. **활성화**를 선택합니다.
