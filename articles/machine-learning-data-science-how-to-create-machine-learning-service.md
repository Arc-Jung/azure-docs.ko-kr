# 기계 학습을 위한 클라우드 데이터 과학 프로세스

이 클라우드 데이터 과학 프로세스 가이드에서는 데이터 과학에 Azure 플랫폼을 사용하는 방법과 Azure 기계 학습을 사용하여 조작 가능한 모델을 구축하는 방법을 보여 줍니다. 맵에서는 다양한 소스에서 데이터를 수집하고, 수집한 데이터를 Azure 기계 학습에 사용할 수 있도록 준비하고, Azure 기계 학습을 사용하여 최종 사용자 응용 프로그램에서 프로그래밍 방식으로 또는 기타 방식으로 사용할 수 있는 조작 가능한 모델을 만드는 등 일반적인 데이터 과학 작업을 설명합니다. 맵에서 보여 주는 단계는 일반적인 종단 간 데이터 과학에 관련된 핵심 단계입니다. 모든 단계가 반드시 필요한 것은 아니며 데이터의 위치, 크기 및 복잡성에 따라 순서가 달라질 수 있습니다.

**Azure의 고급 데이터 처리**라는 제목의 단계는 두 번째 맵으로 연결됩니다. 두 번째 맵에서 돌아오면 해당 지점부터 계속 이어서 Azure 기계 학습 스튜디오에서 처리된 데이터를 수집하여 모델을 구축 및 배포합니다.
