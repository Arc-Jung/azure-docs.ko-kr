# 자습서: 미리 구성된 IoT 솔루션 시작

## 소개

미리 구성된 Azure IoT Suite 솔루션은 다양한 Azure IoT 서비스를 연결하여 IoT(사물 인터넷) 비즈니스 시나리오를 만족하는 종단간 솔루션을 예로 보여 줍니다.

이 자습서는 미리 구성된 솔루션인 **원격 모니터링**을 프로비전하는 방법을 보여 줍니다. 또한 미리 구성된 원격 모니터링 솔루션의 기본 기능을 확인하는 방법을 보여 줍니다.

이 자습서를 완료하려면 다음이 필요합니다.

-   활성 Azure 구독.

    계정이 없는 경우 몇 분 만에 무료 평가판 계정을 만들 수 있습니다. 자세한 내용은 [Azure 무료 평가판][]을 참조하세요.

## 미리 구성된 원격 모니터링 솔루션 프로비전

1.  https://www.azureiotsuite.com에 로그온하고 **+**를 클릭하여 새 솔루션을 만듭니다.

2.  솔루션 형식으로 **원격 모니터링**을 선택합니다.

3.  미리 구성된 원격 모니터링 솔루션에 **솔루션 이름**을 입력합니다.

4.  이 솔루션을 프로비전하려는 **지역** 및 **구독**의 유효성을 검사합니다.

5.  **솔루션 만들기**를 클릭합니다.

## 원격 모니터링 솔루션 대시보드 보기

1.  프로비전이 완료되고 미리 구성된 솔루션에 대한 타일이 **준비**를 가리키면 **시작**을 클릭하여 새 탭에서 원격 모니터링 솔루션 대시보드를 엽니다.

2.  기본적으로는 **대시보드**가 왼쪽 메뉴에서 선택됩니다. 솔루션 대시보드입니다.

## 솔루션 장치 목록 보기

1.  왼쪽 메뉴에서 **장치**를 클릭하여 해당 솔루션에 대한 장치 목록으로 이동합니다.

2.  프로비전하는 즉시 프로비전된 4개의 시뮬레이션된 장치가 나타납니다.

3.  장치 목록에서 **개별 장치**를 클릭하여 해당 장치와 연결된 장치 세부 정보를 봅니다.

## 장치에 명령 보내기

1.  선택한 시뮬레이션된 장치에 대한 장치 세부 정보 창에서 **명령 전송**을 클릭합니다.

2.  명령 목록에서 **PingDevice**를 선택합니다.

3.  **명령 전송**을 클릭합니다.

4.  명령 기록에 표시된 대로 명령의 상태를 참조합니다.

## 시뮬레이션된 새 장치 추가

1.  **←**(뒤로 화살표)를 클릭하여 장치 목록으로 돌아갑니다.

2.  왼쪽 아래 모서리에서 **+ 장치 추가**를 클릭하여 새 장치를 추가합니다.

3.  **시뮬레이션된 장치**에 대해 **새 장치 추가**를 클릭합니다.

4.  **나만의 장치 ID 정의**를 선택하고 고유한 장치 ID 이름을 추가합니다.

5.  **만들기**를 클릭합니다.

6.  **←**(뒤로 화살표)를 클릭하여 장치 목록으로 돌아갑니다.

7.  장치 목록에서 장치 **실행**을 참조하세요.

## 솔루션 규칙 보기 및 편집

1.  **솔루션 대시보드**의 **경보 기록**을 참고합니다.

2.  이러한 경보는 **규칙**에서 지정된 규칙 출력 **AlarmTemp**에 의해 트리거됩니다.

3.  왼쪽 메뉴에서 **규칙**을 클릭하여 해당 솔루션에 대한 규칙으로 이동합니다.

4.  프로비전할 때 이미 사용하도록 설정된 하나의 규칙이 표시됩니다.

5.  규칙 목록에서 **규칙**을 클릭하여 관련된 규칙 속성을 봅니다.

6.  규칙 속성 창에서 **편집**을 클릭합니다.

7.  **임계값**을 30이 되도록 변경하고 다른 모든 속성을 동일하게 유지합니다.

8.  **규칙 저장 및 보기**를 클릭합니다.

9.  **솔루션 대시보드**의 **경보 기록** 테이블로 돌아가고 업데이트된 규칙의 결과로 트리거에서 변경 사항을 확인합니다.
