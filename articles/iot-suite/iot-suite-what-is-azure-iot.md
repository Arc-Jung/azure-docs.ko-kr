## Azure IoT Suite

Microsoft Azure IoT Suite는 *원격 모니터링* 및 *예측 유지 관리*와 같이 일반적인 IoT 시나리오를 해결하는 미리 구성된 확장 가능 솔루션 집합을 통해 빠르게 시작할 수 있는 엔터프라이즈급 솔루션입니다.

미리 구성된 솔루션은 다음과 같습니다.

- 시작하는 데 도움이 되는 시뮬레이트된 장치를 포함한 완료, 작업, 종단 간 솔루션과 [Azure IoT Hub][], [Azure 이벤트 허브][], [Azure 스트림 분석][], [Azure 기계 학습][], [Azure 저장소][]와 같은 미리 구성된 Azure 서비스 및 솔루션 특정 관리 콘솔입니다.
- 입증되고 프로덕션에 사용할 준비가 된 코드가 포함되어 있습니다. 이 코드를 사용자 지정하고 확장하여 자체 특정 IoT 시나리오를 구현할 수 있습니다.

미리 구성된 수많은 솔루션을 사용하는 [Azure IoT Hub][] 서비스에 관심이 있을 수 있습니다. [Azure IoT Hub][]는 샘플 솔루션 아키텍처에 정의된 다양한 데이터 처리 및 분석 기능을 제공합니다.
