# Power BI Embedded REST API

**Microsoft Power BI Embedded**를 사용하면 REST API를 사용하여 **Power BI 보고서**를 응용 프로그램에 통합하는 응용 프로그램을 프로그래밍 방식으로 개발할 수 있습니다.

**Power BI Embedded** REST API에 대한 자세한 내용은 [Power BI REST 작업](https://msdn.microsoft.com/library/mt711493)을 참조하세요.
