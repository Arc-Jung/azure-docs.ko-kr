# Operational Insights 검색
Microsoft Azure Operational Insights의 핵심은 사용자 환경 내에서 여러 원본의 모든 컴퓨터 데이터를 서로 연결하고 결합할 수 있게 하는 검색 기능입니다. 인텔리전스 팩도 검색에서 제공되어 특정 문제 영역 주위에 피벗된 메트릭을 가져옵니다.


## 데이터에 대한 검색
Azure Operational Insights의 **검색** 기능을 사용하여 검색 중인 데이터를 찾을 수 있습니다. 

![검색 타일의 이미지](./media/operational-insights-search/overview-search.png)

**검색** 페이지에서, 쿼리를 만든 다음 검색 시 패싯 컨트롤을 사용하여 결과를 필터링할 수 있습니다. 변환, 필터링 및 결과를 보고하는 고급 쿼리를 만들 수도 있습니다.

![검색 페이지의 이미지](./media/operational-insights-search/search-01.png)

일반적인 검색 쿼리는 대부분의 검색 페이지에 표시됩니다. Operational Insights 콘솔에서는 타일을 클릭하거나 **검색**을 사용하여 항목에 대한 세부 정보를 보도록 다른 항목으로 드릴인투할 수 있습니다.
패싯을 사용하고 컨트롤을 필터링하여 결과 범위를 좁힐 수 있습니다. 자동 완성 기능으로 입력을 보조하면 구문을 이해하고 결과를 신속하게 생성하는 데 도움이 됩니다.
