# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>프리미엄 블록 blob storage 계정에 대 한 확장성 목표

프리미엄 성능 블록 blob 저장소 계정은 더 작은 킬로바이트 범위 개체를 사용 하는 응용 프로그램에 최적화 되어 있습니다. 높은 트랜잭션 요금 또는 일관 된 낮은 대기 시간 저장소를 필요로 하는 응용 프로그램에 적합 합니다. 프리미엄 성능 블록 blob 저장소는 응용 프로그램으로 확장할 수 있도록 설계 되었습니다. 시나리오에서 초당 수백만 개의 요청 또는 저장소 용량 페타바이트 필요한 응용 프로그램을 배포 해야 하는 경우 지원 요청을 [Azure Portal](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)에 제출 하 여 Microsoft에 문의 하세요.
