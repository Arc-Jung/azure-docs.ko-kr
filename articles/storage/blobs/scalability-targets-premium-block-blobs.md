# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>프리미엄 블록 Blob 저장소 계정에 대한 확장성 목표

프리미엄 성능 블록 Blob 저장소 계정은 더 작은 킬로바이트 범위의 개체를 사용하는 응용 프로그램에 최적화되어 있습니다. 높은 트랜잭션 속도 또는 일관된 낮은 대기 시간 저장소가 필요한 응용 프로그램에 이상적입니다. 프리미엄 성능 블록 Blob 저장소는 응용 프로그램에 맞게 확장하도록 설계되었습니다. 시나리오에서 초당 수십만 개의 요청또는 페타바이트 규모의 저장소 용량이 필요한 응용 프로그램을 배포해야 하는 경우 [Azure Portal에서](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)지원 요청을 제출하여 Microsoft에 문의하십시오.
