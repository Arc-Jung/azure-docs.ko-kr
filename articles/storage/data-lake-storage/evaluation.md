# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Azure Data Lake Storage Gen1 고객에 대한 지침

Azure Data Lake Storage Gen2 미리 보기에서는 Data Lake Storage Gen1에서 지원되는 Hadoop 호환 가능 파일 시스템 외에도 데이터 계층, 수명 관리 및 강화된 네트워크 보안을 비롯한 기본 Blob Storage 기능을 제공합니다.

기존 Data Lake Storage Gen1 고객의 경우 Microsoft에서는 사용자가 Data Lake Storage Gen2의 새롭게 강화된 기능을 평가하고 [미리 보기에 참여](https://aka.ms/adlsgen2signup)하여 피드백을 제공해주시기를 바랍니다. 또한 Data Lake Storage Gen1도 계속 지원됩니다. Data Lake Storage Gen1 및 Gen2는 공존할 수 있습니다. 따라서 Data Lake Storage Gen1을 활용하는 기존 리소스 및 응용 프로그램이 업데이트하지 않고 계속 작동합니다. 편의에 따라 Data Lake Storage Gen2에 대한 마이그레이션을 계획할 수 있습니다.

지금 Data Lake Storage Gen1을 평가하고 계시다면 Microsoft에서는 사용자가 Gen2 미리 보기에 참여하여 Data Lake Storage Gen2에서 제공하는 새롭게 강화된 기능을 평가해주시기를 바랍니다. 마찬가지로 기존 Blob Storage 고객이신 경우 Microsoft에서는 사용자가 Hadoop 호환가 능 파일 시스템이 분석 요구 사항에 적합한지를 평가해주시기를 바랍니다.
