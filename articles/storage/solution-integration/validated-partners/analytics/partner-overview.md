# <a name="azure-storage-analytics-and-big-data-partners"></a>Azure Storage 분석 및 빅 데이터 파트너

이 문서에서는 Azure Data Lake Storage와 통합 된 Microsoft 파트너 회사를 강조 표시 합니다. 이러한 파트너 솔루션은 최신 데이터 웨어하우스 워크 로드, 고급 분석 및 실시간 분석과 같은 워크 로드를 다룹니다. 이러한 파트너는 Azure Storage의 [계층적 네임 스페이스](../../../blobs/data-lake-storage-namespace.md) 를 활용 하 여 솔루션을 최적화 하 고 Azure에서 효율적으로 실행 합니다.
