# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Azure Storage 보관, 백업 및 재해 복구 파트너

이 문서에서는 보관 및 BCDR 워크 로드에 대 한 Azure Storage 통합 된 Microsoft 파트너를 강조 표시 합니다. 이는 백업 문제를 효과적으로 해결 하거나, 재해 복구 사이트를 만들거나, 장기 보존을 위해 사용 하지 않는 콘텐츠를 보관 하는 Azure Storage의 규모 및 [비용 혜택](https://docs.microsoft.com/azure/storage/blobs/storage-lifecycle-management-concepts?tabs=azure-portal) 을 활용 합니다. 이러한 솔루션은 Azure Storage 충족 하는 모든 준수 표준과 [변경 불가능 한 저장소](https://docs.microsoft.com/azure/storage/blobs/storage-blob-immutable-storage) 및 수명 주기 관리를 통해 쉽게 테이프 기반 백업을 대체 하 고 주문형 경제적 복구 사이트를 제공할 수 있습니다.
