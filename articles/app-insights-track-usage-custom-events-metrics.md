# 사용자 지정 이벤트 및 메트릭으로 사용량 추적

### 서비스 예정

Application Insights SDK를 통해 사용자 활동을 모니터할 코드 줄을 응용 프로그램에 삽입할 수 있으므로, 요구에 맞춰 앱을 조정할 수 있습니다.

Microsoft Azure의 새로운 Application Insights 포털로 이 기능을 계속 이식하고 있습니다. 원하는 경우 [이전 포털에서 작동 방법을 확인][이전 포털에서 작동 방법을 확인]할 수 있습니다.

그러나 지금 할 수 있는 작업은 [Trace, NLog 또는 Log4Net으로 생성된 진단 이벤트 로그를 캡처 및 검색][Trace, NLog 또는 Log4Net으로 생성된 진단 이벤트 로그를 캡처 및 검색]하는 것입니다.
