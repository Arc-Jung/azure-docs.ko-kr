# <a name="supported-azure-blockchain-service-ledger-versions"></a>지원 되는 Azure Blockchain Service 원장 버전

Azure Blockchain Service는 Ethereum 기반 [쿼럼](https://github.com/jpmorganchase/quorum/wiki) 원장 Azure Blockchain Service에서 컨소시엄으로 식별 하는 알려진된 참가자 그룹 내의 개인 트랜잭션 처리를 위해 설계 되었습니다.

현재, Azure Blockchain 서비스는 지원 [쿼럼 버전 2.2.1](https://github.com/jpmorganchase/quorum/releases/tag/v2.2.1) 하 고 [Tessera 트랜잭션 관리자](https://github.com/jpmorganchase/tessera)합니다.

## <a name="managing-updates-and-upgrades"></a>업데이트 및 업그레이드 관리

쿼럼에 대 한 버전 관리는 주 릴리스, 주요 릴리스 및 부 릴리스를 통해 수행 됩니다. 예를 들어, 쿼럼 버전 2.0.1 2 주 릴리스 인 경우 0은 주요 지점 릴리스, 및 1 부 지점 릴리스입니다. 서비스는 자동으로 사소한 문제로 릴리스의 원장을 패치합니다. 현재 업그레이드 및 주요 포인트 릴리스는 지원 되지 않습니다.

## <a name="availability-of-new-ledger-versions"></a>새 원장 버전 가용성

Azure Blockchain 서비스 원장 제조업체에서 제공 되 고 60 일 이내 원장의 최신 버전을 제공 합니다. 네 가지 주요 포인트 릴리스 중 최대 컨소시엄을 새 멤버를 프로 비전 할 때 선택할 consortia 제공 됩니다.
