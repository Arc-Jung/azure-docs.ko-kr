# <a name="java-codeless-application-monitoring-on-premises---azure-monitor-application-insights---public-preview"></a>Java 코드리스 응용 프로그램 모니터링 온-프레미스 - Azure 모니터 응용 프로그램 인사이트 - 공개 미리 보기

Java 코드리스 응용 프로그램 모니터링은 단순성입니다 - 코드 변경이 없으며 Java 에이전트는 몇 가지 구성 변경을 통해서만 활성화할 수 있습니다.

## <a name="overview"></a>개요

Java 에이전트가 활성화되면 가장 널리 사용되는 라이브러리 및 프레임워크에서 수많은 요청, 종속성, 로그 및 메트릭을 자동으로 수집합니다.

온-프레미스를 포함한 모든 환경에 대한 [자세한 지침을](https://docs.microsoft.com/azure/azure-monitor/app/java-in-process-agent) 따르십시오.
