# <a name="java-codeless-application-monitoring-on-premises---azure-monitor-application-insights---public-preview"></a>Java 코드 없는 응용 프로그램 모니터링 온-프레미스-Azure Monitor Application Insights-공개 미리 보기

Java 코드 없는 응용 프로그램 모니터링은 모두 간단 하 고, 코드를 변경 하지 않고, 몇 가지 구성 변경 만으로 Java 에이전트를 사용 하도록 설정할 수 있습니다.

## <a name="overview"></a>개요

사용 하도록 설정 하면 Java 에이전트가 가장 널리 사용 되는 라이브러리 및 프레임 워크에서 다양 한 요청, 종속성, 로그 및 메트릭을 자동으로 수집 합니다.

온-프레미스를 포함 하 여 모든 환경에 대 한 [자세한 지침](https://docs.microsoft.com/azure/azure-monitor/app/java-in-process-agent) 을 따르세요.
