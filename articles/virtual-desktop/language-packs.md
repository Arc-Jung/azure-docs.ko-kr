# <a name="install-language-packs"></a>언어 팩 설치

Windows 가상 데스크톱 배포를 국제적으로 설정할 때배포가 여러 언어를 지원하는지 확인하는 것이 좋습니다. Windows 10 엔터프라이즈 다중 세션 가상 컴퓨터(VM) 이미지에 언어 팩을 설치하여 조직에 필요한 만큼의 언어를 지원할 수 있습니다. 이 문서에서는 사용자가 자신의 표시 언어를 선택할 수 있도록 언어 팩을 설치하고 이미지를 캡처하는 방법을 설명합니다.

Azure 포털이 있는 [가용성 영역에서 Windows 가상 컴퓨터 만들기에서](../virtual-machines/windows/create-portal-availability-zone.md)Azure에서 VM을 배포하는 방법에 대해 자세히 알아봅니다.

>[!NOTE]
>이 문서는 Windows 10 엔터프라이즈 다중 세션 VM에 적용됩니다.

## <a name="install-a-language-pack"></a>언어 팩 설치

언어 팩을 사용하여 VM 이미지를 만들려면 먼저 컴퓨터에 언어 팩을 설치하고 이미지 캡처해야 합니다.

언어 팩을 설치하려면 다음을 수행하십시오.

1. 관리자로 로그인합니다.
2. 모든 최신 Windows 및 Windows 스토어 업데이트를 설치했는지 확인합니다.
3. 언어**영역**& **설정** > **시간으로** > 이동합니다.
4. **국가 또는 지역에서**드롭다운 메뉴에서 원하는 국가 또는 지역을 선택합니다.
    이 예제에서는 다음 스크린샷과 같이 **프랑스를**선택합니다.

    ![지역 페이지의 스크린샷입니다. 현재 선택된 지역은 프랑스입니다.](media/region-page-france.png)

5. 그런 다음 **언어를**선택한 다음 **언어 추가를 선택합니다.** 목록에서 설치할 언어를 선택한 다음 **다음을**선택합니다.
6. 언어 **기능 설치** 창이 열리면 언어 **설치 팩이라고**표시된 확인란을 선택하고 Windows 표시 언어로 설정합니다.
7. **설치**를 선택합니다.
8. 한 번에 여러 언어를 추가하려면 **언어 추가를**선택한 다음 프로세스를 반복하여 5단계와 6단계에서 언어를 추가합니다. 설치하려는 각 언어에 대해 이 프로세스를 반복합니다. 그러나 한 번에 하나의 언어만 표시 언어로 설정할 수 있습니다.

    빠른 시각적 데모를 살펴보겠습니다. 다음 이미지는 프랑스어 및 네덜란드어 팩을 설치한 다음 프랑스어를 표시 언어로 설정하는 방법을 보여 준다.

    ![프로세스 시작 시 언어 페이지의 스크린샷입니다. 선택한 Windows 표시 언어는 영어입니다.](media/language-page-default.png)

    ![언어 선택 창의 스크린샷입니다. 사용자가 프랑스어 패키지를 찾기 위해 검색 모음에 "프랑스어"를 입력했습니다.](media/select-language-french.png)

    ![언어 설치 기능 페이지의 스크린샷입니다. 프랑스어가 기본 언어로 선택됩니다. 선택한 옵션은 "내 표시 언어 설정", "언어 팩 설치", "음성 인식" 및 "필기"입니다.](media/install-language-features.png)

    언어 팩을 설치한 후에는 언어 팩의 이름이 언어 목록에 표시됩니다.

    ![새 언어 팩이 설치된 언어 페이지의 스크린샷입니다. 프랑스어 및 네덜란드 언어 팩은 "기본 언어"로 나열됩니다.](media/language-page-complete.png)

9. 세션에서 로그아웃하라는 창이 나타납니다. 로그아웃한 다음 다시 로그인합니다. 이제 표시 언어가 선택한 언어가 되어야 합니다.

10.   > 제어판**시계 및 지역** > **영역으로** **이동합니다.**

11.  **지역** 창이 열리면 **관리** 탭을 선택한 다음 **복사 설정을**선택합니다.

12.  **시작 화면 및 시스템 계정 및** 새 사용자 **계정으로**레이블이 지정된 확인란을 선택합니다.

13.  **확인**을 선택합니다.

14.  창이 열리고 세션을 다시 시작하라는 지시가 표시됩니다. **지금 다시 시작을 선택합니다.**

15.  다시 로그인한 후 **제어판** > **시계 및 지역** > 으로**돌아갑니다.**

16.  관리 탭을 **선택합니다.**

17.  **시스템 로캘 변경을 선택합니다.**

18. **현재 시스템 로캘**아래의 드롭다운 메뉴에서 사용할 로캘 언어를 선택합니다. 그런 다음 **확인을**선택합니다.

19. 세션을 다시 시작하려면 **지금** 다시 시작을 선택합니다.

축하합니다, 당신은 당신의 언어 팩을 설치했습니다!

계속하기 전에 시스템에 최신 버전의 Windows 및 Windows 저장소가 설치되어 있는지 확인합니다.

## <a name="sysprep"></a>Sysprep

다음으로, 이미지 캡처 프로세스를 준비하기 위해 컴퓨터를 sysprep해야 합니다.

기계를 sysprep하려면:

1. 관리자 권한으로 PowerShell을 엽니다.
2. 다음 cmdlet을 실행하여 올바른 디렉토리로 이동합니다.
   
    ```powershell
    cd Windows\System32\Sysprep
    ```

3. 그런 다음 다음 cmdlet을 실행합니다.
    
    ```powershell
    .\sysprep.exe
    ```

4. 시스템 준비 도구 창이 열리면 **일반화라는**확인란을 선택한 다음 **종료 옵션으로** 이동하여 드롭다운 메뉴에서 **종료를** 선택합니다.

>[!NOTE]
>syprep 프로세스를 완료하는 데 몇 분 정도 걸릴 것입니다. VM이 종료되면 원격 세션의 연결이 끊어집니다.

### <a name="resolve-sysprep-errors"></a>sysprep 오류 해결

sysprep 프로세스 중에 오류 메시지가 표시되면 다음과 같은 작업을 수행해야 합니다.

1. **드라이브 C를** 열고 **Windows** > **System32 Sysprep** > **표범으로**이동한 다음 설치 **파일을** 엽니다.

   오류 파일의 텍스트는 다음 이미지와 같이 특정 언어 패키지를 제거해야 한다는 것을 알려줍니다. 다음 단계에 대한 언어 패키지 이름을 복사합니다.

   ![설정 파일의 스크린샷입니다. 패키지 이름이 있는 텍스트는 진한 파란색으로 강조 표시됩니다.](media/setuperr-package-name.png)

2. 새 PowerShell 창을 열고 2단계에서 복사한 패키지 이름으로 다음 cmdlet을 실행하여 언어 패키지를 제거합니다.

   ```powershell
   Remove-AppxPackage <package name>
   ```

3. cmdlet을 다시 실행하여 패키지를 제거했는지 `Remove-AppxPackage` 확인합니다. 패키지를 성공적으로 제거한 경우 제거하려는 패키지가 없는 메시지가 표시됩니다.

4. cmdlet을 `sysprep.exe` 다시 실행합니다.

## <a name="capture-the-image"></a>이미지 캡처

이제 시스템이 준비되었으므로 구성 프로세스를 반복하지 않고도 다른 사용자가 시스템에 기반한 VM사용을 시작할 수 있도록 이미지를 캡처할 수 있습니다.

이미지를 캡처하려면 다음을 수행합니다.

1. Azure 포털로 이동하여 언어 팩 설치 및 [sysprep](#sysprep)에서 [구성한](#install-a-language-pack) 컴퓨터의 이름을 선택합니다.

2. **캡처를**선택합니다.

3. 이미지의 이름을 **이름** 필드에 입력하고 다음 이미지와 같이 리소스 **그룹** 드롭다운 메뉴를 사용하여 리소스 그룹에 할당합니다.

   ![이미지 만들기 창의 스크린샷입니다. 사용자가 이 테스트 이미지에 부여한 이름은 "vmwvd-image-fr"이며 "testwvdimagerg" 리소스 그룹에 할당했습니다.](media/create-image.png)

4. **만들기**를 선택합니다.

5. 캡처 프로세스가 완료될 때까지 몇 분 정도 기다립니다. 이미지가 준비되면 알림 센터에 이미지가 캡처되었다는 메시지가 표시됩니다.

이제 새 이미지를 사용하여 VM을 배포할 수 있습니다. VM을 배포할 때 Azure 포털이 있는 [가용성 영역에서 Windows 가상 컴퓨터 만들기의](../virtual-machines/windows/create-portal-availability-zone.md)지침을 따라야 합니다.

### <a name="how-to-change-display-language-for-standard-users"></a>표준 사용자의 표시 언어를 변경하는 방법

표준 사용자는 VM에서 표시 언어를 변경할 수 있습니다.

표시 언어를 변경하려면 다음을 수행하십시오.

1. 언어 **설정으로**이동합니다. 위치를 모르는 경우 시작 메뉴의 검색 모음에 **언어를** 입력할 수 있습니다.

2. Windows 표시 언어 드롭다운 메뉴에서 표시 언어로 사용할 언어를 선택합니다.

3. 세션에서 로그아웃한 다음 다시 로그인합니다. 이제 표시 언어가 2단계에서 선택한 언어가 되어야 합니다.
