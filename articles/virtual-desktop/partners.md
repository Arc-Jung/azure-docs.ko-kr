# <a name="windows-virtual-desktop-partner-integrations"></a>Windows 가상 데스크톱 파트너 통합

이 문서에서는 Windows 가상 데스크톱의 파트너 사전순으로 나열합니다.

## <a name="active-cypher"></a>현재 암호

현재 암호 배우게 될 것을 Windows 가상 데스크톱 부가 가치 서비스 공급자가 엔터프라이즈 위반 및 데이터 손실 로부터 보호 하기 위해 암호화 기술 사용 하 여 IT. 현재 암호를 사용 하 여 혁신적인 암호화 기술을 사용 하는 Azure에서 Windows 가상 데스크톱을 실행 하는 전체 파일 보호 전략을 지원 합니다.

Windows 가상 데스크톱을 사용 하는 경우 의도 하지 않은 데이터 손실을 방지 하는 동안 내게 필요한 옵션 및 사용자 데이터를 이동할 수 있도록 추가 기능을 제공 현재 암호입니다. 현재 암호를 사용 하 여 기존 그룹 보안 정책 및 권한을 Azure Information Protection에서 Windows 가상 데스크톱을 사용 하는 동안 사용자 파일을 암호화 하려면 활용할 수 있습니다.

[파트너 웹 사이트로 이동 합니다.](https://activecypher.com/)

## <a name="cloudjumper"></a>CloudJumper

CloudJumper는 솔루션 공급자와 엔터프라이즈 IT 소프트웨어를 사용 하 여 전체적으로 프로 비전 하 고 Windows 가상 데스크톱 환경 관리를 배우게 될 것을 Windows 가상 데스크톱 부가 가치 서비스 공급자입니다. 소프트웨어 CloudJumper 있습니다를 사용 하 여 Windows 가상 데스크톱 배포의 모든 계층을 관리 합니다. 사용자가 모든 장치에서 어디서 나 데스크톱 신속 하 게 액세스할 수 있도록 워크 로드 및 응용 프로그램의 배달 자동화 됩니다.

배포 및 Azure에서 지속적인 관리 작업을 단순화 하 여 Windows 가상 데스크톱의 가치를 확장 하는 CloudJumper의 소프트웨어, 클라우드 작업 영역 관리 제품군을 사용 합니다. 간편함, 있습니다 단일 창에서 프로 비전, 관리 및 사용자 작업 영역에 대 한 인프라를 최적화 합니다. 간단한 스크립트를 트리거하기 엔진 CloudJumper의 Windows 가상 데스크톱을 프로 비전에 관련 된 작업을 자동화 하는 IT 서비스 플랫폼과 통합 됩니다. 또한 CloudJumper Api 추가 확장성 및 ServiceNow 및 BMC 준비와 같은 기타 엔터프라이즈 시스템에 통합할 수 있습니다.

[파트너 웹 사이트로 이동 합니다.](https://cloudjumper.com/wvd/)

## <a name="lakeside-software"></a>Lakeside 소프트웨어

Lakeside 소프트웨어는 Windows 가상 데스크톱 부가 가치 서비스 공급자 성능을 모니터링 하 고 사용자 워크 로드의 Azure 마이그레이션 준비 상태 평가 소프트웨어를 사용 하 여 IT 팀을 배우게 될 것입니다. IT 향상이이 소프트웨어를 사용 하 여 마이그레이션 프로세스를 간소화 하기 위해 응용 프로그램 사용량 및 리소스 소비에 명확 하 게 표시 합니다. Lakeside 소프트웨어 빠른 문제 해결 및 자산 최적화를 사용 하도록 설정 하면 사용자 환경에서 포괄적인 보고서를 만드는 모든 작업 영역에서 데이터를 수집 합니다.

Lakeside 소프트웨어의 디지털 환경을 모니터링 솔루션을 SysTrack, 성능을 추적 하 고 마이그레이션에 대 한 적합 한 워크 로드를 식별 하 여 뛰어난 사용자 환경을 제공 하는 데 도움이 됩니다. SysTrack 적합 한 크기 조정 평가 통해 Windows 가상 데스크톱의 값 및 사용자 환경의 지속적인 모니터링 확장에 작동 합니다.

[파트너 웹 사이트로 이동 합니다.](https://www.lakesidesoftware.com/assessments/wvd)

## <a name="liquidware"></a>Liquidware

Liquidware가 관리 하 고 Windows 가상 데스크톱 배포를 최적화 하는 소프트웨어를 제공 하는 Windows 가상 데스크톱 부가 가치 서비스 공급자입니다. Liquidware Essentials 도구 모음 계층, 사용자 환경 관리 및 주요 사용자 환경 표시 유형 및 진단을 통해 응용 프로그램 전달을 제공합니다. 마이그레이션 준비 상태를 평가 하 고 사용 메트릭을 분석 솔루션을 사용 하 여 Liquidware 최종 사용자를 원활 하 게 가상 데스크톱 환경을 제공 합니다.

Liquidware Essentials 효율적으로 사용자 프로필을 수집 하 고 사용자 환경에서 Azure로의 마이그레이션을 간소화 하기 위해 주요 사용자 데이터를 수집 하 여 Windows 가상 데스크톱의 값을 확장 합니다. 또한 Liquidware Essentials 통합 사용자 프로필 이미지 관리를 간소화 하 고 관리 설정을 구성할 수 있는 권한을 기반 계층 앱.

[파트너 웹 사이트로 이동 합니다.](https://www.liquidware.com/solutions/solutions-platform/microsoft)

## <a name="nerdio"></a>Nerdio

Nerdio는 쉽게 배포 및 Windows 가상 데스크톱을 관리 하는 Azure IT 자동화 플랫폼입니다. Nerdio 지식과 기술을 배포, 가격, 패키지, 관리 및 고객의 Azure 배포 최적화를 제공 합니다.-Windows 가상 데스크톱 앞과 가운데를 사용 하 여 합니다.

Nerdio 쉽게 Azure 리소스를 프로 비전 및 배포를 간소화 하 여 Windows 가상 데스크톱의 값을 확장 합니다. Azure, IT 수에 대 한 Nerdio를 사용 하 여 자동으로 배포 하 고 2 시간 이내에 Windows 가상 데스크톱을 비롯 한 전체 Azure 환경을 관리 합니다.

[파트너 웹 사이트로 이동 합니다.](https://getnerdio.com/academy/windows-virtual-desktop-explained-for-msps/)

## <a name="thinprint"></a>ThinPrint

ThinPrint에 Windows 가상 데스크톱에서 인쇄 하는 간단 하 고 안전한 클라우드를 제공 하는 Windows 가상 데스크톱 부가 가치 서비스 공급자입니다. 해당 서비스 및 소프트웨어를 사용 하 여 클라우드에서 문서를 인쇄 하려면 인쇄 기존 인프라를 활용할 수 있습니다. ThinPrint 쉽게 인쇄 하는 동안 사무실 또는 원격으로 작업 하는 사용자에 대 한 로컬 연결 및 네트워크 프린터에 있습니다.

ThinPrint의 ezeep 솔루션 기존 엔터프라이즈 인쇄 인프라에 대 한 연결을 사용 하 여 Windows 가상 데스크톱의 값을 확장 합니다. ezeep 위치에 관계 없이 엔터프라이즈에서 사용자 인쇄 제어할을 수 있습니다. 사용자는 ezeep를 사용 하 여 Windows 가상 데스크톱 및 인쇄 하드웨어 간의 격차를 메워야 수 있습니다.

[파트너 웹 사이트로 이동 합니다.](http://www.ezeep.com/wvd-printing)

## <a name="workspot"></a>Workspot

Workspot는 고성능 데스크톱 및 Azure에서 워크스테이션을 사용 하 여 기업 배우게 될 것을 Windows 가상 데스크톱 부가 가치 서비스 공급자입니다. Workspot를 사용 하 여 인프라를 프로 비전 자동화, 즉, 사용자가 어디에서 Windows 가상 데스크톱 환경에 액세스할 수 고가용성으로 전 세계입니다.

Workspot은 클라우드 데스크톱 인프라의 프로 비전 프로세스를 간소화 하 여 Windows 가상 데스크톱의 값을 확장 합니다. Workspot, 리소스 확장할 수 있으며 쉽게 위아래로 다른 사용자 및 사용 사례 요구를 충족 하도록 합니다. CAD 및 엔지니어링 사용자 뿐만 아니라 Windows 응용 프로그램에 필요한 고성능 GPU 워크스테이션에 대 한 배포 및 모든 비즈니스 사용자에 대 한 Windows 10 데스크톱 Workspot 최적화할 수 있습니다.

[파트너 웹 사이트로 이동 합니다.](https://www.workspot.com/wvd)
