# <a name="windows-virtual-desktop-partner-integrations"></a>Windows 가상 데스크톱 파트너 통합

이 문서에서는 Windows 가상 데스크톱에 대한 승인된 파트너 공급자 및 독립 소프트웨어 공급업체를 나열합니다.

## <a name="citrix"></a>Citrix

![시트릭스 로고](./media/partners/citrix.png)

Citrix는 기업이 온-프레미스 배포와 나란히 Azure에서 가상 앱 및 데스크톱 워크로드의 중앙 집중식 하이브리드 관리를 제공하는 승인된 공급자입니다. 가상 앱 및 데스크톱 서비스를 갖춘 Citrix Workspace를 통해 사용자는 고급 Citrix HDX 프로토콜을 활용하여 어디서나 고화질 환경을 제공함으로써 모든 장치에서 앱과 데스크톱에 액세스할 수 있습니다.

Citrix는 강력한 엔터프라이즈 도구를 사용하여 Windows Virtual Desktop의 가치를 확장하여 사용자 밀도와 성능을 개선하고, 주문형 워크로드를 프로비저닝하고, 이미지 및 응용 프로그램 관리를 간소화합니다. IT는 지능형 스케일링 도구를 사용하여 비용을 최적화하는 동시에 업계 전반에서 가장 까다로운 애플리케이션에 대해 현장 테스트를 거친 놀라운 사용자 경험을 제공할 수 있습니다. 또한 Citrix 관리 데스크톱은 모든 장치에 가상 앱 및 데스크톱을 제공하기 위한 간단한 클라우드 기반 관리 솔루션을 제공하는 Windows 가상 데스크톱 지원 데스크톱-서비스형 데스크톱 프로그램입니다.

- [파트너 웹 사이트로 이동합니다.](https://more.citrix.com/wvd)

Windows 응용 프로그램 및 데스크톱. 윈도우 가상 데스크톱의 출시와 함께, VMware는 마이크로 소프트에 호라이즌 클라우드를 사용하는 고객에게 윈도우 가상 데스크톱의 기능을 확장하기 위해 마이크로 소프트와 제휴하고있다. 이 기능은 2019년 말까지 기술 미리 보기에서 사용할 수 있습니다.

클라우드하우스 서비스는 검증된 클라우드하우스 컨테이너화 기술을 활용하여 Windows XP, Windows 7 또는 Windows 8용으로 설계된 응용 프로그램을 포함하여 모든 응용 프로그램을 가져와 코드를 변경하거나 변경할 필요 없이 최신 Windows 가상 데스크톱에 배포합니다. 사용자 경험에 영향을 미칩니다. 인쇄는 거의 모든 사무실 및 소규모 비즈니스 환경에서 기본적인 작업입니다. Windows Virtual Desktop을 최대한 활용하고 뛰어난 사용자 환경을 제공하기 위해서는 사용자가 최소한의 노력과 최대의 안정성으로 프린터에 연결할 수 있도록 하는 것이 중요합니다. Printix를 사용하면 단일 사인온(SSO), 자동 구성, 정기적인 업데이트 및 인쇄 환경의 지속적인 모니터링을 통해 Windows 가상 데스크톱을 최대한 사용할 수 있습니다.

RDPSoft를 사용하면 풍부한 기록 보고 및 포괄적인 대시보드를 클릭 한 번으로 확인할 수 있습니다.
