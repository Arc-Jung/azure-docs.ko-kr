# <a name="azure-data-box-edge-technical-specifications"></a>Azure 데이터 박스 에지 기술 사양

Microsoft Azure 데이터 박스 에지 장치의 하드웨어 구성 요소는 이 문서에서 설명하는 기술 사양 및 규정 표준을 준수합니다. 기술 사양은 전원 공급 장치(PSUs), 저장 용량, 인클로저 및 환경 표준을 설명합니다. 

## <a name="compute-memory-specifications"></a>컴퓨팅, 메모리 사양

데이터 상자 가장자리 장치에는 계산 및 메모리에 대한 다음 사양이 있습니다.

| 사양           | 값                  |
|-------------------------|----------------------------|
| CPU    | 2 X 10 코어 CPU                     |
| 메모리              | 128GB RAM                  |


## <a name="fpga-specifications"></a>FPGA 사양

필드 프로그래밍 가능한 게이트 어레이(FPGA)는 기계 학습(ML) 시나리오를 가능하게 하는 모든 데이터 박스 에지 장치에 포함되어 있습니다. 

| 사양           | 값                  |
|-------------------------|----------------------------|
| Fpga   | 인텔 아리아 10 <br> 사용 가능한 DNN(심층 신경망) 모델은 [클라우드 FPGA 인스턴스에서 지원하는](https://docs.microsoft.com/azure/machine-learning/how-to-deploy-fpga-web-service#whats-supported-on-azure)모델과 동일합니다.| 


## <a name="power-supply-unit-specifications"></a>전원 공급 장치 사양

데이터 박스 에지 장치에는 고성능 팬이 있는 두 개의 100-240V 전원 공급 장치(PSUs)가 있습니다. 두 PSUs는 중복 전원 구성을 제공합니다. PSU에 오류가 발생하면 실패한 모듈을 교체할 때까지 장치가 다른 PSU에서 정상적으로 작동합니다. 다음 표에는 PSUs의 기술 사양이 나열되어 있습니다.

| 사양           | 750 W PSU                  |
|-------------------------|----------------------------|
| 최대 출력 전원    | 750 W                     |
| 빈도               | 50/60Hz                   |
| 전압 범위 선택 | 자동 범위: 100-240V AC |
| 핫 플러그형           | yes                        |

## <a name="network-interface-specifications"></a>네트워크 인터페이스 사양

데이터 박스 에지 장치에는 6개의 네트워크 인터페이스인 PORT1- PORT6가 있습니다.

| 사양           | 설명                 |
|-------------------------|----------------------------|
|  네트워크 인터페이스    | 1GbE 인터페이스 2개 - 1개는 관리용으로 사용자가 구성할 수 없으며 초기 설정에 사용됩니다. 다른 인터페이스는 사용자가 구성 가능하고 데이터 전송에 사용할 수 있으며, 기본적으로 DHCP입니다. <br>25GbE 인터페이스 2개 - 10GbE 인터페이스로도 작동할 수 있습니다. 사용자는 이러한 데이터 인터페이스를 DHCP(기본값) 또는 정적으로 구성할 수 있습니다. <br> 25GbE 인터페이스 2개 - 사용자는 이러한 데이터 인터페이스를 DHCP(기본값) 또는 정적으로 구성할 수 있습니다.                  |

## <a name="storage-specifications"></a>스토리지 사양

데이터 박스 에지 장치에는 각각 1.6TB 용량의 9X 2.5" NVMe SSD가 있습니다. 이러한 SSD 중 1개는 운영 체제 디스크이고 나머지 8개는 데이터 디스크입니다. 장치의 총 사용 가능한 용량은 약 12.5TB입니다. 다음 표에는 장치의 저장 용량에 대한 세부 정보가 있습니다.

|     사양                          |     값             |
|--------------------------------------------|-----------------------|
|    SSD(반도체 드라이브)의 수     |    8                  |
|    단일 SSD 용량                     |    1.6TB             |
|    전체 용량                          |    12.8TB            |
|    총 사용 가능한 용량 *                  |    ~ 12.5 TB            |

**일부 공간은 내부용으로 예약되어 있습니다.*
