# HPC 팩 IaaS 배포 스크립트를 사용하여 Windows VM에 HPC(고성능 컴퓨팅) 클러스터 만들기

## 다음 단계

* 클러스터에서 테스트 워크로드를 실행합니다. 예를 보려면 HPC 팩 [시작하기 가이드](https://technet.microsoft.com/library/jj884144)를 참조하세요.

* 스크립트를 사용하여 클러스터를 만들고 HPC 워크로드를 실행하는 자습서를 보려면 [Azure에서 Excel 및 SOA 워크로드를 실행할 HPC 팩 클러스터 시작하기](virtual-machines-windows-excel-cluster-hpcpack.md)를 참조하세요.

* 사용자가 만든 클러스터에서 계산 노드를 시작, 중지, 추가, 제거하는 HPC 팩의 도구. [Azure에서 HPC 팩 클러스터의 계산 노드 관리](virtual-machines-windows-classic-hpcpack-cluster-node-manage.md)를 참조하세요.
