# <a name="support-for-32-bit-operating-systems-in-azure-virtual-machines"></a>Azure 가상 머신에서 32비트 운영 체제 지원

이제 Microsoft Azure를 통해 사용자가 32비트 Windows 운영 체제를 Azure로 가져올 수 있습니다. 이러한 운영 체제 중 일부는 이미 수명 종료 지원 계약에 도달했기 때문에 Microsoft는 추가 지원을 제공하지 않을 수 있습니다. Linux 기반 또는 VM(Microsoft Azure 가상 머신)에서 실행되는 BSD(버클리 소프트웨어 배포) 기반 운영 체제에 대한 지원도 제공되지 않습니다.

## <a name="more-information"></a>자세한 정보

Azure 가상 컴퓨터에서 지원되는 운영 체제에 대한 자세한 내용은 다음 Microsoft 기술 자료 문서로 이동하십시오.

* [Microsoft Azure 가상 머신에 대한 Microsoft 서버 소프트웨어 지원](https://support.microsoft.com/help/2721672/microsoft-server-software-support-for-microsoft-azure-virtual-machines)
* [Azure의 Linux 및 오픈 소스 기술 지원](https://support.microsoft.com/help/2941892/support-for-linux-and-open-source-technology-in-azure)
