# Azure 가상 컴퓨터에 호스트된 Oracle 소프트웨어

첫 번째 또는 다음 개발/테스트 또는Microsoft Azure 인프라 서비스에서 실행되는 프로덕션 Oracle 데이터베이스를 설정합니다. 새 용량을 포함하도록 Oracle 데이터베이스를 신속히 확장하는 쉬운 구성과 기능을 활용하세요.

## Oracle 가상 컴퓨터 이미지

최신 Windows 및 Linux 기반 가상 컴퓨터 이미지에 대한 [Oracle 가상 컴퓨터 이미지의 목록](virtual-machines-oracle-list-oracle-virtual-machine-images.md) 참조

## Windows 기반 가상 컴퓨터에 대한 고려 사항

Windows 서버 기반 Azure 가상 컴퓨터의 Oracle 소프트웨어에 대한 제한 사항을 이해하려면 [Oracle 가상 컴퓨터 이미지에 대한 기타 고려 사항](virtual-machines-miscellaneous-considerations-oracle-virtual-machine-images.md) 참조

## Oracle 데이터 베이스 가상 컴퓨터

[Azure에서 Oracle 데이터베이스를 만들기](virtual-machines-creating-oracle-database-virtual-machine.md)는 가상 컴퓨터 만들기, 데이터베이스 만들기 및 널리 사용되는 옵션의 구성을 가능하게 합니다.

## Oracle WebLogic Server 12c

[Azure에서 Oracle WebLogic Server 12c 만들기](virtual-machines-creating-oracle-weblogic-server-12c-virtual-machine.md)는 Windows Server 2012에서 실행되는 Oracle WebLogic Server 12c 이미지에 기반한 Azure 가상 컴퓨터 만들기를 가능하게 합니다.

## Oracle WebLogic Server 12c 및 Oracle 데이터베이스 12C

[Azure에서 Oracle WebLogic Server 12c 및 Oracle 데이터 베이스 12c 가상 컴퓨터 만들기](virtual-machines-creating-oracle-weblogic-server-12c-oracle-database-12c-virtual-machine.md)는 가상 컴퓨터 및 데이터 베이스 만들기 및 구성을 가능하게 합니다.

## Oracle WebLogic Server 12c 클러스터

[Azure에서 Oracle WebLogic Server 12 클러스터 만들기](virtual-machines-creating-oracle-weblogic-server-12c-cluster.md)는 클러스터 만들기 및 응용 프로그램 배포를 가능하게 합니다.

## Oracle WebLogic Server 11g

[Azure에서 Oracle WebLogic Server 11를 만들기](virtual-machines-creating-oracle-weblogic-server-11g-virtual-machine.md)는 가상 컴퓨터 만들기, 구성 및 응용 프로그램 설치를 가능하게 합니다.

## Oracle Linux 가상 컴퓨터 준비

[Azure용 Oracle Linux 가상 컴퓨터 준비](virtual-machines-prepare-oracle-linux-virtual-machine.md)는 Oracle Linux 6.4+ 및 Oracle Linux 7.0+에 대한 필수 구성 요소 및 가상 컴퓨터 만들기를 가능하게 합니다.

## Oracle 데이터 가드

[Azure에 대해 Oracle 데이터 가드 구성하기](virtual-machines-configuring-oracle-data-guard.md)는 대기 환경 및 수신기의 구성을 가능하게 합니다.

## Oracle GoldenGate

[Azure에 대한 Oracle Goldengate 구성](virtual-machines-configuring-oracle-goldengate.md)은 데이터 베이스 및 복제 설치를 가능하게 합니다.
