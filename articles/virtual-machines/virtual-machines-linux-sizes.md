# Azure에서 Linux 가상 컴퓨터에 대한 크기

이 문서에서는 앱 및 작업을 실행하는데 사용할 수 있는 Linux 가상 컴퓨터에 대한 크기 및 옵션을 설명합니다. 또한 이러한 리소스의 사용 계획을 세울 때 알아야 할 배포 고려 사항도 제공합니다.또한 이러한 리소스의 사용 계획을 세울 때 알아야 할 배포 고려 사항도 제공합니다. 다양한 크기의 가격 책정에 대한 자세한 내용은 [가상 컴퓨터 가격 책정](https://azure.microsoft.com/pricing/details/virtual-machines/)을 참조하세요. Windows VM에 대한 정보는 [Azure의 Windows 가상 컴퓨터에 대한 크기](virtual-machines-windows-sizes.md)를 참조하세요.
