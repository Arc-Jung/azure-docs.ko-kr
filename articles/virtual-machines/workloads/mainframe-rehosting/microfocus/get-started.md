# <a name="micro-focus-devtest-environments-on-azure"></a>Azure에서 마이크로 포커스 개발/테스트 환경

현대화 활동을 높일 수 있는 좋은 방법 이며 개발 및 테스트 환경을 사용 하 여 Azure에서 시작 파트너 솔루션을 사용 하 여 Azure에서 다시 IBM CICS (고객 정보 제어 시스템) 및 IMS (정보 관리 시스템)와 같은 레거시 시스템에 의존 하는 메인프레임 응용 프로그램을 호스트할 수 있습니다.

Micro Focus는 매력적인 메인프레임 마이그레이션 및 재호스팅 도구를 제공 하는 Microsoft와 협력 했습니다. Azure 및 Microsoft Windows 플랫폼 개발, 테스트 및 프로덕션 시나리오에서 메인프레임 응용 프로그램을 실행할 수 있습니다.

Micro Focus 제공 재호스팅 개발 및 테스트 기존 COBOL 및 PL 확장 하는 솔루션을 다시 호스팅할 / I 비즈니스 응용 프로그램을 Azure.
