## <a name="get-the-training-and-prediction-keys"></a>학습 및 예측 키 가져오기

서비스와 상호 작용하려면 프로젝트에 유효한 구독 키 집합이 필요합니다. 평가판 키 집합을 얻으려면 [Custom Vision 웹 사이트](https://customvision.ai)로 이동하여 Microsoft 계정으로 로그인합니다. 오른쪽 위에서 __기어 아이콘__ 을 선택합니다. __계정__ 섹션에서 __학습 키__, __예측 키__ 및 __예측 리소스 ID__ 필드의 값을 확인합니다. 나중에 필요합니다.
