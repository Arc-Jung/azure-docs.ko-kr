```Bash
pip install azure-cognitiveservices-speech
```

> [!TIP]
> macOS에 있는 경우 위의 pip 명령을 작동 하려면 다음 명령을 실행 해야 할 수 있습니다.
> ```Bash
> python3 -m pip install --upgrade pip
> ```
