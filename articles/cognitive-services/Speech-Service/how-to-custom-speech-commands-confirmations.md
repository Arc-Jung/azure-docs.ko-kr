# <a name="how-to-add-a-confirmation-to-a-custom-command-preview"></a>방법: 사용자 지정 명령에 확인 추가 (미리 보기)

이 문서에서는 명령에 확인을 추가 하는 방법에 대해 알아봅니다.

## <a name="prerequisites"></a>사전 요구 사항

다음 문서의 단계를 완료 해야 합니다.

- [빠른 시작: 사용자 지정 명령 만들기 (미리 보기)](./quickstart-custom-speech-commands-create-new.md)
- [빠른 시작: 매개 변수를 사용 하 여 사용자 지정 명령 만들기 (미리 보기)](./quickstart-custom-speech-commands-create-parameters.md)

## <a name="create-a-setalarm-command"></a>SetAlarm 명령 만들기

유효성 검사를 보여 주기 위해 사용자에 게 경보를 설정 하도록 허용 하는 새 명령을 만들어 보겠습니다.

1. [Speech Studio](https://speech.microsoft.com/) 에서 이전에 만든 사용자 지정 명령 응용 프로그램 열기
1. 새 명령 **Setalarm** 만들기
1. DateTime 이라는 매개 변수 추가

   | 설정           | 제안 값                                          | Description                                                                                      |
   | ----------------- | ---------------------------------------------------------| ------------------------------------------------------------------------------------------------ |
   | 속성              | DateTime                                                 | 명령 매개 변수에 대 한 설명이 포함 된 이름입니다.                                                    |
   | 필수          | true                                                     | 명령을 완료 하기 전에이 매개 변수의 값이 필요한 지 여부를 나타내는 확인란 |
   | 응답 템플릿 | "-시간"                                           | 이 매개 변수 값을 알 수 없는 경우 요청 하는 프롬프트입니다.                              |
   | Type              | DateTime                                                 | 숫자, 문자열 또는 날짜 시간과 같은 매개 변수의 형식입니다.                                      |
   | 날짜 기본값     | 오늘 날짜를 누락 하는 경우                             |                                                                                                  |
   | 시간 기본값     | 시간이 누락 된 경우 하루 시작 사용                      |                                                                                                  | 

1. 몇 가지 샘플 문장 추가
   
   ```
    set an alarm for {DateTime}
    set alarm {DateTime}
    alarm for {DateTime}
   ```

1. 결과를 확인 하려면 완료 규칙을 추가 합니다.

   | 설정    | 제안 값                                         | Description                                        |
   | ---------- | ------------------------------------------------------- | -------------------------------------------------- |
   | 규칙 이름  | 경보 설정                                               | 규칙의 용도를 설명 하는 이름입니다.          |
   | 작업    | SpeechResponse-"-Ok, {DateTime}"에 대 한 경보 집합       | 규칙 조건이 참인 경우 수행할 동작입니다. |

## <a name="try-it-out"></a>기능 직접 사용해 보기

테스트 패널을 선택 하 고 몇 가지 상호 작용을 시도 합니다.

- 입력: 내일 정오에 경보 설정
- 출력: "확인, 12/06/2019 12:00:00에 대 한 경보 설정"

- 입력: 경보 설정
- 출력: "시간"
- 입력: 오후 5 시
- 출력: "확인, 12/05/2019 17:00:00에 대 한 경보 설정"

## <a name="add-the-advanced-rules-for-confirmation"></a>확인에 대 한 고급 규칙 추가

1. 확인에 대 한 고급 규칙을 추가 합니다. 

    이 규칙은 사용자에 게 알람의 날짜 및 시간을 확인 하 고 다음 턴에 확인 (예/아니요)을 예상 하도록 요청 합니다.

   | 설정               | 제안 값                                                                  | Description                                        |
   | --------------------- | -------------------------------------------------------------------------------- | -------------------------------------------------- |
   | 규칙 이름             | 날짜 시간 확인                                                                | 규칙의 용도를 설명 하는 이름입니다.          |
   | 조건            | 필수 매개 변수-DateTime                                                    | 규칙을 실행할 수 있는 시기를 결정 하는 조건    |   
   | 작업               | SpeechResponse-"-{DateTime}"에 대 한 경보를 설정 하 시겠습니까?       | 규칙 조건이 참인 경우 수행할 동작입니다. |
   | 실행 후의 상태 | 입력 대기                                                                   | 다시 설정한 후 사용자에 대 한 상태                  |
   | 수준과          | 확인                                                                     | 다음 턴의 기대                      |

1. 성공 확인을 처리 하는 고급 규칙을 추가 합니다 (사용자가 예).

   | 설정               | 제안 값                                                                  | Description                                        |
   | --------------------- | -------------------------------------------------------------------------------- | -------------------------------------------------- |
   | 규칙 이름             | 수락 된 확인                                                            | 규칙의 용도를 설명 하는 이름입니다.          |
   | 조건            | SuccessfulConfirmation & 필수 매개 변수-DateTime                           | 규칙을 실행할 수 있는 시기를 결정 하는 조건    |   
   | 실행 후의 상태 | 완료 준비 완료                                                             | 턴 후의 사용자 상태                   |

1. 확인 거부를 처리 하는 고급 규칙을 추가 합니다 (사용자가 언급 하지 않음).

   | 설정               | 제안 값                                                                  | Description                                        |
   | --------------------- | -------------------------------------------------------------------------------- | -------------------------------------------------- |
   | 규칙 이름             | 거부 확인                                                                   | 규칙의 용도를 설명 하는 이름입니다.          |
   | 조건            | DeniedConfirmation & 필수 매개 변수-DateTime                               | 규칙을 실행할 수 있는 시기를 결정 하는 조건    |   
   | 작업               | ClearParameter-DateTime & SpeechResponse-"-문제 없음, 시간"     | 규칙 조건이 참인 경우 수행할 동작입니다. |
   | 실행 후의 상태 | 입력 대기                                                                   | 턴 후의 사용자 상태                   |
   | 수준과          | ElicitParameters-DateTime                                                      | 다음 턴의 기대                      |
