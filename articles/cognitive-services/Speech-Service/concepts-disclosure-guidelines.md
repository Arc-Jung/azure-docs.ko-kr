# <a name="disclosure-design-guidelines"></a>디자인 지침 공개
음성 경험의 합성 특성에 대해 투명하게 파악하여 고객과의 신뢰를 구축하고 유지하는 방법을 알아봅니다.

## <a name="what-is-disclosure"></a>공개란 무엇입니까?

공개는 사람들이 합성으로 생성된 음성과 상호 작용하거나 듣고&#39;있음을 알리는 수단입니다.

## <a name="why-is-disclosure-necessary"></a>공개가 필요한 이유는 무엇입니까?

컴퓨터로 생성된 음성의 합성 기원을 공개할 필요성은 비교적 새로운 것입니다. 과거에는 컴퓨터로 생성된 목소리가 분명히 있었는데, 아무도 실제 사람으로 착오하지는 않을 것입니다. 그러나 매일 합성 음성의 리얼리즘이 향상되고 인간의 목소리와 점점 구별되지 않습니다.

## <a name="goals"></a>목표
다음은 합성 음성 환경을 설계할 때 염두에 두어야 할 원칙입니다.

**신뢰 강화**
<br>경험을 저하시키지 않고 튜링 테스트에 실패할 목적으로 설계합니다. 사용자가 가상 음성과 상호 작용하고 있다는 사실에 대해 강조하면서 경험에 원활하게 참여할 수 있도록 합니다.

**사용 컨텍스트에 맞게 조정**
<br>사용자가 합성 음성과 상호 작용하는 시기, 위치 및 방법을 파악하여 적절한 유형의 공개를 적시에 제공합니다.

**명확한 기대치 설정**
<br>사용자가 에이전트의 기능을 쉽게 검색하고 이해할 수 있도록 합니다. 요청 시 합성 음성 기술에 대해 자세히 알아볼 수 있는 기회를 제공합니다.

**실패 수용**
<br>실패의 순간을 사용하여 에이전트의 기능을 강화합니다.

## <a name="how-to-use-this-guide"></a>이 가이드를 사용하는 방법

이 가이드는 합성 음성 경험에 가장 적합한 공개 패턴을 결정하는 데 도움이 됩니다. 그런 다음 사용 방법과 시기에 대한 예제를 제공합니다. 이러한 각 패턴은 인간 중심의 디자인에 충실하면서 합성 음성에 대한 사용자의 투명성을 극대화하도록 설계되었습니다.

음성 경험에 대한 방대한 디자인 지침을 고려하여 여기에서는 다음을 구체적으로 중점합니다.

1. [**공개 평가**](#disclosure-assessment): 합성 음성 경험에 권장되는 공개 유형을 결정하는 프로세스

2. [**공개 방법**](concepts-disclosure-patterns.md): 합성 음성 경험에 적용할 수 있는 공개 패턴의 예

3. [**공개 시기**](concepts-disclosure-patterns.md#when-to-disclose): 사용자 여정 전반에 걸쳐 공개할 최적의 순간

## <a name="disclosure-assessment"></a>공시 평가
사용자가 상호 작용에 대한 기대치와 음성을 경험하게 될 컨텍스트에 대한 기대치를&#39; 고려합니다. 맥락에서 합성 음성이 &quot;말하고 있음을 분명히&quot; 하는 경우 공개는 최소한이거나 순간적이거나 불필요할 수 있습니다. 공개에 영향을 주는 컨텍스트의 주요 유형에는 페르소나 유형, 시나리오 유형 및 노출 수준이 포함됩니다. 또한 누가 듣고 있을지 고려하는 데 도움이 된다.

### <a name="understand-context"></a>컨텍스트 이해

이 워크시트를 사용하여 합성 음성 환경의 컨텍스트를 확인합니다. 공개 수준을 결정하는 다음 단계에서 이를 적용합니다.

|                                    | 사용 컨텍스트                                                                                                                                                                                                                                                                                                                                                       | 잠재적 위험 & 과제                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **1. 페르소나 타입**               | **다음 중 어느 한 가지 라도 적용되는 경우 페르소나는 '인간과 같은 페르소나' 범주에 속합니다.**<br><br><ul><li> 페르소나는 가상의 표현이든 아니든 진정한 인간을 구현합니다. (예: 사진 또는 실제 사람의 컴퓨터 생성 렌더링)<br><br><li> 합성 음성은 널리 인식 할 수있는 실제 인물 (예 : 유명 인사, 정치인)의 목소리를 기반으로합니다. | 페르소나에 인간과 같은 표현을 할수록 사용자가 실제 사용자와 연결하거나 콘텐츠가 컴퓨터로 생성된 것이 아니라 실제 사람이 말하는 것으로 믿게 될 가능성이 높습니다. </ul>                                                                                                                                                                      |
| **2. 시나리오 유형**            | **다음 중 어느 한 가지라도 적용되면 음성 환경이 '민감한' 범주에 속합니다.**<br><br><ul><li> 사용자로부터 개인 정보를 얻거나 표시하는 경우 <br><br> <li> 방송 시간 민감한 뉴스/정보(예: 비상 경보)<br><br><li> 실제 사람들이 서로 통신할 수 있도록 돕는 것을 목표로 합니다(예: 개인 이메일/텍스트 읽기)<br><br> <li> 의료/건강 지원 제공 </ul>            | 주제가 민감하거나 개인적이거나 긴급한 문제와 관련이 있을 때 합성 음성을 사용하는 것이 적절하거나 신뢰할 수 없다고 느낄 수 있습니다. 그들은 또한 실제 인간으로 공감과 문맥 인식의 동일한 수준을 기대할 수 있습니다. |
| **3. 노출 수준** |**다음과 같은 경우 음성 환경이 '높음' 범주에 가장 적합합니다.** <br><br><ul><li>사용자는 합성 음성을 자주 듣거나 상호 작용하거나 장시간 동안 </ul>                                                                                                                                                                             | 장기적인 관계를 수립할 때 투명하고 사용자와의 신뢰를 구축하는 것의 중요성은 더욱 높습니다.                                                                                                                                                                                                                                                                      |
