# <a name="speech-sdk-reference"></a>음성 SDK 참조

음성 SDK(소프트웨어 개발 키트)는 음성 사용이 가능한 소프트웨어를 개발하기 쉽도록 응용 프로그램이 음성 서비스의 기능에 액세스할 수 있게 합니다. 현재 SDK에서는 **음성 텍스트 변환**, **음성 번역** 및 **의도 인식**에 대한 액세스 권한을 제공합니다.
