# <a name="speech-sdk-reference"></a>음성 SDK 참조

음성 SDK(소프트웨어 개발 키트)는 음성 사용이 가능한 소프트웨어를 개발하기 쉽도록 응용 프로그램이 음성 서비스의 기능에 액세스할 수 있게 합니다. 현재, SDK는 **음성 텍스트 변환** 및 **음성 번역**에 대한 액세스를 제공합니다. 텍스트 음성 변환은 HTTP를 통한 [REST](https://docs.microsoft.com/azure/cognitive-services/speech-service/rest-apis)) POST 호출을 사용합니다.

아래 표에서는 현재 지원되는 프로그래밍 언어 및 운영 체제를 설명합니다.

|프로그래밍 언어|플랫폼|API 참조|
|-|-|-|
|C#|Windows|[Browse](https://aka.ms/csspeech/csharpref)
|C/C++|Windows, Linux|[Browse](https://aka.ms/csspeech/cppref)
|Java\*|장치|[Browse](https://aka.ms/csspeech/javaref)

\* *Java SDK는 [음성 장치 SDK](speech-devices-sdk.md).* 에 포함됩니다.
