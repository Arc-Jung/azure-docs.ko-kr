# <a name="quickstart-use-custom-commands-with-custom-voice-preview"></a>빠른 시작: 사용자 지정 음성(미리 보기)으로 사용자 지정 명령 사용

이전 [문서에서는](./quickstart-custom-speech-commands-create-parameters.md)매개 변수를 사용하여 명령에 응답하는 새 사용자 지정 명령 프로젝트를 만들었습니다.

이 문서에서는 만든 응용 프로그램에 대 한 사용자 지정 출력 음성을 선택 합니다.

## <a name="select-a-custom-voice"></a>사용자 지정 음성 선택

1. [이전에 만든](./quickstart-custom-speech-commands-create-parameters.md) 프로젝트 열기
1. 왼쪽 창에서 **설정** 선택
1. 가운데 창에서 **사용자 지정 음성** 선택
1. 테이블에서 원하는 사용자 지정 또는 공용 음성 선택
1. **저장** 선택

> [!div class="mx-imgBorder"]
> ![매개 변수가 있는 샘플 문장](media/custom-speech-commands/select-custom-voice.png)

> [!NOTE]
> 사용자 지정 음성 프로젝트 페이지에서 만들 수 있습니다. 음성 **스튜디오** 링크를 선택한 다음 **사용자 지정 음성을** 클릭하여 시작합니다.

이제 응용 프로그램이 기본 음성 대신 선택한 음성으로 응답합니다.
