# <a name="upload-images"></a>이미지 업로드 #

사용자 인터페이스를 탐색하려면 이미지를 한 번에 5개까지 업로드합니다. 각 이미지 파일 크기는 2MB를 넘지 않아야 합니다. 업로드하기 전에 파일 크기를 확인하세요.

![이미지 업로드](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>조정을 위해 제출 ##
한 번에 6개 이상의 이미지를 업로드하게 되는 경우 휴지통 아이콘을 사용하여 원치 않은 이미지를 제거합니다. 그런 다음, 이미지를 제출하면 자동으로 조정됩니다.

![이미지 제출](images/1-try-image-2.png)
