# <a name="view-dashboard"></a>대시보드 보기

**대시보드** 탭을 클릭하면 도구 내에서 수행된 모든 콘텐츠 검토의 주요 메트릭을 볼 수 있습니다. 이미지 및 텍스트에 대한 총 검토, 완료된 검토, 진행 중인 검토를 볼 수 있습니다. 기본 팀 및 하위 팀에 대한 메트릭을 볼 수 있습니다.
