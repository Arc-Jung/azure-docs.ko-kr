# <a name="quickstart-use-the-text-analytics-client-library-v2"></a>빠른 시작: Text Analytics 클라이언트 라이브러리 v2 사용

Text Analytics 클라이언트 라이브러리 v2를 시작합니다. 이러한 단계에 따라 패키지를 설치하고 기본 작업을 위한 예제 코드를 사용해 봅니다.

Text Analytics 클라이언트 라이브러리 v2를 사용하여 다음을 수행합니다.

* 정서 분석
* 언어 검색
* 엔터티 인식
* 핵심 문구 추출
