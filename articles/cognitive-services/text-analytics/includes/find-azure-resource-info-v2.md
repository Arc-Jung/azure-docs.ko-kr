> [!IMPORTANT]
> Azure Portal에서 키와 엔드포인트를 찾으려면 다음을 수행합니다.
> 1. [https://portal.azure.com/](https://portal.azure.com/ ) 에서 Azure 리소스로 이동합니다.
> 2. **리소스 관리** 아래에 있는 **빠른 시작**을 클릭합니다.
>
> 완료되면 코드에서 자격 증명을 제거하고 공개적으로 게시하지 마세요. 프로덕션의 경우 자격 증명을 안전하게 저장하고 액세스하는 방법을 사용하는 것이 좋습니다. 예를 들어 [Azure Key Vault](https://docs.microsoft.com/azure/key-vault/key-vault-overview)입니다.
