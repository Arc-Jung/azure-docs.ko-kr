# <a name="get-started-with-speech-service"></a>Speech Service 시작

Microsoft Speech Recognition 서비스를 사용하여 오디오를 텍스트로 변환하려면 Speech Recognition [REST API](GetStartedREST.md) 또는 [클라이언트 라이브러리](GetStartedClientLibraries.md)를 참조하세요.

Microsoft 텍스트 음성 변환 서비스를 사용하려면 [Text to Speech API 참조](../api-reference-rest/bingvoiceoutput.md)를 참조하세요.
