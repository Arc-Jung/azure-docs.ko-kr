# <a name="what-is-the-linguistic-analysis-api-preview"></a>Linguistic Analysis API란? (미리 보기)

> [!IMPORTANT]
> Linguistic Analysis 미리 보기는 2018년 8월 9일부로 서비스 해제되었습니다. 텍스트 처리 및 분석에는 [Azure Machine Learning 텍스트 분석 모듈](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics)을 사용하는 것이 좋습니다.

Linguistic Analysis API는 텍스트 구조를 식별하는 NLP(자연어 처리) 도구에 대한 액세스를 제공합니다. 현재 릴리스에서는 세 가지 유형의 분석을 제공합니다.

- [문장 분리 및 토큰화](Sentences-and-Tokens.md)
- [품사 태그 지정](POS-tagging.md)
- [선거구 구문 분석](Constituency-Parsing.md)

## <a name="getting-started"></a>시작하기

위에 나열된 주요 분석 유형에 익숙해지면 [사용 가능한 분석기 목록 가져오기](AnalyzersMethod.md)와 분석기가 [명명된](Analyzer-Names.md) 방식 및 [새로운 입력 분석](AnalyzeMethod.md)에 대한 엔드포인트를 확인할 수 있습니다.
