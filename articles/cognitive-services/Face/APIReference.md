# <a name="face-api-reference-list"></a>얼굴 API 참조 목록

Azure Face는 얼굴 감지 및 인식 알고리즘을 제공하는 클라우드 기반 서비스입니다. Face API는 다음 범주로 구성됩니다.

- 얼굴 알고리즘 API: [감지,](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395236) [유사 찾기,](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395237) [확인,](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f3039523a) [식별](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)및 그룹과 같은 핵심 기능을 다룹니다. [Group](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395238)
- [FaceList API](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f3039524b): [유사한 찾기를](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395237)위한 FaceList를 관리하는 데 사용됩니다.
- [대형 개인 그룹 사람 API](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/599adcba3a7b9412a4d53f40): [식별을](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)위해 대형 개인 그룹 사람 얼굴을 관리하는 데 사용됩니다.
- [대형 사용자 그룹 API](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/599acdee6ac60f11b48b5a9d): [식별을](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)위해 LargePersonGroup 데이터 집합을 관리하는 데 사용됩니다.
- [LargeFaceList API](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/5a157b68d2de3616c086f2cc): [유사한 찾기를](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395237)위한 대형 페이스리스트를 관리하는 데 사용됩니다.
- [개인 그룹 사람 API](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f3039523c): [식별을](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)위해 사람 그룹 사람 얼굴을 관리하는 데 사용됩니다.
- [PersonGroup API](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395244): [식별을](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/563879b61984550f30395239)위해 개인 그룹 데이터 집합을 관리하는 데 사용됩니다.
- [스냅숏 API](https://westus.dev.cognitive.microsoft.com/docs/services/563879b61984550e40cbbe8d/operations/snapshot-take): 구독 간에 데이터 마이그레이션을 위한 스냅숏을 관리하는 데 사용됩니다.
