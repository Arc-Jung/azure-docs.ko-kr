## <a name="overview"></a>개요
Anomaly Finder API를 사용하면 시간에 따라 데이터를 모니터링하고, 올바른 통계 모델을 자동으로 적용하여 고유 데이터에 맞게 수정되는 Machine Learning으로 변칙을 검색할 수 있습니다. Anomaly Finder API를 통해 조직은 데이터 품질을 유지 관리하고, 신뢰할 수 있는 서비스를 제공하며, 비즈니스 인시던트를 식별하고, 비정상적인 데이터를 수동으로 식별하지 않고도 비즈니스 접근 방식을 구체화할 수 있습니다.
