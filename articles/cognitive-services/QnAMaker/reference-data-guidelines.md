# <a name="import-and-export-data-reference"></a>데이터 참조 가져오기 및 내보내기

이 가져오기 및 내보내기 참조를 검토하여 기술 자료 백업, 저장소 및 교체에 대한 최상의 결과를 얻을 수 있습니다.

## <a name="import-and-export-knowledge-base"></a>지식 자료 가져오기 및 내보내기

내보낸 기술 자료의 **TSV 및 XLS 파일은**QnA Maker 포털의 **설정** 페이지에서 파일을 가져와야만 사용할 수 있습니다. 기술 자료를 만드는 동안 또는 **설정** 페이지에서 + **파일 추가** 또는 **+ URL 추가** 기능에서 데이터 원본으로 사용할 수 없습니다.
