# <a name="share-your-custom-search-instance"></a>Custom Search 인스턴스 공유
메일 주소만 사용하여 원하는 모든 사용자와 Custom Search 인스턴스를 공유할 수 있습니다. 공유를 사용하면 다른 사용자가 Custom Search 인스턴스를 수정할 수 있습니다. [Custom Search](https://customsearch.ai)에 로그인하고 Custom Search 인스턴스를 선택한 다음, 설정 아이콘(톱니바퀴 모양으로 표시됨)을 클릭합니다. **인스턴스 공유** 섹션에서 인스턴스를 공유할 사용자의 메일 주소를 입력하고 **공유**를 클릭합니다. **Instance shared with**(인스턴스가 공유된 사용자) 목록에 해당 메일 주소가 표시됩니다. 사용자 지정 검색을 공유할 각 사용자에 대해 반복합니다. 

Custom Search 계정이 없는 사용자의 메일도 목록에 추가할 수 있습니다. 해당 사용자가 구성을 변경하려면 먼저 Custom Search에 등록해야 합니다. 다른 사용자와 인스턴스를 공유하면 해당 Custom Search 인스턴스 목록에 표시됩니다. 한 번에 한 사용자만 인스턴스를 수정할 수 있습니다. 다른 사용자가 편집 중인 인스턴스를 수정하려고 하면 경고가 표시됩니다. 최대 10명의 사용자와 인스턴스를 공유할 수 있습니다.

## <a name="stop-sharing"></a>공유 중지
다른 사용자와 인스턴스 공유를 중지하려면 제거 아이콘을 사용하여 해당 메일 주소를 목록에서 제거합니다.
