# 개요: Azure 인식 서비스에서 화자 인식 API

화자 인식 기능이 응용 프로그램을 향상할 수 있는 방법에 대해 알아봅니다. 화자 인식 API는 화자 검증 및 화자 식별을 위해 가장 발전된 알고리즘을 제공하는 클라우드 기반 API입니다. 화자 인식은 다음과 같은 두 가지 범주로 나눌 수 있습니다.

  * 화자 검증
  * 화자 식별

## 화자 검증

음성은 지문과 마찬가지로 사람을 식별하는 데 사용할 수 있는 고유한 특징이 있습니다. 음성을 액세스 제어 및 인증 시나리오를 위한 신호로 사용하는 혁신적인 도구로 부상했습니다. 기본적으로 고객을 위한 인증 환경을 간소화하는 보안의 수준을 높입니다.

화자 검증 API는 해당 음성 또는 발화를 사용하여 사용자를 자동으로 검증하고 인증할 수 있습니다.

### 화자 검증 등록

화자 검증을 위한 등록은 텍스트에 종속적입니다. 즉, 화자가 등록 및 검증 단계에서 모두 사용할 특정 전달 구를 선택해야 합니다.

등록 단계에서 화자의 음성은 특정 구를 말하도록 기록된 후에 다양한 기능이 추출되고 선택한 구가 인식됩니다. 추출된 기능과 선택한 구가 함께 고유한 음성 시그니처를 구성합니다.

### 입력 음성 및 구 검증

검증에서 동일한 사용자로부터 비롯되었는지 및 올바른 구를 말하고 있는지 여부를 검증하기 위해 입력 음성 및 구문을 등록된 음성 시그니처 및 구와 비교합니다.

화자 검증에 대한 자세한 내용은 [화자 인식 API 설명서의 부분 검증](https://dev.projectoxford.ai/docs/services/563309b6778daf02acc0a508/operations/563309b7778daf06340c9652)을 참조하세요.

## 화자 식별

화자 식별 API는 잠재적인 화자 그룹이 지정된 오디오 파일에서 말하는 사람을 자동으로 식별할 수 있습니다. 입력 오디오는 제공된 화자 그룹에 연결되고 일치하는 항목이 있을 경우 화자의 신원이 반환됩니다.

모든 화자는 먼저 등록 프로세스를 통해 시스템에 등록된 음성 가져오고 음성 인쇄를 생성해야 합니다.

### 화자 식별 등록

화자 식별을 위한 등록은 텍스트에 독립적입니다. 즉, 화자가 오디오로 말한 내용에 제한이 없음을 의미합니다. 화자의 음성을 기록하면 다양한 기능이 추출되어 고유한 음성 시그니처를 구성합니다.

### 화자 인식: 입력 음성 일치

잠재적인 그룹의 화자와 함께 알 수 없는 화자의 오디오는 인식 단계에서 제공됩니다. 누구의 음성인지를 결정하기 위해 모든 화자와 입력 음성을 비교하고 일치를 찾은 경우 화자의 신원을 반환합니다.

화자 식별에 대한 자세한 내용은 [화자 인식 API 설명서의 부분 식별](https://dev.projectoxford.ai/docs/services/563309b6778daf02acc0a508/operations/5645c068e597ed22ec38f42e)을 참조하세요.
