# <a name="how-to-sign-up-for-the-translator-speech-api"></a>Translator Speech API에 등록하는 방법

다음 단계에 따라 Translator Speech API에 등록합니다.

## <a name="1-sign-into-azure"></a>1: Azure에 로그인

- 아직 계정이 없으세요? [Microsoft Azure 계정에 등록](http://azure.com/)
- 이미 계정이 있으신가요? [로그인](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: Microsoft Translator 구독

- Azure에 로그인하면 [Azure Cognitive Services](https://portal.azure.com/#create/Microsoft.CognitiveServices) 섹션으로 이동합니다.
- "API 형식" 아래에서 Translator Speech API를 선택합니다.
- [가격 책정 계층] 섹션에서 요구 사항에 가장 적합한 가격 책정 계층을 선택합니다.
    - 각 구독에는 체험 계층이 있습니다. 체험 계층에는 유료 요금제와 동일한 특징과 기능이 있으며 만료 날짜가 없습니다.
    - 계정에는 체험 구독 하나만 있을 수 있습니다.
- 양식의 나머지 부분을 작성한 다음, [만들기] 단추를 선택합니다.
- 구독은 즉시 사용할 수 있습니다.

## <a name="3-authentication-key"></a>3: 인증 키

- [모든 리소스] 메뉴로 이동하고 구독을 클릭하여 인증 키를 검색합니다.
- [키] 값은 인증에 사용되며 기밀로 유지됩니다. 앱을 개발할 때 키 값이 필요합니다.
