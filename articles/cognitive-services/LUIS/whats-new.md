# <a name="whats-new-in-language-understanding"></a>Language Understanding의 새로운 기능

서비스의 새로운 기능에 대해 알아봅니다. 이러한 항목은 메모, 비디오, 블로그 게시물 및 기타 정보 유형을 릴리스할 수 있습니다. 서비스를 최신 상태로 유지 하려면이 페이지에 책갈피를 설정 합니다.
