분해 가능한 [개념을](../luis-concept-model.md#v3-authoring-model-decomposition)활용하려면 기계 학습 엔터티로 시작합니다. 다음과 같은 예제를 포함하여 이러한 유형의 엔터티를 만드는 방법에 대한 단계는 [기계 학습 엔터티 자습서를](../tutorial-machine-learned-entity.md) 따릅니다.

* **의도 세부 정보 페이지의** 예제 발언에서 구조체 및 설명자(구 목록)를 사용하여 기계 학습 엔터티 만들기 및 [레이블](../tutorial-machine-learned-entity.md#label-text-as-entities-in-example-utterances) 지정
* 엔터티 목록 페이지에서 [미리 빌드된 번호 엔터티를](../tutorial-machine-learned-entity.md#add-prebuilt-number-to-help-extract-data) **추가합니다.**
* 엔터티 세부 정보 페이지에서 [제약 조건으로 하위 구성 요소](../tutorial-machine-learned-entity.md#create-subcomponent-entity-with-constraint-to-help-extract-data) **엔터티를 추가합니다.**
* **의도 세부 정보 페이지에서** 하위 구성 요소가 있는 레이블 예제 [발화](../tutorial-machine-learned-entity.md#label-example-utterance-to-teach-luis-about-the-entity)
