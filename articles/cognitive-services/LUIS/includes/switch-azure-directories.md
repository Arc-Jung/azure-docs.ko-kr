사용자가 만들었거나 공유한 앱이 표시 되지 않는 경우 다른 Azure 디렉터리로 전환 해야 할 수 있습니다.

1. 화면의 오른쪽 위 모퉁이에 있는 아바타를 클릭 합니다. 그런 다음 **다른 Azure 디렉터리로 전환을** 클릭 합니다. 
2. 표시 되는 창에서 사용자와 공유 된 LUIS 리소스가 포함 된 Azure 디렉터리를 선택 해야 합니다.
