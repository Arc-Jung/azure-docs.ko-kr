다른 사용자가 앱을 작성해야 하는 경우 Azure 포털의 작성 리소스에 연결된 전자 메일 주소를 추가해야 합니다.

**Azure 구독의 소유자 또는 관리자인**경우 리소스에 기여기를 추가할 수 있습니다.

**Azure 구독의 소유자 또는 관리자가 아닌**경우 Azure 계정에 `Microsoft.Authorization/roleAssignments/write` 사용 권한이 있어야 합니다.

이 역할 할당에 문제가 있는 경우 다음을 검토합니다.

* [Azure 역할 할당](../../../role-based-access-control/role-assignments-portal.md)
* [Azure 액세스 제어 문제 해결](../../../role-based-access-control/troubleshooting.md#problems-with-azure-role-assignments)
