앱을 다른 사용자가 작성 해야 하는 경우 Azure Portal의 제작 리소스에 연결 된 전자 메일 주소를 추가 해야 합니다.

Azure 구독의 소유자 또는 관리자 인 경우 리소스에 참가자를 추가할 수 있습니다.

Azure 구독의 소유자 또는 관리자가 아닌 경우 Azure 계정에는 `Microsoft.Authorization/roleAssignments/write` 권한이 있어야 합니다.

이 역할 할당에 문제가 있는 경우 다음을 검토 하세요.
