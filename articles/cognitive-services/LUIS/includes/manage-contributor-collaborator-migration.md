공동 작업자의 기여 관리는 앱의 현재 상태에 따라 달라 집니다.

**[리소스 마이그레이션된 앱 제작](../luis-migration-authoring.md)** : _참가자_ 는 **액세스 제어 (IAM)** 페이지를 사용 하 여 제작 리소스에 대 한 Azure Portal에서 관리 됩니다. 협력자의 이메일 주소 및 _참가자_ 역할을 사용 하 여 사용자를 추가 합니다. 

**아직 마이그레이션하지 않은 앱의 경우**: 모든 _협력자_ 는 LUIS 포털의 각 앱 **관리-> 협력자** 페이지에서 관리 됩니다. 이 페이지에서 [사용자의 전자 메일 주소를 추가](../luis-how-to-collaborate.md)합니다. 이 응용 프로그램에 대 한 작성 액세스만 제공 합니다.
