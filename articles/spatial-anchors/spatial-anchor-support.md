# <a name="azure-spatial-anchors-support-options"></a>Azure 공간 앵커 지원 옵션

## <a name="open-a-tech-support-ticket"></a>기술 지원 티켓 열기

Azure 공간 앵커에 대 한 Azure Portal 내에서 기술 지원 티켓을 열려면

1. 사용 하 여 합니다 [Azure portal](https://azure.microsoft.com/account/) 열, 맨 위 메뉴 모음에서 도움말 아이콘을 선택한 다음 선택 합니다 **도움말 + 지원** 단추입니다. 

![Azure portal 도움말 및 지원](./media/spatial-anchor-support.png)

1. 도움말 + 지원 페이지를 열면 선택 **+ 새 지원 요청**.

![Azure portal 새 지원 요청](./media/spatial-anchor-support2.png)

1. 지원 티켓 필드 완료 한 경우: 

- 문제 유형: 기술적
- 서비스: Spatial Anchors

![Azure 포털 지원 티켓 필드](./media/spatial-anchor-support3.png)
