# <a name="azure-spatial-anchors-support-options"></a>Azure 공간 앵커 지원 옵션

## <a name="open-a-tech-support-ticket"></a>기술 지원 티켓 열기

Azure 공간 앵커에 대한 Azure 포털 내에서 기술 지원 티켓을 열려면 다음을 수행하십시오.

1. Azure [포털을](https://azure.microsoft.com/account/) 열려면 상단 메뉴 모음에서 도움말 아이콘을 선택한 다음 **도움말 + 지원** 단추를 선택합니다.

![Azure 포털 도움말 및 지원](./media/spatial-anchor-support.png)

1. 도움말 + 지원 페이지를 열려 면 **+ 새 지원 요청을**선택합니다.

![Azure 포털 새 지원 요청](./media/spatial-anchor-support2.png)

1. 지원 티켓 필드를 완료할 때:

- 문제 유형: 기술
- 서비스: 공간 앵커

![Azure 포털 지원 티켓 필드](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>커뮤니티 지원

사용자 커뮤니티의 지원은 [스택 오버플로의 Azure 공간 앵커를](https://stackoverflow.com/questions/tagged/azure-spatial-anchors)참조하십시오.

## <a name="provide-content-feedback"></a>콘텐츠 피드백 제공

각 콘텐츠 문서의 맨 아래에는 GitHub 문제를 열고 Azure 공간 앵커 설명서 콘텐츠에 대한 피드백을 제공할 수 있습니다.

## <a name="provide-product-feedback"></a>제품 사용자 의견 제공

피드백을 제공하거나 Azure 공간 앵커 서비스에 대한 아이디어 또는 제안을 공유하거나 다른 사용자가 제출한 아이디어에 투표하려면 [Azure 공간 앵커 피드백 포럼을](https://feedback.azure.com/forums/919252-azure-spatial-anchors)방문하십시오.
