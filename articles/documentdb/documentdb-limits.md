# <a name="default-quotas-for-azure-cosmos-db"></a>Azure Cosmos DB에 대한 기본 할당량

Azure Cosmos DB는 응용 프로그램의 요구에 따라 처리량과 저장소 크기를 조정하여 처리할 수 있는 뛰어난 확장성의 데이터베이스입니다. Azure Cosmos DB가 제공하는 규모에 대한 궁금한 사항은 askdocdb@microsoft.com에 메일을 보내 주세요.
