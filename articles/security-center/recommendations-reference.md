# <a name="security-recommendations---a-reference-guide"></a>보안 권장 사항 - 참조 가이드

이 문서에는 Azure 보안 센터에 표시될 수 있는 권장 사항이 나열되어 있습니다. 환경에 표시되는 권장 사항은 보호하는 리소스와 사용자 지정된 구성에 따라 달라집니다.

이러한 권장 사항에 응답하는 방법에 대한 자세한 내용은 [Azure 보안 센터의 권장 사항 해결을](security-center-remediate-recommendations.md)참조하십시오.

보안 점수는 완료한 보안 센터 권장 사항 수를 기반으로 합니다. 먼저 해결할 권장 사항을 결정하려면 각 권장 사항의 심각도와 보안 점수에 미치는 잠재적 영향을 확인합니다.

>[!TIP]
> 권장 사항의 설명에 "관련 정책이 없음"이라고 적혀 있는 경우 일반적으로 권장 사항이 다른 권장 사항 및 *정책에* 따라 달라지기 때문입니다. 예를 들어 "엔드포인트 보호 상태 오류를 해결해야 합니다..."라는 권장 사항은 엔드포인트 보호 솔루션이 *설치되어* 있는지 여부를 확인하는 권장 사항에 의존합니다("엔드포인트 보호 솔루션을 설치해야 합니다..."). 기본 *권장* 사항에는 정책이 있습니다. 정책을 기본 권장 사항으로만 제한하면 정책 관리가 간소화됩니다.
