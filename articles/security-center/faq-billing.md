# <a name="billing-questions"></a>대금 청구 관련 질문

## <a name="how-does-billing-work-for-azure-security-center"></a>Azure Security Center에 대한 청구는 어떻게 작동합니까?
Security Center는 Azure Defender를 사용 하거나 사용 하지 않고 제공 됩니다.

- **Azure Defender를 사용 하지 않으면**azure 리소스의 보안 상태, 기본 보안 정책, 보안 권장 사항 및 파트너의 보안 제품 및 서비스와의 통합에 대 한 가시성을 제공 Security Center.

- **Azure Defender** 는 보안 경고, 위협 인텔리전스, 동작 분석, 변칙 검색 및 위협 특성 보고서를 포함 하는 위협 방지 기능을 추가 합니다. Azure Defender의 30 일 무료 평가판을 시작할 수 있습니다. [업그레이드에 대해 자세히 알아보세요](security-center-pricing.md).

## <a name="how-can-i-track-who-in-my-organization-enabled-azure-defender-changes-in-azure-security-center"></a>내 조직에서 Azure Defender 변경 내용을 사용 하도록 설정한 사용자를 추적할 수 있는 방법 Azure Security Center
Azure 구독에는 가격 책정 설정을 변경할 수 있는 권한이 있는 여러 관리자가 있을 수 있습니다. 변경을 수행한 사용자를 확인 하려면 Azure 활동 로그를 사용 합니다. 

자세한 내용은 [여기](https://techcommunity.microsoft.com/t5/Security-Identity/Tracking-Changes-in-the-Pricing-Tier-for-Azure-Security-Center/td-p/390832)를 참조하세요.
