# <a name="billing-questions"></a>대금 청구 관련 질문

## <a name="how-does-billing-work-for-azure-security-center"></a>Azure Security Center에 대한 청구는 어떻게 작동합니까?
Security Center는 두 계층으로 제공됩니다.

- **프리 티어는** Azure 리소스의 보안 상태, 기본 보안 정책, 보안 권장 사항 및 파트너의 보안 제품 및 서비스와의 통합에 대한 가시성을 제공합니다.

- **표준 계층은** 보안 경고, 위협 인텔리전스, 행동 분석, 변칙 탐지 및 위협 기여 보고서를 포함하는 위협 보호 기능을 추가합니다. 표준 티어 무료 평가판을 시작할 수 있습니다. 업그레이드하려면 보안 정책에서 [가격 책정 계층을](https://docs.microsoft.com/azure/security-center/security-center-pricing) 선택합니다. 자세한 내용은 [가격 책정 페이지](https://azure.microsoft.com/pricing/details/security-center/)를 참조하세요.

## <a name="how-can-i-track-who-in-my-organization-performed-pricing-tier-changes-in-azure-security-center"></a>조직에서 Azure 보안 센터에서 가격 책정 계층 변경을 수행한 사람을 추적할 수 있는 방법은
Azure 구독에는 가격 책정 계층을 변경할 수 있는 권한이 있는 여러 관리자가 있을 수 있습니다. 가격 책정 계층 변경을 수행한 사용자를 확인하려면 Azure 활동 로그를 사용합니다. 자세한 내용은 [여기](https://techcommunity.microsoft.com/t5/Security-Identity/Tracking-Changes-in-the-Pricing-Tier-for-Azure-Security-Center/td-p/390832)를 참조하세요.
