# <a name="what-is-the-azure-education-hub"></a>Azure Education Hub란?

Microsoft Azure [Education Hub](https://portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/quickstart)는 교육 사용자가 다양한 Azure 구독에서 클라우드 크레딧을 프로비저닝하고 관리하는 데 도움이 되는 도구입니다. 이는 교직원 멤버가 많은 클라우드 기반 학생 프로젝트를 관리해야 하는 경우에 유용합니다. Education Hub는 Azure 크레딧 요구 사항이 아직 알려지지 않은 경우에도 연구 목적으로 유용합니다.

사용자는 각 구독 내에서 또는 대량으로 할당된 크레딧 금액을 쉽게 조정할 수 있습니다. 반기 또는 분기 프로젝트가 완료된 경우와 같이 구독을 일시 중단하여 사용 가능한 크레딧을 최대화할 수 있습니다.

## <a name="prerequisites"></a>필수 구성 요소

Education Hub에 액세스하려면 교육용 권한 부여에 대해 승인한 이메일 알림이 이미 수신되어 있어야 합니다. 이메일에는 승인된 크레딧 금액 및 [Education Hub](https://aka.ms/startedu)에 대한 링크가 포함되어 있습니다.

교육용 권한 부여를 사용하려면 Education Hub에서 클래스룸을 만들고 Azure 크레딧에 액세스할 수 있는 클래스룸 내의 구독을 사용해야 합니다.
