# <a name="managing-your-academic-grant"></a>교육용 권한 부여 관리

Azure Education Hub의 기본 방문 페이지는 개요 페이지입니다. 이 페이지에는 사용자가 설정한 클래스 수, 해당 클래스에서 할당 및 사용 된 총 실행 크레딧을 비롯 한 교육 기관 부여에 대 한 모든 관련 정보가 포함 되어 있습니다. 또한 소프트웨어, 학습 및 빠른 시작 템플릿 Education Hub에서 제공 하는 다른 혜택에 대 한 바로 가기도 표시 합니다.

## <a name="overview-page"></a>개요 페이지
:::image type="content" source="media/hub-overview-page/overview.png" alt-text="로그인 오류 메시지" border="false":::

1. **강좌** 에는 강사 계정에 대 한 총 활성 과정 수가 표시 됩니다.
1. **랩에서** 는 학생에 게 전달 된 활성 랩의 총 수를 보여 줍니다.
1. **필요한 작업** 은 랩 초대 수락과 같이 완료 해야 하는 모든 작업을 나열 합니다.
1. **소프트웨어** 는 교육자으로 다운로드할 수 있는 무료 소프트웨어를 표시 합니다.
1. Microsoft Learn를 통해 제공 되는 무료 Azure 학습 경로에 대 한 **학습** 링크입니다.
1. **빠른 시작 템플릿에** 는 일반적인 태스크에 대 한 배포 속도를 높이고 간소화 하는 Azure 템플릿이 포함 되어 있습니다.
