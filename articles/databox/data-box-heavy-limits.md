## <a name="data-box-heavy-limits"></a>데이터 박스 헤비 제한

- 데이터 박스 헤비는 노드당 최대 10억 개의 파일을 저장할 수 있습니다.
- 데이터 박스 헤비는 클라우드의 노드당 최대 512개의 컨테이너 또는 공유를 지원합니다. 사용자 공유 내의 최상위 디렉터리들은 클라우드에서 컨테이너 또는 Azure 파일 공유가 됩니다.
