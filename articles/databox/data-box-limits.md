## <a name="data-box-limits"></a>Data Box 제한

- Data Box는 최대 5억 개의 파일을 저장할 수 있습니다.
- Data Box는 클라우드에서 최대 512의 컨테이너 또는 공유를 지원 합니다. 사용자 공유 내의 최상위 디렉터리는 클라우드의 컨테이너 또는 Azure 파일 공유로 사용 됩니다.
