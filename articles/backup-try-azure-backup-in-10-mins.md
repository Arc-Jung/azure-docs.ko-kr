# 10mins에 Azure 백업을 시도합니다
Azure 백업 설치 프로그램을 매우 쉽게 하 고 다음 방법-하-비디오 시작 하는데 도움이 됩니다.

## 1 단계: Azure 구독 가져오기
+ Azure 구독이 없는 경우 모든 Azure 서비스에 액세스할 수 있게 하는 무료 평가판으로 시작할 얻을 수 있습니다. 다음 링크를 방문 [http://azure.microsoft.com/en-in/pricing/free-trial/](http://azure.microsoft.com/en-in/pricing/free-trial) 에 대해 알아보십시오.
+ Azure 구독을 보유 하는이 단계를 건너뛸 수 있습니다.

## 2 단계: Azure 백업 서비스를 구성 합니다.
비디오에서 설명한 단계를 통해 이동 [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure/) 하려면:

+	백업 자격 증명 모음 만들기

+	"보관 자격 증명" 다운로드

+	Azure 백업 에이전트 다운로드

> [AZURE.VIDEO getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure]

## 3 단계: 온-프레미스 컴퓨터 준비
비디오에서 설명한 단계를 통해 이동 [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup/) 하려면:

+ Windows Server 클라이언트 또는 DPM 서버에 Azure 백업 에이전트를 설치 합니다.

+ 백업 자격 증명 모음에 대해 컴퓨터를 등록 합니다.


> [AZURE.VIDEO getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup]

## 4 단계: 백업 및 복원 데이터
비디오에서 설명한 단계를 통해 이동 [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server/) 하려면:

+ 서버에서 백업 데이터

+ Azure에서 백업 된 데이터 복원

> [AZURE.VIDEO getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server]
