# <a name="log-analytics-view-designer-visualization-part-reference"></a>Log Analytics 뷰 디자이너 시각화 요소 참조
Log Analytics에서 뷰 디자이너를 사용하면 OMS 리포지토리에 있는 데이터의 서로 다른 시각화 요소를 포함하고 있는 OMS 콘솔에서 사용자 지정 보기를 만들 수 있습니다. 이 문서에서는 사용자 지정 보기에 사용할 수 있는 시각화 요소 각각의 설정에 대한 참조가 제공됩니다.

뷰 디자이너에 적용할 수 있는 다른 문서는 다음과 같습니다.

- [뷰 디자이너](log-analytics-view-designer.md) - 뷰 디자이너 개요 및 사용자 지정 보기를 만들고 편집하는 절차
- [타일 참조](log-analytics-view-designer-tiles.md) - 사용자 지정 보기에 사용할 수 있는 타일 각각의 설정에 대한 참조 

다음 표에서는 뷰 디자이너에서 사용할 수 있는 다양한 유형의 타일에 대해 설명합니다.  그 아래에 나오는 섹션에서는 각 타일 형식 및 해당 속성에 대해 자세히 설명합니다.

| 보기 유형 | 설명 |
|:--|:--|
| [쿼리 목록](#list-of-queries-part) | 로그 검색 쿼리 목록을 표시합니다.  각 쿼리를 클릭하면 결과를 표시할 수 있습니다.  보기에는 기본적으로 쿼리 하나가 포함되며, **+ 쿼리**를 클릭하면 쿼리를 추가할 수 있습니다. |
| [1개 숫자 및 목록](#number-amp-list-part) | 머리글에서 로그 검색 쿼리에서 레코드 수를 보여 주는 숫자 하나를 표시합니다.  목록에서 숫자 열의 상대 값을 나타내는 가로 막대와 함께 상위 10개의 쿼리 결과를 표시합니다. |
| [2개 숫자 및 목록](#two-numbers-amp-list-part) | 머리글에서 로그 검색 쿼리에서 레코드 수를 보여 주는 숫자 둘을 표시합니다.  목록에서 숫자 열의 상대 값을 나타내는 가로 막대와 함께 상위 10개의 쿼리 결과를 표시합니다. |
| [도넛형 차트 및 목록](#donut-amp-list-part) | 머리글에서 로그 쿼리의 값 열로부터 요약된 숫자 하나를 표시합니다.  도넛에서 상위 3개 레코드의 결과를 그래픽으로 표시합니다. |
| [2개 타임 라인 및 목록](#two-timelines-amp-list-part) | 머리글에서 두 로그 쿼리의 결과를 로그 쿼리의 값 열로부터 요약된 숫자 하나를 표시하는 설명선이 있는 세로 막대형 차트로 표시합니다.  목록에서 숫자 열의 상대 값을 나타내는 가로 막대와 함께 상위 10개의 쿼리 결과를 표시합니다. |   
| [정보](#information-part) | 머리글에서 정적 텍스트와 선택적 링크를 표시합니다.  목록에서 정적 텍스트 및 제목을 포함한 항목을 하나 이상 표시합니다. |
| [꺾은선형 차트, 설명선 및 목록](#line-chart-callout-amp-list-part) | 머리글에서 로그 쿼리의 여러 시계열 및 요약된 값이 있는 설명선을 포함한 꺾은선형 차트를 표시합니다.  목록에서 숫자 열의 상대 값을 나타내는 가로 막대와 함께 상위 10개의 쿼리 결과를 표시합니다. |
| [꺾은선형 차트 및 목록](#line-chart-amp-list-part) | 머리글에서 로그 쿼리의 여러 시계열이 있는 꺾은선형 차트를 표시합니다.  목록에서 숫자 열의 상대 값을 나타내는 가로 막대와 함께 상위 10개의 쿼리 결과를 표시합니다. |
| [꺾은선형 차트 및 스파크라인](#line-chart-amp-sparklines-part) | 머리글에서 로그 쿼리의 여러 시계열이 있는 꺾은선형 차트를 표시합니다.  목록에서 선택한 기간별 숫자 열의 값을 시각화한 스파크라인이 있는 상위 10개의 쿼리 결과를 표시합니다. |


## <a name="list-of-queries-part"></a>쿼리 목록 요소

로그 검색 쿼리 목록을 표시합니다.  각 쿼리를 클릭하면 결과를 표시할 수 있습니다.  

![쿼리 목록 보기](media/log-analytics-view-designer/view-list-queries.png)

| 설정 | 설명 |
|:--|:--|
| **일반** |
| 제목 | 보기 위쪽에 표시할 텍스트입니다. |
| 새 그룹 | 선택하는 경우 현재 보기에서 시작하는 보기에서 새 그룹을 만듭니다. |
| 미리 선택된 필터 | 사용자가 쿼리를 선택할 때 왼쪽 필터 창에 포함될 속성을 쉼표로 구분한 목록입니다. |
| 렌더링 모드 | 쿼리를 선택할 때 초기 보기가 표시되는 모드입니다.  쿼리를 연 후에는 사용 가능한 보기를 모두 선택할 수 있습니다. |
| **쿼리** |
| 검색 쿼리 | 실행할 쿼리입니다. |
| 친숙한 이름 | 사용자에게 표시할 설명이 포함된 쿼리 이름입니다. |


## <a name="number-&-list-part"></a>1개 숫자 및 목록 요소

머리글에서 로그 검색 쿼리에서 레코드 수를 보여 주는 숫자 하나를 표시합니다.  목록에서 숫자 열의 상대 값을 나타내는 가로 막대와 함께 상위 10개의 쿼리 결과를 표시합니다.

![쿼리 목록 보기](media/log-analytics-view-designer/view-number-list.png)

| 설정 | 설명 |
|:--|:--|
| **일반** |
| 그룹 제목 | 보기 위쪽에 표시할 텍스트입니다. |
| 새 그룹 | 선택하는 경우 현재 보기에서 시작하는 보기에서 새 그룹을 만듭니다. |
| 아이콘 | 머리글에서 결과 옆에 표시할 이미지 파일입니다.
| 아이콘 사용 | 선택하는 경우 아이콘을 표시합니다. |
| **제목** |
| 범례 | 머리글 위쪽에 표시할 텍스트입니다. |
| 쿼리 | 머리글을 위해 실행할 쿼리입니다.  쿼리에서 반환하는 레코드 수가 표시됩니다. |
| **목록** |
| 쿼리 | 목록을 위해 실행할 쿼리입니다.  결과에서 처음 10개 레코드에 대한 처음 2개의 속성이 표시됩니다.  첫 번째 속성은 텍스트 값, 두 번째 속성은 숫자 값이어야 합니다.  숫자 열의 상대 값에 따라 막대가 자동으로 생성됩니다.<br><br>쿼리에 정렬 명령을 사용하면 목록에서 레코드를 정렬합니다.  사용자가 모두 보기를 클릭하면 쿼리를 실행한 다음 모든 레코드를 반환할 수 있습니다. |
| 색 | 막대의 색입니다. |
| 이름 및 값 구분 기호 | 텍스트 속성을 여러 값으로 구문 분석할 경우에 사용하는 단일 문자 구분 기호입니다.  구분 기호를 지정하면 이름 상자에서 동일한 구분 기호로 구분되는 이름을 각 필드에 제공할 수 있습니다.<br><br>예를 들어 *Redmond-Building 41* 및 *Redmond-Building12*와 같은 값을 포함한 위치라는 속성이 있습니다.  이 경우 이름 및 값 구분 기호에는 -(하이픈), 이름에는 *City-Building*을 지정할 수 있습니다.  이렇게 하면 각 값을 *City*와 *Building*이라는 두 속성으로 구문 분석합니다. |
| 탐색 쿼리 | 목록에서 항목을 선택할 때 실행할 쿼리입니다.  {selected item}을 사용하면 선택한 항목의 구문을 포함할 수 있습니다. <br><br>예를 들어 쿼리에 Computer라는 열이 있고 탐색 쿼리가 {selected item}인 경우 컴퓨터를 선택하면 Computer=”MyComputer”와 같은 쿼리가 실행됩니다.  탐색 쿼리가 Type=Event {selected item}이면 Type=Event Computer=”MyComputer” 쿼리가 실행됩니다. |
| **목록** | **> 열 제목** |
| 이름 | 목록에서 첫 번째 열 위쪽에 표시할 텍스트입니다. |
| 값 | 목록에서 두 번째 열 위쪽에 표시할 텍스트입니다. |

## <a name="donut-&-list-part"></a>도넛형 차트 및 목록 요소

머리글에서 로그 쿼리의 값 열로부터 요약된 숫자 하나를 표시합니다.  도넛에서 상위 3개 레코드의 결과를 그래픽으로 표시합니다.

![도넛형 차트 및 목록 보기](media/log-analytics-view-designer/view-donut-list.png)

| 설정 | 설명 |

| **머리글** |
| 제목 | 머리글 위쪽에 표시할 텍스트입니다.
| 부제 | 머리글 위쪽에 있는 제목 밑에 표시할 텍스트입니다.

| 텍스트 | 도넛의 값 밑에 표시할 텍스트입니다. |

| 중앙에 사용되는 결과 값 연산 | 필요에 따라 +(더하기) 기호를 클릭하여 값을 하나 이상 추가합니다.  쿼리 결과는 지정한 속성 값을 포함하는 레코드로 제한됩니다.  값이 추가되지 않으면 쿼리에 모든 레코드가 포함됩니다. |

| 쿼리 | 목록을 위해 실행할 쿼리입니다.  쿼리에서 반환하는 레코드 수가 표시됩니다. |

| 이름 및 값 구분 기호 | 텍스트 속성을 여러 값으로 구문 분석할 경우에 사용하는 단일 문자 구분 기호입니다.  구분 기호를 지정하면 이름 상자에서 동일한 구분 기호로 구분되는 이름을 각 필드에 제공할 수 있습니다.<br><br>예를 들어 *Redmond-Building 41* 및 *Redmond-Building12*와 같은 값을 포함한 위치라는 속성이 있습니다.  이 경우 이름 및 값 구분 기호에는 -(하이픈), 이름에는 *City-Building*을 지정할 수 있습니다.  이렇게 하면 각 값을 *City*와 *Building*이라는 두 속성으로 구문 분석합니다. |

## <a name="line-chart-&-sparklines-part"></a>꺾은선형 차트 및 스파크라인 요소

머리글에서 로그 쿼리의 여러 시계열이 있는 꺾은선형 차트를 표시합니다.  목록에서 선택한 기간별 숫자 열의 값을 시각화한 스파크라인이 있는 상위 10개의 쿼리 결과를 표시합니다.

![꺾은선형 차트 및 스파크라인 보기](media/log-analytics-view-designer/view-line-chart-sparklines.png)
