# <a name="how-to-manage-your-azure-location-based-services-account-and-keys"></a>Azure Location Based Services 계정 및 키를 관리하는 방법

Azure Portal을 통해 Azure Location Based Services 계정 및 키를 관리할 수 있습니다. 계정과 키가 있으면 웹 사이트 또는 모바일 응용 프로그램에서 API를 구현할 수 있습니다.

Azure 구독이 아직 없는 경우 시작하기 전에 [무료 계정](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)을 만듭니다.

## <a name="log-in-to-azure"></a>Azure에 로그인 

Azure Portal( http://portal.azure.com )에 로그인합니다.

## <a name="create-a-new-account"></a>새 계정 만들기

1. Azure Portal의 왼쪽 위 모서리에서 **리소스 만들기**를 클릭합니다.

2. **Location Based Services**를 검색하여 선택한 다음 **만들기**를 클릭합니다.

3. Location Based Services 계정 정보를 입력합니다. 

![포털에서 계정 정보 입력](./media/how-to-manage-account-keys/new-account-portal.png)

## <a name="manage-keys-on-the-account-page"></a>계정 페이지에서 키 관리

왼쪽 메뉴의 설정에서 **키**를 클릭합니다.

![포털에서 계정 키 관리](./media/how-to-manage-account-keys/account-keys-portal.png)

키를 사용하여 Azure Location Based Services를 인증하고 위치 기반 서비스 개발을 시작합니다.

## <a name="delete-an-account"></a>계정 삭제

상단의 탐색 창에서 **삭제** 단추를 클릭하여 계정을 삭제할 수 있습니다.

![포털에서 계정 삭제](./media/how-to-manage-account-keys/account-delete-portal.png)
