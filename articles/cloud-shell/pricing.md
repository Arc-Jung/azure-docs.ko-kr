# <a name="pricing"></a>가격

## <a name="compute-cost"></a>비용 계산
Azure Cloud Shell은 추가 비용 없이 제공되는 컴퓨터에서 실행할 수 있지만 사용할 Azure 파일 공유가 연결되어야 합니다. 

## <a name="storage-cost"></a>저장소 비용
Cloud Shell은 파일 공유에서 기본 5GB 이미지를 만들어서 $HOME 디렉터리를 유지합니다. Azure 파일 공유는 일반적인 비용이 발생합니다.

[Azure 파일 비용에 대한 자세한 내용은 여기](https://azure.microsoft.com/pricing/details/storage/files/)를 확인하세요.
