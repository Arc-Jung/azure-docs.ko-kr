# <a name="security-attributes-for-azure-service-fabric"></a>Azure Service Fabric에 대한 보안 특성

보안은 Azure 서비스의 모든 측면에 통합됩니다. 이 문서에서는 Azure Service Fabric에 기본 제공되는 보안 특성에 대해 설명합니다. 보안 특성은 보안 취약성을 방지, 탐지 또는 대응할 수 있는 서비스의 기능에 기여하는 Azure 서비스의 품질 또는 기능입니다.

보안 특성은 다음과 같이 분류됩니다.
* 예방
* 네트워크 구분
* 감지
* ID 및 액세스 관리 지원
* 감사 내역
* 액세스 제어(사용하는 경우)
* 구성 관리(사용하는 경우)

각 범주에서 특성이 사용되는지 여부를 식별합니다(예/아니오). 일부 서비스의 경우 특성이 적용되지 않을 수 있으며 이 경우 N/A(해당 없음)로 표시됩니다. 특성과 관련된 자세한 정보에 대한 참고 사항 또는 링크가 제공될 수도 있습니다.
