# Microsoft Azure 인증 프로그램을 통해 사전 승인 받기

이 문서에서는 Microsoft Azure 인증을 받는 방법에 대한 지침을 제공합니다. Microsoft Azure 인증 프로그램 및 그 이점에 대한 자세한 개요는 [http://azure.com/certified](http://azure.com/certified)를 참조하세요.

> [AZURE.NOTE]Azure 인증은 Azure 마켓플레이스용 가상 컴퓨터, 솔루션 템플릿 및 개발자 서비스를 만들려는 경우에만 적용할 수 있습니다. 이러한 아티팩트를 만들지 않는 경우에는 이 단계를 건너뛰고 **1단계**, [Microsoft 판매자 계정: 생성 및 등록](marketplace-publishing-accounts-creation-registration.md)으로 이동하세요.

## 시작
1. 응용 프로그램이나 서비스를 인증하고 Azure 마켓플레이스에 등록하는 작업의 첫 번째 단계는 [Microsoft Azure Certified](https://azure.microsoft.com/marketplace/partner-program/) 웹 페이지에서 응용 프로그램 양식을 작성하는 것입니다.

2. 기본적인 자격 조건을 충족하는 경우 Microsoft는 비즈니스와 인증하려는 응용 프로그램 또는 서비스에 대한 추가 정보를 공유할 것을 소유자에게 요청합니다. Microsoft는 모든 정보를 받은 후 응용 프로그램을 검토하여 Azure 마켓플레이스에 적합한지 확인하고 응용 프로그램이 승인되는 경우 Azure 마켓플레이스에 솔루션을 등록하기 위해 소유자와 함께 작업을 시작합니다.

3. 파트너 계정 관리자나 DX 파트너 관리자와 함께 작업하는 경우 Azure 인증 프로그램에 추천해달라고 요청하세요. 또는 [Microsoft Azure Certified](http://azure.com/certified) 웹 페이지로 이동하여 **"정보 요청"**을 클릭하고 응용 프로그램 양식을 작성하면 됩니다.

**"판매" 국가**

> [AZURE.WARNING]Microsoft Azure 마켓플레이스에서 서비스를 판매하기 위해서는 등록된 법인이 승인된 “판매" 국가 중 하나에 속해 있어야 합니다. 이 제한은 지급 및 세금 때문에 있습니다. Microsoft는 가까운 장래에 이 국가 목록을 확장하기 위해 적극 노력하고 있으니 기대해 주세요. 전체 목록은 [http://go.microsoft.com/fwlink/?LinkID=526833](http://go.microsoft.com/fwlink/?LinkID=526833)에 있는 문서의 1b 섹션을 참조하세요.
