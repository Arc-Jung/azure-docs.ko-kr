# Azure 포털 바로 가기 키 

## 허브 열기

| 바로 가기 | 작업 |
|--------|----------|
| N | 알림 허브 열기 |
| S 또는 / | 허브 열기 찾아보기(서비스) | 
| A | 활성 세션 선택 | 
| C | 생성 | 

## 탐색

| 바로 가기 | 작업 |
|--------|----------|
| F | 첫 번째 열린 블레이드로 이동 | 
| H | Azure 포털 홈 및 현재 블레이드 사이 전환 | 
| J | 이전에 열린 블레이드로 이동 | 
| K | 다음 열린 블레이드로 이동 | 
| L | 마지막으로 열린 블레이드로 이동 | 
| - | 블레이드 최소화 또는 최대화에서 복원 | 
| + | 블레이드 최대화 또는 최소화에서 복원 | 
| K | 다음 열린 블레이드로 이동 | 


## 컨트롤
| 바로 가기 | 작업 |
|--------|----------|
| Esc | 허브 메뉴, 대화 상자, 메뉴 계정, 모든 열기 메뉴를 닫습니다. | 
| # | 열려있는 모든 블레이드 닫기 | 
| 홈 | 현재 블레이드 상단으로 스크롤 | 
| 끝 | 현재 블레이드 아래쪽으로 스크롤 | 
| CTRL+M | 컨트롤 간의 탭 이동 및 편집 컨트롤 내 탭 사용 전환 |
