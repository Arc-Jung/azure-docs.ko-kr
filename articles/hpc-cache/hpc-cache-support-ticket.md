# <a name="open-a-support-ticket-for-azure-hpc-cache"></a>Azure HPC 캐시에 대한 지원 티켓 열기

Azure 포털을 사용하여 지원 티켓을 엽니다. 캐시 인스턴스로 이동한 다음 사이드바 하단에 표시되는 **새 지원 요청** 링크를 클릭합니다.

활성 캐시가 없는 경우 티켓을 열려면 Azure 포털의 기본 도움말 + 지원 페이지를 사용합니다. 화면 왼쪽 상단의 컨트롤에서 포털 메뉴를 연 다음 아래쪽으로 스크롤하여 **도움말 + 지원을**클릭합니다.

**새 지원 요청을** 선택하고 Azure HPC 캐시와 관련된 도움말에 대한 **기술을** 선택합니다.

목록에서 구독을 선택합니다.

Azure HPC 캐시 서비스를 찾으려면 **모든 서비스** 단추를 클릭하고 HPC를 검색합니다.

![지원 요청의 스크린 샷 - 기본 탭, 설명된 대로 부분적으로 작성](media/hpc-cache-support-request.png)

나머지 필드에 정보 및 기본 설정을 기입한 다음 준비가 되면 티켓을 제출합니다.

요청을 제출하면 항공권 번호가 적어진 확인 이메일을 받게 됩니다. 지원 담당자가 요청에 대해 연락을 드릴 것입니다.
