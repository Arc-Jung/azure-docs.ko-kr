# <a name="connect-time-series-insights-to-power-bi"></a>타임시리즈 인사이트를 파워 BI에 연결  

타임시리즈 데이터의 상관 관계를 조정하고, 생생한 시각화를 만들고, 조직 전체에서 통찰력을 공유하는 데 도움이 되는 솔루션을 찾고 계십니까?

이제 Azure Time Series Insights는 [Power BI와](https://powerbi.microsoft.com/)원활하게 통합되어 스트리밍 데이터에 대한 보다 강력한 시각화 및 대시보드 기능을 제공하며 조직 전체에서 통찰력과 결과를 공유할 수 있습니다.

## <a name="summary"></a>요약

   [![열렬 인Insights에서 내보내기 옵션](./media/concepts-connect-power-bi/tsi-power-bi-export-example.png)](./media/concepts-connect-power-bi/tsi-power-bi-export-example.png#lightbox)

이제 데이터를 Power BI에 연결하여 다음을 수행할 수 있습니다.

* Power BI에서 지원하는 다른 데이터 원본과의 상관 관계를 수행하고 다양한 데이터 시각화 옵션에 액세스합니다. 
* 타임시리즈 인사이트 데이터를 사용하여 Power BI 대시보드 및 보고서를 만들고 조직과 공유합니다. 
* 간단하고 사용하기 쉬운 방식으로 데이터 인터옵 시나리오를 잠금 해제하고 그 어느 때보다 빠르게 통찰력을 얻을 수 있습니다.
* 강력한 고급 편집기사용을 사용하여 Power BI 내에서 타임시리즈 인사이트 데이터를 수정합니다.

## <a name="native-connector"></a>네이티브 커넥터

데이터를 연결하는 데 1분도 걸리지 않습니다. 타임시리즈 인사이트 탐색기에서 Power BI로 뷰를 빠르게 내보내면 조직의 모든 사용자가 직관적으로 데이터에 액세스하고 시각적으로 변환할 수 있습니다.

구성 옵션에는 **원시** 및 **집계 수준** 데이터 중에서 선택하고 원하는 시간 범위를 선택하는 것이 포함됩니다. 

데이터가 연결되면 Power BI의 모든 풍부한 시각화 및 차트 도구를 사용하여 정보 공유 기능을 더욱 강화할 수 있습니다. 

   [![Power BI의 차트 예제](./media/concepts-connect-power-bi/power-bi-tsi-example.png)](./media/concepts-connect-power-bi/power-bi-tsi-example.png#lightbox)

단계별 세부 사항을 [위해 타임시리즈 인사이트를 Power BI에 연결하는 방법을](./how-to-connect-power-bi.md) 읽어보십시오.

## <a name="flexible-workflow"></a>유연한 워크플로우
 
워크플로의 어느 곳에서나 커넥터를 사용합니다.

* 타임시리즈 인사이트 탐색기를 사용 중인 경우 뷰를 Power BI로 빠르게 내보낼 수 있습니다. 
* Power BI에 이미 있는 경우 Time Series Insights 쿼리 데이터를 사용하여 기존 시각화 시나리오를 보강합니다.

Power BI 내에서 고급 쿼리 및 편집 기능을 사용하는 경우 Power BI의 고급 편집기에서 사용합니다.
