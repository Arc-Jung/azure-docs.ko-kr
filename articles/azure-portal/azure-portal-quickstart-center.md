# <a name="get-started-with-the-azure-quickstart-center"></a>Azure 빠른 시작 센터 시작

Azure 빠른 시작 센터는 Azure에 대한 지식을 향상시키고자 하는 모든 사용자가 사용할 수 있는 Azure 포털의 안내환경입니다. Azure를 새로 접하는 조직의 경우 클라우드 환경을 온보팅하고 설정하는 가장 빠른 방법입니다.

## <a name="overview"></a>개요

Azure 빠른 시작 센터에는 시작하기 위한 세 가지 옵션이 있습니다.

* **설치 가이드**: IT 관리자 및 클라우드 설계자용으로 설계된 이 가이드에서는 Azure 채택을 위한 주요 개념을 소개합니다. 체계적인 단계를 통해 Microsoft의 권장 모범 사례를 적용하여 학습하는 동안 조치를 취할 수 있습니다. 마이그레이션 가이드는 Azure로 전환할 준비를 할 때 준비 상태를 평가하고 계획을 수립하는 데 도움이 됩니다.

* **프로젝트 시작**: 리소스를 만들 준비가 된 경우 이 섹션에서 서비스 옵션을 커밋하기 전에 선택한 작업에 대해 자세히 알아볼 수 있습니다. 서비스에 대해 자세히 알아보고 서비스를 사용해야 하는 이유를 알아보고 비용을 탐색하며 필수 구성 조건을 식별합니다. 선택을 한 후 직접 만들 수 있습니다.

* **온라인 학습**: Azure 빠른 시작 센터의 이 섹션에서는 Microsoft Learn의 무료 입문 과정 모듈을 강조합니다. 타일을 선택하여 강좌를 시작하고 Azure의 클라우드 개념 및 리소스 관리에 대해 자세히 알아봅니다.

## <a name="how-to-use-azure-quickstart-center"></a>Azure 빠른 시작 센터를 사용하는 방법

1. [Azure 포털에](https://portal.azure.com)로그인합니다.

1. Azure 포털 메뉴에서 **모든 서비스를** 선택합니다.

1. **일반** > **빠른 시작 센터를 선택합니다.**
