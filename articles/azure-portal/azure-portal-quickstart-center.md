# <a name="get-started-with-the-azure-quickstart-center"></a>Azure 빠른 시작 센터 시작

Azure 빠른 시작 센터는 Azure에 대 한 지식을 개선 하려는 모든 사용자가 사용할 수 있는 Azure Portal의 단계별 경험입니다. Azure를 처음 접하는 조직의 경우 클라우드 환경을 등록 하 고 설정 하는 가장 빠른 방법입니다.

## <a name="overview"></a>개요

Azure 빠른 시작 센터에는 시작 하기 위한 세 가지 옵션이 있습니다.

* **설정 가이드**: IT 관리자와 클라우드 설계자를 위해 설계 된 가이드에서는 Azure 채택을 위한 주요 개념을 소개 합니다. 구조화 된 단계는 Microsoft의 권장 모범 사례를 적용 하 여 학습할 때 조치를 취하는 데 도움이 됩니다. 마이그레이션 가이드는 Azure로 전환 준비를 할 때 준비 및 계획을 평가 하는 데 도움이 됩니다.

* **프로젝트를 시작 합니다**. 리소스를 만들 준비가 되 면이 섹션에서 서비스 옵션을 커밋하기 전에 선택 항목에 대해 자세히 알아볼 수 있습니다. 서비스 및 서비스를 사용 하 고, 비용을 탐색 하 고, 필수 구성 요소를 식별 하는 이유에 대해 자세히 알아볼 수 있습니다. 선택한 후 직접 만들기로 이동할 수 있습니다.

* **온라인 학습**: Azure 빠른 시작 센터의이 섹션에는 Microsoft Learn에서 제공 하는 무료 소개 과정 모듈이 강조 표시 되어 있습니다. 타일을 선택 하 여 과정을 시작 하 고 Azure에서 클라우드 개념 및 리소스 관리에 대해 자세히 알아보세요.

## <a name="how-to-use-azure-quickstart-center"></a>Azure 빠른 시작 센터를 사용 하는 방법

1. [Azure Portal](https://portal.azure.com)에 로그인합니다.
2. 세로 막대의 위쪽 섹션에서 **모든 서비스** 를 선택 합니다.
1. **일반** > **빠른 시작 센터**를 선택 합니다. 별 모양 아이콘을 선택 하 여 **즐겨찾기에**저장 합니다.
