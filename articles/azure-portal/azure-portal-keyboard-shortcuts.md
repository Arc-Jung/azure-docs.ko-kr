# <a name="azure-portal-keyboard-shortcuts"></a>Azure 포털 바로 가기 키
## <a name="open-hubs"></a>허브 열기
| 바로 가기 | 작업 |
| --- | --- |
| N |알림 |
| / |서비스 찾아보기 |
| 문자열(UTF-8 형식) 또는 |활성 세션 |
| B |결제 |
| C |생성 |

## <a name="navigation"></a>탐색
| 바로 가기 | 작업 |
| --- | --- |
| H |Azure 시작 대시보드 |
| F |첫 번째 열린 블레이드 |
| J |이전에 열린 블레이드 |
| K |다음 열린 블레이드 |
| L |마지막 열린 블레이드 |
| - |블레이드 최소화 또는 최대화에서 복원 |
| + |블레이드 최대화 또는 최소화에서 복원 |
| # |현재 블레이드 닫기 |
| ¦ |모든 블레이드 닫기 |
| . |포커스를 명령 모음으로 이동 |
| 에서도 확인할 수 있습니다. |탐색 모음으로 포커스 이동 |
| ? |바로 가기 키 보기 |
