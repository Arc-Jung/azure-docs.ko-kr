# <a name="add-remove-and-rearrange-favorites"></a>즐겨찾기 추가, 제거 및 재정렬

**즐겨찾기** 목록에서 항목을 추가하거나 제거하여 가장 자주 사용하는 서비스로 빠르게 이동합니다. 즐겨찾기 목록에 몇 가지 **Favorites** 일반적인 서비스가 이미 추가되었지만 사용자 지정할 수 있습니다. **즐겨찾기에**대한 변경 내용을 보는 유일한 사람입니다.

## <a name="add-a-favorite"></a>즐겨찾기 추가

**즐겨찾기** 아래에 나열된 항목은 **모든 서비스에서**선택됩니다. 서비스 이름 위로 마우스를 가져가면 서비스와 관련된 정보 및 리소스가 표시됩니다. 서비스 이름 ![옆에 채워진](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-graystar.png) 별 표별 아이콘은 항목이 **즐겨찾기** 목록에 나타나는 것을 나타냅니다. 별표 아이콘을 선택하여 즐겨찾기 목록에 서비스를 **추가합니다.**

### <a name="add-cost-management--billing-to-favorites"></a>즐겨찾기에 비용 관리 + 청구 추가

1. Azure 포털 메뉴에서 **모든 서비스를** 선택합니다.

    ![선택한 모든 서비스를 보여주는 스크린샷](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-new-all-services.png)

1. 검색 필드에 "비용"이라는 단어를 입력합니다. 제목에 "비용"이 있거나 키워드로 "비용"이 있는 서비스가 표시됩니다.

   ![모든 서비스에서 검색을 보여주는 스크린샷](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-find-service.png)

1. 서비스 이름 위로 마우스를 **가져가면 비용 관리 + 청구** 정보 카드가 표시됩니다. 별 아이콘을 선택합니다.

   ![비용 관리 + 청구 선택 옆의 별을 보여주는 스크린샷](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-add.png)

1. **이제 비용 관리 + 청구가** 즐겨찾기 목록의 마지막 항목으로 **추가됩니다.**

## <a name="remove-an-item-from-favorites"></a>즐겨찾기에서 항목 제거

이제 즐겨찾기 목록에서 직접 항목을 제거할 수 **있습니다.**

1. 포털 메뉴의 **즐겨찾기** 섹션에서 제거할 서비스의 이름 위로 마우스를 가져갑니다.

   ![즐겨찾기에서 마우스를 가져가는 동작을 보여주는 스크린샷](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-remove.png)

2. 정보 카드에서 별이 채워진 것에서 채워지지 않은 것으로 변경되도록 선택합니다. 서비스가 즐겨찾기 목록에서 **제거됩니다.**

## <a name="rearrange-favorites"></a>즐겨찾기 재정렬

즐겨찾는 서비스가 나열된 순서를 변경할 수 있습니다. 메뉴 항목을 **즐겨찾기**아래의 다른 위치로 드래그하여 놓기만 하면 됩니다.

### <a name="move-cost-management--billing-to-the-top-of-favorites"></a>비용 관리 + 즐겨찾기 상단으로 결제 이동

1. 즐겨찾기 목록에서 **비용 관리 + 청구** 항목을 선택하고 **보류합니다.**

   ![비용 관리 + 청구 선택을 보여주는 스크린샷](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-sort.png)

1. 계속 길게 누를 때 항목을 **즐겨찾기** 상단으로 드래그한 다음 놓습니다.
