# 소스 코드 찾아보기 및 편집

Microsoft Azure 포털 미리 보기에서는 Visual Studio Online 팀 프로젝트 소스 코드를 쉽게 찾아 편집할 수 있습니다.


포털에서 팀 프로젝트를 만들고 코드를 체크 인(Team Foundation 버전 제어) 또는 커밋(git)한 경우 포털을 통해 코드를 쉽게 액세스하고 찾아볼 수 있습니다.


1. 먼저 팀 프로젝트 블레이드를 열고 코드 렌즈를 찾습니다.  그런 후 소스 부분에서 찾을 저장소 이름을 클릭합니다(이 경우에는 'BrowseCode').  참고: 리포지토리가 3개 이상 있을 경우 리포지토리 번호(이 경우 1)를 클릭하여 전체 목록을 표시할 수 있습니다.
![Code Lens](./media/visual-studio-online-browse-edit-source-code/Code-Lens.png)
2. 리포지토리 블레이드가 열려 있으므로 코드 부분을 클릭하여 소스 코드 찾기를 시작할 수 있습니다.
![Repository Blade](./media/visual-studio-online-browse-edit-source-code/Repo-Blade.png)
3. 사용할 수 있는 전체 프로젝트를 포함하는 표준 트리 뷰 탐색 블레이드가 표시됩니다.  노드를 클릭하여 폴더를 열거나 특정 파일(이 경우 Index.cshtml)을 클릭하여 파일을 보거나 편집할 수 있습니다.
![Tree Navigation](./media/visual-studio-online-browse-edit-source-code/Tree-Nav.png)
4. 파일을 편집하려면 명령 모음에서 편집 단추를 클릭하기만 하면 됩니다.  파일 내용이 편집 모드로 바뀌고 변경할 수 있게 됩니다. 팁: 전체 화면 편집 환경을 표시하려면 오른쪽 위에 있는 최대화 단추를 클릭합니다.
![Edit Mode](./media/visual-studio-online-browse-edit-source-code/Edit-Mode.png)
5. 파일 편집이 끝나면 명령 모음에서 커밋(또는 체크인) 단추를 클릭합니다.  이렇게 하면 커밋 메시지를 입력할 수 있는 새 블레이드가 시작됩니다.  완료되면 확인을 클릭합니다. 그러면 파일이 리포지토리에 커밋됩니다.
![Commit Code](./media/visual-studio-online-browse-edit-source-code/Commit-Code.png)
6. 블레이드 맨 위에는 리포지토리로의 커밋이 성공적으로 수행되었음을 나타내는 플래시 메시지가 표시됩니다.
![Commit Success](./media/visual-studio-online-browse-edit-source-code/Commit-Success.png)
