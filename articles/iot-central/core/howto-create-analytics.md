# <a name="how-to-use-analytics-to-analyze-device-data"></a>분석을 사용하여 장치 데이터를 분석하는 방법

*이 문서는 운영자, 빌더 및 관리자에게 적용됩니다.*



Azure IoT Central은 기록 추세를 분석하고 장치에서 다양한 원격 분석을 상호 연관시키는 풍부한 분석 기능을 제공합니다. 시작하려면 왼쪽 창에 있는 **애널리틱스를** 방문하세요.

## <a name="understanding-the-analytics-ui"></a>분석 UI 이해
분석 사용자 인터페이스는 세 가지 주요 구성 요소로 구성됩니다.
- **데이터 구성 패널:** 구성 패널에서 먼저 데이터를 분석할 장치 그룹을 선택합니다. 그런 다음 분석할 원격 분석을 선택하고 각 원격 분석에 대한 집계 메서드를 선택합니다. **컨트롤별로 분할하면** 장치 속성을 차원으로 사용하여 데이터를 그룹화하는 데 도움이 됩니다.

- **시간 제어:** 시간 제어는 데이터를 분석할 기간을 선택하는 데 사용됩니다. 시간 슬라이더의 양쪽 끝을 드래그하여 시간 범위를 선택할 수 있습니다. 시간 컨트롤에는 데이터를 집계하는 데 사용되는 버킷 또는 간격 크기를 제어하는 간격 크기 **슬라이더도** 있습니다. 

- **차트 컨트롤:** 차트 컨트롤은 데이터를 표선 차트로 시각화합니다. 차트 범례와 상호 작용하여 특정 선의 가시성을 전환할 수 있습니다. 


  ![분석 UI 개요](media/howto-create-analytics/analyticsui.png)


## <a name="querying-your-data"></a>데이터 쿼리

먼저 **장치 그룹과**분석할 원격 분석을 선택해야 합니다. 작업이 완료되면 **분석옵션을** 선택하여 데이터 시각화를 시작합니다.

- **장치 그룹:** [장치 그룹은](tutorial-use-device-groups.md) 사용자의 사용자 정의 장치 그룹입니다. 예를 들어 오클랜드의 모든 냉장고 또는 모든 버전 2.0 풍력 터빈.

- **원격 분석:** 분석하고 탐색할 원격 분석을 선택합니다. 여러 원격 분석을 선택하여 함께 분석할 수 있습니다. 기본 집계 메서드는 각각 숫자및 문자열 데이터 형식에 대한 개수에 대해 평균으로 설정됩니다. 숫자 데이터 형식에 대해 지원되는 집계 방법은 평균, 최대, 최소, 개수 및 합계입니다.  문자열 데이터 형식에 대해 지원되는 집계 메서드는 개수입니다.

- **분할:** '분할 별' 컨트롤은 장치 속성을 차원으로 사용하여 데이터를 그룹화하는 데 도움이 됩니다. 장치 및 클라우드 속성의 값은 원격 분석과 함께 장치에서 전송될 때 조인됩니다. 클라우드 또는 장치 속성이 업데이트된 경우 차트에서 다른 값으로 그룹화된 원격 분석을 볼 수 있습니다.

    > [!TIP]
    > 각 장치에 대한 데이터를 별도로 보려면 '분할 바이' 컨트롤에서 장치 ID를 선택합니다.

## <a name="interacting-with-your-data"></a>데이터와의 상호 작용

데이터를 쿼리한 후에는 차트에서 데이터를 시각화할 수 있습니다. 원격 분석을 표시/숨기고, 시간 기간을 변경하고, 데이터 그리드에서 원격 분석을 볼 수 있습니다.

- **시간 편집기 패널:** 기본적으로 지난 1일의 데이터를 검색합니다. 시간 슬라이더의 양쪽 끝을 드래그하여 시간 지속 시간을 변경할 수 있습니다. 캘린더 컨트롤을 사용하여 미리 정의된 시간 버킷 중 하나를 선택하거나 사용자 지정 시간 범위를 선택할 수도 있습니다. 시간 컨트롤에는 데이터를 집계하는 데 사용되는 버킷 또는 간격 크기를 제어하는 간격 크기 **슬라이더도** 있습니다.

    ![시간 편집기](media/howto-create-analytics/timeeditorpanel.png)

    - **내부 날짜 범위 슬라이더 도구**: 원하는 시간 범위에 걸쳐 드래그하여 두 끝점 컨트롤을 사용합니다. 이 내부 날짜 범위는 외부 날짜 범위 슬라이더 컨트롤에 의해 제한됩니다.
    
   
    - **외부 날짜 범위 슬라이더 컨트롤**: 끝점 컨트롤을 사용하여 내부 날짜 범위 컨트롤에 사용할 수 있는 외부 날짜 범위를 선택합니다.

    - **날짜 범위 버튼 증가 및 감소**: 원하는 간격에 대해 어느 버튼을 선택하여 시간 범위를 늘리거나 줄입니다.

    - **간격 크기 슬라이더**: 같은 시간 범위에 걸쳐 간격을 확대 및 축소하는 데 사용합니다. 이 작업을 통해 큰 시간 조각 간의 이동을 보다 정확하게 제어할 수 있습니다. 이를 사용하여 데이터의 세분화된 고해상도 뷰를 밀리초까지 볼 수 있습니다. 슬라이더의 기본 시작점은 해상도, 쿼리 속도 및 세분성의 균형을 맞추는 선택 항목의 데이터를 가장 최적의 뷰로 설정합니다.
    
    - **날짜 범위 선택기**:이 웹 컨트롤을 사용하면 원하는 날짜와 시간 범위를 쉽게 선택할 수 있습니다. 컨트롤을 사용하여 표준 시간대 간에 전환할 수도 있습니다. 현재 작업 영역에 적용할 변경 작업을 수행한 후 저장을 선택합니다.

    > [!TIP]
    > 간격 크기는 선택한 시간 범위에 따라 동적으로 결정됩니다. 시간 범위가 작을수록 데이터를 최대 몇 초의 매우 세분화된 간격으로 집계할 수 있습니다.


- **차트 범례:** 차트 범례는 차트에서 선택한 원격 분석을 보여 주었습니다. 범례의 각 항목 위로 마우스를 가져가서 차트에 초점을 맞출 수 있습니다. '분할 기준'을 사용하는 경우 원격 분석은 선택한 차원의 각 값으로 그룹화됩니다. 그룹 이름을 클릭하여 각 특정 원격 분석 또는 전체 그룹의 가시성을 전환할 수 있습니다.  


- **Y축 형식 제어:** y축 모드는 사용 가능한 y축 뷰 옵션을 순환합니다. 이 컨트롤은 다른 원격 분석을 시각화하는 경우에만 사용할 수 있습니다. 다음 세 가지 모드 중 하나를 선택하여 y축을 설정할 수 있습니다.

    - **누적:** 모든 원격 분석에 대한 그래프가 누적되고 각 그래프에는 고유한 y축이 있습니다. 이 모드는 기본값으로 설정됩니다.
    - **공유:** 모든 원격 분석에 대한 그래프는 동일한 y축에 대해 플롯됩니다.
    - **겹침:** 선택한 선에 따라 y축 데이터가 변경되는 등 동일한 y축에 여러 선을 스택하는 데 사용합니다.

  ![다양한 시각화 모드를 사용하여 y-축을 따라 데이터 정렬](media/howto-create-analytics/yaxiscontrol.png)

- **확대/축소 제어:** 줌을 사용하면 데이터를 더 자세히 드릴다운할 수 있습니다. 결과 집합 내에서 포커스를 맞추려는 기간을 찾으면 마우스 포인터를 사용하여 영역을 잡은 다음 선택한 끝점으로 드래그합니다. 그런 다음 선택한 영역을 마우스 오른쪽 버튼으로 클릭하고 확대/축소를 클릭합니다.

  ![데이터 확대](media/howto-create-analytics/zoom.png)

타원 아래에는 데이터와 상호 작용하는 차트 컨트롤이 더 많습니다.

- **표 표시:** 결과를 테이블 형식으로 사용할 수 있으므로 각 데이터 포인트에 대한 특정 값을 볼 수 있습니다.

- **마커 삭제:** '드롭 마커' 컨트롤은 차트의 특정 데이터 포인트를 고정하는 방법을 제공합니다. 여러 기간에 걸쳐 여러 줄에 대한 데이터를 비교하려고 할 때 유용합니다.

  ![분석에 대한 그리드 보기 표시](media/howto-create-analytics/additionalchartcontrols.png)
