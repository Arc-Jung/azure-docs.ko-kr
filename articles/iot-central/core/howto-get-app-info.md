# <a name="about-your-application"></a>애플리케이션 정보

이 문서에서는 IoT Central 응용 프로그램에 대 한 정보를 가져오는 방법을 보여 줍니다. 다음이 필요할 수 있습니다.

- 지원 서비스에 문의 하는 경우이 정보를 제공 합니다.
- Azure Portal에서 청구 정보를 찾기 위해 응용 프로그램에서 사용 하는 Azure 구독입니다.
- REST API 작업 하는 경우 응용 프로그램의 ID입니다.
- 커넥터 추가와 같은 작업을 완료 하는 응용 프로그램의 버전입니다.

## <a name="get-information-about-your-application"></a>애플리케이션에 대한 정보 가져오기

IoT Central 응용 프로그램에 대 한 정보를 얻으려면:

1. 상단 메뉴에서 **도움말** 링크를 선택 합니다.

1. **앱 정보** 를 선택 합니다.

1. **앱** 정보 페이지에는 응용 프로그램에 대 한 정보가 표시 됩니다.  

    :::image type="content" source="media/howto-get-app-info/about-your-app.png" alt-text="앱 스크린샷 정보":::

    정보 **복사** 단추를 사용 하 여 정보를 클립보드에 복사 합니다.
