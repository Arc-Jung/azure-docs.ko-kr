# <a name="what-are-application-templates"></a>애플리케이션 템플릿이란?

Azure IoT Central의 응용 프로그램 템플릿은 솔루션 빌더가 IoT 솔루션 개발을 시작하는 데 도움이 되는 도구입니다. 앱 템플릿을 사용하여 가능한 것에 대한 느낌을 얻는 것부터 완전히 사용자 지정하고 고객에게 재판매할 응용 프로그램에 이르기까지 모든 것을 사용할 수 있습니다.

응용 프로그램 템플릿은 다음과 구성됩니다.

- 샘플 작업자 대시보드
- 샘플 장치 템플릿
- 실시간 데이터를 생성하는 시뮬레이션 된 장치
- 미리 구성된 규칙 및 작업
- 자습서 및 방법-토를 포함 하 여 풍부한 문서

응용 프로그램을 만들 때 응용 프로그램 템플릿을 선택합니다. 응용 프로그램을 만든 후에는 템플릿을 변경할 수 없습니다.

## <a name="custom-templates"></a>사용자 지정 템플릿

응용 프로그램을 처음부터 만들려면 두 가지 사용자 지정 응용 프로그램 템플릿 중 하나를 선택합니다.

- 사용자 지정 애플리케이션
- 사용자 지정 응용 프로그램(레거시)

레거시 템플릿을 사용해야 하는 특별한 이유가 없는 한 **사용자 지정 응용 프로그램** 템플릿을 선택합니다.

## <a name="industry-focused-templates"></a>업계 중심템플릿

Azure IoT Central은 업계에 구애받지 않는 응용 프로그램 플랫폼입니다. 응용 프로그램 템플릿은 현재 이러한 업계에서 사용할 수 있는 업계 중심의 예제이며, 향후 더 많은 것들이 출시될 것입니다.

- [소매](../retail/overview-iot-central-retail-pnp.md)
  - 연결된 물류
  - 디지털 배포 센터
  - 스토어 내 분석 - 조건 모니터링
  - 스토어 내 분석 - 체크 아웃
  - 스마트 재고 관리
- [에너지](../energy/overview-iot-central-energy.md)
  - 스마트 계량기 모니터링
  - 태양전지 패널 모니터링
- [정부](../government/overview-iot-central-government.md)
  - 커넥티드 폐기물 관리
  - 물 소비 모니터링
  - 수질 모니터링
- [헬스케어](../healthcare/overview-iot-central-healthcare.md).
  - 지속적인 환자 모니터링
