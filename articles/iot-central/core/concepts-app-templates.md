# <a name="what-are-application-templates"></a>응용 프로그램 템플릿은 무엇 인가요?

Azure IoT Central의 응용 프로그램 템플릿은 솔루션 빌더가 IoT 솔루션 개발을 kickstart 하는 데 도움이 되는 도구입니다. 응용 프로그램을 고객에 게 재판매 하기 위해 응용 프로그램을 완벽 하 게 사용자 지정 하 고 레이블을 지정 하기 위해 가능한 작업에 대 한 느낌을 얻기 위해 앱 템플릿을 사용할 수 있습니다.

응용 프로그램 템플릿은 다음으로 구성 됩니다.

- Sample operator 대시보드
- 샘플 장치 템플릿
- 실시간 데이터를 생성 하는 시뮬레이션 된 장치
- 미리 구성 된 규칙 및 작업
- 자습서 및 방법을 비롯 한 다양 한 설명서

## <a name="industry-focused-app-templates"></a>업계 집중 앱 템플릿

Azure IoT Central은 업계와 무관 한 응용 프로그램 플랫폼입니다. 응용 프로그램 템플릿은 현재 이러한 업계에서 사용할 수 있는 업계 중심의 예제 이며 향후 제공 될 것입니다.

- [소매](../retail/overview-iot-central-retail-pnp.md)
  - 연결된 물류
  - 디지털 배포 센터
  - 저장소 내 분석-조건 모니터링
  - 저장소 내 분석-체크 아웃
  - 스마트 재고 관리
- [에너지](../energy/overview-iot-central-energy.md)
  - 스마트 측정기 모니터링
  - 태양 창 모니터링
- [자치](../government/overview-iot-central-government.md)
  - 커넥티드 폐기물 관리
  - 용수 사용량 모니터링
  - 용수 품질 모니터링
- [의료](../healthcare/overview-iot-central-healthcare.md).
  - 지속적인 환자 모니터링
