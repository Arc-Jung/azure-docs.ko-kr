# <a name="how-to-use-tiles"></a>타일을 사용 하는 방법
타일을 사용 하 여 응용 프로그램 대시보드, 장치 대시보드를 사용자 지정 하 고 장치 대시보드를 설정 합니다. 여러 타일을 대시보드에 추가할 수 있으며 응용 프로그램 관련 정보를 표시 하려면 해당 타일을 사용자 지정할 수 있습니다. 타일 크기를 조정 하 고 어떤 대시보드에서 든 레이아웃을 사용자 지정할 수도 있습니다. 아래 스크린샷은 여러 타일을 사용 하 여 응용 프로그램 대시보드를 보여 줍니다.

![애플리케이션 대시보드](media/howto-use-tiles/image1a.png)


다음 표에서 Azure IoT Central의 타일의 사용을 보여 줍니다.

 
| 타일 | 대시보드 | 설명
| ----------- | ------- | ------- |
| 연결 | 응용 프로그램 및 장치 대시보드 설정 |링크 타일은 제목 및 설명 텍스트를 표시 하는 클릭할 수 있는 타일입니다. 사용자가 응용 프로그램에 관련 된 URL로 이동할 수 있도록 하려면 링크 타일을 사용 합니다. |
| 이미지 | 응용 프로그램 및 장치 대시보드 설정 |이미지 타일 사용자 지정 이미지를 표시 하 고 클릭할 수 있습니다. 이미지 타일을 대시보드에 그래픽을 추가 하 고 필요에 따라 사용자가 응용 프로그램에 관련 된 URL로 이동할 수 있도록 하려면 사용 합니다.|
| 레이블 | 응용 프로그램 대시보드 |레이블 타일을 대시보드에 사용자 지정 텍스트를 표시합니다. 텍스트의 크기를 선택할 수 있습니다. 이러한 설명, 연락처 세부 정보 또는 도움말 관련 정보를 대시보드에 추가 하려면 레이블 타일 사용|
| 지도 | 응용 프로그램 및 장치 대시보드 설정 |지도 타일 지도에 위치 및 장치 상태를 표시 합니다. 예를 들어 여기서는 장치 및 팬 켜져 있는지 여부를 표시할 수 있습니다.|
| 꺾은선형 차트 | 응용 프로그램 및 장치 대시보드 |꺾은선형 차트 타일의 시간 동안 장치에 대 한 집계 측정 차트를 표시 합니다. 예를 들어, 지난 1 시간 동안 장치의 평균 온도 압력을 보여 주는 꺾은선형 차트를 표시할 수 있습니다.|
| 가로 막대형 차트 | 응용 프로그램 및 장치 대시보드 |가로 막대형 차트 타일 기간에 대 한 장치에 대 한 집계 측정값을 차트로 표시합니다. 예를 들어, 지난 1 시간 동안 장치의 평균 온도 압력을 보여 주는 가로 막대형 차트를 표시할 수 있습니다. |
| 이벤트 기록 | 응용 프로그램 및 장치 대시보드 |이벤트 기록 타일 기간 동안 장치에 대 한 이벤트를 표시합니다. 예를 들어, 지난 1 시간 동안 장치에 대 한 온도 변화를 모든 발생 표시를 사용할 수 있습니다. |
| 상태 기록 | 응용 프로그램 및 장치 대시보드 |상태 기록 타일 시간 동안 측정 값을 표시합니다. 예를 들어, 지난 1 시간 동안 장치에 대 한 온도 값을 표시 하려면 사용할 수 있습니다.|
| KPI | 응용 프로그램 및 장치 대시보드 | KPI 타일 시간 동안 측정 하는 집계 원격 분석 또는 이벤트를 표시합니다. 예를 들어, 지난 1 시간 동안 장치에 도달 하면 최대 온도 표시 하도록 사용할 수 있습니다.|
| 마지막으로 알려진된 값 | 응용 프로그램 및 장치 대시보드 |마지막으로 알려진된 값 타일에 원격 분석 또는 상태 측정에 대 한 최신 값을 표시합니다. 예를 들어, 온도, 압력 및 습도 장치에 대 한 중 가장 최근의 측정값을 표시 하려면이 타일을 사용할 수 있습니다.|
| 장치 설정 표 | 응용 프로그램 및 장치 대시보드 설정 | 장치 설정 표 장치 집합에 대 한 정보를 표시 합니다. 장치 집합의 이름, 위치 및 모든 장치 모델 등의 정보를 표시할 장치 설정 표 형태 타일을 사용 합니다.|


Azure IoT Central 응용 프로그램에서 대시보드를 구성 하는 방법에 대 한 자세한 내용은 참조 하세요 [구성 응용 프로그램 대시보드](howto-configure-homepage.md)합니다.
