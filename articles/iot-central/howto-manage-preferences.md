# <a name="manage-your-application-preferences"></a>애플리케이션 기본 설정 관리

*이 문서는 운영자, 빌더 및 관리자에게 적용됩니다.*

IoT Central은 애플리케이션을 사용자 요구에 맞게 유연하게 사용자 지정할 수 있도록 합니다. 또한 사용자별로 고유한 보기를 사용자 지정할 수 있도록 합니다. 이 문서에서는 사용자가 자신의 프로필에 적용할 수 있는 다양한 사용자 지정 옵션에 대해 설명합니다.

## <a name="changing-language"></a>언어 변경

IoT Central은 17개 언어로 지원됩니다. 위쪽 탐색 모음에서 **언어 선택**을 사용하여 기본 설정 언어를 전환할 수 있습니다. 언어를 변경한 후에는 선택한 언어를 기억했다가 모든 애플리케이션에 적용해야 합니다. 대시보드 이미지와 같은 애플리케이션 내 사용자 지정은 지역화되지 않습니다. 

![IoT Central 언어 선택](media/howto-manage-profile/language-picker.png)

## <a name="changing-theme"></a>테마 변경

밝은 테마와 어두운 테마가 모두 지원됩니다. 어두운 테마가 기본 테마이지만 오른쪽 위에서 이름을 클릭하고 테마를 전환하여 변경할 수 있습니다.

![IoT Central 테마 선택](media/howto-manage-profile/theme-picker.png)
