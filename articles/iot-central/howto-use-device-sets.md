# <a name="use-device-sets-in-your-azure-iot-central-application"></a>Azure IoT Central 애플리케이션에 장치 집합 사용

이 문서에서는 운영자로서 Azure IoT Central 애플리케이션에서 디바이스 집합을 사용하는 방법을 설명합니다.

장치 집합에는 일부 지정 된 조건과 일치 하는 때문에 함께 그룹화 되는 장치의 목록입니다. 디바이스 집합은 디바이스를 더 작은 논리 그룹으로 그룹화하여 디바이스를 대규모로 관리하고 시각화하고 분석하는 데 도움을 줍니다. 예를 들어 시애틀 담당 하는 장치를 찾을 때 엔지니어가 사용 하도록 설정 하려면 모든 공조 장치 목록으로 설정 하는 장치를 만들면 됩니다. 이 문서에서는 디바이스 집합을 만들고 구성하는 방법을 보여줍니다.

## <a name="create-a-device-set"></a>디바이스 집합 만들기

디바이스 집합을 만들려면:

1. 왼쪽 탐색 메뉴에서 **디바이스 집합**을 선택합니다.

1. 선택 **+ 새로 만들기**합니다.

    ![새 디바이스 집합](media/howto-use-device-sets/image1.png)

1. 애플리케이션 전체에서 고유한 장치 집합 이름을 지정합니다. 설명을 추가할 수도 있습니다. 한 디바이스 집합에는 단일 디바이스 템플릿의 디바이스만 포함될 수 있습니다. 이 디바이스 집합에 사용할 디바이스 템플릿을 선택합니다.

1. 속성, 비교 연산자 및 값을 선택하여 디바이스 집합의 디바이스를 식별하는 쿼리를 만듭니다. 여러 쿼리를 추가할 수 있으며 **모든** 조건을 충족하는 디바이스는 디바이스 집합에 배치됩니다. 애플리케이션에 대한 액세스 권한을 가진 사람이라면 누구든지 생성된 장치 집합에 액세스할 수 있으며, 따라서 누구든지 장치 집합을 보고, 수정하고, 삭제할 수 있습니다.

    ![디바이스 집합 쿼리](media/howto-use-device-sets/image2.png)

    > [!NOTE]
    > 디바이스 집합은 동적 쿼리입니다. 디바이스 목록을 볼 때마다 목록의 디바이스가 달라질 수 있습니다. 현재 쿼리 조건을 충족하는 디바이스에 따라 목록이 달라집니다.

1. **저장**을 선택합니다.

## <a name="configure-the-dashboard-for-your-device-set"></a>장치 집합에 대 한 대시보드를 구성 합니다.

디바이스 집합을 만든 후에는 **대시보드**를 구성할 수 있습니다. 합니다 **대시보드** 의 홈페이지입니다. 이미지 및 링크를 배치 하는 위치입니다. 디바이스 집합에 디바이스를 나열하는 그리드를 추가할 수도 있습니다.

1. 왼쪽 탐색 메뉴에서 **디바이스 집합**을 선택합니다.

1. 디바이스 세트를 선택합니다.

1. **대시보드** 탭을 선택합니다.

1. **편집**을 선택합니다.

    ![디자인 모드 켜기](media/howto-use-device-sets/image3.png)

1. 이미지 추가에 대한 내용은 [이미지를 준비하여 Azure IoT Central 애플리케이션에 업로드](howto-prepare-images.md)를 참조하세요.

1. 링크 타일 추가:
    1. 오른쪽 창에서 **링크**를 선택합니다.
    1. 링크 **제목**을 지정합니다.
    1. 링크를 선택 하는 경우 열 수에 대 한 URL을 선택 합니다.
    1. **제목** 아래에 표시될 링크 설명을 입력합니다.
    1. **저장**을 선택합니다.

        ![링크 저장](media/howto-use-device-sets/image7.png)

    1. **대시보드**에서 링크 타일을 이동하고 크기를 조정할 수 있습니다.

1. 그리드를 추가합니다. 그리드는 사용자가 선택하는 열이 포함된 디바이스 집합의 디바이스 표입니다.
    1. 오른쪽 창에서 **그리드**를 선택합니다.
    1. 그리드 **제목**을 지정합니다.
    1. **추가/제거**를 선택하여 표시할 열을 선택합니다. 나타나는 패널에서 표시할 열을 선택하고 오른쪽 화살표를 클릭하여 해당 열을 선택합니다.
    1. **확인**을 선택합니다.
    1. **저장**을 선택합니다.

        ![그리드 저장](media/howto-use-device-sets/image9.png)

    1. **대시보드**에 그리드를 끌어서 놓아 배치합니다.

        > [!NOTE]
        > 여러 이미지, 링크 및 그리드를 추가할 수 있습니다.
  
    1. **완료**를 선택합니다.

Azure IoT Central의 타일을 사용 하는 방법에 대 한 자세한 내용은 참조 하세요 [대시보드 타일을 사용 하 여](howto-use-tiles.md)입니다.

### <a name="configure-a-location-map-in-your-device-sets-dashboard"></a>장치 설정 대시보드에서 위치 맵 구성

장치 집합에서 장치의 위치를 시각화 하도록 지도 추가할 수 있습니다.

대시보드를 설정 하는 장치에 지도 추가 하려면 해야 위치 측정 또는 위치 속성에서에서 구성한 장치 템플릿을 합니다. 자세한 내용은 참조 하세요 [위치 측정을 만듭니다](howto-set-up-template.md) 또는 [위치 속성을 만들](howto-set-up-template.md)합니다.

1. 장치 설정 **대시보드**를 선택 **지도** 라이브러리에서.
2. 제목을 추가 하 고 해당 위치 측정 또는 이전에 구성한 속성을 선택 합니다.
3. 선택 **저장할** 지도 타일 장치 집합에서 장치의 마지막 알려진된 위치를 표시 합니다.
4. 운영자는 장치 설정 대시보드를 볼 때 연산자에 구성한 경우 위치 지도 포함 하 여 모든 타일을 확인 합니다.

대시보드에서 지도 타일 크기를 조정할 수 있습니다. 지도에 pin을 선택 하는 장치 정보, 이름 및 위치를 표시 합니다. 장치 속성 페이지로 이동 하려면 팝업을 선택 합니다.

## <a name="configure-the-list-for-your-device-set"></a>디바이스 집합에 대한 목록 구성

디바이스 집합을 만든 후에는 **목록**을 구성할 수 있습니다. **목록**에는 사용자가 선택하는 열이 포함된 디바이스 집합의 모든 디바이스가 나열됩니다.

1. 왼쪽 탐색 메뉴에서 **디바이스 집합**을 선택합니다.

1. **목록** 탭을 선택합니다.

1. **열 옵션**을 선택합니다.

    ![열 옵션](media/howto-use-device-sets/image11.png)

1. 표시할 열을 선택하고 오른쪽 화살표를 클릭한 후 해당 열을 선택하여 표시할 열을 선택합니다.

    ![열 선택](media/howto-use-device-sets/image12.png)

1. **확인**을 선택합니다.
