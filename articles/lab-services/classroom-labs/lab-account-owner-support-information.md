# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>지원 정보 설정 (Azure Lab Services 랩 계정 소유자)
이 문서에서는 랩 계정 소유자로 랩 작성자 (강사) 및 랩 사용자 (학생)가 랩 계정 또는 랩을 사용 하 여 기술적인 문제가 발생 하는 경우 도움을 얻기 위해 사용할 수 있는 지원 정보를 지정 하는 방법을 설명 합니다. 

지원 정보에는 다음이 포함 됩니다.

- URL
- 메일
- Phone
- 추가 지침 

## <a name="specify-support-information"></a>지원 정보 지정
1. [Azure 포털](https://portal.azure.com)에 로그인합니다.
2. 검색 창에서 **Lab services**를 입력 하 고 검색 결과에서 **lab services** 를 선택 합니다. 
3. 랩 계정 목록에서 랩 계정을 선택 합니다. 
4. **내부 지원** 페이지로 전환 하 여 다음 단계를 수행 합니다.
    1. **지원 URL**을 입력 합니다. 
     2. **지원 전자 메일**을 입력 합니다. 
     3. **지원 전화 번호**를 입력 합니다.
     4. 자세한 **지원 지침** (선택 사항)을 입력 합니다. 랩 소유자 및 사용자는 지원 연락처 정보와 함께이 텍스트를 볼 수 있습니다. Url은 자동으로 링크로 설정 됩니다. 
     5. 도구 모음에서 **저장**을 선택합니다.

         ![내부 지원 페이지](../media/lab-account-owner-support-information/internal-support-page.png)
