# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>지원 정보 설정(Azure 랩 서비스의 랩 계정 소유자)
이 문서에서는 랩 계정 소유자로서 랩 작성자(교수자) 및 랩 사용자(학생)가 랩 계정 또는 랩 사용과 관련된 기술적 문제가 발생하는 경우 도움을 받는 데 사용할 수 있는 지원 정보를 지정하는 방법을 설명합니다. 

지원 정보에는 다음이 포함됩니다.

- URL
- Email
- Phone
- 추가 지침 

## <a name="specify-support-information"></a>지원 정보 지정
1. [Azure 포털에](https://portal.azure.com)로그인합니다.
2. 검색 표시줄에서 **랩 서비스를**입력하고 검색 결과에서 **랩 서비스를** 선택합니다. 
3. 랩 계정 목록에서 랩 계정을 선택합니다. 
4. **내부 지원** 페이지로 전환하고 다음 단계를 수행합니다.
    1. 지원 **URL을**입력합니다. 
     2. 지원 **이메일을**입력합니다. 
     3. 지원 **전화를**입력합니다.
     4. 자세한 **지원 지침을 입력합니다(선택** 사항). 랩 소유자와 사용자는 지원 연락처 정보와 함께 이 텍스트를 볼 수 있습니다. URL이 자동으로 링크로 바뀌게 됩니다. 
     5. 도구 모음에서 **저장**을 선택합니다.

         ![내부 지원 페이지](../media/lab-account-owner-support-information/internal-support-page.png)
