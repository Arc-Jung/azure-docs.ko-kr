# <a name="view-support-information-lab-user-in-azure-lab-services"></a>지원 정보 보기 (Azure Lab Services의 랩 사용자)
이 문서에서는 랩 사용자로 다음 지원 정보를 볼 수 있는 방법에 대해 설명 합니다.

- URL
- 메일
- Phone
- 추가 지침

이 정보를 사용 하 여 랩 계정에서 랩을 사용할 때 기술 문제가 발생할 때 도움을 받을 수 있습니다.

 
## <a name="view-support-information"></a>지원 정보 보기
1. [Azure Lab Services 포털](https://labs.azure.com)에 로그인 합니다.
2. 도움이 필요한 **랩 또는 가상 컴퓨터** 를 선택 하 고 **?** 를 선택 합니다. 페이지의 오른쪽 위 모퉁이에 있습니다. 
3. **보기 지원 웹 사이트**, **전자 메일 지원**및 **지원 전화 번호**에 대 한 링크가 표시 되는지 확인 합니다.

    ![지원 정보 보기](../media/lab-user-support-information/support-information.png)
4. 드롭다운 목록에서 해당 랩으로 전환 하 여 다른 랩에 대 한 지원 연락처 정보를 볼 수 있습니다. 

    ![다른 랩으로 전환](../media/lab-user-support-information/switch-another-lab.png)
5. 이제 다른 랩에 대 한 지원 연락처 정보가 표시 됩니다. 

    ![기타 랩의 지원 정보](../media/lab-user-support-information/second-lab-support-information.png)
