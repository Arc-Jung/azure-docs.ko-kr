#RemoteApp에 앱을 게시하는 방법

클라우드 또는 하이브리드 컬렉션을 만든 후 사용자가 사용할 수 있게 하려는 앱이나 리소스를 게시해야 합니다. 구독과 함께 제공된 템플릿 이미지에는 기본적으로 몇 개의 앱만 게시되어 있습니다. 다른 앱을 공유하려면 게시해야 합니다.
 
포털의 **게시** 탭에서 **게시**를 클릭합니다. 템플릿 이미지의 시작 메뉴에서 앱을 추가하거나 템플릿 이미지에서 앱이 설치되어 있는 경로를 제공합니다. 시작 메뉴에서 추가하도록 선택한 경우 목록에서 게시할 앱을 선택합니다. 앱의 경로를 제공하도록 선택한 경우 앱의 이름과 경로를 입력합니다. 경로에 변수를 사용합니다. 예를 들어 "c:\" 대신 "%systemdrive%"를 사용합니다.

**참고**: 시작 메뉴에서 앱에 추가하려는 경우 *템플릿 이미지의 시작 메뉴에 해당 앱이 추가되어 있어야 합니다.* 그렇지 않으면 RemoteApp에 시작 메뉴에 *있는" 항목만 표시되므로 혼동을 줍니다. 템플릿을 만들 때 앱을 시작 메뉴에 추가할 것을 잊은 경우 앱의 경로를 추가합니다. 또는 템플릿 이미지를 다시 만들 수 있지만 훨씬 더 많은 작업이 필요합니다.
