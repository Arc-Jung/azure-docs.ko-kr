# Azure 검색을 사용하는 첫 검색 솔루션 만들기

## 서비스 예정

Azure 검색의 기능을 보여 주는 샘플 응용 프로그램을 마무리하는 동안 기다려 주셔서 감사합니다. 며칠 후에 샘플 응용 프로그램이 제공됩니다. 준비가 되면 샘플 코드를 다운로드하고 지침에 따라 Azure 검색 서비스에서 실행할 수 있습니다.

지금 몇 단계를 완료할 수 있습니다. 몇 가지 항목을 다운로드하고 서비스를 구성하여 빨리 시작하세요.

-   Visual Studio 2013 Express for the Web
-   Azure 검색 미리 보기 구성

### Visual Studio 2013 Express for the Web 다운로드

샘플 응용 프로그램은 Visual Studio 2013으로 작성되었습니다. 기존 Visual Studio 2013 버전이 이미 있는 경우 해당 버전을 사용해야 합니다. 그렇지 않으면 Microsoft에서 무료 버전을 다운로드할 수 있습니다.

[다운로드 센터][]로 이동하여 Visual Studio 2013 Express for the Web을 다운로드합니다.

### 무료 검색 서비스 구성

이 자습서에는 무료 서비스를 사용하는 것이 좋습니다. 이는 모든 구독자가 추가 비용 없이 사용할 수 있는 제한된 검색 버전입니다. 구독이 없는 경우 체험을 등록할 수 있습니다.

서비스 설정에 대한 자세한 내용은 [Azure 미리 보기 포털에서 검색 구성][]을 참조하세요.

지침에는 실제로 필요한 것보다 많은 단계가 있습니다. 무료 서비스를 만든 후 즉시 중지할 수 있습니다. 이 자습서를 완료하려면 서비스 URL과 api-key가 필요합니다.
