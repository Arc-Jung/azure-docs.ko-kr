# Azure 청구 및 구독 문제에 대한 지원 티켓을 만드는 방법

이 문서에서는 Azure 지원 티켓을 여는 데 필요한 정보를 제공합니다.

>[AZURE.NOTE]Azure 계정 관리자만 계정 센터에 액세스할 수 있는 권한이 있습니다.

1. [Azure 지원](http://azure.microsoft.com/support) 웹 사이트로 가서 **지원 받기**를 클릭합니다. Microsoft Azure 포털 또는 Azure 계정 센터에서 인시던트를 제출할 수도 있습니다.

 * [Azure 포털](https://manage.windowsazure.com/)에서 지원 티켓을 제출하려면 해당 계정 이름을 클릭한 다음 Microsoft 지원에 문의를 클릭합니다.
 * [Azure 계정 센터](https://account.windowsazure.com/Subscriptions)에서 지원 티켓을 제출하려면 구독을 선택하고 Microsoft 지원에 문의를 클릭합니다.

2. **지원 티켓 만들기** 창에서 데이터 및 서비스를 마이그레이션하려는 구독을 선택합니다.

3. **지원 유형**으로 **청구**를 선택합니다.

4. 해당 지역 및 언어를 선택하고 **티켓 만들기**를 클릭합니다.

5. 다음 페이지에서 **문제 유형**으로 **내 구독이 사용 안 함으로 표시됩니다.**를 선택합니다. 지원 티켓 제출과 관련해서 지원이 필요한 경우 **고객 서비스와 채팅**을 클릭하여 고객 서비스 담당자와 라이브 대화를 시작합니다.

6. 연락처 정보를 확인하고 연락받을 수 있는 전화 번호를 입력한 다음 **계속**을 클릭합니다.

7. 다음 페이지에서 다음 정보를 제공합니다.

 * **문제 세부 정보**에서 요청에 대한 인시던트 제목을 입력한 다음 텍스트 상자를 사용하여 필요한 정보를 제공하고 요청에 대한 추가 정보를 포함합니다.
 * **심각도 결정**에서 요청의 심각도를 나타내는 옵션을 선택합니다. 심각도 유형에 대한 자세한 내용은 [Microsoft Azure 인시던트 심각도 표](http://support.microsoft.com/gp/AzureSevDetails)를 참조하세요.
 * **파일 업로드** 도구를 사용하여 관련 문서를 요청에 첨부합니다.

8. Microsoft 서비스 계약 내용에 동의하면 **제출**을 클릭합니다.

계약 내용에 따라 Azure 지원 담당자의 연락을 받게 됩니다.
