# <a name="how-to-create-a-support-ticket-for-azure-billing-and-subscription-issues"></a>Azure 청구 및 구독 문제에 대한 지원 티켓을 만드는 방법
이 문서에서는 Azure 지원 티켓을 여는 데 필요한 정보를 제공합니다.

1. [Azure 포털](https://portal.azure.com/)에 로그인합니다. **도움말 + 지원**>**새 지원 요청**을 선택합니다.

    ![helpandsupportbutton](./media/billing-how-to-create-billing-support-ticket/helpandsupport.png)
2. **기본** 블레이드에서 문제 유형, 구독 및 지원 계획을 선택합니다.
3. **문제** 블레이드에서 다음 정보를 제공합니다.

   * **심각도**. 요청의 심각도를 나타내는 옵션을 선택합니다. 심각도 유형에 대한 자세한 내용은 [Microsoft Azure 인시던트 심각도 표](http://support.microsoft.com/gp/AzureSevDetails)를 참조하세요.
   * **문제 유형**. 문제 유형을 선택합니다.
   * **세부 정보**. 요청에 대한 인시던트 제목을 입력한 다음 텍스트 상자를 사용하여 필요한 정보를 제공하고 요청에 대한 추가 정보를 포함합니다.
   * **시간 프레임** (옵션). 가능한 경우 가장 최근에 발생한 문제에 대한 날짜, 시간 및 표준 시간대 정보를 제공합니다.
   * **파일 업로드** 도구를 사용하여 관련 문서를 요청에 첨부합니다.
4. **연락처 정보** 블레이드에서 연락처 정보와 기본 연락 방법을 확인한 다음 **만들기**를 클릭합니다.

계약 내용에 따라 Azure 지원 담당자의 연락을 받게 됩니다.

또한 Azure 지원 웹 사이트 또는 Azure 계정 센터에서 지원 요청을 제출할 수도 있습니다.

* [Azure 지원 웹 사이트](https://azure.microsoft.com/support/options/)에서 지원 티켓을 제출하려면 **지원 받기**를 클릭합니다.
* [Azure 계정 센터](https://account.windowsazure.com/Subscriptions)에서 지원 티켓을 제출하려면 구독을 선택하고 **Microsoft 지원에 문의**를 클릭합니다.

  > [!NOTE]
  > Azure 계정 관리자만 계정 센터에 액세스할 수 있는 권한이 있습니다. 구독의 계정 관리자를 알아내는 방법에 대한 자세한 내용은 [질문과 대답](billing-subscription-transfer.md#faq)을 참조하세요.
