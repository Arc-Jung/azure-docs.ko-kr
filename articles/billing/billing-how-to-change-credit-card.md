# <a name="change-the-credit-card-used-to-pay-for-an-azure-subscription"></a>Azure 구독 지불에 사용하는 신용 카드 변경

이 문서에서는 Azure 계정에 연결된 신용 카드를 추가, 변경 또는 제거하는 방법 및 지불 방법에 대해 설명합니다.

<a id="addcard"></a>
## <a name="add-a-credit-card-as-a-payment-method"></a>결제 방법으로 신용 카드 추가

계정에 신용 카드를 추가하려면 다음 단계를 수행합니다.

1. [Azure 계정 센터](https://account.windowsazure.com/Subscriptions) 에 계정 관리자로 로그온합니다.
2. **세부 정보 및 사용량을 보려면 구독을 클릭하세요**에서 결제 방법을 추가하려는 구독을 선택합니다.
3. 페이지의 오른쪽에서 **결제 방법 관리**를 선택합니다.
4. **결제 방법 관리** 페이지에서 "+"를 선택하여 신용 카드를 추가합니다.
5. 신용 카드 종류를 선택한 다음 필요한 모든 정보를 제공합니다.
6. **다음**을 선택하여 **결제 방법 관리** 페이지로 돌아갑니다. 추가한 카드가 유효한 경우 사용 가능한 지불 방법 목록에 추가됩니다.

> [!NOTE]
>신용 카드 정보 제출 후 오류 메시지가 나타나는 경우 [사용자의 직불 카드 또는 신용 카드가 Azure 등록 시 거부되었습니다](billing-credit-card-fails-during-azure-sign-up.md) 문서를 참조하세요.
>
>

## <a name="edit-payment-information-for-an-existing-credit-card"></a>기존 신용 카드에 대한 결제 정보 편집

결제 방법을 확인하고 기존 신용 카드의 세부 정보를 업데이트하려면 다음 단계를 수행합니다.

1. [Azure 계정 센터](https://account.windowsazure.com/Subscriptions) 에 계정 관리자로 로그온합니다.

   > [!NOTE]
   > 계정 관리자만 계정 센터에 액세스할 수 있는 권한이 있습니다. 계정 관리자에 대한 자세한 내용은 [Azure 관리자 역할을 추가 또는 변경하는 방법](billing-add-change-azure-subscription-administrator.md)을 참조하세요.
   >
   >
2. **세부 정보 및 사용량을 보려면 구독을 클릭하세요**에서 신용 카드에 연결된 구독을 선택합니다.</br> ![selectsub](./media/billing-how-to-change-credit-card/selectsub.png)
3. 페이지의 오른쪽에서 **결제 방법 관리**를 선택합니다.</br> ![changesub](./media/billing-how-to-change-credit-card/changesub_new.png)

  현재 지불 방법이 나열됩니다. 선택하여 필요한 정보를 확인합니다.

  **기존 신용 카드의 세부 정보를 업데이트하는 방법**

  신용 카드가 갱신되고 번호가 동일하게 유지되면 만료 날짜처럼 기존 신용 카드 세부 정보를 간단하게 업데이트할 수 있습니다. 그러나 카드 분실, 도난 또는 만료로 인해 카드 번호가 변경되는 경우 [계정에 신용 카드를 추가](#addcard)해야 할 수 있습니다. 신용 카드 번호가 변경되면 카드에 있는 CVV 보안 코드가 변경되기 때문입니다.

4. **결제 방법 관리** 페이지에서 신용 카드 번호 옆에 있는 **편집**을 선택합니다.</br> ![changesub](./media/billing-how-to-change-credit-card/editcard_new.png)
5. **편집** 페이지에서 올바른 신용 카드 종류와 카드 번호를 선택했는지 확인합니다.
6. 필요한 카드 세부 정보를 변경한 후 **다음**을 선택하여 **결제 방법 선택** 페이지로 돌아갑니다.

## <a name="change-the-credit-card-that-you-use-to-pay-an-azure-bill"></a>Azure 청구서를 지불하는 데 사용하는 신용 카드 변경

1. [Azure 계정 센터](https://account.windowsazure.com/Subscriptions) 에 계정 관리자로 로그온합니다.
2. **세부 정보 및 사용량을 보려면 구독을 클릭하세요**에서 결제 방법을 업데이트하려는 구독을 선택합니다.
3. 페이지의 오른쪽에서 **결제 방법 관리**를 선택합니다.
4. **결제 방법 관리** 페이지에서 **대신 사용**을 클릭하여 사용할 신용 카드를 선택합니다.

## <a name="remove-a-credit-card-from-the-account"></a>계정에서 신용 카드 제거
1. [Azure 계정 센터](https://account.windowsazure.com/Subscriptions) 에 계정 관리자로 로그온합니다.
2. **세부 정보 및 사용량을 보려면 구독을 클릭하세요**에서 결제 방법을 업데이트하려는 구독을 선택합니다.
3. 페이지의 오른쪽에서 **결제 방법 관리**를 선택합니다.
4. **결제 방법 관리** 페이지에서 **삭제**를 클릭하여 삭제할 신용 카드를 선택합니다.

   > [!NOTE]
   > 신용 카드가 다른 활성 Microsoft 구독과 연결된 경우 신용 카드를 삭제할 수 없게 됩니다. 카드가 사용 중이라는 오류가 발생하면 Microsoft에 등록된 모든 활성 구독에서 카드가 연결 해제되었는지 확인하세요.
   >
   >

##  <a name="how-to-make-payments"></a>지불 방법

결제 방법으로 신용 카드 또는 직불 카드를 설정한 경우 자동으로 결제됩니다.
[결제를 송장으로 처리](https://azure.microsoft.com/pricing/invoicing/)한다면, 송장 밑에 나열된 장소에 결제 금액을 보냅니다.
