# <a name="change-the-profile-information-for-your-azure-account"></a>Azure 계정에 대 한 프로필 정보 변경

Microsoft에서는 계정 관리자 프로필의 전자 메일 주소를 사용 하 여 Azure 계정에 대 한 중요 한 청구, 서비스 및 권장 사항 관련 알림을 보냅니다. 정기적으로 확인 하는 전자 메일 주소를 지정 하는 것이 좋습니다. 계정 관리자는 프로필을 업데이트 하 여 계정에 대 한 메일 주소 및 전화 번호를 변경할 수도 있습니다.

Microsoft는 계정 관리자가 아닌 사용자의 프로필에 있는 전자 메일 주소를 사용 하 여 서비스 관련 알림 및 계정 권장 사항을 보냅니다. 모든 청구 알림은 계정 관리자에 게 전송 됩니다. 또한 메일 주소 및 전화 번호를 변경 하는 프로필을 업데이트할 수 있지만 계정 메일 주소 또는 전화 번호는 변경 되지 않습니다.

## <a name="change-your-profile-information"></a>프로필 정보 변경

1. [Azure 계정 센터](https://account.azure.com/subscriptions)에 로그인합니다.
1. **세부 정보 편집** 단추를 클릭하고 **프로필** 정보를 업데이트합니다.

   ![Azure 프로필 정보의 예](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>추가 전자 메일 주소로 알림 보내기

계정에 연결 된 계정 관리자의 전자 메일 주소 외에도 서비스 알림 및 계정 권장 사항을 전자 메일 주소로 보낼 수 있습니다. 계정 관리자의 전자 메일 주소는 청구 관련 업데이트를 포함 하 여 계정의 모든 구독에 대 한 중요 한 알림을 가져옵니다.

### <a name="change-the-account-administrators-email-address-in-your-profile"></a>프로필에서 계정 관리자의 전자 메일 주소 변경

계정 관리자는 Azure 계정을 설정하고 계정의 모든 구독에 대한 중요한 메일 알림을 받는 사람입니다. 계정 관리자가 정기적으로 확인하는 연락처 메일 주소를 지정하는 것이 좋습니다.

### <a name="update-your-sign-in-email-address"></a>로그인 메일 주소 업데이트

프로필 전자 메일 주소를 업데이트 해도 로그인 전자 메일 주소는 업데이트 되지 않습니다. 로그인 메일 주소를 변경 하려면 계정의 소유권을 이전 해야 합니다.

### <a name="update-the-billing-address-for-your-credit-card"></a>신용 카드에 대 한 청구 주소 업데이트

청구 정보를 업데이트하는 방법을 알아보려면 [Azure 구독 지불에 사용하는 신용 카드 변경](billing-how-to-change-credit-card.md)을 참조하세요.

### <a name="update-your-countryregion"></a>국가/지역 업데이트

기술 제약 조건 때문에 기존 계정에 대 한 국가/지역을 변경할 수 없습니다. 그러나 원하는 국가/지역에서 계정을 만든 다음 Azure 지원에 문의 하 여 해당 계정으로 서비스를 마이그레이션할 수 있습니다.

## <a name="service-and-marketing-emails"></a>서비스 및 마케팅 전자 메일

90 일 마다 전자 메일 주소를 확인 하거나 업데이트 하는 [Azure Portal](https://portal.azure.com) 에 메시지가 표시 됩니다. Microsoft는 다음에 대 한 Azure 계정 관련 정보를 사용 하 여 프로필 전자 메일 주소로 전자 메일을 보냅니다.

- 서비스 알림
- 보안 경고
- 청구 목적
- Support(지원)
- 마케팅 커뮤니케이션
- Azure 사용량에 따라 모범 사례 권장 사항

계정에 대 한 통신을 받을 전자 메일 주소를 입력 합니다. 전자 메일 주소를 입력 하면 Microsoft와의 통신을 받기 위해 옵트인 하 게 됩니다.

![연락처 정보를 업데이트 하는 메시지의 예](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>연락처 전자 메일 주소 변경

다음 방법 중 하나를 사용 하 여 연락처 전자 메일 주소를 변경할 수 있습니다.

* [Azure 계정 센터](https://account.azure.com/subscriptions)에 로그인합니다. **세부 정보 편집** 을 선택 하 고 프로필 전자 메일 주소를 업데이트 합니다. 프로필 전자 메일 주소를 업데이트 해도 로그인 전자 메일 주소는 업데이트 되지 않습니다.

* Azure Portal의 [연락처 정보](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) 영역으로 이동 하 여 전자 메일 주소를 입력 합니다. 제공 하는 주소는 서비스 및 마케팅 통신을 수신 합니다. Azure 계정 센터 프로필의 전자 메일 주소가 자동으로 일치 하도록 업데이트 됩니다. 프로필 전자 메일 주소를 업데이트 해도 로그인 전자 메일 주소는 업데이트 되지 않습니다.

* [Azure Portal](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)에서 이니셜 또는 그림이 있는 아이콘을 선택 합니다. 그런 다음 상황에 맞는 메뉴 ( **...** )를 선택 합니다. 그런 다음 메뉴에서 **내 연락처 정보** 를 선택 하 고 전자 메일 주소를 입력 합니다. 제공 하는 주소는 서비스 및 마케팅 통신을 수신 합니다. Azure 계정 센터 프로필의 전자 메일 주소가 자동으로 일치 하도록 업데이트 됩니다. 프로필 전자 메일 주소를 업데이트 해도 로그인 전자 메일 주소는 업데이트 되지 않습니다.

![Azure에서 전자 메일 주소를 업데이트 하는 예제](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>마케팅 전자 메일 옵트아웃 (Opt out)

마케팅 전자 메일 수신을 옵트아웃 (opt out) 하려면 다음을 수행 합니다.

1. [요청 양식](https://account.microsoft.com/profile/permissions-link-request) 으로 이동 하 여 프로필 전자 메일 주소를 사용 하 여 요청을 제출 합니다. 기본 설정을 업데이트 하는 링크를 전자 메일로 받게 됩니다.
2. **통신 권한 관리** 페이지를 열려면 링크를 선택 합니다. 이 페이지에서는 전자 메일 주소를 옵트인 하는 마케팅 통신의 유형을 보여 줍니다. 옵트아웃 (opt out) 하려는 모든 항목의 선택을 취소 하 고 **저장**을 선택 합니다.

![통신 권한 관리 페이지의 예](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

마케팅 통신을 옵트아웃 하면 계정에 따라 서비스 알림이 계속 수신 됩니다.

## <a name="change-the-subscription-name"></a>구독 이름 변경

1. [Azure Portal](https://portal.azure.com)에 로그인 하 고 왼쪽 창에서 **구독** 을 선택한 다음 이름을 바꿀 구독을 선택 합니다.
1. **개요**를 선택한 다음 명령 모음에서 **이름 바꾸기** 를 선택 합니다.
    ![Azure 구독 이름 바꾸기 예제](./media/billing-how-to-change-azure-account-profile/rename-sub.png)
1. 이름을 변경한 후에는 **저장**을 선택 합니다.
