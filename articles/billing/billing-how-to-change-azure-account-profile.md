# <a name="change-your-azure-account-profile-information"></a>Azure 계정 프로필 정보 변경

Azure 계정에 대 한 연락을 프로필 정보 사용 됩니다. Microsoft 중요 한 알림에 대 한 경고 수 있도록 업데이트 해야 유지 해야 합니다.

계정 관리자 프로필의 전자 메일 주소는 중요 한 대금 청구, 서비스 및 계정에 대 한 권장 사항 관련 알림을 보내도록 Microsoft에서 사용 됩니다. 정기적으로 확인 하는 전자 메일 주소를 지정 하는 것이 좋습니다. 계정 관리자는 또한 계정의 메일 주소 및 전화 번호를 변경 하려면 프로필을 업데이트할 수 있습니다.

사용자의 프로필, 계정 관리자가 아닌 계정 사용자의에서 전자 메일 주소는 Microsoft에서 서비스 관련 알림 및 계정 권장 사항에 사용 됩니다. 모든 청구 알림 계정 관리자에 전송 됩니다. 메일 주소 및 전화 번호를 변경 하려면 프로필을 업데이트할 수도 있습니다 하지만 계정 메일 주소 또는 전화 번호 변경 되지 않습니다.

## <a name="change-your-profile-information"></a>프로필 정보 변경

1. [Azure 계정 센터](https://account.azure.com/Profile)에 로그인합니다.
1. **세부 정보 편집** 단추를 클릭하고 **프로필** 정보를 업데이트합니다.

   ![Azure 프로필 정보를 보여 주는 예제](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>추가 전자 메일 주소로 알림 보내기

서비스 알림 및 사용자 계정과 연결 된 계정 관리자 메일 주소 외에도 전자 메일 주소에 전송 하는 계정 권장 사항 있을 수 있습니다. 계정 관리자의 전자 메일 주소는 대금 청구 관련 업데이트를 포함 하 여 계정에서 모든 구독에 대 한 중요 한 알림을 가져옵니다.

### <a name="change-the-account-administrator-email-address-in-your-profile"></a>프로필에 계정 관리자 메일 주소를 변경 합니다.

계정 관리자는 Azure 계정을 설정하고 계정의 모든 구독에 대한 중요한 메일 알림을 받는 사람입니다. 계정 관리자가 정기적으로 확인하는 연락처 메일 주소를 지정하는 것이 좋습니다.

### <a name="updating-your-profile-email-doesnt-update-your-sign-in-email-address"></a>로그인 전자 메일 주소를 업데이트 하지 않는 전자 메일 프로필 업데이트

프로필 메일 주소를 업데이트 하면 로그인 메일 주소를 업데이트 되지 않습니다. 로그인 전자 메일 주소를 변경 하려면 계정 소유권을 전송 해야 합니다.

### <a name="updating-your-profile-address-doesnt-update-your-credit-card-billing-address"></a>신용 카드 청구 주소를 업데이트 하지 않는 프로필 주소를 업데이트 하는 중

청구 정보를 업데이트하는 방법을 알아보려면 [Azure 구독 지불에 사용하는 신용 카드 변경](billing-how-to-change-credit-card.md)을 참조하세요.

### <a name="you-cant-update-your-countryregion"></a>국가/지역을 업데이트할 수 없습니다.

기술 제약 조건으로 인해 기존 계정의 국가/지역 변경할 수 없습니다. 그러나 원하는 국가/지역에 계정을 만들 수 있으며 해당 계정으로 서비스를 마이그레이션하려면 Azure 지원부에 문의 하십시오.

## <a name="service-and-marketing-emails"></a>서비스 및 전자 메일 마케팅

Microsoft Azure 팀 중요 한 경고를 알리는 서비스와 관련 된 전자 메일을 보냅니다. 팀에서는 또한 계정 권장 사항 전자 메일을 라고도 마케팅 전자 메일을 최적화 하 고 프로젝트를 확장 하는 데 보냅니다.

메시지가 표시 되는 [Azure portal](https://portal.azure.com) 확인 또는 90 일 마다 전자 메일 주소를 업데이트 합니다. 알림 목적은 Microsoft 긴급 서비스 관련 알림에 경우 연락처 정보는 업데이트 되었는지 확인 하는 것입니다. Microsoft에 대 한 Azure 계정 관련 정보를 사용 하 여 프로필 전자 메일 주소로 전자 메일을 보냅니다.

- 서비스 알림
- 보안 경고
- 청구 목적으로
- 지원
- 마케팅 통신
- Azure 사용량에 따라, 모범 사례 권장 사항

계정에 대 한 정보를 받게 하려는 전자 메일 주소를 입력 합니다. 전자 메일 주소를 입력 하면 옵트인에 Microsoft의 커뮤니케이션.

![연락처 정보 상자에 업데이트를 보여 주는 예제](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>연락처 전자 메일 주소를 변경 합니다.

다음 메서드를 사용 하 여 연락처 전자 메일 주소를 변경할 수 있습니다.

**옵션 1**: 에 로그인 합니다 [Azure 계정 센터](https://account.azure.com/Profile)합니다. 선택 된 **세부 정보 편집** 하 고 다음 프로필 전자 메일 주소 정보를 업데이트 합니다. 프로필 메일 주소를 업데이트 하면 로그인 메일 주소를 업데이트 되지 않습니다.

**옵션 2**: 로 이동 합니다 [연락처 정보](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) 영역 Azure portal에서 전자 메일 주소를 입력 합니다. 서비스 및 통신 마케팅 제공한 주소를 받습니다. Azure 계정 센터 프로필의 메일 주소는 자동으로 일치 하도록 업데이트 됩니다. 프로필 메일 주소를 업데이트 하면 로그인 메일 주소를 업데이트 되지 않습니다.

**옵션 3**: 에 [Azure portal](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)이니셜이 있는 아이콘을 선택 하거나 그림입니다. 그런 다음 상황에 맞는 메뉴를 선택 (**...** ). 다음으로, 선택 **내 연락처 정보** 메뉴에서 전자 메일 주소를 입력 합니다. 서비스 및 통신 마케팅 제공한 주소를 받습니다. Azure 계정 센터 프로필에 전자 메일 주소는 자동으로 일치 하도록 업데이트 됩니다. 프로필 메일 주소를 업데이트 하면 로그인 메일 주소를 업데이트 되지 않습니다.

![Azure에서 연락처 정보 업데이트를 보여 주는 예제](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>옵트아웃 전자 메일 마케팅

옵트아웃 하려면 마케팅 전자 메일 수신입니다.

1. 로 이동 합니다 [요청 양식](https://account.microsoft.com/profile/permissions-link-request) 프로필 전자 메일 주소를 사용 하 여 요청을 제출 합니다. 기본 설정을 업데이트 하는 전자 메일로 링크를 받게 됩니다.
2. 이 링크에 옵트인 전자 메일 주소는 마케팅 통신 형식을 보여 줍니다. 옵트아웃 하 고 클릭 하려는 모든 항목 지우기 **저장할**합니다.


![관리 통신 권한 옵션을 보여 주는 예제](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

### <a name="you-still-receive-service-notifications-if-you-opt-out-of-marketing-communications"></a>옵트아웃할 있습니다 마케팅 통신의 경우 여전히 서비스 알림 수신

통신에 대 한 전자 메일 주소를 제공한 경우 계정에 따라 서비스 알림을 받게 됩니다. 경우 옵트아웃할 있습니다, 있습니다만 옵트아웃 마케팅 통신-서비스 알림은 계속 받게 됩니다.
