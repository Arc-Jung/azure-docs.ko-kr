# <a name="change-the-profile-information-for-your-azure-account"></a>Azure 계정의 프로필 정보 변경

Microsoft 계정 관리자 프로필의 전자 메일 주소를 사용 하 여 중요 한 대금 청구, 서비스 및 Azure 계정에 대 한 권장 사항 관련 알림을 보냅니다. 정기적으로 확인 하는 전자 메일 주소를 지정 하는 것이 좋습니다. 계정 관리자와 계정의 메일 주소 및 전화 번호를 변경 하려면 프로필을 업데이트할 수도 있습니다.

Microsoft는 사용자의 프로필, 계정 관리자가 서비스 관련 알림 및 계정 권장 사항을 보낼 수 없는 사용자는 계정 사용자의에서 전자 메일 주소를 사용 합니다. 모든 청구 알림 계정 관리자에 전송 됩니다. 메일 주소 및 전화 번호를 변경 하려면 프로필을 업데이트할 수도 있습니다 하지만 계정 메일 주소 또는 전화 번호 변경 되지 않습니다.

## <a name="change-your-profile-information"></a>프로필 정보 변경

1. [Azure 계정 센터](https://account.azure.com/Profile)에 로그인합니다.
1. **세부 정보 편집** 단추를 클릭하고 **프로필** 정보를 업데이트합니다.

   ![Azure 프로필 정보의 예](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>추가 전자 메일 주소로 알림 보내기

서비스 알림 및 사용자 계정과 연결 된 계정 관리자의 전자 메일 주소 외에도 전자 메일 주소에 전송 하는 계정 권장 사항 있을 수 있습니다. 계정 관리자의 전자 메일 주소는 대금 청구 관련 업데이트를 포함 하 여 계정에서 모든 구독에 대 한 중요 한 알림을 가져옵니다.

### <a name="change-the-account-administrators-email-address-in-your-profile"></a>프로필에 계정 관리자의 전자 메일 주소를 변경 합니다.

계정 관리자는 Azure 계정을 설정하고 계정의 모든 구독에 대한 중요한 메일 알림을 받는 사람입니다. 계정 관리자가 정기적으로 확인하는 연락처 메일 주소를 지정하는 것이 좋습니다.

### <a name="update-your-sign-in-email-address"></a>로그인 전자 메일 주소 업데이트

프로필 메일 주소를 업데이트 하면 로그인 메일 주소를 업데이트 되지 않습니다. 로그인 전자 메일 주소를 변경 하려면 계정 소유권을 전송 해야 합니다.

### <a name="update-the-billing-address-for-your-credit-card"></a>신용 카드 요금 청구서 주소를 업데이트 합니다.

청구 정보를 업데이트하는 방법을 알아보려면 [Azure 구독 지불에 사용하는 신용 카드 변경](billing-how-to-change-credit-card.md)을 참조하세요.

### <a name="update-your-countryregion"></a>국가/지역을 업데이트합니다

기술 제약 조건으로 인해 기존 계정의 국가/지역 변경할 수 없습니다. 그러나 원하는 국가/지역에서 계정을 만들어야 하는 계정으로 서비스를 마이그레이션하려면 Azure 지원부에 문의 하십시오.

## <a name="service-and-marketing-emails"></a>서비스 및 전자 메일 마케팅

메시지가 [Azure portal](https://portal.azure.com) 을 확인 하거나 전자 메일 주소를 90 일 마다 업데이트 합니다. Microsoft에 대 한 Azure 계정 관련 정보를 사용 하 여 프로필 전자 메일 주소로 전자 메일을 보냅니다.

- 서비스 알림
- 보안 경고
- 청구 목적으로
- Support(지원)
- 마케팅 통신
- Azure 사용량에 따라, 모범 사례 권장 사항

계정에 대 한 정보를 받게 하려는 전자 메일 주소를 입력 합니다. 전자 메일 주소를 입력 하 여 microsoft에서 커뮤니케이션에 옵트인 하 고 있습니다.

![귀하의 연락처 정보를 업데이트 하려면 프롬프트의 예](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>연락처 전자 메일 주소를 변경 합니다.

다음 방법 중 하나를 사용 하 여 연락처 전자 메일 주소를 변경할 수 있습니다.

* [Azure 계정 센터](https://account.azure.com/Profile)에 로그인합니다. 선택 **세부 정보 편집** 프로필 전자 메일 주소를 업데이트 합니다. 프로필 메일 주소를 업데이트 하면 로그인 메일 주소를 업데이트 되지 않습니다.

* 로 이동 합니다 [연락처 정보](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) 영역 Azure portal에서 전자 메일 주소를 입력 합니다. 서비스 및 통신 마케팅 사용자가 제공한 주소를 받습니다. Azure 계정 센터 프로필의 메일 주소는 자동으로 일치 하도록 업데이트 됩니다. 프로필 메일 주소를 업데이트 하면 로그인 메일 주소를 업데이트 되지 않습니다.

* 에 [Azure portal](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)이니셜이 있는 아이콘을 선택 하거나 그림입니다. 그런 다음 상황에 맞는 메뉴를 선택 ( **...** ). 다음으로, 선택 **내 연락처 정보** 메뉴에서 전자 메일 주소를 입력 합니다. 서비스 및 통신 마케팅 사용자가 제공한 주소를 받습니다. Azure 계정 센터 프로필에 전자 메일 주소는 자동으로 일치 하도록 업데이트 됩니다. 프로필 메일 주소를 업데이트 하면 로그인 메일 주소를 업데이트 되지 않습니다.

![Azure에서 전자 메일 주소를 업데이트 하는 예제](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>전자 메일 마케팅 옵트아웃

마케팅 전자 메일 수신 옵트아웃 하려면:

1. 로 이동 합니다 [요청 양식](https://account.microsoft.com/profile/permissions-link-request) 프로필 전자 메일 주소를 사용 하 여 요청을 제출 합니다. 기본 설정을 업데이트 하는 전자 메일로 링크를 받게 됩니다.
2. 여는 링크를 선택 합니다 **통신 사용 권한을 관리할** 페이지입니다. 이 페이지는 전자 메일 주소를 옵트인 하는 마케팅 통신 형식을 보여 줍니다. 선택 및 선택 하려는 항목 지우기 **저장할**합니다.

![통신 사용 권한 관리에 대 한 페이지의 예](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

마케팅 통신을 옵트아웃 하는 경우 알림 서비스 계정에 따라 계속 나타납니다.

1. 선택 **개요**를 선택한 후 **이름 바꾸기** 명령 모음에서.
    ![Azure 구독 이름 바꾸기의 예제](./media/billing-how-to-change-azure-account-profile/rename-sub.png)
1. 이름을 변경한 후 선택할 **저장할**합니다.
