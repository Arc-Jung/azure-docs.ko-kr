# <a name="view-your-billing-accounts-in-azure-portal"></a>Azure portal에서 청구 계정 보기  

대금 청구 계정 로그인 microsoft Azure를 사용 하 여 각 계약에 대해 만들어집니다. 청구 계정을 사용 하 여 송장 및 지불 방법, 관리 하 고 Azure 서비스에 대 한 비용을 추적 합니다. 여러 청구 계정에 대 한 액세스를 할 수 있습니다. 예를 들어 수 등록 한 Azure에 대 한 개인 프로젝트에 대 한 합니다. 조직의 기업 계약 또는 Microsoft 고객 계약을 통해 액세스할을 수도 수 있습니다. 이러한 시나리오 각각에 대 한 별도 청구 계정을 사용 해야 합니다.

Azure portal에는 현재 다음과 같은 유형의 청구 계정 지원합니다.

- **Microsoft Online Services 프로그램**: Microsoft Online Services 프로그램에 대 한 대금 청구 계정에 Azure 웹 사이트를 통해 Azure에 등록할 때 만들어집니다. 예를 들어, 로그인 하면 등록을 [Azure 무료 계정](https://azure.microsoft.com/offers/ms-azr-0044p/), [Azure 종 량 제 제품](https://azure.microsoft.com/offers/ms-azr-0003p/) 또는 [Visual studio 구독자](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/)합니다.

- **기업 계약**: 기업 계약에 대 한 대금 청구 계정은 조직 Azure를 사용 하는 엔터프라이즈 규약 생성 됩니다. 기업 계약에 대 한 대금 청구 계정에 관리자는 부서 및 계정 관리 및 조직에 대 한 청구를 구성 하는 범위를 대금 청구를 사용 합니다.

- **Microsoft 고객 계약**: 조직이 Azure를 사용 하는 Microsoft 고객 계약에 서명 하는 경우 Microsoft 고객 계약에 대 한 대금 청구 계정을 생성 됩니다. Microsoft 고객 계약에 대 한 대금 청구 계정에 관리자는 청구 프로필을 사용 하 고 섹션에서는 조직에 대 한 청구를 구성 하는 범위를 청구 송장. 자세한 내용은 [Microsoft 고객 계약에 대 한 대금 청구 계정 시작](billing-mca-overview.md)합니다.

## <a name="change-billing-account-view-in-the-azure-portal"></a>Azure portal에서 계정 보기 청구 변경

범위는 보기 및 청구를 관리 하는 데 사용할 청구 계정 내에서 노드입니다. 예를 들어, Microsoft Online Services 프로그램에 대 한 계정 청구 범위가-대금 청구 계정, 부서 및 계정 범위에 계정을 기업 계약에 대 한 청구 계정에 청구 합니다. 기본적으로 여러 청구 범위에 액세스를 해야 하는 경우 Azure Cost Management에 표시 됩니다 범위 중 하나 + 페이지를 청구 합니다. 아래 지침에 따라 범위를 변경할 수 있습니다.

1. [Azure Portal](https://portal.azure.com)에 로그인합니다.

2. **Cost Management + 청구**에서 검색합니다.

   ![Azure Portal 검색을 보여 주는 스크린샷](./media/billing-view-all-accounts/billing-search-cost-management-billing.png)

3. 선택 **모든 청구 범위** 왼쪽 창에서.

   ![모든 청구 범위를 보여 주는 스크린샷](./media/billing-view-all-accounts/billing-list-of-accounts.png)

4. 세부 정보를 보려면 범위를 선택 합니다.
