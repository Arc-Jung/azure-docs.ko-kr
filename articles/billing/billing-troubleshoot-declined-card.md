# <a name="troubleshoot-a-declined-card-at-azure-sign-up"></a>Azure 등록 시 거부 된 카드 문제 해결

이 문서는 Azure Portal 또는 Azure 계정 센터에서 Azure 등록 시 신용 카드가 거부 된 문제를 해결 하는 데 도움이 됩니다. 문제 해결을 시작 하기 전에 다음 사항을 확인 하십시오.

- 연락처 메일, 주소 및 전화 번호와 같은 Azure 계정 프로필에 대해 제공한 정보가 올바른지 확인 하세요.
- 신용 카드 정보가 정확한지 확인합니다.
- 동일한 정보의 Microsoft 계정이 이미 있는지 확인합니다.
- 직불 카드는 허용 되지 않습니다.

## <a name="issues"></a>문제

다음 목록에는 Azure 등록 시 신용 카드를 거부할 수 있는 일반적인 문제가 나열 되어 있습니다.

### <a name="the-credit-card-provider-is-not-accepted-for-your-country"></a>신용 카드 공급자는 귀하의 국가에서 허용 되지 않습니다.

카드를 선택 하면 선택한 국가에서 유효한 카드 옵션이 Azure에 표시 됩니다. 은행 또는 카드 발급자에 게 문의 하 여 국제 거래에 신용 카드가 사용 되는지 확인 합니다. 지원 되는 국가 및 통화에 대 한 자세한 내용은 [Azure 구매 FAQ](https://azure.microsoft.com/pricing/faq/)를 참조 하세요.

>[!Note]
>미국 Express 신용 카드는 현재 인도 결제 수단으로 지원 되지 않습니다. 승인 된 형태의 지불 인 경우에는 시간 프레임을 사용할 수 없습니다.

### <a name="youre-using-a-virtual-or-prepaid-card"></a>가상 카드 또는 선불 카드를 사용 하 고 있습니다. 

가상 카드, 선불 카드 또는 직불 카드는 Azure 구독의 지불 옵션으로 허용되지 않습니다.

### <a name="your-credit-information-is-inaccurate-or-incomplete"></a>신용 정보가 부정확 하거나 불완전 합니다. 

입력 한 이름, 주소 및 CVV 코드는 카드에 인쇄 되는 것과 정확 하 게 일치 해야 합니다.

### <a name="the-card-is-inactive-or-blocked"></a>카드가 비활성화 되었거나 차단 되었습니다. 

은행에 문의 하 여 카드가 활성 상태 인지 확인 합니다.

다른 등록 문제가 발생 했을 수 있습니다. 

Azure 등록 문제를 해결 하는 방법에 대 한 자세한 내용은 다음 기술 자료 문서를 참조 하세요. 

[Azure Portal 또는 Azure 계정 센터에서 Azure에 등록할 수 없습니다.](billing-troubleshoot-azure-sign-up.md)

### <a name="you-represent-a-business-that-doesnt-want-to-pay-by-card"></a>카드로 지불 하지 않으려는 비즈니스를 나타냅니다. 

비즈니스를 대표 하는 경우 확인, 야간 검사 또는 실시간 전송과 같은 청구서 지불 방법을 사용 하 여 Azure 구독에 대 한 요금을 지불할 수 있습니다. 청구서로 지불 하도록 계정을 설정한 후에는 Microsoft 고객 계약이 있고 Azure 웹 사이트를 통해 Azure에 등록 한 경우를 제외 하 고는 다른 지불 옵션으로 변경할 수 없습니다.

청구서로 지불 하는 방법에 대 한 자세한 내용은 [청구서로 지불 하는 Azure 구독에 대 한 요청 제출을](billing-how-to-pay-by-invoice.md)참조 하세요.

### <a name="your-credit-card-information-is-outdated"></a>신용 카드 정보가 만료 되었습니다. 

카드를 변경 하거나 제거 하는 등 카드 정보를 관리 하는 방법에 대 한 자세한 내용은 [Azure에 대 한 크레딧 추가, 업데이트 또는 제거](billing-how-to-change-credit-card.md)를 참조 하세요.
