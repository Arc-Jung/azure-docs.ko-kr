Azure 데이터 관리: 올바른 선택
==============================

**요약:** Azure는 데이터를 저장하고 관리하는 방법을 다양하게 제공합니다. 이런 다양성 때문에 플랫폼 사용자는 다양한 문제를 해결할 수 있습니다. 하지만 다양성은 선택의 문제를 낳습니다. 특정 상황에 맞는 데이터 관리 옵션은 무엇인지 결정해야 합니다.

간략한 이 개요는 이런 질문에 대답할 수 있게 도움을 드립니다. Azure의 데이터 관리 기술인 SQL 데이터베이스, VM의 SQL Server, Blob 저장소, 테이블 저장소 등에 대해 빠른 개요를 확인하십시오. 그런 다음, 몇 가지 시나리오에 대해 읽어 보고 각 경우에 가장 적합한 기술은 어느 것인지 알아 보십시오.

**내용:**

-   Azure 데이터 관리: 요약
-   데이터 관리 기술 선택: 예
    -   단일 조직을 위한 엔터프라이즈 응용 프로그램 또는 부서 응용 프로그램 만들기
    -   많은 조직을 위한 SaaS 응용 프로그램 만들기
    -   확장 가능한 소비자 응용 프로그램 만들기
    -   빅데이터 응용 프로그램 만들기
    -   데이터 스트리밍 응용 프로그램 만들기
    -   Windows Server 응용 프로그램을 Azure로 이동
    -   클라우드 데이터를 사용하는 온프레미스 응용 프로그램 만들기
    -   클라우드에 백업 데이터 저장
