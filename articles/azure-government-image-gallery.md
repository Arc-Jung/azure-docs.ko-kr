#  Microsoft Azure 정부 이미지 갤러리

<p> Microsoft Azure 정부 이미지 갤러리에는 공용 Azure 포털에 비슷한 환경을입니다.  고객은 Microsoft와 파트너가 협력에서 미리 작성 된 이미지를 배포 하거나 자신의 Vhd를 업로드 하도록 선택할 수 있습니다. 필요한 경우 사용자 고유의 표준화 된 이미지를 배포 하는 유연성을 제공 합니다.
