# <a name="security-recommendations-for-azure-marketplace-images"></a>Azure Marketplace의 보안 권장 사항 이미지

이미지는 이러한 보안 구성 권장 사항을 충족해야 합니다. 그러면 Azure Marketplace에서 파트너 솔루션 이미지의 높은 보안 수준을 유지할 수 있습니다.

제출하기 전에 항상 이미지에서 보안 취약점 검색을 실행합니다. 게시된 이미지에서 보안 취약점이 감지되면 고객에게 취약점과 이를 해결하는 방법을 적시에 알려야 합니다.

## <a name="contacting-customers"></a>고객에게 연락하기

고객 및 고객 연락처 이메일을 식별하려면 다음 을 수행하십시오.

1.  클라우드 파트너 포털에서 왼쪽 레일에서 **인사이트를**선택합니다.
2.  주문 **및 사용** 탭에서 **시작 날짜** 및 **종료 날짜** 필드를 사용하여 필요한 날짜 범위 내에서 사용량을 쿼리합니다. 이는 매일 오퍼에 사용된 Azure 구독을 보여 주며, 이 에 대해 보여 줄 수 있습니다. 이 데이터를 내보냅니다. 
3.  마찬가지로 **고객** 탭에서 고객 기반을 쿼리하고 내보냅니다.
4.  2단계에서 구독 ID로 3단계에서 구독 ID로 일치하여 필요한 고객 정보를 찾습니다.
