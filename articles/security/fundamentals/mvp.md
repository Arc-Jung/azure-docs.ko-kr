# <a name="azure-security-mvp-program"></a>Azure 보안 MVP 프로그램

Microsoft MVP(Most Valuable Professional)는 다른 사용자들을 모범적으로 지원해 온 커뮤니티 리더입니다. MVP는 Microsoft 기술과 관련된 경험을 다른 사용자들이 최대한 활용할 수 있도록 지원합니다. 열정, 실제 지식 및 기술적 전문성을 커뮤니티 및 Microsoft와 공유합니다.

Microsoft Azure에서는 Azure 보안에 특별한 전문성을 갖춘 커뮤니티 전문가들을 공인하고 있습니다. Microsoft MVP는 Azure 보안 기여 부문에서 Microsoft Azure의 MVP를 수상할 수 있습니다.

![Microsoft MVP](./media/mvp/azure-security-mvp-fig1.png)

MVP 지정을 위한 별도의 벤치마크는 없습니다. 그 이유 중 하나는 기술과 수명 주기별로 MVP 지정 기준이 달라지기 때문입니다. MVP 지정의 몇 가지 기준은 다음과 같습니다.

- Microsoft Answers, TechNet, MSDN 등의 온라인 포럼에 후보자가 기고한 콘텐츠의 영향력
- Wiki 및 온라인 콘텐츠
- 회의 및 사용자 그룹
- 팟캐스트, 웹 사이트, 블로그 및 소셜 미디어
- 문서 및 도서
