# <a name="index-of-built-in-security-controls-articles-for-azure-services"></a>Azure 서비스에 대한 "기본 제공 보안 제어" 아티클 의 인덱스

이 인덱스는 "기본 제공 보안 컨트롤" 문서에 대 한 링크를 제공 합니다. 보안 제어는 보안 취약점을 방지, 탐지 및 대응하는 서비스의 기능에 기여하는 Azure 서비스의 품질 또는 기능입니다.

기본 제공 보안 제어 문서는 다음 서비스에 사용할 수 있습니다.

- [API 관리](../../api-management/api-management-security-controls.md)
- [Azure App Service](../../app-service/app-service-security-controls.md)
- [Azure 리소스 관리자](../../azure-resource-manager/management/azure-resource-manager-security-controls.md)
- [Azure 백업](../../backup/backup-security-controls.md)
- [Azure Event Hubs](../../event-hubs/event-hubs-security-controls.md)
- [Azure ExpressRoute](../../expressroute/expressroute-security-controls.md)
- [Azure Load Balancer](../../load-balancer/load-balancer-security-controls.md)
- [Azure 서비스 버스 메시징](../../service-bus-messaging/service-bus-messaging-security-controls.md)
- [Azure Service Bus Relay](../../service-bus-relay/service-bus-relay-security-controls.md)
- [Azure 서비스 패브릭](../../service-fabric/service-fabric-security-controls.md)
- [Azure SQL Database](../../sql-database/sql-database-security-controls.md)
- [Azure Virtual Machine Scale Sets](../../virtual-machine-scale-sets/virtual-machine-scale-sets-security-controls.md)
- [리눅스 가상 머신](../../virtual-machines/linux/virtual-machines-linux-security-controls.md)
- [윈도우 가상 머신](../../virtual-machines/windows/virtual-machines-windows-security-controls.md)
- [Azure VPN 게이트웨이](../../vpn-gateway/vpn-gateway-security-controls.md)
