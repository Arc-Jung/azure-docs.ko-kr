# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Azure Security 벤치 마크 v2 및 CI 매핑 Microsoft Azure 기초 벤치 마크

CI Microsoft Azure 기초 벤치 마크는 Azure에 대 한 보안 기준 구성을 설정 하기 위해 Center for Internet Security에서 제공 하는 보안 지침입니다. 벤치 마크의 범위는 Azure Cloud를 채택 하는 동안 보안의 기본 수준을 설정 하는 것입니다. 이 벤치 마크는 [Azure Security 벤치 마크](overview.md)v 2와 연계 합니다.

[Ci Microsoft Azure 기초 벤치 마크 최신 버전 v 1.3.0](https://www.cisecurity.org/benchmark/azure/) 및 [Azure Security 벤치 마크 v2](overview.md)간의 매핑을 제공 합니다. 이렇게 하면 두 벤치 마크에 대 한 권장 구성이 정렬 되는 방식을 이해 하는 데 도움이 됩니다. [여기](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx)에서 전체 매핑을 다운로드할 수 있습니다.
