# Azure 보안 및 규정 준수 블로그

[Azure 보안 및 규정 준수 블로그](https://blogs.msdn.microsoft.com/azuresecurity/)는 Azure 보안 및 규정 준수에 대한 가장 최신 기술 정보를 확인할 수 있는 적합한 곳입니다.

또한 Azure 보안 및 규정 준수 팀 블로그에서 [Azure 보안 센터](https://azure.microsoft.com/documentation/services/security-center/)에 대한 기술 항목의 최신업데이트를 살펴볼 수도 있습니다.

기술 정보 외에도 Azure 보안 및 규정 준수 팀 블로그에서는 Azure 블로그에 나타나는 Azure 보안 및 규정 준수에 대한 최신 공지 사항을 소개하고 있습니다.

Azure 보안과 관련된 모든 사항의 최신 소식을 확인하려면 Azure 보안 및 규정 준수 팀 블로그를 확인하세요.
