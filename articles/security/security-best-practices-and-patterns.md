# Azure 보안 모범 사례 및 패턴

Azure는 솔루션을 빌드할 수 있도록 안전한 플랫폼을 제공합니다. 우리는 또한 Azure에서 더 안전한 서비스를 빌드하는 데 사용할 수 있도록 상당히 많은 서비스 및 기술을 제공합니다. 사용할 수 있는 옵션이 많기 때문에 많은 분들이 Microsoft가 Azure 배포를 위한 보안 개선의 모범 사례 및 패턴으로 추천하는 것에 관해 관심을 표명했습니다.

우리는 여러분의 관심사를 이해하며 이를 위해 적절한 상화에서 Azure 배포의 보안을 개선하는데 유용할 것이라고 믿는 여러분이 할 수 있는 일을 설명하는 문서 컬렉션을 만들었습니다.

이러한 모범 사례 및 패턴 문서에서 특정 항목에 대한 모범 사례 및 유용한 패턴 컬렉션을 설명합니다. 이러한 모범 사례 및 패턴은 이런 기술에 대한 우리 경험과 여러분 같은 고객의 경험에서 얻은 것입니다.

각 모범 사례에 대해 다음 사항을 설명하겠습니다.

- 각 모범 사례
- 해당 모범 사례를 사용해야 하는 이유
- 해당 모범 사례를 사용하지 않을 경우에 발생할 수 있는 결과
- 해당 모범 사례를 대체할 수 있는 대안
- 해당 모범 사례를 사용하는 방법을 알아보는 방법
