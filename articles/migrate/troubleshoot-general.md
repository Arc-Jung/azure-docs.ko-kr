# <a name="troubleshoot-azure-migrate"></a>Azure Migrate 문제 해결

[Azure Migrate는](migrate-services-overview.md) 평가 및 마이그레이션을 위한 도구 허브와 타사 ISV(독립 소프트웨어 공급업체) 제품을 제공합니다. 이 문서에서는 Azure 마이그레이션, Azure 마이그레이션 서버 평가 및 Azure 마이그레이션 서버 마이그레이션과 관련된 문제를 해결하는 데 도움이 됩니다.

## <a name="how-do-i-create-or-find-a-project"></a>프로젝트를 만들거나 찾으려면 어떻게 해야 합니까?

Azure [마이그레이션 프로젝트 문제 해결 가이드를](troubleshoot-project.md)검토합니다.

## <a name="i-cant-get-the-appliance-working"></a>어플라이언스가 작동하지 않습니다.

어플라이언스 [배포와](troubleshoot-appliance-discovery.md) 관련된 일반적인 문제에 대한 답변을 검토합니다.

## <a name="machines-arent-discovered"></a>컴퓨터가 검색되지 않습니다.

일반적인 검색 문제를 검토합니다.

## <a name="app-discovery-isnt-working"></a>앱 검색이 작동하지 않습니다.

온-프레미스 컴퓨터에서 실행되는 앱, 역할 및 기능 검색은 현재 VMware VM에서만 지원됩니다. 앱 검색을 위한 [일반적인 오류를 검토합니다.](troubleshoot-appliance-discovery.md#common-app-discovery-errors)

## <a name="assessment-isnt-working"></a>평가가 작동하지 않습니다.

일반적인 평가 문제 및 오류를 [검토합니다.](troubleshoot-assessment.md)
