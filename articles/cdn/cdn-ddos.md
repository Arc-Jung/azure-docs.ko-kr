# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS 보호

콘텐츠 전송 네트워크는 의도적으로 DDoS 보호를 제공합니다. Azure CDN은 볼륨 공격을 흡수하는 전역 용량 외에도 추가 비용 없이 아래에 설명된 추가 DDoS 보호 기능을 보유하고 있습니다.

## <a name="azure-cdn-from-microsoft"></a>Microsoft의 Azure CDN

마이크로 소프트의 Azure CDN은 [Azure 기본 DDoS에](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview)의해 보호됩니다. 기본적으로 추가 비용 없이 Microsoft 플랫폼의 Azure CDN에 통합됩니다. Microsoft의 전 세계에 배포된 네트워크에서 Azure CDN의 전체 규모와 용량은 상시 트래픽 모니터링 및 실시간 완화를 통해 일반적인 네트워크 계층 공격에 대한 방어를 제공합니다. 기본 DDoS 보호는 CDN 끝점을 대상으로 하는 가장 일반적이고 자주 발생하는 계층 7 DNS 쿼리 플러드 및 계층 3 및 4 대용량 공격도 방어합니다. 또한 이 서비스는 대규모 공격으로부터 Microsoft의 엔터프라이즈 및 소비자 서비스를 보호하는 데 있어 입증된 실적을 보유하고 있습니다.

## <a name="azure-cdn-from-verizon"></a>Verizon의 Azure CDN

버라이존의 Azure CDN은 Verzion의 독자적인 DDoS 완화 플랫폼에 의해 보호됩니다. 기본적으로 버라이존의 Azure CDN에 통합되어 있으며 추가 비용없이 통합됩니다. CDN 끝점을 대상으로 하는 가장 일반적이고 자주 발생하는 계층 7 DNS 쿼리 플러드 및 계층 3 및 4 대권 공격에 대한 기본 보호 기능을 제공합니다.

## <a name="azure-cdn-from-akamai"></a>Akamai의 Azure CDN

Akamai의 Azure CDN은 Akamai의 독자적인 DDoS 완화 플랫폼으로 보호됩니다. 기본적으로 Akamai의 Azure CDN에 통합되어 있으며 추가 비용없이 통합됩니다. CDN 끝점을 대상으로 하는 가장 일반적이고 자주 발생하는 계층 7 DNS 쿼리 플러드 및 계층 3 및 4 대권 공격에 대한 기본 보호 기능을 제공합니다.
