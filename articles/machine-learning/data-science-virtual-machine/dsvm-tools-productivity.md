# <a name="productivity-tools-on-the-data-science-virtual-machine"></a>Data Science Virtual Machine의 생산성 도구

DSVM에는 데이터 과학 및 프로그래밍 도구 외에도 동료와의 통찰력을 캡처하고 공유 하는 데 도움이 되는 생산성 도구가 포함 되어 있습니다. Office 365 ProPlus는 기업에 가장 생산적이 고 안전한 Office 환경으로, 팀이 언제 어디서 나 원활 하 게 공동 작업할 수 있습니다. Power BI Desktop를 사용 하 여 데이터에서 정보로 이동 하 여 작업을 수행할 수 있습니다. 그리고 Microsoft Edge 브라우저는 최신의 빠르고 안전한 웹 브라우저입니다.
