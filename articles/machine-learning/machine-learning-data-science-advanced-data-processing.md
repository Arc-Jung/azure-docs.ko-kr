# 학습 가이드: Azure에서 고급 데이터 처리 

이 맵에서는 Azure 기계 학습 실험에서 사용할 데이터를 준비하는 프로세스를 안내합니다. 기계 학습을 준비하기 위해 데이터를 이동, 처리 및 샘플링하는 방법을 보여 줍니다. 여기에 설명된 데이터 준비 과정은 [Azure에서 고급 분석 솔루션 빌드](machine-learning-data-science-how-to-create-machine-learning-service.md)에서 제공된 맵의 단계입니다. ADAPT 가이드에서는 일반적으로 Azure 기계 학습으로 예측 모델 빌드 및 배포와 관련된 데이터 과학 작업을 간략하게 설명합니다.
