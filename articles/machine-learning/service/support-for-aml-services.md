## <a name="get-support-for-azure-machine-learning-service"></a>Azure Machine Learning 서비스에 대한 지원 받기

다음 지원 리소스를 확인하세요.

+ **기술 지원**: [Azure 기술 지원](https://azure.microsoft.com/support/options/)을 방문하여 Machine Learning을 선택합니다. 

+ **사용자 포럼**: [MSDN의 Azure Machine Learning 서비스 지원 포럼](https://aka.ms/aml-forum-service)에서 질문하고 답변하고 다른 사용자와 연락할 수 있습니다.

+ **스택 오버플로**: “Azure-Machine-Learning” 태그가 지정되어 있는 [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning)의 Azure Machine Learning 커뮤니티를 방문해보세요.

+ 각 문서의 맨 아래에 있는 링크를 사용하여 액세스할 수 있는 [Azure Machine Learning 피드백 채널](https://feedback.azure.com/forums/257792-machine-learning)에서 기능 요청 및 **제품 제안을 공유**합니다.
