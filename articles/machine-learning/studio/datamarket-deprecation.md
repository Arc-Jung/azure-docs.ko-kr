# <a name="datamarket-is-now-deprecated"></a>DataMarket은 이제 더 이상 사용되지 않습니다.

[Microsoft DataMarket](https://datamarket.azure.com/datasets)은 이제 더 이상 사용되지 않으므로 거기에 호스트된 Machine Learning API도 더 이상 사용되지 않습니다.

이제 이러한 기능(및 기타 기능)은 [Cognitive Services API](https://www.microsoft.com/cognitive-services)에서 찾을 수 있습니다.

## <a name="text-analytics-api"></a>Text Analytics API

이전 DataMarket Text Analytics API 고객은 [Cognitive Services Text Analytics API](https://www.microsoft.com/cognitive-services/text-analytics-api)를 사용해야 합니다.
Cognitive Services Text Analytics API는 이전 DataMarket 버전에서 수행한 모든 시나리오와 언어 감지 및 토픽 감지와 같은 추가 시나리오를 지원합니다.
