+ **Azure Machine Learning 피드백 채널**에서 [제품 제안 및 기능 요청을 공유](https://feedback.azure.com/forums/257792-machine-learning)합니다. 피드백을 공유하려면 각 문서의 끝에 있는 **제품 피드백** 단추를 선택합니다.
