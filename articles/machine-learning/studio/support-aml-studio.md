## <a name="get-service-support"></a>서비스 지원 받기

다음 지원 리소스를 확인하세요.

+ **Azure 고객을 위한 기술 지원**: Azure Portal을 통해 [지원 요청을 제출하고 관리](https://docs.microsoft.com/azure/azure-supportability/how-to-create-azure-support-request)합니다.

+ **사용자 포럼**: [MSDN의 Azure Machine Learning Studio 지원 포럼](https://aka.ms/aml-forum-studio)에서 질문하고 답변하고 다른 사용자와 연락할 수 있습니다.

+ **Stack Overflow**: “Azure-Machine-Learning” 태그가 지정되어 있는 [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning)의 Azure Machine Learning 커뮤니티를 방문해보세요.

+ [Azure Machine Learning 피드백 채널](https://feedback.azure.com/forums/257792-machine-learning)에서 **제품 제안 및 기능 요청을 공유**합니다. 피드백을 공유하려면 각 문서의 끝에 있는 **제품 피드백** 단추를 선택합니다.
