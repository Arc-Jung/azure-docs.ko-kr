# <a name="download-notebookspreview-data"></a>노트북 (미리 보기) 데이터 다운로드

노트북 (미리 보기) 기능은 *4 월 13 2020*일에 제거 될 예정입니다. 이 날짜 후에는 노트북 (미리 보기) 탭이 사라지고 노트북 데이터를 복원할 수 없습니다. 4 월 13 2020 일 전에 노트북 데이터를 다운로드 합니다.

이 문서에서는 노트북 (미리 보기) 데이터를 다운로드 하는 방법에 대 한 단계별 지침을 제공 합니다.

1. Azure Machine Learning Studio (클래식)에서 **노트북** 탭으로 이동 합니다.
    ![노트북 탭으로 이동 합니다.](./media/download-notebooks/notebooks-list.png)

1. 다운로드 하려는 노트북을 선택 합니다. Jupyter에서 열립니다.
    한 번에 여러 전자 필기장을 다운로드 하는](./media/download-notebooks/select-notebook.png) 노트북을 선택할 ![지원 되지 않습니다.

1. **파일** -> **다운로드**로 이동 하 여 서식 옵션을 선택 합니다.
    ![노트북 다운로드 옵션](./media/download-notebooks/download-options.PNG)

1. 옵션을 선택한 후에는 전자 필기장 파일이 다운로드 되기 시작 합니다. 표시 되는 500 또는 502 오류는 무시 합니다. 다운로드에는 영향을 주지 않습니다.
