# <a name="deep-learning-vs-machine-learning"></a>딥 러닝 과 기계 학습

이 문서에서는 딥 러닝과 기계 학습을 비교하는 데 도움이 됩니다. 두 개념을 비교하는 방법과 인공 지능의 광범위한 범주에 어떻게 부합하는지 알아봅니다. 또한 사기 탐지, 음성 및 얼굴 인식, 감정 분석 및 열렬 예측과 같은 실제 시나리오에 딥 러닝을 적용하는 방법에 대해서도 설명합니다.

## <a name="deep-learning-machine-learning-and-ai"></a>딥 러닝, 머신 러닝 및 AI

![관계 다이어그램: AI 와 머신 러닝, 딥 러닝](./media/concept-deep-learning-vs-machine-learning/ai-vs-machine-learning-vs-deep-learning.png)

딥 러닝과 AI를 이해하기 위해 다음 정의를 고려하십시오.

- **딥 러닝은** 인공 신경망을 기반으로 하는 기계 학습의 하위 집합입니다. 인공 신경망의 구조는 여러 입력, 출력 및 숨겨진 계층으로 구성되므로 _학습 과정은_ _깊습니다._ 각 레이어에는 입력 데이터를 다음 레이어가 특정 예측 작업에 사용할 수 있는 정보로 변환하는 단위가 포함되어 있습니다. 이 구조 덕분에 기계는 자체 데이터 처리를 통해 학습할 수 있습니다.

- **머신 러닝은** 기계가 경험을 사용하여 작업을 개선할 수 있도록 하는 기술(예: 딥 러닝)을 사용하는 인공 지능의 하위 집합입니다. _학습 과정은_ 다음 단계를 기반으로 합니다.

   1. 데이터를 알고리즘에 공급합니다. 이 단계에서는 피처 추출을 수행하여 모델에 추가 정보를 제공할 수 있습니다.
   1. 이 데이터를 사용하여 모델을 학습시킵니다.
   1. 모델을 테스트한 후 배포합니다.
   1. 배포된 모델을 사용하여 자동화된 예측 작업을 수행합니다. 즉, 배포된 모델을 호출하고 사용하여 모델에서 반환된 예측을 수신합니다.

- <bpt i="1000001" x="1000001" type="formatting">{b&gt;</bpt>AI(인공 지능)<ept i="1000001">&lt;b}</ept>는 컴퓨터가 인간 지능을 모방할 수 있도록 하는 기술입니다. 여기에는 기계 학습이 포함됩니다. 
 
AI, 머신 러닝 및 딥 러닝 간의 관계를 이해하는 것이 중요합니다. 기계 학습은 인공 지능을 달성하는 방법입니다. 기계 학습 및 딥 러닝 기술을 사용하여 일반적으로 인간의 지능과 관련된 작업을 수행하는 컴퓨터 시스템 및 응용 프로그램을 빌드할 수 있습니다. 이러한 작업에는 이미지 인식, 음성 인식 및 언어 번역이 포함됩니다.

## <a name="techniques-of-deep-learning-vs-machine-learning"></a>딥 러닝 과 기계 학습의 기술 

이제 기계 학습과 딥 러닝에 대한 개요를 살펴보겠습니다. 기계 학습에서 알고리즘은 더 많은 정보를 사용하여 정확한 예측을 수행하는 방법을 설명해야 합니다(예: 기능 추출 수행). 딥 러닝에서 알고리즘은 인공 신경망 구조 덕분에 자체 데이터 처리를 통해 정확한 예측을 만드는 방법을 배울 수 있습니다.

다음 표에서는 두 기술을 자세히 비교합니다.

| |모든 기계 학습 |딥 러닝만|
|---|---|---|
|  **데이터 소수점 수** | 적은 양의 데이터를 사용하여 예측을 할 수 있습니다. | 예측을 위해 많은 양의 학습 데이터를 사용해야 합니다. |
|  **하드웨어 종속성** | 저가형 기계에서 작동할 수 있습니다. 많은 양의 계산 능력이 필요하지 않습니다. | 하이 엔드 기계에 따라 다릅니다. 본질적으로 많은 수의 행렬 곱셈 작업을 수행합니다. GPU는 이러한 작업을 효율적으로 최적화할 수 있습니다. |
|  **위화 과정** | 사용자가 기능을 정확하게 식별하고 만들어야 합니다. | 데이터에서 높은 수준의 기능을 학습하고 자체적으로 새로운 기능을 만듭니다. |
|  **학습 접근 방식** | 학습 프로세스를 더 작은 단계로 나눕니다. 그런 다음 각 단계의 결과를 하나의 출력으로 결합합니다. | 종단 간 문제를 해결하여 학습 프로세스를 진행합니다. |
|  **실행 시간** | 몇 초에서 몇 시간에 이르기까지, 훈련하는 데 비교적 적은 시간이 걸립니다. | 딥 러닝 알고리즘에는 여러 계층이 포함되므로 일반적으로 학습하는 데 시간이 오래 걸립니다. |
|  **출력** | 출력은 일반적으로 점수 나 분류와 같은 숫자 값입니다. | 출력에는 텍스트, 점수 또는 사운드와 같은 여러 형식이 있을 수 있습니다. |

## <a name="deep-learning-use-cases"></a>딥 러닝 사용 사례

인공 신경망 구조로 인해 딥 러닝은 이미지, 사운드, 비디오 및 텍스트와 같은 비정형 데이터에서 패턴을 식별하는 데 탁월합니다. 이러한 이유로 딥 러닝은 의료, 에너지, 금융 및 운송을 포함한 많은 산업을 빠르게 변화시키고 있습니다. 이러한 산업은 이제 기존의 비즈니스 프로세스를 재고하고 있습니다. 

딥 러닝을 위한 가장 일반적인 응용 프로그램 중 일부는 다음 단락에 설명되어 있습니다.

### <a name="named-entity-recognition"></a>명명된 엔터티 인식

명명된 엔터티 인식은 텍스트를 입력으로 사용하여 미리 지정된 클래스로 변환하는 딥 러닝 방법입니다. 이 새 정보는 우편 번호, 날짜, 제품 ID일 수 있습니다. 그런 다음 정보를 구조화된 스키마에 저장하여 주소 목록을 작성하거나 ID 유효성 검사 엔진의 벤치마크 역할을 할 수 있습니다.

### <a name="object-detection"></a>개체 감지

딥 러닝은 많은 개체 감지 사용 사례에 적용되었습니다. 개체 감지는 이미지 분류및 이미지 지역화의 두 부분으로 구성됩니다. 이미지 _분류는_ 자동차 나 사람과 같은 이미지의 개체를 식별합니다. 이미지 _지역화는_ 이러한 개체의 특정 위치를 제공합니다. 

객체 감지는 이미 게임, 소매, 관광 및 자율 주행 자동차와 같은 산업에서 사용됩니다.

### <a name="image-caption-generation"></a>이미지 캡션 생성

이미지 인식과 마찬가지로 이미지 캡션에서 지정된 이미지의 경우 시스템은 이미지의 내용을 설명하는 캡션을 생성해야 합니다. 사진에서 물체를 감지하고 레이블을 지정할 수 있는 경우 다음 단계는 해당 레이블을 설명적인 문장으로 바꾸는 것입니다. 

일반적으로 이미지 캡션 응용 프로그램은 컨볼루션 신경망을 사용하여 이미지의 개체를 식별한 다음 반복 신경망을 사용하여 레이블을 일관된 문장으로 바뀝니다.

### <a name="machine-translation"></a>기계 번역

기계 번역은 한 언어의 단어나 문장을 가져와 자동으로 다른 언어로 번역합니다. 기계 번역은 오랜 시간 동안 사용되어 왔지만 딥 러닝은 텍스트 자동 번역(텍스트 로 음성 번역) 및 이미지 자동 번역이라는 두 가지 특정 영역에서 인상적인 결과를 얻을 수 있습니다.

적절한 데이터 변환을 통해 신경망은 텍스트, 오디오 및 시각적 신호를 이해할 수 있습니다. 기계 번역은 더 큰 오디오 파일에서 사운드 의 조각을 식별하고 텍스트로 음성 단어 또는 이미지를 전사하는 데 사용할 수 있습니다.

### <a name="text-analytics"></a>텍스트 분석

딥 러닝 방법을 기반으로 하는 텍스트 분석에는 대량의 텍스트 데이터(예: 의료 문서 또는 비용 영수증)를 분석하고 패턴을 인식하고 조직적이고 간결한 정보를 만드는 작업이 포함됩니다.

기업은 딥 러닝을 사용하여 텍스트 분석을 수행하여 내부자 거래 및 정부 규정 준수를 감지합니다. 또 다른 일반적인 예는 보험 사기입니다: 텍스트 분석은 종종 보험 청구가 사기일 가능성을 인식하기 위해 많은 양의 문서를 분석하는 데 사용되었습니다. 

## <a name="artificial-neural-networks"></a>인공 신경망

인공 신경망은 연결된 노드계층에 의해 형성됩니다. 딥 러닝 모델은 많은 수의 계층이 있는 신경망을 사용합니다. 

다음 섹션에서는 가장 인기 있는 인공 신경망 유형극을 살펴봅니다.

### <a name="feedforward-neural-network"></a>피드포워드 신경망

피드포워드 신경망은 가장 기본적인 유형의 인공 신경망입니다. 피드포워드 네트워크에서 정보는 입력 계층에서 출력 계층으로 한 방향으로만 이동합니다. 피드포워드 신경망은 일련의 숨겨진 계층을 통해 입력을 변환합니다. 모든 층은 뉴런 세트로 구성되며, 각 층은 이전에 레이어의 모든 뉴런에 완전히 연결됩니다. 마지막으로 완전히 연결된 레이어(출력 계층)는 생성된 예측을 나타냅니다.

### <a name="recurrent-neural-network"></a>재발신경망

재발성 신경망은 널리 사용되는 인공 신경망입니다. 이러한 네트워크는 레이어의 출력을 저장하고 레이어의 결과를 예측하는 데 도움이 되는 입력 계층에 다시 공급합니다. 재발신경망은 훌륭한 학습 능력을 가지고 있습니다. 그들은 널리 타임 시리즈 예측, 필기 학습 및 언어 인식과 같은 복잡한 작업에 사용됩니다.

### <a name="convolutional-neural-networks"></a>컨볼루션 신경망

컨볼루션 신경망은 특히 효과적인 인공 신경망이며 고유한 아키텍처를 제공합니다. 레이어는 너비, 높이 및 깊이의 세 가지 차원으로 구성됩니다. 한 층의 뉴런은 다음 레이어의 모든 뉴런에 연결되지 않고 레이어의 뉴런의 작은 영역에만 연결됩니다. 최종 출력은 깊이 차원을 따라 구성된 확률 점수의 단일 벡터로 줄어듭니다. 

컨볼루션 신경망은 비디오 인식, 이미지 인식 및 추천 시스템과 같은 분야에서 사용되어 왔습니다.
