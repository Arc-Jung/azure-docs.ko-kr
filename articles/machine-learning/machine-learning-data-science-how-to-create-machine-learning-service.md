# Azure에서 고급 분석 솔루션 구축

이 ADAPT(고급 분석 프로세스 및 기술) 가이드에서는 일반적으로 Azure 기계 학습으로 예측 모델 빌드 및 배포와 관련된 데이터 과학 작업을 간략하게 설명합니다. 또한 Azure 플랫폼에서 다양한 소스에서 데이터를 수집하고, 수집한 데이터를 Azure 기계 학습에 사용할 수 있도록 준비하고, Azure 기계 학습 실험을 사용하여 최종 사용자 응용 프로그램에서 프로그래밍 방식 또는 기타 방식으로 사용할 수 있는 조작 가능한 모델을 만드는 등의 작업을 수행하는 방법을 보여 줍니다. 맵에서 보여 주는 단계는 일반적인 종단 간 데이터 과학에 관련된 일련의 핵심 단계입니다. 모든 단계가 반드시 필요한 것은 아니며 데이터의 위치, 크기 및 복잡성에 따라 정확한 순서가 달라질 수 있습니다.

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

**Azure에서 고급 데이터 처리**라는 단계는 Azure 기계 학습 실습에서 사용할 데이터를 준비하는 과정을 안내하는 두 번째 가이드에 연결됩니다. 이 가이드를 마친 경우 Azure 기계 학습 스튜디오에서 처리된 데이터를 수집하는 이 가이드의 후속 단계부터 모델을 사용하는 최종 단계까지 계속 진행합니다.
