# <a name="remove-duplicate-rows-module"></a>중복 행 제거 모듈

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)의 모듈에 대해 설명합니다.

이 모듈을 사용하여 데이터 집합에서 잠재적인 중복을 제거합니다.

예를 들어 데이터가 다음과 같이 보이고 환자에 대한 여러 레코드를 나타낸다고 가정합니다. 

| 참을성 있는 ID | Initials| 성별|Age|인정|
|----|----|----|----|----|
|1|오후| M| 53| 1월|
|2| F.A.M.| M| 53| 1월|
|3| F.A.M.| M| 24| 1월|
|3| 오후| M| 24| 2월|
|4| 오후| M| 23| 2월|
| | 오후| M| 23| |
|5| F.A.M.| M| 53| |
|6| F.A.M.| M| NaN| |
|7| F.A.M.| M| NaN| |

이 예제에는 중복 데이터가 있는 여러 열이 있습니다. 실제로 중복되는지 여부는 데이터에 대한 지식에 따라 다릅니다. 

+ 예를 들어, 많은 환자가 같은 이름을 가지고 있음을 알 수 있습니다. 이름 열을 사용하여 중복을 제거하지는 않을 뿐 아니라 **ID** 열만 제거합니다. 이렇게 하면 환자가 이름이 같은지 여부에 관계없이 중복 ID 값이 있는 행만 필터링됩니다.

+ 또는 ID 필드에서 중복을 허용하고 파일의 다른 조합을 사용하여 이름, 성, 나이 및 성별과 같은 고유한 레코드를 찾을 수 있습니다.  

행이 중복되는지 여부에 대한 조건을 설정하려면 **키로**사용할 단일 열 또는 열 집합을 지정합니다. 두 행은 **모든** 키 열의 값이 같을 때만 중복된 행으로 간주됩니다. 행에 **대한**값이 누락된 경우 중복 행으로 간주되지 않습니다. 예를 들어 위의 표에서 성별과 나이가 키로 설정된 경우 6행과 7행은 Age에서 값이 누락된 경우 중복행이 아닙니다.

모듈을 실행하면 후보 데이터 집합이 생성되고 지정한 열 집합에 중복이 없는 행 집합을 반환합니다.

> [!IMPORTANT]
> 원본 데이터 집합은 변경되지 않습니다. 이 모듈은 지정한 기준에 따라 중복을 제외하도록 필터링된 새 데이터 집합을 만듭니다.

## <a name="how-to-use-remove-duplicate-rows"></a>중복 행 제거 사용 방법

1. 파이프라인에 모듈을 추가합니다. **데이터 변환,** **조작**에서 중복 **행 제거** 모듈을 찾을 수 있습니다.  

2. 중복 행을 검사할 데이터 집합을 연결합니다.

3. **속성** 창에서 키 **열 선택 필터 표현식**에서 **시작 열 선택기를**클릭하여 중복 을 식별하는 데 사용할 열을 선택합니다.

    이 컨텍스트에서 **Key는** 고유 식별자를 의미하지 않습니다. 열 선택기에서 선택한 모든 열이 **키 열로 지정됩니다.** 선택되지 않은 모든 열은 키가 아닌 열로 간주됩니다. 키로 선택하는 열의 조합에 따라 레코드의 고유성이 결정됩니다. (여러 등가 조인을 사용하는 SQL 문으로 생각하십시오.)

    예제:

    + "ID가 고유하도록 하고 싶습니다": ID 열만 선택합니다.
    + "이름, 성, ID의 조합이 고유한지 확인하고 싶습니다.": 세 개의 열을 모두 선택합니다.

4. 첫 **번째 중복 행 유지** 확인란을 사용하여 중복이 발견될 때 반환할 행을 나타냅니다.

    + 이 옵션을 선택하면 첫 번째 행이 반환되고 다른 행은 삭제됩니다. 
    + 이 옵션을 선택 취소하면 마지막 중복 행이 결과에 유지되고 다른 행은 삭제됩니다. 

5. 파이프라인을 제출합니다.

6. 결과를 검토하려면 모듈을 마우스 오른쪽 단추로 클릭하고 **시각화를**선택합니다. 

> [!TIP]
> 결과를 이해하기 어렵거나 일부 열을 고려 대상에서 제외하려는 경우 [데이터 집합의 열 선택](./select-columns-in-dataset.md) 모듈을 사용하여 열을 제거할 수 있습니다.
