# <a name="remove-duplicate-rows-module"></a>모듈 중복 행 제거

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

데이터 집합에서 잠재적인 중복을 제거 하려면이 모듈을 사용 합니다.

예를 들어 데이터는 다음과 같은 및 환자에 대 한 여러 레코드를 나타냅니다. 

| PatientID | Initials| 성별|Age|연결|
|----|----|----|----|----|
|1|F.M.| M| 53| Jan|
|2| F.A.M.| M| 53| Jan|
|3| F.A.M.| M| 24| Jan|
|3| F.M.| M| 24| Feb|
|4| F.M.| M| 23| Feb|
| | F.M.| M| 23| |
|5| F.A.M.| M| 53| |
|6| F.A.M.| M| NaN| |
|7| F.A.M.| M| NaN| |

물론이 예에서는 잠재적으로 중복 데이터를 사용 하 여 여러 열이 있습니다. 실제로 중복 된 여부는 데이터의 지식에 따라 달라 집니다. 

+ 예를 들어 게 많은 환자가 이름이 같은 알 수 있습니다. 이름 열에만 사용 하 여 중복을 제거 하지 않습니다.는 **ID** 열입니다. 이런 방식으로 환자가 이름이 없거나 여부에 관계 없이, 중복 ID 값을 가진 행만 필터링 됩니다.

+ 또는, ID 필드에 중복을 허용 하 고 몇 가지 다른 조합 파일의 이름, 마지막 이름, 연령 및 성별 등의 고유 레코드를 찾는 데 사용 하기로 결정할 수 있습니다.  

단일 열 또는으로 사용할 열의 집합 지정 조건을 행 중복 인지 여부를 설정 하려면 **키**합니다. 두 행에는 중복으로 간주 됩니다 경우에만 값 **모든** 키 열이 같음. 모든 행에 대 한 누락 된 값이 있으면 **키**, 이러한 고려 되지 것입니다 중복 행. 예를 들어, 성별 및 연령 테이블, 6 및 7 행 위에 키가 지정 된 중복 행으로 설정 되어 있는지 누락 값 시대에 있습니다.

모듈을 실행 하면 후보 데이터 집합을 만들고 지정 된 열 집합에서 중복 행이 있는 행 집합을 반환 합니다.

> [!IMPORTANT]
> 원본 데이터 집합은 변경 되지 않습니다. 이 모듈은 사용자가 지정한 조건에 따라 중복 항목을 제외 하도록 필터링 된 새 데이터 집합을 만듭니다.

## <a name="how-to-use-remove-duplicate-rows"></a>중복 행 제거를 사용 하는 방법

1. 모듈을 실험에 추가 합니다. 찾을 수 있습니다는 **중복 행 제거** 아래의 모듈 **데이터 변환**를 **조작**합니다.  

2. 중복 된 행에 대해 확인 하려는 데이터 집합을 연결 합니다.

3. **속성** 창 아래에 있는 **키 열 선택 필터 식**, 클릭 **열 선택기 시작**, 중복 항목 식별에 사용할 열을 선택 합니다.

    이 컨텍스트에서 **키** 고유 식별자를 의미 하지는 않습니다. 열 선택기를 사용 하 여 선택한 모든 열으로 지정 된 **키 열**합니다. 선택 되지 않은 모든 열에는 키가 아닌 열으로 간주 됩니다. 키로 선택 하는 열 조합 레코드의 고유성을 확인 합니다. (생각할 여러 정확할 조인을 사용 하는 SQL 문으로 있습니다.)

    예제:

    + ""을 Id가 고유한 지 확인 합니다. ID 열만 선택 합니다.
    + ""을 적용 하도록 첫 번째 이름, 성, 이름 및 ID 조합이 고유한 지 확인 하십시오. 모든 세 개의 열을 선택 합니다.

4. 사용 된 **첫 번째 중복 행을 보관** 중복이 발견 될 때 반환 하도록 행을 표시 하려면이 확인란을:

    + 첫 번째 행이 반환을 선택 하 고 다른 사용자가 삭제 됩니다. 
    + 이 옵션을 선택 취소 하는 경우에 마지막 중복 행을 결과에 유지 되 고 다른 사용자가 삭제 됩니다. 

5. 실험을 실행합니다.

6. 결과 검토 하려면 모듈을 마우스 오른쪽 단추로 클릭, 선택 **결과 데이터 집합**, 클릭 **시각화**합니다. 

> [!TIP]
> 결과 이해 하기 어려운 아니면 고려 사항에서 일부 열을 제외 하려는 경우 사용 하 여 열을 제거 합니다 [Select Columns in Dataset](./select-columns-in-dataset.md) 모듈입니다.
