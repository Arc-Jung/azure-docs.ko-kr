# <a name="module-assign-data-to-clusters"></a>모듈: 클러스터에 데이터 할당

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)에서 *클러스터에 데이터 할당 모듈을* 사용하는 방법을 설명합니다. 이 모듈은 *K-평균 클러스터링* 알고리즘으로 학습된 클러스터링 모델을 통해 예측을 생성합니다.

클러스터에 데이터 할당 모듈은 각 새 데이터 포인트에 대해 가능한 할당을 포함하는 데이터 집합을 반환합니다. 

## <a name="how-to-use-assign-data-to-clusters"></a>클러스터에 데이터 할당을 사용하는 방법
  
1. Azure 기계 학습 디자이너에서 이전에 학습된 클러스터링 모델을 찾습니다. 다음 방법 중 하나를 사용하여 클러스터링 모델을 만들고 학습할 수 있습니다.  
  
    - K-평균 클러스터링 모듈을 사용하여 [K-평균 클러스터링](k-means-clustering.md) 알고리즘을 구성하고 데이터 집합 및 클러스터링 모델 학습 모듈(이 문서)을 사용하여 모델을 학습합니다.  
  
    - 작업 공간에 **저장된 모델** 그룹에서 학습된 기존 클러스터링 모델을 추가할 수도 있습니다.

2. 학습된 모델을 클러스터에 데이터 할당의 왼쪽 입력 **포트에 연결합니다.**  

3. 새 데이터 집합을 입력으로 연결합니다. 

   이 데이터 집합에서 레이블은 선택 사항입니다. 일반적으로 클러스터링은 자율 학습 방법입니다. 범주를 미리 알 수 없습니다. 그러나 입력 열은 클러스터링 모델을 학습하는 데 사용된 열과 같아야 하거나 오류가 발생합니다.

    > [!TIP]
    > 클러스터 예측에서 디자이너에게 기록되는 열 수를 줄이려면 [데이터 집합의 선택 열을](select-columns-in-dataset.md)사용하고 열의 하위 집합을 선택합니다. 
    
4. 결과에 결과(클러스터 할당)가 표시되는 열을 포함하여 전체 입력 데이터 집합을 포함하도록 하려는 경우 결과에 대해서만 선택된 결과 확인란을 추가확인서 를 그대로 **두거나 선택을 취소합니다.**
  
    이 확인란을 선택 취소하면 결과만 반환됩니다. 이 옵션은 웹 서비스의 일부로 예측을 만들 때 유용할 수 있습니다.
  
5.  파이프라인을 제출합니다.  
  
### <a name="results"></a>결과

+  데이터 집합의 값을 보려면 모듈을 마우스 오른쪽 단추로 클릭한 다음 **시각화를**선택합니다. 또는 모듈을 선택하고 오른쪽 패널의 **출력** 탭으로 전환하고 포트 출력의 히스토그램 아이콘을 클릭하여 결과를 **시각화합니다.**
