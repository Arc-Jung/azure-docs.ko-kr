# <a name="edit-metadata-module"></a>메타 데이터 모듈 편집

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

이 모듈을 사용 하 여 데이터 집합의 열과 연결 된 메타 데이터를 변경 합니다. 사용한 후 데이터 집합의 값 및 데이터 형식은 변경 되는 **메타 데이터 편집** 모듈입니다. 
 
일반적인 메타 데이터 변경 내용이 포함 될 수 있습니다.
  
+ 범주 값으로 부울 또는 숫자 열을 처리 하는 방법  
  
+ 포함 된 열 표시 합니다 *클래스* 레이블이나 또는 범주화 하거나 예측 하려는 값  
  
+ 열을 기능으로 표시
  
+ 숫자 값 또는 그 반대로 값 날짜/시간 변경  
  
+ 열 이름 바꾸기
  
 사용 [메타 데이터 편집 다운스트림 모듈에 대 한 요구 사항에 맞게 일반적으로 열의 정의 수정 하려면 언제 든 지 합니다. 예를 들어 일부 모듈 수 특정 데이터 형식 에서만 작동 하거나 열에 대 한 플래그와 같은 필요한 `IsFeature` 또는 `IsCategorical`합니다.  
  
 필요한 작업을 수행한 후 메타 데이터를 원래 상태로 재설정할 수 있습니다. 
  
## <a name="configure-edit-metadata"></a>메타 데이터 편집 구성
  
1.  Azure Machine Learning에서 추가 [메타 데이터 편집](./edit-metadata.md) 모듈을 실험 하 고 업데이트 하려는 데이터 집합을 연결 합니다. 아래에서 찾을 수 있습니다 **데이터 변환**를 **조작** 범주입니다.
  
2.  클릭 **열 선택기 시작** 열 또는 열을 사용 하 여 작업 집합을 선택 합니다. 이름 또는 인덱스 별로 열을 개별적으로 선택할 수 있습니다 또는 형식으로 열 그룹을 선택할 수 있습니다.  
  
3.  선택 된 **데이터 형식** 선택한 열에 다른 데이터 형식을 할당 해야 하는 경우 옵션입니다. 데이터 변경 형식이 필요할 수 있습니다 특정 작업에 대 한 합니다: 예를 들어, 원본 데이터 집합에 숫자를 텍스트로 처리 하는 경우 변경 해야 해당 숫자 데이터 형식으로 수학 연산에 사용 하기 전에 합니다. 

    + 지원 되는 데이터 형식은 `String`, `Integer`, `Double`, `Boolean`, `DateTime`. 

    + 여러 열을 선택 하면 메타 데이터 변경 내용을 적용 해야 합니다 **모든** 열을 선택 합니다. 예를 들어 2-3 숫자 열을 선택 하면 가정해 보겠습니다. 문자열 데이터 형식에 모든 변경 하 고 하나의 작업으로 바꿀 수 없습니다. 그러나 변경할 수 없습니다 문자열 데이터 형식에 하나의 열과 다른 열 부동 소수점에서 정수입니다.
  
    + 새 데이터 형식을 지정 하지 않으면, 열 메타 데이터를 변경 되지 않습니다. 
    
    + 열 유형 및 값을 변경할 수는 후 수행 합니다 [메타 데이터 편집](./edit-metadata.md) 작업 합니다. 사용 하 여 언제 든 지 원래 데이터 형식을 복구할 수 있습니다 [메타 데이터 편집](./edit-metadata.md) 에 열 데이터 형식을 다시 설정 합니다.  

    > [!NOTE]
    > 모든 형식의 숫자를 변경 하는 경우는 **날짜/시간** 입력을 합니다 **날짜/시간 형식** 필드를 비워 합니다. 현재, 대상 데이터 형식을 지정할 수는 없습니다.  

      
4.  선택 된 **범주별** 범주로 선택된 된 열에서 값을 처리 해야 함을 지정 하는 옵션입니다. 

    예를 들어 0, 1 및 2, 숫자가 포함 된 열이 있는 하지만 숫자 "Smoker", "비-smoker" 및 "Unknown" 실제로 의미는 알 수 있습니다. 이 경우 열을 범주로 플래그를 지정 하 여 숫자 계산에 데이터를 그룹화에만 값은 사용 되지 않음을 확인할 수 있습니다. 
  
5.  사용 된 **필드** Azure Machine Learning 모델의 데이터를 사용 하는 방법을 변경 하려는 경우 옵션입니다.

    + **기능**: 열에 플래그를 사용 하 여 기능 열 에서만 작동 하는 모듈에 대 한 기능으로 하려면이 옵션을 사용 합니다. 기본적으로 모든 열은 처음에 기능으로 처리 됩니다.  
  
    + **레이블**: 레이블 (예측 가능한 특성 또는 대상 변수 라고도 함)를 표시 하려면이 옵션을 사용 합니다. 대부분의 모듈 중 하나 이상이 (및 하나만) 필요한 레이블 열이 데이터 집합에 있어야 합니다. 
    
        대부분의 경우에서 열 클래스 레이블을 포함 하지만이 메타 데이터를 설정 하 여 확실히 열 올바르게 식별 되는 Azure Machine Learning 유추할 수 있습니다. 이 옵션을 설정 값을 변경 하지 데이터, 일부 기계 학습 알고리즘은 데이터를 처리 하는 방법만 합니다.
  

  
    > [!TIP]
    >  이러한 범주에 맞지 않는 데이터가 있나요?  예를 들어 데이터 집합 변수로 도움이 되지 않는 고유 식별자와 같은 값을 포함할 수 있습니다. 경우에 따라 Id에는 모델에 사용 하는 경우 문제가 발생할 수 있습니다. 
    >   
    >  다행히 "에서" Azure Machine Learning 유지 모든 데이터를 하므로 데이터 집합에서 이러한 열을 삭제할 필요가 없습니다. 일부 특수 한 열 집합에 대 한 작업을 수행 해야 할 때만 다른 모든 열 일시적으로 사용 하 여 제거 합니다 [Select Columns in Dataset](./select-columns-in-dataset.md) 모듈입니다. 사용 하 여 데이터 집합에 다시 열을 병합할 수는 나중에 [열 추가](./add-columns.md) 모듈입니다.  
  
6. 이전에 선택한 항목의 선택을 취소 한 기본값으로 복원 메타 데이터를 다음 옵션을 사용 합니다.  
  
    + **기능 지우기**: 기능 플래그를 제거 하려면이 옵션을 사용 합니다.  
  
         모든 열을 수치 연산을 수행 하는 모듈에 대 한 기능으로 처음 처리 하므로 숫자 열 변수로 처리 하지 않도록 하려면이 옵션을 사용 해야 합니다.
  
    + **레이블 지우기**: 제거 하려면이 옵션을 사용 합니다 **레이블** 지정된 된 열에서 메타 데이터입니다.  
  
    + **점수 지우기**: 제거 하려면이 옵션을 사용 합니다 **점수** 지정된 된 열에서 메타 데이터입니다.  
  
         현재 점수 열을 명시적으로 표시할 수는 Azure Machine Learning에서 사용할 수 없습니다. 그러나 일부 작업 내부적으로 점수를 표시 되는 열을 생성 합니다. 또한 사용자 지정 R 모듈 출력 점수 값 수입니다.
  
  
7.  에 대 한 **새 열 이름**, 선택한 열 또는 열의 새 이름을 입력 합니다.  
  
    + 열 이름에는 u t F-8에서 지원 되는 문자만 사용할 수 있습니다 인코딩. 빈 문자열, null 또는 공백 전적으로 이루어진 이름은 허용 되지 않습니다.  
  
    + 여러 열 이름을 바꾸려면 열 인덱스 순서 대로 쉼표로 구분 된 목록으로 이름을 입력 합니다.  
  
    + 선택한 모든 열 이름을 변경 해야 합니다. 생략 하거나 열을 건너뛸 수 없습니다.  
  
  
8.  실험을 실행합니다.
