# <a name="apply-math-operation"></a>수학 연산 적용

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)의 모듈에 대해 설명합니다.

수학 적용 작업을 사용하여 입력 데이터 집합의 숫자 열에 적용되는 계산을 만듭니다. 

지원되는 수학 연산에는 곱셈 및 분할, 삼각 함수, 다양한 반올림 함수 및 감마 및 오류 함수와 같은 데이터 과학에 사용되는 특수 함수와 같은 일반적인 산술 함수가 포함됩니다.  

작업을 정의하고 파이프라인을 실행하면 값이 데이터 집합에 추가됩니다. 모듈을 구성하는 방법에 따라 다음을 수행할 수 있습니다.

+ 결과를 데이터 집합에 보도록 합니다. 이 기능은 작업 결과를 확인할 때 특히 유용합니다.
+ 열 값을 계산된 새 값으로 바꿉습니다.
+ 결과에 대한 새 열을 생성하고 원래 데이터를 표시하지 않습니다. 

다음 범주에서 필요한 작업을 찾습니다.  

- [Basic](#basic-math-operations)  
  
     **Basic** 범주의 함수를 사용하여 값의 단일 값 또는 열을 조작할 수 있습니다. 예를 들어 열에 있는 모든 숫자의 절대 값을 얻거나 열에서 각 값의 제곱근을 계산할 수 있습니다.  
  
-   [비교](#comparison-operations)  
  
      **비교** 범주의 함수는 모두 비교에 사용됩니다: 두 열의 값에 대해 쌍으로 비교하거나 열의 각 값을 지정된 상수와 비교할 수 있습니다. 예를 들어 열을 비교하여 두 데이터 집합에서 값이 동일한지 여부를 확인할 수 있습니다. 또는 최대 허용 값과 같은 상수를 사용하여 숫자 열에서 이상값을 찾을 수 있습니다.  
  
-   [작업](#arithmetic-operations)  
  
     **작업** 범주에는 덧셈, 뺄셈, 곱셈 및 분할과 같은 기본 수학 함수가 포함됩니다. 열이나 상수로 작업할 수 있습니다. 예를 들어 열 A의 값을 열 B의 값에 추가할 수 있습니다. 또는 열 A의 각 값에서 이전에 계산된 평균과 같은 상수를 빼을 수 있습니다.  
  
-   [반올림](#rounding-operations)  
  
     **반올림** 범주에는 반올림, 천장, 바닥 및 다양한 수준의 정밀도로 의자선 과 같은 작업을 수행하기 위한 다양한 기능이 포함되어 있습니다. 소수점과 정수 모두에 대한 정밀도 수준을 지정할 수 있습니다.  
  
-   [특별](#special-math-functions)  
  
     **특수** 범주에는 타원 적분 및 가우시안 오류 함수와 같이 데이터 과학에 특히 사용되는 수학적 함수가 포함됩니다.  
  
-   [삼각](#trigonometric-functions)  
  
     **삼각법** 범주에는 모든 표준 삼각 함수가 포함됩니다. 예를 들어 라디안을 각도로 변환하거나 라디안 또는 도에서 접선과 같은 함수를 계산할 수 있습니다.
     이러한 함수는 unary, 즉 그들은 입력으로 값의 단일 열을 가지고 삼각함수를 적용 하 고 결과로 값의 열을 반환 합니다.  따라서 입력 열이 적절한 형식이고 지정된 작업에 적합한 종류의 값이 포함되어 있는지 확인해야 합니다.   

## <a name="how-to-configure-apply-math-operation"></a>수학 적용 작업을 구성하는 방법  

**수학 작업 적용** 모듈에는 숫자만 포함하는 하나 이상의 열이 포함된 데이터 집합이 필요합니다. 숫자는 불연속또는 연속일 수 있지만 문자열이 아닌 숫자 데이터 형식이어야 합니다.

동일한 작업을 여러 숫자 열에 적용할 수 있지만 모든 열은 동일한 데이터 집합에 있어야 합니다. 

이 모듈의 각 인스턴스는 한 번에 하나의 작업 유형만 수행할 수 있습니다. 복잡한 수학 연산을 수행하려면 **수학 작업 적용** 모듈의 여러 인스턴스를 함께 연결해야 할 수 있습니다.  
  
1.  파이프라인에 **수학 작업 적용** 모듈을 추가합니다.

1. 숫자 열이 하나 이상 포함된 데이터 집합을 연결합니다.  

1.  계산을 수행할 하나 이상의 소스 열을 선택합니다.   
  
    - 선택한 모든 열은 숫자 데이터 형식이어야 합니다. 
    - 데이터 범위는 선택한 수학 연수에 대해 유효해야 합니다. 그렇지 않으면 오류 또는 NaN(숫자 아님) 결과가 발생할 수 있습니다. 예를 들어 Ln(-1.0)은 잘못된 작업이며 `NaN`의 값을 생성합니다.
  
1.  **범주를** 클릭하여 **type** 수행할 수학 연산을 선택합니다.
    
1. 해당 범주의 목록에서 특정 작업을 선택합니다.
  
1.  각 작업 유형에 필요한 추가 매개 변수를 설정합니다.  
  
1.  출력 **모드** 옵션을 사용하여 수학 연산을 생성하는 방법을 나타냅니다. 

    - **Append**. 입력으로 사용되는 모든 열은 출력 데이터 집합에 포함되며 수학 연수 결과를 포함하는 추가 열이 추가됩니다.
    - **인장소**. 입력으로 사용되는 열의 값이 새 계산값으로 바뀝습니다. 
    - **결과만**. 수학 연수 결과를 포함하는 단일 열이 반환됩니다.
  
1.  파이프라인을 제출합니다.  
  
## <a name="results"></a>결과

**추가** 또는 **ResultOnly** 옵션을 사용하여 결과를 생성하는 경우 반환된 데이터 집합의 열 제목은 작업 및 사용된 열을 나타냅니다. 예를 들어 **Equals** 연산자(Equals 연산자)를 사용하여 두 열을 비교하면 결과는 다음과 같습니다.  
  
-   **[Col2_Col1)은**Col1에 대해 Col2를 테스트했음을 나타냅니다.  
-   열 2를 상수 10과 비교했음을 나타내는 **같음(Col2_$10)입니다.**  

**Inplace** 옵션을 사용하더라도 원본 데이터는 삭제되거나 변경되지 않습니다. 원래 데이터 집합의 열은 디자이너에서 계속 사용할 수 있습니다. 원본 데이터를 보려면 열 추가 [모듈을](add-columns.md) 연결하고 **수학 적용 작업의**출력에 조인할 수 있습니다.  
    
## <a name="basic-math-operations"></a>기본 수학 연산 

**Basic** 범주의 함수는 일반적으로 열에서 단일 값을 가져 와서 미리 정의된 작업을 수행하고 단일 값을 반환합니다. 일부 함수의 경우 상수 또는 열 집합을 두 번째 인수로 지정할 수 있습니다.  
  
 Azure 기계 학습은 **기본** 범주에서 다음 기능을 지원합니다.  

### <a name="abs"></a>Abs

선택한 열의 절대 값을 반환합니다.  
  
### <a name="atan2"></a>Atan2

4사분면 역탄젠트를 반환합니다.  

점 좌표를 포함하는 열을 선택합니다. x 좌표에 해당하는 두 번째 인수의 경우 상수를 지정할 수도 있습니다.  

Matlab의 ATAN2 함수에 해당합니다.  

### <a name="conj"></a>Conj

선택한 열의 값에 대한 컨쥬게이트를 반환합니다.  

### <a name="cuberoot"></a>CubeRoot

선택한 열의 값에 대한 큐브 루트를 계산합니다.  

### <a name="doublefactorial"></a>DoubleFactorial  
 선택한 열의 값에 대한 이중 계수를 계산합니다. 이중 계수는 일반 요인 함수의 확장이며 x로 표시됩니다!!.  

### <a name="eps"></a>Eps

현재 값과 다음으로 높은 배정밀도 값 사이의 크기 차이를 반환합니다. Matlab의 EPS 함수에 해당합니다.  
  
### <a name="exp"></a>Exp

선택한 열의 값 의 전원으로 발생 e를 반환합니다. 이것은 엑셀 EXP 함수와 동일합니다.  

### <a name="exp2"></a>Exp2

인수의 기본-2 지수를 반환하여 x *<sup>2t에</sup> 대해 해결하며 여기서 t는 지수를 포함하는 값의 열입니다.  

**열 집합에서**지수 값 t가 포함된 열을 선택합니다.

**Exp2의** 경우 상수 또는 다른 값 열일 수 있는 두 번째 인수 x를 지정할 수 있습니다. **두 번째 인수 형식에서**승수 x를 상수로 제공할지 또는 열의 값을 제공할지 여부를 나타냅니다.  

예를 들어 승수와 지수 에 {0,1,2,3,4,5} 대한 값이 있는 열을 선택하면 함수가 {0, 2, 8, 24, 64 160을 반환합니다.  

### <a name="expminus1"></a>ExpMinus1 

선택한 열의 값에 대한 음수 지수를 반환합니다.  

### <a name="factorial"></a>계승값
선택한 열의 값에 대한 요인을 반환합니다.  

### <a name="hypotenuse"></a>빗변
한 면의 길이가 값의 열로 지정되고 두 번째 면의 길이가 상수 또는 두 개의 열로 지정된 삼각형에 대한 hypotenuse를 계산합니다.  

### <a name="ln"></a>Ln

선택한 열의 값에 대한 자연 로그백을 반환합니다.  

### <a name="lnplus1"></a>LnPlus1

선택한 열의 값에 대해 자연 로그렘과 1을 반환합니다.  

### <a name="log"></a>로그

지정된 밑을 사용하여 선택한 열에 있는 값의 로그를 반환합니다.  

기준(두 번째 인수)을 상수로 지정하거나 다른 값 열을 선택하여 지정할 수 있습니다.  

### <a name="log10"></a>Log10

선택한 열에 대한 기본 10 로그릿헴 값을 반환합니다.  

### <a name="log2"></a>Log2

선택한 열에 대한 기본 2 로그릿헴 값을 반환합니다.  

### <a name="nthroot"></a>NthRoot
지정한 n을 사용하여 값의 n번째 루트를 반환합니다.  

**ColumnSet** 옵션을 사용하여 루트를 계산할 열을 선택합니다.  

**두 번째 인수 유형에서**루트를 포함하는 다른 열을 선택하거나 루트로 사용할 상수를 지정합니다.  

두 번째 인수가 열인 경우 열의 각 값은 해당 행에 대한 n값으로 사용됩니다. 두 번째 인수가 상수인 경우 두 **번째 인수** 텍스트 상자에 n에 대한 값을 입력합니다.
### <a name="pow"></a>Pow

선택한 열의 각 값에 대해 Y의 힘으로 올려진 X를 계산합니다.  

먼저 **ColumnSet** 옵션을 사용하여 float가 되어야 하는 **기준을**포함하는 열을 선택합니다.  

**두 번째 인수 유형에서**지수를 포함하는 열을 선택하거나 지수로 사용할 상수를 지정합니다.  

두 번째 인수가 열이면, 이 열의 각 값을 해당 행의 지수로 사용합니다. 두 번째 인수가 상수인 경우 **두 번째 인수** 텍스트 상자에 지수에 대한 값을 입력합니다.  

### <a name="sqrt"></a>Sqrt

선택한 열에서 값의 제곱근을 반환합니다.  

### <a name="sqrtpi"></a>SqrtPi

선택한 열의 각 값에 대해 값에 pi를 곱한 다음 결과의 제곱근을 반환합니다.  

### <a name="square"></a>Square

선택한 열의 값을 제곱합니다.  

## <a name="comparison-operations"></a>비교 연산  

두 값 집합을 서로 테스트해야 하는 Azure 기계 학습 디자이너의 비교 함수를 사용할 수 있습니다. 예를 들어 파이프라인에서 다음과 같은 비교 작업을 수행해야 할 수 있습니다.  

- 임계값에 대해 확률 점수 모델의 열을 평가합니다.
- 두 결과 집합이 동일한지 확인합니다. 다른 각 행에 대해 추가 처리 또는 필터링에 사용할 수 있는 FALSE 플래그를 추가합니다.  

### <a name="equalto"></a>EqualTo

값이 동일한 경우 True를 반환합니다.  

### <a name="greaterthan"></a>GreaterThan

**Column 집합의** 값이 지정된 상수보다 크거나 비교 열의 해당 값보다 큰 경우 True를 반환합니다.  

### <a name="greaterthanorequalto"></a>GreaterThanOrEqualTo

**Column 집합의** 값이 지정된 상보다 크거나 같거나 비교 열의 해당 값보다 크거나 같으면 True를 반환합니다.  

### <a name="lessthan"></a>LessThan

**Column 집합의** 값이 지정된 상수보다 적거나 비교 열의 해당 값보다 적은 경우 True를 반환합니다.  
  
### <a name="lessthanorequalto"></a>LessThanOrEqualTo

**Column 집합의** 값이 지정된 상보다 낮거나 같거나 비교 열의 해당 값보다 낮거나 같으면 True를 반환합니다.  

### <a name="notequalto"></a>NotEqualTo

**열 집합**의 값이 상수 또는 비교 열과 다르면 True를 반환하고, 동일하면 False를 반환합니다.  

### <a name="pairmax"></a>PairMax

**열 집합**의 값과 상수 또는 비교 열의 값 중 더 큰 값을 반환합니다.  

### <a name="pairmin"></a>PairMin

**열 집합의** 값 또는 상수 또는 비교 열의 값과 더 작은 값을 반환합니다.  
  
##  <a name="arithmetic-operations"></a>산술 연산   

기본 산술 연산( 덧셈 및 뺄셈, 분할 및 곱셈)을 포함합니다.  대부분의 작업은 두 개의 숫자가 필요한 이진 작업이므로 먼저 작업을 선택한 다음 첫 번째 및 두 번째 인수에서 사용할 열이나 숫자를 선택합니다.

분할 및 뺄셈에 대한 열을 선택하는 순서는 직관적이지 않은 것처럼 보일 수 있습니다. 그러나 결과를 보다 쉽게 이해할 수 있도록 열 제목은 작업 이름과 열이 사용된 순서를 제공합니다.

작업(Operation)|Num1|Num2|결과 열|결과 값|
----|----|----|----|----
|더하기|1|5|추가(Num2_Num1)| 4|
|곱하기|1|5|다중(Num2_Num1)|5|
|빼기|1|5|빼기(Num2_Num1)|4|
|빼기|0|1|빼기(Num2_Num1)|0|
|나누기|1|5|분할(Num2_Num1)|5|
|나누기|0|1|분할(Num2_Num1)|Infinity|

### <a name="add"></a>추가

**Column set을**사용하여 소스 열을 지정한 다음 두 번째 **인수에**지정된 숫자를 해당 값에 추가합니다.  

두 열에 값을 추가하려면 **column 집합을**사용하여 열 또는 열을 선택한 다음 **두 번째 인수를**사용하여 두 번째 열을 선택합니다.  

### <a name="divide"></a>나누기

**열 집합의** 값을 상수 또는 **두 번째 인수에**정의된 열 값으로 나눕니다.  즉, 먼저 제수기와 배당금을 선택합니다. 출력 값은 몫입니다.

### <a name="multiply"></a>곱하기

**열 집합**의 값을 지정된 상수 또는 열 값으로 곱합니다.  

### <a name="subtract"></a>빼기

**열 집합** 옵션을 사용하여 다른 열을 선택하여 작동할 값 *열(minuend)을*지정합니다. 그런 다음 **두 번째 인수** 드롭다운 목록을 사용하여 빼는 숫자(subtrahend)를 지정합니다. *subtrahend* 값의 상수 또는 열을 선택할 수 있습니다.

##  <a name="rounding-operations"></a>반올림 작업 

Azure 기계 학습 디자이너는 다양한 반올림 작업을 지원합니다. 많은 작업의 경우 반올림 할 때 사용할 정밀도양을 지정해야 합니다. 상수로 지정된 정적 정밀도 수준을 사용하거나 값 열에서 얻은 동적 정밀도 값을 적용할 수 있습니다.  

- 상수를 사용하는 경우 **정밀도 유형을** **상수로** 설정한 다음 **정수** 텍스트 상자에 숫자 수를 정수로 입력합니다. 정수가 아닌 것을 입력하면 모듈에서 오류를 발생시키지 않지만 결과가 예기치 않을 수 있습니다.  

- 데이터 집합의 각 행에 대해 다른 정밀도 값을 사용하려면 **정밀도 유형을** **ColumnSet으로**설정한 다음 적절한 정밀도 값이 포함된 열을 선택합니다.

## <a name="technical-notes"></a>기술 정보

두 번째 연산자로 두 개 이상의 열을 선택할 때주의하십시오. 모든 열에 상수를 추가하는 등 작업이 간단한지 쉽게 이해할 수 있습니다. 

데이터 집합에 여러 열이 있다고 가정하고 데이터 집합을 자체에 추가합니다. 결과에서 각 열은 다음과 같이 자체에 추가됩니다.  
  
|Num1|Num2|숫자 3|추가(Num1_Num1)|추가(Num2_Num2)|추가(Num3_Num3)|
|----|----|----|----|----|----|
|1|5|2|2|10|4|
|2|3|-1|4|6|-2|
|0|1|-1|0|2|-2|

더 복잡한 계산을 수행해야 하는 경우 **수학 적용 작업의**여러 인스턴스를 연결할 수 있습니다. 예를 들어 한 **수치 연산 적용** 인스턴스를 사용하여 두 개의 열을 추가한 후 다른 **수치 연산 적용** 인스턴스를 사용하여 상수로 합계를 나누어 평균을 계산합니다.
