# <a name="linear-regression-module"></a>선형 회귀 모듈
이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

이 모듈을 사용 하 여 실험에서 사용에 대 한 선형 회귀 모델을 만듭니다.  선형 회귀는 하나 이상의 독립 변수 및 숫자 결과 또는 종속 변수 간에 선형 관계를 설정 하려고 합니다. 

이 모듈을 사용 하 여 선형 회귀 메서드를 정의 하 고 레이블이 지정 된 데이터 집합을 사용 하 여 모델을 학습 합니다. 학습된 된 모델을 예측 한 다음 사용할 수 있습니다.

## <a name="about-linear-regression"></a>선형 회귀에 대 한

선형 회귀는 machine learning에서 도입 되었으며 줄 맞춤 하 고 오류를 측정 하기 위한 여러 새로운 메서드를 사용 하 여 향상 된 일반적인 통계 메서드에입니다. 가장 기본적인 관점에서 회귀는 숫자 대상의 예측을 말합니다. 기본 예측 작업에 대 한 간단한 모델을 원하는 경우 선형 회귀는 여전히 적합 합니다. 선형 회귀는 또한 복잡 하지 않은 높은 차원의 스파스 데이터 집합에서 잘 작동 하는 경향이 있습니다.

Azure Machine Learning에는 다양 한 회귀 모델의 경우 선형 회귀 외에도 지원합니다. 그러나 "재발" 라는 용어는 느슨하게, 해석 될 수 있으며 다른 도구에서 제공 하는 회귀의 일부 형식은 지원 되지 않습니다.

+ 클래식 회귀 문제에는 종속 변수와 독립 변수를 하나만 포함 됩니다. 이 이라고 *단순 회귀*합니다.  이 모듈에서는 간단한 회귀를 지원 합니다.

+ *다중 선형 회귀* 단일 종속 변수가에 기여 하는 두 개 이상의 독립 변수를 포함 합니다. 단일 숫자 결과 예측에 사용 되는 여러 입력 문제 라고도 *다변량 선형 회귀*합니다.

    합니다 **선형 회귀** 모듈 대부분의 다른 회귀 모듈 수 있습니다. 이러한 문제를 해결할 수 있습니다.

+ *다중 레이블 회귀* 단일 모델 내에서 여러 종속 변수를 예측 하는 작업입니다. 예를 들어, 다중 레이블 로지스틱 회귀에서 샘플을 여러 다른 레이블에 할당할 수 있습니다. (이 단일 클래스 변수 내에서 여러 수준을 예측 하는 작업과 다릅니다.)

    이 유형의 회귀는 Azure Machine Learning에서 지원 되지 않습니다. 여러 변수를 예측 하려면 예측 하려는 각 출력에 대 한 별도 학습자를 만듭니다.

오랫동안 통계학자 점점 더 고급 회귀 방법을 개발한 합니다. 선형 회귀에 대해서도 마찬가지입니다. 이 모듈에서는 오류를 측정 하 고 회귀선을 맞추기는 두 가지 방법: 최소 자승법 메서드와 기울기 하강 합니다.

- **기울기 하강** 모델 학습 프로세스의 각 단계에서 오류의 양을 최소화 하는 메서드입니다. 기울기 하강에 많은 변동이 발생 하 고 다양 한 학습 문제에 대 한 최적화는 광범위 하 게 연구 되었습니다. 이 옵션을 선택 하면 **솔루션 메서드**, 학습 속도, 다양 한 단계 크기를 제어 하는 매개 변수를 설정 하 고 등 수 있습니다. 또한이 옵션을 통합된 매개 변수 비우기 사용을 지원합니다.

- **최소 자승법** 선형 회귀에 자주 사용 하는 기술 중 하나입니다. 예를 들어, 최소 자승법은 Microsoft Excel 용 분석 도구에 사용 되는 메서드.

    최소 자승법은 실제 값에서 예측 선까지 거리의 제곱의 합으로 오류를 계산 하 고 제곱된 된 오류를 최소화 하 여 모델에 적합 한 손실 함수를 가리킵니다. 이 메서드는 입력과 종속 변수 간의 강력한 선형 관계를 가정합니다.

## <a name="configure-linear-regression"></a>선형 회귀를 구성 합니다.

이 모듈에서는 다양 한 옵션을 사용 하 여 회귀 모델을 맞추는 두 가지 방법이 있습니다.

+ [온라인 기울기 하강을 사용 하 여 회귀 모델 만들기](#bkmk_GradientDescent)

    기울기 하강에는 더 복잡 한는 또는 변수 수를 지정 하는 학습 데이터가 너무 적은 모델에 대 한 더 나은 손실 함수입니다.



+ [최소 자승법을 사용 하 여 회귀 모델을 맞추기](#bkmk_OrdinaryLeastSquares)

    작은 데이터 집합에 대 한 최소 자승법을 선택 하는 것이 좋습니다. 그러면 하면 비슷한 결과 Excel로 제공 됩니다.

## <a name="bkmk_OrdinaryLeastSquares"></a> 최소 자승법을 사용 하 여 회귀 모델 만들기

1. 추가 된 **선형 회귀 모델** 모듈 인터페이스에서 실험을 합니다.

    이 모듈에서 찾을 수 있습니다 합니다 **Machine Learning** 범주입니다. 확장 **모델 초기화**, 확장 **회귀**를 끌어서 놓습니다 합니다 **선형 회귀 모델** 모듈을 실험 합니다.

2. 에 **속성** 창에서 합니다 **솔루션 메서드** 드롭다운 목록에서 선택 **최소 자승법**합니다. 이 옵션에는 회귀선을 찾는 데 계산 방법을 지정 합니다.

3. **L2 정규화 가중치**, L2 정규화의 가중치로 사용할 값을 입력 합니다. 0이 아닌 값을 사용 하 여 과잉 맞춤을 방지 하는 것이 좋습니다.

     정규화 모델 맞춤에 미치는 영향에 대 한 자세한 내용은이 문서를 참조 하세요. [L1 및 L2 정규화 Machine Learning에 대 한](https://msdn.microsoft.com/magazine/dn904675.aspx)

4. 옵션을 선택 **절편 용어 포함**절편 용어를 확인 하려는 경우.

    회귀 수식을 검토할 필요가 없는 경우이 옵션의 선택을 취소 합니다.

5. 에 대 한 **임의 숫자 초기값**, 모델에서 사용 하는 난수 생성기 시드 값을 입력할 수도 있습니다.

    초기값을 사용 하는 것은 여러 번 실행할 때 동일한 실험의 동일한 결과 유지 하려는 경우에 유용 합니다. 그렇지 않은 경우 기본값은 시스템 클록에서 값을 사용 하도록.


7. 추가 합니다 [모델 학습](./train-model.md) 모듈을 실험에 레이블이 지정 된 데이터 집합을 연결 합니다.

8. 실험을 실행합니다.

## <a name="bkmk_GradientDescent"></a> 온라인 기울기 하강을 사용 하 여 회귀 모델 만들기

1. 추가 된 **선형 회귀 모델** 모듈 인터페이스에서 실험을 합니다.

    이 모듈에서 찾을 수 있습니다 합니다 **Machine Learning** 범주입니다. 확장 **모델 초기화**를 확장 하 고 **회귀**, 끌어서 합니다 **선형 회귀 모델** 모듈을 실험

2. 에 **속성** 창에서 합니다 **솔루션 메서드** 드롭다운 목록에서 선택 **온라인 기울기 하강** 회귀선을 찾는 데 계산 방법으로 합니다.

3. 에 대 한 **트레이너 모드 만들기**, 미리 정의 된 매개 변수 집합을 사용 하 여 모델을 학습 것인지 또는 매개 변수 비우기를 사용 하 여 모델을 최적화 하려는 경우를 나타냅니다.

    + **매개 변수를 단일**: 선형 회귀 네트워크를 구성 하려는 방법을 알고 있는 경우 특정 값 집합을 인수로 제공할 수 있습니다.

   
4. 에 대 한 **학습 속도**, 추측 기울기 하강 최적화 프로그램의 초기 학습 속도 지정 합니다.

5. 에 대 한 **학습 epoch 수**, 알고리즘 횟수를 지정 하는 값 예제를 반복 해야 하는 형식입니다. 적은 수의 예제를 사용 하 여 데이터 집합의 경우이 번호는 수렴에 도달할 정도로 이어야 합니다.

6. **기능 정규화**: 모델 학습에 사용 되는 숫자 데이터 정규화 이미 있는 경우이 옵션을 선택을 취소할 수 있습니다. 기본적으로 모듈 0과 1 사이의 범위에 대 한 모든 숫자 입력을 정규화합니다.

    > [!NOTE]
    > 
    > 새 데이터 점수 매기기에 사용 되는 동일한 정규화 메서드를 적용 해야 합니다.

7. **L2 정규화 가중치**, L2 정규화의 가중치로 사용할 값을 입력 합니다. 0이 아닌 값을 사용 하 여 과잉 맞춤을 방지 하는 것이 좋습니다.

    정규화 모델 맞춤에 미치는 영향에 대 한 자세한 내용은이 문서를 참조 하세요. [L1 및 L2 정규화 Machine Learning에 대 한](https://msdn.microsoft.com/magazine/dn904675.aspx)


9. 옵션을 선택 **학습 속도가 감소**반복이 진행 됨에 따라 감소를 학습 속도 하려는 경우.  

10. 에 대 한 **임의 숫자 초기값**, 모델에서 사용 하는 난수 생성기 시드 값을 입력할 수도 있습니다. 초기값을 사용 하는 것은 여러 번 실행할 때 동일한 실험의 동일한 결과 유지 하려는 경우에 유용 합니다.


12. 레이블이 지정 된 데이터 집합 및 학습 모듈 중 하나를 추가 합니다.

    매개 변수 비우기의 사용 하지 않는 경우 사용 합니다 [모델 학습](train-model.md) 모듈입니다.

13. 실험을 실행합니다.
