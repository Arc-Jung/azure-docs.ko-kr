# <a name="two-class-support-vector-machine-module"></a>2 클래스 Support Vector Machine 모듈

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

이 모듈을 사용 하 여 지원 벡터 컴퓨터 알고리즘을 기반으로 하는 모델을 만듭니다. 

Svm (지원 벡터 컴퓨터)는 거친 클래스 감독된 학습 방법입니다. 이 특정 구현 연속 또는 범주 변수를 기반으로 하는 두 가지 가능한 결과 예측에 적합 합니다.

모델 매개 변수를 정의한 후 학습 모듈을 사용 하 고 제공 하 여 모델 학습을 *데이터 집합을 태그가 지정 된* 레이블 또는 결과 열을 포함 하는 합니다.

## <a name="about-support-vector-machines"></a>지원 벡터 컴퓨터에 대 한

지원 벡터 컴퓨터는 초기 기계 학습 알고리즘 및 SVM 모델 텍스트 및 이미지 분류에 대 한 정보 검색에서 많은 응용 프로그램에서 사용 되었습니다. 분류와 회귀 작업에 대해 Svm은 사용할 수 있습니다.

이 SVM 모델은 레이블이 지정 된 데이터를 필요로 하는 감독 된 학습 모델입니다. 학습 프로세스에서 알고리즘 입력된 데이터를 분석 하 고 호출 하는 다차원 기능 공간에서 패턴을 인식 합니다 *평면이*합니다.  모든 입력된 예제가이 공간에 요소로 표시 되 고 범주 범주 차원으로 나눈을 최대한 차이 선택을 취소 하는 방식으로 출력에 매핑됩니다.

SVM 알고리즘을 예측에 대 한 하나의 범주 또는 다른, 동일한 해당 공간에 매핑시키고에 새 예제를 할당 합니다. 

## <a name="how-to-configure"></a>구성 방법 

이 모델 유형에 대 한 분류자를 학습에 사용 하기 전에 데이터 집합을 정규화 하는 것이 좋습니다.
  
1.  추가 된 **2 클래스 Support Vector Machine** 모듈을 실험 합니다.  
  
2.  설정 하 여 학습 모델을 지정 합니다 **트레이너 모드 만들기** 옵션입니다.  
  
    -   **매개 변수를 단일**: 모델을 구성 하려는 방법를 알고 있으면 특정 값 집합을 인수로 제공할 수 있습니다.  

3.  에 대 한 **반복 횟수**, 모델을 빌드할 때 사용 하는 반복 횟수를 나타내는 숫자를 입력 합니다.  
  
     이 매개 변수 학습 속도 정확도 간의 균형을 제어할 사용할 수 있습니다.  
  
4.  에 대 한 **람다**, L1 정규화의 가중치로 사용할 값을 입력 합니다.  
  
     이 정규화 계수는 모델을 조정 하 사용할 수 있습니다. 값이 클수록 더욱 복잡 한 모델의 처벌 합니다.  
  
5.  옵션을 선택 **기능 정규화**학습 전에 기능을 정규화 할 경우.
  
     학습 하기 전에 정규화를 적용 하는 경우 데이터 요소 평균 중심이 업데이트 하 고 표준 편차가 한 단위가 되도록 확장 됩니다.
  
6.  옵션을 선택 **프로젝트 단위 구체에**, 계수를 정규화 합니다.
  
     값 단위 공간에 프로젝션 의미 학습 하기 전에 데이터 요소는 0이 중심이 표준 편차가 한 단위가 되도록 크기를 조정 합니다.
  
7.  **임의 숫자 초기값**, 실행 간에 재현 가능성을 보장 하려는 경우 초기값으로 사용할 정수 값을 입력 합니다.  이 고, 그렇지 시스템 클록 값은 실행 간에 약간 다른 결과를 초래할 수 있습니다 초기값으로 사용 됩니다.
  
9. 레이블이 지정 된 데이터 집합 및 중 하나에 연결 합니다 [학습 모듈](module-reference.md):
  
    -   설정 하는 경우 **트레이너 모드 만들기** 를 **단일 매개 변수**를 사용 합니다 [모델 학습](train-model.md) 모듈입니다.
  

10. 실험을 실행합니다.
