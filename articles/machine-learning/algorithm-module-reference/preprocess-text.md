# <a name="preprocess-text"></a>텍스트 전처리

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)의 모듈에 대해 설명합니다.

텍스트 **전처리** 모듈을 사용하여 텍스트를 정리하고 단순화합니다. 다음과 같은 일반적인 텍스트 처리 작업을 지원합니다.

* 중지 단어 제거
* 정규식을 사용하여 특정 대상 문자열을 검색하고 바꾸기
* 여러 관련 단어를 단일 표준 형식으로 변환하는 Lemmatization
* 대/소문자 정규화
* 숫자, 특수 문자 및 "aaaa"와 같은 반복된 문자 시퀀스와 같은 특정 클래스의 문자 제거
* 이메일 및 URL 식별 및 제거

**텍스트 전처리** 모듈은 현재 영어만 지원합니다.

## <a name="configure-text-preprocessing"></a>텍스트 전처리 구성  

1.  Azure 기계 학습에서 파이프라인에 **프로세스 전처리 텍스트** 모듈을 추가합니다. **텍스트 분석**에서 이 모듈을 찾을 수 있습니다.

1. 텍스트가 포함된 열이 하나 이상 있는 데이터 집합을 연결합니다.

1. **언어** 드롭다운 목록에서 언어를 선택합니다.

1. **정리할 텍스트 열**: 사전 처리할 열을 선택합니다.

1. **중지 단어 제거**: 텍스트 열에 미리 정의된 중지 단어 목록을 적용하려면 이 옵션을 선택합니다. 

    중지 단어 목록은 언어에 따라 다르며 사용자 지정이 가능합니다.

1. **Lemmatization**: 단어를 표준 형식으로 표시하려면 이 옵션을 선택합니다. 이 옵션은 유사한 텍스트 토큰의 고유 발생 횟수를 줄이는 데 유용합니다.

    lemmatization 프로세스는 언어에 따라 다릅니다.

1. **문장 감지**: 해석을 수행할 때 모듈이 문장 경계 표시를 삽입하도록 하려면 이 옵션을 선택합니다.

    이 모듈에서는 일련의 세 `|||` 개의 파이프 문자를 사용하여 문장 종기를 나타냅니다.

1. 정규식을 사용하여 선택적 찾기 및 바꾸기 작업을 수행합니다.

    * **사용자 지정 정규식**: 검색할 텍스트를 정의합니다.
    * **사용자 지정 대체 문자열**: 단일 대체 값을 정의합니다.

1. **소문자로 대/소문자로 정규화**: ASCII 대문자문자를 소문자 형태로 변환하려면 이 옵션을 선택합니다.

    문자가 정규화되지 않으면 대문자와 소문자에서 동일한 단어가 두 개의 서로 다른 단어로 간주됩니다.

1. 처리된 출력 텍스트에서 다음과 같은 유형의 문자 또는 문자 시퀀스를 제거할 수도 있습니다.

    * **숫자 제거**: 이 옵션을 선택하여 지정된 언어의 모든 숫자 문자를 제거합니다. 식별 번호는 도메인에 따라 다르며 언어에 따라 다릅니다. 숫자 문자가 알려진 단어의 필수적인 부분인 경우 숫자가 제거되지 않을 수 있습니다.
    
    * **특수 문자 제거**: 이 옵션을 사용하여 영숫자가 아닌 특수 문자를 제거합니다.
    
    * **중복 문자 제거**: 이 옵션을 선택하여 두 번 이상 반복되는 시퀀스에서 추가 문자를 제거합니다. 예를 들어 "aaaaa"와 같은 시퀀스는 "aaa"로 줄어듭니다.
    
    * **이메일 주소 제거**: 이 옵션을 선택하여 `<string>@<string>`형식의 시퀀스를 제거합니다.  
    * **URL 제거**: 다음 URL 접두사가 포함된 시퀀스를 `http`제거하려면 이 옵션을 선택합니다. `https` `ftp``www`
    
1. **동사 수축 확장**: 이 옵션은 동사 수축을 사용하는 언어에만 적용됩니다. 현재, 영어만. 

    예를 들어 이 옵션을 선택하면 *"거기에 머무르지 않을 것"이라는* 문구를 *"거기에 머물지 않을 것"으로*바꿀 수 있습니다.

1. **슬래시로 백슬래시 정규화**: 이 옵션을 선택하여 의 모든 `\\` 인스턴스를 에 매핑합니다. `/`

1. **특수 문자에 대한 분할 토큰**: `&`이 옵션을 선택하여 와 `-`같은 문자에 대한 단어를 나누려면 이 옵션을 선택합니다. 이 옵션은 특수 문자를 두 번 이상 반복할 때 줄일 수도 있습니다. 

    예를 `MS---WORD` 들어 문자열은 세 개의 토큰, `MS`및 `-`. `WORD`

1. 파이프라인을 제출합니다.
