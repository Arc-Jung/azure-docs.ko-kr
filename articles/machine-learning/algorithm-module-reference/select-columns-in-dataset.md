# <a name="select-columns-in-dataset-module"></a>데이터 집합 모듈에서 열 선택

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)의 모듈에 대해 설명합니다.

이 모듈을 사용하여 다운스트림 작업에 사용할 열의 하위 집합을 선택합니다. 모듈은 원본 데이터 집합에서 열을 물리적으로 제거하지 않습니다. 대신 데이터베이스 *보기* 또는 *프로젝션과*같은 열하위 집합을 만듭니다.

이 모듈은 다운스트림 작업에 사용할 수 있는 열을 제한해야 하거나 불필요한 열을 제거하여 데이터 집합의 크기를 줄이려는 경우에 유용합니다.

데이터 집합의 열은 원래 데이터와 동일한 순서로 출력되며, 다른 순서로 지정한 경우에도 마찬가지입니다.

## <a name="how-to-use"></a>사용 방법

이 모듈에는 매개 변수가 없습니다. 열 선택기를 사용하여 포함하거나 제외할 열을 선택합니다.

### <a name="choose-columns-by-name"></a>이름으로 열 선택

모듈에는 이름으로 열을 선택하는 여러 옵션이 있습니다. 

+ 필터링 및 검색

    이름 **별** 옵션을 클릭합니다.

    이미 채워진 데이터 집합을 연결한 경우 사용 가능한 열 목록이 나타납니다. 열이 나타나지 않으면 열 목록을 보려면 업스트림 모듈을 실행해야 할 수 있습니다.

    목록을 필터링하려면 검색 상자를 입력합니다. 예를 들어 검색 상자에 `w` 문자를 입력하면 목록이 필터링되어 문자가 `w`포함된 열 이름을 표시합니다.

    열을 선택하고 오른쪽 화살표 버튼을 클릭하여 선택한 열을 오른쪽 창의 목록으로 이동합니다.

    + 연속 열 이름 범위를 선택하려면 **Shift + 를 클릭합니다.**
    + 선택 영역에 개별 열을 추가하려면 **Ctrl + 를 클릭합니다.**

    확인 표시 버튼을 클릭하여 저장하고 닫습니다.

+ 다른 규칙과 함께 이름 사용

    사용 규칙 옵션을 **클릭합니다.**
    
    특정 데이터 형식의 열을 표시하는 등의 규칙을 선택합니다.

    그런 다음 이름으로 해당 유형의 개별 열을 클릭하여 선택 목록에 추가합니다.

+ 쉼표로 구분된 열 이름 목록을 입력하거나 붙여넣기

    데이터 집합이 넓은 경우 열을 개별적으로 선택하는 것보다 인덱스 또는 생성된 이름 목록을 사용하는 것이 더 쉬울 수 있습니다. 목록을 미리 준비했다고 가정합니다.

    1. 사용 규칙 옵션을 **클릭합니다.** 
    2. 열 없음을 선택하고 **포함을**선택한 다음 빨간색 느낌표가 있는 텍스트 상자 내부를 **클릭합니다.** 
    3. 이전에 유효성이 검사된 열 이름의 쉼표 로 분리된 목록을 붙여넣거나 입력합니다. 열에 잘못된 이름이 있는 경우 모듈을 저장할 수 없으므로 미리 이름을 확인하십시오.
    
    이 메서드를 사용하여 인덱스 값을 사용하여 열 목록을 지정할 수도 있습니다. 

### <a name="choose-by-type"></a>유형별 선택

**WITH RULES** 옵션을 사용하는 경우 열 선택 항목에 여러 조건을 적용할 수 있습니다. 예를 들어 숫자 데이터 형식의 피쳐 열만 얻어야 할 수 있습니다.

**시작** 옵션을 사용하면 시작점이 결정되며 결과를 이해하는 데 중요합니다. 

+ **ALL COLUMNS** 옵션을 선택하면 모든 열이 목록에 추가됩니다. 그런 다음 **제외** 옵션을 사용하여 특정 조건을 충족하는 열을 *제거해야* 합니다. 

    예를 들어 모든 열로 시작한 다음 이름또는 유형별로 열을 제거할 수 있습니다.

+ **열 없음** 옵션을 선택하면 열 목록이 비어 시작합니다. 그런 다음 목록에 열을 *추가할* 조건을 지정합니다. 

    여러 규칙을 적용하는 경우 각 조건이 **가산됩니다.** 예를 들어 열없이 시작한 다음 규칙을 추가하여 모든 숫자 열을 가져옵니다. 자동차 가격 데이터 집합에서 16개의 열이 생성됩니다. 그런 다음 기호를 **+** 클릭하여 새 조건을 추가하고 모든 **피처 포함을**선택합니다. 결과 데이터 집합에는 모든 숫자 열과 일부 문자열 피쳐 열을 포함한 모든 피쳐 열이 포함됩니다.

### <a name="choose-by-column-index"></a>열 인덱스로 선택

열 인덱스는 원래 데이터 집합 내의 열 순서를 나타냅니다.

+ 열은 1부터 순차적으로 번호가 매겨져 있습니다.  
+ 다양한 열을 얻으려면 하이픈을 사용합니다. 
+ 와 같은 `1-` 개방형 사양은 `-3` 허용되지 않습니다.
+ 중복 인덱스 값(또는 열 이름)은 허용되지 않으며 오류가 발생할 수 있습니다.

예를 들어 데이터 집합에 8개 이상의 열이 있다고 가정하면 다음 예제에 붙여서 여러 개의 연속되지 않은 열을 반환할 수 있습니다. 

+ `8,1-4,6`
+ `1,3-8`
+ `1,3-6,4` 

마지막 예제에서는 오류가 발생하지 않습니다. 그러나 단일 열 `4`인스턴스를 반환합니다.



### <a name="change-order-of-columns"></a>열의 순서 변경

선택 **항목에서 중복 허용 및 열 순서 유지** 옵션은 빈 목록으로 시작하고 이름 또는 인덱스별로 지정하는 열을 추가합니다. 항상 "자연 순서"로 열을 반환하는 다른 옵션과 달리 이 옵션은 열의 이름을 지정하거나 나열하는 순서대로 열을 출력합니다. 

예를 들어 Col1, Col2, Col3 및 Col4 열이 있는 데이터 집합에서 다음 목록 중 하나를 지정하여 열의 순서를 반대로 하고 열 2를 제외할 수 있습니다.

+ `Col4, Col3, Col1`
+ `4,3,1`
