# <a name="select-columns-in-dataset-module"></a>데이터 집합 모듈에 열을 선택 합니다.

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

이 모듈을 사용 하 여 다운스트림 작업에서 사용 하 여 열의 하위 집합을 선택 합니다. 모듈 열 원본 데이터 집합에서 물리적으로 제거 되지 않습니다. 대신 데이터베이스와 마찬가지로 열의 하위 집합을 만듭니다 *뷰* 하거나 *프로젝션*합니다.

이 모듈을 다운스트림 작업에 대 한 사용 가능한 열을 제한 해야 할 때 또는 불필요 한 열을 제거 하 여 데이터 집합의 크기를 줄이고 싶은 경우에 유용 합니다.

데이터 집합의 열이 다른 순서를 지정 하는 경우에 원래 데이터와 동일한 순서로 출력 합니다.

## <a name="how-to-use"></a>사용 방법

이 모듈에 매개 변수가 없습니다. 열 선택기를 사용 하 여 포함 하거나 제외 하려면 열을 선택 합니다.

### <a name="choose-columns-by-name"></a>이름별으로 열을 선택 합니다.

이름으로 열을 선택 하는 것에 대 한 모듈의 여러 옵션이 있습니다. 

+ 필터 및 검색

    클릭 합니다 **이름순** 옵션입니다.

    가 이미 채워져 있는 데이터 집합에 연결 하는 경우 사용 가능한 열 목록이 표시 됩니다. 열을 표시 하는 경우에 열 목록을 보려면 업스트림 모듈을 실행 해야 합니다.

    목록을 필터링 하려면 검색 상자에 입력 합니다. 예를 들어 문자를 입력할 `w` 문자를 포함 하는 열 이름을 표시 하려면 검색 상자에서 목록이 필터링 됩니다 `w`합니다.

    열을 선택 하 고 오른쪽 창에서 목록에 선택한 열을 이동 하려면 오른쪽 화살표 단추를 클릭 합니다.

    + 열 이름의 연속 범위를 선택 하려면 **Shift + 클릭**합니다.
    + 키를 눌러 선택 영역에 개별 열을 추가할 **Ctrl + 클릭**합니다.

    저장 후 닫기 확인 표시 단추를 클릭 합니다.

+ 다른 규칙과 함께 이름을 사용 하 여

    클릭 합니다 **사용 하 여 규칙** 옵션입니다.
    
    규칙을 표시 하는 특정 데이터 형식의 열을 선택 합니다.

    선택 목록에 추가할 이름으로 해당 형식의 개별 열을 차례로 클릭 합니다.

+ 입력 또는 열 이름의 쉼표로 구분 된 목록을 붙여 넣습니다.

    데이터 집합에 넓은 경우 인덱스를 사용 하 여 더 쉬울 수 있습니다 또는 개별적으로 선택한 열이 아닌 이름 목록을 생성 합니다. 목록을 사전 준비한 것으로 가정 합니다.

    1. 클릭 합니다 **사용 하 여 규칙** 옵션입니다. 
    2. 선택 **열이 없습니다**를 선택 **포함**, 및 빨간색 느낌표가 텍스트 상자 내부를 클릭 합니다. 
    3. 붙여 넣거나 이전에 유효성이 검사 된 열 이름의 쉼표로 구분 된 목록을 입력 합니다. 모듈 저장 열에 잘못 된 이름이 있으면 되므로 확신할 수 없습니다 미리 이름을 확인 하려면.
    
    또한 인덱스 값을 사용 하 여 열 목록을 지정 하려면이 메서드를 사용할 수 있습니다. 

### <a name="choose-by-type"></a>형식으로 선택 합니다.

사용 하는 경우는 **사용 하 여 규칙** 옵션을 선택한 열에 여러 조건을 적용할 수 있습니다. 예를 들어 숫자 데이터 형식의 기능 열만 가져오도록 해야 할 수 있습니다.

합니다 **시작** 옵션 시작점을 결정 하 고 결과 이해 하는 데 중요 합니다. 

+ 선택 하는 경우는 **모든 열** 옵션을 모든 열 목록에 추가 됩니다. 그런 다음 사용 해야 합니다 **제외** 옵션을 *제거* 특정 조건을 충족 하는 열입니다. 

    예를 들어 모든 열을 사용 하 여 시작 수도 있으며 다음 또는 형식으로 이름별로 열을 제거할 수 있습니다.

+ 선택 하는 경우는 **NO 열** 옵션을 열 목록을 빈 상태로 시작 합니다. 그런 다음 조건을 지정할 *추가* 목록에는 열입니다. 

    여러 규칙을 적용 하면 각 조건은 **가산적**합니다. 예를 들어 열이 없는 시작한 모든 숫자 열을 가져오려면 규칙을 추가 합니다. 자동차 가격 데이터 집합의 결과는 16 개의 열입니다. 차례로 클릭 합니다 **+** 새 조건을 추가 하 고 선택 로그인 **모든 기능이 포함 되어**. 결과 데이터 집합의 모든 숫자 열을 더한 일부 문자열 기능 열을 비롯 한 모든 기능 열을 포함 합니다.

### <a name="choose-by-column-index"></a>열 인덱스로 선택

원래 데이터 집합 내에 있는 열의 순서와 열 인덱스를 나타냅니다.

+ 열 1에서 시작 하는 순차적으로 번호가 매겨집니다.  
+ 열 범위는 하이픈을 사용 합니다. 
+ 와 같은 개방형 사양 `1-` 또는 `-3` 허용 되지 않습니다.
+ 중복 인덱스 값 (또는 열 이름), 허용 되지 않으며 오류가 발생할 수 있습니다.

예를 들어를 가정 하 고 데이터 집합에 적어도 8 개 열 수 없습니다 붙여넣을 여러 연속 되지 않은 열을 반환 하기 위해 다음 예제에서: 

+ `8,1-4,6`
+ `1,3-8`
+ `1,3-6,4` 

마지막 예에서는 컬렉션에 오류가 발생 하지 않습니다. 그러나 단일 인스턴스의 열을 반환 합니다 `4`합니다.



### <a name="change-order-of-columns"></a>열 순서 변경

옵션 **중복을 허용 하 고 선택 영역에서 열 순서를 유지** 는 빈 목록으로 시작 하 고 이름이 나 인덱스를 지정 하는 열을 추가 합니다. 항상 "일반적인 방향의 순서"에서 열을 반환 하는 다른 옵션과 달리이 옵션 이름을 지정 하는 순서 대로 열을 출력 하거나 나열 합니다. 

예를 들어, Col1, Col2, Col3 및 Col4 열을 사용 하 여 데이터 집합에서 열의 순서를 반대로 하 고 다음 목록 중 하나를 지정 하 여 2, 열을 생략 수 있습니다.

+ `Col4, Col3, Col1`
+ `4,3,1`
