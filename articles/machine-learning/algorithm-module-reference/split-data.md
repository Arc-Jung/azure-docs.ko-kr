# <a name="split-data-module"></a>데이터 분할 모듈

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

이 모듈을 사용 하 여 두 개의 집합으로 데이터 집합을 분할 합니다.

이 모듈은 데이터를 학습 집합과 테스트 집합으로 분리 해야 할 때 특히 유용 합니다. 데이터도 분할 되는 방식을 사용자 지정할 수 있습니다. 일부 옵션 임의 데이터를 지원합니다. 다른 사용자가 특정 데이터 형식 또는 모델 형식에 맞게 조정 됩니다.

## <a name="how-to-configure"></a>구성 방법

> [!TIP]
> 분할의 유형을 결정 하는 모든 옵션을 읽고 분할 모드를 선택 하기 전에 다음을 수행 해야 합니다.
> 분할 모드를 변경 하면 다른 모든 옵션 초기화할 수 없습니다.

1. 추가 된 **분할 데이터** 모듈 인터페이스에서 실험을 합니다. 이 모듈에서 찾을 수 있습니다 **데이터 변환**를 **샘플 및 분할** 범주입니다.

2. **분할 모드**: 형식에 있는 데이터 및 분할 하려는 방법에 따라 다음 모드 중 하나를 선택 합니다. 각 분할 모드에 다른 옵션이 있습니다. 에 대 한 내용과 자세한 지침은 다음 항목을 클릭 합니다. 

    - **행 분할**: 두 부분으로 데이터를 분할 하려는 경우이 옵션을 사용 합니다. 각 분할에 배치 하는 데이터의 백분율을 지정할 수 있지만 기본적으로 데이터에는 50 ~ 50 구분 됩니다.

        각 그룹의 행이 선택 불규칙화 하 고 층 화 샘플링을 사용 하 여 수도 있습니다. 층 화 샘플링의 두 결과 데이터 집합 간에 동일 하 게 할당 될 값 하려는 데이터의 단일 열을 선택 해야 합니다.  

    - **일반 식 분할** 값에 대 한 단일 열을 테스트 하 여 데이터 집합을 분할 하려는 경우이 옵션을 선택 합니다.

        예를 들어, 감정 분석 하는 경우 텍스트 필드에서 특정 제품 이름의 존재 여부 확인 하 고 데이터 집합에 없는 및 대상 제품 이름을 사용 하 여 행을 나누는 수 있습니다.

    - **상대 식 분할**:  숫자 열에 조건을 적용 하려고 할 때마다이 옵션을 사용 합니다. 날짜/시간 필드, 포함 된 보관 기간을 열 또는 달러 금액 백분율도 될 수 있습니다. 예를 들어, 다음 연령 범위를 기준으로 그룹 사용자나 달력 날짜에서 별도 데이터 항목의 비용에 따라 데이터 집합을 분할 하는 것이 좋습니다.

### <a name="split-rows"></a>행 분할
1.  추가 합니다 [분할 데이터](./split-data.md) 모듈 인터페이스에서 실험을 분할 하려면 데이터 집합 연결 합니다.
  
2.  에 대 한 **분할 모드**, 선택 **행 분할**합니다. 

3.  **첫 번째 출력 데이터 집합의 행 분수**합니다. 첫 번째 (왼쪽) 출력으로 이동 행 개수를 확인 하려면이 옵션을 사용 합니다. 다른 모든 행의 두 번째 (오른쪽) 출력으로 이동 합니다.

    비율을 0과 1 사이의 10 진수 숫자를 입력 해야 하므로 첫 번째 출력 데이터 집합으로 전송 된 행의 비율을 나타냅니다.
     
     예를 들어 값으로 0.75를 입력 하면 데이터 집합은 75:25 비율을 사용 하 여 첫 번째 출력 데이터 집합으로 전송 되는 행의 75%가 고 25% 두 번째 출력 데이터 집합으로 전송 하 여 분할 되며 합니다.
  
4. 선택 된 **무작위 분할** 옵션이 선택 된 데이터는 두 그룹으로 임의로 지정 하려는 경우. 학습 및 테스트 데이터 집합을 만들 때 기본 옵션입니다.

5.  **임의 초기값**: 의사 난수 시퀀스 사용할 인스턴스를 초기화 하는 음수가 아닌 정수 값을 입력 합니다. 이 기본 초기값은 난수를 생성 하는 모든 모듈에 사용 됩니다. 

     일반적으로 재현 가능한 결과 통해 초기값을 지정 합니다. 분할 작업의 결과 반복 해야 할 경우 난수 생성기의 초기값을 지정 해야 합니다. 무작위 초기값 0으로, 시스템 클록에서 최초 초기값 가져온 의미를 기본적으로 설정 됩니다. 결과적으로, 데이터 분포 분할을 수행할 때마다 약간 다른 수. 있습니다 

6. **층 화 분할**: 이 옵션을 설정 **True** 라는 두 개의 출력 데이터 집합에 있는 값의 대표적인 샘플을 포함 하도록 합니다 *층 열* 하거나 *층 화 키 열*합니다. 

    층 화 샘플링을 사용 하 여 데이터에는 각 출력 데이터 집합을 각 대상 값과 동일한 비율로 가져옵니다 대략적으로 나뉩니다. 예를 들어, 교육 및 테스트 집합의 결과 관련 하 여 분산 대략적으로 확인 하려고 하거나 사용 하 여 간주 ot gender와 같은 일부 다른 열 수 있습니다.

7. 실험을 실행합니다.


## <a name="regular-expression-split"></a>정규식 분할

1.  추가 합니다 [데이터 분할](./split-data.md) 모듈을 실험에 분할 하려는 데이터 집합에 대 한 입력으로 연결 합니다.  
  
2.  에 대 한 **분할 모드**를 선택 **정규식 분할**합니다.

3. 에 **정규식** 상자에서 유효한 정규식을 입력 합니다. 
  
   정규식은 Python 정규식 구문을 따라야 합니다.


4. 실험을 실행합니다.

    제공 된 정규식에 따라 데이터 집합 두 행 집합으로 구분 됩니다: 식과 나머지 모든 행과 일치 하는 값을 가진 행입니다. 

## <a name="relative-expression-split"></a>상대 식 분할 합니다.

1. 추가 합니다 [데이터 분할](./split-data.md) 모듈을 실험에 분할 하려는 데이터 집합에 대 한 입력으로 연결 합니다.
  
2. 에 대 한 **분할 모드**를 선택 **상대식 분할**합니다.
  
3. 에 **관계식** 텍스트 상자에 단일 열에는 비교 연산을 수행 하는 식을 입력 합니다.


 - 숫자 열:
    - 열의 날짜/시간 데이터 형식을 비롯 한 모든 숫자 데이터 형식의 숫자를 포함 합니다.

    - 식의 열 이름 하나를 참조할 수 있습니다.

    - 앰퍼샌드 문자를 사용 (&) OR 연산에 파이프 문자 (|) 및 작업을 사용 합니다.

    - 다음 연산자가 지원 됩니다. `<`, `>`, `<=`, `>=`, `==`, `!=`

    - 사용 하 여 작업을 그룹화 할 수 없습니다 `(` 고 `)`입니다.

 - 문자열 열: 
    - 다음 연산자가 지원 됩니다. `==`, `!=`



4. 실험을 실행합니다.

    식 두 행 집합으로 데이터 집합을 나눕니다: 조건을 충족 하는 값을 가진 행과 나머지 모든 행입니다.
