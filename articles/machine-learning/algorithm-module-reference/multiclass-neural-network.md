# <a name="multiclass-neural-network-module"></a>다중 클래스 신경망 모듈

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

이 모듈을 사용 하 여 여러 값이 포함 된 대상을 예측 하는 데 사용할 수 있는 신경망 모델을 만듭니다. 

예를 들어, 숫자 또는 문자 인식, 문서 분류, 패턴 인식 등의 복잡 한 컴퓨터 비전 작업에서 이러한 종류의 신경망을 사용할 수 있습니다.

신경망을 사용 하 여 분류는 지도 학습 방법 이며 하므로 *데이터 집합을 태그가 지정 된* 레이블 열을 포함 하는 합니다.

모델 및 태그가 지정 된 데이터 집합에 대 한 입력으로 제공 하 여 모델을 학습할 수 있습니다 [모델 학습](./train-model.md)합니다. 그런 다음 학습 된 모델은 새 입력된 예제에 대 한 값을 예측 하 사용할 수 있습니다.  

## <a name="about-neural-networks"></a>신경망에 대 한

신경망은 상호 연결 된 계층 집합입니다. 첫 번째 계층은 입력과 가중치가 적용 된 지 및 노드 수로 구성 하는 방향이 있는 비순환 그래프에서 출력 계층에 연결 됩니다.

입력 및 출력 계층 간에 여러 숨겨진된 계층을 삽입할 수 있습니다. 대부분의 예측 작업은 숨겨진된 계층을 하나 또는 몇 개만 사용 하 여 쉽게 수행할 수 있습니다. 그러나 최근 연구에 따르면 많은 계층을 사용 하 여 deep neural network DNN ()는 이미지 또는 음성 인식과 같은 복잡 한 작업 효율적일 수 있습니다. 후속 레이어는 계속 증가 의미 수준을 모델링 하는 데 사용 됩니다.

입력과 출력 간의 관계는 입력된 데이터에 대해 신경망 학습에서 확인 됩니다. 출력 계층 및 숨겨진된 계층을 통해 입력에서 그래프의 방향을 진행 됩니다. 계층의 모든 노드는 다음 계층의 노드로 가중치가 적용 된 지를 통해 연결 됩니다.

특정 입력에 대 한 네트워크의 출력에 계산 하려면 숨겨진 계층과에서 출력 계층의 각 노드에서 값을 계산 합니다. 값은 이전 계층에서 노드 값의 가중치 합을 계산 하 여 설정 됩니다. 활성화 함수는 해당 가중치 합에 적용 됩니다.

## <a name="configure-multiclass-neural-network"></a>다중 클래스 신경망 네트워크를 구성 합니다.

1. 추가 된 **다중 클래스 신경망** 모듈 인터페이스에서 실험을 합니다. 이 모듈에서 찾을 수 있습니다 **Machine Learning**, **초기화**에서 합니다 **분류** 범주입니다.

2. **트레이너 모드 만들기**: 모델이 학습 되는 방법을 지정 하려면이 옵션을 사용 합니다.

    - **매개 변수를 단일**: 모델을 구성 하려는 방법을 알고 있는 경우이 옵션을 선택 합니다.

    

3. **숨겨진 계층 사양**: 만들려는 네트워크 아키텍처의 유형을 선택 합니다.

    - **대/소문자를 완벽 하 게 연결**: 기본 신경망 아키텍처를 사용 하 여 모델을 만들려면이 옵션을 선택 합니다. 다중 클래스 신경망 모델에 대 한 기본값은 다음과 같습니다.

        - 숨겨진된 계층 하나가
        - 출력 계층은 숨겨진된 계층에 완전히 연결 됩니다.
        - 숨겨진된 계층은 입력된 계층에 완전히 연결 됩니다.
        - 입력된 계층의 노드 수는 학습 데이터의 기능 수로 결정 됩니다.
        - 숨겨진된 계층의 노드 수는 사용자가 설정할 수 있습니다. 기본값은 100입니다.
        - 출력 계층의 노드 수가 클래스의 수에 따라 달라 집니다.
  
   

5. **숨겨진된 노드 수가**: 이 옵션의 기본 아키텍처에서 숨겨진 노드의 수를 사용자 지정할 수 있습니다. 숨겨진 노드의 수를 입력 합니다. 기본값은 100 노드와 하나의 숨겨진된 계층이 있습니다.

6. **학습 속도**: 수정 하기 전에 각 반복에서 수행 되는 단계 크기를 정의 합니다. 학습 속도 값이 크면 모델이 빠르게 수렴 하면 되지만 로컬 최소값이 과도 복원 지점을 지나친 수 있습니다.

7. **학습 반복 횟수**: 알고리즘 학습 사례를 처리 해야 하는 최대 횟수를 지정 합니다.

8. **초기 학습 가중치 지름**: 학습 프로세스의 시작 부분에 노드 가중치를 지정 합니다.

9. **운동량**: 이전 반복에서 노드를 학습 하는 동안 적용할 가중치를 지정 합니다.
  
11. **예제 섞기**: 반복 간 사례를 섞 습니다.이 옵션을 선택 합니다.

    이 옵션의 선택을 취소 하면 경우 실험을 실행할 때마다 정확히 동일한 순서로 처리 됩니다.

12. **난수 초기값**: 동일한 실험의 실행에서 반복성을 유지 하려는 경우에 초기값으로 사용할 값을 입력 합니다.

14. 학습 데이터 집합 및 중 하나에 연결 합니다 [학습 모듈](module-reference.md): 

    - 설정 하는 경우 **트레이너 모드 만들기** 하 **단일 매개 변수**를 사용 하 여 [모델 학습](train-model.md)합니다.  
  

## <a name="results"></a>결과

완료 된 후 교육:

- 가중치를 학습 및 신경망 네트워크의 다른 매개 변수에서 배운 기능와 함께 모델의 매개 변수 요약을 볼 출력을 마우스 오른쪽 단추로 클릭 [모델 학습](./train-model.md) 선택한 **시각화**.  

- 학습된 된 모델의 스냅숏을 저장 하려면 마우스 오른쪽 단추로 클릭 합니다 **Trained 모델** 선택한 출력 **학습 된 모델을 저장**합니다. 이 모델은 동일한 실험의 후속 실행 시 업데이트 되지 않습니다.
