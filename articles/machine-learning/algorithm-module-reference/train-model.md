# <a name="train-model-module"></a>모델 학습 모듈

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)의 모듈에 대해 설명합니다.

이 모듈을 사용하여 분류 또는 회귀 모델을 학습합니다. 학습은 모델을 정의하고 매개 변수를 설정한 후 수행되며 태그가 지정된 데이터가 필요합니다. 또한 기차 **모델을** 사용하여 새 데이터로 기존 모델을 다시 학습할 수도 있습니다. 

## <a name="how-the-training-process-works"></a>교육 프로세스의 작동 방식

Azure 기계 학습에서 기계 학습 모델을 만들고 사용하는 것은 일반적으로 3단계 프로세스입니다. 

1. 특정 유형의 알고리즘을 선택하고 매개 변수 또는 하이퍼매개 변수를 정의하여 모델을 구성합니다. 다음 모델 유형 중 하나를 선택합니다. 

    + 신경망, 의사 결정 트리 및 의사 결정 포리스트 및 기타 알고리즘을 기반으로 하는 **분류** 모델입니다.
    + 표준 선형 회귀를 포함하거나 신경망 및 베이지안 회귀를 포함한 다른 알고리즘을 사용하는 **회귀** 모델입니다.  

2. 레이블이 지정되고 알고리즘과 호환되는 데이터가 있는 데이터 집합을 제공합니다. 모델 **학습에**데이터와 모델을 모두 연결합니다.

    학습에서 생성되는 것은 데이터에서 학습한 통계 패턴을 캡슐화하는 특정 이진 형식인 iLearner입니다. 이 형식을 직접 수정하거나 읽을 수 없습니다. 그러나 다른 모듈에서는 이 학습된 모델을 사용할 수 있습니다. 
    
    모델의 속성을 볼 수도 있습니다. 자세한 내용은 결과 섹션을 참조하세요.

3. 교육이 완료된 후 채점 모듈 중 하나와 함께 학습된 모델을 사용하여 새 데이터를 [예측합니다.](./score-model.md)

## <a name="how-to-use-train-model"></a>기차 모형 사용 방법 
  
1.  Azure 기계 학습에서 분류 모델 또는 회귀 모델을 구성합니다.
    
2. **파이프라인에 학습 모델** 모듈을 추가합니다.  이 모듈은 기계 **학습** 범주에서 찾을 수 있습니다. **기차를**확장한 다음 **학습 모델** 모듈을 파이프라인으로 드래그합니다.
  
3.  왼쪽 입력에서 학습되지 않은 모드를 연결합니다. 학습 데이터 집합을 **학습 모델의**오른쪽 입력에 연결합니다.

    학습 데이터 집합에는 레이블 열이 포함되어야 합니다. 레이블이 없는 행은 무시됩니다.
  
4.  **레이블 열의**경우 모듈의 오른쪽 패널에서 **열 편집을** 클릭하고 모델이 교육에 사용할 수 있는 결과가 포함된 단일 열을 선택합니다.
  
    - 분류 문제의 경우 레이블 열에는 **범주값** 또는 **불연속** 값이 포함되어야 합니다. 예/아니요 등급, 질병 분류 코드 또는 이름 또는 소득 그룹등 몇 가지 예가 있습니다.  분류되지 않은 열을 선택하면 학습 중에 모듈에서 오류가 반환됩니다.
  
    -   회귀 문제의 경우 레이블 열에는 응답 변수를 나타내는 **숫자** 데이터가 포함되어야 합니다. 이상적으로 숫자 데이터는 연속 배율을 나타냅니다. 
    
    예를 들어 신용 위험 점수, 하드 드라이브에 대한 실패 예상 시간 또는 지정된 일 또는 시간에 콜 센터에 대한 예상 통화 횟수일 수 있습니다.  숫자 열을 선택하지 않으면 오류가 발생할 수 있습니다.
  
    -   사용할 레이블 열을 지정하지 않으면 Azure 기계 학습에서는 데이터 집합의 메타데이터를 사용하여 적절한 레이블 열을 유추합니다. 잘못된 열을 선택하는 경우 열 선택기를 사용하여 수정합니다.
  
    > [!TIP] 
    > 열 선택기 사용에 문제가 있는 경우 [데이터 집합에서 열 선택](./select-columns-in-dataset.md) 에 대한 팁을 참조하세요. **WITH RULES** 및 **BY NAME** 옵션을 사용하기 위한 몇 가지 일반적인 시나리오와 팁을 설명합니다.
  
5.  파이프라인을 제출합니다. 데이터가 많은 경우 시간이 걸릴 수 있습니다.

## <a name="results"></a><a name="bkmk_results"></a>결과

모델을 학습한 후:


+ 다른 파이프라인에서 모델을 사용하려면 모듈을 선택하고 오른쪽 패널의 출력 탭에서 **데이터 집합 등록** **아이콘을 선택합니다.** **데이터 집합**아래의 모듈 팔레트에서 저장된 모델에 액세스할 수 있습니다.

+ 새 값을 예측할 때 모델을 사용하려면 새 입력 데이터와 함께 [모델 점수](./score-model.md) 매기기 모듈에 연결합니다.
