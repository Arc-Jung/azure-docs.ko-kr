# <a name="train-model-module"></a>모델 학습 모듈

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

이 모듈을 사용 하 여 분류 또는 회귀 모델을 학습 합니다. 학습 모델을 정의 하 고 해당 매개 변수를 설정 후에 발생 하며 태그가 지정 된 데이터입니다. 사용할 수도 있습니다 **모델 학습** 새 데이터로 기존 모델 다시 학습 하려면. 

## <a name="how-the-training-process-works"></a>학습 프로세스의 작동 원리

Azure Machine Learning 만들기 및 기계 학습 모델을 사용 하 여 일반적으로 3 단계로 이루어진 프로세스입니다. 

1. 특정 유형의 알고리즘을 선택 하 고 해당 매개 변수 또는 하이퍼 매개 변수를 정의 하 여 모델을 구성 합니다. 다음 모델 유형 중 하나를 선택 합니다. 

    + **분류** 신경망, 의사 결정 트리 및 의사 결정 포리스트 및 기타 알고리즘을 기반으로 모델입니다.
    + **회귀** 표준 선형 회귀를 포함할 수 있는 또는 베이 지 언 회귀 및 신경망을 비롯 한 다른 알고리즘을 사용 하는 모델입니다.  

2. 레이블이 지정 되 고 데이터가 알고리즘을 사용 하 여 호환 되는 데이터 집합을 제공 합니다. 데이터와 모델에 연결 **모델 학습**합니다.

    특정 이진 형식으로 데이터에서 배운 통계 패턴을 캡슐화 하는 iLearner는 어떤 교육을 생성 합니다. 직접 수정 하거나이 형식으로 읽을 수 없습니다. 그러나 다른 모듈 학습 된 모델을 사용할 수 있습니다. 
    
    또한 모델의 속성을 볼 수 있습니다. 자세한 내용은 결과 섹션을 참조 하세요.

3. 학습이 완료 된 다음 중 하나를 사용 하 여 학습 된 모델을 사용 합니다 [모듈을 점수 매기기](./score-model.md), 새 데이터에 예측을 수행 하 합니다.

## <a name="how-to-use-train-model"></a>사용 하는 방법을 **모델 학습**  
  
1.  Azure Machine Learning에서 모델 분류 또는 회귀 모델을 구성 합니다.
    
2. 추가 된 **모델 학습** 모듈을 실험 합니다.  이 모듈에서 찾을 수 있습니다 합니다 **Machine Learning** 범주입니다. 확장 **학습**를 끌어서 놓습니다 합니다 **모델 학습** 실험에 모듈입니다.
  
3.  왼쪽 입력에서 학습 되지 않은 모드를 연결 합니다. 학습 데이터 집합의 오른쪽 입력에 연결 **모델 학습**합니다.

    학습 데이터 집합에 레이블 열을 포함 해야 합니다. 레이블이 없는 모든 행은 무시 됩니다.
  
4.  에 대 한 **레이블 열**, 클릭 **열 선택기 시작**, 결과 모델 학습에 사용할 수 있는 단일 열을 선택 합니다.
  
    - 분류 문제의 경우 레이블 열 포함 해야 **범주** 값 또는 **불연속** 값입니다. 몇 가지 예는 예/아니요 등급, 질병 분류 코드 또는 이름 또는 소득 그룹이 수 있습니다.  범주가 아닌 열을 선택 하면 모듈을 학습 하는 동안 오류가 반환 됩니다.
  
    -   회귀 문제의 경우 레이블 열이 포함 되어야 합니다 **숫자** 응답 변수를 나타내는 데이터입니다. 이상적으로 숫자 데이터는 연속적인 눈금을 나타냅니다. 
    
    예로 예상된 실패 시간 하드 드라이브 또는 콜 센터 통화 횟수 예측된에 대 한 특정된 날짜 또는 시간에는 신용 위험 점수를 수 있습니다.  숫자 열을 선택 하지 않으면 오류가 표시 될 수 있습니다.
  
    -   사용 하 여 레이블 열을 지정 하지 않으면 Azure Machine Learning 데이터 집합의 메타 데이터를 사용 하 여 적절 한 레이블 열을를 유추 하려고 합니다. 잘못 된 열을 선택 하는 경우 열 선택기를 사용 하 여 수정 해야 합니다.
  
    > [!TIP] 
    > 열 선택기를 사용 하는 데 문제가 있으면 문서를 참조 [Select Columns in Dataset](./select-columns-in-dataset.md) 팁에 대 한 합니다. 몇 가지 일반적인 시나리오 및 사용을 위한 팁에 설명 합니다는 **사용 하 여 규칙** 하 고 **이름순** 옵션입니다.
  
5.  실험을 실행합니다. 데이터가 많은 경우이 걸릴 수 있습니다.

## <a name="bkmk_results"></a> 결과

후 모델을 학습 합니다.

+ 모델 매개 변수 및 기능 가중치를 확인 하려면 선택한 출력을 마우스 오른쪽 단추로 **시각화**합니다.
+ 다른 실험에서 모델을 사용 하려면 모델을 마우스 오른쪽 단추로 클릭 하 고 선택 **모델 저장**합니다. 모델의 이름을 입력 합니다. 

    이 실험의 반복된 실행 하 여 업데이트 되지 않는 스냅숏으로 모델을 저장 합니다.
+ 새 값을 예측 모델을 사용 하려면 연결 된 [모델 점수 매기기](./score-model.md) 새 입력된 데이터와 함께 모듈입니다.
