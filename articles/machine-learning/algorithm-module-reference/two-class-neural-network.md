# <a name="two-class-neural-network-module"></a>2클래스 신경망 모듈

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)의 모듈에 대해 설명합니다.

이 모듈을 사용하여 값이 두 개만 포함된 대상을 예측하는 데 사용할 수 있는 신경망 모델을 만듭니다.

신경망을 사용하는 분류는 감독된 학습 방법이므로 레이블 열을 포함하는 *태그가 지정된 데이터 집합이*필요합니다. 예를 들어 이 신경망 모델을 사용하여 환자가 특정 질병을 앓고 있는지 여부 또는 지정된 기간 내에 컴퓨터가 실패할 가능성이 있는지 여부와 같은 이진 결과를 예측할 수 있습니다.  

모델을 정의한 후 태그가 지정된 데이터 집합과 모델을 학습 모델에 대한 입력으로 제공하여 모델을 [학습합니다.](./train-model.md) 그런 다음 학습된 모델을 사용하여 새 입력에 대한 값을 예측할 수 있습니다.

### <a name="more-about-neural-networks"></a>신경망에 대해 자세히 알아보기

신경망은 상호 연결된 계층의 집합입니다. 입력은 첫 번째 레이어이며 가중치 가중 모서리 및 노드로 구성된 비순환 그래프로 출력 레이어에 연결됩니다.

입력 레이어와 출력 레이어 사이에 여러 숨겨진 레이어를 삽입할 수 있습니다. 대부분의 예측 작업은 숨겨진 계층을 하나 또는 몇 개만 사용하여 쉽게 완료할 수 있습니다. 그러나 최근 연구에 따르면 여러 계층을 가진 심층 신경망(DNN)이 이미지 나 음성 인식과 같은 복잡한 작업에 효과적일 수 있습니다. 연속된 레이어는 증가하는 의미 깊이 수준을 모델링하는 데 사용됩니다.

입력과 출력 간의 관계는 입력 데이터에 대한 신경망을 학습하여 학습합니다. 그래프의 방향은 입력에서 숨겨진 레이어를 통해 출력 레이어로 진행됩니다. 레이어의 모든 노드는 가중치가 있는 모서리에 의해 다음 레이어의 노드에 연결됩니다.

특정 입력에 대한 네트워크 출력을 계산하기 위해 숨겨진 레이어의 각 노드와 출력 계층에서 값이 계산됩니다. 값은 이전 계층에서 노드 값의 가중 합계를 계산하여 설정됩니다. 그런 다음 활성화 함수가 해당 가중 합계에 적용됩니다.
  
## <a name="how-to-configure"></a>구성 방법

1.  파이프라인에 **2클래스 신경망** 모듈을 추가합니다. 이 모듈은 **기계 학습,** **초기화,** **분류** 범주에서 찾을 수 있습니다.  
  
2.  **트레이너 만들기 모드** 만들기 옵션을 설정하여 모델을 학습할 방법을 지정합니다.  
  
    -   **단일 매개 변수**: 모델을 구성하는 방법을 이미 알고 있는 경우 이 옵션을 선택합니다.

    -   **매개 변수 범위**: 최상의 매개 변수를 잘 모르는 경우 모델 하이퍼 매개 변수 조정 모듈을 사용하여 최적의 매개 변수를 찾을 수 [있습니다.](tune-model-hyperparameters.md) 일부 값 범위를 제공하고 트레이너는 여러 설정 조합을 통해 이월하여 최상의 결과를 생성하는 값의 조합을 결정합니다.  

3.  **숨겨진 계층 사양의**경우 만들 네트워크 아키텍처 유형을 선택합니다.  
  
    -   **완전히 연결된 사례**: 다음과 같이 2클래스 신경망에 대해 정의된 기본 신경망 아키텍처를 사용합니다.
  
        -   하나의 숨겨진 레이어가 있습니다.
  
        -   출력 계층은 숨겨진 계층에 완전히 연결되며 숨겨진 계층은 입력 계층에 완전히 연결됩니다.
  
        -   입력 계층의 노드 수는 학습 데이터의 피처 수와 같습니다.
  
        -   숨겨진 계층의 노드 수는 사용자가 설정합니다. 기본값은 100입니다.
  
        -   노드 수는 클래스 수와 같습니다. 2클래스 신경망의 경우 모든 입력이 출력 계층의 두 노드 중 하나에 매핑되어야 합니다.

5.  **학습 속도의**경우 수정 하기 전에 각 반복에서 수행 하는 단계의 크기를 정의 합니다. 학습 속도 값이 크면 모델이 빠르게 수렴되지만 로컬 최소값이 과도해질 수 있습니다.

6.  **학습 반복 횟수의**경우 알고리즘이 학습 사례를 처리해야 하는 최대 횟수를 지정합니다.

7.  **초기 학습 가중치 지름의**경우 학습 프로세스시작 시 노드 가중치를 지정합니다.

8.  **모멘텀의**경우 이전 반복의 노드에 학습하는 동안 적용할 가중치를 지정합니다.  

10. 반복 간에 사례를 섞는 **경우 보기 선택** 옵션을 선택합니다. 이 옵션을 선택 취소하면 파이프라인을 실행할 때마다 서비스 케이스가 정확히 동일한 순서로 처리됩니다.
  
11. **난수 시드의**경우 시드로 사용할 값을 입력합니다.
  
     시드 값을 지정하면 동일한 파이프라인의 실행에서 반복성을 보장하려는 경우에 유용합니다.  그렇지 않으면 시스템 클럭 값이 시드로 사용되어 파이프라인을 실행할 때마다 약간 다른 결과가 발생할 수 있습니다.
  
13. 태그가 지정된 데이터 집합을 파이프라인에 추가하고 교육 모듈 중 하나를 [연결합니다.](module-reference.md)  
  
    -   **트레이너 만들기 모드를** 단일 매개 **변수로**설정하면 [학습 모델](train-model.md) 모듈을 사용합니다.  
  
14. 파이프라인을 제출합니다.

## <a name="results"></a>결과

교육이 완료된 후:

+ 학습된 모델의 스냅샷을 저장하려면 **학습 모델** 모듈의 오른쪽 패널에서 **출력** 탭을 선택합니다. 모델을 재사용 가능한 모듈로 저장하려면 **데이터 집합 등록** 아이콘을 선택합니다.

+ 점수를 매기기 위해 모델을 사용하려면 **파이프라인에 모델 점수 매기기** 모듈을 추가합니다.
