# <a name="two-class-neural-network-module"></a>2 클래스 신경망 모듈

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

이 모듈을 사용 하 여 두 값만 포함 된 대상을 예측 하는 신경망 모델을 만듭니다.

신경망을 사용 하 여 분류는 지도 학습 방법 이며 하므로 *데이터 집합 태그*, 레이블 열을 포함 하는 합니다. 예를 들어 여부 환자에 게 특정 질환이, 같은 또는 컴퓨터를 지정 된 기간 내에 실패할 가능성이 있는지 여부를 이진 결과 예측에 신경망 모델에이 사용할 수 있습니다.  

모델을 정의한 후에 대 한 입력으로 태그가 지정 된 데이터 집합 및 모델을 제공 하 여 학습 [모델 학습](./train-model.md)합니다. 그런 다음 학습된 된 모델에 대 한 새 입력 값을 예측 하 사용할 수 있습니다.

### <a name="more-about-neural-networks"></a>신경망에 대 한 자세한 정보

신경망은 상호 연결 된 계층 집합입니다. 첫 번째 계층은 입력과 가중치가 적용 된 지 및 노드 수로 구성 하는 방향이 있는 비순환 그래프에서 출력 계층에 연결 됩니다.

입력 및 출력 계층 간에 여러 숨겨진된 계층을 삽입할 수 있습니다. 대부분의 예측 작업은 숨겨진된 계층을 하나 또는 몇 개만 사용 하 여 쉽게 수행할 수 있습니다. 그러나 최근 연구에 따르면 많은 계층을 사용 하 여 deep neural network DNN ()는 이미지 또는 음성 인식과 같은 복잡 한 작업 효율적일 수 있습니다. 후속 레이어는 계속 증가 의미 수준을 모델링 하는 데 사용 됩니다.

입력과 출력 간의 관계는 입력된 데이터에 대해 신경망 학습에서 확인 됩니다. 출력 계층 및 숨겨진된 계층을 통해 입력에서 그래프의 방향을 진행 됩니다. 계층의 모든 노드는 다음 계층의 노드로 가중치가 적용 된 지를 통해 연결 됩니다.

특정 입력에 대 한 네트워크의 출력에 계산 하려면 숨겨진 계층과에서 출력 계층의 각 노드에서 값을 계산 합니다. 값은 이전 계층에서 노드 값의 가중치 합을 계산 하 여 설정 됩니다. 활성화 함수는 해당 가중치 합에 적용 됩니다.
  
## <a name="how-to-configure"></a>구성 방법

1.  추가 된 **2 클래스 신경망** 모듈을 실험 합니다. 이 모듈에서 찾을 수 있습니다 **Machine Learning**, **초기화**에서 합니다 **분류** 범주입니다.  
  
2.  설정 하 여 학습 모델을 지정 합니다 **트레이너 모드 만들기** 옵션입니다.  
  
    -   **매개 변수를 단일**: 모델을 구성 하려는 방법을 알고 있는 경우이 옵션을 선택 합니다.  

3.  에 대 한 **숨겨진 계층 사양**를 만들려면 네트워크 아키텍처의 유형을 선택 합니다.  
  
    -   **대/소문자를 완벽 하 게 연결**: 다음과 같이 2 클래스 신경망에 대 한 정의 기본 신경망 아키텍처를 사용 합니다.
  
        -   에 하나의 숨겨진된 계층이 있습니다.
  
        -   출력 계층은 숨겨진된 계층에 완전히 연결 하 고 숨겨진된 계층은 입력된 계층에 완전히 연결 됩니다.
  
        -   입력된 계층의 노드 수를는 학습 데이터의 기능 수를 같습니다.
  
        -   숨겨진된 계층의 노드 수는 사용자가 설정 됩니다. 기본값은 100입니다.
  
        -   노드 수가 클래스 수를 같습니다. 2 클래스 신경망에 대 한 모든 입력을 출력 계층의 두 노드 중 하나에 매핑해야 하을 의미 합니다.

5.  에 대 한 **학습 속도**를 수정 하기 전에 각 반복에서 수행 되는 단계 크기를 정의 합니다. 학습 속도 값이 크면 모델이 빠르게 수렴 하면 되지만 로컬 최소값이 과도 복원 지점을 지나친 수 있습니다.

6.  에 대 한 **학습 반복 횟수**, 알고리즘에서 학습 사례를 처리 해야 하는 최대 횟수를 지정 합니다.

7.  에 대 한 **초기 학습 가중치 지름**, 학습 프로세스의 시작 부분에 노드 가중치를 지정 합니다.

8.  에 대 한 **운동량**, 이전 반복에서 노드를 학습 하는 동안 적용할 가중치를 지정 합니다.  

10. 선택 된 **예제 섞기** 반복 간 사례를 섞을 수 있습니다. 이 옵션의 선택을 취소 하면 경우 실험을 실행할 때마다 정확히 동일한 순서로 처리 됩니다.
  
11. 에 대 한 **임의 숫자 초기값**를 초기값으로 사용할 값을 입력 합니다.
  
     초기값을 지정 값은 유용 동일한 실험의 실행에서 반복성을 유지 하려는 경우.  그렇지 않으면 시스템 클록 실험을 실행할 때마다 약간 다른 결과 일으킬 수 있는 초기값으로 사용 됩니다.
  
13. 태그가 지정 된 데이터 집합을 실험에 추가 하 고 연결 중 하나는 [학습 모듈](module-reference.md)합니다.  
  
    -   설정 하는 경우 **트레이너 모드 만들기** 를 **단일 매개 변수**를 사용 합니다 [모델 학습](train-model.md) 모듈입니다.  
  
14. 실험을 실행합니다.

## <a name="results"></a>결과

완료 된 후 교육:

+ 가중치를 학습 및 신경망 네트워크의 다른 매개 변수에서 배운 기능와 함께 모델의 매개 변수 요약을 볼 출력을 마우스 오른쪽 단추로 클릭 [모델 학습](./train-model.md), 선택한 **시각화**.  

+ 학습된 된 모델의 스냅숏을 저장 하려면 마우스 오른쪽 단추로 클릭 합니다 **Trained 모델** 선택한 출력 **학습 된 모델을 저장**합니다. 이 모델은 동일한 실험의 후속 실행 시 업데이트 되지 않습니다.
