# <a name="clip-values"></a>값 잘라내기

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)의 모듈에 대해 설명합니다.

클립 값 모듈을 사용하여 지정된 임계값 위 또는 아래에 있는 데이터 값을 평균, 상수 또는 기타 대체 값으로 식별하고 선택적으로 바꿉니다.  

모듈을 클립할 숫자가 있는 데이터 집합에 연결하고 작업할 열을 선택한 다음 임계값 또는 값 범위 및 대체 방법을 설정합니다. 모듈은 결과만 출력하거나 원래 데이터 집합에 추가된 변경된 값을 출력할 수 있습니다.

## <a name="how-to-configure-clip-values"></a>클립 값을 구성하는 방법

시작하기 전에 클립할 열과 사용할 방법을 식별합니다. 먼저 데이터의 작은 하위 집합에서 클리핑 메서드를 테스트하는 것이 좋습니다.

모듈은 선택 영역에 포함된 **모든** 열에 동일한 기준 및 대체 방법을 적용합니다. 따라서 변경하지 않으려는 열은 제외해야 합니다.

일부 열에 클리핑 메서드 또는 다른 조건을 적용해야 하는 경우 유사한 열의 각 집합에 대해 **새 클립 값** 인스턴스를 사용해야 합니다.

1.  **파이프라인에 클립 값** 모듈을 추가하고 수정하려는 데이터 집합에 연결합니다. **데이터 변환,** **배율 및 축소** 범주에서 이 모듈을 찾을 수 있습니다. 
  
1.  **열 목록에서**열 선택기를 사용하여 **클립 값이** 적용될 열을 선택합니다.  
  
1.  임계값 집합의 경우 드롭다운 목록에서 다음 옵션 중 하나를 **선택합니다.** 이러한 옵션은 잘라야 하는 값과 허용 가능한 값에 대해 상한 및 하위 경계를 설정하는 방법을 결정합니다.  
  
    - **ClipPeaks**: 피크별로 값을 클립할 때 상위 경계만 지정합니다. 해당 경계 값보다 큰 값이 대체됩니다.
  
    -  **ClipSubpeaks**: 하위 피크별로 값을 클립할 때 더 낮은 경계만 지정합니다. 해당 경계 값보다 적은 값이 대체됩니다.  
  
    - **ClipPeaksAndSubpeaks**: 피크와 하위 피크별로 값을 클립할 때 위쪽 및 하한 경계를 모두 지정할 수 있습니다. 해당 범위를 벗어난 값은 대체됩니다. 경계 값과 일치하는 값은 변경되지 않습니다.
  
1.  이전 단계에서 선택한 항목에 따라 다음 임계값을 설정할 수 있습니다. 

    + **낮은 임계값**: **클립서브픽스를** 선택한 경우에만 표시됩니다.
    + **상한값**: **클립피크를** 선택한 경우에만 표시됩니다.
    + **임계값**: **클립픽스및서브픽스를** 선택한 경우에만 표시됩니다.

    각 임계값 유형에 대해 **상수** 또는 **백분위수**중 하나를 선택합니다.

1. **상수를**선택하면 텍스트 상자에 최대 값 또는 최소 값을 입력합니다. 예를 들어 999 값이 자리 표시자 값으로 사용된 것을 알고 있다고 가정합니다. 상한 임계값에 대해 **상수를** 선택하고 **상한값에 대해 상수 값에 999를 입력할**수 있습니다.
  
1. **백분위수를**선택하면 열 값을 백분위수 범위로 제한합니다. 

    예를 들어 10-80 백분위수 범위의 값만 유지하고 다른 값을 모두 바꾸려고 한다고 가정합니다. **백분위수를**선택한 다음 낮은 **임계값에 대해 백분위수 값에**대해 10을 입력하고 상위 임계값에 대해 **백분위수 값에**대해 80을 입력합니다. 

    백분위수 범위를 사용하는 방법에 대한 몇 가지 예는 [백분위수](#examples-for-clipping-using-percentiles) 섹션을 참조하십시오.  
  
1.  대체 값을 정의합니다.

    지정한 경계와 정확히 일치하는 숫자는 허용된 값 범위 내에 있는 것으로 간주되므로 대체되지 않습니다. 지정된 범위를 벗어난 모든 숫자는 대체 값으로 바뀝습니다. 
  
    + **피크에 대한 대체 값**: 지정된 임계값보다 큰 모든 열 값을 대체할 값을 정의합니다.  
    + **하위 피크에 대한 대체 값**: 지정된 임계값보다 작은 모든 열 값의 대체값으로 사용할 값을 정의합니다.  
    + **ClipPeaksAndSubpeaks** 옵션을 사용하는 경우 위쪽 및 아래쪽 클리핑된 값에 대해 별도의 대체 값을 지정할 수 있습니다.  

    다음 대체 값이 지원됩니다.  
  
    -   **임계값**: 잘린 값을 지정된 임계값으로 바꿉습니다.  
  
    -   **평균**: 잘린 값을 열 값의 평균으로 바꿉습니다. 값이 잘리기 전에 평균이 계산됩니다.  
  
    -   **중앙값**: 잘린 값을 열 값의 중앙값으로 바꿉습니다. 값이 잘리기 전에 중앙값이 계산됩니다.   
  
    -   **누락된**. 잘린 값을 누락된(빈) 값으로 바꿉니다.  
  
1.  **표시기 열 추가**: 지정된 클리핑 작업이 해당 행의 데이터에 적용되는지 여부를 알려주는 새 열을 생성하려면 이 옵션을 선택합니다. 이 옵션은 새 클리핑 및 대체 값 집합을 테스트할 때 유용합니다.  
  
1. **플래그 덮어쓰기**: 새 값을 생성할 방법을 나타냅니다. 기본적으로 **클립 값은** 원하는 임계값에 잘린 피크 값을 사용하여 새 열을 생성합니다. 새 값은 원래 열을 덮어씁니다.  
  
    원래 열을 유지하고 잘린 값이 있는 새 열을 추가하려면 이 옵션을 선택 해제합니다.  
  
1.  파이프라인을 제출합니다.  
  
    **클립 값** 모듈을 마우스 오른쪽 단추로 클릭하고 **모듈을 시각화하거나** 선택하고 오른쪽 패널의 **출력** 탭으로 전환하고 **포트 출력의**히스토그램 아이콘을 클릭하여 값을 검토하고 클리핑 작업이 기대에 부합하는지 확인합니다.  
 
### <a name="examples-for-clipping-using-percentiles"></a>백분위수를 사용하여 클리핑하는 예

백분위수를 사용한 자르기의 작동 방식을 이해하기 위해 행이 10개인 데이터 집합의 각 행에 값 1-10이 하나씩 포함되어 있다고 가정해 보겠습니다.  
  
- 상한 임계값으로 백분위수를 사용하는 경우 90번째 백분위수의 값에서 데이터 집합 내 모든 값의 90%는 해당 값보다 작아야 합니다.  
  
- 하한 임계값으로 백분위수를 사용하는 경우 10번째 백분위수의 값에서 데이터 집합 내 모든 값의 10%는 해당 값보다 작아야 합니다.  
  
1.  **임계값 집합의**경우 **클립픽스And서브픽스를 선택합니다.**  
  
1.  **상한값의**경우 **백분위수**및 **백분위수**, 유형 90을 선택합니다.  
  
1.  **대체 값의 경우**누락 **값을**선택합니다.  
  
1.  **하한 임계값**에는 **백분위수**를 선택하고 **백분위수 값**으로는 10을 입력합니다.  
  
1.  **대체 값이 낮을수록** **누락 값을**선택합니다.  
  
1.  옵션 **덮어쓰기 플래그를**선택 취소하고 옵션, **표시기 추가 열을**선택합니다.  
  
이제 60을 상위 백분위수 임계값으로 사용하고 30을 낮은 백분위수 임계값으로 사용하여 동일한 파이프라인을 시도하고 임계값값을 대체 값으로 사용합니다. 다음 표에는 두 결과를 비교한 내용이 나와 있습니다.  
  
1.  누락된 것으로 바꿉습니다. 상한값 = 90; 낮은 임계값 = 20  
  
1.  임계값으로 바꿉습니다. 위백분위수 = 60; 낮은 백분위수 = 40
