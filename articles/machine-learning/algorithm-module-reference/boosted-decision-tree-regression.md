# <a name="boosted-decision-tree-regression-module"></a>승격 된 의사 결정 트리 회귀 모듈

이 문서에서는 Azure Machine Learning 서비스에 대 한 시각적 인터페이스 (미리 보기)의 모듈을 설명 합니다.

사용 하 여 회귀 트리의 앙상블을 만드는 데이 모듈을 사용 합니다. *부스트* 각 트리가 이전 트리 종속 임을 의미 합니다. 나머지 부분 트리를 앞에 오는 것으로 알고리즘은 학습 합니다. 따라서 작은 위험이 적은 검사를 사용 하 여 정확도 개선 하는 의사 결정 트리 앙상블 승격 되 면 합니다.  
  
이 회귀 메서드는 지도 학습 방법 이며 하므로 *데이터 집합에 레이블이 지정 된*합니다. 레이블 열 숫자 값을 포함 해야 합니다.  

> [!NOTE]
> 숫자 변수를 사용 하는 데이터 집합에만이 모듈을 사용 합니다.  

모델을 정의한 후 사용 하 여 학습 된 [모델 학습](./train-model.md)합니다.

> [!TIP]
> 생성 된 트리에 대 한 자세히 알고 싶으세요? 모델을 학습 한 후 출력을 마우스 오른쪽 단추로 클릭 합니다 [모델 학습](./train-model.md) 모듈과 선택 **시각화** 각 반복에 대해 생성 된 트리를 보려면. 각 트리에 대 한 분할으로 드릴 다운 하 고 각 노드에 대 한 규칙을 볼 수 있습니다.

## <a name="how-to-configure-boosted-decision-tree-regression"></a>승격 된 의사 결정 트리 회귀를 구성 하는 방법

1.  추가 된 **승격 된 의사 결정 트리** 모듈을 실험 합니다. 이 모듈에서 찾을 수 있습니다 **Machine Learning**, **초기화**아래에 있는 합니다 **회귀** 범주입니다. 
  
2.  설정 하 여 학습 모델을 지정 합니다 **트레이너 모드 만들기** 옵션입니다.  
  
    -   **매개 변수를 단일**: 모델을 구성 하 고 값을 인수로 특정 집합을 제공 하고자 하는 방법을 알고 있는 경우이 옵션을 선택 합니다.  
   
  
3. **트리 당 리프의 최대**: 트리의 만들 수 있는 터미널 노드 (리프)의 최대 수를 나타냅니다.  

    이 값을 늘려 잠재적으로 트리의 크기를 늘릴을 높아지지만 학습 시간이 길어지고 과잉 맞춤이 향상 전체 자릿수를 가져옵니다.  

4. **리프 노드당 샘플의 최소 수**: 트리의 터미널 노드 (리프)를 만드는 데 필요한 최대 사례 최소 수를 나타냅니다.

    이 값을 늘려 새 규칙 만들기에 대 한 임계값을 늘려야 합니다. 예를 들어, 단일 사례도 1의 기본값을 사용 하 여 새 규칙을 만드는 발생할 수 있습니다. 값을 5로 늘리면 학습 데이터를 동일한 조건을 만족 하는 사례가 다섯 개 이상 포함 해야 합니다.

5. **학습 속도**: 학습 하는 동안 단계 크기를 정의 하는 0과 1 사이의 숫자를 입력 합니다. 학습 속도 학습자 최적의 솔루션에 수렴 속도 결정 합니다. 단계 크기가 너무 큰 경우 최적의 솔루션 복원 지점을 지나친 수 있습니다. 단계 크기를 너무 작은 경우 학습에 가장 적합 한 솔루션에 수렴 오래 걸립니다.

6. **생성 되는 트리의 수**: 앙상블에서 만들려는 의사 결정 트리의 총 수를 나타냅니다. 추가 의사 결정 트리를 만들면 적용 범위가 확대를 잠재적으로 가져올 수 있습니다 있지만 학습 시간이 증가 합니다.

    이 값에는 또한 학습된 된 모델을 시각화 하는 경우에 표시 되는 트리 수를 제어 합니다. 보거나 ingle 트리로 인쇄 하려는 경우 값을 1로 설정할 수 있습니다. 그러나 하나의 트리만 생성 됩니다 (초기 매개 변수 집합을 사용 하 여 트리) 추가 반복에서 수행 됩니다.

7. **난수 초기값**: 난수 초기값으로 사용할 선택적 음수가 아닌 정수를 입력 합니다. 동일한 데이터 및 매개 변수는 실행 간의 재현 가능성을 보장 초기값을 지정 합니다.

    기본적으로 무작위 초기값은 시스템 클록에서 초기 시드 값을 가져오는 것을 의미 하는 0으로 설정 됩니다.
  
8. **알 수 없는 범주 수준 허용**: 학습 및 유효성 검사 집합에서 알 수 없는 값에 대 한 그룹을 만들려면이 옵션을 선택 합니다. 이 옵션의 선택을 취소 하면 모델 학습 데이터에 포함 된 값만 수락할 수 있습니다. 모델을 알려진된 값에 대해 덜 정확한 수 있지만 새로운 (알 수 없는) 값에 대 한 더 나은 예측을 제공할 수 있습니다.

9. 학습 데이터 집합 및 학습 모듈 중 하나를 추가 합니다.

    - 설정 하는 경우 **트레이너 모드 만들기** 옵션을 **단일 매개 변수**를 사용 합니다 [모델 학습](train-model.md) 모듈입니다.  
  
    

10. 실험을 실행합니다.  
  
### <a name="results"></a>결과

완료 된 후 교육:

+ 각 반복에 대해 생성 된 트리를 보려면 출력을 마우스 오른쪽 단추로 클릭 합니다 [모델 학습](train-model.md) 모듈과 선택 **시각화**합니다.
  
     각 트리에 분할으로 드릴 다운 하 여 각 노드에 대 한 규칙을 클릭 합니다.  

+ 모델 점수 매기기를 사용 하려면 연결 [모델 점수 매기기](./score-model.md)를 새로운 입력된 예제에 대 한 값을 예측 합니다.

+ 학습된 된 모델의 스냅숏을 저장 하려면 마우스 오른쪽 단추로 클릭 합니다 **Trained 모델** 학습 모듈을 선택 하는 출력 **다른 이름으로 저장**합니다. 저장 하는 학습된 된 모델의 복사본을 실험의 후속 실행에서 업데이트 되지 않습니다.
