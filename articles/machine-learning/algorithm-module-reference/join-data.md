# <a name="join-data"></a>데이터 조인

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)에서 **데이터 조인** 모듈을 사용하여 데이터베이스 스타일 조인 작업을 사용하여 두 데이터 집합을 병합하는 방법을 설명합니다.  

## <a name="how-to-configure-join-data"></a>조인 데이터를 구성하는 방법

두 데이터 집합에서 조인을 수행하려면 키 열과 관련되어야 합니다. 여러 열을 사용하는 복합 키도 지원됩니다. 

1. 결합할 데이터 집합을 추가한 다음 **데이터 조인** 모듈을 파이프라인으로 드래그합니다. 

    **모듈은 데이터 변환** 범주에서 **조작**에서 찾을 수 있습니다.

1. 데이터 집합을 **데이터 조인** 모듈에 연결합니다. 
 
1. 시작 **열 선택기를** 선택하여 키 열을 선택합니다. 왼쪽 및 오른쪽 입력 모두에 대해 열을 선택해야 합니다.

    단일 키의 경우:

    두 입력에 대해 단일 키 열을 선택합니다.
    
    복합 키의 경우:

    왼쪽 입력에서 모든 키 열을 선택하고 오른쪽 입력을 동일한 순서로 선택합니다. 모든 키 열이 일치하면 **데이터 조인** 모듈이 테이블에 조인됩니다. 열 순서가 원래 테이블과 같지 않은 경우 **중복 허용 및 선택 항목에서 열 순서 유지** 옵션을 선택합니다. 

    ![열 선택기](media/module/join-data-column-selector.png)


1. 텍스트 열 조인에서 대/소문자 구분을 유지하려면 **사례 일치** 옵션을 선택합니다. 
   
1. **조인 유형** 드롭다운 목록을 사용하여 데이터 집합을 결합하는 방법을 지정합니다.  
  
    * **내부 조인**: *내부 조인이* 가장 일반적인 조인 작업입니다. 키 열의 값이 일치하는 경우에만 결합된 행을 반환합니다.  
  
    * **왼쪽 외부 조인**: *왼쪽 외부 조인은* 왼쪽 테이블의 모든 행에 대해 조인된 행을 반환합니다. 왼쪽 테이블의 행에 오른쪽 테이블에 일치하는 행이 없는 경우 반환된 행에는 오른쪽 테이블에서 온 모든 열에 대한 누락된 값이 포함됩니다. 누락된 값에 대한 대체 값을 지정할 수도 있습니다.  
  
    * **전체 외부 조인**: *전체 외부 조인은* 왼쪽 테이블(표1)과 오른쪽 테이블(표2)에서 모든 행을 반환합니다.**table1****table2**  
  
         다른 테이블에 일치하는 행이 없는 두 테이블의 각 행에 대해 결과에 누락된 값이 포함된 행이 포함됩니다.  
  
    * **왼쪽 세미 조인**: *왼쪽 세미 조인은* 키 열의 값이 일치할 때 왼쪽 테이블의 값만 반환합니다.  

1. 옵션의 경우 **조인된 테이블에서 오른쪽 키 열을 유지합니다.**

    * 두 입력 테이블에서 키를 보려면 이 옵션을 선택합니다.
    * 왼쪽 입력에서 키 열만 반환하려면 선택 해제합니다.

1. 파이프라인을 제출합니다.

1. 결과를 보려면 **데이터 조인을** 마우스 오른쪽 단추로 클릭하고 **시각화를**선택합니다.
