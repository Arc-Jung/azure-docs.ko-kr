# <a name="join-data"></a>데이터 조인

이 문서에서는 사용 하는 방법을 설명 합니다 **데이터를 조인** 데이터베이스 스타일 조인 작업을 사용 하는 두 개의 데이터 집합을 병합 하려면 Azure Machine Learning 서비스 시각적 인터페이스에는 모듈입니다.  

## <a name="how-to-configure-join-data"></a>데이터 연결을 구성 하는 방법

두 데이터 집합에 대해 조인을 수행을 하는 키 열에 의해 관련 되어야 합니다. 여러 열을 사용 하 여 복합 키도 지원 됩니다. 

1. 결합 한 다음 끌어 하려는 데이터 집합을 추가 합니다 **조인 데이터** 실험에 모듈입니다. 

    모듈을 찾을 수 있습니다 합니다 **데이터 변환** 범주 아래에 있는 **조작**합니다.

1. 데이터 집합을 연결 합니다 **조인 데이터** 모듈입니다. 
 
1. 선택 **열 선택기 시작** 키 열을 선택 합니다. 왼쪽 및 오른쪽 입력 열을 선택 해야 합니다.

    단일 키:

    두 입력에 대 한 단일 키 열을 선택 합니다.
    
    복합 키:

    왼쪽 입력과 동일한 순서로 오른쪽 입력에서 모든 키 열을 선택 합니다. 합니다 **데이터를 조인** 모듈의 모든 키 열과 일치 하는 경우 테이블을 조인 합니다. 옵션 선택 **중복을 허용 하 고 선택 영역에서 열 순서를 유지** 경우 열 순서는 원래 테이블에서와 동일 하지는 않습니다. 

    ![열 선택기](media/module/join-data-column-selector.png)


1. 선택 된 **대/소문자** 텍스트 열 조인에서 대/소문자를 유지 하려는 경우 옵션입니다. 
   
1. 사용 된 **조인 유형** 데이터 집합을 결합 해야 하는 방법을 지정 하려면 드롭다운 목록입니다.  
  
    * **내부 조인**: *내부 조인* 가장 일반적인 조인 작업입니다. 키 열의 값이 일치 하는 경우에 결합 된 행을 반환 합니다.  
  
    * **왼쪽 우선 외부 조인**: A *왼쪽된 우선 외부 조인을* 조인 된 왼쪽된 테이블의 모든 행에 대 한 행을 반환 합니다. 왼쪽된 테이블의 행 오른쪽 테이블에서 일치 하는 행에 있는 경우 반환 되는 행 오른쪽 테이블에서 제공 하는 모든 열의 누락 값을 포함 합니다. 누락 값에 대 한 대체 값을 지정할 수도 있습니다.  
  
    * **전체 외부 조인**: A *완전 외부 조인* 왼쪽된 테이블의 모든 행을 반환 합니다 (**table1**) 오른쪽 테이블에서 (**table2**).  
  
         다른 일치 하는 행이 있는 테이블 중 하나에서 행의 각각에 대 한 결과 누락 값이 포함 된 행을 포함 합니다.  
  
    * **세미 조인 왼쪽**: A *왼쪽 우선 부분 조인* 키 열의 값이 일치 하면 왼쪽된 테이블의 값만 반환 합니다.  

1. 옵션에 대 한 **조인 된 테이블에서 오른쪽 키 열 유지**:

    * 두 입력된 테이블의 키를 보려면이 옵션을 선택 합니다.
    * 왼쪽된 입력에서 키 열만 반환 하려면 선택을 취소 합니다.

1. 실험을 실행 하거나 데이터를 조인 모듈을 선택 하 고, 선택한 **선택한 실행** 조인을 수행 하 합니다.

1. 결과 보려면 마우스 오른쪽 단추로 클릭 합니다 **데이터 조인** > **결과 데이터 집합** > **시각화**합니다.
