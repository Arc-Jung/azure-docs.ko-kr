# <a name="two-class-boosted-decision-tree-module"></a>2클래스 부스트 의사 결정 트리 모듈

이 문서에서는 Azure 기계 학습 디자이너(미리 보기)의 모듈에 대해 설명합니다.

이 모듈을 사용하여 향상된 의사 결정 트리 알고리즘을 기반으로 하는 기계 학습 모델을 만듭니다. 

증폭된 결정 트리는 두 번째 트리가 첫 번째 트리의 오류를 수정하고 세 번째 트리가 첫 번째 트리와 두 번째 트리의 오류를 수정하는 앙상블 학습 방법입니다.  예측은 예측을 만드는 나무의 전체 앙상블을 기반으로 합니다.
  
일반적으로 제대로 구성된 경우 향상된 의사 결정 트리는 다양한 기계 학습 작업에서 최고의 성능을 얻을 수 있는 가장 쉬운 방법입니다. 그러나 메모리 집약적인 학습자 중 하나이기도 하며 현재 구현은 모든 것을 메모리에 보유합니다. 따라서 향상 된 의사 결정 트리 모델 일부 선형 학습자가 처리할 수 있는 큰 데이터 집합을 처리 하지 못할 수 있습니다.

## <a name="how-to-configure"></a>구성 방법

이 모듈은 학습되지 않은 분류 모델을 만듭니다. 분류는 모델을 학습하기 위해 감독되는 학습 방법이기 때문에 모든 행에 대한 값이 있는 레이블 열이 포함된 *태그가 지정된 데이터 집합이* 필요합니다.

이러한 유형의 모델을 [학습할](././train-model.md)수 있습니다. 

1.  Azure 기계 학습에서 **파이프라인에 향상된 의사 결정 트리** 모듈을 추가합니다.
  
2.  **트레이너 만들기 모드** 만들기 옵션을 설정하여 모델을 학습할 방법을 지정합니다.
  
    + **단일 매개 변수**: 모델을 구성하는 방법을 알고 있는 경우 특정 값 집합을 인수로 제공할 수 있습니다.
  
    + **매개 변수 범위**: 최상의 매개 변수를 잘 모르는 경우 모델 하이퍼 매개 변수 조정 모듈을 사용하여 최적의 매개 변수를 찾을 수 [있습니다.](tune-model-hyperparameters.md) 일부 값 범위를 제공하고 트레이너는 여러 설정 조합을 통해 이월하여 최상의 결과를 생성하는 값의 조합을 결정합니다.
  
3.  **트리당 최대 잎 수의**경우 모든 트리에서 만들 수 있는 터미널 노드(잎)의 최대 수를 나타냅니다.
  
     이 값을 늘리면 트리 크기가 커지고 정밀도는 높아질 수 있지만 학습 시간이 더 길어지고 과잉 맞춤이 발생할 수 있습니다.
  
4.  **리프 노드당 최소 샘플 수의**경우 트리에서 터미널 노드(리프)를 만드는 데 필요한 사례 수를 나타냅니다.  
  
     이 값을 늘려 새 규칙을 작성하기 위한 임계값을 늘립니다. 예를 들어, 기본값이 1이면 단일 사례만으로도 새 규칙을 하나 작성할 수 있습니다. 값을 5로 늘리면 학습 데이터에 동일한 조건을 만족하는 사례가 다섯 개 이상 있어야 합니다.
  
5.  **학습 속도의**경우 학습 하는 동안 단계 크기를 정의 하는 0과 1 사이의 숫자를 입력 합니다.  
  
     학습 속도에 따라 학습자가 최적의 솔루션에 수렴하는 속도 또는 속도가 결정됩니다. 단계 크기가 너무 크면 최적의 솔루션을 오버슈트할 수 있습니다. 단계 크기가 너무 작으면 최상의 솔루션에 수렴하는 데 교육시간이 더 오래 걸립니다.
  
6.  **구성된 트리 수의**경우 앙상블에서 작성할 총 결정 트리 수를 나타냅니다. 추가 의사 결정 트리를 만들면 적용 범위가 확대될 수 있지만 학습 시간이 증가됩니다.
  
     이 값은 학습된 모델을 시각화할 때 표시되는 트리 수도 제어합니다. 단일 트리를 보거나 인쇄하려면 값을 1로 설정합니다. 그러나 이렇게 하면 하나의 트리(초기 매개 변수 집합이 있는 트리)만 생성되고 추가 반복이 수행되지 않습니다.
  
7.  **난수 시드의**경우 선택적으로 음수가 아닌 정수를 입력하여 임의 시드 값으로 사용합니다. 시드를 지정하면 동일한 데이터와 매개 변수가 있는 런에서 재현성이 보장됩니다.  
  
     임의 시드는 기본적으로 0으로 설정되며, 이는 초기 시드 값이 시스템 클럭에서 가져온 것을 의미합니다.  임의 시드를 사용한 연속 실행은 다른 결과를 가질 수 있습니다.
  

9. 모델을 학습시킵니다.
  
    + **트레이너 만들기 모드를** 단일 매개 **변수로**설정하면 태그가 지정된 데이터 집합과 [모델 학습](./train-model.md) 모듈을 연결합니다.  
   
## <a name="results"></a>결과

교육이 완료된 후:

+ 학습된 모델의 스냅샷을 저장하려면 **학습 모델** 모듈의 오른쪽 패널에서 **출력** 탭을 선택합니다. 모델을 재사용 가능한 모듈로 저장하려면 **데이터 집합 등록** 아이콘을 선택합니다.

+ 점수를 매기기 위해 모델을 사용하려면 **파이프라인에 모델 점수 매기기** 모듈을 추가합니다.
