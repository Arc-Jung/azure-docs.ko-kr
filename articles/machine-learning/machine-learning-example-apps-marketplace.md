# Azure 마켓플레이스의 기계 학습 앱

데이터 과학자는 Azure 기계 학습을 통해 스마트폰 앱 개발자처럼 자신이 보유한 노하우와 창의성을 수익화하는 채널을 확보할 수 있습니다. 기계 학습 서비스를 Azure 마켓플레이스에 게시하여 전 세계 모든 사용자가 데이터 과학자가 게시한 클라우드 API를 결제 및 구독하여 자신의 앱에 사용하도록 할 수 있습니다.

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

그러나 Azure 마켓플레이스는 기본 분석 API 이상을 제공할 수 있습니다. 예를 들어 특정 비즈니스 요구 사항을 해결하기 위해 완성된 일련의 종단 간 기계 학습 응용 프로그램을 작성했습니다. 이러한 기계 학습 앱을 사용하는 데에는 데이터 과학자 또는 기계 학습 전문가가 필요 없습니다. 솔루션에 과학이 이미 반영되어 있습니다. 사용자는 자신의 고유한 데이터를 제공하는 것으로 앱을 사용할 수 있습니다.

예를 들어 이러한 앱에는 다음이 포함됩니다.

##추천##
 셀프 서비스이지만 매우 강력한 추천 엔진으로서, 웹 사이트나 메일 캠페인에 개인 설정된 제품 추천을 쉽게 추가할 수 있게 해줍니다.

[DataMarket 링크](http://datamarket.azure.com/dataset/amla/recommendations)

[빠른 시작 가이드](machine-learning-recommendation-api-quick-start-guide.md)
  

##이상 감지##
시계열 데이터(예: 서버 원격 분석 또는 센서 기반 데이터)에서 비정상적인 이벤트를 확인할 수 있는 API입니다.

[DataMarket 링크](https://datamarket.azure.com/dataset/aml_labs/anomalydetection)

[설명서](machine-learning-apps-anomaly-detection.md)

이러한 API를 사용하면 데이터 과학자가 되거나 스스로 앱을 개발하는 데 시간을 들일 필요 없이 Azure 기계 학습에 빌드된 강력한 기계 학습 기반 서비스를 사용할 수 있습니다. 사용자 고유의 데이터를 가져와 사용하면 됩니다.

##텍스트 분석##
이 API를 사용하여 정서 분석 및 핵심 문구 추출과 같은 작업에 대한 구조화되지 않은 텍스트를 분석할 수 있습니다. 학습 데이터 없이 이 API를 사용할 수 있으며, 텍스트 데이터를 가져오기만 하면 됩니다. 이 API는 내장된 고급 자연어 처리 기술을 사용합니다.

[DataMarket 링크](https://datamarket.azure.com/dataset/aml_labs/anomalydetection)

[설명서](machine-learning-apps-text-analytics.md)

 
 ##고객 변동 예측## 고객 변동 예측은 Azure 기계 학습을 사용하여 빌드된 변동 분석 서비스입니다. 고객(플레이어, 구독자, 사용자 등)이 회사 또는 서비스와의 관계를 종료할 가능성이 큰 시기를 예측하도록 설계되었습니다. 회사와의 관계를 종료할 위험이 큰 고객을 예측할 수 있으면 고객에게 접근하여 종료 가능성을 줄일 수 있는 기회 창이 회사에 제공됩니다.

[DataMarket 링크](https://datamarket.azure.com/dataset/amla/customer-churn-prediction)

[설명서](https://churn.cloudapp.net/documentation)
