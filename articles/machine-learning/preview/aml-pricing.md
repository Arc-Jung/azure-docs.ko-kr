# <a name="pricing-and-billing-for-azure-machine-learning-services"></a>Azure Machine Learning 서비스에 대한 가격 책정 및 청구

자세한 내용이나 샘플 청구서를 보려면 [Azure 가격 책정 페이지](https://azure.microsoft.com/pricing/details/machine-learning-services/)를 방문하세요.  

다음은 청구 및 가격 정책과 관련하여 자주 묻는 질문입니다.

## <a name="can-i-use-azure-machine-learning-for-free-during-preview"></a>미리 보기 동안 Azure Machine Learning을 사용할 수 있습니까?    

Azure Machine Learning Workbench 응용 프로그램은 Azure 구독자가 무료로 사용할 수 있습니다.

실험 서비스 및 모델 관리는 공개 미리 보기 중에 할인 요금으로 사용할 수 있는 유료 계층 외에 무료 계층도 제공합니다.

## <a name="am-i-charged-based-on-how-many-experiments-i-run"></a>실행하는 실험 수에 따라 비용이 청구되나요?

아니요. 실험 서비스에서는 원하는 수만큼 많은 실험을 수행할 수 있으며, 비용은 사용자 수에 따라서만 청구됩니다. 실험 계산 리소스의 비용은 개별적으로 청구됩니다.  솔루션에 가장 적합한 모델을 찾을 수 있도록 여러 실험을 수행하는 것이 좋습니다. 

## <a name="am-i-charged-based-on-how-many-times-my-web-services-is-called"></a>내 웹 서비스 호출 빈도에 따라 비용이 청구되나요?

번호 웹 서비스는 모델 관리 비용에 영향을 미치지 않고 필요한 대로 자주 호출할 수 있습니다. 응용 프로그램의 요구 사항을 충족시키기 위해 배포 범위를 완벽하게 제어할 수 있어야 합니다.

## <a name="how-can-i-scale-the-units-i-purchase-in-the-azure-machine-learning-model-management"></a>Azure Machine Learning 모델 관리에서 구매한 단위를 어떻게 조정할 수 있나요?

Azure Portal 또는 CLI를 사용하여 단위 수를 높거나 낮게 변경할 수 있습니다. 

## <a name="what-does-a-bill-look-like"></a>청구서는 어떤 모양인가요?

청구서는 매일 생성됩니다. 청구 목적상 하루는 UTC 기준 자정에 시작됩니다. 청구서는 매달 생성됩니다. Azure Machine Learning과 함께 사용하는 Azure 서비스에 대해서는 별도의 수수료가 발생합니다. 수수료는 다음을 포함하지만 이에 국한되지 않습니다. 
- 계산 요금
- HDInsight
- Azure Container Service
- Azure Container Registry 
- Azure Blob Storage
- Application Insights
- Azure Key Vault
- Visual Studio Team Services
- Azure Event Hub
- Azure Stream Analytics 자세한 내용이나 샘플 청구서를 보려면 [Azure 가격 책정 페이지](https://azure.microsoft.com/pricing/details/machine-learning-services/)를 방문하세요.
