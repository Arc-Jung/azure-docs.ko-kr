# <a name="how-to-code-and-test-azure-functions-locally"></a>Azure 함수를 로컬로 코딩 및 테스트하는 방법
> [!NOTE]
> Azure Functions 로컬 개발 및 도구 환경은 현재 미리 보기로 제공되며 환경은 최종 릴리스 전에 상당히 개선될 예정입니다. Azure Functions 호스트를 로컬로 실행하는 가장 쉬운 방법은 [Azure Functions CLI](https://go.microsoft.com/fwlink/?linkid=832752)를 사용하고 npm에서 설치하는 것입니다. 지금은 Windows만 지원됩니다. 전체 설명서는 곧 제공될 예정입니다. Azure Functions CLI에 대한 피드백 또는 파일 버그를 제공하려면 [azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script) GitHub 리포지토리에 문제를 제출해 주세요.
