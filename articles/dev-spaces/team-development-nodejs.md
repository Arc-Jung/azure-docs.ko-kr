### <a name="make-a-code-change"></a>코드 변경
`mywebapi`에 대한 VS Code 창으로 이동하고, `server.js`에서 기본 `/` GET 처리기에 대한 코드를 편집합니다. 예를 들어 다음과 같습니다.

```javascript
app.get('/', function (req, res) {
    res.send('mywebapi now says something new');
});
```
