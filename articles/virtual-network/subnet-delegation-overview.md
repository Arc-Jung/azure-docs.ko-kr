# <a name="what-is-subnet-delegation"></a>서브넷 위임이란 무엇입니까?

서브넷 위임을 사용하면 가상 네트워크에 삽입해야 하는 원하는 Azure PaaS 서비스에 대한 특정 서브넷을 지정할 수 있습니다. Subnet 위임은 Azure 서비스를 가상 네트워크에 통합하는 관리를 고객에게 완전히 제어합니다.

Azure 서비스에 서브넷을 위임할 때 해당 서비스가 해당 서브넷에 대한 몇 가지 기본 네트워크 구성 규칙을 설정하도록 허용하여 Azure 서비스가 안정적인 방식으로 인스턴스를 운영하는 데 도움이 됩니다. 따라서 Azure 서비스는 다음과 같은 배포 전 또는 사후 조건을 설정할 수 있습니다.
- 공유 서브넷과 전용 서브넷에 서비스를 배포할 수 있습니다.
- 서비스가 제대로 작동하는 데 필요한 배포 후 네트워크 의도 정책 집합을 서비스에 추가합니다.

##  <a name="advantages-of-subnet-delegation"></a>서브넷 위임의 장점

서브넷을 특정 서비스에 위임하면 다음과 같은 이점이 있습니다.

- 이를 사용하면 하나 이상의 Azure 서비스에 대한 서브넷을 지정하고 요구 사항에 따라 서브넷의 인스턴스를 관리할 수 있습니다. 예를 들어 가상 네트워크 소유자는 다음과 같이 리소스 및 액세스를 더 잘 관리하기 위해 위임된 서브넷에 대해 다음을 정의할 수 있습니다.
    - 네트워크 보안 그룹과 트래픽 정책을 필터링합니다.
    - 사용자 정의 경로로 정책을 라우팅합니다.
    - 서비스 끝점 구성과의 서비스 통합.
- 주입된 서비스를 사용하여 네트워크 의도 정책 의 형태로 배포의 사전 조건을 정의하여 가상 네트워크와 더 잘 통합할 수 있습니다. 이렇게 하면 PUT에서 주입된 서비스의 기능에 영향을 줄 수 있는 모든 작업을 차단할 수 있습니다.


## <a name="who-can-delegate"></a>누가 위임할 수 있습니까?
서브넷 위임은 가상 네트워크 소유자가 특정 Azure 서비스에 대한 서브넷 중 하나를 지정하기 위해 수행해야 하는 연습입니다. Azure Service는 차례로 고객 워크로드에서 소비하기 위해 인스턴스를 이 서브넷에 배포합니다.

## <a name="impact-of-subnet-delegation-on-your-subnet"></a>서브넷에 대한 서브넷 위임의 영향
각 Azure 서비스는 다음과 같이 삽입을 위해 위임된 서브넷에서 수행할 또는 지원하지 않는 속성을 정의할 수 있는 자체 배포 모델을 정의합니다.
- 동일한 서브넷에 설정된 다른 Azure 서비스 또는 VM/가상 시스템 규모와 공유서브넷을 공유하거나 이 서비스의 인스턴스만 있는 전용 서브넷만 지원합니다.
- 는 위임된 서브넷과의 NSG 연결을 지원합니다.
- 위임된 서브넷과 연결된 NSG를 지원하므로 다른 서브넷과도 연결할 수 있습니다.
- 을 사용하면 위임된 서브넷과 라우트 테이블 연결을 사용할 수 있습니다.
- 을 사용하면 위임된 서브넷과 연결된 경로 테이블을 다른 서브넷과 연결할 수 있습니다.
- 은 위임된 서브넷의 최소 IP 주소 수를 지정합니다.
- 위임된 서브넷의 IP 주소 공간을 개인 IP 주소 공간(10.0.0.0/8, 192.168.0.0/16, 172.16.0.0/12)에서 사용할 수 있습니다.
- 즉, 사용자 지정 DNS 구성에 Azure DNS 항목이 있습니다.

주입된 서비스는 다음과 같이 자체 정책을 추가할 수도 있습니다.
- **보안 정책**: 지정된 서비스가 작동하는 데 필요한 보안 규칙 의 컬렉션입니다.
- **경로 정책**: 지정된 서비스가 작동하는 데 필요한 경로의 컬렉션입니다.

## <a name="what-subnet-delegation-does-not-do"></a>서브넷 위임이 하지 않는 일

위임된 서브넷에 주입되는 Azure 서비스에는 다음과 같이 위임되지 않은 서브넷에 사용할 수 있는 기본 속성 집합이 있습니다.
-  Azure 서비스는 고객 서브넷에 인스턴스를 삽입할 수 있지만 기존 워크로드에 영향을 미치지는 않습니다.
-  이러한 서비스가 적용되는 정책 이나 경로는 유연 하 고 고객에 의해 재정의 될 수 있습니다.
