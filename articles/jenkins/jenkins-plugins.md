# <a name="jenkins-plugins-for-azure"></a>Azure에 대 한 Jenkins 플러그 인

다음 Jenkins 플러그 인은 Azure에서 사용 하기 위한 다양 한 기능을 지원 합니다.
  
| Jenkins 플러그 인 | 설명                                   |
|------------------------------------------------------------------------------|
| [Azure App Service 플러그 인](https://plugins.jenkins.io/azure-app-service)     | Azure App Service를 배포 하는 Jenkins 플러그 인 (현재 웹 앱만 지원) | 
| [Azure AD 플러그 인](https://plugins.jenkins.io/azure-ad)                       | Azure Active Directory를 통해 인증 & 권한 부여를 지 원하는 Jenkins 플러그 인입니다. | 
| [Azure 아티팩트 관리자 플러그 인](https://plugins.jenkins.io/azure-artifact-manager) | Azure 아티팩트 관리자 플러그 인은 아티팩트를 Azure Blob Storage에 저장할 수 있도록 하는 아티팩트 관리자입니다. Azure 아티팩트 관리자 플러그 인은 Jenkins 및 작업에 투명 하 게 작동 하며, 기본 아티팩트 관리자와 비슷합니다. | 
| [Azure 컨테이너 에이전트 플러그 인](https://plugins.jenkins.io/azure-container-agents) | Azure 컨테이너 에이전트 플러그 인은 Jenkins에서 컨테이너를 에이전트로 실행 하는 데 도움이 될 수 있습니다. | 
| [Azure Container Registry 작업 플러그 인](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins 플러그 인을 통해 [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview)에 docker 빌드 요청을 보낼 수 있습니다. |
| [Azure Container Service 플러그 인](https://plugins.jenkins.io/azure-acs)       | Azure Container Service (AKS)에 구성을 배포할 Jenkins 플러그 인입니다. | 
| [Azure 자격 증명 플러그 인](https://plugins.jenkins.io/azure-credentials)      | Jenkins 플러그 인을 통해 Azure 자격 증명을 관리 합니다. | 
| [Azure Function 플러그 인](https://plugins.jenkins.io/azure-function)           | Jenkins 플러그 인을 통해 Azure 함수를 배포 합니다. | 
| [Azure IoT Edge 플러그 인](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge 플러그 인을 사용 하면 Jenkins에서 IoT Edge 솔루션을 개발 하기 위한 CI/CD 파이프라인을 쉽게 설정할 수 있습니다. | 
| [Azure Service Fabric 플러그 인](https://plugins.jenkins.io/service-fabric)     | Linux Azure Service Fabric 프로젝트용 Jenkins 플러그 인 |
| [Azure Storage 플러그인](https://plugins.jenkins.io/windows-azure-storage)     | Microsoft Azure Blob storage에 빌드 아티팩트를 업로드 하거나 빌드 종속성을 다운로드 하기 위한 플러그 인입니다. | 
| [Azure VM 에이전트 플러그 인](https://plugins.jenkins.io/azure-vm-agents)         | Jenkins 플러그 인을 통해 Azure virtual machines에서 Jenkins 에이전트를 만들 수 있습니다 (Azure Resource Manager 템플릿 통해). | 
| [Azure 가상 머신 확장 집합 플러그 인](https://plugins.jenkins.io/azure-vmss)           | Jenkins 플러그 인을 통해 VM 이미지를 Azure 가상 머신 확장 집합에 배포 합니다. |
