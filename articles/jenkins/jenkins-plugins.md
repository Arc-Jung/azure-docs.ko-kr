# <a name="jenkins-plugins-for-azure"></a>Azure용 젠킨스 플러그인

다음 Jenkins 플러그인Azure와 함께 사용할 수 있는 다양 한 기능을 지원 합니다.
  
| 젠킨스 플러그인 | 설명                                   |
|------------------------------------------------------------------------------|
| [Azure 앱 서비스 플러그인](https://plugins.jenkins.io/azure-app-service)     | Jenkins 는 Azure 앱 서비스를 배포하기 위해 플러그인을 사용합니다(현재 웹 앱만 지원). | 
| [Azure AD 플러그 인](https://plugins.jenkins.io/azure-ad)                       | Azure Active Directory를 통한 인증 & 권한 부여를 지원하는 Jenkins 플러그인입니다. | 
| [Azure 아티팩트 관리자 플러그인](https://plugins.jenkins.io/azure-artifact-manager) | Azure 아티팩트 관리자 플러그인은 아티팩트를 Azure Blob 저장소에 저장할 수 있는 아티팩트 관리자입니다. Azure 아티팩트 관리자 플러그인은 젠킨스와 작업에 투명하게 작동하며 기본 아티팩트 관리자와 같습니다. | 
| [Azure 컨테이너 에이전트 플러그 인](https://plugins.jenkins.io/azure-container-agents) | Azure 컨테이너 에이전트 플러그인은 Jenkins에서 에이전트로 컨테이너를 실행 하는 데 도움이 수 있습니다. | 
| [Azure 컨테이너 레지스트리 작업 플러그인](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins 플러그인 [Azure 컨테이너 레지스트리에](/azure/container-registry/container-registry-tasks-overview)도커 빌드 요청을 보낼 수 있습니다. |
| [Azure Container Service 플러그 인](https://plugins.jenkins.io/azure-acs)       | Jenkins 플러그인 Azure 컨테이너 서비스 (AKS)에 구성을 배포 합니다. | 
| [Azure 자격 증명 플러그 인](https://plugins.jenkins.io/azure-credentials)      | 젠킨스 는 Azure 자격 증명을 관리 하려면 플러그인. | 
| [Azure Function 플러그 인](https://plugins.jenkins.io/azure-function)           | 젠킨스 는 Azure 함수를 배포하는 플러그인. | 
| [Azure IoT 에지 플러그인](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge 플러그인을 사용하면 Jenkins에서 IoT Edge 솔루션을 개발하기 위한 CI/CD 파이프라인을 쉽게 설정할 수 있습니다. | 
| [Azure Service Fabric 플러그 인](https://plugins.jenkins.io/service-fabric)     | 리눅스 Azure 서비스 패브릭 프로젝트에 대 한 젠킨스 플러그인. |
| [Azure Storage 플러그인](https://plugins.jenkins.io/windows-azure-storage)     | 빌드 아티팩트를 업로드하거나 Microsoft Azure Blob 저장소에서 빌드 종속성을 다운로드하기 위한 플러그인입니다. | 
| [Azure VM 에이전트 플러그인](https://plugins.jenkins.io/azure-vm-agents)         | 젠킨스 플러그인 (Azure 리소스 관리자 템플릿을 통해) Azure 가상 컴퓨터에서 젠킨스 에이전트를 만듭니다. | 
| [Azure 가상 시스템 규모 집합 플러그인](https://plugins.jenkins.io/azure-vmss)           | Jenkins 플러그인 Azure 가상 시스템 규모 집합에 VM 이미지를 배포합니다. |
