# Azure 데이터 팩터리 SDK

Azure 포털, Azure PowerShell, .NET 클래스 라이브러리 또는 REST API를 사용하여 팩터리를 만들고 모니터링하고 관리할 수 있습니다.

메서드 | 리소스 위치 | 개발자 참조
---------------------------------------------------- | ------------------------------ | -----------
Azure 포털 | [https://portal.azure.com/](https://portal.azure.com) | 
Azure PowerShell | 최신 [Azure PowerShell](http://go.microsoft.com/?linkid=9811175&clcid=0x409) 다운로드 | [Cmdlet 참조](https://msdn.microsoft.com/library/dn820234.aspx)<p>[JSON 스크립팅 참조](https://msdn.microsoft.com/library/azure/dn835050.aspx) </p> 
.NET 클래스 라이브러리 | Azure 데이터 팩터리 .NET SDK를 사용하여 Azure 데이터 팩터리를 만들고 모니터링하고 관리할 수 있고 .NET 활동을 사용하여 데이터 팩터리를 확장할 수 있습니다. 쉽게 시작하려면 [Azure 데이터 팩터리 파이프라인에서 사용자 지정 작업 사용](data-factory-use-custom-activities.md) 및 [데이터 팩터리 .NET SDK를 사용하여 Azure 데이터 팩터리 만들기, 모니터링 및 관리](data-factory-create-data-factories-programmatically.md)를 참조하세요.<p><b>최신 Nuget 다운로드</b><br/>최신 Azure 데이터 팩터리 관리 라이브러리 Nuget 패키지는 [https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/](https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/) 페이지에서 다운로드할 수 있습니다.</p><p>**Visual Studio에서 패키지 관리자 콘솔 사용**<br/>Visual Studio의 패키지 관리자 콘솔에서 다음 명령을 실행하면 최신 Azure 데이터 팩터리 관리 라이브러리를 구할 수 있습니다.</p>Install-Package Microsoft.Azure.Management.DataFactories</p> | [.NET SDK 참조](https://msdn.microsoft.com/library/dn883654.aspx)
REST API | 데이터 팩터리 REST API를 사용하여 Azure 데이터 팩터리를 만들고 모니터링하고 관리할 수 있습니다. | [REST API 참조](https://msdn.microsoft.com/library/dn906738.aspx)
