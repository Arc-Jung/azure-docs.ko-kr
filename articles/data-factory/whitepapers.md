# <a name="azure-data-factory-whitepapers"></a>Azure 데이터 팩터리 백서 

백서를 사용하면 더 자세한 수준에서 Azure 데이터 팩터리를 탐색할 수 있습니다. 이 문서에서는 Azure 데이터 팩터리에 사용할 수 있는 백서 목록을 제공합니다.

| **백서** | **설명** |
| --- | --- |
|[Azure 데이터 팩터리- 클라우드의 데이터 통합](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | 이 백서에서는 Azure Data Factory에서 최신 데이터 웨어하우스를 구축하고, 고급 분석을 사용하여 지능형 SaaS 응용 프로그램을 구동하고, SQL Server 통합 서비스 패키지를 Azure로 리프트할 수 있는 방법을 설명합니다.|
|[Azure 데이터 팩터리를 사용하여 온-프레미스 관계형 데이터 웨어하우스에서 Azure로 데이터 마이그레이션](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | 이 백서에서는 Azure 데이터 팩터리를 사용하여 기존 온-프레미스 관계형 데이터 웨어하우스(예: Netezza, Oracle, Teradata, SQL Server)에서 Azure(예: Blob 저장소 또는 Azure Data Lake Storage)로 수십 개의 TB 데이터를 마이그레이션하는 복잡성을 다룹니다. Azure 데이터 팩터리별 Azure로의 빅 데이터 수집 여정에 대한 복원력, 성능, 확장성, 관리 및 보안에 대한 과제와 모범 사례를 설명합니다. |
|[Azure 데이터 팩터리를 사용한 SAP 데이터 통합](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | 이 백서에서는 대상 시나리오, SAP 커넥터 옵션 및 다양한 요구 사항에 대한 비교, 데이터 팩터리의 각 SAP 커넥터에 대한 소개를 포함하여 SAP 데이터 통합에 대한 Azure Data Factory의 현재 지원을 요약합니다. | 
|[Azure 데이터 팩터리: 클라우드의 SSIS](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| 이 백서에서는 기존 SSIS 워크로드를 Azure Data Factory로 마이그레이션하고 일반적인 고려 사항 및 문제를 해결하는 이유를 살펴봅니다. 그런 다음 Azure-SSIS IR을 만드는 기술적 세부 사항을 살펴보고 Sql Server 관리 스튜디오(SSMS)와 같이 익숙한 도구를 사용하여 Azure Data Factory를 통해 패키지를 업로드, 실행 및 모니터링하는 방법을 보여 줍니다. |
|[Azure 데이터 팩터리 –전달 매개 변수](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| 이 문서에서는 파이프라인과 활동 간에 매개 변수를 전달하는 방법과 활동 간에 매개 변수를 전달하는 방법을 안내합니다. |
|[Azure 데이터 팩터리 -데브옵스](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| 이 백서에서는 Azure 데이터 팩터리의 지속적인 통합 및 배포와 관련된 몇 가지 모범 사례를 다룹니다. |
