## <a name="symptoms"></a>증상

IoT Hub에 대한 요청은 오류 **503003 PartitionNot**.

## <a name="cause"></a>원인

이 오류는 IoT Hub 내부이며 일시적일 수 있습니다. [IoT Hub 내부 서버 오류 원인을](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause)참조하십시오.
