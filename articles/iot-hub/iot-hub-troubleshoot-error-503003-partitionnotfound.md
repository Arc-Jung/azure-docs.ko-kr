## <a name="cause"></a>원인

이 오류는 IoT Hub 내부 이며 일시적일 수 있습니다. [IoT Hub 내부 서버 오류 원인](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause)을 참조 하세요.
