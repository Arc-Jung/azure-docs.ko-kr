<div class="content">
    <h1>Web Apps 설명서</h1>
    <p style="padding-bottom: 0px; max-width: 1050px;">Azure Web Apps를 사용하면 인프라를 관리할 필요 없이 선택한 프로그래밍 언어로 웹 사이트를 빌드하고 호스팅할 수 있습니다. Azure Web Apps는 자동 크기 조정 및 고가용성을 제공하고, Windows 및 Linux(미리 보기)를 지원하며, GitHub, Team Services 또는 Git 리포지토리에서 자동화된 배포를 지원합니다. 빠른 시작, 자습서 및 샘플을 통해 Azure Web Apps를 사용하는 방법을 알아보세요.</p>
