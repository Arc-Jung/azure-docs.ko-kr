# Azure 앱 서비스 웹 앱 학습 맵
Azure 앱 서비스 웹 응용 프로그램을 테스트하고, 웹 자산을 Azure 앱 서비스 웹 응용 프로그램으로 신속하게 마이그레이션하고, 전 세계 고객을 위한 풍부하고 원활한 환경을 만들 수 있습니다. 웹 응용 프로그램 콘텐츠의 효과적인 학습 경로를 알려면 이 페이지의 지침을 따르세요.
