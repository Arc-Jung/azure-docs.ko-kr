# Azure AD를 사용하여 시작하는 방법

[Azure AD란?](active-directory-whatis.md)<br> [작동 원리](active-directory-works.md)<br> [다음 단계](active-directory-next-steps.md)<br> [자세한 정보](active-directory-learn-map.md)


시작할 준비가 되셨습니까? 누구나 Azure AD를 무료로 사용할 수 있습니다. Azure 평가판 구독을 지금 등록하면 디렉터리를 사용할 수 있습니다. 디렉터리는 구입 여부에 상관없이 여러분의 것입니다. [지금 Azure AD를 사용해 보세요.](http://azure.microsoft.com/trial/get-started-active-directory/)

## 테스트, 스테이징, 프로덕션에 사용할 디렉터리를 어떻게 판단하나요?

Office 365와 같은 Microsoft 클라우드 서비스에 대해 기존 구독이 있으면 Azure AD에 이미 디렉터리가 있으며 바로 사용할 수 있습니다. 해당 디렉터리는 Office 365에 로그인하는 사용자를 인증합니다. Office 365 로그인에 사용하는 회사 또는 학교 계정과 동일한 계정을 사용하여 Azure에 등록하기만 하면 됩니다. 그러면 Azure 관리 포털에서 디렉터리를 관리할 수 있으며 테스트 등 필요한 경우 디렉터리를 추가할 수 있습니다. [Azure에서 Office 365 구독의 디렉터리를 관리하는 방법에 대해 자세히 알아보세요](active-directory-how-subscriptions-associated-directory.md#manage-the-directory-for-your-office-365-subscription-in-azure).

Microsoft 클라우드 서비스 구독이 아직 없는 경우 Azure에 등록하면 새 디렉터리가 회사 또는 학교에 맞게 프로비전되며 이 디렉터리를 기본 디렉터리로 사용할 수 있습니다.


## 등록된 도메인 이름을 어떻게 추가하나요?
사용자 지정 도메인 이름을 추가하거나 등록된 도메인 이름을 사용하시겠습니까? [사용자 지정 도메인을 추가하는 방법에 대해 자세히 알아보세요](active-directory-add-domain.md).

## 회사의 브랜딩을 어떻게 추가하나요?
Azure AD Premium으로 Azure 로그인 페이지 및 액세스 패널 페이지에 회사의 로고 및 기타 브랜딩을 추가할 수 있습니다. [로그인 및 액세스 패널 페이지에 회사 브랜딩 추가하기](active-directory-add-company-branding.md)

## Azure AD를 다른 디렉터리와 어떻게 통합하나요?
온-프레미스 디렉터리와 Azure AD 간에 암호 동기화를 포함한 디렉터리 동기화를 설정할 수 있으며 응용 프로그램 온-프레미스 및 클라우드의 액세스에 대해 Single Sign-On을 사용하도록 설정할 수 있습니다. [디렉터리 통합에 대해 자세히 알아보세요](active-directory-aadconnect.md).

## 사용자를 어떻게 만들고 편집하나요?
Microsoft 클라우드 서비스에 액세스할 모든 사용자에 대한 계정을 만듭니다. 기본적으로 사용자에게는 관리자 권한이 없지만 할당할 수 있습니다. [Azure AD 사용자 만들기에 대해 자세히 알아보세요](active-directory-create-users.md).

## 액세스 및 사용 보고서 보기
Azure AD는 관리자를 위한 다양한 보고서를 제공합니다. Azure AD의 유료 버전에서는 더 많은 보고서를 사용할 수 있으며 사용자 지정 보고서를 만드는 보고 API가 있습니다. [Azure AD 보고서에 대해 자세히 알아보세요](active-directory-view-access-usage-reports.md).
