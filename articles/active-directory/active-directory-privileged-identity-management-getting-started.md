# Azure AD Privileged Identity Management 시작

## 권위 있는 ID 관리 확장 시작

1. Azure Active Directory에서 디렉터리를 만듭니다.
2. 몇 가지 전역 관리자를 포함하여 새 디렉터리에 사용자를 추가합니다.

    >[AZURE.NOTE] 사용자의 회사 또는 학교 계정을 사용하여 사용자를 설정합니다. 현재 Microsoft 계정은 Azure AD PIM에 등록할 수 없습니다.

3. 해당 디렉터리의 전역 관리자로 [Azure 포털](https://portal.azure.com/)에 로그인합니다.
4. Azure 포털의 오른쪽 위에서 사용자 이름을 클릭합니다.
5. 방금 만든 디렉터리를 선택합니다.
6. 왼쪽 탐색 창에서 **새로 만들기** 아이콘을 클릭합니다.
7. 만들기 메뉴에서 **보안 + ID**를 선택합니다.
8. **Azure AD Privileged Identity Management**를 선택합니다.
9. **대시보드에 고정** 확인란을 선택된 상태로 두고 **만들기** 단추를 클릭합니다. 권위 있는 ID 관리 대시보드가 열립니다.
