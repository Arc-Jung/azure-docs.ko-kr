# <a name="box-and-azure-active-directory"></a>Box 및 Azure Active Directory

조직 전체에서 상자의 채택을 가속화 하 고 미리 작성 된 통합으로 사용자 채택을 늘립니다. Box를 Azure Active Directory (Azure AD)에 연결 하 여 사용자가 신속 하 게 공동 작업을 수행할 수 있습니다.

![Box 앱 로고를 표시 합니다.](./media/box.png)

> [!div class="nextstepaction"]
> [Azure AD >의 시도 상자](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Box와 Azure AD를 사용 하 여 무엇을 할 수 있나요?

- **간편한 single sign-on** -사용자가 Azure AD 계정을 사용 하 여 Box에 로그인 할 수 있도록 하 여 여러 사용자 이름 및 암호를 제거 하 고 액세스를 간소화 합니다.
- **사용자 프로 비전 자동화** -사용자가 조직에 참여 하거나 이동 하거나 탈퇴할 때 사용자 계정의 생성, 유지 관리 및 제거를 Box에 자동화 합니다.
- 조건부 액세스 정책을 적용 하 고, 액세스 검토를 통해 액세스를 제어 하 고, 로그인 로그를 사용 하 여 액세스를 모니터링 하 여 Box에 대 한 액세스 보안 액세스 **를 보호, 관리 및 모니터링** 합니다.

## <a name="get-started"></a>시작

간편한 구성 가이드를 사용 하 여 Box에 대 한 [사용자 프로 비전](https://aka.ms/BoxSSODoc) 및 [single sign-on](https://aka.ms/BoxProvisioningDoc) 을 설정 합니다.
