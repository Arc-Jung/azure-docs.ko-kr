# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet 및 Azure Active Directory

조직 전체에서 Smartsheet 도입을 가속화 하 고 미리 작성 된 통합으로 사용자 채택을 늘립니다. 사용자가 Smartsheet를 Azure Active Directory (Azure AD)에 연결 하 여 신속 하 게 공동 작업을 수행할 수 있습니다.

![Smartsheet 앱 로고를 표시 합니다.](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Azure AD >를 사용 하 여 Smartsheet 사용해 보기](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Smartsheet 및 Azure AD로 무엇을 할 수 있나요?

- **간편한 single sign-on** -사용자가 Azure AD 계정을 사용 하 여 smartsheet에 로그인 할 수 있도록 하 여 여러 사용자 이름 및 암호를 제거 하 고 액세스를 간소화 합니다.
- **사용자 프로 비전 자동화** -사용자가 조직에 참여 하거나 이동 하거나 탈퇴할 때 사용자 계정의 생성, 유지 관리 및 제거를 smartsheet에 자동화 합니다.
- 조건부 액세스 정책을 적용 하 고, 액세스 검토를 통해 액세스를 제어 하 고, 로그인 로그를 사용 하 여 액세스를 모니터링 하 여 Smartsheet에 대 한 **액세스를 안전 하 게 보호, 제어 및 모니터링** 합니다.

## <a name="get-started"></a>시작

간단한 구성 가이드를 사용 하 여 Smartsheet에 대 한 [사용자 프로 비전](https://aka.ms/SmartsheetProvisioningDoc) 및 [single sign-on](https://aka.ms/SmartsheetSSODoc) 을 설정 합니다.
