## <a name="what-can-you-do-with-workday-and-azure-ad"></a>Workday 및 Azure AD로 무엇을 할 수 있나요?

- **사용자 프로 비전 자동화** -그룹, 역할 또는 Workday의 다른 특성을 기반으로 하 여 직원에 게 응용 프로그램을 자동으로 할당 합니다.
- **직원 프로필 동기화** -Azure AD와 Workday 간에 직원 데이터가 최신 상태를 유지 하는지 확인 합니다.
- **온 보 딩 및 오프 보** 딩-새 채용을 위해 사용자 계정을 자동으로 프로 비전 합니다. 그러면 직원이 나갈 때 사용자 계정을 자동으로 프로 비전 해제 하 여 보안 위험을 줄일 수 있습니다.
- **간편한 single sign-on** 사용자 이름 및 암호를 제거 하 고 사용자가 Azure AD에서 호스트 되는 조직 계정을 사용 하 여 로그인 할 수 있도록 합니다.
