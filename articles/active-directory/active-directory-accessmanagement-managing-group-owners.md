# 그룹에 대한 소유자 관리

## 그룹에 소유자 할당

**Azure 포털에서 그룹에 소유자를 추가 하려면**

1. Azure 포털에서 **Active Directory**를 클릭한 다음 조직의 디렉터리의 이름을 클릭합니다.
2. **그룹** 탭을 클릭합니다.
2. **그룹** 페이지에서 소유자를 추가하려는 그룹의 이름을 클릭합니다.
3. 해당 그룹의 페이지에서 **소유자 추가**를 클릭합니다.
4. **소유자 추가** 페이지에서 이 그룹의 소유자로 추가할 사용자의 이름을 클릭하고 이 이름이 **선택 항목** 창에 추가되었는지 확인합니다.


**Azure 포털의 그룹에서 소유자를 제거하려면**

1. Azure 포털에서 **Active Directory**를 클릭한 다음 조직의 디렉터리의 이름을 클릭합니다.
2. **그룹** 탭을 클릭합니다.
3. **그룹** 페이지에서 소유자를 제거하려는 그룹의 이름을 클릭합니다.
4. 해당 그룹의 페이지에서 **소유자** 탭을 클릭합니다.
5. 해당 그룹의 페이지에서, 이 그룹에서 제거할 소유자의 이름을 클릭하고 **제거**를 클릭합니다.
