# <a name="help-i-forgot-my-password"></a>암호를 잊어버렸어요. 도와주세요!

먼저 겁 먹지 마세요.

다음 시나리오에 해당하는 경우 도움을 줄 수 있습니다.

* 계정에 액세스하는 방법을 모르거나 암호를 기억할 수 없습니다.
* 암호가 지정되지 않았고 관리자가 여기로 보냈습니다.

## <a name="unlock-your-account"></a>계정 잠금 해제

계정을 잠금 해제하려면 다음 단계를 수행합니다. 아래의 6단계에서 **새 암호 선택**이 표시되면 암호를 잠금 해제하거나 변경할 수 있으며, 잠금 해제된 상태로 있어야 합니다.

## <a name="reset-my-password"></a>내 암호 재설정

자신의 계정으로 돌아가려면 아래 단계를 수행합니다.
1. 회사 또는 학교 로그인 페이지에서 **계정에 액세스할 수 없으세요? 링크**, **회사 또는 학교 계정**을 차례로 클릭하거나 [암호 재설정 페이지](https://passwordreset.microsoftonline.com/)로 직접 이동합니다.

    ![계정에 액세스할 수 없으세요?][Login]

2. 회사 또는 학교 **사용자 ID**를 입력하고, CAPTCHA 챌린지를 전달하고 제시된 텍스트를 입력한 후 **다음**을 클릭하여 로봇이 아님을 증명합니다.
3. 관리자가 작업을 구성한 방법에 따라 다음 항목 중 하나 이상이 표시됩니다.
    * **내 암호 확인용 이메일 주소로 이메일 보내기** - 암호 확인용 메일 또는 인증 전자 메일(선택) 중 하나에 6자리 코드가 포함된 메일을 보냅니다.
    * **내 휴대폰으로 문자 메시지 보내기** - 휴대폰 또는 인증 전화(선택) 중 하나에 6자리 코드가 포함된 문자 메시지를 보냅니다.
    * **내 휴대폰으로 전화 걸기** - 휴대폰 또는 인증 전화(선택) 중 하나에 전화를 겁니다. # 키를 눌러 전화를 확인합니다.
    * **내 사무실 전화로 전화 걸기** - 사무실 전화를 겁니다. # 키를 눌러 전화를 확인합니다.
    * **보안 질문에 답변** - 답변할 수 있도록 미리 등록된 보안 질문을 표시합니다.
4. 챌린지 정보에 대한 응답을 필수 필드에 채우고 **다음**을 클릭합니다.

    ![인증 데이터 확인][Verification]

5. 관리자가 추가 확인 단계를 요구할 수 있으며, 이 경우 다른 옵션을 선택하여 4단계를 반복해야 할 수 있습니다.
6. **새 암호 선택** 페이지에서 조직 요구 사항을 충족하는 새 암호를 입력하고 해당 암호를 확인한 다음 **마침**을 클릭합니다.

    ![암호 변경][Change]

7. **암호가 재설정되었습니다.**가 표시되면 새 암호로 로그인할 수 있습니다.

    ![암호가 재설정되었습니다.][Complete]

> [!NOTE]
> 관리자가 이 기능을 사용하도록 설정하지 않은 경우 관리자가 전자 메일 또는 자체 웹 포털을 통해 지원을 제공할 수 있도록 "관리자에게 문의하세요." 링크가 표시됩니다.
>

이 방법을 사용하여 암호를 잠금 해제하거나 다시 설정하면 "조직을 대신하여 Microsoft에서 보냄"과 같이 계정에서 비롯된 이 프로세스가 완료되었음을 알리는 메일을 받을 수 있습니다. 이러한 메일을 받았고 계정에 대한 액세스 권한을 다시 얻기 위해 셀프 서비스 암호 재설정을 사용하지 않았으면 관리자에게 문의합니다.

## <a name="change-my-password"></a>내 암호 변경

이미 알고 있는 암호를 변경해야 하는 경우 다음 단계를 수행합니다.

### <a name="change-your-password-from-the-office-365-portal"></a>Office 365 포털에서 암호 변경

1. 오른쪽 위에서 자신의 프로필을 클릭한 다음 **계정 보기**를 클릭합니다.
2. **보안 및 개인 정보 보호**
3. **암호**
4. 이전 암호를 입력하고, 새 암호를 입력한 다음, 해당 암호를 확인합니다.
5. **제출**

### <a name="change-your-password-from-the-azure-access-panel"></a>Azure 액세스 패널에서 암호 변경

1. 기존 암호를 사용하여 [Azure 액세스 포털](https://myapps.microsoft.com/)에 로그인합니다.
2. 오른쪽 위에서 자신의 프로필을 클릭한 다음 **프로필**을 클릭합니다.
3. **암호 변경**
4. 이전 암호를 입력하고, 새 암호를 입력한 다음, 해당 암호를 확인합니다.
5. **제출**
