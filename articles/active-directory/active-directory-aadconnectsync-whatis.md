# Azure AD Connect Sync: 사용자 지정 동기화 옵션

Azure Active Directory 연결 동기화 서비스 (Azure AD Connect Sync)는 클라우드 내 온-프레미스 환경과 Azure AD 간의 id 데이터 동기화에 관련된 모든 작업을 처리하는 Azure AD Connect의 기본 구성 요소입니다. 동기화 관점에서 볼 때 Azure AD Connect Sync는 DirSync, Azure AD Sync 및 Active Directory Connector가 구성된 Forefront Identity Manager의 후속 제품입니다.

<center>![Azure AD Connect Sync](./media/active-directory-aadconnectsync-whatis/sync01.png) </center>


이 항목의 목적은 Auzre AD Connect Sync 관련 항목에 대한 링크를 제공하는 것입니다.

이제 무엇을 수행하시겠습니까?

- 기술 개념에 대한 개요는 [기술 개념](active-directory-aadconnectsync-technical-concepts.md)을 참조하십시오.
- Azure AD Connect Sync에서 지원되는 다중 포리스트 시나리오에 대한 개요는 [다중 포리스트 시나리오 개요](active-directory-aadconnectsync-scenario-overview.md)를 참조하십시오.
- 필터 구현 방법에 대해 자세히 알아보려면 [필터링 구성](active-directory-aadconnectsync-configure-filtering.md)을 참조하십시오.
- 암호 동기화에 대해 자세히 알아보려면 [암호 동기화 구현](active-directory-aadconnectsync-implement-password-synchronization.md)을 참조하십시오.
- Azure AD Connect Sync에서 사용자 및 연락처에 대해 자세히 알아보려면 [사용자 및 연락처 이해](active-directory-aadconnectsync-understanding-users-and-contacts.md)를 참조하십시오.
- 기본 구성을 변경하려면 [active-directory-aadconnectsync-best-practices-changing-default-configuration](active-directory-aadconnectsync-best-practices-changing-default-configuration.md)을 참조하십시오.
- Azure AD로 동기화되는 특성을 살펴보려면 [active-directory-aadconnectsync-attributes-synchronized](active-directory-aadconnectsync-attributes-synchronized.md)를 참조하십시오.
- 선언적 프로비전에 대한 자세한 내용을 보려면 [선언적 프로비전 식 이해](active-directory-aadconnectsync-understanding-declarative-provisioning-expressions.md)를 참조하십시오.
- 선언적 프로비전 기능을 살펴보려면 [함수 참조](active-directory-aadconnectsync-functions-reference.md)를 참조하십시오.
