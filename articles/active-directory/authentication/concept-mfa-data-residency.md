# <a name="data-residency-and-customer-data-for-azure-multi-factor-authentication"></a>Azure 다단계 인증을 위한 데이터 상주 및 고객 데이터

고객 데이터는 Office 365 및 Azure와 같은 Microsoft 온라인 서비스를 구독할 때 조직에서 제공한 주소를 기반으로 지리적 위치에 Azure AD에 의해 저장됩니다. 고객 데이터가 저장되는 위치에 대한 자세한 내용은 Microsoft 신뢰 센터의 [데이터 위치](https://www.microsoft.com/trustcenter/privacy/where-your-data-is-located) 섹션을 사용할 수 있습니다.

클라우드 기반 Azure 다단계 인증 및 Azure 다단계 인증 서버 프로세스 및 일부 양의 개인 데이터 및 조직 데이터를 저장합니다. 이 문서에서는 데이터가 저장되는 위치와 위치를 간략하게 설명합니다.

다음 다단계 인증 활동은 현재 언급된 경우를 제외하고 미국 데이터 센터에서 시작됩니다.

* 전화 통화 또는 SMS를 사용하는 이중 인증은 일반적으로 미국 데이터 센터에서 시작되며 글로벌 공급자에 의해 라우팅됩니다.
    * 유럽 또는 오스트레일리아와 같은 다른 지역의 범용 사용자 인증 요청은 현재 해당 지역의 데이터 센터에서 처리됩니다. 셀프 서비스 암호 재설정, Azure B2C 이벤트 또는 NPS 확장 또는 AD FS 어댑터를 사용하는 하이브리드 시나리오와 같은 다른 이벤트는 모두 현재 미국 데이터 센터에서 처리됩니다.
* Microsoft 인증자 앱을 사용하는 푸시 알림은 미국 데이터 센터에서 시작됩니다. 또한 장치 공급업체별 서비스도 다른 지역에서 작동할 수 있습니다.
* OATH 코드는 일반적으로 현재 미국에서 검증됩니다.
    * 유럽 이나 오스트레일리아 와 같은 다른 지역에서 발생 하는 범용 사용자 인증 이벤트는 해당 지역의 데이터 센터에서 처리 됩니다. 추가 이벤트는 현재 미국 데이터 센터에서 처리됩니다.

## <a name="personal-data-stored-by-azure-multi-factor-authentication"></a>Azure 다단계 인증에 의해 저장된 개인 데이터

개인 데이터는 특정 사용자와 관련된 사용자 수준 정보입니다. 다음 데이터 저장소에는 개인 정보가 포함되어 있습니다.

* 차단된 사용자
* 사용자 우회
* Microsoft 인증자 장치 토큰 변경 요청
* 다단계 인증 활동 보고서
* 마이크로소프트 인증자 활성화

이 정보는 90일 동안 보관됩니다.

Azure 다단계 인증은 사용자 이름, 전화 번호 또는 IP 주소와 같은 개인 데이터를 기록하지 않지만 사용자에게 다단계 인증 시도를 식별하는 *UserObjectId가* 있습니다. 로그 데이터는 30일 동안 저장됩니다.

## <a name="organizational-data-stored-by-azure-multi-factor-authentication"></a>Azure 다단계 인증에 의해 저장된 조직 데이터

조직 데이터는 구성 또는 환경 설정을 노출할 수 있는 테넌트 수준 정보입니다. 다음 Azure 포털 다단계 인증 페이지의 테넌트 설정은 들어오는 전화 인증 요청에 대한 잠금 임계값 또는 발신자 ID 정보와 같은 조직 데이터를 저장할 수 있습니다.

* 계정 잠금
* 사기 행위 경고
* 공지
* 전화 통화 설정

Azure 다단계 인증 서버의 경우 다음 Azure 포털 페이지에조직 데이터가 포함될 수 있습니다.

* 서버 설정
* 일회성 바이패스
* 캐싱 규칙
* 다단계 인증 서버 상태

## <a name="log-data-location"></a>로그 데이터 위치

로그 정보가 저장되는 곳은 로그 정보가 처리되는 지역에 따라 다릅니다. 대부분의 지역에는 네이티브 Azure 다단계 인증 기능이 있으므로 다단계 인증 요청을 처리하는 동일한 지역에 로그 데이터가 저장됩니다. 네이티브 Azure 다단계 인증 지원이 없는 지역에서는 미국 또는 유럽 지역에서 서비스를 받고 다단계 인증 요청을 처리하는 동일한 지역에 로그 데이터가 저장됩니다.

일부 핵심 인증 로그 데이터는 미국에만 저장됩니다. 21Vianet에서 운영하는 마이크로소프트 Azure 독일과 마이크로소프트 Azure는 항상 해당 클라우드에 저장됩니다. Microsoft 정부 클라우드 로그 데이터는 항상 미국에 저장됩니다.
