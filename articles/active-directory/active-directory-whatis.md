# Azure Active Directory란?


###이 항목에 대한 다른 문서
[Azure AD란?](active-directory-whatis.md)<br> [작동 원리](active-directory-works.md)<br> [시작](active-directory-get-started.md)<br> [다음 단계](active-directory-next-steps.md)<br> [자세한 정보](active-directory-learn-map.md)

Azure Active Directory(Azure AD)는 클라우드 및 온-프레미스 모두에서비즈니스가 ID와 액세스를 쉽게 관리하는 방법을 제공합니다. 사용자는 iOS, Mac OS X, Android 및 Windows 장치를 포함하여 좋아하는 장치를 사용하여, 클라우드 및 온-프레미스 웹 응용 프로그램에 대한 SSO의 회사 또는 학교 계정을 하나 사용할 수 있습니다. 조직에서는 안전한 로컬 및 원격 액세스를 보장하는 통합된 Multi-Factor Authentication을 사용하여 온-프레미스 및 클라우드 모두에서 중요한 데이터와 응용 프로그램을 보호할 수 있습니다. 또는 정보 근로자가 안전하고 일관되게 자신의 회사 리소스에 액세스하기 위해 단일 조직 계정을 사용할 수 있도록 온-프레미스 디렉토리를 확장합니다. Azure AD는 포괄적인 보고서, 분석 및 실프 서비스 기능을 제공하여 비용을 절감하고 보안을 강화할 수도 있습니다. Azure AD의 엔터프라이즈 규모 및 SLA를 사용하여 비즈니스는 항상 계속 실행됩니다.

![][1]


> [AZURE.NOTE]Azure Active Directory를 사용하려면 Azure 계정이 필요합니다. 계정이 없으면 [무료 Azure 계정을 등록](http://azure.microsoft.com/pricing/free-trial/)할 수 있습니다.
