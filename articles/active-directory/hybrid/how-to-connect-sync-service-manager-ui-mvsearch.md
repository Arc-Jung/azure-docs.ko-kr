# <a name="sync-service-manager-metaverse-search"></a>Sync Service Manager 메타버스 검색

![Sync Service Manager](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

메타버스 검색 탭은 데이터 관련 문제를 해결하는 데 유용합니다. 위쪽 절반에서 특성의 조합에 따라 쿼리를 만들 수 있습니다. 쿼리가 만족스러우면 **검색**을 클릭합니다. 결과가 아래쪽 그리드에 표시됩니다. **열 설정**과 함께 표시되어야 하는 열을 선택할 수 있습니다.

검색 결과에서 **메타버스 개체 속성** 을 보려면 개체 및 [속성](tshoot-connect-object-not-syncing.md#metaverse-object-properties)을 선택합니다.
