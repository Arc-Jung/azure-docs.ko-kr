# <a name="add-or-change-work-information-for-a-user-in-azure-active-directory"></a>Azure Active Directory에서 사용자에 대한 작업 정보 추가 또는 변경
이 문서는 Azure AD(Azure Active Directory)에서 사용자의 전화 번호 또는 부서 이름과 같은 작업 정보를 추가 또는 변경하는 방법을 설명합니다. 조직 내에서 새 사용자 추가에 대한 자세한 내용은 [Azure Active Directory에 새 사용자 추가](active-directory-users-create-external-azure-portal.md)를 참조하세요.

## <a name="to-change-work-information"></a>작업 정보를 변경하려면
1. 디렉터리에 대한 전역 관리자인 계정으로 [Azure AD 관리 센터](https://aad.portal.azure.com)에 로그인합니다.
2. **사용자 및 그룹**을 선택합니다.

   ![사용자 관리 열기](./media/active-directory-users-work-info-azure-portal/create-users-user-management.png)
3. **모든 사용자**를 선택합니다.

   ![모든 사용자 그룹 열기](./media/active-directory-users-work-info-azure-portal/create-users-open-users-blade.png)
4. 목록에서 사용자를 선택합니다.
5. 선택한 사용자에 대해 **작업 정보**를 선택합니다.

    ![작업 정보 열기](./media/active-directory-users-work-info-azure-portal/active-directory-create-users-work-info.png)
6. 작업 정보를 추가하거나 변경합니다. 그런 다음 명령 모음에서 **저장**을 선택합니다.
