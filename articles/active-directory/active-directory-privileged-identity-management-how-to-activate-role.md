# Azure 권위 있는 ID 관리: 역할을 활성화하거나 비활성화하는 방법

## 역할 활성화 또는 비활성화

1. [Azure 포털](http://portal.azure.com)에 로그인합니다.
2. [Azure 권한 있는 ID 관리 시작](active-directory-privileged-identity-management-getting-started.md)의 단계에 따라 Azure 포털 대시보드에 Azure PIM을 배치합니다.
3. 보안 마법사의 단계를 완료하면 Azure PIM의 주 메뉴가 표시됩니다.
4. **내 역할 활성화**를 클릭합니다.
5. 할당된 역할 목록이 나타납니다.
6. 활성화하려는 역할을 클릭합니다.
7. **활성화**를 클릭합니다. **역할 활성화 요청** 블레이드가 나타납니다.
8. 텍스트 필드에 활성화 요청 이유를 입력합니다.
9. **확인**을 클릭합니다. 이제 역할이 활성화됩니다.
10. 역할이 활성화된 후 **비활성화**를 클릭하여 역할을 비활성화할 수도 있습니다. 또한 [역할 추가 또는 제거](active-directory-privileged-identity-management-how-to-add-role-to-user.md)의 단계에 따라 사용자에서 역할을 제거할 수 있습니다.

- 역할 활성화 설정에 대한 자세한 내용은 [역할 활성화를 구성하는 방법](active-directory-privileged-identity-management-how-to-configure-role-activation.md)을 참조하세요.
- 역할 활성화 설정과 관련된 보안 경고에 대한 자세한 내용은 [보안 경고를 구성하는 방법](active-directory-privileged-identity-management-how-to-configure-security-alerts)을 참조하세요.
