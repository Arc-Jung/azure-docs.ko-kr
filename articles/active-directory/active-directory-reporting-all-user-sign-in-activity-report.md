# Azure Active Directory 모든 사용자 로그인 활동 보고서

*이 설명서는 [Azure Active Directory Reporting 가이드](active-directory-reporting-guide.md)의 일부입니다.*

 >[AZURE.IMPORTANT]>- 모든 사용자 로그인 활동 보고서는 비공개 미리 보기의 프리미엄 보고서입니다. 보고서에 대한 액세스를 가져오려면 도메인 이름을 사용하여 aadreportinghelp@microsoft.com을 메일로 보냅니다.(**contoso.onmicrosoft.com**과 같이)


모든 사용자 로그인 활동 보고서는 사용자가 로그인하는 날짜 및 시간, UPN, IP 주소 및 응용 프로그램을 포함하여 디렉터리의 모든 사용자에게서 모든 로그인을 포함합니다.


 >[AZURE.IMPORTANT]>- 모든 사용자 로그인 활동 보고서는 현재 보고 API를 통해서만 사용할 수 있습니다. 자세히 알아보려면 [Reporting API 시작](active-directory-reporting-api-getting-started.md)을 방문합니다.
