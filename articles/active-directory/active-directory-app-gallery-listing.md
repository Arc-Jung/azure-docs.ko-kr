# Azure Active Directory 응용 프로그램 갤러리에 응용 프로그램 나열

[Azure AD 갤러리](http://azure.microsoft.com/marketplace/active-directory/all/) 및 Azure 마켓플레이스에서 Azure Active Directory를 사용한 single sign-on을 지원하는 응용 프로그램을 나열하려면 Azure AD 팀에서 통합을 확인해야 합니다. 이렇게 하려면 다음 정보를 <waadpartners@microsoft.com>으로 보내야 합니다.

- 응용 프로그램에 로그인할 수 있는 Azure Active Directory 테스트 인스턴스 테스트 사용자 계정의 사용자 이름 및 암호
- 이 테스트 사용자가 응용 프로그램에 로그인하는 URL 및/또는 방법에 대한 지침
- 고객이 응용 프로그램과 Azure AD 간 연결을 만드는 방법에 대한 간단한 설명을 제공합니다. 다음과 같이 제공할 수 있습니다.
  - Azure AD 동의 프레임워크를 사용하여 응용 프로그램에 기본 제공
  - 응용 프로그램의 관리 포털을 사용하여 Azure AD를 ID 공급자(SAML 2.0, WS-Federation 또는 OpenID Connect)로 구성
  - 조직의 누군가를 연결하여 수동으로 연결 생성
- 응용 프로그램에서 셀프 서비스 관리 포털을 통해 Azure AD를 ID 공급자로 구성하는 것을 지원하는 경우 확인을 위해 테스트 관리자 계정도 제공해야 합니다.

테스트가 완료되면 응용 프로그램을 Azure Active Directory 응용 프로그램 갤러리 및 Azure 마켓플레이스에 나열할 수 있습니다. 아래 질문을 완료하여 <waadpartners@microsoft.com>으로 보냅니다.
