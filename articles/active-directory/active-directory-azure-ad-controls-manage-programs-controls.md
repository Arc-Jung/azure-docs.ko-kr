# <a name="manage-programs-and-their-controls"></a>프로그램 및 해당 컨트롤 관리 

Azure AD에는 액세스 권한이 있는 사용자에 대한 감시를 보장하는 컨트롤의 예인 그룹 구성원 및 응용 프로그램 액세스에 대한 액세스 검토가 포함됩니다. 조직에서 이러한 컨트롤을 통해 거버넌스, 위험 관리 및 준수 요구 사항을 효율적으로 처리할 수 있습니다.  

## <a name="how-to-manage-programs-and-their-controls"></a>프로그램 및 해당 컨트롤을 관리하는 방법
액세스 검토를 프로그램으로 구성하여 다양한 용도로 추적하고 수집하는 작업을 간소화할 수 있습니다.  각 액세스 검토는 프로그램에 연결될 수 있으므로 감사자를 위한 보고서를 준비할 때 특정 이니셔티브의 범위에 있는 액세스 검토만 볼 수 있습니다.

프로그램 목록을 보려면 [액세스 검토 페이지](https://portal.azure.com/#blade/Microsoft_AAD_ERM/DashboardBlade/)로 이동하고 **프로그램** 탭으로 변경합니다.

항상 "기본 프로그램"이라는 하나의 프로그램이 있습니다.  전역 관리자 역할에 속하는 경우 추가 프로그램을 만들 수 있습니다.  예를 들어 준수 이니셔티브 또는 비즈니스 목표 각각에 대한 프로그램을 하나씩 선택할 수 있습니다.

프로그램이 더 이상 필요하지 않고 연결된 컨트롤이 없는 경우 해당 프로그램을 삭제할 수 있습니다.
