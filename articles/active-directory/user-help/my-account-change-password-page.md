# <a name="change-your-work-or-school-account-password-from-the-change-password-page"></a>암호 변경 페이지에서 직장 또는 학교 계정 비밀번호 변경

**내 계정** 포털의 **암호 변경** 페이지에서는 암호를 기억하고 계정이 잠기지 않았다고 가정하여 회사 또는 학교 계정의 기존 암호를 업데이트하는 데 도움이 됩니다. 비밀번호가 기억나지 않거나, 계정이 잠겨 있거나, 조직에서 비밀번호를 받지 않은 경우 보안 정보와 모바일 장치를 사용하여 비밀번호를 재설정할 수 있습니다.

>[!Important]
>이 문서는 기존 직장 또는 학교 계정에 대해 알려진 암호를 업데이트하려는 사용자를 위한 것입니다. Xbox, 핫메일 또는 Outlook.com 같은 개인 계정에 액세스하려는 사용자인 경우 Microsoft 계정 문서에 [로그인할 수 없는 경우](https://support.microsoft.com/help/12429/microsoft-account-sign-in-cant) 에서 제안 사항을 시도해 보십시오. 직원 또는 다른 사용자에 대한 셀프 서비스 암호 재설정을 테스트하는 방법에 대한 자세한 내용을 찾으려는 관리자의 경우 [셀프 서비스 암호 재설정을](https://docs.microsoft.com/azure/active-directory/authentication/quickstart-sspr)참조하십시오.

## <a name="update-a-password-from-the-change-password-page"></a>암호 변경 페이지에서 암호 업데이트

1. 기존 암호를 사용하여 직장 또는 학교 계정에 로그인한 다음 **내** 계정(페이지)로https://myprofile.microsoft.com/) 이동합니다.

2. 왼쪽 탐색 창에서 **암호를** 선택하거나 **암호** 블록에서 **암호 변경** 링크를 선택합니다.

    ![내 계정 페이지, 강조 표시된 암호 변경 링크 표시](media/my-account-portal/my-account-portal-change-password.png)

3. 이전 암호를 입력한 다음 새 암호를 만들고 확인합니다.

    ![암호 필드 표시, 암호 페이지 변경](media/my-account-portal/my-account-portal-change-password-page.png)

4. **제출**을 선택합니다.

    비밀번호가 변경되고 직장 또는 학교 계정에 다시 로그인하라는 메시지가 표시됩니다.

## <a name="next-steps"></a>다음 단계

암호를 변경한 후 다음을 수행할 수 있습니다.

- [보안 정보를](user-help-security-info-overview.md)보거나 관리합니다.

- 연결된 [장치를](my-account-portal-devices-page.md)보거나 관리합니다.

- [조직을](my-account-portal-organizations-page.md)보거나 관리합니다.

- 로그인 [활동을](my-account-portal-sign-ins-page.md)봅니다.

- 조직에서 개인 정보 관련 데이터를 사용하는 방법을 [확인합니다.](my-account-portal-privacy-page.md)
