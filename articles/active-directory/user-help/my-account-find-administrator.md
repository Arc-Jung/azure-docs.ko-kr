# <a name="find-an-administrator-to-change-my-information-in-my-account"></a>내 계정에서 내 정보를 변경 하는 관리자 찾기

IT 또는 인적 자원 부서가 프로필 정보 중 일부를 별도의 시스템에 유지할 수 있습니다. 예를 들어 프로필 사진이 나 사용자의 이름 또는 제목입니다. 이 정보를 직접 변경할 수 없으면 인적 자원 부서나 관리자에 게 문의 하 여 변경 하도록 합니다.

## <a name="why-cant-you-just-give-me-the-name-of-who-to-contact"></a>연락할 사용자의 이름을 제공할 수 없는 이유는 무엇 인가요?

연락할 사용자의 이름 또는 전자 메일 주소를 제공 하는 것이 좋아요 하지만 Microsoft는이 정보를가지고 있지 않습니다. 이 문서에서는 관리자가 누구 인지 확인 하는 데 도움이 될 수 있는 몇 가지 제안 사항을 설명 합니다.

## <a name="how-do-i-find-out-who-my-admin-is"></a>관리자가 누구 인지 확인 하는 어떻게 할까요?

내 계정 관리자를 검색 하 여 특정 정보를 업데이트 하거나, 암호를 다시 설정 하거나, 계정을 삭제 하거나, 다른 작업을 수행할 때 연락할 사용자에 대 한 몇 가지 포인터는 다음과 같습니다.

- 대학 및 학교: 기술 지원 팀에 문의 하세요. 일반적으로 대학 사이트에 대 한 링크를 찾을 수 있습니다. 소형 학교에는 관리자 권한이 있는 두 명의 기술 사용자만 있을 수 있습니다.

- 대기업: 조직의 내부 지원 센터 또는 기술 지원에 문의 하세요.

- 중소기업: 비즈니스 소유자 또는 공동 소유자에 게 문의 하세요. 비즈니스에 대 한 모든 컴퓨터 유지 관리 작업을 수행 하는 IT 컨설턴트에 게 관리자 권한을 부여 하는 경우가 많습니다.

회사 또는 학교에서 도움을 요청 하는 사람을 모르는 경우 내 계정 사용자 계정 및 암호를 부여한 사람에 게 문의 하세요.
