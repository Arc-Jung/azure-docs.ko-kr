# <a name="manage-your-connected-devices-from-the-devices-page"></a>장치 페이지에서 연결된 장치 관리

**내 계정** 포털의 **장치** 페이지에서 회사 또는 학교 계정에 연결된 장치를 관리하는 데 도움이 됩니다. **장치** 페이지에서 다음을 수행할 수 있습니다.

- 직장 또는 학교 계정 연결 장치를 봅니다.

- 더 이상 소유하지 않거나 분실했거나 도난당한 장치를 사용하지 않도록 설정합니다.

>[!Important]
>이 문서는 직장 또는 학교 계정에 연결된 장치 정보를 업데이트하려는 사용자를 위한 것입니다. 직원 및 기타 용도에 대한 장치 관리에 대한 정보를 찾는 관리자인 경우 [장치 ID 설명서를](https://docs.microsoft.com/azure/active-directory/devices/index)참조하십시오.

## <a name="view-your-connected-devices"></a>연결된 디바이스 보기

1. 직장 또는 학교 계정에 로그인한 다음 내 계정(페이지)로 **My Account** https://myprofile.microsoft.com/) 이동합니다.

2. 왼쪽 탐색 창에서 **디바이스**를 선택하거나 **디바이스** 블록에서 **디바이스 관리** 링크를 선택합니다.

    ![내 계정 페이지, 강조 표시된 장치 링크 표시](media/my-account-portal/my-account-portal-devices.png)

3. 연결된 디바이스에 대한 정보를 검토하여 모든 디바이스를 인식하는지 확인합니다.

    ![연결된 디바이스를 보여주는 디바이스 페이지](media/my-account-portal/my-account-portal-devices-page.png)

## <a name="disable-a-device"></a>디바이스 사용 안 함

언제든지 연결된 디바이스를 비활성화할 수 있습니다. 디바이스가 익숙하지 않거나 디바이스를 분실 또는 도난당한 경우에 특히 중요합니다. 장치를 사용하지 않도록 설정한 후에는 더 이상 조직에서 인증할 수 없으며 조직의 리소스에 액세스할 수 없습니다.

>[!Important]
>실수로 디바이스를 사용 중지하면 실행 취소할 수 없습니다. 디바이스를 추가하려면 조직의 관리자에게 문의해야 합니다.

1. 제거하려는 디바이스에서 **사용 안 함**을 선택합니다.

    ![강조 표시된 사용 안 함 링크가 있는 디바이스 페이지](media/my-account-portal/my-account-portal-devices-disable.png)

2. **예**를 선택하여 디바이스를 사용하지 않도록 설정할지 확인합니다.

## <a name="view-a-bitlocker-key"></a>비트락커 키 보기

기기가 잠겨 있거나 치명적인 오류가 있는 경우 다른 기기로 이동하여 **내 계정** 포털에서 BitLocker 키를 볼 수 있습니다.

1. 잠겨 있지 않은 다른 기기에서는 **내 계정** 포털의 **장치** 페이지로 이동합니다.

    ![비트락커 키 옵션이 있는 장치 페이지](media/my-account-portal/my-account-portal-devices-bitlocker.png)

2. 잠긴 장치에 대한 **비트 로커 키 보기를** 선택하고 잠긴 장치에 대한 BitLocker 키를 적어 둡니다.
