# <a name="delegate-user-management-with-my-staff-preview"></a>내 직원과 사용자 관리 위임 (미리 보기)

조직에서 **내 직원** 을 사용 하 여 직원 들이 필요한 응용 프로그램에 액세스 하는 데 도움이 되는 매장 관리자 또는 팀 리더와 같은 권한 수치에 사용자 관리 작업을 위임할 수 있습니다. 팀 멤버가 암호를 잊은 경우 응용 프로그램에 액세스할 수 없는 경우 생산성이 손실 됩니다. 또한 지원 비용을 계산 하 고 관리 프로세스에 병목 현상이 발생 합니다.  직원 들이 자신의 계정에 액세스할 수 없는 팀 멤버는 두 번의 클릭 만으로 액세스할 수 있으며 관리자의 도움 없이도 액세스할 수 있습니다.

## <a name="manage-your-staff-in-my-staff"></a>내 직원의 직원 관리

내 직원의 팀 멤버를 관리 하는 것은 간단 합니다. 시작 하려면 [내 직원으로 이동](https://aka.ms/mystaff)하 여 팀 또는 위치를 선택한 다음 사용자를 선택 합니다. 위치의 위치 및 팀 멤버는 IT 관리자에 의해 결정 되며 변경할 수 없습니다.

둘 이상의 위치를 관리 하는 경우 내 직원으로 이동 하는 경우 위치를 선택 하 여 해당 위치에 할당 된 팀 멤버를 확인 해야 합니다.

직원에 게 액세스 하는 데 충분 한 권한이 없는 경우 다음 메시지가 표시 됩니다 .이 경우에는 내 직원을 볼 수 있는 권한이 없는 것 처럼 보입니다. 자세한 내용은 관리자에 게 문의 하세요. "

### <a name="find-a-staff-member-in-my-staff"></a>내 직원의 직원 구성원 찾기

직원 구성원의 프로필 관리를 시작 하려면 먼저 해당 프로필을 열어야 합니다.

1. [내 직원을 열고](https://aka.ms/mystaff) 필요한 경우 위치를 선택 합니다.

    ![내 직원의 팀 멤버에 대 한 위치를 선택 합니다.](media/my-staff-team-manager/allaus.png)

1. 팀 멤버의 프로필을 엽니다.

    ![내 직원의 특정 위치에 있는 사용자 중 하나를 선택 합니다.](media/my-staff-team-manager/aupage.png)

## <a name="reset-a-user-password"></a>사용자 암호 다시 설정

조직에서 사용자에 게 권한을 부여 하는 경우 직원 구성원에 대 한 암호를 다시 설정할 수 있습니다.

1. [내 직원을 엽니다](https://aka.ms/mystaff).
1. 직원 구성원의 프로필을 엽니다.
1. **암호 다시 설정**을 선택합니다.

    ![내 직원의 사용자 암호 다시 설정](media/my-staff-team-manager/resetpassword1.png)

1. 새 암호를 생성 하거나 입력 합니다. 자동으로 생성 된 임시 암호가 표시 되거나 사용자에 대 한 임시 암호를 입력 하 라는 메시지가 표시 될 수 있습니다.

    ![내 직원을 다시 설정한 후에 임시 사용자 암호를 복사 합니다.](media/my-staff-team-manager/resetpassword2.png)

사용자의 암호를 다시 설정한 후 사용자에 게 임시 암호를 지정 합니다. 사용자가 임시 암호를 사용 하 여 로그인 하는 경우 변경 해야 합니다.

## <a name="manage-a-users-phone-number"></a>사용자의 전화 번호 관리

조직에서 사용자에 게 권한을 부여 하는 경우 직원 구성원의 전화 번호를 관리할 수 있습니다.

### <a name="add-a-phone-number"></a>전화 번호 추가

1. [내 직원을 엽니다](https://aka.ms/mystaff).
1. 직원 구성원의 프로필을 엽니다.
1. **전화 번호 추가**를 선택 합니다.

    ![내 직원의 사용자 전화 번호 추가](media/my-staff-team-manager/addphone1.png)

1. 전화 번호를 추가 하 고 **저장**을 선택 합니다.

    ![직원에 게 추가 된 사용자 전화 번호를 저장 합니다.](media/my-staff-team-manager/addphone2.png)

사용자의 전화 번호를 등록 한 후에는이를 사용 하 여 SMS에 로그인 하거나, 2 단계 인증을 수행 하거나, 조직의 설정에 따라 자신의 암호를 다시 설정할 수 있습니다.

![내 직원에 게 등록 된 새 전화 번호](media/my-staff-team-manager/addphone3.png)

### <a name="edit-a-phone-number"></a>전화 번호 편집

1. [내 직원을 엽니다](https://aka.ms/mystaff).
1. 직원 구성원의 프로필을 엽니다.
1. **전화 번호 편집**을 선택 합니다.

    ![내 직원의 사용자 프로필에서 편집을 선택 합니다.](media/my-staff-team-manager/editphone2.png)

1. 새 전화 번호를 입력 하 고 **저장**을 선택 합니다.

    ![내 직원의 직원 구성원 전화 번호 편집](media/my-staff-team-manager/editphone1.png)

### <a name="enable-phone-number-sign-in-for-a-user"></a>사용자에 대 한 전화 번호 로그인 사용

조직에서 전화 번호를 사용자 이름으로 사용 하 여 로그인 (SMS 로그인)을 사용 하는 경우 기존 사용자 전화 번호에이 인증을 추가할 수 있습니다.

1. [내 직원을 엽니다](https://aka.ms/mystaff).
1. 직원 구성원의 프로필을 엽니다.
1. 사용자 이름으로 전화 번호를 사용 하 여 로그인 하 라는 메시지가 화면 아래쪽에 표시 되 면 **사용** 을 선택 하 여 프로세스를 시작 합니다. 사용자가 전화 번호로 로그인 할 수 있도록 설정 된 경우이 메시지가 나타납니다.

    ![휴대폰 로그인이 내 직원의 특정 위치에서 지원 되는 경우의 메시지를 참조 하세요.](media/my-staff-team-manager/enableforms1.png)

1. 준비가 되면 **확인**을 선택합니다.

    ![내 직원의 직원 구성원 전화 번호 제거](media/my-staff-team-manager/enableforms2.png)

### <a name="remove-a-phone-number"></a>전화 번호 제거

1. [내 직원을 엽니다](https://aka.ms/mystaff).
1. 직원 구성원의 프로필을 엽니다.
1. **전화 번호 제거**를 선택 합니다.
1. 완료 되 면 **삭제** 를 선택 합니다.

    ![내 직원의 직원 구성원 전화 번호 제거](media/my-staff-team-manager/deletephone1.png)
