- **Microsoft Edge** 에서 Microsoft Store로 이동 합니다 [내 앱 보안 로그인 확장](https://www.microsoft.com/p/my-apps-secure-sign-in-extension/9pc9sckkzk84?rtc=1&activetab=pivot%3Aoverviewtab) 기능을 선택한 후 **가져오기**합니다.
