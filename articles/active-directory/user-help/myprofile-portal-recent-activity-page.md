# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>내 프로필(미리 보기) 포털에서 최근 로그인 활동 보기
**내 프로필** 포털의 **최근 활동** 페이지에서 회사 또는 학교 계정을 사용하여 최근 로그인 활동을 모두 볼 수 있습니다.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>최근 로그인 활동 보기
1. 회사 또는 학교 계정에 로그인한 다음, https://myprofile.microsoft.com/ 페이지로 이동합니다.

2. 왼쪽 탐색 창에서 **최근 활동**을 선택하거나 **최근 활동** 블록에서 **검토 활동** 링크를 선택합니다.

    ![강조 표시된 최근 활동 링크가 표시된 내 프로필 페이지](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. 각 로그인 항목을 확장하고 검토하여 각 항목을 인식할 수 있는지 확인합니다. 친숙하지 않은 로그인 항목이 있으면 계정이 손상된 경우 계정을 보호할 수 있도록 암호를 변경하는 것이 좋습니다.

    ![확장 로그인 세부 정보가 포함된 최근 활동 페이지](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >계정이 손상되었다고 판단되는 경우 암호를 재설정하거나 계정 차단을 해제하는 방법에 대한 추가 도움이 필요하면 [손상된 Microsoft 계정에 액세스하는 방법](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account)을 참조하세요.

## <a name="next-steps"></a>다음 단계

- 최근 로그인 활동을 확인한 후 [보안 정보](user-help-security-info-overview.md), 연결된 [디바이스](myprofile-portal-devices-page.md) 또는 [조직](myprofile-portal-organizations-page.md)을 확인 또는 관리하거나, 조직에서 [데이터를 사용](myprofile-portal-privacy-page.md)하는 방식을 확인할 수 있습니다.
