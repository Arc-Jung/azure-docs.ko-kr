# <a name="what-is-the-my-apps-preview-portal"></a>내 앱(미리 보기) 포털이란?

웹 기반 **내 앱** 포털에서 회사 또는 학교 계정을 사용하여 다양한 조직의 클라우드 기반 앱을 보고 구성하고 시작할 수 있습니다. **내 앱** 포털에 액세스할 수 없는 경우 조직의 기술 지원 팀에 사용 권한을 문의하세요.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-and-workspaces.md)]

>[!Important]
>이 콘텐츠는 사용자를 위한 것입니다. 관리자인 경우 [애플리케이션 관리 설명서](https://docs.microsoft.com/azure/active-directory/manage-apps/access-panel-workspaces)에서 클라우드 기반 앱을 설정 및 관리하는 방법에 대한 더 자세한 정보를 확인할 수 있습니다.

**My Apps** 포털에서 사용할 수 있는 기능에 대한 자세한 내용 및 지침을 보려면 다음 문서를 참조하세요.

|문서 |설명 |
|------|------------|

| [내 앱(미리 보기) 포털에서 작업 영역 액세스 및 사용](my-applications-portal-workspaces.md) | **내 앱** 웹 포털을 사용하여 클라우드 기반 앱을 조직의 범주로 구성하는 방법에 대한 지침입니다. |
| [내 앱(미리 보기) 포털에서 애플리케이션 사용 권한 편집 또는 해지](my-applications-portal-permissions-saved-accounts.md) | **내 앱** 웹 포털에서 앱 사용 권한 및 저장된 계정 자격 증명을 검토하고 관리하는 방법에 대한 지침입니다. |
