# <a name="update-your-groups-info-on-the-my-apps-portal"></a>내 앱 포털에서 그룹 정보 업데이트

웹 기반 **My Apps** 포털을 통해 회사 또는 학교 계정을 사용하여 다양한 조직의 클라우드 기반 앱을 확인 및 시작하고, 일부 프로필 및 계정 정보를 업데이트하고, **그룹** 정보를 보고, 앱 및 그룹에 대한 **액세스 검토**를 수행할 수 있습니다. **My Apps** 포털에 액세스할 수 없는 경우 사용 권한에 대해 기술 지원팀에 문의해야 합니다.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

>[!Important]
>이 콘텐츠는 사용자를 위한 것입니다. 관리자인 경우 [애플리케이션 관리 설명서](https://docs.microsoft.com/azure/active-directory/manage-apps)에서 클라우드 기반 앱을 설정 및 관리하는 방법에 대한 더 자세한 정보를 확인할 수 있습니다.

## <a name="view-your-groups-information"></a>그룹 정보 보기

관리자가 **그룹** 타일을 볼 수 있는 권한을 부여한 경우 다음을 수행할 수 있습니다.

- **그룹 멤버로.** 세부 정보를 보거나 그룹을 탈퇴합니다.

- **그룹 소유자로.** 세부 정보 보기, 새 그룹 생성, 멤버 추가 또는 제거 또는 그룹 삭제

### <a name="to-view-your-groups-information"></a>그룹 정보를 보려면

1. 회사 또는 학교 계정에 로그인합니다.

2. 웹 브라우저를 열고 https://myapps.microsoft.com로 이동하거나 조직에서 제공하는 링크를 사용합니다. 예를 https://myapps.microsoft.com/contoso.com들어.

    조직에서 소유하고 사용할 수 있는 모든 클라우드 기반 앱을 표시하는 **앱** 페이지가 나타납니다.

    ![내 앱 포털의 앱 페이지](media/my-apps-portal/my-apps-portal-apps-page-groups-tile.png)

3. **그룹** 타일을 선택하여 그룹 관련 정보를 확인합니다.

    ![소유 그룹과 구성원 그룹이 모두 있는 그룹 페이지](media/my-apps-portal/my-apps-portal-groups-page.png)

4. 사용 권한에 따라 **그룹** 페이지를 사용하여 다음을 수행할 수 있습니다.

    - **소유한 그룹을 검토합니다.** 내가 소유한 그룹 영역에서 조직 내에서 소유한 그룹에 대한 정보를 **봅니다.** 특정 그룹 이름을 선택하면 그룹 유형, 구성원 수, 조인 정책 및 활성 구성원 목록을 포함하여 그룹에 대한 자세한 정보를 확인할 수 있습니다.

    - **새 그룹을 만듭니다.** 내가 소유한 그룹의 소유자로 새 **그룹을** 만듭니다. 특정 단계에 대 한이 문서의 [새 그룹 만들기](#create-a-new-group) 섹션을 참조 하십시오.

    - **기존 그룹을 편집합니다.** 자신의 그룹에 대한 세부 정보를 편집합니다. 특정 단계에 대 한이 문서의 [기존 그룹 편집](#edit-an-existing-group) 섹션을 참조 하십시오.

    - **멤버를 추가하거나 제거합니다.** 소유한 그룹의 구성원을 추가하거나 제거합니다. 특정 단계에 대한 자세한 내용은 이 문서의 구성원 추가 또는 제거 섹션을 [참조하세요.](#add-or-remove-a-member)

    - **Office 365 그룹을 갱신합니다.** 조직에서 허용하는 경우 Office 365 그룹을 갱신할 수 있습니다. 특정 단계는 이 문서의 [Office 365 그룹 갱신](#renew-an-office-365-group) 섹션을 참조하십시오. 

    - **그룹을 삭제합니다.** 소유한 모든 그룹을 삭제합니다. 특정 단계에 대 한이 문서의 [그룹 삭제](#delete-a-group) 섹션을 참조 하십시오.

    - **소속그룹도 검토합니다.** 내가 속한 그룹의 구성원인 **그룹의** 이름을 봅니다. 특정 그룹 이름을 선택하면 그룹 유형, 구성원 수, 조인 정책 및 활성 구성원 목록을 포함하여 그룹에 대한 자세한 정보를 확인할 수 있습니다.

    - **그룹에 가입합니다.** 내가 지역에 있는 그룹에서 아직 회원이 아닌 기존 **그룹에** 가입하세요. 특정 단계에 대한 [기존 그룹 조인](#join-an-existing-group)을 참조하십시오.

## <a name="create-a-new-group"></a>새 그룹 만들기

1. **그룹** 페이지에서 **내가 소유한 그룹에서** 그룹 **만들기를** 선택합니다.

    **그룹 만들기** 상자가 나타납니다.

    ![그룹 상자 만들기](media/my-apps-portal/my-apps-portal-create-group-page.png)

2. 필요한 정보를 입력합니다.

    - **그룹 유형:**

        - **보안.** 사용자 그룹의 공유 리소스에 대한 멤버 및 컴퓨터 액세스를 관리하는 데 사용됩니다. 예를 들어 특정 보안 정책의 보안 그룹을 만들 수 있습니다. 이렇게 하면 각 멤버에 개별적으로 권한을 추가할 필요 없이 한 번에 모든 멤버에게 권한 집합을 제공할 수 있습니다.

        - **사무실 365.** 공유 사서함, 일정, 파일, SharePoint 사이트 등에 대한 액세스를 멤버에게 부여하여 협업 기회를 제공합니다. 이 옵션을 통해 조직 외부에 있는 사용자에게 그룹에 대한 액세스를 제공할 수 있습니다.

    - **그룹 이름입니다.** 기억하기 쉽고 의미 있는 그룹 이름을 추가합니다.

    - **그룹 설명(선택 사항).** 그룹에 선택적 설명을 추가합니다.

    - **그룹 정책입니다.** 모든 사용자가 그룹에 가입하도록 허용하거나 그룹의 소유자만 구성원을 추가할 수 있도록 허용하도록 선택합니다.

3. **만들기**를 선택합니다.

    새 그룹은 소유자로 당신과 함께 만들어지고 **내가 소유 한 그룹** 목록에 나타납니다. 소유자이기 때문에 이 그룹은 목록에 있는 **그룹에도** 나타납니다.

## <a name="edit-an-existing-group"></a>기존 그룹 편집

그룹을 만든 후 기존 정보 업데이트를 포함하여 세부 정보를 편집할 수 있습니다.

1. **그룹** 페이지에서 편집할 그룹을 선택한 다음 * &lt;group_name&gt; * 페이지에서 **세부 정보 편집을** 선택합니다.

    **세부 정보 편집** 상자가 나타나고 처음 그룹을 만들 때 추가한 정보를 업데이트할 수 있습니다.

2. 모든 변경 내용을 변경한 다음 **Update를**선택합니다.

## <a name="add-or-remove-a-member"></a>멤버 추가 또는 제거

소유한 모든 그룹에 대해 구성원을 추가하거나 제거할 수 있습니다.

1. 구성원을 추가할 그룹을 선택한 다음 * &lt;group_name&gt; * **+** 페이지에서 선택합니다.

    ![+ 기호가 강조 표시된 그룹 구성원 추가](media/my-apps-portal/my-apps-portal-add-member-link.png)

2. **멤버 추가** 상자에서 추가할 멤버를 검색한 다음 에 **추가를**선택합니다.

    ![새 멤버를 추가하여 멤버 상자 추가](media/my-apps-portal/my-apps-portal-add-member-page.png)

    조직의 앱에 액세스하기 시작하려면 새 구성원에게 초대가 전송됩니다.

3. 실수로 구성원을 추가했거나 구성원이 조직을 떠난 경우 * &lt;group_name&gt; * 페이지에서 구성원 이름 옆에 있는 **구성원 제거를** 선택하여 구성원을 제거할 수 있습니다.

    ![제거 링크가 강조 표시된 멤버 제거](media/my-apps-portal/my-apps-portal-remove-member-link.png)

## <a name="renew-an-office-365-group"></a>Office 365 그룹 갱신

조직에서 허용하는 경우 만료 날짜를 연장하여 Office 365 그룹을 갱신할 수 있습니다.

1. 갱신할 Office 365 그룹을 선택한 다음 **그룹 갱신을**선택합니다.

    ![Office 365 그룹을 갱신하여 만료 날짜를 연장합니다.](media/my-apps-portal/my-apps-portal-renew-group-link.png)

2. 확인 메시지를 닫려면 **확인을** 클릭합니다.

    페이지를 새로 고치면 업데이트된 **마지막 갱신** 날짜와 **그룹 만료** 날짜가 표시됩니다.

## <a name="delete-a-group"></a>그룹 삭제

언제든지 자신의 그룹을 삭제할 수 있습니다. 그러나 실수로 그룹을 삭제하는 경우 그룹을 만들고 구성원을 다시 추가해야 합니다.

1. 영구적으로 삭제할 그룹을 선택한 다음 * &lt;group_name&gt; * 페이지에서 **그룹 삭제를** 선택합니다.

    ![<Group_name> 페이지가 그룹 삭제 링크가 강조 표시된 경우](media/my-apps-portal/my-apps-portal-delete-group-link.png)

2. 확인 메시지에서 **예를** 선택합니다.

    그룹이 영구적으로 삭제됩니다.

## <a name="join-an-existing-group"></a>기존 그룹에 가입

**그룹** 페이지에서 기존 그룹에 가입하거나 나갈 수 있습니다.

1. **그룹** 페이지에서 내가 속한 그룹에서 **그룹 가입을** 선택합니다. **Groups I'm in**

    **조인 그룹** 페이지가 나타납니다.

    ![그룹 조인 버튼이 강조 표시된 그룹 조인 페이지](media/my-apps-portal/my-apps-portal-join-group-link.png)

2. 그룹 **조인** 페이지에서 가입할 그룹의 이름을 선택하고, 연결된 그룹 세부 정보를 보고, 그룹을 사용할 수 있는 경우 **그룹 가입을**선택합니다.

    그룹 소유자가 멤버십을 승인하도록 요구하는 경우 그룹에 가입해야 하는 이유에 대한 비즈니스 근거를 입력한 다음 **요청을**선택하라는 메시지가 표시됩니다. 그룹에 승인이 필요하지 않으면 즉시 구성원으로 추가되고 그룹이 내가 목록에 **있는 그룹에** 표시됩니다.

3. 실수로 그룹에 가입했거나 그룹에 더 이상 참여할 필요가 없는 경우 **그룹 가입** 페이지에서 그룹 이름을 선택한 다음 **그룹 탈퇴를**선택할 수 있습니다.

    ![그룹 그룹 탈퇴 버튼이 강조 표시된 그룹 페이지 가입](media/my-apps-portal/my-apps-portal-leave-group-link.png)
