# <a name="microsoft-authenticator-app"></a>Microsoft Authenticator 앱

>[!Important]
>이 콘텐츠는 사용자를 위한 것입니다. 관리자의 경우 [Azure Active Directory 문서](https://docs.microsoft.com/en-us/azure/active-directory)에서 Azure AD(Azure Active Directory) 환경을 설정하고 관리하는 방법에 대한 자세한 정보를 찾을 수 있습니다.

Microsoft Authenticator 앱은 암호 입력 후 2단계 인증을 통해 계정을 안전하게 지킬 수 있게 도와줍니다. 하지만 개인 Microsoft 계정에 대한 암호를 완전히 바꿀 수 있다는 점을 아십니까? Microsoft Authenticator 앱을 설정하고 사용하는 방법을 알아보세요.
