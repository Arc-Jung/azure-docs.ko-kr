# <a name="update-your-profile-and-account-information-on-the-my-apps-portal"></a>내 앱 포털에서 프로필 및 계정 정보 업데이트

웹 기반 **내 앱** 포털에서 회사 또는 학교 계정을 사용하여 

- 조직의 많은 클라우드 기반 앱 보기 및 시작
- 일부 프로필 및 계정 정보 업데이트
- **그룹** 정보 보기
- 앱 및 그룹에 대한 **액세스 검토를** 수행합니다. 

**My Apps** 포털에 액세스할 수 없는 경우 사용 권한에 대해 기술 지원팀에 문의해야 합니다.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

> [!Important]
> 이 콘텐츠는 **내 앱** 사용자를 위한 것입니다. 관리자인 경우 [애플리케이션 관리 설명서](https://docs.microsoft.com/azure/active-directory/manage-apps)에서 클라우드 기반 앱을 설정 및 관리하는 방법에 대한 더 자세한 정보를 확인할 수 있습니다.

## <a name="view-your-organization-related-profile-information"></a>조직 관련 프로필 정보 보기

조직에서 **내 앱** 포털의 **프로필** 페이지에 포함하도록 선택한 항목에 따라 장치 및 활동 및 사용자가 속한 추가 조직과 함께 특정 업무 관련 세부 정보를 볼 수 있습니다.

### <a name="to-view-your-profile-information"></a>프로필 정보를 보려면

1. 직장 또는 학교 계정에 로그인하여 [ **내 앱** 포털로 이동합니다.](my-apps-portal-end-user-access.md)

2. **앱** 페이지에서 오른쪽 상단 모서리에서 프로필 사진을 선택한 다음 이름과 조직을 입력한 다음 **프로필을**선택합니다.

    **프로필** 페이지가 나타나 프로필 정보를 표시합니다.

    ![사용 가능한 모든 정보를 표시하는 프로필 페이지](media/my-apps-portal/my-apps-portal-profile-page.png)

3. **프로필** 페이지에서 다음을 수행할 수 있습니다.

    - **조직 관련 세부 정보를 검토합니다.** 사진, 이름, 제목, 관련 이메일 주소 및 사무실 정보를 봅니다. 이 정보는 조직에서 관리하므로 변경할 수 없습니다. 오류가 표시되면 헬프데스크에 문의하십시오.

    - **장치 & 활동을 검토합니다.** 각 장치가 조직에 익숙하고 적절하게 연결되어 있는지 확인합니다. 기기를 인식하지 못하는 경우 **기기 사용 안 함으로** 선택하여 계정과의 연결을 제거합니다. 장치를 사용하지 않도록 설정하면 이 페이지에서 제거됩니다.

    - **조직을 검토합니다.** 지정된 각 조직에서 계속 작업해야 합니다. 조직에서 더 이상 작업하지 않는 경우 조직을 **나가려면 로그인을**선택하는 것이 좋습니다. 조직을 떠난 후에는 이 페이지에서 제거됩니다.

## <a name="manage-your-work-or-school-account-information"></a>직장 또는 학교 계정 정보 관리

**내 앱** 포털의 **프로필** 페이지에서 회사 또는 학교 계정 정보를 업데이트하고 관리합니다. 이 페이지는 다음을 수행할 수 있습니다.

- 직장 또는 학교 계정 암호를 변경합니다.

- 암호 재설정을 켜십시오(관리자가 이 기능을 켜면).

- 추가 보안 확인 정보를 제공합니다.

- 조직의 사용 약관을 검토합니다.

- 어디서나 로그아웃합니다.

## <a name="change-your-password"></a>암호 변경

직장 또는 학교 계정의 암호를 변경하려면 **프로필** 페이지의 **계정 관리** 영역에서 **암호 변경을** 선택할 수 있습니다.

### <a name="to-change-your-password"></a>비밀번호를 변경하려면

1. **프로필** 페이지의 **계정 관리** 영역에서 **암호 변경을**선택합니다.

2. 암호 **변경** 페이지에서 사용자 ID가 올바른지 확인한 다음 이전 암호와 새 암호를 상자에 입력합니다.

    ![암호 페이지 변경](media/my-apps-portal/my-apps-portal-change-password-page.png)

3. **제출**을 선택합니다.

    암호가 변경되었습니다. 이전에 로그인한 모든 앱에 로그인하여 직장 또는 학교 계정을 사용해야합니다.

## <a name="set-up-and-use-password-reset"></a>암호 재설정 설정 및 사용

비밀번호를 잊어버렸거나 회사 지원에서 암호를 받지 못했거나 계정이 잠긴 경우 자신의 비밀번호를 재설정할 수 있습니다.

>[!Important]
>관리자는 이 기능을 켜야 하며 참여하려면 등록해야 합니다. 등록 방법 및 암호를 재설정하는 방법에 대한 자세한 내용은 [셀프 서비스 암호 재설정 등록및](active-directory-passwords-reset-register.md) 직장 또는 학교 [비밀번호 재설정을](active-directory-passwords-update-your-own-password.md)참조하십시오.

## <a name="change-your-security-verification-information"></a>보안 확인 정보 변경

조직에서 2단계 확인을 사용해야 하는 경우 **추가 보안 확인** 페이지에서 관련 보안 정보를 추가, 업데이트 및 삭제할 수 있습니다.

2단계 인증을 사용하려면 계정이나 조직의 정보로 이동하기 전에 암호및 PIN과 같은 두 가지 확인 정보를 사용해야 합니다. 2단계 인증에 대한 자세한 내용은 [2단계 인증을 위해 내 계정 설정을](multi-factor-authentication-end-user-first-time.md)참조하십시오.

### <a name="to-change-your-security-information"></a>보안 정보를 변경하려면

1. **프로필** 페이지의 **계정 관리** 영역에서 추가 **보안 확인을**선택합니다.

    ![추가 보안 확인 페이지](media/my-apps-portal/my-apps-portal-additional-verification-page.png)

2. 추가 **보안 확인** 페이지에서 다음 정보를 추가, 변경 또는 삭제할 수 있습니다.

    - **기본 확인 옵션입니다.** 이중 확인에 사용할 기본 보조 방법을 선택합니다. 이 방법은 사용자 이름과 암호를 입력한 후 2단계 확인이 필요할 때마다 자동으로 사용됩니다.

    - **확인 방법을 추가, 업데이트 또는 제거합니다.** 더 이상 적용되지 않는 이전 정보를 새로 추가하거나 기존 정보를 업데이트하거나 삭제할 수 있습니다.

    - **Microsoft 인증자 앱을 설정합니다.** Microsoft 인증자 앱을 확인 방법으로 작동하도록 설정하도록 선택할 수 있습니다. Microsoft 인증자 앱에 대한 자세한 내용은 [Microsoft 인증자 앱이란 무엇입니까?](user-help-auth-app-overview.md)

3. **저장**을 선택하여 변경 내용을 저장합니다.

## <a name="review-your-organizations-terms-of-use-statement"></a>조직의 사용 약관 검토

사용 가능한 경우 조직의 사용 약관을 검토하도록 선택할 수 있습니다.

1. **프로필** 페이지의 **계정 관리** 영역에서 사용 **약관 검토를**선택합니다.

2. 조직의 이용 약관을 검토하고 **수락을** 선택하여 조직의 사용 약관을 읽고 이해했는지 확인합니다.

    ![내 앱 포털의 이용 약관 페이지](media/my-apps-portal/my-apps-portal-tou-page.png)

    조직에 사용 약관이 없는 경우 **완료를** 선택하여 **프로필** 페이지로 돌아갈 수 있습니다.

## <a name="sign-out-of-everywhere"></a>어디서나 로그아웃

현재 로그인한 모든 곳에서 직장 또는 학교 계정을 사용하여 로그아웃하도록 선택할 수 있습니다. 여기에는 모든 앱과 장치가 포함됩니다.

### <a name="to-sign-out-of-everywhere"></a>모든 곳에서 로그아웃하려면

1. **프로필** 페이지의 **계정 관리** 영역에서 모든 **위치에서 로그아웃을**선택합니다.

2. **사방 로그아웃** 확인 란에서 **예를** 선택하여 모든 세션 및 장치에서 로그아웃할지 확인합니다. 마음이 바뀌면 **아니요를** 선택합니다.
