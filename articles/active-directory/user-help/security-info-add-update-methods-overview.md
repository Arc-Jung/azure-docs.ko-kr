# <a name="overview-about-changing-your-security-info-preview-methods-and-information"></a>보안 정보(미리 보기) 방법 및 정보 변경에 대한 개요

**보안 정보** 페이지 내에서 기본 보안 정보 방법을 추가, 편집, 삭제 및 변경할 수 있습니다. 각 보안 정보 방법 및 정보를 변경하는 방법에 대한 자세한 내용은 관련 보안 정보 방법 기반 문서를 참조하세요.

## <a name="two-factor-verification-versus-password-reset-authentication"></a>2단계 인증 및 암호 재설정 인증

보안 정보 방법은 2단계 인증 및 암호 재설정에 모두 사용됩니다. 그러나 모든 보안 정보 방법을 둘 모두에 사용할 수 있는 것은 아닙니다.

| 방법 | 용도 |
| ------ | -------- |
| 인증자 앱 | 2단계 인증 및 암호 재설정 인증입니다. |
| 문자 메시지 | 2단계 인증 및 암호 재설정 인증입니다. |
| 전화 통화 | 2단계 인증 및 암호 재설정 인증입니다. |
| 이메일 계정 | 암호 재설정 인증 전용입니다. 2단계 인증에는 다른 보안 정보 방법을 선택해야 합니다. |
| 보안 질문 | 암호 재설정 인증 전용입니다. 2단계 인증에는 다른 보안 정보 방법을 선택해야 합니다. |
