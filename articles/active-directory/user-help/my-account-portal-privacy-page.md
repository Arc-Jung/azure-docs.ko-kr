# <a name="view-how-your-organization-uses-your-privacy-related-data"></a>조직에서 개인 정보 관련 데이터를 사용하는 방법 보기

조직에서 **내 개정** 포털의 **개인 정보** 페이지에서 데이터를 사용하는 방법을 볼 수 있습니다.

## <a name="view-your-privacy-related-info"></a>개인 관련 정보 보기

1. 회사 또는 학교 계정에 로그인한 다음, https://myaccount.microsoft.com/ 페이지로 이동합니다.

2. 왼쪽 탐색 창에서 **개인 정보**를 선택하거나 **개인 정보** 블록에서 **개인 정보 검토** 링크를 선택합니다.

    ![강조 표시된 개인 정보 링크를 보여주는 내 계정 페이지](media/my-account-portal/my-account-portal-privacy.png)

3. 다음을 포함한 정보를 검토합니다.

    - **서비스.** 회사 또는 학교 계정을 사용하여 연결된 온라인 서비스 목록입니다.

    - **사용 약관**. 조직의 사용 약관입니다.

## <a name="next-steps"></a>다음 단계

조직에서 데이터를 사용하는 방법을 확인한 후 다음을 수행할 수 있습니다.

- [보안 정보](user-help-security-info-overview.md)를 보거나 관리합니다.

- 연결된 [디바이스](my-account-portal-devices-page.md)를 보거나 관리합니다.

- [조직](my-account-portal-organizations-page.md)을 보거나 관리합니다.

- [로그인 활동](my-account-portal-sign-ins-page.md)을 봅니다.

- [Office 포털, 보안 및 개인 정보 페이지](https://portal.office.com/account/#security)에서 Microsoft Office 관련 연락처 기본 설정 및 개인 정보를 볼 수도 있습니다.
