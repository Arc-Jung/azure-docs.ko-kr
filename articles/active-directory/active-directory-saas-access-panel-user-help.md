# <a name="common-access-panel-tasks"></a>일반적인 액세스 패널 작업

액세스 패널은 다음을 수행할 수 있는 웹 포털입니다.

- 클라우드 기반 응용 프로그램 관리 및 실행
- 그룹 구성원 관리
- 프로필 설정 관리 

![액세스 패널][1] 

이 항목에서는 일반적인 작업을 수행하고 문제를 해결하기 위한 지침을 제공합니다.



## <a name="how-to-troubleshoot-accessing-an-app"></a>앱 액세스 문제를 해결하는 방법

1.  브라우저에 액세스 패널 확장이 제대로 설치되어 있는지 확인합니다.  
[다음 단계](active-directory-saas-ie-troubleshooting.md)에 따라 IE 확장 문제를 해결합니다.
2.  브라우저가 비공개 모드 또는 시크릿 모드가 아닌지 확인합니다.
3.  여전히 앱에 액세스할 수 없는 경우 관리자 또는 기술 지원팀에 문의하여 도움을 요청합니다.

## <a name="how-to-change-your-password"></a>암호를 변경하는 방법

1.  **액세스 패널**에서 오른쪽 위의 사용자 정의 컨트롤을 클릭한 다음 **프로필**을 클릭합니다.
2.  **계정 관리**에서 **암호 변경**을 클릭합니다.
3.  이전 암호를 입력하고 새 암호를 입력한 다음 확인합니다.
4.  **제출**을 클릭합니다.

자세한 내용은 [고유한 암호를 업데이트하는 방법](active-directory-passwords-update-your-own-password.md)을 참조하세요.


## <a name="how-to-setup-password-reset"></a>암호를 다시 설정하는 방법

1.  **액세스 패널**에서 오른쪽 위의 사용자 정의 컨트롤을 클릭한 다음 **프로필**을 클릭합니다.
2.  **계정 관리**에서 **셀프 서비스 암호 재설정 지정**을 클릭합니다.
3.  대화 상자의 단계에 따라 제출합니다.

자세한 내용은 [사용자 암호를 다시 설정하는 방법](active-directory-passwords-update-your-own-password.md#how-to-reset-your-password)을 참조하세요.


## <a name="how-to-add-new-apps-or-request-permission-to-app-access"></a>새 앱을 추가하거나 앱 액세스 권한을 요청하는 방법

1.  **액세스 패널**에서 **앱 추가**를 클릭합니다.
2.  추가하려는 앱을 검색하거나 찾은 다음 **추가**를 클릭합니다.  
앱에 권한이 필요한 경우 관리자 승인을 기다려야 할 수 있습니다.

찾고 있는 앱을 찾을 수 없는 경우 관리자에게 문의하여 도움을 요청합니다.


## <a name="how-to-manage-your-group-memberships"></a>그룹 구성원을 관리하는 방법

1. **액세스 패널**에서 **그룹** 앱 타일을 클릭합니다.
2. 그룹을 만들려면 **내가 소유한 그룹**에서 **그룹 만들기**를 클릭하고 지침을 따릅니다.
3. 그룹에 가입하려면 **내가 속한 그룹**에서 **그룹 참여**를 클릭하고 지침을 따릅니다.

**주의**:
  
- 구성원으로 속한 그룹을 클릭하면 세부 정보를 보고 그룹을 탈퇴할 수 있습니다.
- 소유한 그룹을 클릭하면 세부 정보를 보고, 구성원을 추가하거나 제거하고, 그룹을 탈퇴할 수 있습니다.
