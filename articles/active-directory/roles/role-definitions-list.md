# <a name="list-azure-ad-role-definitions"></a>Azure AD 역할 정의 나열

역할 정의는 읽기, 쓰기 및 삭제와 같이 수행할 수 있는 사용 권한 컬렉션입니다. 일반적으로 단지 역할이라고 합니다. Azure Active Directory에는 60 이상의 기본 제공 역할이 있거나 사용자 고유의 사용자 지정 역할을 만들 수 있습니다. "이러한 역할을 수행 하는 작업은 무엇 인가요?"에 대해 궁금 하면 각 역할에 대 한 자세한 사용 권한 목록을 확인할 수 있습니다.

이 문서에서는 Azure AD 기본 제공 및 사용자 지정 역할을 해당 권한과 함께 나열 하는 방법을 설명 합니다.

## <a name="list-all-roles"></a>모든 역할 나열

1. [AZURE AD 관리 센터](https://aad.portal.azure.com) 에 로그인 하 고 **Azure Active Directory** 를 선택 합니다.

1. **역할 및 관리자** 를 선택 하 여 사용 가능한 모든 역할 목록을 표시 합니다.

    ![Azure AD 포털의 역할 목록](./media/role-definitions-list/view-roles-in-azure-active-directory.png)

1. 오른쪽에서 줄임표를 선택 하 고 **설명** 을 선택 하 여 역할에 대 한 사용 권한의 전체 목록을 표시 합니다.

    이 페이지에는 역할 관리를 안내 하는 데 도움이 되는 관련 설명서에 대 한 링크가 포함 되어 있습니다.

    !["전역 관리자-설명" 페이지를 보여 주는 스크린샷](./media/role-definitions-list/role-description.png)
