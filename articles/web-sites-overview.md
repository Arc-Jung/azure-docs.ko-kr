웹 사이트란?
============

[WACOM.INCLUDE [disclaimer](../includes/disclaimer.md)]

웹 사이트는 기존 응용 프로그램의 코드를 변경할 필요 없이 .NET, Node.js 및 PHP와 같이 많이 사용되는 웹 응용 프로그램 기술을 지원하는 웹 응용 프로그램 호스트입니다. .NET, Node.js 및 PHP 개발자가 파일, 데이터베이스와 같은 리소스에 액세스하는 데 사용하는 모든 일반적인 프로그래밍 모델 및 리소스는 Azure에 배포된 웹 사이트에서 예상대로 계속 작동합니다. 웹 사이트에서 파일 및 데이터베이스 리소스에 액세스할 수 있고 현재 웹 응용 프로그램에서 사용되는 모든 표준 구성 파일(예: web.config, php.ini 및 config.js)에 액세스할 수 있습니다. 웹 사이트는 제어 및 관리할 수 있는, 클라우드에서 실행되는 웹 응용 프로그램의 호스트입니다.
