# <a name="commands"></a>명령
|||
|---|---|
|az mysql server|서버를 관리합니다. 서버는 데이터베이스, 데이터 웨어하우스 및 탄력적 풀을 포함합니다.|
|az mysql server create|MySQL 서버를 만듭니다.|
|az mysql server delete|MySQL 서버를 삭제합니다.|
|az mysql server list|리소스 그룹의 모든 MySQL 서버 목록을 가져옵니다.|
|az mysql server show|MySQL 서버의 세부 정보를 표시합니다.|
|az mysql server update|서버를 업데이트합니다.|
|az mysql server restore|이전의 특정 시점으로 서버를 복원합니다.|
|az mysql server firewall-rule|서버의 방화벽 규칙을 관리합니다.|
|az mysql server firewall-rule create|방화벽 규칙을 만듭니다.|
|az mysql server firewall-rule delete|방화벽 규칙을 삭제합니다.|
|az mysql server firewall-rule list|방화벽 규칙을 나열합니다.|
|az mysql server firewall-rule show|방화벽 규칙의 세부 정보를 표시합니다.|
|az mysql server firewall-rule update|방화벽 규칙을 업데이트합니다.|
|az mysql db|데이터베이스를 관리합니다.|
|az mysql db list|서버에서 모든 MySQL 데이터베이스를 나열합니다.|
|az mysql server-logs list|서버의 로그 파일을 나열합니다.|
|az mysql server-logs download|로그를 서버에서 로컬로 다운로드합니다.|
|az mysql server configuration list|서버 구성 매개 변수를 나열합니다.|
|az mysql server configuration show|서버 구성 매개 변수 세부 정보를 표시합니다.|
|az mysql server configuration update|서버 구성 매개 변수 값을 수정합니다.|
|||
