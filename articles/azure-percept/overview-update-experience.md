# <a name="the-azure-percept-dk-update-experience"></a>Azure Percept 진한 업데이트 환경

Azure Percept 진한 사용자는 dev kit OS 및 펌웨어를 업데이트 하는 두 가지 옵션 (OTA (공중파) 또는 USB를 통해)을 사용할 수 있습니다. OTA는 장치를 대규모로 쉽게 최신 상태로 유지 하는 가장 좋은 방법입니다. USB 업데이트는 OTA를 사용할 수 없거나 장치를 공장 기본 설정으로 설정 하려는 경우에 적합 한 옵션입니다.

## <a name="update-experience-guides"></a>업데이트 환경 가이드

어떤 업데이트 방법을 사용 하는 것이 가장 적합 한지 확인 하기 위해 사용자에 게 도움이 되는 방법 가이드의 컬렉션을 포함 했습니다.

- [Azure IoT Hub를 설정 하 여 air 업데이트를 통해 Azure Percept에 배포 하는 방법](./how-to-set-up-over-the-air-updates.md)
- [무선을 통해 Azure Percept 진한을 업데이트 하는 방법](./how-to-update-over-the-air.md)
- [USB를 통해 Azure Percept 진한 업데이트](./how-to-update-via-usb.md)
