# Azure 리소스 커넥터 시작 및 논리 앱에 추가
>[AZURE.NOTE] 이 문서 버전은 논리 앱 2014-12-01-미리 보기 스키마 버전에 적용됩니다.

Azure 리소스 커넥터를 사용하여 논리 앱 내에서 Azure 리소스를 쉽게 관리할 수 있습니다.

## Azure 리소스 커넥터 만들기
Azure 리소스 커넥터 API 앱을 사용하려면 먼저 해당 앱의 인스턴스를 만들어야 합니다. 이 작업은 논리 앱을 만드는 동안 인라인으로 수행하거나 Azure 마켓플레이스에서 Azure 리소스 커넥터 API 앱을 선택하여 수행할 수 있습니다.

앱을 구성하려면 Azure에서 수행할 작업이 무엇이든 해당 작업을 수행할 권한이 있는 서비스 주체를 설정해야 합니다. 모든 호출은 사용자가 설정하는 이 서비스 주체를 대신하여 수행됩니다. 따라서 사용자가 원하는 작업만 정확하게 커넥터가 사용하고 다른 작업은 하지 않도록 범위를 지정할 수 있습니다.

David Ebbo가 이를 설정하는 방법에 대해 [훌륭한 블로그 게시물](http://blog.davidebbo.com/2014/12/azure-service-principal.html)을 작성했습니다. 이 블로그의 모든 지침에 따라 **테넌트 ID**, **클라이언트 ID** 및 **비밀**을 가져옵니다. 이들 세 필드와 함께 **구독 ID**가 커넥터를 구성하는 데 필요한 항목입니다.

## 논리 앱 디자이너에서 Azure 리소스 커넥터 사용
### 트리거
커넥터에서는 두 개의 트리거가 지원됩니다.

이름 | 설명
---- | -----------
이벤트 발생 | 구독에서 리소스에 대해 이벤트가 발생할 때 트리거됩니다.
메트릭이 임계값 초과 | 메트릭이 특정 임계값을 충족할 때 트리거됩니다.

### 작업

마찬가지로, Azure 구독 내의 수많은 동작을 제공할 수 있습니다.

**리소스 그룹**의 경우 다음을 할 수 있습니다.

이름 | 설명
---- | -----------
리소스 그룹 나열 | 구독의 모든 리소스 그룹을 나열합니다.
리소스 그룹 가져오기 | 해당 ID별로 리소스 그룹을 가져옵니다.
리소스 그룹 만들기 | 리소스 그룹을 만들거나 업데이트합니다.
리소스 그룹 삭제 | 리소스 그룹을 삭제합니다.

**리소스**의 경우 다음을 할 수 있습니다.

이름 | 설명
---- | -----------
리소스 나열 | 다양한 유형의 필터에 따라 구독의 리소스를 나열합니다.
리소스 가져오기 | 리소스 ID에 따라 단일 리소스를 가져옵니다.
리소스 만들기 또는 업데이트 | 리소스를 만들거나 기존 리소스를 업데이트합니다. 해당 리소스에 대한 모든 속성을 제공해야 합니다.
리소스 작업 | 리소스에서 다른 작업을 수행합니다. 작업 이름 및 이 작업이 수행되는 페이로드(있는 경우)를 알아야 합니다.
리소스 삭제 | 리소스를 삭제합니다.

**리소스 공급자**의 경우 다음을 할 수 있습니다.

이름 | 설명
---- | -----------
리소스 공급자 나열 | 구독에서 사용할 수 있는 모든 리소스 공급자를 나열합니다.

**리소스 그룹 배포**의 경우 다음을 할 수 있습니다.

이름 | 설명
---- | -----------
배포 나열 | 리소스 그룹의 모든 배포를 나열합니다.
배포 가져오기 | ID에 따라 템플릿 배포를 가져옵니다.
배포 만들기 | 템플릿을 제공하여 새 리소스 그룹을 만듭니다.

리소스에 대한 **이벤트**의 경우 다음을 할 수 있습니다.

이름 | 설명
---- | -----------
이벤트 가져오기 | 구독 또는 리소스의 이벤트를 가져옵니다.

리소스에 대한 **메트릭**의 경우 다음을 할 수 있습니다.

이름 | 설명
---- | -----------
메트릭 가져오기 | 리소스 ID에 해당하는 메트릭을 가져옵니다.

## 커넥터의 추가 기능
이제 커넥터를 만들었으므로 논리 앱을 사용하여 비즈니스 흐름에 추가할 수 있습니다. [논리 앱 정의](app-service-logic-what-are-logic-apps.md)를 참조하세요.

>[AZURE.NOTE] Azure 계정을 등록하기 전에 Azure Logic Apps를 시작하려는 경우 [논리 앱 평가](https://tryappservice.azure.com/?appservice=logic)로 이동합니다. 앱 서비스에서 즉시 단기 스타터 논리 앱을 만들 수 있습니다. 신용 카드는 필요하지 않으며 약정도 필요하지 않습니다.

[커넥터 및 API 앱 참조](http://go.microsoft.com/fwlink/p/?LinkId=529766)의 Swagger REST API 참조를 봅니다.
