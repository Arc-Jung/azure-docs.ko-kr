# Azure 구독 지불에 사용하는 신용 카드를 변경하는 방법

신용 또는 직불 카드를 사용하여 지불하거나 청구서로 지불 승인을 받을 수 있습니다. 청구서로 지불하는 방법에 대한 자세한 내용은 [Azure 청구](http://azure.microsoft.com/pricing/invoicing/)를 참조하세요.

## 기존 신용 카드에 대한 결제 정보 편집
1. [Azure 계정 센터](https://account.windowsazure.com/Subscriptions)에 계정 관리자로 로그온합니다.
2. **세부 정보 및 사용량을 보려면 구독을 클릭하세요**에서 신용 카드에 연결된 구독을 클릭합니다.</br> ![selectsub](./media/billing-how-to-change-credit-card/selectsub.png)
3. 페이지의 오른쪽에서 **지불 방법 변경**을 클릭합니다</br> ![changesub](./media/billing-how-to-change-credit-card/changesub.png).
4. **지불 방법 선택** 페이지에서 신용 카드 옆에 있는 **편집** 을 클릭합니다.</br> ![changesub](./media/billing-how-to-change-credit-card/editcard.png)
5. **편집** 페이지에서 올바른 신용 카드 종류와 카드 번호를 선택했는지 확인합니다.
6. 필요한 카드 세부 정보를 변경한 다음 **다음**을 클릭하여 지불 방법 선택 페이지로 돌아갑니다.

## 지불 방법으로 새 신용 카드 추가
1. [Azure 계정 센터](https://account.windowsazure.com/Subscriptions)에 계정 관리자로 로그온합니다.
2. **세부 정보 및 사용량을 보려면 구독을 클릭하세요**에서 새 지불 방법을 추가하려는 구독을 클릭합니다.
3. 페이지의 오른쪽에서 **지불 방법 변경**을 클릭합니다.
4. **지불 방법 선택** 페이지에서 "+"를 선택하여 새 신용 카드를 추가합니다.
5. 신용 카드 종류를 선택한 다음 필요한 모든 정보를 제공합니다.
6. **다음**을 클릭하여 지불 방법 선택 페이지로 돌아갑니다. 추가한 카드가 유효한 경우 사용 가능한 지불 방법 목록에 추가됩니다. 신용 카드 옆에 있는 드롭다운 화살표를 클릭하면 사용 가능한 모든 신용 카드를 볼 수 있습니다.

## Azure 청구서를 지불하는 데 사용하는 신용 카드 변경
1. [Azure 계정 센터](https://account.windowsazure.com/Subscriptions)에 계정 관리자로 로그온합니다.
2. **세부 정보 및 사용량을 보려면 구독을 클릭하세요**에서 지불 방법을 업데이트하려는 구독을 클릭합니다.
3. 페이지의 오른쪽에서 **지불 방법 변경**을 클릭합니다.
4. 지불 방법 선택 페이지에서 신용 카드 옆에 있는 드롭다운 화살표를 클릭하고 사용할 신용 카드를 선택한 다음 ![checkbutton](./media/billing-how-to-change-credit-card/checkbutton.png)을 클릭합니다. 이렇게 하면 창이 닫힙니다.
