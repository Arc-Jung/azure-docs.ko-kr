# <a name="tutorial-publish-your-device"></a>자습서: 디바이스 게시

축하합니다! 디바이스를 성공적으로 인증했습니다. 제품이 Azure에서 잘 작동하는 예외 디바이스의 에코시스템에 참여하고 있습니다. 디바이스가 인증되면 필요에 따라 고객이 검색하여 구매할 수 있도록 [Azure Certified Device 카탈로그](https://devicecatalog.azure.com)에 디바이스 세부 정보를 게시할 수 있습니다.

이 자습서에서는 다음 작업 방법을 알아봅니다.

> [!div class="checklist"]
> * Azure Certified Device 카탈로그에 디바이스 게시

## <a name="prerequisites"></a>필수 구성 요소

- 로그인되어 있고 [Azure Certified Device 포털](https://certify.azure.com)에서 만든 디바이스용 **승인된** 프로젝트가 있어야 합니다. 인증된 디바이스가 없는 경우 이 [자습서](tutorial-01-creating-your-project.md)를 확인하여 시작할 수 있습니다.

## <a name="publishing-your-device"></a>디바이스 게시

디바이스를 게시하는 것은 고객이 Azure Certified Device 카탈로그에서 제품을 찾는 데 유용한 간단한 프로세스입니다.

1. 디바이스를 게시하려면 프로젝트 요약 페이지에서 `Publish to Device Catalog`를 클릭합니다.

    ![카탈로그에 게시](./media/images/publish-to-catalog.png)

1. 팝업 창에서 게시 확인

    ![카탈로그에 게시 확인](./media/images/publish-to-catalog-confirm.png)

1. 디바이스가 Azure Certified Device 카탈로그를 처리하면 회사 프로필의 이메일 주소로 알림을 받게 됩니다.

## <a name="next-steps"></a>다음 단계

축하합니다! 이제 인증된 디바이스가 Azure Certified Device 카탈로그에 추가되었습니다. 여기서는 고객이 Azure를 통해 성능이 보장된 제품을 구매할 수 있습니다. 인증된 IoT 제품의 에코시스템에 참여해 주셔서 감사합니다. 이제 프로젝트 페이지가 읽기 전용으로 표시됩니다. 디바이스 정보를 업데이트하려는 경우 방법 가이드를 참조하세요.
