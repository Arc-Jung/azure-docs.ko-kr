# <a name="deprecated-configure-baidu-cloud-push-settings-for-a-notification-hub-in-the-azure-portal"></a>더 이상 사용되지 않는: Azure 포털의 알림 허브에 대해 Baidu 클라우드 푸시 설정을 구성합니다.

이 문서에서는 Azure 포털을 사용하여 Azure 알림 허브에 대한 Baidu Cloud 푸시 설정을 구성하는 방법을 보여 주며 있습니다.

> [!IMPORTANT]
> 이 자습서는 더 이상 사용되지 않습니다. 

## <a name="prerequisites"></a>사전 요구 사항
알림 허브를 아직 만들지 않은 경우 지금 만듭니다. 자세한 내용은 [Azure Portal에 Azure 알림 허브 만들기](create-notification-hub-portal.md)를 참조하세요. 

## <a name="configure-baidu-cloud-push"></a>Baidu 클라우드 푸시 구성
다음 절차에서는 알림 허브에 대해 Baidu 클라우드 푸시 설정을 구성하는 단계를 제공합니다.

1. Azure 포털에서 알림 **허브** 페이지에서 왼쪽 메뉴에서 **바이두(Android China)를** 선택합니다. 
2. Baidu 클라우드 푸시 프로젝트의 Baidu 콘솔에서 가져온 **Api 키**를 입력합니다. 
3. Baidu 클라우드 푸시 프로젝트의 Baidu 콘솔에서 가져온 **비밀 키**를 입력합니다. 
4. **저장**을 선택합니다. 

    ![푸시 알림에 대한 Baidu(Android China) 구성을 보여주는 Notification Hubs의 스크린샷](./media/notification-hubs-baidu-get-started/AzureNotificationServicesBaidu.png)

## <a name="next-steps"></a>다음 단계
Azure 알림 허브 및 바이두 클라우드 푸시를 사용하여 Baidu에 알림을 푸시하기 위한 단계별 지침이 있는 자습서의 경우 [Baidu](notification-hubs-baidu-china-android-notifications-get-started.md)를 사용하여 알림 허브 시작 을 참조하십시오.
