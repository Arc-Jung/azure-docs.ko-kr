# <a name="content-delivery-network-on-azure-with-azure-cdn-service"></a>Azure CDN 서비스를 사용 하는 Azure의 Content delivery network
[Azure CDN](https://azure.microsoft.com/services/cdn/) 은 전 세계에서 수백 개의 PoP (현재 위치)에 콘텐츠를 캐싱하여 사용자에 게 비디오, 이미지 또는 정적 파일과 같은 고대역폭 콘텐츠를 신속 하 게 제공할 수 있는 글로벌 솔루션을 개발자에 게 제공 합니다. 또한 캐시 될 수 없는 동적 콘텐츠를 가속화 하 여 로드 시간을 줄이고, 대역폭을 절약 하 고, 웹 사이트 또는 모바일 응용 프로그램을 개발 또는 관리 하거나, 스트리밍 및 배포를 수행할 때의 응답성을 높일 수 있습니다. Azure CDN 미디어, 게임 소프트웨어, 펌웨어 업데이트 또는 IoT 끝점입니다.

## <a name="azure-cdn-features"></a>Azure CDN 기능
  - 은 웹 응용 프로그램, Media Services, 저장소 및 Cloud Services를 비롯 한 Azure 서비스와 **원활** 하 게 작동 합니다.
  - **트래픽 급증 및 과도 한 로드를 쉽게 처리**합니다.
  - **Akamai 및 Verizon과 통합** 하 여 콘텐츠 배달 전략을 구축 하 고 조정 합니다.
  - 동적 사이트 가속.
  - **캐싱 규칙**을 사용 하 여 Azure CDN 캐싱 동작을 제어 합니다.
  - HTTPS **사용자 지정 도메인 지원**.
  - **Azure 진단 로그** 를 통해 핵심 분석을 보고 Blob Storage, Event Hubs 또는 Azure Monitor으로 내보낼 수 있습니다.
  - **파일을 압축**하 여 성능을 향상 시킵니다.
  - **지역 필터링** 을 통해 국가/지역별 Azure CDN 콘텐츠를 제한 합니다.
