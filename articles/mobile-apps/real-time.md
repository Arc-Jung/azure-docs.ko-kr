# <a name="build-real-time-applications-with-azure-signalr-service"></a>Azure SignalR Service를 사용 하 여 실시간 응용 프로그램 빌드

[Azure SignalR 서비스](https://azure.microsoft.com/services/signalr-service/)를 사용 하면 응용 프로그램에 실시간 기능을 쉽게 추가할 수 있습니다. 이 실시간 기능을 사용 하면 서비스에서 연결 된 모바일 응용 프로그램에 콘텐츠 업데이트를 푸시할 수 있습니다. 따라서 서버를 폴링하고 업데이트에 대 한 새 HTTP 요청을 제출할 필요 없이 앱이 업데이트 됩니다.

## <a name="azure-signalr-service-features"></a>Azure SignalR Service 기능
- 에는 웹 및 모바일 브라우저, 데스크톱 응용 프로그램, 서버 프로세스, IoT 장치, 게임 콘솔 등 광범위 한 클라이언트에 대 한 광범위 한 클라이언트 지원이 있습니다.
- 여러 인스턴스를 함께 사용 하 여 수백만 개의 모바일 응용 프로그램 연결로 확장할 수 있습니다. 또한 분할, 고가용성 또는 재해 복구를 위해 여러 글로벌 지역을 지원 합니다.
- 는 특정 연결, 모든 연결 또는 특정 사용자에 게 속하거나 임의의 그룹에 배치 된 연결의 하위 집합에 메시지를 보낼 수 있는 다양 한 메시징 패턴을 위한 다양 한 Api를 제공 합니다.
