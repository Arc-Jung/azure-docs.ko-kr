# <a name="build-real-time-applications-with-azure-signalr-service"></a>Azure SignalR 서비스로 실시간 응용 프로그램 빌드

[Azure SignalR 서비스를](https://azure.microsoft.com/services/signalr-service/)사용하면 응용 프로그램에 실시간 기능을 쉽게 추가할 수 있습니다. 이 실시간 기능을 통해 서비스는 연결된 모바일 애플리케이션에 콘텐츠 업데이트를 푸시할 수 있습니다. 따라서 앱을 서버를 폴링하거나 업데이트에 대한 새 HTTP 요청을 제출할 필요 없이 업데이트됩니다.

## <a name="azure-signalr-service-features"></a>Azure 신호서비스 기능
- 웹 및 모바일 브라우저, 데스크톱 응용 프로그램, 서버 프로세스, IoT 장치 및 게임 콘솔과 같은 광범위한 클라이언트에 대한 광범위한 클라이언트 지원을 제공합니다.
- 여러 인스턴스가 함께 작동하여 수백만 개의 모바일 응용 프로그램 연결로 확장할 수 있습니다. 또한 샤딩, 고가용성 또는 재해 복구 를 위해 여러 글로벌 지역을 지원합니다.
- 특정 연결, 모든 연결 또는 특정 사용자에 속하거나 임의의 그룹에 배치된 연결의 하위 집합에 메시지를 보낼 수 있는 다양한 메시징 패턴에 대한 풍부한 API를 제공합니다.
