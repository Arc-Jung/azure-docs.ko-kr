# <a name="store-sync-and-query-mobile-application-data-from-the-cloud"></a>클라우드에서 모바일 애플리케이션 데이터 저장, 동기화 및 쿼리
어떤 종류의 응용 프로그램을 구축하든 데이터를 생성하고 처리할 수 있습니다. 응용 프로그램의 사용자는 높은 기대치를 가지고 있습니다. 그들은 응용 프로그램이 모든 상황에서 빠르고 원활하게 작동하기를 원합니다. 대부분의 응용 프로그램은 여러 장치에서도 작동합니다. 데스크톱 또는 모바일 장치에서 응용 프로그램에 액세스할 수 있습니다. 여러 사용자가 동시에 응용 프로그램을 사용하고 데이터에 대한 즉각적인 실시간 액세스를 얻을 것으로 예상하여 데이터를 공유할 수 있습니다.

응용 프로그램 사용자가 항상 인터넷에 연결되지는 않습니다. 응용 프로그램은 인터넷 연결 과 함께 또는 인터넷 연결 없이 작동하도록 설계되고 예상됩니다. 개발자는 애플리케이션에 뛰어난 고객 경험을 제공하기 위해 데이터를 클라우드에 저장하고 동기화하는 데 적합한 솔루션을 선택해야 합니다.

Microsoft는 서버를 스핀업하거나, 데이터베이스를 선택하거나, 가능한 한 풍부한 환경을 제공하기 위해 규모 나 보안에 대해 걱정할 필요가 없는 다양한 서비스를 제공합니다. 이러한 서비스는 SQL 또는 NoSQL API를 사용하여 응용 프로그램 데이터를 클라우드에 저장할 수 있는 훌륭한 개발자 환경을 제공합니다. 또한 모든 장치에서 데이터를 동기화하고 응용 프로그램이 네트워크 연결과 함께 또는 네트워크 연결 없이 작업할 수 있도록 하여 확장 가능하고 강력한 응용 프로그램을 빌드할 수 있습니다.

다음 서비스를 사용하여 클라우드에서 모바일 응용 프로그램 데이터를 관리하고 저장합니다.

## <a name="visual-studio-app-center"></a>Visual Studio App Center
[앱 센터 데이터는](/appcenter/data/) 데이터 관리 서비스입니다. 이를 통해 애플리케이션은 온라인 및 오프라인 시나리오에서 다양한 장치 와 플랫폼에서 클라우드에서 애플리케이션 데이터를 관리, 유지 및 동기화할 수 있습니다. Azure Cosmos DB 위에 구축된 앱 센터 데이터는 사용자 기반과 응용 프로그램 수가 증가함에 따라 확장됩니다. 모든 데이터에 대해 낮은 대기 시간, 고가용성 및 높은 확장성을 보장합니다.

**주요 기능**
- Visual Studio 앱 센터 포털에서 새 Azure Cosmos DB 데이터베이스 또는 기존 Azure Cosmos DB 데이터베이스에 쉽게 프로비전할 수 있습니다.
- NoSQL 데이터베이스는 응용 프로그램 데이터를 쉽게 저장, 동기화 및 쿼리할 수 있도록 지원합니다.
- Azure Cosmos DB 위에 구축된 이 서비스는 Azure Cosmos DB에서 제공하는 대부분의 주요 기능을 상속하며 비즈니스 요구 사항에 맞게 전 세계적으로 확장할 수 있습니다.
- 온라인 및 오프라인 동기화 기능을 통해 여러 장치 간에 데이터를 동기화할 수 있습니다.
- 개인 응용 프로그램 데이터를 쉽게 관리하는 데 사용할 수 있는 모바일 클라이언트 SDK입니다.
- iOS, 안드로이드, 자마린 및 리폰 네이티브에 대한 플랫폼 지원.

**참조**

- [앱 센터 데이터 시작](/appcenter/data/getting-started)

## <a name="azure-cosmos-db"></a>Azure Cosmos DB
[Azure Cosmos DB](https://azure.microsoft.com/services/cosmos-db/)는 전 세계에 배포된 다중 모델 데이터베이스 서비스입니다. 이를 사용하여 행성 규모의 응용 프로그램을 빌드할 수 있습니다. Azure Cosmos DB를 사용하면 전 세계 모든 Azure 리전에서 처리량 및 저장소를 탄력적이고 독립적으로 확장할 수 있습니다. 즐겨 찾는 API 표면을 사용하여 한 자리 수 밀리초의 빠른 데이터 액세스를 활용할 수 있습니다. 이러한 표면에는 SQL, 몽고DB, 카산드라, 테이블 또는 그렘린이 포함됩니다. Azure Cosmos DB는 처리량, 대기 시간, 가용성 및 일관성에 대한 포괄적인 서비스 수준 계약(SLA)을 고유하게 제공합니다.

**주요 기능**
- SQL(코어) API, 카산드라 API, 몽고DB API, 그렘린 API 및 테이블 API를 포함하는 광범위한 API를 지원합니다.
- 턴키 글로벌 배포는 사용자가 어디에 있든 데이터를 복제합니다. 사용자는 자신과 가장 가까운 데이터의 복제본과 상호 작용할 수 있습니다.
- 데이터베이스 엔진이 완전히 스키마에 구애받지 않으므로 스키마 또는 인덱스 관리가 없습니다.
- Azure Cosmos DB는 퍼블릭 클라우드의 54개 이상의 리전을 포함하는 전 세계 모든 Azure 지역에서 사용할 수 있기 때문에 유비쿼터스 지역 입니다.
- Azure Cosmos DB의 다중 마스터 복제 프로토콜은 잘 정의된 다섯 가지 일관성 선택을 제공하도록 신중하게 설계되었기 때문에 정확하게 정의된 여러 일관성 선택. 이 다섯 가지 선택은 강력하고, 경계가 있는 부실, 세션, 일관된 접두사 및 최종입니다.
- 읽기와 쓰기 모두에 대해 99.999% 가용성.
- 프로그래밍 방식으로(또는 Azure 포털을 통해) Azure Cosmos DB 계정의 지역 장애 조치(failover)를 호출하여 응용 프로그램이 지역 재해를 견딜 수 있도록 설계되었는지 확인합니다.
- 전 세계 99번째 백분위수에서 낮은 대기 시간을 보장합니다.

**참조**

## <a name="azure-sql-database"></a>Azure SQL Database
 [Azure SQL Database는](https://azure.microsoft.com/services/sql-database/) 범용 관계형 데이터베이스 관리 서비스입니다. Azure 클라우드의 응용 프로그램 및 솔루션에 대해 가용성이 높은 고성능 데이터 저장소 계층을 만드는 데 사용할 수 있습니다.

**주요 기능**
- **탄력적 데이터베이스 모델 및 도구:** 탄력적 데이터베이스를 사용하면 개발자는 확장을 위해 데이터베이스 그룹 간에 리소스를 풀링할 수 있습니다. 이러한 리소스를 관리하려면 스크립트를 작업으로 제출합니다. 그런 다음 SQL 데이터베이스는 데이터베이스 전체에서 스크립트를 수행합니다.
- **고성능:** 처리량이 높은 응용 프로그램은 최신 버전을 활용할 수 있습니다. 25% 더 많은 프리미엄 데이터베이스 전력을 제공합니다.
- **백업, 복제 및 고가용성:** 데이터베이스 수준에서 기본 제공 복제 및 Microsoft 지원 SLA는 응용 프로그램 연속성을 제공하고 치명적인 이벤트에 대한 보호를 제공합니다. 활성 지역 복제를 사용하면 장애 조치 및 셀프 서비스 복원을 구성할 수 있으므로 "oops 복구"를 완전히 제어할 수 있습니다. 데이터 복원은 최대 35일의 데이터 백업에서 사용할 수 있습니다.
- **0에 가까운 유지 보수:** 자동 소프트웨어는 서비스의 일부입니다. 내장된 시스템 복제본은 고유한 데이터 보호, 데이터베이스 가동 시간 및 시스템 안정성을 제공하는 데 도움이 됩니다. 시스템 복제본이 자동으로 새 컴퓨터로 이동됩니다. 오래된 것들이 실패하면 즉석에서 프로비저닝됩니다.
- **보안:** Azure SQL Database는 조직 또는 업계에서 요구하는 규정 준수 정책을 충족하는 보안 기능 포트폴리오를 제공합니다.
    - 감사는 개발자에게 규정 준수 관련 작업을 수행하고 활동에 대한 지식을 얻을 수 있는 기능을 제공합니다.
    - 개발자와 IT는 데이터베이스 수준에서 정책을 구현하여 행 수준 보안, 동적 데이터 마스킹 및 Azure SQL Database에 대한 투명한 데이터 암호화를 통해 중요한 데이터에 대한 액세스를 제한할 수 있습니다.
    - Azure SQL Database는 HIPAA BAA, ISO/IEC 27001:2005, FedRAMP 및 EU 모델 조항과 같은 주요 Azure 규정 준수 인증 및 승인 범위의 일부로 주요 클라우드 감사기관에서 확인합니다.
