# <a name="store-sync-and-query-mobile-application-data-from-the-cloud"></a>클라우드에서 모바일 응용 프로그램 데이터 저장, 동기화 및 쿼리
작성 하는 응용 프로그램의 종류에 관계 없이 데이터를 생성 하 고 처리 하는 경우가 많습니다. 사용자 기대치는 많고, 모든 상황에서 응용 프로그램이 신속 하 고 원활 하 게 작동 하도록 하려고 합니다. 대부분의 응용 프로그램은 여러 장치에서 작동 하므로 응용 프로그램에 액세스할 때 데스크톱 이나 모바일 장치에서 응용 프로그램에 액세스할 수 있습니다. 여러 사용자가 동시에 응용 프로그램을 사용 하 고 데이터를 공유 하 여 데이터에 대 한 즉각적인 실시간 액세스를 얻을 수 있습니다.

이를 기반으로 응용 프로그램 사용자는 항상 인터넷에 연결 되지 않습니다. 응용 프로그램은 인터넷 연결을 사용 하거나 사용 하지 않고 설계 되 고 사용할 것으로 예상 됩니다. 이러한 모든 복잡성이 증가 하는 것은 개발자가 자신의 응용 프로그램에 뛰어난 고객 환경을 제공 하기 위해 데이터를 저장 하 고 클라우드에 동기화 하는 데 적합 한 솔루션을 선택할 수 있는 큰 작업입니다.

Microsoft에서 제공 하는 다양 한 서비스를 통해 서버를 실행 하거나, 데이터베이스를 선택 하거나, 규모 또는 보안에 대해 걱정 하 여 가능한 한 풍부한 환경을 제공할 필요가 없습니다. 이러한 서비스는 SQL 또는 NoSQL Api를 사용 하 여 클라우드에 응용 프로그램 데이터를 저장 하 고, 모든 장치에서 데이터를 동기화 하 고, 응용 프로그램을 네트워크 연결을 사용 하거나 사용 하지 않고 작동 하 여 확장 가능 하 고 강력한 기능을 구축할 수 있도록 하는 뛰어난 개발자 환경을 제공 합니다. 프로그램도.

다음 서비스를 사용 하 여 클라우드에서 모바일 응용 프로그램 데이터를 관리 하 고 저장 합니다.

## <a name="visual-studio-app-center"></a>Visual Studio App Center
[App Center 데이터](/appcenter/data/) 는 응용 프로그램이 온라인 및 오프 라인 시나리오에서 서로 다른 장치 및 플랫폼에서 클라우드의 응용 프로그램 데이터를 관리, 유지 및 동기화 할 수 있도록 하는 데이터 관리 서비스입니다. 사용자 기반으로 확장 되 고 응용 프로그램 수가 증가 함에 따라 확장 되는 Cosmos DB 서비스를 기반으로 **구축** 된이 서비스는 모든 데이터에 대해 낮은 대기 시간, 고가용성 및 높은 확장성을 보장 합니다.

**주요 기능**
- **새 Cosmos DB 데이터베이스를 쉽게 프로 비전** 하거나 App Center 포털에서 **기존 Cosmos DB 데이터베이스에 연결** 합니다.
- 응용 프로그램 데이터를 쉽게 저장, 동기화 및 쿼리할 수 있는 **Nosql 데이터베이스 지원**
- **Cosmos DB**을 기반으로 구축 된이 서비스는 Azure Cosmos DB에서 제공 하는 주요 기능을 대부분 상속 하며 비즈니스 요구 사항에 맞게 **전역적으로 확장할** 수 있습니다.
- 장치 간에 데이터를 동기화 하는 **온라인 및 오프 라인 동기화** 기능.
- 두 개인 응용 프로그램 데이터를 쉽게 관리할 수 있는 모바일 **클라이언트 sdk**
- **플랫폼 지원** -IOS, Android, Xamarin, 네이티브에 반응 합니다.

## <a name="azure-cosmos-db"></a>Azure Cosmos DB
[Azure Cosmos DB](https://azure.microsoft.com/services/cosmos-db/) 은 전 세계적 규모의 응용 프로그램을 구축 하는 데 도움이 되는 Microsoft의 전 세계적으로 분산 된 다중 모델 데이터베이스 서비스입니다. Cosmos DB에서 단추를 한 번만 클릭하면 전 세계의 수많은 Azure 지역에서 처리량과 스토리지의 크기를 탄력적이고 독립적으로 조정할 수 있습니다. 처리량 및 저장소의 크기를 조정 하 고, SQL, MongoDB, Cassandra, 테이블 또는 Gremlin를 포함 하 여 즐겨 사용 하는 API 화면을 사용 하 여 빠르고 단일 자리 밀리초 데이터 액세스를 활용할 수 있습니다. Cosmos DB는 처리량, 대기 시간, 가용성 및 일관성에 대 한 포괄적인 Sla (서비스 수준 계약)를 고유 하 게 제공 합니다.

**주요 기능**
- 는 SQL (Core) API, Cassandra API, MongoDB API, Gremlin API 및 Table API를 포함 하는 **광범위 한 api** 를 지원 합니다.
- **턴키 전역 배포** 는 사용자가 어디에 있든 데이터를 복제 하므로 사용자가 가장 가까운 데이터의 복제본과 상호 작용할 수 있습니다.
- 데이터베이스 엔진으로 **스키마 또는 인덱스 관리** 는 완전히 스키마에 구애 받지 않습니다.
- 전 세계 모든 Azure 지역에서 **Cosmos DB를 사용할** 수 있으며, 공용 클라우드의 54 개 이상의 지역을 포함 합니다.
- **정확 하 게 정의 된 여러 일관성 선택** 은 Cosmos DB의 다중 마스터 복제 프로토콜입니다. 강력, 제한 된 부실, 세션, 일관적인 접두사 및 최종의 5 가지 잘 정의 된 일관성 선택 항목을 제공 하도록 신중 하 게 설계 되었습니다.
- 읽기 및 쓰기 둘 다에 대해 **99.999%의 가용성** 을 제공 합니다.
- **프로그래밍 방식으로 (또는 포털을 통해) Cosmos 계정의 지역 장애 조치 (failover)를 호출** 하 여 응용 프로그램이 지역 재해를 견딜 수 있도록 설계 되었는지 확인 합니다.
- 전 세계 **99 번째 백분위 수의 짧은 대기 시간을 보장**합니다.

## <a name="azure-sql-database"></a>Azure SQL Database
 [Azure SQL Database](https://azure.microsoft.com/services/sql-database/) 는 Azure 클라우드의 응용 프로그램 및 솔루션을 위한 고가용성 및 고성능 데이터 저장소 계층을 만들 수 있도록 하는 범용 관계형 데이터베이스 관리 서비스입니다.

**주요 기능**
- **탄력적 데이터베이스 모델 및 도구** -탄력적 데이터베이스를 사용 하면 크기를 조정 하기 위해 데이터베이스 그룹에서 사용 하는 리소스를 풀 수 있으며,이를 통해 스크립트를 작업으로 제출 하 여 관리 하 고 SQL Database을 수행할 수 있습니다. 데이터베이스 전체를 스크립팅 합니다.
- **고성능-처리량** 이 높은 응용 프로그램은 25% 더 premium 데이터베이스 기능을 제공 하는 최신 버전을 활용할 수 있습니다.
- **백업, 복제 및 고가용성** -데이터베이스 수준의 MICROSOFT 지원 SLA는 응용 프로그램 연속성 및 치명적인 이벤트 로부터 보호를 제공 합니다. 활성 지역 복제를 사용 하면 장애 조치 (failover) 및 셀프 서비스 복원을 구성할 수 있으며,이를 통해 "복구" (최대 35 일 동안 사용 가능한 데이터 백업에서 데이터 복원)를 완벽 하 게 제어할 수 있습니다.
- **0에 가까운 유지 관리** -자동 소프트웨어는 서비스의 일부 이며 기본 제공 시스템 복제본은 기본적인 데이터 보호, 데이터베이스 작동 시간 및 시스템 안정성을 제공 하는 데 도움이 됩니다. 시스템 복제본은 새로운 컴퓨터로 자동으로 이동되어, 이전 시스템에 장애가 발생하는 즉시 프로비전됩니다.
- **보안** SQL Database 조직 또는 업계 규정 준수 정책을 충족 하는 보안 기능 포트폴리오를 제공 합니다.
    - 감사는 개발자에 게 규정 준수 관련 작업을 수행 하 고 작업에 대 한 정보를 얻을 수 있는 기능을 제공 합니다.
    - 개발자와 IT는 데이터베이스 수준에서 정책을 구현 하 여 행 수준 보안, 동적 데이터 마스킹 및 Azure SQL Database에 대 한 투명 한 데이터 암호화를 통해 중요 한 데이터에 대 한 액세스를 제한할 수 있습니다.
    - SQL Database은 HIPAA BAA, ISO/IEC 27001:2005, FedRAMP 및 EU Model 절과 같은 주요 Azure 준수 인증 및 승인 범위의 일부로 주요 클라우드 감사자에 의해 확인 됩니다.
