# <a name="cognitive-services-helps-to-solve-business-problems-by-using-ai"></a>Cognitive Services AI를 사용 하 여 비즈니스 문제를 해결 하는 데 도움이 됩니다.
[Azure Cognitive Services](https://azure.microsoft.com/services/cognitive-services/) 는 사용자가 인공 지능 (인공 지능)으로 비즈니스를 변환 하는 데 필요한 사용자 요구를 확인 하 고, 듣고, 말하고, 이해 하 고, 해석 하기 위한 지능형 알고리즘으로 앱을 의사 소통 방식는 강력한 서비스를 제공 합니다 사용자와 관련 하 여 모바일 응용 프로그램을 스마트, 사용 및 컨텍스트 하 게 만들 수 있습니다. 개발자는 Api, Sdk 및 서비스를 사용 하 여 AI를 제품에 추가 하 고 직접 AI 또는 데이터 과학 기술을 사용 하지 않고도 지능형 응용 프로그램을 빌드할 수 있습니다.

## <a name="cognitive-services-capabilities"></a>Cognitive Services 기능
- **Vision**
    - 이미지 태그 지정, 미리 보기
    - OCR (필기 인식)
    - 얼굴 감지
    - 감정 감지
    - 비디오 인사이트
    - 이미지 및 비디오 중재
    - 사용자 지정 된 이미지 인식

- **음성**
    - 음성 텍스트 (음성 기록)
    - 사용자 지정 음성 기록 (복잡 한 단어, 잡음이 있는 환경)
    - 텍스트 음성 변환
    - 스피커 ID 및 인증
    - 실시간 음성 번역

- **언어**
    - 상황에 맞는 언어 이해, 사용자 지정 된 의도 분석
    - 감정 분석, 키 구 검색
    - 60 개 언어 까지의 텍스트 번역
    - 텍스트 조정
    - 맞춤법 검사

- **결정** 
    - 콘텐츠 중재자
    - 개인 설정 된 콘텐츠 배달
    - Anomaly Detector

- **검색**
    - 자동 검색 제안 
    - 엔터티 정보 확대
    - 포괄적인 뉴스, 이미지, 비디오, 웹 및 로컬 비즈니스 엔터티 검색
    - 맞춤형 및 사용자 지정 된 검색 환경

- Cognitive Services에 대 한 컨테이너 지원
