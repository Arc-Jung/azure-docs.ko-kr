# <a name="cognitive-services-helps-to-solve-business-problems-by-using-ai"></a>코그너티브 서비스는 AI를 사용하여 비즈니스 문제를 해결하는 데 도움이 됩니다.
[Azure Cognitive Services는](https://azure.microsoft.com/services/cognitive-services/) 인공 지능(AI)으로 비즈니스를 혁신하기 위해 사용자의 요구를 보고, 듣고, 말하고, 이해하고, 해석하는 지능형 알고리즘을 앱에 주입하는 강력한 서비스를 제공합니다. 모바일 애플리케이션을 스마트하고 매력적이며 상황에 따라 사용자에게 적합하게 만들 수 있습니다. 개발자는 API, SDK 및 서비스를 사용하여 제품에 AI를 추가하고 직접적인 AI 또는 데이터 과학 기술 없이도 지능형 애플리케이션을 구축할 수 있습니다.

## <a name="cognitive-services-capabilities"></a>코그너티브 서비스 기능
- **시각**
    - 이미지 태그 지정, 썸네일
    - OCR(필기 인식)
    - 얼굴 감지
    - 감정 감지
    - 비디오 인사이트
    - 이미지 및 비디오 검토
    - 맞춤형 이미지 인식

- **Speech**
    - 텍스트 음성(음성 녹음)
    - 사용자 정의 음성 전사 (복잡한 단어, 시끄러운 환경)
    - 텍스트 음성 변환
    - 스피커 ID 및 인증
    - 실시간 음성 번역

- **언어**
    - 컨텍스트 언어 이해, 맞춤형 의도 분석
    - 감정 분석, 핵심 문구 감지
    - 최대 60개 언어로 텍스트 번역
    - 텍스트 조정
    - 맞춤법 검사

- **의사 결정** 
    - 콘텐츠 중재자
    - 개인화된 콘텐츠 전송
    - Anomaly Detector

- **검색**
    - 자동 검색 제안 
    - 엔터티 정보 보강
    - 종합 뉴스, 이미지, 비디오, 웹 및 지역 비즈니스 엔터티 검색
    - 맞춤형 맞춤형 검색 환경

- 코그너티브 서비스를 위한 컨테이너 지원
