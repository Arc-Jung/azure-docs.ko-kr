# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Azure Maps를 사용 하 여 지도, 공간 분석 및 이동성 솔루션을 추가 하는 지리 공간적 Api

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) 는 클라우드에서 기본적으로 위치 플랫폼을 제공 하 고 새로운 매핑 데이터를 사용 하 여 웹 및 모바일 응용 프로그램에 정확한 지리적 컨텍스트를 제공 하는 지리 공간적 서비스 포트폴리오를 제공 합니다. 개발자, 엔터프라이즈 및 Isv는 Azure Maps을 사용 하 여 위치 인식 응용 프로그램 및 IoT, 이동성, 물류 및 자산 추적 솔루션을 만들 수 있습니다.

## <a name="azure-maps-features"></a>Azure Maps 기능
- **검색**: 주변 관심 지점에 대 한 세부 정보를 제공 하 고, 지도에 위치를 렌더링 하 고, geocode (또는 역방향 geocode) 주소를 사용 하 여 해당 위도 및 경도 좌표를 가져옵니다.
- **트래픽**: 이동 시간을 개선 하 고 gridlock을 방지 하는 데 도움을 줍니다.
- **Maps**: 솔루션에 통합 하기 위한 명확 하 고 읽기 쉬운 맵을 제공 합니다.
- **라우팅**: 특정 시간에 여러 대상에서 사용할 수 있는 가장 짧은 경로 또는 가장 빠른 경로를 표시 합니다. 또한 라우팅은 워커, bicyclists 및 상업적 차량에 대 한 특수 경로와 팩트를 제공 합니다.
- **표준 시간대**: 전 세계 어디에 있든 시간을 쉽게 확인할 수 있습니다.
- **공간 작업**: 위치 인텔리전스를 향상 시킵니다.
- **지리적**위치: IP 주소의 국가를 찾습니다.
- **데이터 서비스**: 공간 작업 또는 이미지 컴퍼지션에 사용할 지리 공간 데이터를 업로드 및 저장 합니다.
- **이동성**: 가까운 공용 전송 서비스에 대 한 실시간 위치 인텔리전스를 제공 합니다. 중지, 경로 정보 및 이동 시간 예측이 포함 됩니다.
