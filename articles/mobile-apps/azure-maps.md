# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Azure 지도를 사용하여 맵, 공간 분석 및 모빌리티 솔루션을 추가하는 지리 공간 API

[Azure Maps는](https://azure.microsoft.com/services/azure-maps/) 기본적으로 클라우드에 위치 플랫폼을 제공하며 웹 및 모바일 응용 프로그램에 정확한 지리적 컨텍스트를 제공하기 위해 최신 매핑 데이터를 사용하는 지리 공간 서비스 포트폴리오를 제공합니다. 개발자, 기업 및 ISV는 Azure Maps를 사용하여 위치 인식 응용 프로그램 및 IoT, 모빌리티, 물류 및 자산 추적 솔루션을 만들 수 있습니다.

## <a name="azure-maps-features"></a>Azure 지도 기능
- **검색**: 관심 있는 근처의 지점에 대한 세부 정보를 제공하고, 맵에서 위치를 렌더링하고, 지오코드(또는 역방향 지오코드) 주소를 사용하여 해당 위도 및 경도 좌표를 가져옵니다.
- **교통**: 여행 시간을 개선하고 그리드 락을 방지하는 데 도움이됩니다.
- **지도**: 명확하고 읽기 쉬운 맵을 제공하여 솔루션에 통합할 수 있습니다.
- **라우팅**: 특정 시간에 여러 목적지에서 사용할 수 있는 가장 짧거나 빠른 경로를 제공합니다. 라우팅은 또한 보행자, 자전거 및 상용차에 대한 전문 경로 및 사실을 제공합니다.
- **시간대**: 세계 어느 곳에서나 몇 시에 있는지 쉽게 볼 수 있습니다.
- **공간 작업**: 위치 인텔리전스를 향상시킵니다.
- **지리적 위치**: IP 주소의 국가를 찾습니다.
- **데이터 서비스**: 공간 작업 또는 이미지 구성과 함께 사용할 지리 공간 데이터를 업로드하고 저장합니다.
- **모빌리티**: 인근 대중 교통 서비스에 대한 실시간 위치 인텔리전스를 제공합니다. 경유지, 경로 정보 및 이동 시간 예측이 포함됩니다.
