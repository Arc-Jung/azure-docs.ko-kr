# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Azure Maps를 사용 하 여 지도, 공간 분석 및 이동성 솔루션을 추가 하는 지리 공간적 Api

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) 는 Azure 클라우드에서 기본적으로 위치 플랫폼을 제공 하며, 새로운 매핑 데이터를 사용 하 여 웹 및 모바일 응용 프로그램에 정확한 지리적 컨텍스트를 제공 하는 지리 공간적 서비스 포트폴리오를 제공 합니다. 이를 통해 개발자, 엔터프라이즈 및 Isv는 위치 인식 응용 프로그램 및 IoT, 이동성, 물류 및 자산 추적 솔루션을 만들 수 있습니다.

## <a name="azure-maps-features"></a>Azure Maps 기능
- **검색** 은 해당 하는 위도/경도 좌표를 가져오기 위해 근처의 관심 지점 (poi), 지도의 렌더링 위치 및 geocode (또는 역방향 geocode) 주소에 대 한 세부 정보를 제공 합니다.
- **트래픽**사용을 통해 이동 시간을 개선 하 고 gridlock을 사용 하지 않습니다.
- 명확 하 고 읽기 쉬운 **지도** 를 솔루션에 통합 합니다.
- **경로** 를 사용 하 여 사용 가능한 최단 경로 또는 가장 빠른 경로를 한 번에 여러 대상으로 제공 하거나 워커, bicyclists 및 상업적 차량에 대 한 특수 한 경로와 팩트를 제공 합니다.
- **표준 시간대** 를 사용 하면 전 세계 어디에서 든 지 사용자가 쉽게 확인할 수 있습니다.
- **공간 작업** 을 통해 위치 인텔리전스를 향상 시킬 수 있습니다.
- **지리적** 위치는 IP 주소의 국가를 조회 합니다.
- 공간 작업 또는 이미지 컴퍼지션에 사용할 지리 공간 데이터를 업로드 및 저장 하는 **데이터 서비스** 입니다.
- **이동성** 은 중지, 경로 정보 및 이동 시간 추정치를 포함 하 여 가까운 공용 전송 서비스에 대 한 실시간 위치 인텔리전스를 제공 합니다.
