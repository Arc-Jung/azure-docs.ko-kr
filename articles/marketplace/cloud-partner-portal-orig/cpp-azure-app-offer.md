# <a name="azure-application-offer"></a>Azure 애플리케이션 제품

<table> <tr> <td>이 섹션에서는 관리되는 새 Azure 애플리케이션 제품을 <a href="https://azuremarketplace.microsoft.com/marketplace/">Microsoft Azure Marketplace</a>에 게시하는 방법을 설명합니다.  각 관리되는 애플리케이션에는 일반적으로 하나 이상의 가상 머신 및 기타 지원 Azure 기반 서비스 또는 웹 기반 서비스를 포함하는 애플리케이션에서 사용하는 모든 기술 자산을 정의하는 Azure Resource Manager 템플릿이 포함됩니다. </td> <td><img src="./media/cloud-partner-portal-publish-managed-app/azureapp-icon1.png"  alt="Azure application icon" /></td> </tr> </table>

## <a name="publishing-overview"></a>게시 개요

[솔루션 템플릿 및 Azure Marketplace에 대한 관리되는 애플리케이션 빌드](https://channel9.msdn.com/Events/Build/2018/BRK3603) 비디오는 Azure Resource Manager 템플릿을 작성하여 Azure 애플리케이션 솔루션을 정의하는 방법 및 이후에 앱 제품을 Azure Marketplace에 게시하는 방법을 간략하게 보여줍니다.

> [!VIDEO https://channel9.msdn.com/Events/Build/2018/BRK3603/player]


## <a name="publishing-process"></a>게시 프로세스

다음 문서에서는 관리되는 Azure 애플리케이션을 만들고 게시하는 프로세스를 안내합니다.

1. Azure Marketplace에 게시하기 위한 승인을 받으려면 [Microsoft 개발자 센터에 계정을 등록합니다](./cloud-partner-portal-dev-center-accounts-registration.md).
2. Azure Marketplace에 [새 Azure 애플리케이션 제품을 게시합니다](./cloud-partner-portal-managed-app-publish.md).
3. [제품을 라이브로 전환합니다](./cloud-partner-portal-managed-app-offer-go-live.md)! 


## <a name="next-steps"></a>다음 단계

다음으로, 이 제품 유형에 필요한 요구 사항을 확인하고 [관리되는 애플리케이션 작성](./cloud-partner-portal-managed-app-publish.md)에 설명된 대로 게시 프로세스를 시작해야 합니다.
