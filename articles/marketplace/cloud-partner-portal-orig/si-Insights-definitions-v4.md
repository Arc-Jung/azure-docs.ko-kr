<a name="seller-insights-definitions"></a>Seller Insights 정의
=======================

아래 표에는 Seller Insights에서 사용되는 여러 용어의 정의가 나와 있습니다.

|  **용어**                 |  **정의**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Azure 라이선스 유형**                               | Azure 구매 고객이 체결하는 라이선싱 계약 유형입니다.  채널이라고도 합니다.                                                  |
| **Azure 라이선스 유형: 클라우드 솔루션 공급자**      | 재판매인 역할을 하는 클라우드 솔루션 공급자를 통해 최종 고객이 Azure 및 Marketplace 제품을 조달받습니다.                 |
| **Azure 라이선스 유형: 기업**                   | Microsoft와 직접 체결하는 기업계약을 통해 최종 고객이 Azure 및 Marketplace 제품을 조달받습니다.                  |
| **Azure 라이선스 유형: 기업(재판매인 있음)**  | 최종 고객은 Microsoft와의 엔터프라이즈 계약을 용이하게 하는 리셀러를 통해 Azure 및 Marketplace 오퍼를 조달합니다.     |
| **Azure 라이선스 유형: 종량제**                | 최종 고객은 Microsoft와 직접 서명한 "Pay as You Go" 계약을 통해 Azure 및 마켓플레이스 오퍼를 조달합니다.                |
| **청구 금액(CC)**                              | 청구에 대한 고객 통화(CC)로 고객에게 청구되는 금액입니다. *Customer Currency*                                                                 |
| **청구 금액(PC)**                               | *고객에게* 청구된 금액(PC)입니다.                                                                      |
| **청구 날짜**                                      | 고객의 요금이 계산된 날짜(일반적으로 사용 기간 직후).                                             |
| **클라우드 인스턴스 이름**                              | VM이 배포된 Microsoft Cloud입니다.                                   |
| **클라우드 인스턴스 이름: Azure 전역**                | 퍼블릭 글로벌 마이크로소프트 클라우드.                           |
| **클라우드 인스턴스 이름: Azure 정부**        | 중국, 독일 또는 미국 중 하나에 대 한 정부 별 Microsoft 클라우드입니다.                           |
| **컬렉션 상태**  | 특정 요금의 경우 청구 및 수집 수명 주기의 최신 상태입니다.  예를 들어 상태는 진행 중인 컬렉션, 수집 또는 환불이 될 수 있습니다.                  |
| **코어 크기**  | 오퍼가 실행 중인 가상 시스템에 의해 제공되는 가상 코어 의 수입니다.               |
| **고객 구/군/시**  | 고객이 제공한 도시 이름입니다. 이는 고객의 Azure 구독의 도시와 다를 수 있습니다.  |
| **고객 통신 언어**        | 고객이 통신용으로 선택한 기본 설정 언어입니다.                                  |
| **고객 회사 이름**                  | 고객이 제공한 회사 이름입니다. 이는 고객의 Azure 구독의 도시와 다를 수 있습니다.                                  |
| **고객 국가**                       | 고객이 제공한 국가/지역 이름입니다.  이는 고객의 Azure 구독의 국가/지역과 다를 수 있습니다.                               |
| **고객 통화(CC)**                 | 고객이 가격 및 청구용으로 선택한 기본 설정 통화입니다.                            |
| **고객 이메일**                         | 최종 고객이 제공한 전자 메일 주소입니다.  이는 고객의 Azure 구독의 전자 메일 주소와 다를 수 있습니다.                   |
| **고객 이름**                              | 고객이 제공한 이름입니다.  이는 고객의 Azure 구독에 제공된 이름과 다를 수 있습니다.                    |
| **고객 성**                               | 고객이 제공한 성입니다.  이는 고객의 Azure 구독에 제공된 이름과 다를 수 있습니다.                    |
| **고객 ID**                               | 고객에게 할당된 고유 식별자입니다.  고객은 Azure 마켓플레이스 구독이 0개 이상일 수 있습니다.                    |
| **고객 지불 유형**                            | 고객이 사용하는 지불 유형입니다.  예를 들어 카드, 송장 또는 전자 수표일 수 있습니다.                             |
| **고객 우편 번호**                             | 고객이 제공한 우편 번호입니다.  이는 고객의 Azure 구독에 제공된 우편 번호와 다를 수 있습니다.                                                                 |
| **고객 시/도**                                   | 고객이 제공한 상태(주소)입니다.  이는 고객의 Azure 구독에 제공된 성과 다를 수 있습니다.                                                                 |
| **취득일**                                    | 고객이 귀하가 게시한 모든 쿠폰을 구매한 첫 번째 날짜입니다.                 |
| **취소일**                                        | 고객이 이전에 구매한 모든 오퍼의 마지막 을 취소한 마지막 날짜입니다.                                                         |
| **예상 요금 소계(CC)**                   | 지정된 SKU(고객통화)의 사용량 단위 수량에 대한 예상 연장 요금입니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                            |
| **예상 요금 소계(PC)**                   | 지정된 SKU의 사용 단위 수량에 대한 예상 확장 요금은 사용 일자에 대한 외환 변환을 기준으로 계산됩니다(게시자의 통화). 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                  |
| **예상 지급액(PC)**                            | 사용량이 계산된 날짜(게시자의 통화)에 대한 환율 변환을 기준으로 지정된 SKU의 사용량 단위 수량에 대한 예상 지급액입니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                        |
| **예상 가격(PC)**                             | 지정된 SKU의 사용 단위에 대한 예상 가격은 사용 일자에 대한 외환 변환을 기준으로 계산됩니다(게시자의 통화). 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                                             |
| **새로운 고객입니다.**                                  | 이 값을 통해 마켓플레이스에서 처음으로 제품을 구매하거나 구매하는 신규 고객이 있는지 알 수 있습니다.  "취득한 날짜"의 동일한 달인 경우 값은 "예"가 됩니다.  고객이 보고된 달 이전에 쿠폰을 구매한 경우 이 값은 "아니요"가 됩니다.                                                       |
| **미리 보기 SKU입니다.**                                  | 이 값은 SKU에 "미리 보기"로 태그를 지정한 경우 알려드립니다. SKU에 따라 태그가 지정된 경우 값은 "예"이며 이 이미지를 배포하고 사용할 수 있는 권한이 지정된 Azure 구독만 사용할 수 있습니다.  SKU가 "미리 보기"로 식별되지 않은 경우 값은 "아니요"가 됩니다.                                          |
| **프로모션 연락처 옵트인**                   | 이 값을 통해 고객이 게시자의 프로모션 연락처를 사전에 선택했는지 확인할 수 있습니다. 현재는 고객에게 해당 옵션이 제공되지 않으므로 보드 전체에 “아니요”가 표시됩니다. 이 기능이 배포되면 그에 따라 업데이트를 시작할 예정입니다.                                          |
| **Marketplace 라이선스 유형**                         | Marketplace 제품의 청구 방법입니다.                             |
| **Marketplace 라이선스 유형: Azure를 통해 청구됨**   | Microsoft가 이 Marketplace 제품의 대리인으로 게시자 대신 고객에게 대금을 청구합니다. (종량제 신용 카드 또는 기업 송장)       |
| **Marketplace 라이선스 유형: BYOL(사용자 라이선스 필요)** | VM을 배포하려면 고객이 제공하는 라이센스 키가 필요합니다. Microsoft는 이 Marketplace 제품 사용에 대해 고객에게 대금을 청구하지 않습니다. |
| **Marketplace 라이선스 유형: 무료**                   | Marketplace 제품이 모든 사용자에게 무료로 제공되도록 구성됩니다. Microsoft는 이 Marketplace 제품 사용에 대해 고객에게 대금을 청구하지 않습니다.    |
| **Marketplace 라이선스 유형: Microsoft - 재판매인**  | Microsoft가 이 Marketplace 제품의 재판매인 역할을 합니다.                               |
| **마켓플레이스 구독 회사 이름**  | Azure 구독에서 고객이 제공한 회사 이름입니다.                           |
| **마켓플레이스 구독 ID**            | 고객이 마켓플레이스 오퍼를 구입하는 데 사용한 Azure 구독과 연결된 고유 식별자입니다.  이전에는 Azure 구독 GUID였습니다.  SAAS 주문과 같은 경우에 따라 구독이 필요하지 않습니다.           |
| **Microsoft 수수료(CC)**                               | Microsoft의 거래 수수료(단위: 고객 통화)입니다.                              |
| **쿠폰 이름**                                | 마켓플레이스 오퍼링의 이름입니다.                                                             |
| **오퍼 유형**                                | Microsoft 마켓플레이스 오퍼링의 유형입니다.  예를 들어 관리되는 응용 프로그램, 가상 컴퓨터 라이센스 또는 컨테이너일 수 있습니다.                           |
| **주문 취소 날짜**                         | Marketplace 주문이 취소된 날짜입니다.                                                       |
| **주문 ID**                                  | 마켓플레이스 서비스에 대한 고객 주문의 고유 식별자입니다.  가상 컴퓨터 사용 기반 오퍼는 주문과 연결되지 않습니다.                 |
| **주문 구매 날짜**                       | Marketplace 주문이 생성된 날짜입니다.                                                        |
| **Order Quantity**                       | Azure 구독과 연결된 동일한 SKU에 대한 주문 수입니다.  SAAS 주문의 경우 이 주문 인스턴스와 연결된 시트 수를 나타냅니다.                                            |
| **주문 상태**                              | 데이터를 마지막으로 새로 고칠 때의 Marketplace 주문 상태입니다.  예를 들어 주문이 활성 상태이거나 취소될 수 있습니다.             |
| **주문 상태: 활성**                           | 고객이 주문을 구매했으며 주문을 취소하지 않았습니다.   |
| **주문 상태: 취소됨**                        | 고객은 이전에 주문을 구매한 후 주문을 취소했습니다.        |
| **지급 금액(PC)**                             | 게시자에게 지급되는 금액입니다. 게시자의 기본 설정 PC(지급 통화) 단위입니다.                                       |
| **PC(지급 통화)**                           | 지급에 사용되는 통화입니다.                                                                |
| **지급 날짜**                                    | Microsoft에서 선택한 금융 기관으로 결제 요청을 보낸 날짜입니다.                                                                         |
| **지급 상태**                                  | 데이터가 마지막으로 새로 고쳐진 시점의 지급 현황입니다.                                   |
| **지급 상태: 진행 중**               | 거래에 대한 지급이 준비되지 않았습니다. (자세한 내용은 컬렉션 상태 참조)                             |
| **지급 상태: 지급됨**                          | 거래가 이전 지급 계산에 포함되었습니다. 계산 결과가 양수인 경우 해당 금액이 지급되었으며, 음수인 경우에는 총 지불 금액에서 차감되었습니다. |
| **지급 상태: 지급 예정**                   | 거래에 대한 지급이 준비되었으며 사용 가능한 다음 지급 계산에 포함될 예정입니다.                      |
| **가격(CC)**                                   | 지정된 SKU의 사용 단위 가격(단위: 고객 통화)입니다.                                                                  |
| **공급자 이메일**                                 | Microsoft와 최종 고객 간의 관계에 관련된 공급자의 전자 메일 주소입니다. 고객이 리셀러를 통해 엔터프라이즈인 경우 리셀러가 됩니다.  CSP(클라우드 솔루션 공급자)가 관련된 경우 CSP가 됩니다.                                                                 |
| **공급자 이름**                                 | Microsoft와 최종 고객 간의 관계에 관련된 공급자의 이름입니다. 고객이 리셀러를 통해 엔터프라이즈인 경우 리셀러가 됩니다.  CSP(클라우드 솔루션 공급자)가 관련된 경우 CSP가 됩니다.                                                                 |
| **리소스 URI**                                  | 배포 아티팩트가 배포된 경로입니다.                                            |
| **Sku**                                           | 게시 중에 정의한 SKU 이름입니다. 한 제품에 여러 SKU가 포함될 수는 있지만 각 SKU는 제품 하나에만 연결할 수 있습니다.              |
| **SKU 청구 유형**                                 | SKU의 청구 방법입니다.                                                                                                             |
| **SKU 청구 유형: 자신의 라이선스 가져오기**         | VM 또는 서비스에는 고객이 배포하기 위해 제공하는 라이센스 키가 필요합니다. Microsoft는 이러한 마켓플레이스 제품의 사용에 대해 고객에게 요금을 청구하지 않습니다.   |
| **SKU 청구 유형: 마이크로소프트 리셀러로**          | Microsoft가 이 SKU의 재판매인 역할을 합니다.   |
| **SKU 청구 유형: 무료**                           | SKU가 모든 사용자에게 무료로 제공되도록 구성됩니다. Microsoft는 이 SKU 사용에 대해 고객에게 대금을 청구하지 않습니다.                           |
| **SKU 청구 유형: 유료**                           | Microsoft가 이 SKU의 대리인으로 게시자 대신 고객에게 대금을 청구합니다. (종량제 신용 카드 또는 기업 송장)                   |
| **SKU 청구 유형: 평가판**                          | 고객이 평가판을 사용하는 중이며, 제품 사용을 취소하거나 삭제하지 않으면 유료로 전환됩니다.                                         |
| **세금액(CC)**                                  | CC(고객 통화) 단위로 고객 청구 금액에 적용되는 세금액입니다.                                                               |
| **거래 날짜**                                 | 지급 보고에 기록된 거래 날짜입니다.                                                                           |
| **트랜잭션 유형**                               | 보고 중인 거래 유형입니다.  예를 들어 요금, 환불 또는 지급 조정일 수 있습니다.                                                                                        |
| **평가판 종료 날짜**                                 | 이 주문의 평가 기간이 종료되었거나 종료될 예정인 날짜입니다.                                                                           |
| **사용 날짜**                                       | 고객이 SKU를 사용한 날짜입니다.                                                                                                             |
| **사용 수량**                                       | 보고된 고객의 SKU 사용량입니다. VM 이미지의 경우 사용량 기록은 해당 VM 크기와 SKU에 대한 보고 대상 기간의 사용량을 나타냅니다.                                               |
| **사용량 기준**                                       | 지급 보고서의 항목과 연결된 특정 SKU를 고객이 사용한 기간(1일 이상)의 식별자입니다.                                               |
| **사용 유형**                                       | 측정 중인 사용에 대한 설명입니다. (정규화된 사용량 또는 원시 사용량)                                                              |
| **사용 유형: 정규화된 사용량**                | 사용량 계산에 포함되는 VM 코어 수를 고려하여 정규화된 시간 단위 사용량입니다. [VM 코어 수] x [원시 사용량(시간)]으로 계산됩니다. “SHARECORE”로 지정된 VM은 [VM 코어 수] 승수로 1/6(0.1666)을 사용합니다.                                        |
| **사용 유형: 원시 사용**                        | Marketplace 제품의 사용 시간입니다.  그것은 "달력"시간의 관점에서 표시됩니다.  가상 시스템이 실행된 시간입니다.                           |
| **사용 단위**                                   | 명시된 사용량의 측정 단위입니다. VM의 경우 항상 시간별 측정 단위로 측정됩니다.                               |
| **VM 크기**                                       | Azure 제품에 맞게 조정된 가상 머신 하드웨어 크기를 나타냅니다. `Basic_A0`, `Standard_A11`, `Standard_D12`, `Standard_G4` 등이 포함됩니다.   |
