<a name="seller-insights-definitions"></a>Seller Insights 정의
=======================

아래 표에는 Seller Insights에서 사용되는 여러 용어의 정의가 나와 있습니다.

|  **용어**                 |  **정의**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Azure 라이선스 유형**                               | Azure 구매 고객이 체결하는 라이선싱 계약 유형입니다.  채널이라고도 합니다.                                                  |
| **Azure 라이선스 유형: 클라우드 솔루션 공급자**      | 재판매인 역할을 하는 클라우드 솔루션 공급자를 통해 최종 고객이 Azure 및 Marketplace 제품을 조달받습니다.                 |
| **Azure 라이선스 유형: Enterprise**                   | Microsoft와 직접 체결하는 기업계약을 통해 최종 고객이 Azure 및 Marketplace 제품을 조달받습니다.                  |
| **Azure 라이선스 유형: 재판매인을 통해 엔터프라이즈**  | 최종 고객은 Microsoft와의 기업계약을 용이 하 게 하는 재판매인을 통해 Azure 및 Marketplace 제품을 procures 합니다.     |
| **Azure 라이선스 유형: 종 량 제**                | 최종 고객은 Microsoft에 직접 서명 하는 "종 량 제" 계약을 통해 Azure 및 Marketplace 제품을 procures.                |
| **청구 금액 (CC)**                              | 청구를 위해 고객에 게 청구 되는 고객 *통화* (CC) 금액입니다.                                                                 |
| **청구 금액 (PC)**                               | *지급 Currency* (PC)에서 고객에 게 청구 되는 금액입니다.                                                                      |
| **청구 날짜**                                      | 고객의 요금이 계산 된 날짜 (일반적으로 사용 기간 바로 다음)입니다.                                             |
| **클라우드 인스턴스 이름**                              | VM이 배포된 Microsoft Cloud입니다.                                   |
| **클라우드 인스턴스 이름: Azure Global**                | 공용 글로벌 Microsoft 클라우드.                           |
| **클라우드 인스턴스 이름: Azure Government**        | 중국, 독일 또는 미국의 정부 중 하나인 정부 관련 Microsoft 클라우드.                           |
| **수집 상태**  | 특정 요금에 대 한 청구 및 수집 수명 주기의 최신 상태입니다.  예를 들어 상태는 진행 중, 수집 됨 또는 환불 됨의 컬렉션 일 수 있습니다.                  |
| **핵심 크기**  | 제품이 실행 되 고 있는 가상 컴퓨터에 표시 되는 가상 코어의 수입니다.               |
| **고객 도시**  | 고객이 제공한 도시 이름입니다. 이는 고객의 Azure 구독에 있는 도시와 다를 수 있습니다.  |
| **고객 통신 언어**        | 고객이 통신용으로 선택한 기본 설정 언어입니다.                                  |
| **고객 회사 이름**                  | 고객이 제공한 회사 이름입니다. 이는 고객의 Azure 구독에 있는 도시와 다를 수 있습니다.                                  |
| **고객 국가**                       | 고객에 의해 제공 되는 국가/지역 이름입니다.  이는 고객의 Azure 구독에서 국가/지역과 다를 수 있습니다.                               |
| **고객 통화 (CC)**                 | 고객이 가격 및 청구용으로 선택한 기본 설정 통화입니다.                            |
| **고객 전자 메일**                         | 최종 고객이 제공한 전자 메일 주소입니다.  이는 고객의 Azure 구독에 있는 전자 메일 주소와 다를 수 있습니다.                   |
| **고객 이름**                              | 고객이 제공한 이름입니다.  이는 고객의 Azure 구독에 제공 된 이름과 다를 수 있습니다.                    |
| **고객 성**                               | 고객이 제공한 성입니다.  이는 고객의 Azure 구독에 제공 된 이름과 다를 수 있습니다.                    |
| **고객 ID**                               | 고객에 게 할당 된 고유 식별자입니다.  고객은 Azure Marketplace 구독이 0 개 이상 있을 수 있습니다.                    |
| **고객 지불 유형**                            | 고객이 사용하는 지불 유형입니다.  예를 들어 카드, 송장 또는 전자 확인이 있습니다.                             |
| **고객 우편 번호**                             | 고객이 제공한 우편 번호입니다.  이는 고객의 Azure 구독에서 제공 되는 우편 번호와 다를 수 있습니다.                                                                 |
| **고객 상태**                                   | 고객이 제공한 상태 (주소)입니다.  이 이름은 고객의 Azure 구독에 제공 된 성에 따라 다를 수 있습니다.                                                                 |
| **가져온 날짜**                                    | 고객이 게시 한 제품을 처음 구입한 날짜입니다.                 |
| **손실 날짜**                                        | 고객이 이전에 구매한 모든 제품의 마지막 날짜를 취소 한 마지막 날짜입니다.                                                         |
| **예상 연장 요금 (CC)**                   | 지정 된 SKU (고객의 통화)의 사용 단위 수량에 대 한 예상 연장 요금입니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                            |
| **예상 연장 요금 (PC)**                   | 날짜 사용에 대 한 외부 exchange 변환을 기반으로 지정 된 SKU에 대 한 사용량 단위에 대 한 예상 연장 비용은 계산 됩니다 (게시자의 통화). 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                  |
| **예상 지급 (PC)**                            | 사용이 계산 된 날짜에 대 한 외부 교환 변환 (게시자의 통화)에 따라 지정 된 SKU에 대 한 사용량 단위에 대 한 예상 지불입니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                        |
| **예상 가격 (PC)**                             | 지정 된 SKU의 사용 단위에 대 한 예상 가격은 날짜 사용에 대 한 외부 exchange 변환을 기반으로 하 여 계산 됩니다 (게시자의 통화). 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                                             |
| **새 고객**                                  | 이 값을 사용 하 여 처음으로 하나 이상의 Marketplace 제품을 사용 하거나 구매할 수 있는 새 고객 인지 알 수 있습니다.  "날짜를 구한 날짜"와 동일한 달력 월 인 경우 값은 "Yes"가 됩니다.  고객이 보고 한 월 이전에 제품을 구매한 경우이 값은 "아니요"입니다.                                                       |
| **미리 보기 SKU**                                  | 이 값은 SKU를 "preview"로 태그가 지정 되었는지 여부를 알려 주는 데 사용 됩니다. SKU에 해당 하는 태그가 지정 되어 있고이 이미지를 배포 하 고 사용할 수 있는 Azure 구독만이 값을 "예"로 지정할 수 있습니다.  SKU가 "preview"로 식별 되지 않은 경우이 값은 "아니요"입니다.                                          |
| **판촉 연락처 옵트인**                   | 이 값은 고객이 게시자의 판촉 담당자를 사전에 옵트인 했는지 여부를 알려 주는 데 사용 됩니다. 현재는 고객에게 해당 옵션이 제공되지 않으므로 보드 전체에 “아니요”가 표시됩니다. 이 기능이 배포되면 그에 따라 업데이트를 시작할 예정입니다.                                          |
| **Marketplace 라이선스 유형**                         | Marketplace 제품의 청구 방법입니다.                             |
| **Marketplace 라이선스 유형: Azure를 통해 청구**   | Microsoft가 이 Marketplace 제품의 대리인으로 게시자 대신 고객에게 대금을 청구합니다. (종량제 신용 카드 또는 기업 송장)       |
| **Marketplace 라이선스 유형: 사용자 라이선스를 가져옵니다.** | VM을 배포 하려면 고객이 제공한 라이선스 키가 필요 합니다. Microsoft는 이 Marketplace 제품 사용에 대해 고객에게 대금을 청구하지 않습니다. |
| **Marketplace 라이선스 유형: 무료**                   | Marketplace 제품이 모든 사용자에게 무료로 제공되도록 구성됩니다. Microsoft는 이 Marketplace 제품 사용에 대해 고객에게 대금을 청구하지 않습니다.    |
| **Marketplace 라이선스 유형: Microsoft as 재판매인**  | Microsoft가 이 Marketplace 제품의 재판매인 역할을 합니다.                               |
| **Marketplace 구독 회사 이름**  | 사용자가 Azure 구독에서 제공 하는 회사 이름입니다.                           |
| **Marketplace 구독 ID**            | 고객이 Marketplace 제품을 구매 하는 데 사용한 Azure 구독과 연결 된 고유 식별자입니다.  이전에는 Azure 구독 GUID 였습니다.  SAAS 주문과 같은 경우에는 구독이 필요 하지 않습니다.           |
| **Microsoft 요금 (CC)**                               | Microsoft의 거래 수수료(단위: 고객 통화)입니다.                              |
| **제안 이름**                                | Marketplace 제품의 이름입니다.                                                             |
| **제품 유형**                                | Microsoft Marketplace 제공 형식입니다.  예를 들어 관리 되는 응용 프로그램, 가상 컴퓨터 라이선스 또는 컨테이너가 될 수 있습니다.                           |
| **주문 취소 날짜**                         | Marketplace 주문이 취소된 날짜입니다.                                                       |
| **주문 ID**                                  | Marketplace 서비스의 고객 주문에 대 한 고유 식별자입니다.  가상 컴퓨터 사용량 기반 제안은 주문에 연결 되지 않습니다.                 |
| **주문 구매 날짜**                       | Marketplace 주문이 만들어진 날짜입니다.                                                        |
| **주문 수량**                       | Azure 구독과 연결 된 동일한 SKU에 대 한 주문 수량입니다.  SAAS 주문의 경우 주문의이 인스턴스와 연결 된 사용자 수를 나타냅니다.                                            |
| **주문 상태**                              | 데이터를 마지막으로 새로 고칠 때의 Marketplace 주문 상태입니다.  예를 들어, 주문이 활성화 되거나 취소 될 수 있습니다.             |
| **주문 상태: 활성**                           | 고객이 주문을 구매 했 고 주문을 취소 하지 않았습니다.   |
| **주문 상태: 취소 됨**                        | 이전에 고객이 주문을 구매 하 고 이후에 주문을 취소 했습니다.        |
| **지급 Amount (PC)**                             | 선호 하는 지급 Currency (PC)에서 사용자에 게 지불 된 금액입니다.                                       |
| **지급 Currency (PC)**                           | 지급에 사용되는 통화입니다.                                                                |
| **지급 날짜**                                    | 지불 요청이 Microsoft에서 선택한 금융 기관으로 전송 된 날짜입니다.                                                                         |
| **지급 상태**                                  | 데이터를 마지막으로 새로 고친 시점의 지급 상태입니다.                                   |
| **지급 상태: 진행 중**               | 거래에 대한 지급이 준비되지 않았습니다. (자세한 내용은 수집 상태를 참조 하세요.)                             |
| **지급 상태: 지불 됨**                          | 거래가 이전 지급 계산에 포함되었습니다. 계산 결과가 양수인 경우 해당 금액이 지급되었으며, 음수인 경우에는 총 지불 금액에서 차감되었습니다. |
| **지급 상태: 예정 된 지급**                   | 거래에 대한 지급이 준비되었으며 사용 가능한 다음 지급 계산에 포함될 예정입니다.                      |
| **가격 (CC)**                                   | 지정된 SKU의 사용 단위 가격(단위: 고객 통화)입니다.                                                                  |
| **공급자 전자 메일**                                 | Microsoft와 최종 고객 간의 관계와 관련 된 공급자의 전자 메일 주소입니다. 고객이 대리점을 통해 엔터프라이즈 인 경우 재판매인이 됩니다.  CSP (클라우드 솔루션 공급자)가 관련 된 경우 CSP가 됩니다.                                                                 |
| **공급자 이름**                                 | Microsoft와 최종 고객 간의 관계에 관련 된 공급자의 이름입니다. 고객이 대리점을 통해 엔터프라이즈 인 경우 재판매인이 됩니다.  CSP (클라우드 솔루션 공급자)가 관련 된 경우 CSP가 됩니다.                                                                 |
| **리소스 URI**                                  | 배포 아티팩트가 배포 된 경로입니다.                                            |
| **SKU**                                           | 게시 하는 동안 정의 된 SKU 이름입니다. 제품에는 많은 Sku가 있을 수 있지만 SKU는 단일 제품에만 연결 될 수 있습니다.              |
| **SKU 청구 유형**                                 | SKU의 청구 방법입니다.                                                                                                             |
| **SKU 청구 유형: 사용자의 라이선스를 가져옵니다.**         | VM 또는 서비스에는 고객이 배포 하기 위해 제공 하는 라이선스 키가 필요 합니다. Microsoft는 이러한 Marketplace 제품 사용에 대해 고객에 게 요금을 청구 하지 않습니다.   |
| **SKU 청구 유형: Microsoft As 재판매인**          | Microsoft는이 SKU에 대 한 대리점입니다.   |
| **SKU 청구 유형: 무료**                           | SKU는 모든 사용자에 게 무료로 구성 됩니다. Microsoft는이 SKU의 사용에 대 한 고객의 요금을 청구 하지 않습니다.                           |
| **SKU 청구 유형: 유료**                           | Microsoft는이 SKU에 대 한 에이전트 이며 사용자를 대신 하 여 고객에 게 요금을 청구 합니다. (종량제 신용 카드 또는 기업 송장)                   |
| **SKU 청구 유형: 평가판**                          | 고객이 평가판을 사용하는 중이며, 제품 사용을 취소하거나 삭제하지 않으면 유료로 전환됩니다.                                         |
| **세금 금액 (CC)**                                  | 고객 통화 (CC)에서 고객의 청구서에 적용 되는 세금입니다.                                                               |
| **트랜잭션 날짜**                                 | 지급 보고에 기록된 거래 날짜입니다.                                                                           |
| **트랜잭션 유형**                               | 보고 중인 거래 유형입니다.  예를 들어 요금 청구, 환불 또는 지급 조정이 있을 수 있습니다.                                                                                        |
| **평가판 종료 날짜**                                 | 이 주문의 평가 기간이 종료되었거나 종료될 예정인 날짜입니다.                                                                           |
| **사용 날짜**                                       | 고객이 SKU를 사용한 날짜입니다.                                                                                                             |
| **사용 수량**                                       | SKU의 보고 된 고객 사용량입니다. VM 이미지의 경우 사용 레코드는 해당 VM 크기 및 SKU에 대해 보고 된 기간의 사용량을 나타냅니다.                                               |
| **사용 참조**                                       | 지급 보고서의 항목과 연결된 특정 SKU를 고객이 사용한 기간(1일 이상)의 식별자입니다.                                               |
| **사용 유형**                                       | 측정 중인 사용에 대한 설명입니다. (정규화 된 사용 또는 원시 사용량)                                                              |
