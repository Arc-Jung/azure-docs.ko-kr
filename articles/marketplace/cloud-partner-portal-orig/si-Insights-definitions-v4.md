<a name="seller-insights-definitions"></a>Seller Insights 정의
=======================

아래 표에는 Seller Insights에서 사용되는 여러 용어의 정의가 나와 있습니다.

|  **용어**                                        |  **정의**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| Azure 라이선스 유형                               | Azure 구매 고객이 체결하는 라이선싱 계약 유형입니다. 채널이라고도 합니다.                                                  |
| Azure 라이선스 유형: 클라우드 솔루션 공급자      | 재판매인 역할을 하는 클라우드 솔루션 공급자를 통해 최종 고객이 Azure 및 Marketplace 제품을 조달받습니다.                 |
| Azure 라이선스 유형: 기업                   | Microsoft와 직접 체결하는 기업계약을 통해 최종 고객이 Azure 및 Marketplace 제품을 조달받습니다.                  |
| Azure 라이선스 유형: 기업(재판매인 있음)  | Microsoft와의 기업계약을 진행해주는 재판매인을 통해 최종 고객이 Azure 및 Marketplace 제품을 조달받습니다.     |
| Azure 라이선스 유형: 종량제                | Microsoft와 직접 체결하는 종량제 계약을 통해 최종 고객이 Azure 및 Marketplace 제품을 조달받습니다.                |
| Azure 구독 GUID                          | 고객이 Marketplace 제품을 구매하는 데 사용한 Azure 구독의 GUID(Globally Unique Identifier)입니다.                     |
| 청구 금액(CC)                               | 고객의 청구 통화(CC)로 고객에게 청구되는 금액입니다.                                                                 |
| 청구 금액(PC)                               | *지급액 통화(PC)* 로 고객에게 청구되는 금액입니다.                                                                      |
| 청구 날짜                                      | 고객의 청구 금액이 계산된 날짜(대개 사용 기간 직후)입니다.                                             |
| 클라우드 인스턴스 이름                              | VM이 배포된 Microsoft Cloud입니다. (Azure Gov - 미국 정부용 Microsoft Cloud 인스턴스, Azure China - 중국 내의 Microsoft Cloud 인스턴스 Azure Germany - 독일 내의 Microsoft Cloud 인스턴스, Azure Global - 기타 모든 국가의 Microsoft Cloud 인스턴스)                                                          |
| 사용자 지정 측정 사용량                               | 사용자 지정 측정 제품에서 사용하는 측정 단위입니다.                                                                                        |
| 고객                                         | Marketplace를 통해 게시된 제품을 획득하거나 이용하거나 기타 방식으로 확인하는 Azure 고객 또는 최종 사용자입니다. 고유 *Azure 구독 GUID*로 고객을 식별할 수 있습니다.                                                                                                                                                                        |
| 고객 구/군/시                                    | 최종 고객이 입력한 구/군/시 이름입니다.                                                                                                  |
| 고객 통신 언어                  | 고객이 통신용으로 선택한 기본 설정 언어입니다.                                                                                    |
| 고객 회사 이름                            | 최종 고객이 입력한 회사 이름입니다.                                                                                               |
| 고객 국가                                 | 최종 고객이 입력한 국가 이름입니다.                                                                                               |
| 고객 통화(CC)                           | 고객이 가격 및 청구용으로 선택한 기본 설정 통화입니다.                                                                              |
| 고객 이메일                                   | 최종 고객이 입력한 이메일 주소입니다.                                                                                              |
| 고객 이름                              | 고객의 이름입니다.                                                                                                              |
| 고객 성                               | 고객의 성입니다.                                                                                                               |
| 고객 지불 유형                            | 고객이 사용하는 지불 유형입니다.                                                                                         |
| 고객 우편 번호                             | 최종 고객이 입력한 우편 번호입니다.                                                                                                |
| 고객 시/도                                   | 최종 고객이 입력한 시/도입니다.                                                                                                      |
| 취득일                                    | 게시된 제품을 Azure 구독으로 최초 구매한 날짜입니다.                                                                  |
| 취소일                                        | 게시된 모든 제품을 Azure 구독에서 취소한 날짜입니다.                                                                     |
| 예상 요금 소계(CC)                   | 지정된 SKU의 사용 단위 수량에 대해 예상되는 요금 소계(단위: 고객 통화)입니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                                                                                                                                                                                           |
| 예상 요금 소계(PC)                   | 지정된 SKU의 사용 단위 수량에 대해 예상되는 요금 소계입니다. 사용량 계산 날짜의 환율(게시자 통화)을 기준으로 합니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                                                                                                                |
| 예상 지급액(PC)                            | 지정된 SKU의 사용 단위 수량에 대해 예상되는 지급액입니다. 사용량 계산 날짜의 환율(게시자 통화)을 기준으로 합니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                                                                                                                |
| 예상 가격(PC)                             | 지정된 SKU의 사용 단위에 대해 예상되는 가격입니다. 사용량 계산 날짜의 환율(게시자 통화)을 기준으로 합니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.                                                                                                                                            |
| 최종 수금 상태                          | 특정 요금에 대한 청구 및 수금 수명 주기의 최신 상태입니다. 상태는 *수금 진행 중*, *수금됨*, *지급됨*, *환불* 또는 *탕감* 중 하나일 수 있습니다.                                                                                                                                                                                      |
| 최종 수금 상태: 수금됨               | 요금이 수금되었습니다.                                                                                                                    |
| 최종 수금 상태: 수금 진행 중  | 고객에게 요금이 아직 수금되지 않았습니다. Microsoft Azure 청구를 통해 고객에게 요금을 청구하거나 수금하는 중입니다.             |
| 최종 수금 상태: 환불                  | 요금이 전액 또는 부분 환불되었습니다.                                                                      |
| 최종 수금 상태: 탕감               | 요금이 회수 불능 금액(대손)으로 간주되어 탕감되었습니다.                                                                                                      |
| 신규 고객                                   | 해당 월에 처음으로 해당하는 *SKU*를 배포한 고객입니다.                                                               |
| 시장                                           | 최종 고객이 입력한 국가 이름입니다.                                                                                               |
| Marketplace 라이선스 유형                         | Marketplace 제품의 청구 방법입니다.                                                                                                 |
| Marketplace 라이선스 유형: Azure를 통해 청구됨   | Microsoft가 이 Marketplace 제품의 대리인으로 게시자 대신 고객에게 대금을 청구합니다. (종량제 신용 카드 또는 기업 송장)       |
| Marketplace 라이선스 유형: BYOL(사용자 라이선스 필요) | VM을 배포하려면 고객이 제공한 라이선스 키가 필요합니다. Microsoft는 이 Marketplace 제품 사용에 대해 고객에게 대금을 청구하지 않습니다. |
| Marketplace 라이선스 유형: 무료                   | Marketplace 제품이 모든 사용자에게 무료로 제공되도록 구성됩니다. Microsoft는 이 Marketplace 제품 사용에 대해 고객에게 대금을 청구하지 않습니다.    |
| Marketplace 라이선스 유형: Microsoft - 재판매인  | Microsoft가 이 Marketplace 제품의 재판매인 역할을 합니다.                                                                                       |
| Marketplace 주문                                | VM의 경우 Marketplace 주문은 고유 Azure 구독에서 단일 *SKU*에 연결된 VM 하나 이상의 배포를 나타냅니다. 단일 Marketplace 주문이 코어 크기가 각기 다른 여러 배포를 나타낼 수 있습니다. 관리형 애플리케이션 및 개발 서비스의 경우 Marketplace 주문은 *Azure 구독 GUID*로 구매하는 단일 *SKU*를 나타냅니다.           |
| Microsoft 수수료(CC)                               | Microsoft의 거래 수수료(단위: 고객 통화)입니다.                                                                                 |
| 정규화된 사용량                                 | 사용량 계산에 포함되는 VM 코어 수를 고려하여 정규화된 시간 단위 사용량입니다. [VM 코어 수] x [원시 사용량(시간)]으로 계산됩니다. “SHARECORE”로 지정된 VM은 [VM 코어 수] 승수로 1/6(0.1666)을 사용합니다.                                                                                                                                       |
| 제품 이름                                       | Marketplace 제품의 이름입니다.                                                                                                           |
| 제품 유형                                       | 솔루션의 유형입니다.                                                                                                                        |
| 제품 유형: 사용자 지정 유료 솔루션                | 고객이 Marketplace에서 사용자 지정 유료 솔루션을 배포했습니다.                                                                     |
| 제품 유형: 관리형 애플리케이션                  | 고객이 관리형 서비스 애플리케이션에 대한 구독을 구매했습니다.                                                                   |
| 제품 유형: 단일 VM                            | 고객이 단일 VM을 선택하여 이미지를 배포했습니다.                                                                                     |
| 제품 유형: 솔루션 템플릿                    | 고객이 솔루션 템플릿을 배포했습니다.                                                                                                |
| 주문 취소 날짜                                | Marketplace 주문이 취소된 날짜입니다.                                                                                                 |
| 주문 수                                      | 해당하는 *Azure 구독 GUID*에 연결된 주문 중 활성 및 취소된 주문 수                               |
| 주문 구매 날짜                              | *Marketplace 주문*이 생성된 날짜입니다. VM의 경우 고객 Azure 구독에 이미지를 배포한 최초 날짜입니다. 최초 배포 이후 동일 Azure 구독에 같은 이미지를 배포하는 작업은 모두 주문 1개로 간주됩니다.                                                                                                       |
| 주문 상태                                     | 데이터를 마지막으로 새로 고칠 때의 Marketplace 주문 상태입니다.                                                                   |
| 주문 상태: 활성                             | 관리형 애플리케이션의 경우 주문이 활성 상태입니다. VM의 경우 고객의 Azure 구독에 이미지가 하나 이상 배포되어 있습니다.   |
| 주문 상태: 취소됨                          | 관리형 애플리케이션의 경우 주문이 취소되었습니다. VM의 경우 고객이 Azure 구독에서 *SKU*의 모든 배포를 삭제했습니다.        |
| OrderID                                          | 주문의 고유한 식별자입니다. 관리형 애플리케이션의 경우 고객이 서비스의 월간 구독을 구매했습니다. VM의 경우 고객이 이미지를 배포했습니다. 최초 배포 이후 동일 Azure 구독에 같은 이미지를 배포하는 작업은 모두 주문 1개로 간주됩니다.                                                                       |
| 지불 유형: 카드                               | 고객이 신용 카드를 사용하여 Marketplace 요금을 지불합니다.                                                                          |
| 지불 유형: 송장                            | 고객이 송장을 통해 Marketplace 요금을 지불합니다.                                                                                 |
| 지급 금액(PC)                               | 게시자에게 지급되는 금액입니다. 게시자의 기본 설정 *PC(지급 통화)* 단위입니다.                                                                            |
| PC(지급 통화)                             | 지급에 사용되는 통화입니다.                                                                                                          |
| 지급 날짜                                      | Microsoft에서 은행으로 지급 요청을 보낸 날짜입니다.                                                                           |
| 지급 상태                                    | 지급 수명 주기에서 거래의 상태를 나타냅니다. *지급됨*, *지급 예정*, *지급 준비 안 됨* 중 하나일 수 있습니다.                        |
| 지급 상태: 지급 준비 안 됨              | 거래에 대한 지급이 준비되지 않았습니다. (자세한 내용은 *최종 수금 상태* 참조)                                                    |
| 지급 상태: 지급됨                          | 거래가 이전 지급 계산에 포함되었습니다. 계산 결과가 양수인 경우 해당 금액이 지급되었으며, 음수인 경우에는 총 지불 금액에서 차감되었습니다. |
| 지급 상태: 지급 예정                   | 거래에 대한 지급이 준비되었으며 사용 가능한 다음 지급 계산에 포함될 예정입니다.                                           |
| 미리 보기 SKU                                      | *SKU*에 "미리 보기" 태그를 지정했습니다. 이 경우 게시자가 허용 목록에 포함한 Azure 구독만 배포 가능하며 이 이미지를 사용할 수 있습니다.                       |
| 가격(CC)                                       | 지정된 SKU의 사용 단위 가격(단위: 고객 통화)입니다.                                                                  |
| 홍보 연락 옵트인                       | 고객이 게시자의 홍보를 위한 연락을 받도록 사전에 옵트인했는지 여부를 나타냅니다. 현재는 고객에게 해당 옵션이 제공되지 않으므로 보드 전체에 “아니요”가 표시됩니다. 이 기능이 배포되면 그에 따라 업데이트를 시작할 예정입니다.                                                                                          |
| PC(게시자 통화)                          | 게시자가 지급용으로 선택한 기본 설정 통화입니다.                                                                                          |
| 원시 사용량                                        | Marketplace 제품의 사용 시간입니다.                                                                                                      |
| 재판매인 이메일                                   | 최종 고객에게 제품을 판매한 재판매인의 이메일 주소입니다.                                                                  |
| 재판매인 이름                                    | 최종 고객을 관리하는 Microsoft 재판매인의 이름입니다.                                                                                |
| 리소스 URI                                     | 개별 VM 또는 개발자 서비스 배포를 위한 고유 식별자입니다.                                                                    |
| SKU                                              | 게시 중에 정의한 *SKU* 이름입니다. 한 제품에 여러 *SKU*가 포함될 수는 있지만 각 *SKU*는 제품 하나에만 연결할 수 있습니다.              |
| SKU 청구 유형                                 | *SKU*의 청구 방법입니다.                                                                                                             |
| SKU 청구 유형: BYOL                           | VM을 배포하려면 고객이 제공한 라이선스 키가 필요합니다. Microsoft는 이 Marketplace 제품 사용에 대해 고객에게 대금을 청구하지 않습니다.   |
| SKU 청구 유형: 무료                           | *SKU*가 모든 사용자에게 무료로 제공되도록 구성됩니다. Microsoft는 이 *SKU* 사용에 대해 고객에게 대금을 청구하지 않습니다.                            |
| SKU 청구 유형: Microsoft - 재판매인          | Microsoft가 이 *SKU*의 재판매인 역할을 합니다.                                                                                                   |
| SKU 청구 유형: 유료                           | Microsoft가 이 *SKU*의 대리인으로 게시자 대신 고객에게 대금을 청구합니다. (종량제 신용 카드 또는 기업 송장)                   |
| SKU 청구 유형: 평가판                          | 고객이 평가판을 사용하는 중이며, 제품 사용을 취소하거나 삭제하지 않으면 유료로 전환됩니다.                                         |
| 세금액(CC)                                  | CC(*고객 통화*) 단위로 고객 청구 금액에 적용되는 세금액입니다.                                                               |
| 트랜잭션 날짜                                 | 지급 보고에 기록된 거래 날짜입니다.                                                                               |
