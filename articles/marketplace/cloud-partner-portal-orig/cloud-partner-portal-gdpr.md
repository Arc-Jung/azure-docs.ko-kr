<a name="cloud-partner-portal-gdpr-compliance"></a>클라우드 파트너 포털 GDPR 규정 준수
====================================

[GDPR(일반 데이터 보호 규정)](https://ec.europa.eu/info/law/law-topic/data-protection/reform_en/)은 EU(유럽 연합)의 데이터 보호 및 프라이버시 법률입니다. GDPR은 회사, 정부 기관, 비영리 단체 및 기타 조직에 대해 상품 및 서비스를 제공하거나 EU 거주자와 연결된 데이터를 수집하고 분석하는 규칙을 수립합니다.

클라우드 파트너 포털에서는 고객 데이터 처리 및 해당 데이터를 보호하는 데 사용되는 보안 조치와 관련된 상세 정보를 제공합니다. 제품 내 환경을 통해 이 정보에 액세스할 수 있습니다.

<a name="discover"></a>검색
--------

Cloud 파트너 포털 사용자 탭의 모든 데이터에 대 한 액세스를 제공 합니다. 여기에서 개인 데이터를 식별할 수 있습니다. 사용자 추가/제거 방법을 알아보려면 [설명서](./cloud-partner-portal-manage-users.md)를 참조하세요.

<a name="manage"></a>관리
------

사용자 탭에서 사용자 계정에 대해 삭제를 눌러 개인 데이터를 관리할 수 있습니다. 각 특정 사용자에 게 할당할 수 있는 **소유자** 및 **기여자** 역할이 있으며 둘 다 할당/복사/업데이트/삭제할 수 있습니다.

<a name="consent"></a>동의
-------

클라우드 파트너 포털에서 동의를 관리할 때는 항상 게시자 계약이 먼저 표시됩니다. 이 계약에 동의해야 Microsoft의 게시자로 등록됩니다. 게시자 계약에 동의하지 않는 게시자의 경우 Microsoft를 통한 솔루션 판매가 제한됩니다.
