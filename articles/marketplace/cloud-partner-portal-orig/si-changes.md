# <a name="seller-insights-release-notes"></a>Seller Insights 릴리스 정보 

(발매일: 2019년 3월 1일)

이 문서에서는 [클라우드 파트너 포털](https://cloudpartner.azure.com/#insights)의 Seller Insights 기능 변경 내용에 대한 정보를 제공합니다.

## <a name="release-highlights-for-march-1-2019"></a>2019년 3월 1일 릴리스 하이라이트

* 요약에 추가된 *고객 추세*
* 요약의 *상위 5개 고객은* 고객이 보유하고 있는 모든 Azure 구독을 반영합니다.
* *정규화 된 사용 동향 & 활성 주문 추세* 요약에 한 눈에 월별 *주문에서* 이동
* *지급 조정 보고서가* 업데이트되었습니다.
* 결제 상위 *5개 고객은* 고객이 보유하고 있는 모든 Azure 구독을 반영합니다.
* 고객 ID로 업데이트된 *사용 보고서*
* 주문에 대한 *고객 의 기간* & 사용량은 고객이 가지고 있는 모든 Azure 구독을 반영합니다.


(릴리스 날짜: 2018년 7월 28일)

## <a name="release-highlights-for-july-28-2018"></a>2018년 7월 28일 릴리스 하이라이트


-   *예상 가격*에서 환율이 적용된 고객 요금을 확인할 수 있습니다.
-   *예상 지급액*에서는 지급 가능액을 미리 확인할 수 있습니다.
-  *사용량 기준 식별자*는 고객 사용량과 주문 및 지급액의 정확한 데이터를 제공합니다.
-   *일 단위 사용량*에서는 고객 사용량을 더 세분화하여 자세히 확인할 수 있습니다.


### <a name="changes-to-data-structure-and-taxonomy"></a>데이터 구조 및 분류 변경 내용

아래 표에는 이번 릴리스에서 추가되거나 크게 변경된 메트릭이 나와 있습니다. 

| **새 용어**                   |    **정의**                                                             |
|--------------------------------|  ---------------------------------------------------------------------------- |
| 가격(CC)                     | 지정된 SKU의 사용 단위 가격(고객 통화)입니다.       |
| 예상 요금 소계(CC) | 지정된 SKU(고객통화)의 사용량 단위 수량에 대한 예상 연장 요금입니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.   |
| PC(게시자 통화)        | 게시자가 지급용으로 선택한 기본 설정 통화입니다.                               |
| 예상 가격(PC)           | 사용일의 외환 변환을 기준으로 지정된 SKU의 사용 단위에 대한 예상 가격은 게시자의 통화로 계산됩니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.   |
| 예상 요금 소계(PC) | 지정된 SKU의 사용 단위수에 대한 예상 확장 요금은 사용 일자에 대한 외환 변환을 기준으로 계산됩니다(게시자의 통화). 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다. |
| 예상 지급액(PC)          | 사용이 계산된 날짜(게시자의 통화)에 따른 환율 변환을 기준으로 지정된 SKU의 사용량 단위수에 대한 예상 지급액입니다. 반올림 또는 버림 오류로 인해 이 값은 정확하지 않을 수도 있습니다.   |
| 사용량 기준                | 지급 보고서의 항목과 연결된 특정 SKU를 고객이 사용한 기간(1일 이상)의 식별자입니다. |
|  |  |
