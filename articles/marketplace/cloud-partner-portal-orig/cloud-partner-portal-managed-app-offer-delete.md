<a name="delete-an-offersku-from-azure-marketplace"></a>Azure Marketplace에서 제품/SKU 삭제 
==========================================

다양한 이유로 Marketplace에서 제품을 제거하도록 결정할 수 있습니다. 제품을 제거하는 경우 신규 고객이 제품을 더 이상 구매하거나 배포할 수는 없지만 기존 고객에게는 아무런 영향이 없습니다.
제품 종료는 귀하와 기존 고객 간의 서비스 및/또는 사용권 계약이 종료되는 프로세스입니다. 제품 제거 및 종료 관련 지침과 정책은 [Microsoft Marketplace 게시자 계약](http://go.microsoft.com/fwlink/?LinkID=699560)(7항 참조) 및
7) [참가 정책](https://azure.microsoft.com/support/legal/marketplace/participation-policies/)(6.2항 참조)을 따릅니다. 이 문서에서는 지원되는 각 삭제 시나리오와 수행할 수 있는 단계에 대해 설명합니다.

<a name="delete-a-live-sku-from-azure-marketplace"></a>Azure Marketplace에서 라이브 SKU 삭제 
----------------------------------------

다음 단계를 수행하여 Azure Marketplace에서 라이브 SKU를 삭제할 수 있습니다.

1.  [클라우드 파트너 포털](http://cloudpartner.azure.com)에 로그인합니다.
2.  모든 제품 탭에서 제품을 선택합니다.
3.  화면 왼쪽의 창에서 SKU 탭을 선택합니다.
4.  삭제할 SKU를 선택하고 해당 SKU의 삭제 단추를 클릭합니다.
5.  제품을 Azure Marketplace에 다시 게시합니다.

Azure Marketplace에 제품이 라이브되면 SKU가 Azure Marketplace 및 Azure Portal에서 삭제됩니다.

<a name="roll-back-to-a-previous-sku-version"></a>이전 SKU 버전으로 롤백 
----------------------------------

다음 단계를 수행하여 Azure Marketplace에서 라이브 SKU의 현재 버전을 삭제할 수 있습니다. 이 프로세스가 완료되면 SKU가 이전 버전으로 롤백됩니다.

1.  [클라우드 파트너 포털](http://cloudpartner.azure.com)에 로그인합니다.
2.  모든 제품 탭에서 제품을 선택합니다.
3.  화면 왼쪽의 창에서 SKU 탭을 선택합니다.
4.  게시된 패키지 목록에서 최신 패키지 버전을 삭제합니다.
5.  제품을 Azure Marketplace에 다시 게시합니다.

Azure Marketplace에 제품이 다시 라이브되면 나열된 SKU의 현재 버전이 Azure Marketplace 및 Azure Portal에서 삭제됩니다.
SKU는 이전 버전으로 롤백됩니다.

<a name="delete-a-live-offer"></a>라이브 제품 삭제 
-------------------

Azure Marketplace에서 라이브 제품을 제거하려면 다음 단계를 수행하여 지원 팀의 지침을 확인하세요.

1.  이 [링크](https://go.microsoft.com/fwlink/?linkid=844975)를 사용하여 지원 티켓 제기
2.  문제 형식 목록에서 제품 관리를 선택하고 범주 목록에서 프로덕션 환경에서 제품 및/또는 SKU 수정을 선택합니다.
3.  요청을 제출합니다.

지원 팀은 제품 삭제 프로세스를 안내합니다.

SKU/제품을 삭제해도 이미 진행된 SKU/제품의 현재 구매에는 영향을 주지 않습니다. 즉, 기존 SKU/제품은 이전처럼 계속 작동하지만 신규 구매는 진행할 수 없습니다.
