# <a name="notifying-microsoft-regarding-the-publisher-agreement"></a>Microsoft에 게시자 계약에 대 한 알림

지원 티켓을 사용 하 여 microsoft 게시자 계약과 관련 하 여 Microsoft에 법적 통지를 제출 하 여 적절 한 조치 및 추적을 보장 합니다.

## <a name="submit-notice-regarding-the-publisher-agreement"></a>게시자 계약과 관련 된 알림 제출

1. 파트너 센터 지원의 [계약 또는 법적 질문](https://partner.microsoft.com/support/?stage=2&topicid=05a1a389-1256-d441-89c9-a140217de6b9) 섹션으로 이동 합니다.

1. **문제 세부 정보 제공** (필요한 경우 파트너 센터에 먼저 로그인)을 선택 합니다.

    :::image type="content" source="media/notice-submit/recommended-solutions.png" alt-text="' 권장 솔루션 ' 창을 표시 합니다.":::

1. 원하는 경우 200 자까지 **문제 요약** 을 편집 합니다.
1. 여기에서 **파일 끌기 또는 파일 찾아보기** 링크를 사용 하 여 해당 하는 파일을이 지원 요청에 연결 합니다.

    :::image type="content" source="media/notice-submit/create-support-ticket.png" alt-text="지원 티켓을 만들기 위한 창의 필드를 표시 합니다.":::

1. **추가 세부 정보** 를 제공 합니다.

    - **제품 유형** (이 알림이 특정 제품 유형과 관련이 없는 경우 "확인 안 함" 선택)
    - 알림 **세부 정보**
    - 알림의 **시작 날짜** (오늘 이후의 개시 날짜를 포함 하는 경우)
    - **연락처 정보** 및 기본 설정

    이 지원 요청 페이지에는 게시자 ID 및 판매자 ID를 찾는 방법에 대 한 지침도 제공 됩니다.

1. 필요한 모든 정보를 입력 한 후 **제출** 을 선택 하 여 지원 티켓을 만들고 공지를 Microsoft에 보냅니다.
