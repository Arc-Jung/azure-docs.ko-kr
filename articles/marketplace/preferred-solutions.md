# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Azure Marketplace 및 AppSource의 기본 설정 솔루션

Microsoft 기본 설정 솔루션은 특정 수직 시장 또는 솔루션 영역에서 고객의 요구를 해결할 수 있는 품질, 성능 및 기능을 갖추어 선택된 클라우드 애플리케이션입니다. Microsoft 전문가 팀은 입증된 특정 역량과 기능을 갖춘 파트너의 솔루션을 검증합니다. 이러한 솔루션은 클라우드 마켓플레이스 스토어프론트, Azure 마켓플레이스 및 AppSource뿐만 아니라 Azure 포털에도 제공됩니다. AppSource에서 선호하는 솔루션은 산업 분야별로 검색할 수 있습니다. Azure Marketplace 및 Azure 포털에서 기본 솔루션을 가로 솔루션 범주에서 검색할 수 있습니다.
