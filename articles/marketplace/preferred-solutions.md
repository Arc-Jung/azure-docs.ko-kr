# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Azure Marketplace 및 AppSource의 기본 설정 솔루션

Microsoft 기본 설정 솔루션은 특정 수직 시장 또는 솔루션 영역에서 고객의 요구를 해결할 수 있는 품질, 성능 및 기능을 갖추어 선택된 클라우드 애플리케이션입니다. Microsoft 전문가 팀에서 특정 입증 된 역량 및 기능을 사용 하 여 파트너 솔루션을 확인합니다. Azure portal에서 뿐만 아니라 클라우드 마켓플레이스 상점 인, Azure Marketplace 및 AppSource에 이러한 솔루션 추천 됩니다. AppSource에서 기본 솔루션 버티 컬 산업에서 검색할 수 있습니다. Azure Marketplace 및 Azure portal에서 기본 솔루션 가로 솔루션 범주에서 검색할 수 있습니다.
