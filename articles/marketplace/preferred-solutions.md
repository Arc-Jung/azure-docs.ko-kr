# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Azure Marketplace 및 AppSource의 기본 설정 솔루션

Microsoft 기본 설정 솔루션은 특정 수직 시장 또는 솔루션 영역에서 고객의 요구를 해결할 수 있는 품질, 성능 및 기능을 갖추어 선택된 클라우드 응용 프로그램입니다. Microsoft 산업 및 영업 전문가 팀은 Microsoft 파트너 네트워크 [골드 멤버십](https://partner.microsoft.com/en-us/membership/core-benefits) 파트너의 솔루션을 선택합니다. 이러한 솔루션은 Microsoft의 클라우드 마켓플레이스 상점인 Azure Marketplace 및 AppSource에 제공됩니다. Azure Marketplace에서는 솔루션 영역 범주 내에서 클릭할 수 있지만, AppSource에서는 수직 시장별로 기본 설정 솔루션을 검색할 수 있습니다. 

Microsoft는 해당 솔루션 중 하나 이상이 기본 설정 솔루션으로 선택되면 솔루션 공급자에게 연락합니다.
