# <a name="how-to-configure-your-consulting-service-offer-listing-details"></a>컨설팅 서비스 제품 목록 세부 정보를 구성 하는 방법

파트너 센터의 **제품 목록** 페이지에 제공 하는 정보는 Microsoft 상업적 marketplace online 스토어에 표시 됩니다. 여기에는 제품 이름, 설명, 미디어 및 기타 마케팅 자산이 포함 됩니다.

> [!NOTE]
> 영어 이외의 언어로 서비스를 제공 하는 경우 제품 목록에 해당 언어를 사용할 수 있지만 설명은 "이 서비스는 &lt; 제품 콘텐츠>의 언어로 제공 됩니다." 라는 영어로 시작 하거나 끝나야 합니다. 제품 목록 세부 정보에 사용 된 것과 다른 언어로 지원 문서를 제공할 수도 있습니다.

**제공 목록** 페이지에서 아래에 설명 된 정보를 제공 합니다. 컨설팅 서비스 제품에 대 한 목록 세부 정보에 대 한 자세한 내용을 보려면 [제안 목록 세부 정보](./plan-consulting-service-offer.md#offer-listing-details)를 검토 하세요.

## <a name="offer-details"></a>솔루션 세부 정보

1.  **이름** 상자는 이전에 **새 제품** 대화 상자에서 입력 한 이름으로 미리 채워져 있지만 언제 든 지 변경할 수 있습니다. 이 이름은 온라인 상점에서 제품의 제목으로 표시 됩니다.

    > [!IMPORTANT]
    > 제품 이름은 *이름: 기간 + 형식* 으로 지정 해야 합니다. 자세한 내용은 [제품 목록 세부 정보](./plan-consulting-service-offer.md#offer-listing-details)를 참조 하세요.

2. **검색 결과 요약** 상자에서 200 자 이하로 제품의 목적 또는 목표를 설명 합니다.
3. **설명** 필드에서 컨설팅 서비스 제품을 설명 합니다. HTML 태그를 사용 하 여 설명 서식을 지정할 수 있습니다. HTML 태그와 공백을 포함 하 여이 상자에 최대 2000 자의 텍스트를 입력할 수 있습니다. HTML 서식 지정에 대 한 자세한 내용은 [제품 설명에서 지원 되는 html 태그](./supported-html-tags.md)를 참조 하세요.
4. **검색 키워드** 를 3 개까지 입력할 수 있는 옵션이 있습니다. 이러한 키워드는 고객이 온라인 스토어에서 제품을 찾는 데 도움이 됩니다. 제품 이름 및 설명을 포함할 필요가 없습니다.
5. **기간** 드롭다운 목록에서 컨설팅 서비스의 예상 시간을 입력 합니다. 선택한 기간은 제품 이름에서 언급 한 기간과 일치 해야 합니다.

## <a name="contact-information"></a>연락처 정보

회사의 두 사람에 대 한 이름, 전자 메일 주소 및 전화 번호를 입력 합니다 (둘 중 하나를 사용할 수 있음). Microsoft는이 정보를 사용 하 여 귀하의 제품에 대 한 정보와 통신 합니다. 이 정보는 고객에 게 표시 되지 않지만, CSP (클라우드 솔루션 공급자) 파트너에 게 제공 될 수 있습니다.

## <a name="supporting-documents"></a>지원 문서

제품 목록을 지원 하려면 하나 이상의 PDF 문서를 추가 합니다. 각 파일을 업로드 한 후 문서 이름을 제공 합니다. 이 이름은 제품 목록에 표시 됩니다.

## <a name="marketplace-media"></a>마켓플레이스 미디어

> [!NOTE]
> 파일을 업로드하는 동안 문제가 발생한 경우 로컬 네트워크가 파트너 센터에서 사용하는 `https://upload.xboxlive.com` 서비스를 차단하지 않는지 확인하세요.

### <a name="add-logos"></a>로고 추가

**로고** 에서 216 x 216 및 350 x 350 픽셀 사이의 PNG 형식으로 **많은** 로고를 업로드 합니다. 파트너 센터는 **작은** (48 x 48) 로고를 자동으로 만듭니다 .이 로고는 나중에 바꿀 수 있습니다.

두 로고 크기는 온라인 상점에서 다른 위치에 사용 됩니다.

* Azure Marketplace 또는 AppSource의 제품 목록 페이지에 **많은** 로고가 표시 됩니다.
* **작은** 로고는 Azure Marketplace 검색 결과 또는 appsource 기본 페이지 및 검색 결과 페이지에 표시 됩니다.

### <a name="add-screenshots"></a>스크린샷 추가

제품을 설명 하는 이미지를 하나 이상, 최대 5 개까지 추가 합니다. 모든 이미지는 1280 x 720 픽셀 크기 및의 크기 여야 합니다. PNG 형식입니다.

1. **스크린샷** 에서을 끌어 놓습니다. **스크린샷** 상자에 PNG 파일을 놓습니다.
2.  **이미지 캡션 추가** 옆의 **편집** 아이콘을 선택 합니다.
3.  표시 되는 대화 상자에서 캡션을 입력 합니다.
4.  1 ~ 3 단계를 반복 하 여 추가 스크린샷 추가

### <a name="add-videos-optional"></a>비디오 추가 (선택 사항)

제품을 설명 하는 YouTube 또는 Vimeo 비디오에 대 한 링크를 추가할 수 있습니다. 이러한 비디오는 고객과 함께 고객에 게 표시 됩니다. 비디오의 축소판 이미지를 입력 해야 합니다. 크기는 1280 x 720 픽셀 크기와입니다. PNG 형식입니다. 제품 당 최대 4 개의 비디오를 추가 합니다.

1.  비디오 **에서** **비디오 추가** 링크를 선택 합니다.
2.  표시 되는 상자에 비디오의 이름과 링크를 입력 합니다.
3.  을 끌어서 놓습니다. 회색 축소판 상자에 PNG 파일 (1280 x 720 픽셀)을 표시 합니다.
4.  다른 비디오를 추가 하려면 1 ~ 3 단계를 반복 합니다.
