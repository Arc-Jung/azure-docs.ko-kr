# <a name="orders-dashboard-in-commercial-marketplace-analytics"></a>상용 marketplace 분석의 주문 대시보드

이 문서에서는 파트너 센터의 **주문 대시보드에** 대 한 정보를 제공 합니다. 이 대시보드는 주문에 대 한 정보를 그래픽 및 다운로드 가능한 형식으로 표시 합니다.

파트너 센터 분석 도구의 **주문 대시보드에** 액세스 하려면 상업적 Marketplace에서 **[분석 대시보드](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** 를 엽니다.

>[!NOTE]
> 분석 용어에 대 한 자세한 정의는 [상업적 marketplace 분석에 대 한 질문과 용어](./faq-terminology.md)를 참조 하세요.

## <a name="orders-dashboard"></a>주문 대시보드

**분석** 메뉴의 **주문 대시보드에** 는 모든 SaaS 제품에 대 한 현재 주문이 표시 됩니다. 다음 항목에 대 한 그래픽 표현을 볼 수 있습니다.

- [주문 요약](#order-summary)
- [지리 별 주문](#orders-by-geography)
- [제품 별 주문](#orders-by-offers)
- [사이트별 주문 추세 및 사용자 단위](#orders-trend-per-site-versus-per-seat)
- [Sku 별 주문](#orders-by-skus)
- [주문 및 좌석 추세](#orders-and-seats-trend)
- [주문 정보 테이블](#order-details-table)

> [!NOTE]
> 분석 보고서가 Cloud 파트너 포털 (CPP)에 표시 되는 방식과 파트너 센터의 새로운 상업용 marketplace 프로그램 간에 차이가 있습니다. 특정 한 방법 중 하나는 CPP의 **판매자 통찰력** 에는 사용 현황 기반 제공 및 비 사용량 기반 제품에 대 한 데이터를 표시 하는 **주문 & 사용** 탭이 있다는 것입니다. 파트너 센터의 **주문** 페이지에는 SaaS 제품에 대 한 별도의 탭이 있습니다.

## <a name="order-dashboard-details"></a>주문 대시보드 세부 정보

이 섹션에서는 분석 보고서에 대해 자세히 설명 합니다.

### <a name="order-summary"></a>주문 요약

주문 요약 섹션에는 구매한 주문 (취소 된 주문 제외), 취소 된 주문 및 좌석 수가 표시 됩니다.

총 주문 옆의 백분율 값은 선택한 날짜 범위의 증가를 나타냅니다.

![파트너 센터 분석 주문 요약](./media/order-summary.png)

- 위쪽을 가리키는 녹색 삼각형은 긍정 급수 추세를 나타냅니다.
- 아래쪽을 가리키는 빨간색 삼각형은 이전 월을 기준으로 하는 음의 증가 추세를 나타냅니다.
- 성장 추세는 마이크로 막대 그래프로 표시 됩니다. 차트의 열 위로 마우스를 가져가면 각 월의 값을 표시할 수 있습니다.
- 취소 된 주문은 이전에 구매한 후 선택한 날짜 범위 동안 취소 된 주문 수입니다.
- 좌석은 선택한 날짜 범위 동안 생성 된 사용자 수입니다.

### <a name="orders-by-geography"></a>지리 별 주문

**지리 별 주문** 열 지도는 전 세계 지도의 주문 수를 표시 하 고 고객 국가를 기준으로 매핑된 사용자를 표시 합니다. 이 열 지도는 **[geography 열 지도로 고객과](./customer-dashboard.md#customer-by-geography)** 동일 하 게 작동 합니다.

![지리 별 주문 분석 파트너 센터](./media/orders-by-geography.png)

### <a name="orders-by-offers"></a>제품 별 주문

**제공 된 주문 별 주문** 도넛형 그래프는 제품 이름에 따라 주문 (취소 된 주문 포함)을 구성 합니다.

- 최상위 제품이 그래프에 표시 되 고 나머지 제안은 ' Rest a l l '로 그룹화 됩니다.
- 범례에서 특정 제품을 선택 하 여 그래프에 제공 된 항목만 표시할 수 있습니다.
- 그래프에서 조각을 마우스로 가리키면 모든 제품의 총 주문 수와 비교한 주문의 수와 백분율이 표시 됩니다.
- **제공 된 주문 별 추세** 는 월 단위 급수 추세를 표시 합니다. 월 열은 제품 이름별 주문 수를 나타냅니다. 꺾은선형 차트는 z 축에 그려진 증가율 추세를 표시 합니다.
- 차트 위쪽의 슬라이더를 사용 하 여 x 축을 따라 오른쪽 및 왼쪽으로 스크롤하고 특정 데이터 요소에 초점을 맞출 수 있습니다.
- 범례에서 특정 항목을 선택 하 여 추세 차트를 표시할 수 있습니다.
- **취소 된 주문의**추세와 데이터를 표시 하도록 선택할 수도 있습니다. 그래프는 **주문 별 주문** 그래프와 동일한 방식으로 작동 합니다.

### <a name="orders-trend-per-site-versus-per-seat"></a>사이트별 주문 추세 및 사용자 단위

사이트별 **및 사용자 단위 도넛형 차트** 는 사이트별 saas 및 고객에 의해 구매한 saas 주문에 대 한 분석을 나타냅니다 .이 차트에는 취소 된 주문이 포함 됩니다. 세로 막대형 차트는 사이트별 SaaS 및 고객에 의해 구매한 SaaS 주문에 대 한 추세를 나타냅니다 .이 차트에는 취소 된 주문이 포함 됩니다.

### <a name="orders-by-skus"></a>Sku 별 주문

**Sku 별 주문** 차트는 모든 제품에 대 한 SKU (재고 보존 유닛) 수준에서 주문의 추세 (취소 된 주문을 포함)를 나타냅니다. 도넛형 차트는 상위 5 개 SKU 주문의 분석을 나타내고 세로 막대형 차트는 상위 5 개 Sku에 대 한 주문의 추세를 나타냅니다.

### <a name="orders-and-seats-trend"></a>주문 및 좌석 추세

**주문 및 좌석 추세** 차트는 가장 많은 수의 주문에 대 한 상위 50 제안을 제공 합니다. 이러한 값은 리더 보드에 표시 되 고 가장 높은 순서 개수와 순서 백분율로 정렬 됩니다.

- **Sku 별 주문**: 차트에서 상위 5 개 sku의 주문 수에 대 한 분석을 볼 수 있는 제품을 선택 합니다.
- **Sku 별**사용자: 상위 5 개 sku에 대 한 사용자의 월 추세입니다. 선택 하는 제품이 사용자 단위 제품이 아닌 경우이 영역 차트에 데이터가 표시 되지 않습니다.

### <a name="canceled-orders-by-offers"></a>제안에 의해 취소 된 주문

**제공 된 주문 별 취소 된 주문** 원형 차트는 해당 제품 이름에 따라 취소 된 모든 주문을 구성 합니다. 맨 위에 있는 제품이 그래프에 표시 되 고 나머지 제품은 "나머지 모두"로 그룹화 됩니다. 범례에서 그래프에 표시할 특정 제품을 선택할 수 있습니다.

- 그래프에서 조각을 마우스로 가리키면 모든 제품의 총 주문 수와 비교 하 여 선택한 제안의 주문 수와 백분율이 표시 됩니다.
- 세로 막대형 차트는 월 단위 추세를 표시 합니다. 열은 제안 이름별로 취소 된 주문의 수를 나타냅니다. 차트 맨 위에 있는 슬라이더를 사용 하 여 x 축을 따라 오른쪽 및 왼쪽으로 스크롤하고 특정 데이터 요소에 초점을 맞출 수 있습니다. 범례에서 특정 항목을 선택 하 여 추세 차트를 표시할 수 있습니다.

### <a name="order-details-table"></a>주문 정보 테이블

주문 세부 정보 테이블에는 획득 날짜별로 정렬 된 1000 상위 주문의 번호가 매겨진 목록이 표시 됩니다.

- 표의 각 열을 정렬할 수 있습니다.
- 레코드 수가 1000 보다 작은 경우 데이터를 TSV 파일로 추출할 수 있습니다.
- 1000를 초과 하는 레코드를 기록 하는 경우 내보낸 데이터는 다음 30 일 동안 비동기적으로 다운로드 페이지에 배치 됩니다.
- **Order details 테이블** 에 필터를 적용 하 여 관심 있는 데이터만 표시할 수 있습니다. 데이터는 Country, Azure 라이선스 유형, Marketplace 라이선스 유형, 제품 유형, 주문 상태, 무료 내역, Marketplace 구독 ID, 고객 ID 및 회사 이름을 기준으로 필터링 할 수 있습니다.

#### <a name="orders-page-filters"></a>주문 페이지 필터

이러한 필터는 페이지 수준에서 적용 됩니다.

여러 필터를 선택 하 여 보려는 조건에 대 한 차트와 **자세한 주문 데이터** 그리드/내보내기에 표시 하려는 데이터를 렌더링할 수 있습니다. 필터는 주문 페이지의 오른쪽 위 모서리에서 선택한 데이터 범위에 대해 추출 된 데이터에 적용 됩니다.

- 제안 유형 및 제안 이름은 선택한 날짜 범위 동안 주문 하는 제품에 대해서만 나열 됩니다. 목록에서 선택한 제품 유형에 대 한 제안 이름이 목록에 표시 됩니다.
- 적용 된 필터는 선택한 각 필터에 대 한 각 선택 항목 내의 총 메트릭을 표시 합니다. 기본 선택 항목을 선택 하면 적용 된 필터는 표시 되지 않습니다.
- 드롭다운 목록 중 하나에 대해 **모두** 를 선택 하면 선택한 페이지의 모든 메트릭이 집계 됩니다. 예를 들어 제품 유형 필터 옵션의 "모두"는 모든 제품 유형을 선택 했음을 의미 합니다. 드롭다운 목록에 대 한 기본 선택입니다. 적용 된 필터 **모든** 항목을 선택 하면 표시 되지 않습니다 .를 선택 합니다.
- **다중 값 선택**: 드롭다운 목록에서 선택한 모든 항목에 대해 페이지의 모든 메트릭이 집계 됩니다. 여러 항목을 선택 하는 경우 적용 된 필터에는 선택한 모든 항목의 수가 표시 됩니다. 참조는 아래 이미지를 참조 하세요.

    ![필터에 여러 값이 적용 된 파트너 센터의 주문 분석](./media/filters-applied.png)

- **단일 값 선택**: 하나의 값을 선택 하는 경우 적용 된 필터에는 선택한 필터 수가 표시 됩니다. 참조는 아래 이미지를 참조 하세요.

     ![필터에 단일 값이 적용 된 파트너 센터의 주문 분석](./media/filters-applied-single.png)
