# <a name="orders-dashboard-in-commercial-marketplace-analytics"></a>상업용 마켓플레이스 분석의 주문 대시보드

이 문서에서는 파트너 센터의 **주문 대시보드**에 대한 정보를 제공합니다. 이 대시보드는 주문에 대한 정보를 그래픽 및 다운로드 가능한 형식으로 표시합니다.

파트너 센터 분석 도구에서 **주문 대시보드**에 액세스하려면 상업용 Marketplace에서 **[대시보드 분석](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** 을 엽니다.

>[!NOTE]
> 분석 용어에 대한 자세한 정의는 [상업용 Marketplace 분석에 대한 자주 묻는 질문과 용어](./faq-terminology.md)를 참조하세요.

## <a name="orders-dashboard"></a>주문 대시보드

**분석** 메뉴의 **주문 대시보드**에는 모든 SaaS 제품에 대한 현재 주문이 표시됩니다. 다음 항목에 대한 그래픽 표현을 볼 수 있습니다.

- [주문 요약](#order-summary)
- [지리별 주문](#orders-by-geography)
- [제품별 주문](#orders-by-offers)
- [사이트 단위 대 사용자 단위 주문 추세](#orders-trend-per-site-versus-per-seat)
- [요금제 별 주문](#orders-by-plans)
- [주문 및 사용자 추세](#orders-and-seats-trend)
- [주문 세부 정보 표](#order-details-table)

파트너 센터에서 주문 생성 및 보고 사이의 최대 대기 시간은 48 시간입니다.

## <a name="order-dashboard-details"></a>주문 대시보드 세부 정보

이 섹션에서는 분석 보고서에 대해 자세히 설명합니다.

### <a name="order-summary"></a>주문 요약

주문 요약 섹션에는 구매한 모든 주문(취소된 주문 제외), 취소된 주문 및 사용자 수가 표시됩니다.

총 주문 옆의 백분율 값은 선택한 날짜 범위의 증가량을 나타냅니다.

![파트너 센터 분석 주문 요약](./media/order-summary.png)

- 위쪽을 가리키는 녹색 삼각형은 양의 증가 추세를 나타냅니다.
- 아래쪽을 가리키는 빨간색 삼각형은 이전 월을 기준으로 하는 음의 증가 추세를 나타냅니다.
- 증가 추세는 마이크로 막대 그래프로 표시됩니다. 차트의 열 위로 마우스를 가져가면 각 월의 값을 표시할 수 있습니다.
- 취소된 주문은 이전에 구매한 후 선택한 날짜 범위 동안 취소된 주문 수입니다.
- 사용자는 선택한 날짜 범위 동안 생성된 사용자 수입니다.

### <a name="orders-by-geography"></a>지리별 주문

**지리 별 주문** 열 맵은 전 세계 지도의 주문 수를 표시 하 고 고객 국가/지역에 따라 매핑된 사용자를 표시 합니다. 이 열 지도는 **[지리별 고객 열 지도](./customer-dashboard.md#customer-by-geography)** 와 동일하게 작동합니다.

![파트너 센터 분석 지리별 주문](./media/orders-by-geography.png)

### <a name="orders-by-offers"></a>제품별 주문

**제품별 주문** 도넛형 그래프는 제품 이름에 따라 주문(취소된 주문 포함)을 구성합니다.

- 최상위 제품이 그래프에 표시되고 나머지 제품은 'Rest All'로 그룹화됩니다.
- 범례에서 특정 제품을 선택하여 그래프에 해당 제품만 표시할 수 있습니다.
- 그래프에서 조각을 마우스로 가리키면 모든 제품의 총 주문 수와 비교한 주문의 수와 백분율이 표시 됩니다.
- **제품별 주문 추세**는 월별 증가 추세를 표시합니다. 월 열은 제품 이름별 주문 수를 나타냅니다. 꺾은선형 차트는 z축에 그려진 증가율 추세를 표시합니다.
- 차트 위쪽의 슬라이더를 사용하여 x축을 따라 좌우로 스크롤하고 특정 데이터 요소에 초점을 맞출 수 있습니다.
- 범례에서 특정 항목을 선택하여 추세 차트를 표시할 수 있습니다.
- **취소된 주문**에 대한 추세와 데이터를 표시하도록 선택할 수도 있습니다. 그래프는 **제품별 주문** 그래프와 동일한 방식으로 작동합니다.

### <a name="orders-trend-per-site-versus-per-seat"></a>사이트 단위 대 사용자 단위 주문 추세

**사이트별** 및 사용자 단위 도넛형 차트는 고객이 구매한 saas 주문 및 사이트별 saas의 분석을 나타냅니다 .이 차트에는 취소 된 주문이 포함 됩니다. 세로 막대형 차트는 고객이 구매한 사이트별 SaaS 및 사용자별 SaaS 주문에 대한 추세를 나타냅니다(취소된 주문 포함).

### <a name="orders-by-plans"></a>요금제 별 주문

**계획 별 주문** 차트는 취소 된 주문을 포함 하 여 모든 제품에 대 한 계획 수준에서 주문의 추세를 나타냅니다. 도넛형 차트는 상위 5 개의 계획 주문에 대 한 분석을 나타내고 세로 막대형 차트는 상위 5 개 요금제에 대 한 주문의 추세를 나타냅니다.

### <a name="orders-and-seats-trend"></a>주문 및 사용자 추세

**주문 및 사용자 추세** 차트는 가장 많은 수의 주문에 대한 상위 50개 제품을 나타냅니다. 이러한 값은 리더 보드에 표시 되 고 가장 높은 순서 개수와 순서 백분율로 정렬 됩니다.

- **요금제 별 주문** – 차트에서 상위 5 개 요금제의 주문 수에 대 한 분석 결과를 볼 수 있는 제품을 선택 합니다.
- **요금제 별** 사용자-상위 5 개 요금제에 대 한 사용자의 월간 추세입니다. 선택 하는 제품이 사용자 단위 제품이 아닌 경우 여기에 데이터가 표시 되지 않습니다.

### <a name="canceled-orders-by-offers"></a>제품별 취소된 주문

**제품별 취소된 주문** 원형 차트는 해당 제품 이름에 따라 취소된 모든 주문을 구성합니다. 최상위 제품이 그래프에 표시되고 나머지 제품은 "Rest All"로 그룹화됩니다. 범례에서 특정 제품을 선택하여 그래프에 표시할 수 있습니다.

- 그래프에서 조각을 마우스로 가리키면 선택한 제품의 주문 수와 백분율이 모든 제품의 총 주문 수와 비교 하 여 표시 됩니다.
- 세로 막대형 차트는 월 단위 추세를 표시합니다. 열은 제품 이름별로 취소된 주문의 수를 나타냅니다. 차트 위쪽의 슬라이더를 사용하여 x축을 따라 좌우로 스크롤하고 특정 데이터 요소에 초점을 맞출 수 있습니다. 범례에서 특정 항목을 선택하여 추세 차트를 표시할 수 있습니다.

### <a name="order-details-table"></a>주문 세부 정보 표

주문 세부 정보 표에는 획득 날짜별로 정렬된 1000개의 상위 주문의 번호가 매겨진 목록이 표시됩니다.

- 그리드의 각 열을 정렬할 수 있습니다.
- 레코드 수가 1000보다 작은 경우 데이터를 TSV 파일로 추출할 수 있습니다.
- 레코드 수가 1000을 초과하는 경우 내보낸 데이터는 다음 30일 동안 비동기적으로 다운로드 페이지에 배치됩니다.
- **주문 정보 테이블** 에 필터를 적용 하 여 관심 있는 데이터만 표시 합니다. 국가/지역, Azure 라이선스 유형, 상업적 마켓플레이스 라이선스 유형, 제안 유형, 주문 상태, 무료 내역, 상업적 marketplace 구독 ID, 고객 ID 및 회사 이름으로 필터링 합니다.
- SaaS는 Azure Marketplace 또는 AppSource를 통해 구매한 제품을 제공 하기 때문에 Azure 구독이 필요 하지 않습니다. Marketplace 구독 ID는 **자세한 주문 데이터** 섹션에 00000000-0000-0000-0000-000000000000으로 표시 됩니다.

#### <a name="orders-page-filters"></a>주문 페이지 필터

이러한 필터는 페이지 수준에서 적용됩니다.

여러 필터를 선택하여 보려는 조건 및 원하는 데이터에 대한 차트를 **자세한 주문 데이터** 그리드/내보내기에 렌더링하여 표시할 수 있습니다. 필터는 주문 페이지의 오른쪽 위 모서리에서 선택한 데이터 범위에 대해 추출 된 데이터에 적용 됩니다.

- 제품 유형 및 제품 이름은 선택한 날짜 범위 동안 주문하는 제품에 대해서만 나열됩니다. 목록에서 선택한 제품 유형에 대한 제품 이름이 목록에 표시됩니다.
- 적용된 필터는 선택한 각 필터에 대한 각 선택 항목 내의 총 메트릭을 표시합니다. 기본 선택 영역을 선택하면 적용된 필터는 표시되지 않습니다.
- 드롭다운 목록 중 하나에 대해 **모두**를 선택하면 선택한 페이지의 모든 메트릭이 집계됩니다. 다음은 그 예입니다.  제품 유형 필터 옵션에서 "모두"는 모든 제품 유형을 선택했음을 의미합니다. 이는 드롭다운 목록에 대한 기본 선택입니다. **모두**를 선택하면 적용된 필터는 아무것도 표시하지 않습니다.
- **여러 값 선택**: 페이지의 모든 메트릭은 드롭다운 목록에서 선택한 모든 항목에 대해 집계됩니다. 여러 항목을 선택하는 경우 적용된 필터에는 선택한 모든 항목의 수가 표시됩니다. 참고로 아래 이미지를 참조하세요.

    ![필터에 여러 값이 적용된 파트너 센터 분석 주문](./media/filters-applied.png)

- **단일 값 선택**: 하나의 값을 선택하는 경우 적용된 필터에는 선택한 하나의 필터 수가 표시됩니다. 참고로 아래 이미지를 참조하세요.

     ![필터에 단일 값이 적용된 파트너 센터 분석 주문](./media/filters-applied-single.png)
