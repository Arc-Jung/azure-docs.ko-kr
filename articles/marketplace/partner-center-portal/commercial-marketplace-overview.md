# <a name="overview-of-the-commercial-marketplace-portal-in-partner-center"></a>파트너 센터에서 상용 Marketplace 포털 개요

Microsoft Partner Center에서 상용 Marketplace 포털을 시작 합니다. 합니다 [상용 Marketplace 포털](https://partner.microsoft.com/dashboard/commercial-marketplace/) 를 쉽게 제품 및 서비스를 수익 화, 고객에 게 도달 하 고 비즈니스를 성장 시키세요.

이 설명서에서는 다룰 만들고 파트너 센터 계정에 사용 하 여 기본 서비스 (SaaS) 제공에 나열 될 수 있습니다 하는 새 소프트웨어를 만드는 [Azure Marketplace](https://azuremarketplace.microsoft.com/) 하거나 [AppSource](https://appsource.microsoft.com/), 기존 SaaS 제품을 관리 하 고 질문에 도움이 사용자 의견을 수집 하는 리소스를 제공 합니다. 

## <a name="about-microsofts-commercial-marketplace"></a>Microsoft의 상용 Marketplace에 대 한

기업, 정부 및 모든 규모의 조직에서는 비즈니스 솔루션, 혁신적인 기술 및 지원 서비스를 통해 찾을 합니다 [Azure Marketplace](https://azuremarketplace.microsoft.com/) 하 고 [AppSource](https://appsource.microsoft.com/) 웹 저장 [ Azure portal의 제품 내 환경을](https://portal.azure.com), [클라우드 솔루션 공급자](https://partner.microsoft.com/cloud-solution-provider), 및 Microsoft 판매 팀입니다. 

파트너 센터는 Microsoft 파트너의 환경 허브입니다. Microsoft 마케팅 및 marketplace를 활용로 조직을 위해 파트너 센터는는 게시자 및 제품 관리, 테스트 드라이브 데모를 제공 만들고 제품 성능에 분석을 보려면 포털입니다. 

## <a name="migration-from-cloud-partner-portal"></a>클라우드 파트너 포털에서 마이그레이션

소프트웨어 게시자를 사용 하 여 Microsoft의 관계의 모든 측면을 더 잘 통합 되도록 파트너 센터를 사용 하는 진화 하 고 있습니다. 이 진화의 일환으로,의 기능을 마이그레이션하는 것은 [클라우드 파트너 포털](https://cloudpartner.azure.com/) 파트너 센터에 있습니다. 

**현재 파트너 센터 생성 및 관리 소프트웨어-as a Service (SaaS) 제품에 지원합니다.**

기존 마켓플레이스 게시자는 직관적인 인터페이스를 사용 하 여 친숙 한 있지만 향상 된 사용자 환경을 찾습니다. 알림 전자 메일을 통해 포털에서 마이그레이션 게시자를 알려줍니다. 적절 한 위치로 사용자를 원활 하 게 전송 됩니다. 이 파트너 환경 업그레이드 하 여 고객에 게는 영향이 없습니다. 

### <a name="improvements-on-user-experience-and-billing"></a>사용자 환경 및 청구의 기능 개선

파트너 센터로 전환을 사용 하 여 새 기능은 게시자의 경우 사용자 경험에 향상 된 기능 외에도 사용할 수 있습니다. 청구할 수 있는 월간 또는 연간, SaaS 제품에는 사용자 당 사이트별 라이선스 (-요금이) 외에도 (사용자 기반) 라이선스를 지원 합니다. 이 새로운 수익 모델로 비즈니스 소프트웨어에 적합합니다. SaaS 및 사용자 단위 라이선스 미치는 비즈니스 및 산업 솔루션에 대 한 Microsoft 상용 marketplace, AppSource에서 고객은 전체 구매를 허용 하는 새로운 기능으로 맞춥니다. 

### <a name="account-and-user-management-updates"></a>계정 및 사용자 관리 업데이트

파트너 센터에서 계정 및 사용자 관리 하면 쉽게 개인 포털에 액세스 하 고 적절 한 권한을 적용 하려면 관리자가 있습니다. 파트너 센터에서의 역할 기반 액세스 제어 기능 이전 제품 관리 포털에 비해, 게시 프로세스 보다 간단해 대규모 조직에는 보안 향상 기능을 제공 합니다. 

### <a name="improved-certification-experience"></a>인증 환경 개선

파트너 센터는 향상 된 인증 환경을 제공 합니다. 이제 인증 자동화 및 향상 된 소요 시간에 대 한 기계 학습 및 보다 신속한 처리를 활용할 됩니다. 정확도 유지할 최우선, 클라우드 마켓플레이스는 안전 하 고 고객에 대 한 유용한 제품을 포함 하는 상태로 유지 됩니다. 요구 사항 변경 되지 것입니다 되지만 좀 더 자세 하 고 설명이 포함 된 인증 보고서 참조 파트너 센터를 통해 제공 제출할 때 이제 받습니다 [공용 정책](https://docs.microsoft.com/legal/marketplace/certification-policies)합니다. 

### <a name="analytics-for-performance-reporting"></a>성능 보고에 대 한 분석

파트너 센터 analytics 제품의 성능 보기를 제공 하 고 보고 도구를 향상 합니다. 게시자는 데이터 시각화 및 정보 그래프를 사용 하 여 최적화 목록에 sales를 모니터링할 수 있습니다. 향상 된 분석 도구를 사용 하면 고객 및 대리점 관계 개선을 성능 결과 작업할 수 있습니다.
