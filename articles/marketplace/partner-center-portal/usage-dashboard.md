# <a name="usage-dashboard-in-commercial-marketplace-analytics"></a>상용 마켓플레이스 분석의 사용 대시보드

이 문서에서는 파트너 센터의 사용 대시보드에 대한 정보를 제공합니다. 이 대시보드에는 VM 사용량 및 계량 결제 메트릭이 두 개의 별도 탭(VM 사용량 및 계량 된 청구 사용량)으로 모든 VM제공 메트릭이 표시됩니다.

사용 대시보드에 액세스하려면 상업용 마켓플레이스에서 **[분석](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** 대시보드를 엽니다.

>[!NOTE]
> 분석 용어에 대한 자세한 정의는 [상용 마켓플레이스 분석을 위한 자주 묻는 질문 및 용어를](./faq-terminology.md)참조하십시오.

## <a name="usage-dashboard"></a>사용량 대시보드

사용 대시보드는 모든 VM(가상 머신)에 대한 메트릭을 나타내며 사용량 및 유료 청구 사용량을 제공합니다. VM 사용량과 데이터 결제 사용량의 두 개의 별도 탭에서 찾을 수 있습니다.

VM 사용 탭에는 다음 항목의 그래픽 표현이 있습니다.

- [사용 요약](#usage-summary)
- [지역별 사용법](#usage-by-geography)
- [오퍼별 사용](#usage-by-offers)
- [오퍼 및 SUS별 사용 추세](#usage-trend-by-offers-and-skus)
- [오퍼 유형별 사용](#usage-by-offer-type)
- [VM 크기별 사용법](#usage-by-vm-size)
- [판매 채널별 사용량](#usage-by-sales-channel)
- [자세한 사용 데이터](#detailed-usage-data)

> [!NOTE]
> 분석 보고서는 CPP(클라우드 파트너 포털) 및 파트너 센터에서 다르게 표시됩니다. CPP의 **판매자 인사이트에는** 사용 기반 오퍼와 비사용량 기반 오퍼에 대한 데이터를 표시하는 주문 및 사용 탭이 있습니다. 파트너 센터에서 사용 메트릭은 별도의 페이지에 표시됩니다.

### <a name="usage-summary"></a>사용 요약

사용 요약 테이블에는 구입한 모든 오퍼에 대한 고객 사용 시간이 표시됩니다.

- 정규화된 사용 시간은 VM 코어 수([VM 코어 수] x [원시 사용 시간])를 고려하여 정규화된 사용 시간으로 정의됩니다. "SHAREDCORE"로 지정된 VM은 1/6(또는 0.1666)을 [VM 코어 수] 승수로 사용합니다.
- 원시 사용 시간은 VM이 시간 단위로 실행된 시간으로 정의됩니다.
- 백분율 값은 선택한 날짜 범위([지난 달 사용량 - 첫 달 사용량])/첫 달 사용량에 대한 사용량 증가 변화를 나타냅니다.
- 위쪽을 가리키는 녹색 삼각형은 성장 변화를 나타냅니다.
- 아래쪽을 가리키는 빨간색 삼각형은 전월에 비해 마이너스 성장 변화를 나타냅니다.
- 마이크로 막대 그래프는 월별 값을 나타냅니다. 차트 내의 열 위로 마우스를 가져가서 매월 값을 표시할 수 있습니다.

### <a name="usage-by-geography"></a>지역별 사용법

**지리 히트맵별 정규화된 사용량은** 고객 국가에 따라 매핑된 사용 시간을 표시합니다. 국가 색상 변화는 정규화된 사용 농도를 나타냅니다. 맵의 **홈** 버튼을 눌러 원래 보기로 되돌림합니다.

### <a name="usage-by-offers"></a>오퍼별 사용

- 제공 원형 **차트에 의한 정규화된 사용량은** 선택한 날짜 범위에 따라 오퍼별로 정규화된 사용 시간을 분석한 것을 표시합니다. 상위 5개 오퍼는 그래프로 표시되고 나머지는 "모두 나머지" 범주로 그룹화됩니다.
- 막대 차트는 선택한 날짜 범위에 대한 월별 성장 추세를 표시합니다. 월 열은 해당 월에 대한 사용 시간이 가장 높은 오퍼의 사용 시간을 나타냅니다. 선부 차트는 보조 Y축에 그려진 성장 백분율 추세를 나타냅니다.
- 차트 상단의 슬라이더를 사용하여 x축을 따라 오른쪽에서 왼쪽으로 스크롤하거나 특정 데이터 포인트에 초점을 맞춥니다.

### <a name="usage-trend-by-offers-and-skus"></a>오퍼 및 SUS별 사용 추세

이 차트는 선택한 오퍼의 SCO에 대한 정규화된 사용 추세를 표시합니다. 오퍼 리더는 사용량이 가장 많은 상위 50개 오퍼를 표시하며 사용 시간별로 정렬됩니다. SKU 순위표에는 선택한 오퍼에 대해 가장 높은 사용량을 가진 상위 50개 SKU가 표시됩니다.

### <a name="usage-by-offer-type"></a>오퍼 유형별 사용

- 오퍼 유형 원형 **차트별 사용량은** 오퍼 유형에 따라 사용량을 구성합니다.
- 상위 오퍼는 차트에 표시되고 나머지 오퍼는 '모두 휴식'으로 그룹화됩니다.
- **추세** 차트에는 월별 성장 추세가 표시됩니다. 월 열은 해당 월의 상위 오퍼 유형별로 사용량을 나타냅니다.

### <a name="usage-by-vm-size"></a>VM 크기별 사용법

이 차트는 모든 오퍼/SCO 중 선택한 VM 크기(최대 5)의 사용 추세를 나타냅니다. 열수 차트는 선택한 VM 크기의 사용 시간과 함께 누적됩니다.

순위표에는 사용량이 가장 많은 상위 50VM 크기와 사용 시간별로 정렬된 크기가 표시됩니다.

### <a name="usage-by-sales-channel"></a>판매 채널별 사용량

- 판매 채널 원형 차트별 사용량은 판매 채널에 따라 사용량을 구성합니다.
- 사용량이 가장 많은 상위 판매 채널이 차트에 표시되고 나머지 판매 채널은 '모두 휴식'으로 그룹화됩니다.
- 월 열은 해당 월의 상위 판매 채널별 사용량을 나타냅니다.
- 이 차트의 기능은 '쿠폰별 사용' 차트와 동일합니다.

### <a name="detailed-usage-data"></a>자세한 사용 데이터

**사용 세부 정보 테이블에는** 사용량별로 정렬된 상위 1000개 사용 레코드의 번호가 매겨진 목록이 표시됩니다.

- 그리드의 각 열은 정렬할 수 있습니다.
- 레코드 수가 1000개 미만이면 데이터를 CSV 파일로 추출할 수 있습니다.
- 레코드 수가 1000개를 초과하면 내보내기 데이터가 다운로드 페이지에 비동기적으로 배치되어 향후 30일 동안 사용할 수 있습니다.
- **필터는 세부 사용 데이터에** 적용하여 관심 있는 데이터만 표시할 수 있습니다. 데이터는 국가, 판매 채널, 마켓플레이스 라이선스 유형, 사용 유형, 쿠폰 유형, 쿠폰 유형, 무료 평가판, 마켓플레이스 구독 ID, 고객 ID 및 회사 이름으로 필터링할 수 있습니다.

> [!NOTE]
> 페이지 필터에서 **사용 유형을** 선택하여 "정규화된 보기" 또는 "원시 보기"에서 페이지에서 차트를 봅니다. 이러한 차트의 기본 보기는 "정규화된 보기"입니다.

**사용 페이지 필터는** 페이지 수준에서 적용됩니다. 여러 필터를 선택하여 표시하도록 선택한 조건과 "자세한 사용 데이터" 그리드/내보내기에 표시하려는 데이터에 대한 차트를 렌더링할 수 있습니다. 필터는 주문 페이지의 오른쪽 상단 모서리에서 선택한 데이터 범위에 대해 추출된 데이터에 적용됩니다.

- **제안 유형** 및 **제안 이름은** 선택한 기간 동안 획득한 오퍼에 대해서만 나열됩니다. 목록에서 선택한 제품 유형에 대해 목록에 제품 이름이 표시됩니다.
- 기본 선택은 **사용 유형을**제외한 각 필터 옵션에 대해 "모두"입니다. **사용 유형에** 대한 기본 선택은 정규화된 사용량입니다. 차트에 원시 사용량을 표시하려면 "원시 사용량"을 선택합니다.
- 적용된 필터는 만들어진 필터 선택 항목에 대한 개수 선택을 표시합니다. 적용된 필터는 기본 선택 항목에 대해 표시되지 않습니다.

> [!NOTE]
> "상세 순서 데이터" 그리드, 페이지 필터 및 가능한 모든 선택 항목의 각 필드에 대한 자세한 정의는 [FAQ 및 용어](link needed) 문서의 데이터 사전 섹션에 정의되어 있습니다.

**유료 결제 사용** 탭에는 미터 차원별로 사용량이 측정되는 오퍼 유형에 대한 사용량 정보가 표시됩니다. SaaS 오퍼 타입 초과분은 현재 제시되어 있습니다. 이 탭은 SaaS 데이터 결제 사용량에 대한 초과 추세에 대한 그래픽 표현을 제공합니다.

- **미터 차원별 초과 추세**: 오퍼의 선택한 미터 치수에 대한 월별 초과 추세를 표시합니다. X축은 월을 나타내고 Y축은 사용 량을 나타냅니다. 사용자 지정 미터의 측정 단위도 Y축에 표시됩니다.
- **SKU별 초과 추세**: SKU에 의해 선택한 미터 차원의 사용 수량 의 추세를 나타냅니다. 표시되는 SCO는 선택한 오퍼에 대해 가장 많은 사용량을 가진 상위 5개 SCO를 나타냅니다.
- **상위 50 대 고객에 의한 초과 추세**: 가장 높은 사용 시간을 가진 상위 50 개 이벤트는 ***리더 보드에*** 표시되며 사용자 정의 미터의 가장 높은 사용에 의해 순위가 매겨지고 있습니다. 순위표에서 고객을 선택하여 선택한 미터 차원의 사용 추세를 확인합니다.
- **상위 고객별 초과 추세**: 전체 사용량의 %에 기여하는 최고 고객 백분위수(들)를 제시합니다. 상위 고객 백분위수는 X축을 따라 표시되며 고객의 사용 수량에 따라 결정됩니다. Y축은 사용 량을 표시합니다. 표선 차트를 따라 점 위로 마우스를 가져가서 세부 정보를 표시할 수 있습니다.

> [!NOTE]
> 이 페이지의 사용 세부 정보와 모든 차트는 페이지 필터에 대해 선택한 측정기준에 대해 표시됩니다.
