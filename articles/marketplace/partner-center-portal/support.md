# <a name="support-for-the-commercial-marketplace-program-in-partner-center"></a>파트너 센터에서 상업적 Marketplace 프로그램 지원

Microsoft는 광범위한 제품 및 서비스에 대한 지원을 제공합니다. 적절 한 적절 한 응답을 보장 하려면 올바른 지원 팀을 찾아야 합니다. 다음은 쿼리를 적절한 팀으로 라우팅하는 데 도움이 되는 시나리오입니다. 

- 게시자이 고 고객의 질문이 있는 경우 고객에 게 [Azure Portal](https://portal.azure.com/)지원 링크를 사용 하 여 지원을 요청 하도록 요청 하세요. 

- 게시자이 고 앱 이나 서비스와 관련 된 질문이 있는 경우 다음 지원 옵션을 검토 하세요.

## <a name="support-options-for-publishers"></a>게시자에 대 한 지원 옵션

1. 회사 계정을 사용 하 여 [파트너 센터의 상용 Marketplace 프로그램](https://partner.microsoft.com/dashboard/commercial-marketplace/overview) 에 로그인 합니다. 아직 수행 하지 않은 경우 [파트너 센터 계정을 만들어야](./create-account.md)합니다.

1. 페이지 오른쪽 상단 메뉴에서 **지원** 아이콘을 선택 합니다. 
 
   ![지원 아이콘](./media/commercial-marketplace-support.png)

1. **도움말 및 지원** 창이 페이지의 오른쪽에 표시 됩니다. 계정이 등록 된 파트너 센터의 각 프로그램에 대 한 드롭다운이 있습니다. **상업적 마켓플레이스** 드롭다운 메뉴를 선택 합니다. 
 
   ![지원 드롭다운 메뉴](./media/commercial-marketplace-support-pane.png)

1. **[설명서](../index.md)** 를 선택 하 여 질문 및 리소스에 대 한 포괄적인 대답을 검토 합니다. 

1. 다른 Microsoft 게시자의 지식을 활용 하 여 질문에 답변 하려면 **[Marketplace 파트너 커뮤니티 포럼](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/bd-p/2222)** 을 선택 합니다. 

1. **[추가 도움말](https://support.microsoft.com/supportforbusiness/productselection?sapId=48734891-ee9a-5d77-bf29-82bf8d8111ff)** 을 선택 하 여 **새 지원 요청** 티켓을 엽니다. 

## <a name="how-to-open-a-support-ticket"></a>지원 티켓을 여는 방법

파트너 센터의 상업 마켓플레이스 지원 메뉴에서 **추가 도움말** 을 선택 하면 비즈니스용 Microsoft 지원 페이지로 이동 합니다. 몇 가지 드롭다운 메뉴가 표시 됩니다. 처음 몇 개의 드롭다운 메뉴는 자동으로 채워집니다. Marketplace 지원의 경우 제품 제품군을 **클라우드 및 온라인 서비스**, **marketplace 게시자**로, 제품 버전을 **marketplace 게시**로 식별 합니다. 이러한 미리 채워진 드롭다운 메뉴 선택 항목은 변경 하지 마십시오. 

- **문제를 가장 잘 설명 하는 범주 선택**: 질문과 문제에 가장 적합 한 범주를 선택 합니다. 범주를 선택 하면 추가 드롭다운 메뉴가 표시 되어 문제를 가장 잘 설명 하는 문제를 선택 하 라는 메시지가 표시 됩니다.

- 문제 **를 가장 잘 설명 하는 문제 선택**: 문제에 가장 적합 한 나열 된 옵션에서 문제를 선택 합니다. 문제에 따라 새 섹션이 표시 될 수 있습니다. 여기에는 문제와 관련 된 링크를 제공 하는 데 도움이 될 수 있는 몇 가지 정보가 나와 있습니다. **다음** 단추를 선택 하기 전에 이러한 링크를 검토 하 여 질문이 나 문제가 해결 되었는지 확인 합니다.

**다음** 단추를 선택 하면 **문제 세부 정보** 페이지로 이동 합니다. 

#### <a name="tell-us-more-about-your-issue"></a>문제에 대한 자세한 정보

**문제 세부 정보** 섹션에서 도움이 필요한 항목을 설명 합니다.

- 국가를 선택 합니다.
- 지원을 받을 언어를 선택 합니다.
- 문제에 대 한 제목 (3 ~ 200 자)을 입력 합니다.  
- 문제에 대 한 간략 한 설명을 입력 합니다 (최대 2000 자).
- **다음**을 선택합니다.

#### <a name="select-a-support-plan"></a>지원 플랜 선택

**지원 계획** 섹션에는 정보가 필요 하지 않습니다. **다음**을 선택합니다. 

#### <a name="select-the-severity-of-your-issue"></a>문제의 심각도를 선택 합니다.

**심각도** 섹션에는 정보가 필요 하지 않습니다. **다음**을 선택합니다. 

#### <a name="who-should-we-contact"></a>누구에 게 연락 해야 하나요?

**연락처 정보** 섹션에서 기본 연락처의 이름, 전화 번호 및 전자 메일이 올바른지 확인 합니다. 
- 잘못 된 항목이 표시 되는 경우이 정보를 업데이트 합니다.
- 원하는 경우 회사의 다른 연락처를이 지원 요청에 추가할 수 있습니다.
- **다음**을 선택합니다.

#### <a name="review-and-submit-your-support-request"></a>지원 요청 검토 및 제출

**검토** 섹션에서 지원 요청 정보를 검토 합니다. 
- 모든 정보가 올바르면 **제출**을 선택 합니다. 
- 정보를 업데이트 해야 하는 경우 업데이트 필요에서 섹션 옆의 **변경** 단추를 선택 합니다.

## <a name="track-your-existing-support-requests"></a>기존 지원 요청 추적 

지원 요청을 추적 하려면 [상업적 마켓플레이스 지원 페이지](https://support.microsoft.com/supportforbusiness/productselection?sapId=48734891-ee9a-5d77-bf29-82bf8d8111ff)를 방문 하세요. 

위쪽 파란색 메뉴 모음에서 "비즈니스 지원" 옆에 있는 **지원 요청** 링크를 선택 합니다. 

**Microsoft 전문 지원** 섹션에서 모든 오픈 및 폐쇄형 지원 요청을 검토 합니다.
