# <a name="become-a-publisher-in-the-storefronts-on-the-marketplace"></a>마켓플레이스에서 상점의 게시자 되기  
마켓플레이스의 상점에 게시자로 등록하려면 다음 단계를 참조하세요. Microsoft 및 선택된 열거 유형을 사용하는 기존 방법에 따라 몇 가지 단계가 필요하지 않을 수 있습니다.  

|  | 등록 단계 | Time | 세부 정보 |  
|:--- |:--- |:--- |:--- |  
| 1 | Microsoft 파트너 네트워크에 등록 | 15분 | [Microsoft 파트너 네트워크에 등록](#register-in-microsoft-partner-network) |  
| 2 | Microsoft ID 만들기 | 15분 | [Microsoft ID 만들기](#create-a-microsoft-id) |  
| 3 | Marketplace 추천 양식 제출 | 1-3일 | [Marketplace 추천 양식 제출](#submit-the-marketplace-nomination-form) |  
| 4 | Microsoft 개발자 센터(개발자 센터)에 등록 | 5-10일 | [개발자 센터에 등록](#register-in-dev-center) |  
| 5 | AppSource에 로그인 | 15분 | [AppSource에 로그인](#sign-into-appSource) |  
| 5 |  Cloud 파트너 포털에 로그인 | 15분 | [Cloud 파트너 포털에 로그인](#sign-into-cloud-partner-portal) |  

## <a name="register-in-microsoft-partner-network"></a>Microsoft 파트너 네트워크에 등록  
| 15분 |  
MPN(Microsoft 파트너 네트워크)에 조인하여 공식 Microsoft 파트너가 되고 Azure Marketplace에서 게시를 위한 추가 혜택 및 지원을 받습니다. MPN에 등록하고 `Enroll`을 클릭합니다.  
*   MPN에 등록하려면 멤버십 페이지([partner.microsoft.com/membership](https://partner.microsoft.com/membership))를 방문하세요.  

---  

조직에 기존 멤버십이 있는 경우 등록 중에 조직에 가입할 수 있습니다. 등록한 후 조직의 MPN ID를 기록합니다. Microsoft는 Cloud 파트너 포털에서 게시자 프로필을 활성화하기 전에 MPN ID를 요청합니다. 
*   게시자 프로필 활성화에 대한 자세한 내용은 [개발자 센터에 등록](#register-in-dev-center) 섹션을 참조하세요.  

---  

## <a name="create-a-microsoft-id"></a>Microsoft ID 만들기  
| 15분 |  
Microsoft ID는 Cloud 파트너 포털에 액세스하는 데 사용됩니다. 메일 주소는 Microsoft ID로 등록되어야 하며 Cloud 파트너 포털과 개발자 센터에 모두 사용됩니다. 선택한 이메일 주소는 회사 도메인에 포함된 것이거나, IT 팀에서 관리하는 것이 좋습니다.  
*   개발자 센터에 등록하는 방법에 대한 자세한 내용은 [개발자 센터 등록](#register-in-dev-center) 섹션을 참조하세요.  
*   Cloud 파트너 포털에 로그인하는 방법에 대한 자세한 내용은 [Cloud 파트너 포털에 로그인](#sign-into-cloud-partner-portal) 섹션을 참조하세요.  
*   Microsoft ID를 만들기 위한 지침을 검토하려면 [Marketplace 계정을 관리하기 위한 Microsoft ID를 만들기 위한 지침](./guidelines.md#guidelines-for-creating-a-microsoft-id-to-manage-a-marketplace-account) 섹션을 방문하세요.

## <a name="submit-the-marketplace-nomination-form"></a>Marketplace 추천 양식 제출  
| 1-3일 |  
마켓플레이스에서 상점의 게시자가 되려면 자신의 조직을 추천합니다. 
*   Azure Marketplace 추천 양식을 제출하려면 Azure Marketplace 추천 양식 페이지([aka.ms/listonazuremarketplace](http://aka.ms/listonazuremarketplace))를 방문하세요.  
*   AppSource 추천 양식을 제출하려면 앱에 대한 정보 입력 페이지([aka.ms/listonappsource](http://aka.ms/listonappsource))를 방문하세요.  

추천 양식에 다음 정보를 포함해야 합니다.  
*   조직 세부 정보  
*   초기 앱 또는 제안에 대한 세부 정보  
*   고객에게 제공하는 지원 수준에 대한 세부 정보  

양식을 제출하면 앱이나 제안이 검토되고 유효성이 검사됩니다. 승인되면 승인된 파트너가 되기 위한 다음 단계가 포함된 메일을 받게 됩니다. 메일에는 개발자 센터 계정을 만들 때 등록비를 면제 받을 수 있는 프로모션 코드도 포함되어 있습니다. 또한 Cloud 파트너 포털 또는 AppSource에 액세스할 수 있습니다. AppSource에서 첫 번째 앱을 완성하거나 Cloud 파트너 포털에서 첫 번째 제안을 완성합니다. AppSource에서 추가 앱을 만들거나 Cloud 파트너 포털에서 추가 제안을 만듭니다.  
*   개발자 센터에 등록하는 방법에 대한 자세한 내용은 [개발자 센터 등록](#register-in-dev-center) 섹션을 참조하세요.  

---  

## <a name="register-in-dev-center"></a>개발자 센터에 등록  
| 5-10일 |  
Transact 열거 유형의 앱(예: 가상 머신), Azure 앱(예: 솔루션 템플릿), Azure 앱(예: 관리되는 앱) 등을 게시하려면 개발자 센터 계정이 필요합니다. 이 요구 사항을 통해 Microsoft는 회사의 법률, 세금 및 은행 정보를 확인할 수 있습니다. 등록자는 조직의 정식 담당자여야 하며 신원을 확인할 수 있는 개인 정보를 제공해야 합니다. Microsoft ID를 사용하여 등록해야 합니다. Microsoft ID는 회사 내에서 공유되어야 합니다. Cloud 파트너 포털에서 동일한 Microsoft ID를 사용해야 합니다.  
*   Microsoft ID 만들기에 대한 자세한 내용은 [Microsoft ID 만들기](#create-a-microsoft-id) 섹션을 참조하세요.  

>[!NOTE]
>Marketplace 추천 양식을 완료하지 않을 경우 $99의 등록 요금을 지불해야 합니다. 이 요금을 면제 받으려면 Marketplace 추천 양식을 완료하고 메일을 통해 프로모션 코드를 받아야 합니다.  

>[!Important]
>개발자 센터 계정을 만들기 전에 회사에 개발자 센터 계정이 있는지 확인합니다. 
>*   자세한 내용은 [개발자 센터에 등록하는 방법](#how-to-register-in-dev-center) 섹션을 참조하세요.

## <a name="sign-into-appsource"></a>AppSource에 로그인  
| 15분 |  
추천이 승인되고 MPN과 개발자 센터에 모두 등록한 후에는 AppSource에 대한 액세스 권한이 있는 Marketplace 계정이 만들어집니다. 최초 로그인 자격 증명이 포함된 추천 승인 메일이 전송됩니다. 
*   MPN에 액세스하려면 멤버십 페이지([partner.microsoft.com/membership](https://partner.microsoft.com/membership))를 방문하세요.  
*   개발자 센터에 액세스하려면 Windows 개발자 센터([dev.windows.com](https://dev.windows.com))를 방문하세요.  
*   AppSource에 액세스하려면 AppSource 페이지([appsource.microsoft.com](https://appsource.microsoft.com))를 방문하세요.  
*   Microsoft Cloud 파트너 포털에 액세스하려면 Cloud 파트너 포털 페이지([cloudpartner.azure.com](https://cloudpartner.azure.com))를 방문하세요.  
*   Cloud 파트너 포털 사용에 대한 자세한 내용은 학습 섹션([cloudpartner.azure.com/#Learn](https://cloudpartner.azure.com/#Learn))을 참조하세요.  
*   Cloud 파트너 포털에 대한 자세한 내용은 문서 섹션([cloudpartner.azure.com/#documentation](https://cloudpartner.azure.com/#documentation))을 방문하세요.  

---  

## <a name="sign-into-cloud-partner-portal"></a>Cloud 파트너 포털에 로그인
| 15분 |  
추천이 승인되고 MPN 및 개발자 센터에 모두 등록되면 Cloud 파트너 포털에 대한 액세스 권한이 있는 Marketplace 계정이 만들어집니다. 최초 로그인 자격 증명이 포함된 추천 승인 메일이 전송됩니다. 
*   MPN에 액세스하려면 멤버십 페이지([partner.microsoft.com/membership](https://partner.microsoft.com/membership))를 방문하세요.  
*   개발자 센터에 액세스하려면 Windows 개발자 센터([dev.windows.com](https://dev.windows.com))를 방문하세요.  
*   Microsoft Cloud 파트너 포털에 액세스하려면 Cloud 파트너 포털 페이지([cloudpartner.azure.com](https://cloudpartner.azure.com))를 방문하세요.  
*   Cloud 파트너 포털 사용에 대한 자세한 내용은 학습 섹션([cloudpartner.azure.com/#Learn](https://cloudpartner.azure.com/#Learn))을 참조하세요.  
*   Cloud 파트너 포털에 대한 자세한 내용은 문서 섹션([cloudpartner.azure.com/#documentation](https://cloudpartner.azure.com/#documentation))을 방문하세요.
