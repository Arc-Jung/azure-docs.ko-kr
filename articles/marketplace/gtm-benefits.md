# <a name="gtm-benefits-on-the-marketplace"></a>Marketplace의 GTM 혜택   
Marketplace의 상점에 나온 새로운 항목은 Marketplace GTM 혜택을 무료로 받을 자격을 얻게 됩니다. 목록에 게시된 후 Microsoft의 마케팅 전문가가 귀하 또는 귀하의 마케팅 담당자에게 활동 시작을 위해 연락을 드립니다. 연락을 받으시면 Microsoft와 협력하는 것 외에 추가로 필요한 것은 없습니다.  

Microsoft가 제공하는 활동은 Marketplace에서 귀하의 솔루션 상태에 따라 다릅니다. 목록에 추가하면 혜택이 크게 증가합니다.  

| 상점 | 사용 가능한 추가 혜택  |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Microsoft Gold 파트너</td></tr> <tr><td>평가판 앱</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>거래 제품</td></tr> <tr><td>평가판 제품</td></tr> </table> |  

이러한 활동의 영향력을 최대화하기 위해 구현 계획을 준비하는 것이 좋습니다. 시작 전술 대부분에 방문 페이지를 사용할 수 있습니다.  

>[!NOTE]
>OCP 카탈로그(One Commercial Partner Catalog)는 Microsoft 파트너 네트워크에 속하는 멤버를 위한 혜택입니다.  

![Marketplace의 GTM 혜택](./media/marketplace-publishers-guide/gtmmay2018.png)  

시장 진출 지원에는 비즈니스 홍보를 위한 템플릿, 웹 콘텐츠, 교육 및 도구가 포함됩니다.  
*   시장 진출 지원에 대한 자세한 내용은 [partner.microsoft.com/reach-customers/gtm](https://partner.microsoft.com/reach-customers/gtm)에 있는 Microsoft 시장 진출 서비스 페이지를 방문하세요.
