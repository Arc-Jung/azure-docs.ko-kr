# <a name="support-for-microsoft-commercial-marketplace"></a>Microsoft 상업적 marketplace에 대 한 지원

Microsoft 상업적 마켓플레이스 게시자로 시작 하는 데 대 한 질문이 있나요? 다음은 상용 marketplace에 대 한 지원 옵션 목록입니다. 다음 리소스 외에도 [C + AI 커뮤니티 포럼의 Marketplace 채널](https://www.microsoftpartnercommunity.com/t5/Marketplace/bd-p/2222)에서 많은 질문에 대 한 답변을 받을 수 있습니다.  

## <a name="onboarding"></a>온보딩

온 보 딩 및 시작에 대 한 문제에 대 한 Microsoft [marketplace 게시자 지원](https://partner.microsoft.com/support/v2/?stage=1) 으로 티켓을 엽니다.
