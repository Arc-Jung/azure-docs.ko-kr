# <a name="insights-page"></a>인사이트 페이지

[**인사이트**](https://cloudpartner.azure.com/#insights) 페이지에서는 Marketplace 제품을 이해하고 판매량을 최대화하는 데 도움이 되는 다음과 같은 대시보드 세트를 제공합니다.  각 대시보드는 페이지의 맨 위에 있는 동일한 이름의 탭을 통해 액세스할 수 있습니다.


|  **대시보드**    |  **표시된 콘텐츠**               |
|  -------------    |  ---------------------               |
| **요약**       | 그래프, 추세 및 게시자의 제품에 대한 Marketplace 작업을 요약하는 집계 데이터의 값 |
| **지급**        | 그래픽 및 다운로드할 수 있는 형식의 지급 및 관련된 트랜잭션 |
| **주문 및 사용량** | 그래픽 및 다운로드할 수 있는 형식의 주문 및 사용량 |
| **Customer**      | 해당 구매 프로필을 포함하는 고객 정보 |
| **배포**    | 그래프와 이벤트 수준 형식으로 표시되는 배포 성공 및 실패 정보 |
| **다운로드**     | 지난 30일 간 다운로드 요청의 목록 |
| **분석**     | 웹 분석의 요약 및 캠페인 성능의 분석 |
|  |  |


## <a name="summary-dashboard"></a>요약 대시보드
요약 대시보드는 첫 번째 탭으로, 인사이트 페이지로 이동하는 경우 표시되는 기본 탭입니다.  이 의도적인 디자인은 이 대시보드가 게시자에게 해당 제품의 판매 활동에 대한 가장 광범위한 보기를 제공한다는 것을 반영합니다.  정규화된 사용량 추세, 월간 사용량, 지리적 시장, 제품 및 고객별 사용량 추세, 지급 추세에 대한 그래픽 표현을 제공합니다.  분석에 대한 시작 및 종료 날짜를 지정하여 출력을 사용자 지정할 수 있습니다.  다음 이미지는 2018년 6월부터 8월까지 Contoso의 예제 요약 대시보드입니다.
