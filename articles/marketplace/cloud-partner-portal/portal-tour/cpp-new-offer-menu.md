# <a name="new-offer-menu"></a>새 제품 메뉴

> 이 메뉴는 새 제품 유형 및 제품 이름 변경 사항을 반영하도록 주기적으로 업데이트되므로 표시 항목이 위의 테이블에 나온 항목과 일치하지 않을 수도 있습니다.
