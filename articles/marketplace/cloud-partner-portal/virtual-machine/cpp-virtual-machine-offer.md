# <a name="azure-virtual-machine-offer"></a>Azure Virtual Machine 제품

[Cloud 파트너 포털](https://cloudpartner.azure.com/)은 Azure Virtual Machine 제품의 생성 및 관리를 더 이상 지원하지 않습니다. 이 기능은 Microsoft [파트너 센터](https://partner.microsoft.com/pcv/)로 이전되었습니다. 자세한 내용은 [Azure Virtual Machine 제품 만들기](../../partner-center-portal/azure-vm-create-offer.md)를 참조하세요.
