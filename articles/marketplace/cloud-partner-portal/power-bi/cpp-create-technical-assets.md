# <a name="create-power-bi-app-technical-assets"></a>Power BI 앱 기술 자산 만들기

이 제안 유형에 필요한 기본 기술 자산은 Power BI 애플리케이션으로, 기본 데이터 세트, 보고서 및(또는) 대시보드와 선택적인 연결된 서비스 및 포함된 데이터 세트의 모음입니다. Power BI 애플리케이션은 이전에 [콘텐츠 팩](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction)이라고 알려진 자산 유형이 발전된 것입니다.  이러한 애플리케이션을 개발하는 방법에 대한 자세한 내용은 [Power BI 앱](https://go.microsoft.com/fwlink/?linkid=2028636)을 참조하세요.


## <a name="obtaining-an-installation-url"></a>설치 URL 가져오기

Power BI 앱은 [Power BI 서비스](https://powerbi.microsoft.com/) 내에서만 빌드할 수 있으며, 이 경우 [Power BI Pro 라이선스](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro)를 사용하여 로그인해야 합니다. Power BI에서 서비스 애플리케이션이 만들어지고 테스트되면 Power BI에서 생성하는 애플리케이션 설치 URL을 저장합니다.  이 URL은 Cloud 파트너 포털 제출 양식의 [기술 정보 탭](./cpp-technical-info-tab.md)에 제공해야 합니다.

Power BI 앱이 만들어져 테스트되고 관련 제안이 만들어졌으면 [Power BI 제안을 게시](./cpp-publish-offer.md)할 수 있습니다.
