# <a name="manage-azure-and-appsource-marketplace-offers"></a>Azure 및 AppSource Marketplace 제품 관리

이 섹션에서는 Azure 및 AppSource Marketplace의 제품을 관리하는 데 사용되는 일반적인 작업을 소개합니다.  예를 들어 [Cloud 파트너 포털](https://cloudpartner.azure.com/)을 사용하여 마켓플레이스 제품을 만들고 게시하는 방법을 설명합니다.  이 섹션에서는 텍스트 기반 및 이미지 자산과 같은 대부분의 제품 유형에 공통된 기술 문제에 관해서도 설명합니다.


## <a name="standard-offer-operations"></a>표준 제품 작업

Cloud 파트너 포털을 통해 게시자는 제품에 대해 다음과 같은 기본 작업을 수행할 수 있습니다.

|     작업(Operation)      |  설명                                           |
|     ---------      |  -----------                                           |
| [제품 만들기](./cpp-create-offer.md)   | 온보딩 리포지토리에 보류 중인 제품 항목을 만들지만 Microsoft 마켓플레이스에 자동으로 게시하지 않습니다. | 
| [제품 상태 보기](./cpp-view-status-offer.md)   | 제품의 게시 상태를 제공합니다.  포털에는 상태 정보가 표시되는 여러 위치가 있습니다. |
| [제품 게시](./cpp-publish-offer.md) | 유효성을 검사한 후 완료된 제품 항목을 적절한 Microsoft 마켓플레이스에 게시합니다.  그러면 제품이 ‘라이브’ 상태라고 합니다.** |
| [제품 업데이트](./cpp-update-offer.md)   | 기존 제품의 특정 특성을 수정할 수 있습니다.  보류 중인 제품과 게시된 제품 둘 다에 적용됩니다. |
| [제품 삭제](./cpp-delete-offer.md)   | 기존의 보류 중인 제품이나 라이브 제품을 삭제합니다.  | 
|  |  |
