# <a name="cloud-partner-portal-management"></a>Cloud 파트너 포털 관리

이 섹션에서는 [Cloud 파트너 포털](https://cloudpartner.azure.com)를 사용 하는 데 필요한 계정을 만들고 관리 하는 데 사용할 수 있는 작업 유형에 대해 설명 합니다.  이러한 계정은 [Azure Marketplace](https://azuremarketplace.microsoft.com) 및 [appsource](https://appsource.microsoft.com)에 필요 합니다.  


## <a name="background-information"></a>배경 정보

Cloud 파트너 포털를 사용 하기 전에 다음 문서를 숙지 해야 합니다.

- [Azure Marketplace 정책과 약관](../../policies-terms.md) 에는 및 게시자의 사용자에 게 [Azure Marketplace](https://azuremarketplace.microsoft.com) 및 [appsource](https://appsource.microsoft.com)에 적용 되는 다양 한 법적 정보와 기술 요구 사항 및 규칙이 나열 됩니다.
- [클라우드 Marketplace 게시자](https://docs.microsoft.com/azure/marketplace/become-publisher) 는 공식 Microsoft 클라우드 게시자가 되기 위해 필요한 모든 단계에 대 한 개요를 제공 합니다.


## <a name="account-creation-and-management"></a>계정 만들기 및 관리

이 섹션에는 marketplace 게시자 계정을 관리 하는 방법을 설명 하는 다음과 같은 문서가 포함 되어 있습니다.

- [계정 관리를 위한 필수 구성 요소](./cpp-manage-prerequisites.md) 는 클라우드 파트너 포털을 사용 하기 위한 요구 사항인 등록 된 클라우드 게시자가 될 수 있는 방법을 설명 합니다.
- [Azure Marketplace 및 AppSource 게시자 프로필 관리](./cpp-manage-publisher-profile.md) Cloud 파트너 포털를 사용 하 여 Microsoft Marketplace 게시자 프로필을 보고 관리 하는 방법을 설명 합니다. 
- [Cloud 파트너 포털에서 사용자를 관리](./cpp-manage-users.md) 하 여 조직의 클라우드 게시 계정에 사용자를 추가 하 고 해당 액세스를 관리 하는 방법으로 여러 사람이 동일한 제품에서 작업 하도록 할 수 있습니다.


## <a name="next-steps"></a>다음 단계

계속 하기 전에 [계정 관리를 위한 필수 구성 요소](./cpp-manage-prerequisites.md) 를 충족 해야 합니다.  그런 다음 [Cloud 파트너 포털 둘러보기를](../portal-tour/cpp-portal-tour.md) 사용 하 여 포털의 사용자 인터페이스를 숙지 해야 합니다.
