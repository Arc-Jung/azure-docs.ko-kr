# <a name="azure-saas-application-offer"></a>Azure SaaS 애플리케이션 제품

[Cloud 파트너 포털](https://cloudpartner.azure.com/) 는 더 이상 SaaS 응용 프로그램 제품의 생성과 관리를 지원 하지 않습니다.  대신이 기능은 Microsoft [파트너 센터 포털로](https://partner.microsoft.com/pcv/)이동 되었습니다.  자세한 내용은 [새 SaaS 제품 만들기](../../partner-center-portal/create-new-saas-offer.md)를 참조 하세요. 


## <a name="next-steps"></a>다음 단계

새 SaaS 앱 제안을 만들기 전에 [saas 응용 프로그램 제공 게시 가이드](../../marketplace-saas-applications-technical-publishing-guide.md) 를 읽고 [제품 만들기 검사 목록](../../partner-center-portal/offer-creation-checklist.md)에 나열 된 파트너 센터 포털 단계를 검토 해야 합니다.
