# <a name="azure-saas-application-offer"></a>Azure SaaS 애플리케이션 제품

[클라우드 파트너 포털은](https://cloudpartner.azure.com/) 더 이상 SaaS 애플리케이션 오퍼의 생성 및 관리를 지원하지 않습니다.  대신 이 기능이 Microsoft 파트너 [센터 포털로](https://partner.microsoft.com/pcv/)이동되었습니다.  자세한 내용은 [새 SaaS 오퍼 만들기를](../../partner-center-portal/create-new-saas-offer.md)참조하십시오. 


## <a name="next-steps"></a>다음 단계

새 SaaS 앱 오퍼를 만들기 전에 [SaaS 응용 프로그램 제공 게시 가이드를](../../marketplace-saas-applications-technical-publishing-guide.md) 읽고 [제안 만들기 체크리스트에](../../partner-center-portal/offer-creation-checklist.md)나열된 파트너 센터 포털 단계를 검토해야 합니다.
