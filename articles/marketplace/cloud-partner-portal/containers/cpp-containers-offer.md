[Cloud 파트너 포털](https://cloudpartner.azure.com/)에서는 더 이상 Azure Container 제품의 생성 및 관리를 지원하지 않습니다. 이 기능은 Microsoft [파트너 센터](https://partner.microsoft.com/pcv/)로 이동했습니다. 자세한 내용은 [Azure Container 제품 생성](../../partner-center-portal/create-azure-container-offer.md)을 참조하세요.
