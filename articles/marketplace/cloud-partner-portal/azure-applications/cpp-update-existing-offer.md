# <a name="update-an-existing-azure-application-offer"></a>기존 Azure 애플리케이션 제품 업데이트

[Cloud 파트너 포털](https://cloudpartner.azure.com/) 는 더 이상 Azure 응용 프로그램 제품의 생성 및 관리를 지원 하지 않습니다. 대신이 기능은 Microsoft [파트너 센터 포털로](https://partner.microsoft.com/)이동 되었습니다. 자세한 내용은 [Azure 응용 프로그램 제품 만들기](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-new-azure-apps-offer)를 참조 하세요.

새 Azure 응용 프로그램 제품을 만들기 전에 [파트너 센터 계정 만들기](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account) 를 읽은 다음 **개요** 페이지가 선택 된 상태로 [상용 마켓플레이스 대시보드](https://partner.microsoft.com/dashboard/directory) 를 열어야 합니다. [Azure 응용 프로그램 제품 만들기](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-new-azure-apps-offer) 의 지침에 따라 제품을 만듭니다.
