# <a name="co-sell-partner-engagement"></a>공동 판매 파트너 참여

파트너 인 경우 Microsoft 공동 판매 프로그램을 사용 하 여 공동 판매 기회를 통해 Microsoft와 직접 작업할 수 있습니다. 이 프로그램을 사용 하면 파트너가 방대한 고객 커뮤니티에 연결 하 고 비즈니스 성장을 가속화 하는 새로운 판매 기회를 만들 수 있습니다.

## <a name="partner-selling-motions"></a>파트너 판매 동작

Microsoft는 세 가지 파트너 판매 동작을 제공 합니다.

* *파트너를 통해 판매*: 판매자에 게 Microsoft 기술을 재판매 수 있습니다.
* *서비스 공동 판매*: 판매자와 파트너는 Microsoft 기술을 포함 하는 프로젝트 또는 관리 서비스를 판매 하기 위해 공동 작업 합니다.
* *IP 공동 판매*: 판매자와 파트너는 반복 되 고 패키지 된 ip 솔루션을 판매 하기 위해 협력 합니다. *

\*이러한 IP 공동 판매 솔루션 중 일부는 Microsoft 현장 판매 팀과 함께 성공적으로 판매 및 배포 될 때 추가 성과급을 받을 수 있습니다.
