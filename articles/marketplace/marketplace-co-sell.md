# <a name="co-sell-partner-engagement"></a>공동 판매 파트너 참여

파트너는 Microsoft 공동 판매 프로그램을 사용하여 Microsoft와 직접 협력하면서 공동 판매 기회를 창출할 수 있습니다. 이 프로그램을 사용하면 파트너가 방대한 고객 커뮤니티에 접촉하고 비즈니스 성장을 가속화하는 새로운 판매 기회를 창출할 수 있습니다.

## <a name="partner-selling-motions"></a>파트너 판매 동작

Microsoft는 다음과 같은 세 가지 파트너 판매 동작을 제공합니다.

* *파트너를 통한 판매* – 고유한 판매자를 통해 Microsoft 기술을 재판매할 수 있습니다.
* *서비스 공동 판매* - 판매자와 파트너는 Microsoft 기술이 포함된 프로젝트 또는 관리형 서비스를 판매하기 위해 협력합니다.
* *IP 공동 판매* - 판매자와 파트너는 반복되는 패키징 IP 솔루션을 판매하기 위해 협력합니다.*

\* 이러한 IP 공동 판매 솔루션 중 일부는 Microsoft 현장 판매 팀을 통해 성공적으로 판매 및 배포될 때 추가 성과급을 받을 수 있습니다.
