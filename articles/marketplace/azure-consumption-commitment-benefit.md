# <a name="azure-consumption-commitment-benefit"></a>Azure 사용량 약정 혜택

Microsoft 상업적 marketplace 제품의 선택 집합은 조직에 다음 계약 중 하나가 있는 경우 조직의 Microsoft Azure 사용 약정에 기여 합니다.

- MACC (Microsoft Azure 사용량 약정)
- 사용할 Azure 커밋 (CtC)

이 프로그램에 참여 하는 모든 제품의 유효성을 검사 하 여 고품질 솔루션을 받을 수 있도록 합니다.

이 혜택을 활용 하려면 Azure 계약과 관련 된 구독을 사용 하 여 Azure Marketplace에 대 한 적격 제안을 구매 하기만 하면 됩니다. Azure 선불 및 금액 약정은 이러한 혜택을 받을 자격이 없습니다.

> [!IMPORTANT]
> 제외는이 marketplace 혜택 전에 서명 된 CtC 계약에 적용 될 수 있습니다. 자격에 대 한 질문이 있는 경우 Microsoft 계정 임원에 문의 하세요.
