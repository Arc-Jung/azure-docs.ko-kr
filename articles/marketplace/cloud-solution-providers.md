# <a name="cloud-solution-providers"></a>클라우드 솔루션 공급 기업

소프트웨어 제품은 [Microsoft 웹 상점](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace)을 통한 제품의 공개 가용성 외에도 CSP (클라우드 솔루션 공급자) 프로그램에서 파트너가 제공 하는 수많은 microsoft 고객에 게 도달할 수 있습니다.

게시자는 옵트인 (기본값은 제외 됨)에 따라 CSP 프로그램의 가용성에 대 한 제안을 구성 합니다. 새 제품이 나 기존 제품의 경우 파트너가 제품을 판매 하 고 고객을 위한 번들 솔루션을 만들 수 있도록 합니다.

게시자는 최종 고객에 게 중단-수정 지원을 제공 하 고 CSP 파트너 및/또는 고객이 지원 담당자에 게 연락할 수 있는 메커니즘을 제공 하는 일을 담당 합니다. Csp 파트너가 고객의 계층 1 지원 요청을 처리할 수 있도록 사용자 설명서, 교육 및 서비스 상태/중단 알림 (해당 하는 경우)을 제공 하는 것이 모범 사례입니다.

다음 표에서는 CSP 프로그램을 옵트인 (opt in) 할 수 있는 제품 유형 및 가격 책정 모델을 보여 줍니다.

| **제품 유형**    | **BYOL**  |  **Free** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| VM  | 예 | 예 | 예 |
| 컨테이너 | 해당 없음 | 예 | 해당 없음 |
| 솔루션 템플릿 | 해당 없음 | 예 | 해당 없음 |
| Managed Applications | 해당 없음 | 예 | 예 |
| SaaS(Software-as-a-Service) | 해당 없음 | 예 | 예 |
|   |   |   |

## <a name="how-to-configure-an-offering"></a>제품을 구성 하는 방법

CSP 프로그램 옵트인 설정은 파트너 센터 또는 Cloud 파트너 포털 제품 만들기 환경에서 구성 됩니다. [변경 게시자 환경에 대해 자세히 알아보세요.](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/Cloud-Marketplace-In-Partner-Center/m-p/9738#M293)

Cloud 파트너 포털 Marketplace 또는 Storefront 탭에서 설정 됩니다. 파트너 센터에서는 설치 정보의 제품 설정 탭에서 설정 됩니다.

![CPP의 CSP 옵트인 (opt in) 환경](media/marketplace-publishers-guide/csp-opt-in.png)

아래 표시 된 것 처럼 파트너 센터에서 게시자는 csp **프로그램에서 제공**하는 확인란을 선택 하 여 csp 프로그램에서 해당 제품을 사용할 수 있도록 선택할 수 있습니다.

![파트너 센터의 CSP 옵트인 (opt in) 환경](media/marketplace-publishers-guide/pc-csp-opt-in.png)

## <a name="sharing-sales-and-support-materials-with-csp-partners"></a>CSP 파트너와 판매 및 지원 자료 공유

클라우드 솔루션 공급자 프로그램의 파트너가 제품을 가장 효과적으로 표시 하 고 조직에 참여 하 게 하려면 대리점에서 사용할 수 있는 판매 및 지원 자료를 제출 합니다. 이러한 리소스는 marketplace 상점 고객에 게 노출 되지 않습니다.

아래 Cloud 파트너 포털에 표시 된 것 처럼 게시자는 관련 마케팅 자료 및 채널 연락처 정보를 CSP 채널로 호스트 하는 URL을 입력할 수 있습니다.

![CSP 참고 자료 정보](media/marketplace-publishers-guide/cpp-csp-information.png)
