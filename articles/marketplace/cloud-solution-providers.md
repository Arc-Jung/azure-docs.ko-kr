# <a name="cloud-solution-providers"></a>클라우드 솔루션 공급 기업

소프트웨어 제품을 통해 제품의 일반 공급 하는 것 외에도 클라우드 솔루션 공급자 (CSP) 프로그램에서 파트너가 제공 하는 정규화 된 Microsoft 고객은 수많은 도달할 수 있습니다 [Microsoft의 웹 상점](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace)합니다.

옵트인 단위로 CSP 프로그램의 가용성에 대 한 제품을 구성 하는 게시자 (기본은 제외 됨), 고객에 대 한 솔루션을 번들 파트너가 제품을 판매 하 고 만들 수 있게을 새 제품 또는 기존 합니다.

게시자는 최종 고객에 게 중단-수정 지원을 제공 하는 것에 대 한 지원을 위한 연락을 고객이 및/또는 CSP 파트너에 대 한 메커니즘을 제공 하는 데 책임이 있습니다. 교육, 사용자 설명서를 통해 CSP 파트너를 제공 하 고 서비스를 상태/중단 알림 CSP 파트너는 고객의 계층 1 지원 요청을 처리 하는 데 참조할 수 있도록 하기 위한 모범 사례는 것입니다.

다음 표에서 제품 유형 및 CSP 프로그램에 옵트인 할 수는 가격 책정 모델을 보여 줍니다.

| **제품 유형**    | **BYOL**  |  **Free** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| VM  | 예 | 예 | 예 |
| 컨테이너 | N/A | 예 | N/A |
| 솔루션 템플릿 | N/A | 예 | N/A |
| Managed Applications | N/A | 사용자 계정 컨트롤 | 예 |
| SaaS(Software-as-a-Service) | N/A | 사용자 계정 컨트롤 | 예 |
|   |   |   |

## <a name="how-to-configure-an-offering"></a>제품을 구성 하는 방법

CSP 프로그램 옵트인 설정은 파트너 센터 또는 클라우드 파트너 포털 제품 만들기 환경에서 구성 됩니다. ([변경 게시자 환경에 자세히 알아봅니다. ](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/Cloud-Marketplace-In-Partner-Center/m-p/9738#M293))

클라우드 파트너 포털의 Marketplace 또는 Storefront 탭에서 설정 됩니다. 파트너 센터에서 설치 세부 정보 아래에서 제품 설정 탭에서 설정 됩니다.

![CPP에서 CSP 옵트인 경험](media/marketplace-publishers-guide/csp-opt-in.png)

파트너 센터에서 아래와 같이 게시자를 선택할 수 있는 상자를 확인 하 여 해당 제품 CSP 프로그램에 사용할 수 있도록 **내 제품을 CSP 프로그램에서 사용할 수 있도록**입니다.

![파트너 센터에서 CSP 옵트인 경험](media/marketplace-publishers-guide/pc-csp-opt-in.png)

## <a name="sharing-sales-and-support-materials-with-csp-partners"></a>CSP 파트너를 사용 하 여 판매 및 지원 자료 공유

파트너 클라우드 솔루션 공급자 프로그램에 가장 효과적으로 귀하의 제품을 나타내는 조직과 engage, 판매를 제출 하 고 지 원하는 감소 한 대리점을 사용할 수 있는 자료를 사용할 수 있도록 합니다. 이러한 리소스는 marketplace 상점에서 고객에 게 노출 되지 않습니다.

Cloud 파트너 포털에서 아래와 같이 게시자 수 CSP 채널을 채널 연락처 정보 및 마케팅 관련 자료를 호스트 하는 URL을 입력 합니다.

![CSP 참고 자료 정보](media/marketplace-publishers-guide/cpp-csp-information.png)
