# <a name="azure-appsource-and-marketplace-review-policies"></a>Azure AppSource 및 Marketplace 리뷰 정책

Azure AppSource 및 Marketplace에 대한 등급 및 리뷰 정책이 여기에 나열되어 있습니다.

* AppSource 및 Azure Marketplace 리뷰 시, Microsoft는 다음과 같은 행동 또는 콘텐츠를 허용하지 않습니다.
    * 봇/사이버 공격
    * 개인 정보 취급 방침 위반
    * 스팸 콘텐츠
    * 스캠 콘텐츠
    * 불쾌감을 주는 콘텐츠: 외설적, 모욕적 또는 공격적인 언어나 제스처
    * 불법 콘텐츠
    * 악의적, 적대적 또는 위협적 콘텐츠
    * 명예 훼손 콘텐츠
    * 동일한 또는 유사한 콘텐츠의 반복 제출
    * 다른 앱 및 서비스의 프로모션을 비롯한 광고

* Microsoft는 어떤 이유로든 사용자가 제출한 리뷰 관련 콘텐츠를 제거할 권리를 보유합니다.
* Microsoft는 어떤 이유로든 사용자가 리뷰 관련 콘텐츠를 제출하지 못하게 차단할 수 있는 권리를 보유합니다.
* 게시자의 애플리케이션과 관련된 리뷰 관련 콘텐츠가 제거되어도 해당 게시자에게 알림이 제공되지 않습니다.
* 사용자가 제출한 리뷰 관련 콘텐츠가 제거되어도 해당 사용자에게 알림이 제공되지 않습니다.
* Microsoft는 리뷰 관련 콘텐츠를 제거한 이유를 설명할 의무가 없습니다.
* Microsoft는 제거된 리뷰 관련 콘텐츠를 복원해 달라는 요청을 수락하지 않습니다.

## <a name="review-guidelines-for-customers"></a>고객용 리뷰 지침

* [AppSource 및 Azure Marketplace 리뷰 정책](./rating-review-policies.md)을 위반할 수 있는 콘텐츠를 제출하지 마세요.
* 이메일 주소, 실제 주소, 전화 번호 등의 PII(개인 식별 정보)를 포함하지 마세요.
* 리뷰는 간결하고 읽을 수 있고 정보를 제공해야 합니다.
* 리뷰의 관점이 균형적이고 중립적이어야 합니다.
* 리뷰는 공개 포럼에 게시되므로 그에 맞게 적절하게 행동해야 합니다.
