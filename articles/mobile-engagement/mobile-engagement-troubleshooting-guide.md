# Azure Mobile Engagement - 문제 해결 가이드

## 소개
다음 문제 해결 가이드에서는 Azure Mobile Engagement와 관련하여 가장 흔히 발생하는 문제에 대해 다룹니다. 단, [청구][Link 11] 관련 질문과 [미리 보기][Link 7] 등록 지원에 대해서는 설명하지 않습니다 이러한 문제 해결 가이드를 통해 문제를 해결할 수 없으면 [Azure 지원 계약](http://azure.microsoft.com/support/options/)이 있는 경우 Azure 서비스 요청을 개설할 수 있으며, 그렇지 않은 경우에는 [MSDN 포럼][Link 8]에서 지원을 요청할 수 있습니다 기술 지원 담당자가 문제를 조사할 수 있도록 서비스 요청을 개설할 때는 [서비스 요청 정보](mobile-engagement-troubleshooting-guide-sr-info.md)의 모든 문제 해결 정보를 포함해야 합니다.
