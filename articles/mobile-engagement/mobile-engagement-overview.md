#Azure Mobile Engagement 개요

Azure Mobile Engagement는 연결된 모든 장치에서 실시간 세분화된 사용자 구분, 앱 사용자 분석 및 컨텍스트 인식 스마트 푸시 알림과 앱 내 메시징을 지원하는 SaaS 제공 데이터 기반 사용자 참여 플랫폼입니다.

###개요 동영상
> [AZURE.VIDEO azure-mobile-engagement-overview]

Azure Mobile Engagement를 통해 앱 게시자와 마케팅 전문가는 첨단 상호 작용을 만들 수 있습니다.

##조치 가능한 실시간 분석으로 앱 사용 늘리기

빅데이터 수집과 실시간 처리를 결합하여 앱 사용자 동작 및 인구 통계에 따라 참여 시나리오를 트리거하세요. Azure Mobile Engagement는 특정 비즈니스 요구 사항과 관련된 거의 모든 질문에 답변할 수 있습니다. 예를 들어 사용자 지정 대시보드를 만들어 KPI를 측정하고, 사용자 경로에서 사용 병목 상태를 빠르게 찾아 해결하고, 재방문 주기 및 연결 유지를 추적함으로써 어떤 캠페인이 가장 높은 ROI를 제공하는지 결정할 수 있습니다. Azure Mobile Engagement의 360도 사용자 경로 뷰를 통해 사용자 환경을 지속적으로 쉽게 향상하고 최적화하여 재방문 주기율을 높이고 앱 사용을 늘릴 수 있습니다.

##부가 가치 푸시 및 통신 플랫폼

Azure Mobile Engagement는 Google의 GCM, Apple의 APNS 또는 Microsoft MPNS와 같은 네이티브 푸시 알림 게이트웨이와도 문제없이 작동할 뿐만 아니라 지능적인 고객층 및 앱 내 메시징 기능을 제공하여 대상 캠페인을 만들 수 있는 기능도 제공합니다.

따라서 앱 개발자자 매우 효율적인 비주입식 방법으로 고객과 연결하는 이점을 제공합니다.

##개방형 API 및 간편한 통합

개방형 API와 간편한 통합을 제공하여 CRM, CMS 및 다른 비즈니스 관련 시스템에서 기존 데이터를 활용할 수 있습니다. 따라서 고객 대상을 더 향상하고 투자를 보호할 수 있습니다.

##전 세계에서 데이터 보호 및 프라이버시

Microsoft는 오랫동안 개인 정보 보호를 위해 노력해 왔으며 개인 정보 보호는 Microsoft 제품 및 서비스 수명 주기의 필수 요소입니다. 우리는 투명한 개인 정보 관행을 통해 고객에게 중요한 개인 정보 선택권을 제공하고, 우리가 저장하는 데이터를 책임감 있게 관리하고 있습니다. 모든 Azure 서비스와 마찬가지로 Mobile Engagement에서는 고객의 재산에서 파생된 정보를 공유하거나 판매하지 않습니다. Azure Mobile Engagement는 전 세계에 배포 가능한 솔루션이므로, 지역 또는 국가별 데이터 프라이버시 법률을 따를 수 있습니다.
