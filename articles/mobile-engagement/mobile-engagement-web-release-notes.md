#Azure Mobile Engagement 웹 SDK 릴리스 정보

##2\.0.1 (06/10/2016)

-   IE8 및 IE9에서는 SDK를 사용할 수 없습니다.
-   Opera 웹 브라우저 감지 문제가 수정되었습니다.

##2\.0.0 (06/07/2016)

-   Azure Mobile Engagement의 최초 릴리스입니다.
-   appId 구성이 연결 문자열 구성으로 대체됩니다.
-   에이전트를 활성화/비활성화하는 API가 추가되었습니다.
-   보안이 개선되었습니다.
-   브라우저에서 네이티브 JSON API를 사용합니다.
-   장치 간에 메시지를 보내고 받기 위한 API가 제거되었습니다.
-   사용이 중단된 XMPP 연결과 관련된 콜백이 제거되었습니다. 
-   이 버전은 도달률 기능을 지원하지 않습니다.
