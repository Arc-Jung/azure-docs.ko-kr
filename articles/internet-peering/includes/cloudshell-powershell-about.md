cmdlet을 실행하기 위해 무료 대화형 셸인 Azure Cloud Shell을 사용할 수 있습니다. 공용 Azure 도구가 사전 설치되어 계정에서 사용하도록 구성되어 있습니다. **복사를** 선택하여 코드를 복사하고 클라우드 셸에 붙여넣습니다. 그런 다음 **Enter를** 선택하여 실행합니다. 클라우드 셸을 시작하는 몇 가지 방법이 있습니다.


|  |   |
|-----------------------------------------------|---|
| 브라우저에서 Cloud Shell을 엽니다. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Azure 포털의 오른쪽 상단에 있는 도구 모음에서 **클라우드 셸** 단추를 선택합니다. | [![포털의 Cloud Shell](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Azure Cloud Shell을 사용하지 않으려면, 대신 PowerShell을 로컬에 설치할 수 있습니다. PowerShell을 로컬에 설치하여 사용하는 경우 최신 버전의 Azure Resource Manager PowerShell cmdlet을 설치해야 합니다. PowerShell cmdlet은 자주 업데이트됩니다. 일반적으로 최신 기능 기능을 얻으려면 PowerShell cmdlet을 업데이트해야 합니다. 그렇지 않으면 문제가 발생할 수 있습니다.

로컬에서 실행 중인 PowerShell 버전을 찾으려면 **'모듈 -목록 사용 가능한 Az'** cmdlet을 사용합니다. 업데이트하려면 [Azure PowerShell 모듈 설치](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)를 참조하세요. 자세한 내용은 [Azure PowerShell 을 설치하고 구성하는 방법을](https://docs.microsoft.com/powershell/azure/azurerm/overview)참조하세요.

macOS에서 PowerShell을 사용하는 경우 [macOS에서 PowerShell 설치](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6) 단계를 따릅니다.
