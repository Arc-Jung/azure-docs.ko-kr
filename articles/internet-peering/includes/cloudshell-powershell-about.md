cmdlet을 실행하기 위해 무료 대화형 셸인 Azure Cloud Shell을 사용할 수 있습니다. 공용 Azure 도구가 사전 설치되어 계정에서 사용하도록 구성되어 있습니다. **복사**를 클릭하여 코드를 복사하고, Cloud Shell에 붙여넣고, Enter 키를 눌러 실행합니다. Cloud Shell을 시작하는 몇 가지 방법이 있습니다.


|  |   |
|-----------------------------------------------|---|
| 브라우저에서 Cloud Shell을 엽니다. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| 포털 오른쪽 상단의 메뉴에서 **클라우드 셸** 단추를 클릭합니다. | [![포털의 Cloud Shell](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Azure CloudShell을 사용하지 않으려면 대신 PowerShell을 로컬로 설치할 수 있습니다. PowerShell을 로컬로 설치하고 사용하도록 선택한 경우 리소스 관리자 PowerShell cmdlet의 최신 버전을 설치해야 합니다. PowerShell cmdlet은 자주 업데이트되며 일반적으로 PowerShell cmdlet을 업데이트하여 최신 기능 기능을 얻지 못하여 문제가 발생할 수 있습니다.

로컬에서 실행 중인 PowerShell 버전을 찾으려면 '모듈 -목록 사용 가능한 Az' cmdlet을 사용합니다. 업데이트하려면 [Azure PowerShell 모듈 설치](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)를 참조하세요. 자세한 내용은 [Azure PowerShell을 설치하고 구성하는 방법을](https://docs.microsoft.com/powershell/azure/azurerm/overview)참조하세요.

macOS에서 PowerShell을 사용하는 경우 [macOS에서 PowerShell 코어 설치](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6) 단계를 따릅니다.
