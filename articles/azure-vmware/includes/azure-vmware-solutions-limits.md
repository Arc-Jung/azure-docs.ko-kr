다음 표에서는 Azure VMware 솔루션에 대 한 최대 제한 사항을 설명 합니다.

| **리소스** | **제한** |
| --- | --- |
| 사설 클라우드 당 클러스터 | 4 |
| 클러스터 당 최소 노드 수 | 3 |
| 클러스터당 최대 노드 수 | 16 |
| 사설 클라우드 당 노드 수 | 64 |
|  사설 클라우드 당 vCenter | 1  |
| HCX 사이트 쌍 | 3 with Advanced edition, 10 Enterprise edition |
| Express 경로 제한 | 울트라 게이트웨이를 사용 하는 경우 4 SDDCs, 10 SDDCs |
| VWAN을 통해 노출 되는 공용 Ip | 256 |
|  vSAN 용량 제한 | 사용 가능한 총 75% (SLA에 25% 사용 가능)  |
