Microsoft는 사설 클라우드에서 VMware 소프트웨어 (ESXi, vCenter, PSC 및 NXS)의 수명 주기 관리를 담당 합니다.

소프트웨어 업데이트는 다음과 같습니다.

- **패치** -VMware에서 릴리스된 보안 패치 또는 버그 수정
- **업데이트** -VMware stack 구성 요소의 부 버전 변경
- **업그레이드** -VMware stack 구성 요소의 주요 버전 변경

Microsoft는 VMware에서 사용할 수 있게 되는 즉시 중요 한 보안 패치를 테스트 합니다.

문서화 된 VMware 해결 방법은 다음에 예약 된 업데이트가 배포 될 때까지 해당 패치를 설치 하는 대신 구현 됩니다.
