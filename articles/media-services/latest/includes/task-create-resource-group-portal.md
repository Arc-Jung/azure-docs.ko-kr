## <a name="create-a-resource-group"></a>리소스 그룹 만들기

1. Azure Portal의 홈 화면에서 **리소스 만들기** 를 선택합니다. Marketplace 화면이 표시됩니다.
1. **리소스 그룹** 을 선택합니다.  리소스 그룹 목록이 표시됩니다.
1. **추가** 를 선택합니다. 리소스 그룹 만들기 화면이 표시됩니다.
1. 이 리소스 그룹에 사용할 구독을 선택합니다.
1. **리소스 그룹** 필드에 리소스 그룹 이름을 입력합니다.
1. 리소스 그룹에 대한 **지역** 을 선택합니다.
1. **검토 + 만들기** 를 선택합니다.
