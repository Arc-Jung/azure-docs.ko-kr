## <a name="create-a-key-vault"></a>키 자격 증명 모음 만들기

1. Azure Portal의 홈 화면에서 **리소스 만들기** 를 선택합니다.
1. **Marketplace** 검색 필드에 *Key Vault* 를 입력하고 검색 결과에 표시되면 **Key Vault** 를 선택합니다.
1. **만들기** 를 선택합니다.  키 자격 증명 모음 만들기 화면이 나타납니다.
1. 사용하려는 **리소스 그룹** 을 선택하거나 새 리소스 그룹을 만듭니다.
1. **Key Vault** 필드에 입력하여 Key Vault에 이름을 지정합니다.
1. 이 예제에서는 **복구 옵션** 에 대한 기본 설정을 그대로 둡니다.
1. 완료되면 **다음: 액세스 정책 >** 을 선택합니다. 액세스 정책 화면이 표시됩니다.
1. 키 자격 증명 모음에 대해 나열된 사용자에게 충분한 권한을 부여합니다.  기본 권한은 충분해야 합니다.
1. **다음: 네트워킹** 을 선택합니다. 네트워킹 화면이 표시됩니다.
1. 사용할 엔드포인트 유형을 선택합니다.
1. **검토 + 만들기** 를 클릭합니다.
