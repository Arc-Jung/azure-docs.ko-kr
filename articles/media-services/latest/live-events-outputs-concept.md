# <a name="live-events-and-live-outputs"></a>라이브 이벤트 및 라이브 출력

Azure Media Services를 사용하면 Azure 클라우드에서 고객에게 라이브 이벤트를 전달할 수 있습니다. Media Services v3에서 라이브 스트리밍 이벤트를 구성하려면 [라이브 이벤트](https://docs.microsoft.com/rest/api/media/liveevents) 및 [라이브 출력](https://docs.microsoft.com/rest/api/media/liveoutputs)에 대해 알고 있어야 합니다.
