# <a name="customizing-content-models-in-video-indexer"></a>Video Indexer에서 사용자 지정 콘텐츠 모델

Video Indexer를 사용 하 여 특정 사용 사례에 적응 해야 해당 모델의 일부를 사용자 지정할 수 있습니다. 이러한 모델에 포함 됩니다 [브랜드](customize-brands-model-overview.md)를 [언어](customize-language-model-overview.md), 및 [person](customize-person-model-overview.md)합니다. Video Indexer 웹 사이트 또는 API를 사용 하 여 이러한 모델을 쉽게 지정할 수 있습니다.

이 문서에서는 각 유형의 사용자 지정의 이점을 설명 하는 개념 문서에 대 한 링크를 제공 합니다. 이 문서는 또한 각 모델의 사용자 지정을 구현 하는 방법을 보여 주는 방법 가이드에 연결 됩니다.
