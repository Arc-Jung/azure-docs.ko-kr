# <a name="customize-a-person-model-with-the-video-indexer-website"></a>Video Indexer 웹 사이트를 사용하여 개인 모델 사용자 지정

비디오 인덱서는 비디오 콘텐츠에 대한 유명 인사 인식을 지원합니다. 유명인 인식 기능은 IMDB, Wikipedia, 상위 LinkedIn 영향력 행사자 등 일반적으로 요청되는 데이터 원본을 기준으로 하는 약 백만 개의 얼굴을 처리합니다. 자세한 개요는 비디오 [인덱서의 사람 모델 사용자 지정을](customize-person-model-overview.md)참조하세요.

Video Indexer 웹 사이트를 사용하여 이 항목에 설명된 대로 비디오에서 감지된 얼굴을 편집할 수 있습니다. [API를 사용하여 사람 모델 사용자 지정에](customize-person-model-with-api.md)설명된 대로 API를 사용할 수도 있습니다.

## <a name="central-management-of-person-models-in-your-account"></a>계정의 사람 모델 중앙 관리

1. 계정에서 사람 모델을 보고, 편집하고, 삭제하려면 비디오 인덱서 웹 사이트를 찾아 로그인합니다.

2. 페이지의 오른쪽 상단 모서리에 있는 콘텐츠 모델 사용자 지정 단추를 선택합니다.

    ![콘텐츠 모델 사용자 지정](./media/customize-face-model/content-model-customization.png)

3. 피플 탭을 선택합니다.

    계정에 기본 사용자 모델이 표시됩니다. 기본 사람 모델에는 인덱싱 중에 사용자 지정 Person 모델을 지정하지 않은 동영상의 인사이트에서 편집하거나 변경한 얼굴이 있습니다.

    다른 사람 모델을 만든 경우 이 페이지에도 나열됩니다.

    ![콘텐츠 모델 사용자 지정](./media/customize-face-model/content-model-customization-people-tab.png)

## <a name="create-a-new-person-model"></a>새 개인 모델 만들기

1. + **모델 추가** 단추를 선택합니다.

    ![새 사람 모델 추가](./media/customize-face-model/add-new-person.png)

2. 모델 이름을 입력하고 이름 옆에 있는 확인 단추를 선택합니다.

    ![새 사람 모델 추가](./media/customize-face-model/add-new-person2.png)

    새 사람 모델을 만들었습니다. 이제 새 사람 모델에 면을 추가할 수 있습니다.

3. 목록 메뉴 단추를 선택하고 **+ 사람 추가를**선택합니다.

    ![새 사람 모델 추가](./media/customize-face-model/add-new-person3.png)

## <a name="add-a-new-person-to-a-person-model"></a>사람 모델에 새 사람 추가

> [!NOTE]
> 비디오 인덱서를 사용하면 사람 모델에서 이름이 같은 여러 사람을 추가할 수 있습니다. 그러나 유용성과 명확성을 위해 모델의 각 사용자에게 고유한 이름을 지정하는 것이 좋습니다.

1. 사람 모델에 새 면을 추가하려면 얼굴을 추가할 사람 모델 옆의 목록 메뉴 단추를 선택합니다.

1. + 메뉴에서 **사람 추가를** 선택합니다.

    ![새 얼굴 추가](./media/customize-face-model/add-new-face.png)

    팝업을 통해 사용자의 세부 정보를 입력하라는 메시지가 표시됩니다. 사람의 이름을 입력하고 확인 단추를 선택합니다.

    ![새 얼굴 추가](./media/customize-face-model/add-new-face2.png)

    그런 다음 파일 탐색기에서 선택하거나 얼굴의 얼굴 이미지를 드래그 앤 드롭할 수 있습니다. 비디오 인덱서에서는 모든 표준 이미지 파일 유형(예: JPG, PNG 등)을 사용합니다.

    비디오 인덱서에서는 이 새 얼굴을 추가한 사람 모델을 사용하여 인덱싱한 향후 동영상과 이미 인덱싱한 현재 비디오에서 이 사람의 발생을 감지할 수 있습니다. 현재 동영상에 있는 사람을 인식하는 것은 일괄 처리 프로세스이기 때문에 적용되는 데 다소 시간이 걸릴 수 있습니다.

## <a name="rename-a-person-model"></a>사람 모델 이름 바꾸기

기본 사람 모델을 포함하여 계정의 모든 사람 모델이름을 바꿀 수 있습니다. 기본 Person 모델의 이름을 바꾸더라도 계정의 기본 사용자 모델로 계속 사용됩니다.

1. 이름을 바꿀 사람 모델 옆의 목록 메뉴 단추를 선택합니다.
2. 메뉴에서 **이름 바꾸기**를 선택합니다.

    ![사람 모델 이름 바꾸기](./media/customize-face-model/rename-person.png)

3. 모델의 현재 이름을 선택하고 새 이름을 입력합니다.

    ![사람 모델 이름 바꾸기](./media/customize-face-model/rename-person2.png)

4. 모델의 이름을 바꿀 검사 단추를 선택합니다.

## <a name="delete-a-person-model"></a>개인 모델 삭제

계정에서 만든 모든 사람 모델을 삭제할 수 있습니다. 그러나 기본 사용자 모델을 삭제할 수는 없습니다.

1. 메뉴에서 **삭제**를 선택합니다.

    ![사람 모델 삭제](./media/customize-face-model/delete-person.png)

    팝업이 표시되고 이 작업이 사람 모델과 포함된 모든 사람 및 파일을 삭제한다는 것을 알려줍니다. 이 작업은 취소할 수 없습니다.

    ![사람 모델 삭제](./media/customize-face-model/delete-person2.png)

1. 확실한 경우 삭제를 다시 선택합니다.

> [!NOTE]
> 이(현재 삭제된) Person 모델을 사용하여 색인이 생성된 기존 동영상은 동영상에 나타나는 얼굴이름을 업데이트하는 기능을 지원하지 않습니다. 다른 사람 모델을 사용하여 얼굴의 인덱싱을 다시 한 번 받은 후에만 이러한 동영상의 얼굴 이름을 편집할 수 있습니다. Person 모델을 지정하지 않고 다시 인덱싱하면 기본 모델이 사용됩니다.

## <a name="manage-existing-people-in-a-person-model"></a>사람 모델에서 기존 인력 관리

Person 모델의 내용을 확인하려면 사람 모델 의 이름 옆에 있는 화살표를 선택합니다. 드롭다운은 특정 사람 모델의 모든 사람을 표시합니다. 각 사람 옆에 있는 목록 메뉴 단추를 선택하면 관리, 이름 바꾸기 및 삭제 옵션이 표시됩니다.  

![새 얼굴 추가](./media/customize-face-model/manage-people.png)

### <a name="rename-a-person"></a>사용자 이름 바꾸기

1. 사람 모델에서 사용자의 이름을 변경하려면 목록 메뉴 단추를 선택하고 목록 메뉴에서 **이름 바꾸기를** 선택합니다.
1. 사용자의 현재 이름을 선택하고 새 이름을 입력합니다.
1. 확인 버튼을 선택하면 이름이 바뀝니다.

### <a name="delete-a-person"></a>사람 삭제

1. 사람 모델에서 사람을 삭제하려면 목록 메뉴 단추를 선택하고 목록 메뉴에서 **삭제를** 선택합니다.
1. 팝업은 이 작업이 사람을 삭제하고 이 작업을 취소할 수 없다는 것을 알려줍니다.
1. 다시 **삭제를** 선택하면 사람 모델에서 사람이 제거됩니다.

### <a name="manage-a-person"></a>사람 관리

**관리를**선택하면 이 사람 모델이 학습중인 모든 면이 표시됩니다. 이러한 얼굴은 이 사람 모델을 사용하는 동영상이나 수동으로 업로드한 이미지에서 해당 사용자의 발생에서 비롯됩니다.

**이미지 추가를**선택하여 다른 사람에게 더 많은 얼굴을 추가할 수 있습니다.

관리 창을 사용하여 사용자 이름을 변경하고 사람 모델에서 사람을 삭제할 수 있습니다.

## <a name="use-a-person-model-to-index-a-video"></a>사람 모델을 사용하여 비디오 인덱싱

사람 모델을 사용하여 동영상을 업로드하는 동안 사람 모델을 할당하여 새 동영상을 색인을 생성할 수 있습니다.

새 동영상에서 사람 모델을 사용하려면 다음 단계를 수행합니다.

1. 페이지 상단의 **업로드** 버튼을 선택합니다.

    ![사람 모델 업로드](./media/customize-face-model/upload.png)

1. 비디오 파일을 원에 놓거나 파일을 찾아봅을 올립니다.
1. 고급 **옵션** 화살표를 선택합니다.

    ![사람 모델 업로드](./media/customize-face-model/upload2.png)

1. 드롭다운을 선택하고 작성한 사람 모델을 선택합니다.

    ![사람 모델 업로드](./media/customize-face-model/upload3.png)

1. 페이지 하단에서 **업로드** 옵션을 선택하면 사람 모델을 사용하여 새 동영상이 색인이 생성됩니다.

업로드 하는 동안 사람 모델을 지정 하지 않으면 비디오 인덱서 계정의 기본 사람 모델을 사용 하 여 동영상을 인덱싱 합니다.

## <a name="use-a-person-model-to-reindex-a-video"></a>사람 모델을 사용하여 비디오 인덱싱

사람 모델을 사용하여 컬렉션에서 동영상의 인덱싱을 다시 하려면 비디오 인덱서 홈 페이지의 계정 비디오로 이동하여 다시 인덱싱할 동영상의 이름을 마우스로 가리킵니다.

비디오를 편집, 삭제 및 다시 인덱싱하는 옵션이 표시됩니다.

1. 비디오 인덱싱 옵션을 선택합니다.

    ![사람 모델을 사용하여 비디오 인덱싱](./media/customize-face-model/reindex.png)

    이제 사람 모델을 선택하여 비디오를 다시 인덱싱할 수 있습니다.
1. 드롭다운을 선택하고 사용할 사람 모델을 선택합니다.

    ![사람 모델을 사용하여 비디오 인덱싱](./media/customize-face-model/reindex2.png)

1. 다시 **인덱싱** 버튼을 선택하면 사람 모델을 사용하여 비디오가 다시 인덱싱됩니다.

방금 다시 인덱싱한 비디오에서 감지되고 인식된 얼굴에 대한 새 편집은 비디오를 다시 인덱싱하는 데 사용한 사람 모델에 저장됩니다.

## <a name="managing-people-in-your-videos"></a>동영상의 사용자 관리

감지된 얼굴과 동영상에서 인식된 얼굴을 편집하고 삭제하여 인덱싱한 사람을 관리할 수 있습니다.

얼굴을 삭제면 비디오의 인사이트에서 특정 얼굴이 제거됩니다.

얼굴을 편집하면 동영상에서 감지되어 인식될 가능성이 있는 얼굴의 이름이 바바오게 됩니다. 동영상에서 얼굴을 편집하면 해당 이름은 업로드 및 색인 생성 중에 동영상에 할당된 사람 모델의 사람 항목으로 저장됩니다.

업로드하는 동안 동영상에 사람 모델을 할당하지 않으면 계정의 기본 사람 모델에 편집이 저장됩니다.

### <a name="edit-a-face"></a>얼굴 편집

> [!NOTE]
> Person 모델에 이름이 같은 사람이 두 명 이상인 경우 해당 Person 모델을 사용하는 동영상 내에서 해당 이름에 태그를 지정할 수 없습니다. 비디오 인덱서의 콘텐츠 모델 사용자 지정 페이지의 피플 탭에서 해당 이름을 공유하는 사용자만 변경할 수 있습니다. 따라서 Person 모델의 각 사용자에게 고유한 이름을 지정하는 것이 좋습니다.

1. Video Indexer 웹 사이트로 이동하고 로그인합니다.
1. 계정에서 보고 편집하려는 비디오를 검색합니다.
1. 동영상에서 얼굴을 편집하려면 인사이트 탭으로 이동하여 창의 오른쪽 상단 모서리에 있는 연필 아이콘을 선택합니다.

    ![동영상에서 얼굴 편집](./media/customize-face-model/edit-face.png)

1. 감지된 면 중 어느 한 면을 선택하고 이름을 "알 수 없는 #X"(또는 이전에 면에 할당한 이름)에서 변경합니다.
1. 새 이름을 입력한 후 새 이름 옆에 있는 확인 아이콘을 선택합니다. 이 작업은 새 이름을 저장하고 다른 현재 동영상및 업로드한 향후 동영상에서 이 얼굴의 모든 발생을 인식하고 이름을 지정합니다. 다른 현재 비디오의 얼굴 인식은 배치 프로세스이므로 적용되는 데 다소 시간이 걸릴 수 있습니다.

비디오가 사용하는 Person 모델에서 기존 사람의 이름으로 얼굴이름을 지정하면 해당 사용자의 이 비디오에서 감지된 얼굴 이미지가 모델에 이미 있는 얼굴 이미지와 병합됩니다. 새 이름의 얼굴 이름을 지정하면 동영상에서 사용하는 Person 모델에서 새 사람 항목이 만들어집니다.

![동영상에서 얼굴 편집](./media/customize-face-model/edit-face2.png)

### <a name="delete-a-face"></a>얼굴 삭제

동영상에서 감지된 얼굴을 삭제하려면 Insights 창으로 이동하여 창의 오른쪽 상단 모서리에 있는 연필 아이콘을 선택합니다. 면 이름 아래에 **있는 삭제** 옵션을 선택합니다. 이 작업을 수행하면 비디오에서 감지된 얼굴이 제거됩니다. 사람의 얼굴은 표시되는 다른 동영상에서 여전히 감지되지만 색인이 생성된 후에도 해당 동영상에서 얼굴을 삭제할 수 있습니다.

이름이 지정된 사람인 경우 Person 모델에서 해당 사람을 특별히 삭제하지 않는 한 얼굴을 삭제한 비디오를 인덱싱하는 데 사용된 사람 모델에도 계속 존재합니다.

![동영상에서 얼굴 삭제](./media/customize-face-model/delete-face.png)
