# <a name="search-for-exact-moments-in-videos-with-video-indexer"></a>Video Indexer를 사용 하 여 비디오에서 정확한 순간 검색

이 항목에서는 비디오에서 정확한 순간을 검색할 수 있는 검색 옵션을 보여 줍니다.

1. [Video Indexer](https://www.videoindexer.ai/) 웹 사이트로 이동 하 여 로그인 합니다.
2. 계정에 있는 모든 비디오 중에서 검색합니다.

    다음 예제에서는 보안에 대해 설명 하 고 Satya 표시 되는 모든 비디오를 검색 합니다.

    ![Video Indexer에서 비디오 검색](./media/video-indexer-search/video-indexer-search01.png)

3. 비디오의 요약된 인사이트를 검색합니다.

    비디오에서 **재생** 을 선택 하 여 비디오를 검색할 수 있습니다. 그런 다음 **검색** 탭을 선택 하 여 비디오에서 정확한 순간을 검색할 수 있습니다.

    다음 예제에서는 선택한 비디오 내에서 "보안"을 검색 합니다.

    ![Video Indexer를 사용 하 여 비디오 검색](./media/video-indexer-search/video-indexer-search02.png)

    결과 중 하나를 선택 하면 플레이어는 비디오에서 정확한 순간을 표시 합니다. 앱에서 플레이어/정보 보기 및 동기화를 달성할 수 있습니다. 자세한 내용은 [앱에 Video Indexer 위젯 포함](video-indexer-embed-widgets.md)을 참조 하세요.

4. 비디오에 대한 자세한 분석을 검색합니다.

    찾은 비디오를 기준으로 클립을 직접 만들려면 **편집** 단추를 선택 합니다. 이 페이지에서는 해당 정보를 필터로 사용 하 여 비디오를 표시 합니다. 자세한 내용은 [Video Indexer 인사이트 보기 및 편집](video-indexer-view-edit.md)을 참조하세요.

    비디오 내에서 정확한 순간을 검색 하 여 관심 있는 줄만 표시할 수 있습니다. 사이드 insights를 사용 하 여 보려는 파트를 필터링 합니다. 완료 되 면 클립을 미리 보고 **게시** 를 선택 하 여 갤러리에 표시 되는 새 클립을 만들 수 있습니다.

    다음 예제에서는 "mixed reality" 텍스트를 검색 했습니다. 또한 아래 화면과 같이 추가 필터도 적용했습니다.

    ![Video Indexer에서 정확한 순간 검색](./media/video-indexer-search/video-indexer-search03.png)
