# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>IoT용 Azure Certified 포털에 온보딩

[IoT 포털에 대한 Azure 인증 포털을](https://aka.ms/ACFI) 사용하여 다음을 수행할 수 있습니다.

- IoT 장치에 대한 인증 프로세스를 완료합니다.
- IoT 플러그 앤 플레이 디바이스 기능 모델을 찾아보십시오. 이러한 모델을 사용하여 IoT 지원 장치를 신속하게 구축하고 솔루션과 통합할 수 있습니다.
- 장치 기능 모델을 공용 리포지토리에 게시합니다.

## <a name="sign-in"></a>로그인

Microsoft _회사 계정을_ 사용하여 [IoT 포털에 대한 Azure 인증 포털에](https://aka.ms/ACFI) 로그인합니다. 포털은 로그인할 때 Microsoft 파트너 센터의 멤버십을 확인합니다.

- 조직이 이미 [Microsoft 파트너 센터의](https://docs.microsoft.com/partner-center/)구성원인 경우 포털의 모든 기능에 대한 액세스 권한이 부여됩니다.
- 조직에서 [Microsoft 파트너 센터의](https://docs.microsoft.com/partner-center/)구성원이 아닌 경우 인증 테스트를 실행하고 장치 기능 모델 리포지토리를 찾아볼 수 있습니다. IoT 인증 장치 카탈로그에 장치를 게시할 수 없습니다.
- Microsoft 파트너 네트워크 ID가 누락되었다는 알림 메시지가 표시되면 [파트너 센터 계정을 만들어야](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account)합니다. 계정이 승인되면 포털로 돌아가서 기기를 게시할 수 있습니다. 자세한 내용은 파트너 [센터 FAQ를](https://support.microsoft.com/help/4340639/partner-center-account-faqs)참조하십시오.

장치를 IoT 인증 장치 카탈로그에 게시하기 전에 프로그램 계약을 읽고 수락하십시오. IoT에 대한 Azure 인증 포털의 설정 페이지에서 계약을 찾을 수 있습니다.
