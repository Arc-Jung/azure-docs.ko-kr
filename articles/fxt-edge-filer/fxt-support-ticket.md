# <a name="open-a-support-ticket-for-the-azure-fxt-edge-filer"></a>Azure FXT Edge 필터가 대 한 지원 티켓 열기

이 문서에서는 Azure FXT Edge 필터가 시스템을 사용 하 여 도움을 얻는 방법을 설명 합니다.

Azure 포털을 사용 하 여 Microsoft 고객 서비스 및 지원 센터를 사용 하 여 지원 티켓을 엽니다.

1. 엽니다는 [Azure portal](https://portal.azure.com/) 에 로그인 합니다.
1. 왼쪽된 패널에서 아래쪽으로 스크롤하여 클릭 **도움말 + 지원**합니다.
1. **새 지원 요청**을 클릭합니다. 

   ![새 지원 요청 양식 스크린샷](media/fxt-support-blank.png)

1. 요청 양식을 작성 하세요.  
    1. 에 **기본 사항** 탭을 선택 합니다 **문제 유형** ``Technical``합니다. 
    1. 구독을 선택합니다. 
    1. 아래 **서비스**를 선택 **Azure FXT Edge 필터가**합니다. 클릭 해야 하는 **모든 서비스** 찾을 컨트롤을 필터링 합니다. 
    1. 입력을 **요약** 선택 합니다 **문제 유형**합니다. 
    ![새 지원 요청 양식 스크린샷](media/fxt-support-populated.png) 
    1. 계속하려면 **다음** 을 클릭합니다. 
1. 에 **세부 정보** 탭, 문제 및 사용자에 게 연락 하는 방법에 대 한 자세한 정보를 추가 합니다. 문제와 관련 된 파일을 업로드할 수 있습니다. 
1. **검토 + 만들기**정보를 확인 하 고 클릭 합니다 **만들기** 단추 요청을 제출 합니다.

요청을 제출 하면 티켓 번호를 사용 하 여 확인 전자 메일을 받게 됩니다. 지원 담당자가 연락을 요청에 대 한 합니다.
