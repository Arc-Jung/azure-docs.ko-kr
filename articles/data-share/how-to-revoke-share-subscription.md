# <a name="how-to-revoke-a-consumers-share-subscription-in-azure-data-share"></a>Azure 데이터 공유에서 소비자의 공유 구독을 해지하는 방법

이 문서에서는 Azure 데이터 공유를 사용하여 하나 이상의 소비자로부터 공유 구독을 해지하는 방법을 설명합니다. 이렇게 하면 소비자가 더 이상 스냅숏을 트리거할 수 없습니다. 소비자가 아직 스냅숏을 트리거하지 않은 경우 공유 구독이 해지되면 데이터를 받지 않습니다. 이전에 스냅숏을 트리거한 경우 가지고 있는 최신 데이터가 계정에 유지됩니다.

## <a name="navigate-to-a-sent-data-share"></a>보낸 데이터 공유로 이동

Azure 데이터 공유에서 보낸 공유로 이동하여 **구독 공유 탭을 선택합니다.**

![공유 구독 해지](./media/how-to/how-to-revoke-share-subscription/revoke-share-subscription.png) 

삭제하려는 공유 구독을 수신자 옆에 있는 확인란을 확인한 다음 **해지를**클릭합니다. 소비자는 더 이상 데이터에 대한 업데이트를 받지 않습니다.
