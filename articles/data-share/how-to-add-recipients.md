# <a name="how-to-add-a-recipient-to-an-existing-data-share-in-azure-data-share"></a>Azure 데이터 공유의 기존 데이터 공유에 받는 사람을 추가하는 방법

이 문서에서는 기존 데이터 공유에 받는 사람을 추가하는 방법을 보여 주며, 받는 사람을 추가하는 방법을 보여 줄 것입니다.

공유를 만들 때 받는 사람을 추가하는 방법에 대한 자세한 내용은 [데이터 공유](share-your-data.md) 자습서를 참조하세요.

## <a name="navigate-to-a-sent-data-share"></a>보낸 데이터 공유로 이동

Azure 데이터 공유에서 보낸 공유로 이동하여 **초대 탭을** 선택합니다. 새 것을 추가하려면 **받는 사람 추가**를 클릭합니다.

![받는 사람 추가](./media/how-to/how-to-add-recipients/add-recipient.png)

패널이 페이지의 오른쪽에 나타납니다. **받는 사람 추가를** 클릭한 다음 빈줄에 새 받는 사람의 전자 메일을 입력합니다. 원하는 만큼 받는 사람을 추가합니다.

![받는 사람 추가](./media/how-to/how-to-add-recipients/add-recipient-side.png)

**추가 및 보내기 초대를 클릭합니다.** 새 받는 사람(들)이 이 데이터 공유에 초대 이메일을 전송합니다.
