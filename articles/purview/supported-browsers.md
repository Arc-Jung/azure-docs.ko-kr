# <a name="supported-browsers"></a>지원되는 브라우저 

Azure 부서의 범위는 다음 브라우저를 지원 합니다. 운영 체제와 호환되는 가장 최신 브라우저를 사용하는 것이 좋습니다. 

* Microsoft Edge(최신 버전)
* Safari(최신 버전, Mac만 해당)
* Chrome(최신 버전)
* Firefox(최신 버전)

## <a name="chrome-incognito-mode"></a>Chrome Incognito 모드

 부서의 범위 Studio가 작동 하려면 Chrome Incognito 차단 중인 타사 쿠키를 사용 하지 않도록 설정 해야 합니다.

:::image type="content" source="./media/supported-browsers/incognito-chrome.png" alt-text="Chrome을 보여 주는 스크린샷":::

## <a name="chromium-edge-inprivate-mode"></a>Chromium Edge InPrivate 모드

부서의 범위 Studio가 작동 하려면 엄격한 추적 방지를 사용 하는 Chromium Edge InPrivate를 사용 하지 않도록 설정 해야 합니다.

:::image type="content" source="./media/supported-browsers/incognito-edge.png" alt-text="가장자리를 보여 주는 스크린샷":::
