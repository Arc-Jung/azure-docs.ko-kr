# <a name="access-azure-kinect-dk-microphone-input-data"></a>Azure Kinect 진한 마이크 입력 데이터 액세스

[SPEECH SDK 퀵 스타트](https://docs.microsoft.com/azure/cognitive-services/speech-service) 는 다양 한 프로그래밍 언어에서 AZURE Kinect 진한 마이크 배열을 사용 하는 방법에 대 한 예제를 제공 합니다.
예를 들어 Speech SDK 퀵 스타트를 **사용 하 여 Windows에서 c + +의 음성 인식 기능** 을 참조 하세요. [GitHub에서](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/cpp)코드를 사용할 수 있습니다.

Windows API를 통해서도 마이크 배열에 액세스 합니다. Windows 설명서에 대 한 자세한 내용은 다음 문서를 참조 하세요.

* [Windows 오디오 아키텍처](https://docs.microsoft.com/windows-hardware/drivers/audio/windows-audio-architecture)
* [Windows. Media. 문서 캡처](https://docs.microsoft.com/uwp/api/Windows.Media.Capture)
* [웹캠 캡처에 대 한 자습서](https://docs.microsoft.com/windows/uwp/audio-video-camera/basic-photo-video-and-audio-capture-with-mediacapture)
* [USB 오디오 정보](https://docs.microsoft.com/windows-hardware/drivers/audio/usb-2-0-audio-drivers)

또한 [마이크 배열 하드웨어 사양을](hardware-specification.md#microphone-array)검토할 수 있습니다.
