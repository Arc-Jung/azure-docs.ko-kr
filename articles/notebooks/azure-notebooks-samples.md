# <a name="discover-and-use-sample-notebooks"></a>샘플 Notebook 검색 및 사용

인터넷에는 쉽게 Azure Notebooks로 가져올 수 있는 흥미롭고 유용한 여러 Jupyter Notebook 컬렉션이 있습니다. 이 문서에서는 주목할 만한 샘플 목록을 제공합니다.

살펴보고 싶은 Notebook을 찾았으면 다음과 같은 몇 가지 세부 정보를 이해해야 합니다.

- 이미 Azure Notebooks에 Notebook이 있는 경우 프로젝트 대시보드의 **복제** 단추를 사용하여 사용자 계정에 프로젝트 복사본을 만듭니다.
- 일부 Notebook은 특정 패키지를 실행해야 합니다. `!pip install`(Python) 또는 `install.packages`(R)를 사용하여 Notebook 자체에서 패키지를 설치할 수도 있지만, [적절한 환경을 가진 Azure Notebooks 프로젝트를 구성](configure-manage-azure-notebooks-projects.md)할 수도 있습니다.
