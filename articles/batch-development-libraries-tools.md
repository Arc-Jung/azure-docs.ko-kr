# Azure 배치 개발 라이브러리 및 도구 
<p> 이러한 라이브러리 및 Azure 일괄 처리 및 일괄 처리 응용 프로그램의 응용 프로그램을 개발 하는데 필요한 도구를 얻습니다.

## 배치

+ [.NET 용 클라이언트 라이브러리를 일괄 처리](http://www.nuget.org/packages/Azure.Batch/) (NuGet)-일괄 처리 서비스와 작업을 실행 하는 클라이언트 응용 프로그램 개발
+ [일괄 처리 관리 라이브러리](http://www.nuget.org/packages/Microsoft.Azure.Management.Batch/) (NuGet)-일괄 처리 관리 서비스 계정
+ [일괄 처리 탐색기](https://code.msdn.microsoft.com/windowsazure/Azure-Batch-Explorer-c1d37768) (MSDN) – GUI 응용 프로그램 및 샘플 찾아보기, 액세스 및 작업 및 작업, 작업 가상 컴퓨터 (TVMs) 및 풀 및는 TVM에 파일을 포함 하는 일괄 처리 계정 내에서 주요 요소를 업데이트 합니다.

> [AZURE.TIP]일괄 처리 탐색기는 모니터링 하거나 일괄 처리 작업의 문제를 해결 하려면 하거나 새 일괄 처리 하는 경우 좋은 도구입니다. 코드 샘플 이기 때문에 소스 코드를 보면 기능은 어떻게 일괄 처리.NET API를 사용 하 여 구현 합니다. 참조는 [블로그 게시물](http://blogs.technet.com/b/windowshpc/archive/2015/01/20/azure-batch-explorer-sample-walkthrough.aspx).

## 배치 앱

+ [일괄 처리 앱 클라우드 SDK](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps.Cloud/) (NuGet) – 응용 프로그램을 일괄 처리 서비스에 대 한 작업 오프 로드를 사용 하도록 설정
+ [일괄 처리 응용 프로그램 Visual Studio 확장](https://visualstudiogallery.msdn.microsoft.com/8b294850-a0a5-43b0-acde-57a07f17826a) (Visual Studio 갤러리)-일괄 처리 앱 클라우드 SDK를 사용 하 여 Visual Studio에서 클라우드 사용 응용 프로그램
+ [일괄 처리 응용 프로그램 클라이언트 SDK](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps/) (NuGet)-일괄 처리 서비스에 대 한 작업 오프 로드를 사용 하도록 설정 하는 응용 프로그램과 함께 의견을 나눠보십시오
+ [일괄 처리 앱 Python 클라이언트](https://github.com/Azure/azure-batch-apps-python) (GitHub)-일괄 처리 응용 프로그램 서비스에서 설정 된 응용 프로그램과 상호작용 하는 Python 클라이언트 모듈
+ [일괄 처리 응용 프로그램 Blender 샘플](https://github.com/Azure/azure-batch-apps-blender) (GitHub)-Addon Blender 공개 소스 렌더링 소프트웨어 렌더링 클라우드 기반 플랫폼을 설정 하는 일괄 처리 응용 프로그램 SDK 및 Python 클라이언트를 사용 하는
