# <a name="customize-your-azure-security-center-for-iot-solution"></a>IoT 솔루션을 위한 Azure 보안 센터 사용자 지정 

이 가이드에서는 IoT용 Azure 보안 센터에서 다양한 설정을 사용자 지정하는 방법을 알아봅니다.  

> [!div class="checklist"]
> * 솔루션 권장 사항 구성 
> * 설정 변경 

## <a name="change-settings"></a>설정 변경

IoT에 대한 Azure 보안 센터 관리:

IoT Hub에서 왼쪽 상단모서리의 보안 개요 블레이드로 이동하여 "설정"을 참조하여 IoT 설정에 대한 Azure 보안 센터를 구성하려면 다음을 수행합니다.

1. Azure Portal에서 **IoT Hub**를 엽니다. 
1. **보안**아래의 왼쪽 메뉴에서 **개요를**선택하고 엽니다.
1. 설정에서 변경하려는 솔루션 설정을 선택합니다.   
1. 항상 설정 화면 상단의 **저장을** 클릭하여 설정 변경 사항을 저장해야 합니다. 

## <a name="configure-solution-recommendations"></a>솔루션 권장 사항 구성

IoT 솔루션 권장 사항에 대한 Azure 보안 센터를 구성하려면 다음을 수행합니다.

1. Azure Portal에서 **IoT Hub**를 엽니다. 
1. 왼쪽 메뉴에서 **보안** 아래에서 개요를 선택하고 **엽니다.**
1. 설정에서 왼쪽 메뉴에서 **권장 구성을** 선택합니다. 
1. 조직 및 워크플로와 관련된 솔루션 권장 사항을 사용하지 않도록 설정하거나 사용하도록 설정합니다. 
1. 화면 상단에 **있는 저장을** 선택하여 선택 항목을 저장합니다.
