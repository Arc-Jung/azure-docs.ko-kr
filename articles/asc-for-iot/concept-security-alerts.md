# <a name="azure-security-center-for-iot-security-alerts"></a>IoT용 Azure Security Center 보안 경고

IoT용 Azure 보안 센터는 고급 분석 및 위협 인텔리전스를 사용하여 IoT 솔루션을 지속적으로 분석하여 악의적인 활동을 경고합니다.
또한 예상 된 장치 동작에 대 한 지식에 따라 사용자 지정 경고를 만들 수 있습니다.
경고는 잠재적인 손상의 지표로 작동하며 조사및 수정해야 합니다.

이 문서에서는 IoT Hub 및/또는 IoT 장치에서 트리거할 수 있는 기본 제공 경고 목록을 찾을 수 있습니다.
기본 제공 경고 외에도 IoT용 Azure 보안 센터를 사용하면 예상되는 IoT Hub 및/또는 장치 동작을 기반으로 사용자 지정 경고를 정의할 수 있습니다.
자세한 내용은 [사용자 지정 가능한 경고를](concept-customizable-security-alerts.md)참조하십시오.
