# <a name="introducing-azure-security-center-for-iot"></a>IoT용 Azure Security Center 소개

하이브리드 클라우드 워크로드 및 Azure IoT 솔루션 전반에 걸쳐 보안 관리를 통합하고 종단 간 위협 탐지 및 분석을 지원합니다. 

## <a name="secure-your-entire-iot-solution-from-iot-devices-to-azure-cloud"></a>IoT 디바이스에서 Azure 클라우드로 전체 IoT 솔루션 보호

원활한 에이전트 리스 솔루션 중에서 선택하거나 에이전트 기반의 포괄적인 보안을 활용하는 IoT용 Azure Security Center는 IoT 자산 전반에 걸쳐 모든 디바이스, IoT Edge 및 IoT Hub에 대한 위협 예방 및 분석을 제공합니다.

수십억 개의 새로운 장치가 인터넷에 연결되어 일상 생활과 비즈니스에 통합됨에 따라 보안 운영 팀은 새로운 공격 표면을 다룰 수 있을 만큼 보안 전략이 빠르게 진화하도록 보장해야 합니다. 다른 시스템과 마찬가지로 IoT 솔루션을 포괄적으로 보호하려면 구현의 모든 단계에서 보호가 필요합니다. 

IoT용 Azure 보안 센터는 에지, 온-프레미스, Azure 및 기타 에서 실행되는 워크로드 전반에 걸쳐 통합 된 가시성 및 제어, 적응형 위협 예방 및 지능형 위협 탐지 및 대응을 제공하여 하이브리드 워크로드 보호를 간소화합니다. 구름. 

![IoT용 Azure Security Center 아키텍처](./media/architecture/azure-iot-security-architecture.png)

### <a name="unified-visibility-and-control"></a>통합된 표시 유형 및 제어

Azure IoT 솔루션을 비롯한 모든 온-프레미스 및 클라우드 워크로드에서 보안에 대한 통합된 보기를 가져옵니다. 새 장치를 온보더하고 워크로드(리프 장치, Microsoft Edge 장치, IoT Hub)에 보안 정책을 적용하여 보안 표준을 준수하고 향상된 보안 상태를 준수할 수 있습니다. 

### <a name="adaptive-threat-prevention"></a>적응형 위협 방지

IoT용 Azure 보안 센터를 사용하여 컴퓨터, 네트워크 및 Azure 서비스의 보안을 지속적으로 모니터링합니다. 수백 개의 기본 제공 보안 평가 중에서 선택하거나 IoT Hub 대시보드용 중앙 Azure 보안 센터에서 직접 만들 수 있습니다. 가상 머신, 네트워크, 앱 및 데이터 전반에서 실행 가능한 권장 사항을 사용하여 보안 설정을 최적화하고 보안 점수를 개선합니다. 새로 추가된 IoT 기능을 사용하면 Azure IoT 솔루션의 공격 표면을 줄이고 문제를 악용하기 전에 문제를 수정할 수 있습니다.

### <a name="intelligent-threat-detection-and-response"></a>지능형 위협 검색 및 대응

고급 분석 및 Microsoft 지능형 보안 그래프를 사용하여 진화하는 사이버 공격에 대한 우위를 확보하십시오. 기본 제공 행동 분석 및 기계 학습은 공격과 제로 데이 익스플로잇을 식별합니다. IoT 솔루션을 모니터링하여 들어오는 공격 및 위반 후 활동이 확인됩니다. 대화형 도구 및 상황별 위협 인텔리전스를 통해 장치 조사 및 수정을 간소화합니다.
