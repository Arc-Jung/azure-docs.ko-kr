# <a name="get-started-with-azure-security-center-for-iot"></a>IoT용 Azure Security Center 시작

이 문서에서는 IoT 서비스에 대한 Azure 보안 센터의 다양한 구성 요소에 대해 설명하고 두 가지 가능한 배포 옵션을 사용하여 서비스를 시작하는 방법을 설명합니다.

## <a name="deployment-options"></a>배포 옵션

IoT 장치 및 환경 요구 사항을 가장 잘 충족하는 서비스 시나리오를 선택합니다.

### <a name="built-in-deployment"></a>기본 제공 배포

원활한 기본 제공 배포 옵션을 사용하여 IoT용 Azure 보안 센터를 IoT 허브에 신속하게 통합하고 IoT 허브 구성, 장치 ID 및 관리, 허브 장치 통신 패턴에 대한 보안 분석을 제공할 수 있습니다.

IoT Hub 모니터링 및 권장 사항을 갖춘 [기본 제공 배포를](iot-hub-integration.md) 시작합니다.
    <br>

### <a name="enhanced-deployment"></a>향상된 배포

향상된 보안 기능을 위해 IoT Hub 보안을 사용하도록 설정하는 것 외에도 IoT 에이전트용 Azure 보안 센터를 배포하면 IoT 장치에서 주요 보안 데이터의 에이전트 기반 이벤트 수집, 분석 및 위협 감지뿐만 아니라 포괄적인 보안 상태 관리 기능을 제공합니다.

에이전트 기반의 포괄적인 위협 보호 및 보안 상태 관리 솔루션을 갖춘 [향상된 배포를](security-agents.md) 시작합니다.
