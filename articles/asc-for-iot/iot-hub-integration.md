# <a name="get-started-with-built-in-iot-hub-integration"></a>내장 된 IoT 허브 통합으로 시작하기 

이 옵션을 사용하면 IoT 보안 에이전트용 Azure 보안 센터를 사용하지 않고도 서비스를 사용할 수 있습니다.

## <a name="enable-built-in-iot-hub-integration"></a>내장 된 IoT 허브 통합 활성화
장치 ID 관리, 디바이스-클라우드 및 클라우드에서 장치 통신 패턴을 모니터링하려면 다음을 수행하여 서비스를 시작합니다. 

1. **IoT 허브를**엽니다.
1. 보안 개요 메뉴를 **선택합니다.** 
1. IoT 솔루션 보안을 클릭하고 온보딩 양식을 **작성합니다.** 

축하합니다! IoT Hub에서 IoT 서비스를 위한 Azure 보안 센터 활성화를 완료했습니다.
