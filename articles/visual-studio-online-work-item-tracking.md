# 웹 사이트의 작업 추적
이 항목에서는 작업 항목 추적 도구를 사용하여 웹 사이트의 작업을 관리하는 방법을 알아봅니다.

* <a href="#how-to-manage-your-backlog">방법: 백로그 관리</a> 
* <a href="#how-to-view-recently-completed-work">방법: 최근에 완료된 작업 보기</a> 

## 방법: 백로그에 작업 항목 추가 
1. 팀 프로젝트 블레이드로 이동합니다. 

2. 아래쪽의 "작업" 렌즈 내부에서 백로그 파트를 클릭합니다.

	백로그 블레이드가 시작됩니다. 백로그에는 웹 사이트에 대한 우선 순위를 매긴 작업 목록이 포함됩니다. 

	![Open backlog](./media/visual-studio-online-work-item-tracking/visual-studio-online-open-backlog.png)

3. 위쪽의 텍스트 상자에 제목을 입력하고 Enter 키를 눌러 항목을 추가합니다.

	스크럼 프로세스 템플릿을 사용하고 있으면 오른쪽 드롭다운에서 버그 또는 제품 백로그 항목을 선택하여 추가할 수 있습니다.

4. 작업 항목을 클릭하여 열고 세부 정보를 추가합니다.

	![Open work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-open-work-item.png) 

5. 작업 항목 양식에서 제목을 편집하고, 설명을 추가하고, 사용자에게 작업 항목을 할당하고, 해당 항목을 완료로 표시합니다. 

6. 작업 항목 양식의 아래쪽에서 작업 항목에 메모를 추가하고 관련 기록을 볼 수 있습니다.

	작업 항목의 기록을 자세히 알아보려면 변경 내용을 클릭하면 됩니다. 그러면 내역에 대한 세부 정보가 포함된 블레이드가 열립니다.

## 방법: 백로그 관리

효율적으로 우선 순위가 지정된 백로그를 유지하는 것은 웹 사이트의 작업을 성공적으로 관리하는 데 핵심입니다. 다음 두 부분이 있습니다. 

### 완료 시 작업 제거 

1. 완료로 표시할 작업 항목을 엽니다. 

2. "완료"로 표시된 확인란을 선택합니다. 

3. 작업 항목을 저장합니다. 

4. 이 항목의 스텁은 백로그를 새로 고치거나 백로그를 닫고 다시 열 때까지 백로그에 남아 있습니다.

![Completed work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-completed-work-item.png) 

### 백로그의 우선 순위 유지 

1. 우선 순위를 변경할 작업 항목을 가리킵니다. 

2. 작업 항목 왼쪽에서 위치 조정 막대를 클릭합니다. 

3. 작업 항목을 백로그의 새 위치로 끌어 놓습니다.

![Dragging work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-dragging-work-item.png) 

## 방법: 최근에 완료된 작업 보기 

1. 팀 프로젝트 블레이드에서 완료된 항목 블레이드를 클릭합니다. 

2. 최근에 완료된 작업 목록을 확인합니다. 

![Dragging work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-completed-items.png)
