# Azure 가상 컴퓨터에서 SQL Server 시작

이 항목은 라이브러리의 [새 위치][새 위치]로 이동했습니다.

플랫폼에서 제공되는 이미지를 사용하여 SQL Server 가상 컴퓨터를 만드는 방법에 대한 자세한 내용은 [Azure에서 SQL Server 가상 컴퓨터 프로비전][Azure에서 SQL Server 가상 컴퓨터 프로비전](영문)을 참조하세요.

Azure 가상 컴퓨터에서 SQL Server를 사용하는 방법에 대한 최신 정보를 모두 보려면 라이브러리의 [Azure 가상 컴퓨터의 SQL Server][Azure 가상 컴퓨터의 SQL Server]를 참조하세요.
