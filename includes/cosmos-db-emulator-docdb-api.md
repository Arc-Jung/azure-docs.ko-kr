Azure 구독, 비용 및 약정 없이 [Azure Cosmos DB 무료 체험](https://azure.microsoft.com/try/cosmosdb/)을 사용해 보거나, 처음에 400RU/초 및 5GB의 스토리지를 무료로 사용할 수 있는 [Azure Cosmos DB 체험 계층 계정](../articles/cosmos-db/optimize-dev-test.md#azure-cosmos-db-free-tier)을 만들 수 있습니다. 또한 URI가 `https://localhost:8081`인 [Azure Cosmos DB Emulator](https://docs.microsoft.com/azure/cosmos-db/local-emulator)를 사용할 수도 있습니다. 에뮬레이터에서 사용할 키는 [요청 인증](../articles/cosmos-db/local-emulator.md#authenticate-requests)을 참조하세요.
