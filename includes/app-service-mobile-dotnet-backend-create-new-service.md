1. [Azure 포털에]로그인합니다.

2. **리소스 만들기를 클릭합니다.**

3. 검색 상자에 웹 **앱을**입력합니다.
    
4. 결과 목록에서 마켓플레이스에서 **웹 앱을** 선택합니다.

5. **구독** 및 **리소스 그룹을** 선택합니다(기존 리소스 그룹을 _선택하거나 앱과_ 동일한 이름을 사용하여 새 리소스 그룹을 만듭니다).

6. 웹 앱의 고유한 **이름을** 선택합니다.

7. 기본 **게시** 옵션을 **코드로**선택합니다.

8. **런타임 스택에서** **ASP.NET** 또는 **노드**에서 버전을 선택해야 합니다. .NET 백 엔드를 빌드하는 경우 ASP.NET 아래 버전을 선택합니다. 그렇지 않으면 노드 기반 응용 프로그램을 대상으로 하는 경우 노드에서 버전 중 하나를 선택합니다.

9. 리눅스 또는 윈도우 중 오른쪽 **운영 체제를**선택합니다. 

10. 이 앱을 배포할 **지역을** 선택합니다. 

11. 적절한 **앱 서비스 계획을** 선택하고 검토를 **누르고 을 만듭니다.** 

12. **** 리소스 그룹__ 아래에서 기존 리소스 그룹을 선택하거나 새 리소스 그룹을 만듭니다(앱과 동일한 이름 사용).

13. **만들기**를 클릭합니다. 계속 진행하기 전에 서비스가 성공적으로 배포되도록 몇 분 동안 기다립니다. 상태 업데이트를 확인하려면 포털 머리글에 있는 알림(벨) 아이콘을 지켜봅니다.

14. 배포가 완료되면 배포 세부 **정보** 섹션을 클릭한 다음 **Microsoft.Web/사이트**유형의 리소스를 클릭합니다. 방금 만든 앱 서비스 웹 앱으로 이동합니다. 

15. **설정** 및 **응용 프로그램 설정에서** **구성** 블레이드를 클릭하고 새 응용 **프로그램 설정** 버튼을 클릭합니다.

16. **추가/편집 응용 프로그램 설정** 페이지에서 **이름을** **MobileAppsManagement_EXTENSION_VERSION** 값으로 입력하고 **확인을** 누를 수 있습니다.

새로 만든 앱 서비스 웹 앱을 모바일 앱으로 사용하도록 설정되어 있습니다.
