1. [Azure Portal]에 로그인합니다.

2. **리소스 만들기**를 클릭합니다.

3. 검색 상자에 **Mobile Apps**를 입력합니다.

    [![Mobile Apps 빠른 시작이 강조 표시된 Azure Portal][quickstart]](./media/app-service-mobile-dotnet-backend-create-new-service/search-mobile-apps-quickstart-zoom.png#lightbox)

4. 결과 목록에서 **Mobile Apps 빠른 시작**을 선택한 후 **만들기**를 선택합니다.

5. 고유한 **앱 이름**을 선택합니다. 이 이름은 App Service에 대한 도메인 이름의 일부도 됩니다.

6. **리소스 그룹** 아래에서 기존 리소스 그룹을 선택하거나 새 리소스 그룹을 만듭니다(앱과 동일한 이름 사용).

7. **만들기**를 클릭합니다. 계속 진행하기 전에 서비스가 성공적으로 배포되도록 몇 분 동안 기다립니다. 상태 업데이트를 확인하려면 포털 머리글에 있는 알림(벨) 아이콘을 지켜봅니다.
