Always On은 연속 Webjob 실행을 위해 필요합니다. Azure 스케줄러 무료 또는 표준은 예약된 WebJobs에 필요합니다. <sup>10</sup>장애 조치의 경우 Azure 트래픽 관리자와 함께 여러 인스턴스를 사용하는 배포에 대해 제공되는 99.95%의 SLA입니다.
