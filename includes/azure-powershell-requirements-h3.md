### <a name="requirements"></a>요구 사항

* PowerShell을 로컬로 사용 하도록 선택 하는 경우:
  * [Az PowerShell module을 설치](/powershell/azure/install-az-ps)합니다.
  * [AzAccount](/powershell/module/az.accounts/connect-azaccount) cmdlet을 사용 하 여 Azure 계정에 연결 합니다.
* Cloud Shell 사용 하도록 선택 하는 경우:
  * 자세한 내용은 [Azure Cloud Shell 개요](https://docs.microsoft.com/azure/cloud-shell/overview) 를 참조 하세요.
