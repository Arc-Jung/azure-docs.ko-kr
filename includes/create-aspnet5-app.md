1. Visual Studio 메뉴에서 **파일 > 새로 만들기 > 프로젝트**를 선택합니다.
2. **새 프로젝트** 대화 상자의 **템플릿** 섹션에서 **Visual C# > 웹**을 선택합니다.
3. **ASP.NET Core 웹 응용 프로그램**을 선택합니다.
4. 새 응용 프로그램에 이름을 지정(또는 기본값 사용)하고 **확인**을 선택합니다.
5. **웹 응용 프로그램**을 선택합니다.
6. **Docker 지원 사용** 확인란을 선택하고 **확인**을 클릭합니다.
