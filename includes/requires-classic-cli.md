> [!IMPORTANT]
> 이 문서에는 Azure 클래식 CLI가 필요한 콘텐츠가 포함되어 있습니다. Azure CLI의 현재 릴리스에서는 이 문서에 설명된 기능을 지원하지 않으므로 클래식 CLI가 필요합니다. 
> 
> 클래식 CLI를 최신 Azure CLI와 나란히 설치할 수 있지만 기능 지원을 사용할 수 있는 모든 새 스크립트 및 배포에는 Azure CLI를 사용하는 것이 좋습니다. 클래식 CLI를 설치하려면 [Azure 클래식 CLI 설치](/cli/azure/install-classic-cli)를 참조하고, 최신 CLI를 설치하려면 [Azure CLI 설치](/cli/azure/install-azure-cli)를 참조하세요.
