### <a name="mfa"></a>MFA 페이지 열기

1. Azure Portal에 로그인합니다.
2. **Azure Active Directory-> 모든 사용자**로 이동 합니다.
3. **Multi-Factor Authentication** 를 선택 하 여 multi-factor Authentication 페이지를 엽니다.

   ![로그인](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="users"></a>사용자 선택

1. **Multi-factor authentication** 페이지에서 MFA를 사용 하도록 설정 하려는 사용자를 선택 합니다.
2. **사용**을 선택합니다.

   ![여기서](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
