Vm (가상 컴퓨터)에는 VM을 배포 하는 서브넷에 따라 사용자가 지정한 범위에서 개인 IP 주소가 자동으로 할당 됩니다. Vm은 VM이 삭제 될 때까지 주소를 유지 합니다. Azure는 VM을 만든 서브넷에서 사용 가능한 다음 개인 IP 주소를 동적으로 할당 합니다. VM에 할당 된 서브넷의 특정 IP 주소를 원하는 경우 고정 IP 주소를 할당 합니다.
