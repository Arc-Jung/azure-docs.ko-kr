> [!NOTE]
> Azure AD B2C에서 사용자 흐름 이름 앞에는 `b2c_1_`이 붙습니다(예: `b2c_1_sign_up`).  클라이언트와 서버 둘 다의 모든 앱에서 사용자 흐름을 자유롭게 사용할 수 있습니다.  이전에 다른 B2C 연습에서 사용자 흐름을 만든 경우에는 다시 만들 필요가 없습니다. 애플리케이션의 요구 사항과 일치하는 경우 포털에서 이전에 만든 사용자 흐름을 다시 사용할 수 있습니다.
