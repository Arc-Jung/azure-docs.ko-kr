#### <a name="to-add-or-modify-a-storsimple-backup-schedule"></a>StorSimple 백업 일정을 추가 또는 수정하려면

1. StorSimple 디바이스로 이동하고 **백업 정책** 을 클릭합니다.

2. 정책의 테이블 형식 목록에서 수정하려는 정책을 선택 및 클릭합니다. 상황에 맞는 메뉴를 마우스 오른쪽 단추로 클릭하여 호출한 후 **일정 추가** 를 선택합니다.

    ![상황에 맞는 메뉴에서 일정 추가가 선택 된 상태에서 선택한 백업 정책을 보여 주는 스크린샷](./media/storsimple-8000-add-modify-backup-schedule-u2/addschedule1.png)

3. **일정 추가** 블레이드에서 스냅샷 유형, 백업 빈도, 재방문 주기, 시작 및 종료 시간을 수정합니다. **추가** 를 클릭합니다.

    ![스크린 샷에서는 값이 입력 된 일정 추가 창이 표시 됩니다.](./media/storsimple-8000-add-modify-backup-schedule-u2/addschedule5.png)

4. 백업 정책 만들기가 완료되면 알림이 표시됩니다. 백업 정책 목록도 업데이트됩니다.

    ![스크린샷 두 개의 예약 된 백업을 사용 하 여 선택한 백업 정책을 보여 줍니다.](./media/storsimple-8000-add-modify-backup-schedule-u2/addschedule4.png)
