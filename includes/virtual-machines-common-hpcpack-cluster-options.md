


클라우드 기반 HPC(고성능 컴퓨팅) 클러스터를 만들고 관리하려면 Microsoft HPC Pack과 Azure 계산 및 인프라 서비스를 이용합니다. [HPC Pack](https://technet.microsoft.com/library/jj899572.aspx)은 Microsoft Azure 및 Windows Server 기술로 구축된 Microsoft의 무료 HPC 솔루션으로, 광범위한 HPC 워크로드를 지원합니다. 클라우드 기반 HPC Pack 클러스터는 클러스터 관리자 또는 ISV(Independent Software Vendor)에 유연하고 확장 가능한 플랫폼을 제공하여 계산 집약적인 응용 프로그램을 실행합니다. 온-프레미스 계산 클러스터 인프라에 대한 투자도 줄일 수 있습니다. Azure에서 HPC 및 배치 작업을 실행하기 위한 다른 옵션을 보려면 [배치 및 고성능 컴퓨팅에 대한 기술 리소스](../articles/batch/big-compute-resources.md)를 참조하세요.
