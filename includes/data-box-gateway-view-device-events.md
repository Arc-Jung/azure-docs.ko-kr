디바이스 이벤트를 보려면 Azure Portal에서 다음 단계를 수행합니다. 

1. Azure Portal에서 리소스로 이동한 후 **모니터링 > 장치 이벤트** 로 이동 합니다.
2. 이벤트를 선택 하 고 경고 세부 정보를 확인 합니다. 경고 조건을 해결하려면 적절한 조치를 취합니다.
