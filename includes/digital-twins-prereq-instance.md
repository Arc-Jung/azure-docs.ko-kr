이 문서에서 Azure Digital Twins로 작업하려면 먼저 Azure Digital Twins 인스턴스를 설정하고 이를 사용하는 데 필요한 권한이 필요합니다. 이전 작업에서 이미 Azure Digital Twins 인스턴스가 설정된 경우 해당 인스턴스를 사용할 수 있습니다.

그렇지 않으면 [인스턴스 및 인증 설정](../articles/digital-twins/how-to-set-up-instance-portal.md)의 지침을 따릅니다. 지침에는 각 단계를 성공적으로 완료하고 새 인스턴스를 사용할 준비가 되었는지 확인하는 단계도 포함되어 있습니다.

Azure Digital Twins 인스턴스가 설정되면 해당 인스턴스의 *_호스트 이름_* 이 필요합니다. [Azure Portal](../articles/digital-twins/how-to-set-up-instance-portal.md#verify-success-and-collect-important-values)에서 이름을 찾을 수 있습니다. 나중에 인스턴스에 연결하는 데 사용할 수 있도록 이 값을 기록해 둡니다.
