**파일 > 빌딩 설정** 을 클릭합니다. **유니버설 Windows 플랫폼** 을 클릭한 다음, **플랫폼 전환** 을 클릭합니다. Unity 편집기에서 일부 구성 요소를 먼저 다운로드해야 한다고 표시되면 해당 구성 요소를 다운로드하여 설치합니다. 아래 스크린샷에 따라 빌드 설정을 구성합니다.

:::image type="content" source="./media/object-anchors-quickstarts-unity/aoa-unity-build-settings.png" alt-text="빌드 설정":::

**빌드** 를 클릭하고 출력 폴더를 선택합니다. 이제 출력 폴더에 VS 프로젝트를 생성할 수 있습니다.
