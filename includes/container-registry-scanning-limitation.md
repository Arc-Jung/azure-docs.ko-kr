> [!IMPORTANT]
> * Azure Security Center는 현재 개인 끝점, 선택한 서브넷 또는 IP 주소에 대 한 액세스를 제한 하는 레지스트리에서 [이미지 취약성 검사](../articles/security-center/defender-for-container-registries-introduction.md?bc=%2fazure%2fcontainer-registry%2fbreadcrumb%2ftoc.json&toc=%2fazure%2fcontainer-registry%2ftoc.json) 를 수행할 수 없습니다. 
> * Azure DevOps Services, Web Apps 및 Azure Container Instances를 비롯 한 Azure 서비스 인스턴스도 네트워크 제한 컨테이너 레지스트리에 액세스할 수 없습니다. 
> * 다른 특정 Azure 서비스 인스턴스는 네트워크 제한 컨테이너 레지스트리에 안전 하 게 액세스할 수 있습니다. 자세한 내용은 [신뢰할 수 있는 서비스에서 네트워크 제한 컨테이너 레지스트리에 안전 하 게 액세스 하도록 허용](../articles/container-registry/allow-access-trusted-services.md)을 참조 하세요.
