## NIC

NIC(네트워크 인터페이스 카드)는 VM(가상 컴퓨터)에 연결 될 수 있는 네트워크 인터페이스를 나타냅니다. VM은 NIC를 하나 이상 포함할 수 있습니다.

![단일 VM의 NIC](./media/resource-groups-networking/Figure3.png)

NIC 리소스의 키 속성은 다음과 같습니다.

- IP 설정
- 내부 DNS 이름
- DNS 서버

NIC를 다음 네트워크 리소스에 연결할 수도 있습니다.

- NSG(네트워크 보안 그룹) 
- 부하 분산 장치
