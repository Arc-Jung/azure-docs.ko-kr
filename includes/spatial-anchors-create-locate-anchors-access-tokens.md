### <a name="access-tokens"></a>액세스 토큰

액세스 토큰은 Azure 공간 앵커를 사용 하 여 인증 하는 보다 강력한 방법을입니다. 특히와 프로덕션 배포에 대 한 응용 프로그램을 준비 합니다. 이 방법의 요약 인증 클라이언트 응용 프로그램을 받을 수 안전 하 게 하는 백 엔드 서비스를 설정 하는 것입니다. 런타임에 AAD를 사용 하 여 및 액세스 토큰을 요청 하려면 Azure 공간 앵커 STS 서비스를 사용 하 여 백 엔드 서비스 인터페이스입니다. 이 토큰은 다음 클라이언트 응용 프로그램 배달와 SDK에 인증 하는 데 Azure 공간 앵커입니다.
