Azure에서 함수 앱에 대 한 스트리밍 로그를 켜려면:

1. 명령 팔레트를 열고 다음 검색 하 고 명령을 실행 하 여 F1 키를 눌러 **Azure Functions: 로그 스트리밍을 시작**합니다.

1. Azure에서 함수 앱을 선택한 다음 선택 **예** 함수 앱에 대 한 응용 프로그램 로깅을 사용 하도록 설정 합니다.

1. Azure에서 함수를 트리거하고 로그 데이터는 Visual Studio Code의 출력 창에 표시 합니다.

1. 을 완료 한 후이 명령을 실행 해야 **Azure Functions: 로그 스트리밍 중지** 함수 앱에 대 한 로깅을 사용 하지 않도록 설정 합니다.
