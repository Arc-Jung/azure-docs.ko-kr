Azure에서 함수 앱에 대한 스트리밍 로그를 켜려면 다음을 수행합니다.

1. F1을 선택하여 명령 팔레트를 연 다음 Azure **Functions: 스트리밍 로그 시작**명령을 검색하고 실행합니다.

1. Azure에서 함수 앱을 선택한 다음 **예를** 선택하여 함수 앱에 대한 응용 프로그램 로깅을 사용하도록 설정합니다.

1. Azure에서 함수를 트리거합니다. 로그 데이터는 Visual Studio 코드의 출력 창에 표시됩니다.

1. 작업이 완료되면 **Azure Functions: 스트리밍 로그 중지를 실행하여** 함수 앱에 대한 로깅을 사용하지 않도록 설정합니다.
