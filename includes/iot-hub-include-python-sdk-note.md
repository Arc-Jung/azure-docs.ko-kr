> [!NOTE]
> IoT Hub는 Azure IoT 장치 SDK를 통해 많은 장치 플랫폼 및 언어(C, Java, Java 스크립트 및 파이썬 포함)에 대한 [SDK](https://github.com/azure/azure-iot-sdks)지원을 제공합니다. 파이썬을 사용하여 장치를 이 자습서의 코드에 연결하고 일반적으로 Azure IoT Hub에 연결하는 방법에 대한 지침은 [Azure IoT 파이썬 SDK](https://github.com/Azure/azure-iot-sdk-python)를 참조하십시오.
