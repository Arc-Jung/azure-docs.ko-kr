> [!IMPORTANT]
> Azure SQL Database Machine Learning Services는 현재 미리 보기로 제공됩니다. 미리 보기 등록은 현재 **마감**되었습니다.
>
> 2020년 6월 30일까지 지원은 중단되고 일반 공급을 위해 미리 보기가 릴리스되지 않습니다. 2020년 6월 30일 이후에는 사용 중인 R 스크립트는 작동하지 않습니다.
>
> Azure SQL에서 기계 학습을 계속 사용하려면 [**Azure SQL Managed Instance**의 Machine Learning Services(미리 보기)](../articles/azure-sql/managed-instance/machine-learning-services-overview.md)를 참조하세요.
