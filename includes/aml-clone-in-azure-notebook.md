* **Azure Notebooks** - Azure 클라우드의 무료 Jupyter 기반 Notebook
    1. [![Azure Notebooks](https://notebooks.azure.com/launch.png)](https://aka.ms/aml-clone-azure-notebooks)
    [샘플 Notebook을 Azure Notebooks로 가져옵니다](https://aka.ms/aml-clone-azure-notebooks).
    1.  [Azure Machine Learning 서비스용 개발 환경을 구성하는 방법](https://aka.ms/aml-how-to-configure-environment)의 지침을 사용하여 작업 영역 및 해당 구성 파일(**config.json**)을 만듭니다.
    1. Azure Notebook 도구 모음에서 `+New`를 선택하여 **config.json** 파일을 Notebook과 같은 폴더에 추가합니다.
    1. Notebook을 엽니다.

* **사용자 고유 Jupyter Notebook 서버**

    [GitHub 리포지토리](https://aka.ms/aml-notebooks)를 복제하고 **추가 정보** 지침에 따라 서버에 대한 소프트웨어를 설치합니다.
