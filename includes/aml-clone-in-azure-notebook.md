### <a name="azure-notebooks---free-jupyter-based-notebooks-in-the-azure-cloud"></a>**Azure Notebooks** - Azure 클라우드의 무료 Jupyter 기반 Notebook
SDK는 Azure Notebooks에 이미 설치되고 구성되어 있습니다.
  
1. [빠른 시작: 시작](../articles/machine-learning/service/quickstart-get-started.md)을 수행하여 작업 영역을 만들고 Azure Notebooks를 시작합니다.
1. [Azure Notebooks](https://notebooks.azure.com/)로 이동합니다.
1. 빠른 시작 중에 만든 `Getting Started` 라이브러리에서 `tutorials` 폴더로 이동합니다
1. Notebook을 엽니다.

### <a name="your-own-jupyter-notebook-server"></a>**사용자 고유 Jupyter Notebook 서버**
1. [빠른 시작: Python SDK를 사용하여 시작](../articles/machine-learning/service/quickstart-create-workspace-with-python.md)을 수행하여 SDK를 설치하고 작업 영역을 만듭니다.
1. [GitHub 리포지토리](https://aka.ms/aml-notebooks)를 복제합니다.
1. 빠른 시작 중에 만든 `aml_config` 디렉터리를 복제된 디렉터리에 복사합니다.
1. 복제된 디렉터리에서 노트북 서버를 시작합니다.
1. `tutorials` 폴더로 이동합니다.
1. Notebook을 엽니다.
