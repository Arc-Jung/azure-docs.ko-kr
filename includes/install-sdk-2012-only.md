<h2><a name="setupdevenv"></a>개발 환경 설정</h2>

시작하려면 Azure SDK for .NET을 설치하여 개발 환경 설정을 설정합니다.

1. 해당 SDK를 설치하려면 아래 링크를 클릭하십시오. 이 자습서를 완료하려면 Visual Studio 2012가 필요합니다. Visual Studio를 설치하지 않은 경우 Visual Studio Express for Web이 SDK와 함께 설치됩니다.
   
   * [Visual Studio 2012용 Azure SDK][Azure SDK for Visual Studio 2012]
     
     **참고:** 사용자 컴퓨터에 SDK 종속성이 얼마나 있었는지에 따라 SDK를 설치하는 시간이 몇 분에서&30;분 또는 그 이상이 될 수 있습니다.
2. 실행 가능한 설치 프로그램을 실행할지 또는 저장할지 묻는 메시지가 표시되면 **실행**을 클릭합니다.
3. 웹 플랫폼 설치 관리자 창에서 **설치** 를 클릭하여 설치를 계속합니다.
   
    ![웹 플랫폼 설치 관리자 - Azure SDK for .NET][WebPIAzureSdk]

설치가 완료되면 개발을 시작하는 데 필요한 내용이 모두 준비된 것입니다.
