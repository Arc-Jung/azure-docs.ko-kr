미사용 데이터의 경우:

- 공유에 저장된 데이터에 대한 액세스가 제한됩니다.

    - 공유 데이터에 액세스하는 SMB 클라이언트에는 공유와 연결된 사용자 자격 증명이 필요합니다. 이러한 자격 증명은 공유를 만들 때 정의됩니다.
    - 공유에 액세스하는 NFS 클라이언트의 IP 주소를 공유할 때 추가해야 합니다.
