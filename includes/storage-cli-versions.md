## <a name="versions-of-the-azure-cli"></a>Azure CLI의 버전

현재 사용할 수 있는 두 가지 Azure CLI(명령줄 인터페이스) 버전이 있습니다.

* [Azure CLI](../articles/storage/common/storage-azure-cli.md): 현재 버전의 Azure CLI를 Resource Manager 배포 모델을 사용 합니다.
* [Azure 클래식 CLI](../articles/storage/common/storage-azure-cli-nodejs.md): 이전 버전의 Azure CLI 클래식 배포 모델 에서만 사용 하기 위한 것입니다.
