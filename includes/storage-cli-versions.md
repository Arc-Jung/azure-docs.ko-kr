## <a name="versions-of-the-azure-cli"></a>Azure CLI의 버전

현재 사용할 수 있는 두 가지 Azure CLI(명령줄 인터페이스) 버전이 있습니다.

* [Azure CLI](../articles/storage/common/storage-azure-cli.md): Resource Manager 배포 모델에서 사용할 수 있는 최신 버전의 Azure CLI입니다.
* [Azure 클래식 CLI](../articles/storage/common/storage-azure-cli-nodejs.md): 클래식 배포 모델에서만 사용할 수 있는 이전 버전의 Azure CLI입니다.
