다음 방법 중 하나로 데이터 저장소를 구성하는 경우 이 데이터 저장소에 연결하려면 [자체 호스팅 통합 런타임을](../articles/data-factory/create-self-hosted-integration-runtime.md) 설정해야 합니다.

- 데이터 저장소는 온-프레미스 네트워크 내부, Azure 가상 네트워크 내부 또는 Amazon 가상 사설 클라우드 내에 있습니다.
- 데이터 저장소는 방화벽 규칙에 나열된 IP로 액세스가 제한되는 관리되는 클라우드 데이터 서비스입니다.
