Text Analytics API에 대한 액세스 권한이 있는 [Cognitive Services API 구독](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account)이 있어야 합니다. 구독이 없는 경우 [무료 계정](https://azure.microsoft.com/try/cognitive-services/?api=bing-web-search-api)을 만들 수 있습니다. 계속하려면 계정을 활성화한 후 제공된 Text Analytics 구독 키가 필요합니다.
