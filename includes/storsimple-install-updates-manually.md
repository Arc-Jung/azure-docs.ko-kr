#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>Azure 클래식 포털을 통해 일반 업데이트를 설치하려면
1. **디바이스** 페이지에서 업데이트를 설치할 디바이스를 선택합니다.
2. **장치**  >  **유지 관리** 로 이동 하 여 **소프트웨어 업데이트** 까지 아래로 스크롤합니다.
3. 업데이트를 확인하려면 페이지 아래쪽의 **업데이트 확인** 을 클릭합니다.
4. 소프트웨어 업데이트를 사용할 수 있는 경우 메시지가 나타납니다. 장치 업데이트를 시작하려면 **업데이트 설치** 를 클릭합니다.
   
    업데이트가 성공적으로 설치되면 알림이 표시됩니다.
