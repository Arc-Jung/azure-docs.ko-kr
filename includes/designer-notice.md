> Machine Learning Studio(클래식)를 사용하거나 평가하는 고객은 ML 모듈 끌어서 놓기 __기능과 더불어__ 확장성, 버전 제어 및 엔터프라이즈 보안을 제공하는 [Azure Machine Learning 스튜디오](../articles/machine-learning/overview-what-is-machine-learning-studio.md)를 사용하는 것이 좋습니다. 현재 스튜디오(클래식) 고객은 [Azure Machine Learning으로 마이그레이션](../articles/machine-learning/classic/migrate-overview.md)하는 방법을 학습할 수 있습니다.
