함수 앱은 함수 실행을 호스트합니다. 함수 앱을 통해 함수를 논리 단위로 그룹화하여 더욱 쉽게 관리, 배포 및 리소스 공유할 수 있습니다.

**새로 만들기** > **계산** > **함수 앱**을 선택합니다.

![새 함수 앱](media/functions-create-function-app-portal2/new_fun_app.png)

필수 필드를 입력합니다.

![새 함수 앱](media/functions-create-function-app-portal2/new_fun_app2.png)

| 필드               | 샘플 값 | 주석 |
| ----------------- | ------------ | ------------- |
| 앱 이름 | myFunctionApp | 고유한 이름을 입력합니다. |
| 구독 | MSDN | 사용 중인 구독을 선택합니다. |
| 리소스 그룹 | myFunctionApp | 새 리소스 그룹을 만듭니다. |
| 호스팅 계획 | 소비 계획 |  리소스 사용량에 따라 청구됩니다. |
| 위치 | 미국 서부  | 가까운 위치를 선택합니다. |
| 저장소 | myFunctionApp123  | 새 저장소 계정을 만듭니다. |

**만들기**를 선택하여 새 함수 앱을 프로비전하고 배포합니다.
