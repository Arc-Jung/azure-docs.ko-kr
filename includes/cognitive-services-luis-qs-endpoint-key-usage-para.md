이 빠른 시작에서는 편의를 위해 제작 키를 사용합니다. 이 키는 주로 모델 작성 용도로 사용되지만 소량(1000)의 엔드포인트 요청을 허용합니다. 테스트, 단계 또는 프로덕션 환경에서 더 많은 엔드포인트 요청을 받을 준비가 되었다면 Azure Portal에서 **Language Understanding** 리소스를 만들어 LUIS 포털의 LUIS 앱에 할당합니다.
