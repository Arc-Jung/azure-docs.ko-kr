### <a name="azure-active-directory-authentication"></a>Azure Active Directory 인증

또한 azure 공간 앵커에는 응용 프로그램을을 Azure Active Directory 토큰 사용자를 사용 하 여 인증할 수 있습니다. 예를 들어, ASA 통합할 AAD 토큰을 사용할 수 있습니다. AAD에서 사용자를 관리 하는 엔터프라이즈의 경우에 Azure 공간 앵커 SDK에서 사용자 AAD 토큰을 제공할 수 있습니다. 이렇게 하면 동일한 AAD 테 넌 트의 일부인 계정에 대 한 ASA 서비스에 직접 인증할 수 있습니다.
