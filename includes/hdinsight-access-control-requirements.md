회사 소유 구독과 같이 자신이 관리자 또는 소유자가 아닌 Azure 구독을 사용하는 경우에는 이 문서의 단계를 사용하기 전에 다음을 확인해야 합니다.

* Azure에 로그인하려면 적어도 Azure 리소스 그룹에 대한 참가자 액세스가 필요합니다. 이 리소스 그룹은 Azure HDInsight 클러스터 및 다른 Azure 리소스를 만드는 데 사용됩니다.
* Azure 구독에 대해 참가자 이상의 액세스가 있는 사용자는 사용할 리소스의 공급자를 이전에 등록해 두어야 합니다. 공급자 등록은 구독에 대해 참가자 액세스가 있는 사용자가 구독에서 처음으로 리소스를 만들 때 이루어집니다. [REST를 사용하여 공급자를 등록](https://msdn.microsoft.com/library/azure/dn790548.aspx)하면 리소스를 만들지 않고도 수행될 수 있습니다.
