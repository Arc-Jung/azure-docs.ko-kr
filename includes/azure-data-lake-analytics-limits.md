Azure Data Lake Analytics를 사용하면 분산 인프라 및 복잡한 코드를 쉽게 관리하는 복잡한 작업을 수행할 수 있습니다. 리소스를 동적으로 프로비전하며 이를 사용하여 엑사바이트 규모의 데이터에 대한 분석을 수행할 수 있습니다. 작업이 완료되면 리소스가 자동으로 감깁니다. 사용 된 처리 능력만 지불 합니다. 저장된 데이터의 크기 나 사용된 계산의 양을 늘리거나 줄이면 코드를 다시 작성할 필요가 없습니다. 구독의 기본 한도를 높이려면 지원팀에 문의하세요.

| **리소스** | **제한** | **설명** |
| --- | --- | --- |
| 최대 동시 작업 수 |20 | |
| 계정당 최대 분석 단위(AUS) 수 |250 | 20개의 작업에 최대 250개의 AU 조합을 사용합니다. 이 제한을 높이려면 Microsoft 지원에 문의하십시오. |
| 작업 제출의 최대 스크립트 크기 | 3MB | |
| 구독당 지역별 최대 데이터 레이크 애널리틱스 계정 수 | 5 | 이 제한을 높이려면 Microsoft 지원에 문의하십시오. |
