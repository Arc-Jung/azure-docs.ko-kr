## <a name="configure-the-sensor-fingerprint-provider"></a>센서 지문 공급자 구성

먼저 센서 지문 공급자를 만들고 구성합니다. 센서 지문 공급자는 디바이스의 플랫폼별 센서를 읽고 해당 판독값을 클라우드 앵커 세션에서 사용하는 일반적인 표현으로 변환합니다.
