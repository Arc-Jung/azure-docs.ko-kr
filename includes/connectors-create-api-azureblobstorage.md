1. 연결 정보에 대한 메시지가 표시되면 다음과 같은 세부 정보를 입력합니다.

   *필수*

   | 자산 | 값 | 설명 | 
   |----------|-------|-------------| 
   | 연결 이름 | <*connection-name*> | 연결에 만들 이름 | 
   | Storage 계정 | <*storage-account*> | 목록에서 저장소 계정을 선택합니다. | 
   ||| 

   예: 

   ![연결 만들기](./media/connectors-create-api-azureblobstorage/create-storage-account-connection.png)  

2. 작업을 완료하면 **만들기**를 선택합니다.

3. 연결을 만든 후 [트리거 추가](#add-trigger) 또는 [작업 추가](#add-action)를 계속 진행합니다.
