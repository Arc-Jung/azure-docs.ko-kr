<!-- this tells how to get the connection string for your hub -->
<!-- This assumes the user is looking at his hub in the portal. -->

허브가 만들어졌으면 허브에 대한 연결 문자열을 검색합니다. 연결 문자열은 디바이스와 애플리케이션을 허브에 연결하는 데 사용됩니다. 

1. 설정 등이 포함된 IoT Hub 창을 보려면 허브를 클릭합니다. **공유 액세스 정책**을 클릭합니다.
   
2. **공유 액세스 정책**에서 **iothubowner** 정책을 선택합니다. 

3. **공유 액세스 키**에서 나중에 사용할 **연결 문자열 - 기본 키**를 복사합니다.

    ![연결 문자열을 검색하는 방법 표시](./media/iot-hub-include-find-connection-string/iot-hub-get-connection-string.png)

    자세한 내용은 "IoT Hub 개발자 가이드"의 [액세스 제어](../articles/iot-hub/iot-hub-devguide-security.md)를 참조하세요.
