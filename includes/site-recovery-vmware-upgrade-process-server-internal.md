1. 관리자 권한으로 프로세스 서버에 로그인합니다.
2. 최신 버전의 [통합 설정 설치 관리자](https://aka.ms/unifiedinstaller)를 다운로드합니다.
3. 업데이트 프로세스를 시작하려면 설치 관리자를 두 번 클릭합니다.
4. 설치 관리자가 설치된 Site Recovery 구성 요소를 검색하고 최신 버전으로 업그레이드합니다.
