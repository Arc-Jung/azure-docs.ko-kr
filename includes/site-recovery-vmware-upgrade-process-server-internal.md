1. 프로세스 서버에 관리자로 로그인합니다.
2. 통합 설치 설치 관리자의 최신 버전을 [다운로드합니다.](https://aka.ms/unifiedinstaller)
3. 업데이트 프로세스를 시작하려면 설치 관리자를 두 번 클릭합니다.
4. 설치 관리자는 설치된 사이트 복구 구성 요소를 검색하고 최신 버전으로 업그레이드합니다.
