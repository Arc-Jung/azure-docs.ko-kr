1. 관리자 권한으로 프로세스 서버에 로그인 합니다.
2. 최신 버전의 [통합 설치 관리자](https://aka.ms/unifiedinstaller)를 다운로드 합니다.
3. 업데이트 프로세스를 시작하려면 설치 관리자를 두 번 클릭합니다.
4. 설치 관리자는 설치 된 Site Recovery 구성 요소를 검색 하 고 최신 버전으로 업그레이드 합니다.
