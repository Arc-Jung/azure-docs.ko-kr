> [!NOTE]
>
> 이 문서는 최근에 업데이트되어 새 Azure PowerShell Az 모듈의 명령을 사용합니다. 
>
> * Az 모듈에 대한 자세한 내용은 [새로운 Azure PowerShell Az 모듈 소개](/powershell/azure/new-azureps-module-az)를 참조하세요.
> * Az 모듈 설치에 대한 지침은 [Azure PowerShell 설치](/powershell/azure/install-az-ps)를 참조하세요. Az는 Cloud Shell에서 이미 사용 가능합니다.
> * AzureRM 호환성령을 사용하도록 설정하려면 설치 후 `Enable-AzureRMAlias`를 실행합니다.
>   호환성에 대한 자세한 내용은 [Azure PowerShell AzureRM에서 Az로 마이그레이션](/powershell/azure/migrate-from-azurerm-to-az)을 참조하세요.
