> [!NOTE]
>
> 이 문서는 최근에 업데이트되어 새 Azure PowerShell Az 모듈의 명령을 사용합니다. 이 모듈은 PowerShell 버전 5.x 및 6와 호환되는 Azure의 새 PowerShell 모듈입니다. AzureRM은 계속해서 중요한 버그 수정을 업데이트하지만 새 기능은 Az 모듈에서만 작동합니다.
>
> * Az 모듈과 AzureRM 모듈에 대한 지속적인 지원에 대한 자세한 내용은 [새 Azure PowerShell Az 모듈 소개](/powershell/azure/new-azureps-module-az)를 참조하세요.
> * Az 모듈은 AzureRM 호환성을 자동으로 사용하도록 설정하여 이미 Cloud Shell에서 사용할 수 있습니다.
>   Az 모듈 설치에 대한 지침은 [Azure PowerShell 설치](/powershell/azure/install-az-ps)를 참조하세요.
> * AzureRM 호환성 명령을 사용하도록 설정하려면 설치 후 `Enable-AzureRmAlias`를 실행합니다.
>   호환성에 대한 자세한 내용은 [Azure PowerShell AzureRM에서 Az로 마이그레이션](/powershell/azure/migrate-from-azurerm-to-az)을 참조하세요.
