다음 표에서는 Azure Maps S0 가격 책정 계층의 사용 제한을 보여줍니다. 사용량 제한은 가격 책정 계층에 따라 달라집니다.

| 리소스                              | S0 가격 책정 계층 한도 |
|---------------------------------------|:---------------------:|
| 구독당 최대 요청 속도 |   초당 50회 요청  |

다음 표에서는 Azure 구독의 Azure Maps 계정에 대한 누적 데이터 크기 제한을 보여줍니다. Azure Maps Data 서비스는 S1 가격 책정 계층에서만 사용할 수 있습니다.

| 리소스                               | 제한  |
|----------------------------------------|:---------------------:|
| Azure 구독당 최대 스토리지 |   1 GB |
| 파일 업로드당 최대 크기           | 100MB |


Azure Maps 가격 책정 계층에 대한 자세한 내용은 [Azure Maps 가격 책정](https://azure.microsoft.com/pricing/details/azure-maps/)을 참조하세요.
