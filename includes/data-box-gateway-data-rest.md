미사용 데이터의 경우:

- 공유에 저장 된 데이터에 대 한 액세스가 제한 됩니다.

    - 공유 데이터에 액세스 하는 SMB 클라이언트에는 공유와 연결 된 사용자 자격 증명이 필요 합니다. 이러한 자격 증명은 공유를 만들 때 정의 됩니다.
    - 공유를 만들 때 공유에 액세스 하는 NFS 클라이언트의 IP 주소를 추가 해야 합니다.
