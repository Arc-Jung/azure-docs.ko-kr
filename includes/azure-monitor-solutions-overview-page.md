이 모니터링 솔루션으로 수집된 데이터는 Azure Portal의 Azure Monitor **개요** 페이지에서 사용할 수 있습니다. **인사이트** 섹션에서 **기타**를 클릭하여 **Azure Monitor** 메뉴에서 이 페이지를 엽니다. 각 솔루션이 타일로 표시됩니다. 해당 솔루션에서 수집한 자세한 데이터를 보려면 타일을 클릭합니다.
