Azure Data Lake Storage Gen1은 빅 데이터 분석 작업을 위한 엔터프라이즈 수준 하이퍼 스케일 리포지토리입니다. Data Lake Storage Gen1을 사용하여 운영 및 예비 분석을 위해 한 곳에서 모든 크기, 유형 및 수집 속도의 데이터를 캡처할 수 있습니다. Data Lake Storage Gen1 계정에 저장할 수 있는 데이터 양에는 제한이 없습니다.

| **리소스** | **제한** | **설명** |
| --- | --- | --- |
| 지역별 구독당 최대 데이터 레이크 스토리지 Gen1 계정 수 |10 | 이 제한에 대한 증가를 요청하려면 지원팀에 문의하십시오. |
| 파일 또는 폴더당 최대 액세스 ACL 수 |32 | 이것은 하드 한도입니다. 그룹을 사용하여 적은 항목으로 액세스를 관리합니다. |
| 파일 또는 폴더당 기본 ACL의 최대 수 |32 | 이것은 하드 한도입니다. 그룹을 사용하여 적은 항목으로 액세스를 관리합니다. |
