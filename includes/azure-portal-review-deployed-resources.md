Azure Portal 대시보드 내에서 가상 머신에 대한 데이터를 볼 수 있는지 확인합니다.

1. Azure Portal에서 **대시보드** 를 선택합니다.

    :::image type="content" source="media/azure-portal-review-deployed-resources/navigate-to-dashboards.png" alt-text="대시보드에 대한 Azure Portal 탐색":::

1. 대시보드 페이지에서 **단순 VM 대시보드** 를 선택합니다.

    :::image type="content" source="media/azure-portal-review-deployed-resources/select-simple-vm-dashboard.png" alt-text="단순 VM 대시보드로 이동":::

1. ARM 템플릿(Azure Resource Manager 템플릿)이 만든 대시보드를 검토합니다. 일부 콘텐츠는 정적이지만 처음에 만든 VM의 성능을 표시하는 차트도 있습니다.

    :::image type="content" source="media/azure-portal-review-deployed-resources/review-simple-vm-dashboard.png" alt-text="단순 VM 대시보드로 검토":::
