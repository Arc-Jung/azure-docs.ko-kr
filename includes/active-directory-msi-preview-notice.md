|MSI(관리되는 서비스 ID)는 현재 Azure Active Directory의 미리 보기 기능이며 평가 목적으로만 사용됩니다. MSI는 전역적으로 지역에 배포되고 해당 지역에 나타나는 데 시간이 걸릴 수 있습니다. 시작하기 전에 [알려진 문제](/azure/active-directory/msi-known-issues)를 검토합니다. 미리 보기에 대한 자세한 내용은 [Microsoft Azure 미리 보기에 대한 추가 사용 약관](https://azure.microsoft.com/support/legal/preview-supplemental-terms/)을 참조하세요.|
