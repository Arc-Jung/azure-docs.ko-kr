

1. [Azure 클래식 포털](https://manage.windowsazure.com/)에 로그인한 다음 화면 아래쪽에서 **+새로 만들기**를 클릭합니다.

2. **앱 서비스**, **Service Bus**, **알림 허브**, **빠른 생성**을 차례로 클릭합니다.

   	![](./media/notification-hubs-portal-create-new-hub/notification-hubs-create-from-portal.png)


3. **알림 허브 이름**을 입력합니다. 원하는 **지역**과 **구독**을 선택합니다.
 
	허브에서 만들려는 서비스 버스 네임스페이스가 있는 경우 **네임스페이스 이름**을 선택합니다. 그렇지 않고 네임스페이스 이름을 사용할 수 있는 경우 허브 이름을 기준으로 생성될 기본 **네임스페이스 이름**을 사용할 수 있습니다.

	**새 알림 허브 만들기**를 클릭합니다.

   	![알림 허브 속성 설정](./media/notification-hubs-portal-create-new-hub/notification-hubs-create-from-portal2.png)

4. 네임스페이스와 알림 허브가 생성되면 서비스 버스에 있는 네임스페이스가 표시됩니다. 방금 허브를 만든 네임스페이스를 클릭합니다(일반적으로 ***notification hub name*-ns**).

5. 네임스페이스 페이지에서 위쪽의 **알림 허브** 탭을 클릭한 후 방금 만든 알림 허브를 클릭합니다. 그러면 새 알림 허브의 대시보드가 열립니다.

6. 새 허브의 대시보드에서 **연결 문자열 보**기를 클릭합니다. 두 연결 문자열을 기록해 둡니다. 나중에 사용합니다.

   	![](./media/notification-hubs-portal-create-new-hub/notification-hubs-view-connection-strings.png)

	![](./media/notification-hubs-portal-create-new-hub/notification-hubs-connection-strings.png)
