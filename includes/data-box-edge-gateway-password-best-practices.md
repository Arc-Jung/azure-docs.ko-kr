다음 모범 사례를 염두에 두십시오.

- 모든 암호를 안전한 장소에 저장하여 암호를 잊어버린 경우 재설정할 필요가 없도록 하는 것이 좋습니다. 관리 서비스는 기존 암호를 검색할 수 없습니다. Azure 포털을 통해서만 재설정할 수 있습니다. 암호를 재설정하는 경우 암호를 재설정하기 전에 모든 사용자에게 알려야 합니다.
- HTTP를 통해 장치의 Windows PowerShell 인터페이스에 원격으로 액세스할 수 있습니다. 보안 모범 사례는 신뢰할 수 있는 네트워크에서만 HTTP를 사용해야 합니다.
- 장치 암호가 강력하고 잘 보호되어 있는지 확인합니다. 암호 [모범 사례를](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management)따르십시오.
