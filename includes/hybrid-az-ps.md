이 문서의 단계 및 예제에서는 Azure PowerShell Az modules를 사용 합니다. 컴퓨터에 로컬로 Az modules를 설치 하려면 [install Azure PowerShell](/powershell/azure/install-az-ps)를 참조 하세요. 새 Az module에 대해 자세히 알아보려면 [새 Azure PowerShell Az Module 소개](/powershell/azure/new-azureps-module-az)를 참조 하세요. PowerShell cmdlet은 자주 업데이트 됩니다. 최신 버전을 실행 하지 않는 경우 지침에 지정 된 값이 실패할 수 있습니다. 시스템에서 설치 된 PowerShell 버전을 찾으려면 cmdlet을 사용 합니다 `Get-Module -ListAvailable Az` .
