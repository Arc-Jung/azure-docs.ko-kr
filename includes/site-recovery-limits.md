Azure Site Recovery에는 다음과 같은 제한이 적용 됩니다.

| 제한 식별자 | 제한 |
| --- | --- |
| 구독당 자격 증명 모음 수 |500 |     
| Azure 자격 증명 모음당 서버 수 |250 |
| Azure 자격 증명 모음당 보호 그룹 수 |제한 없음 |
| Azure 자격 증명 모음당 복구 계획 수 |제한 없음 |
| 보호 그룹당 서버 수 |제한 없음 |
| 복구 계획당 서버 수 |50 |
