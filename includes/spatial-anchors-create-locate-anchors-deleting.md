## <a name="delete-anchors"></a>앵커 삭제

클라우드 공간 앵커를 삭제하려면 `DeleteAnchor()` 메서드를 사용합니다. 더 이상 사용하지 않을 때 앵커를 삭제하는 것은 Azure 리소스를 정리하기 위해 개발 프로세스와 관행 초기에 포함하는 것이 좋습니다.
