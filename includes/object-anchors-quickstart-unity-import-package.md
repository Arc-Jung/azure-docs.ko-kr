# <a name="npm-download"></a>[NPM 다운로드](#tab/unity-package-npm)

# <a name="mixed-reality-feature-tool-beta"></a>[Mixed Reality 기능 도구(베타)](#tab/unity-package-mixed-reality-feature-tool)

이 단계를 수행하려면 <a href="https://aka.ms/MRFeatureToolDocs" target="_blank">Mixed Reality 기능 도구</a>가 다운로드되어 사용할 수 있어야 합니다.

> [!NOTE]
> <a a href="https://aka.ms/MRFeatureToolDocs" target="_blank">Mixed Reality 기능 도구</a>는 현재 Windows만 지원합니다.

1. <a href="https://aka.ms/MRFeatureToolDocs" target="_blank">Mixed Reality 기능 도구</a> 설명서에 따라 도구를 설정하고 사용 방법을 알아보세요.
2. Unity 프로젝트 폴더에 "Microsoft Azure Object Anchors" 기능 패키지를 설치합니다.
