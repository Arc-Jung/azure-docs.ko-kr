## <a name="scenario"></a>시나리오

VM에 대 한 고정 IP 주소를 구성 하는 방법을 보다 잘 설명 하기 위해이 문서에서는 다음 시나리오를 사용 합니다.

![가상 네트워크 시나리오: 프런트 엔드 서브넷에 대 한 고정 IP 주소를 사용 하는 프런트 엔드 및 백 엔드 서브넷](./media/virtual-networks-static-ip-scenario-include/static-ip-scenario.png)

이 시나리오에서는 *프런트 엔드* 서브넷에 *DNS01* 이라는 VM을 만든 다음 *192.168.1.101*의 고정 IP 주소를 사용 하도록 설정 합니다.
