## Get SQL server connection information
-->

Azure SQL 데이터베이스에 연결하는 데 필요한 연결 정보를 가져옵니다. 다음 절차를 수행하려면 정규화된 서버 이름, 데이터베이스 이름 및 로그인 정보가 필요합니다.

1. [Azure Portal](https://portal.azure.com/)에 로그인합니다.

1. 왼쪽 메뉴에서 **SQL 데이터베이스**를 선택한 다음, **SQL 데이터베이스** 페이지에서 데이터베이스를 선택합니다.

1. 데이터베이스의 **개요** 페이지에서 **서버 이름** 옆에 있는 정규화된 서버 이름을 검토합니다. 서버 이름을 복사하려면 해당 이름을 마우스로 가리키고 **복사** 아이콘을 선택합니다.  

![서버 이름](./media/sql-database-connect-query-prerequisites-server-connection-info-includes/server-name.png)

로그인 정보를 잊은 경우 서버 이름을 선택하여 **SQL Server** 페이지를 엽니다. 여기에서 **서버 관리자** 이름을 보고, 필요한 경우 **암호 재설정**을 선택할 수 있습니다.
