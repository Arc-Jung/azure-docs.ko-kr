Cognitive Services 또는 Azure를 처음 사용하는 경우 [임시 키](https://azure.microsoft.com/try/cognitive-services/)를 만듭니다. Azure 계정이 있으면 포털에서 새 리소스를 만듭니다. Language Understanding 엔드포인트 쿼리는 두 가지 리소스 종류인 **Language Understanding** 또는 **Cognitive Service** 리소스를 지원합니다. 앱에 키를 [할당](../articles/cognitive-services/luis/luis-how-to-azure-subscription.md)할 때 리소스 이름이 필요합니다.
