## <a name="more-information-about-bots"></a>봇에 대한 자세한 정보

봇으로 이 서비스를 사용하는 방법에 대한 자세한 내용은 다음 리소스를 참조하세요.

|리소스|목적|
|--|--|
|[Azure Bot Service](https://dev.botframework.com/)|Azure Bot Service는 봇 엔드포인트를 사용하여 완전히 클라우드에 호스트된 웹 서비스를 제공합니다. 이 서비스는 여러 언어로 제공되는 [Bot Framework](https://github.com/Microsoft/botframework)를 사용합니다.|
|[Bot Framework](https://github.com/Microsoft/botframework)|Microsoft Bot Framework는 엔터프라이즈급 대화형 AI 환경을 구축하기 위한 포괄적인 플랫폼입니다.|
|[Bot Framework Emulator](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|Bot Framework Emulator는 봇 개발자가 Bot Framework SDK를 사용하여 빌드된 봇을 테스트 및 디버그할 수 있게 해주는 플랫폼 간 데스크톱 애플리케이션입니다. Bot Framework Emulator를 사용하여 머신의 로컬에서 실행 중인 봇을 테스트하거나 원격에서 실행 중인 봇에 연결할 수 있습니다.|
|[봇 도구](https://github.com/Microsoft/botbuilder-tools)|Bot Framework 도구는 엔드투엔드 봇 개발 워크플로를 처리하기 위해 설계된 플랫폼 간 명령줄 도구 모음입니다. |
|[Bot Builder 샘플](https://github.com/Microsoft/BotBuilder-Samples)|완전히 개발된 봇 샘플은 훌륭한 봇을 빌드하기 위해 구현해야 하는 시나리오를 설명하기 위해 설계되었습니다.|
