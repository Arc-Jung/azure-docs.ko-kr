보안 특성을 품질 또는 Azure 서비스의 기능입니다. 이 예방, 감지 및 보안 취약성에 대응 하는 서비스의 기능을 제공 합니다.

보안 특성은 다음과 같이 분류됩니다.
* 예방
* 네트워크 구분
* 감지
* Id 및 액세스 관리에 대 한 지원
* 감사 내역
* 액세스 제어(사용하는 경우)
* 구성 관리(사용하는 경우)

각 범주에 "Yes" 또는 "No" 특성의 사용 여부를 나타내려면 살펴보겠습니다. 일부 서비스의 경우 적용할 수 없는 특성에 대 한 "해당 없음" 표시 합니다. 또는 특성에 대 한 자세한 정보에 대 한 링크를 제공 될 수 있습니다.
