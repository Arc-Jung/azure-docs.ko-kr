1. 포털에서 VM을 선택합니다. **DNS 이름** 아래에서 **구성**을 클릭합니다.
   
   ![DNS 이름 아래의 구성 클릭](./media/virtual-machines-common-portal-create-fqdn/configure.png)

2. 원하는 DNS 이름 레이블을 입력한 다음 **저장**을 선택합니다.
   
   ![공용 IP 리소스에 대한 DNS 이름 레이블을 입력합니다.](./media/virtual-machines-common-portal-create-fqdn/configure-pane.png)


3. VM 개요 블레이드로 돌아가려면 *공용 IP 주소* 블레이드를 닫습니다. 이제 *DNS 이름*이 표시되는지 확인합니다.
