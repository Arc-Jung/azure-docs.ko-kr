## <a name="clean-up-deployment"></a>배포 정리

샘플 스크립트가 실행된 후, 다음 명령을 사용하여 리소스 그룹 및 리소스 그룹과 연관된 모든 리소스를 제거할 수 있습니다.

```azurecli
az group delete --name myResourceGroup
```
