## <a name="set-up-your-device"></a>디바이스 설정

HoloLens에 앱을 배포하려면 HoloLens를 컴퓨터와 페어링해야 합니다.

1. HoloLens에서 **설정 > 업데이트 및 보안 -> 개발자용** 으로 이동합니다.
2. **쌍** 을 클릭하고 첫 번째 배포 중에 Visual Studio에 PIN을 입력할 때까지 화면을 열어둡니다.
