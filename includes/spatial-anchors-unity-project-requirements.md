버전 2.9 이상용 Azure 공간 앵커 SDK는 다음 패키지를 사용 하 여 [UNITY XR 플러그 인 프레임 워크](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) 를 사용 하 여 [unity 2020.3 (lts)](https://unity3d.com/unity/whats-new/2020.3.0) 에서 구성 해야 합니다.

- AR: [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@4.0/manual/index.html)
- Windows XR 플러그 인 (HoloLens 장치 지원): [4.4.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr@4.4/manual/index.html)
- ARCore XR 플러그 인 (Android 장치 지원): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arcore@4.0/manual/index.html)
- ARKit XR 플러그 인 (iOS 장치 지원): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arkit@4.0/manual/index.html)

Azure 공간 앵커 SDK 버전 2.8 또는 이전 버전은 [UNITY XR 플러그 인 프레임 워크](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) 를 지원 하지 않으며 다음 패키지를 사용 하 여 [unity 2019.4 (lts)](https://unity.com/releases/2019-lts) 에서 구성 해야 합니다.

- AR: [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html)
- Windows Mixed Reality (HoloLens 장치 지원): [4.2.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html)
- ARCore XR 플러그 인 (Android 장치 지원): [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html)
- ARKit XR 플러그 인 (iOS 장치 지원): [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html)
