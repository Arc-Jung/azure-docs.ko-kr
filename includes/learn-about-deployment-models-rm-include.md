> [!NOTE]
> Azure에는 리소스를 만들고 사용하는 데 필요한 두 가지 배포 모델이 있습니다. [Azure Resource Manager 및 클래식](../articles/azure-resource-manager/management/deployment-models.md)이라는 두 가지 배포 모델이 있다는 것을 이해해야 합니다. 이 문서에서는 Resource Manager 배포 모델의 사용법에 대해 설명합니다. 새 배포에는 클래식 배포 모델 대신 Resource Manager 배포 모델을 사용하는 것이 좋습니다.
