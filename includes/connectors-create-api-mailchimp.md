### <a name="prerequisites"></a>필수 조건
* [MailChimp](https://www.MailChimp.com/) 계정 

논리 앱에서 MailChimp 계정을 사용하려면 먼저 MailChimp 계정에 연결하도록 논리 앱에 권한을 부여해야 합니다. 다행히 Azure Portal의 논리 앱 내에서 이를 쉽게 수행할 수 있습니다. 

MailChimp 계정에 연결하도록 논리 앱에 권한을 부여하는 단계는 다음과 같습니다.

1. 논리 앱 디자이너에서 MailChimp에 대한 연결을 만들려면 드롭다운 목록에서 **Microsoft 관리 API 표시**를 선택한 다음 검색 상자에 *MailChimp*를 입력합니다. 사용할 트리거 또는 동작을 선택합니다.  
   ![MailChimp 1단계](./media/connectors-create-api-mailchimp/mailchimp-1.png)
2. 이전에 MailChimp에 대한 연결을 만들지 않은 경우 MailChimp 자격 증명을 제공하라는 메시지가 표시됩니다. 이러한 자격 증명을 사용하여 MailChimp 계정의 데이터에 연결하도록 논리 앱에 권한을 부여하고 해당 데이터에 액세스할 수 있습니다.  
   ![MailChimp 2단계](./media/connectors-create-api-mailchimp/mailchimp-2.png)
3. MailChimp 사용자 이름 및 암호를 제공하여 논리 앱에 권한을 부여합니다.  
   ![MailChimp 3단계](./media/connectors-create-api-mailchimp/mailchimp-3.png)   
4. 연결이 만들어졌으므로 이제 논리 앱의 다른 단계를 진행할 수 있습니다.  
   ![MailChimp 4단계](./media/connectors-create-api-mailchimp/mailchimp-4.png)
