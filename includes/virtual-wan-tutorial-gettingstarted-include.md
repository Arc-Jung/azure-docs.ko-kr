**WAN(광역 네트워크)** 페이지에서 만든 WAN을 선택합니다. 왼쪽 메뉴에서 **시작** 페이지를 선택합니다. 이 페이지에는 가상 WAN 리소스 만들기 순서가 표시됩니다. 기능 WAN을 만들려면 1~4 단계를 완료해야 합니다. 사이트와 허브는 서로 종속되지 않으므로 원하는 순서대로 작성할 수 있습니다. 그러나 사이트와 허브를 연결하기 전에 사이트와 허브가 모두 생성되어 있어야 합니다.

절차는 다음과 같습니다. 

1. 사이트 만들기
2. 허브 만들기
3. 사이트와 허브 연결
4. 파일 다운로드
5. 허브와 가상 네트워크(일부 구성에서의 선택 사항) 연결
