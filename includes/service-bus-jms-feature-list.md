다음 표에서는 현재 지원 Azure Service Bus는 JMS (Java Message Service) 기능을 보여 줍니다. 또한 지원 되지 않는 기능을 보여 줍니다.


| 기능 | 상태 |
|---|---|
| 큐   | 지원 여부 |
| 토픽   | 지원 여부 |
| 임시 큐 | 지원 여부 |
| 임시 항목 | 지원 여부 |
| 메시지 선택기 | 지원 여부 |
| 큐 브라우저 | 지원 여부 |
| 공유 내구성이 있는 구독 | 지원 여부|
| 공유 되지 않은 내구성이 있는 구독 | 지원 여부 |
| 영구 공유 되지 않는 공유 구독 | 지원 여부 |
| 공유 되지 않은 내구성이 없는 구독 | 지원 여부 |
| 분산 트랜잭션 | 지원되지 않음 |
| 영 속 종착지 | 지원 안 함 |
