아래 표에는 Azure Service Bus에서 현재 지원 되 고 지원 되지 않는 **JMS (Java Message Service** ) 기능이 나열 되어 있습니다.


| 기능 | 상태 |
|---|---|
| 큐   | 지원됨 |
| 토픽   | 지원됨 |
| 임시 큐 | 지원됨 |
| 임시 항목 | 지원됨 |
| 메시지 선택기 | 지원됨 |
| 큐 브라우저 | 지원됨 |
| 공유 내구성이 있는 구독 | 지원됨|
| 공유 되지 않은 내구성이 있는 구독 | 지원됨 |
| 영구 공유 되지 않는 공유 구독 | 지원됨 |
| 공유 되지 않은 내구성이 없는 구독 | 지원됨 |
| 분산 트랜잭션 | **지원되지 않음** |
| 영 속 종착지 | **지원되지 않음** |
