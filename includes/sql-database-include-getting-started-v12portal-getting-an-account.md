## <a name="connecting-to-the-azure-portal-with-a-subscription"></a>구독으로 Azure 포털에 연결
Microsoft Azure에 대한 기존 구독이 있는 경우 다음 단계에 따라 Azure 포털에 연결합니다. 기존 Microsoft Azure 계정이 없는 경우 계정을 가져오려면 아래 링크 중 하나를 수행한 후 아래 단계에 따라 Azure 포털에 연결합니다.

1. 선택한 브라우저를 열고 http://portal.azure.com/에서 [Azure Portal](https://portal.azure.com/)에 연결합니다.
2. [Azure 포털](https://portal.azure.com/)(http://portal.azure.com/)에 로그인합니다.
3. 로그인 페이지가 표시되면 구독에 대한 자격 증명을 제공합니다.
