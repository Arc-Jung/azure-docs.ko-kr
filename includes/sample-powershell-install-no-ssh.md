이 샘플에는 Azure PowerShell이 필요합니다. 설치되어 있는지 확인하려면 `Get-Module -ListAvailable Az`를 실행합니다. 설치해야 하는 경우 [Azure PowerShell 모듈 설치](/powershell/azure/install-az-ps)를 참조하세요. 

`Connect-AzAccount`를 실행하여 Azure와 연결합니다.
