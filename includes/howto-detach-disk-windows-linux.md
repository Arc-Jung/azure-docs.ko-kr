가상 컴퓨터에 연결된 데이터 디스크가 더 이상 필요하지 않은 경우 쉽게 분리할 수 있습니다. 디스크를 분리하면 가상 컴퓨터에서 디스크가 제거되지만, 저장소에서는 제거되지 않습니다.

디스크에 있는 기존 데이터를 다시 사용하려는 경우 동일한 또는 다른 가상 컴퓨터에 다시 연결할 수 있습니다.

> [AZURE.NOTE]또한 가상 컴퓨터를 삭제하지 않는 한 운영 체제 디스크를 분리할 수 없습니다.


## 디스크 찾기

디스크의 이름을 모르거나, 분리하기 전에 이름을 확인하려면 다음 단계를 따릅니다.


1. 아직 로그인하지 않은 경우 [Azure 포털](http://manage.windowsazure.com)에 로그인합니다.

2. **가상 컴퓨터**를 클릭하고 가상 컴퓨터의 이름을 클릭한 다음 **대시보드**를 클릭합니다.

3. 모든 연결된 디스크의 이름 및 형식이 **디스크** 아래 테이블에 나열됩니다. 예를 들어, 이 화면에는 한 개의 운영 체제(OS) 디스크와 한 개의 데이터 디스크가 있는 가상 컴퓨터가 표시됩니다.

	![데이터 디스크 찾기](./media/howto-detach-disk-windows-linux/FindDataDisks.png)


## 디스크 분리

1. **가상 컴퓨터**를 클릭하고 분리하려는 데이터 디스크가 있는 가상 컴퓨터를 선택한 다음 **대시보드**를 클릭합니다.

2. 명령 모음에서 **디스크 분리**를 클릭합니다.

3. 데이터 디스크를 선택하고 확인 표시를 클릭하여 분리합니다.

	![디스크 분리 세부 정보](./media/howto-detach-disk-windows-linux/DetachDiskDetails.png)

디스크가 저장소에 유지되지만 더 이상 가상 컴퓨터에 연결되어 있지 않습니다.
