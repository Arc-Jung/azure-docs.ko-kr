대금 청구 계정 유형을 결정 하는 규약 유형이 확인
 
1. [Azure Portal]( http://portal.azure.com)에 로그인합니다.
 
2. **Cost Management + 청구**에서 검색합니다.
 
   ![Azure Portal 검색을 보여 주는 스크린샷](./media/billing-check-account-type/billing-search-cost-management-billing.png)    
 
3. 하나의 청구 범위에 액세스할 수 있으면 선택 **속성** 왼쪽에서 합니다.
 
    ![속성 페이지에서 microsoft 고객 계약을 보여 주는 스크린샷](./media/billing-check-account-type/billing-mca-property.png)
    
    합니다 **청구 계정 유형** 속성 페이지는 사용자의 계정 유형을 결정 합니다. Microsoft 온라인 서비스 프로그램, 기업 계약 또는 Microsoft 고객 계약 수 있습니다. 청구 계정 유형에 대 한 자세한 내용은 참조 하세요 [Azure portal에서 청구 계정 보기](../articles/billing/billing-view-all-accounts.md)합니다.  
 
4. 여러 청구 범위에 액세스할 수 있으면 선택 **대금 청구 계정** 하거나 **모든 청구 범위** 왼쪽에서 합니다. 형식을 확인 합니다 **청구 계정 유형** 열입니다.
 
    ![청구 계정 목록 페이지에서 microsoft 고객 계약을 보여 주는 스크린샷](./media/billing-check-account-type/billing-account-type-in-the-list.png)
