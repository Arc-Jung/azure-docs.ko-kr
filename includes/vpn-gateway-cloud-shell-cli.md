이 문서에서는 Azure CLI을 사용 합니다. 명령을 실행 하려면 Azure Cloud Shell를 사용할 수 있습니다. Azure Cloud Shell은 이 항목의 단계를 실행하는 데 무료로 사용할 수 있는 대화형 셸입니다. 공용 Azure 도구가 사전 설치되어 계정에서 사용하도록 구성되어 있습니다.

Cloud Shell을 열려면 코드 블록의 오른쪽 위 모서리에 있는 **사용해 보세요** 를 선택하기만 하면 됩니다. 을 (를) 이동 하 [https://shell.azure.com](https://shell.azure.com) 고, Bash 또는 PowerShell을 반영 하도록 왼쪽 모퉁이의 드롭다운을 전환 하 여 별도의 브라우저 탭에서 Cloud Shell를 시작할 수도 있습니다. **복사** 를 선택하여 코드 블록을 복사하여 Cloud Shell에 붙여넣고, Enter 키를 눌러 실행합니다.
