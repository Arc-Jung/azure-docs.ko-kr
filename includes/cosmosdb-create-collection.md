이제 데이터 탐색기를 사용하여 컬렉션을 만들고 데이터베이스에 데이터를 추가할 수 있습니다. 

1. Azure Portal의 탐색 메뉴에서 **데이터 탐색기**를 클릭합니다. 
2. 데이터 탐색기 블레이드에서 **새 컬렉션**을 클릭한 후 다음 정보를 사용하여 페이지를 채웁니다.

    ![Azure Portal의 데이터 탐색기](./media/cosmosdb-create-collection/azure-cosmosdb-data-explorer.png)

    설정|제안 값|설명
    ---|---|---
    데이터베이스 ID|항목|새 데이터베이스에 대한 ID입니다. 데이터베이스 이름은 1~255자 사이여야 하며 `/ \ # ?` 또는 후행 공백을 포함할 수 없습니다.
    컬렉션 ID|ToDoList|새 컬렉션에 대한 ID입니다. 컬렉션 이름은 데이터베이스 ID와 동일한 문자 요구 사항을 갖습니다.
    저장소 용량| 고정(10GB)|기본값을 그대로 둡니다. 데이터베이스의 저장소 용량입니다.
    처리량|400RU|기본값을 그대로 둡니다. 대기 시간을 단축하려면 나중에 처리량을 증가시킬 수 있습니다.
    파티션 키|/userid|데이터를 각 파티션에 균등하게 배포하는 파티션 키입니다. 올바른 파티션 키를 선택하는 것은 성능이 뛰어난 컬렉션을 만드는 데 중요합니다. 자세한 내용은 [분할 디자인](../articles/cosmos-db/partition-data.md#designing-for-partitioning)에서 확인하세요.    



3. 양식을 작성한 후 **확인**을 클릭합니다.
