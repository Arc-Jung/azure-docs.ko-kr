## <a name="set-up-the-session"></a>세션 설정

호출 `Start()` 환경 데이터를 처리 하 여 세션을 사용 하도록 설정 합니다.

세션에서 발생 한 이벤트를 처리 하려면 이벤트 처리기를 연결 합니다.
