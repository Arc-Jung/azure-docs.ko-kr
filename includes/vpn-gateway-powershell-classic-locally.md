클래식 배포 모델로 작업할 때Azure Cloud Shell을 사용할 수 없습니다. 대신 SM(Azure 서비스 관리) PowerShell cmdlet의 최신 버전을 컴퓨터에 로컬로 설치해야 합니다. 이러한 cmdlet은 AzureRM 또는 Az cmdlet과 다릅니다. SM cmdlet을 설치하려면 [서비스 관리 cmdlet 설치를](/powershell/azure/servicemanagement/install-azure-ps)참조하십시오. 일반적으로 Azure PowerShell에 대한 자세한 내용은 [Azure PowerShell 설명서를](/powershell/azure/)참조하십시오.
