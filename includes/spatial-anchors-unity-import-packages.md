Unity에서 프로젝트를 엽니다. Unity 패키지 관리자를 사용하여 Unity 프로젝트로 다운로드한 Azure Spatial Anchors 패키지를 가져오려면 <a href="https://docs.unity3d.com/Manual/upm-ui-tarball.html" target="_blank">여기</a> 지침을 따르세요.

프로젝트 보기에서 `Packages` 폴더를 확장하여 올바른 패키지를 가져왔는지 확인할 수 있습니다.
