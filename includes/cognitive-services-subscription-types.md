여러 가지 리소스 (여러 서비스 리소스 또는 단일 서비스)를 통해 Azure Cognitive Services에 액세스할 수 있습니다.

* 다중 서비스 리소스:
    * 단일 키와 엔드포인트로 여러 Azure Cognitive Services에 액세스합니다.
    * 는 사용 하는 서비스에서 청구를 통합 합니다.
* 단일 서비스 리소스:
    * 만든 각 서비스에 대해 고유한 키와 끝점을 사용 하 여 단일 Azure 인식 서비스에 액세스 합니다. 
    * 무료 계층을 사용 하 여 서비스를 사용해 보세요.
