> [!IMPORTANT]
> X.509 인증 기관 (CA) 인증을 사용 하는 장치에서 파일 업로드 기능은 공개 미리 보기로 제공 되며 [미리 보기 모드를 사용 하도록 설정 해야 합니다](../articles/iot-hub/iot-hub-preview-mode.md). 일반적으로 x.509 지문 인증을 사용 하는 장치에서 사용할 수 있습니다. IoT Hub 사용 하는 x.509 인증에 대 한 자세한 내용은 [지원 되는 x.509 인증서](../articles/iot-hub/iot-hub-devguide-security.md#supported-x509-certificates)를 참조 하세요.
