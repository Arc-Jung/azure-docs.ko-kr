### <a name="prepare-your-environment-for-the-azure-cli"></a>Azure CLI에 대 한 환경 준비

- Bash 환경을 사용 하 여 [Azure Cloud Shell](/azure/cloud-shell/quickstart) 를 사용 합니다.

   [![Embed 시작](https://shell.azure.com/images/launchcloudshell.png "Azure Cloud Shell 시작")](https://shell.azure.com)   
- 원하는 경우 Azure CLI의 [로컬 설치](/cli/azure/install-azure-cli) 를 수행 하 여 CLI 참조 명령을 실행 합니다.
   - local install을 사용하는 경우 [az login](/cli/azure/reference-index#az-login) 명령을 사용하여 Azure CLI에 로그인합니다.  터미널에 표시된 단계에 따라 인증 프로세스를 완료합니다.  추가 로그인 옵션은 [Azure CLI으로 로그인을](/cli/azure/authenticate-azure-cli) 참조 하세요.
  - 메시지가 표시되면 처음 사용할 때 Azure CLI 확장을 설치합니다.  [Azure CLI에서 확장 사용](/cli/azure/azure-cli-extensions-overview)에서 확장에 대해 자세히 알아봅니다.
  - [az version](/cli/azure/reference-index?#az_version)을 실행하여 설치된 버전과 종속 라이브러리를 찾습니다. 최신 버전으로 업그레이드하려면 [az upgrade](/cli/azure/reference-index?#az_upgrade)를 실행합니다.
