> [!Note]
> Azure IoT Central은 현재 디바이스와 디바이스 템플릿 간에 새로운 상호 작용 패턴을 도입하고 있습니다. 이 새로운 UI 디자인은 디바이스와 디바이스 템플릿을 분리하여 서로 독립적으로 상호 작용할 수 있게 합니다. 왼쪽 탐색 메뉴에 **디바이스 템플릿**  단추가 표시되는 경우 새로운 UI 디자인을 사용할 수 있습니다.
