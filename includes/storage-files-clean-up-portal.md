작업을 완료하면 리소스 그룹을 삭제할 수 있습니다. 그러면 저장소 계정, Azure 파일 공유 및 리소스 그룹 내부에 배포한 다른 리소스를 삭제합니다.

1. 왼쪽 메뉴에서 **리소스 그룹**을 클릭합니다.
2. 리소스 그룹을 마우스 오른쪽 단추로 클릭하고 **리소스 그룹 삭제**를 선택합니다. 페이지가 열리고 리소스 그룹과 함께 삭제할 리소스에 대한 경고 메시지가 표시됩니다.
3. 리소스 그룹 이름을 입력한 다음, **삭제**를 클릭합니다.
