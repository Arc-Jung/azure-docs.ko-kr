> [!IMPORTANT]
> [Microsoft ID 플랫폼(v2.0)](/azure/active-directory/develop/v2-overview)은 [Azure AD(Azure Active Directory) 개발자 플랫폼(v1.0)의 발전된 형태](../articles/active-directory/azuread-dev/about-microsoft-identity-platform.md)입니다. 이 플랫폼을 사용하면 개발자가 모든 Microsoft ID에 로그인하고, Microsoft Graph와 같은 Microsoft API 또는 개발자가 빌드한 API를 호출하기 위한 토큰을 가져오는 애플리케이션을 빌드할 수 있습니다.
> 이 콘텐츠는 이전의 Azure AD v1.0 엔드포인트에 대한 것입니다. 새 프로젝트에는 v2.0 엔드포인트를 사용하는 것이 좋습니다. 자세한 내용은 [Microsoft ID 플랫폼(v2.0)으로 업데이트하는 이유](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) 및 [Microsoft ID 플랫폼 제한 사항](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations)을 참조하세요.
