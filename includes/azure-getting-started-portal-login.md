## 구독으로 Microsoft Azure 포털에 연결

Microsoft Azure 포털에 연결하려면 구독이 있어야 합니다.

### 새 계정 가져오기

기존 Microsoft Azure 계정이 없는 경우 계정을 가져오려면 아래 링크 중 하나를 수행합니다.

- [무료 계정](https://azure.microsoft.com/get-started/) 가져오기
- [MSDN 구독](https://azure.microsoft.com/pricing/member-offers/msdn-benefits/)을 사용하세요?

###기존 계정을 사용하여 로그인합니다.

[기존 구독](https://account.windowsazure.com/Home/Index)을 사용하여 Azure 포털에 연결하려면 다음 단계를 수행합니다.

1. 선택한 브라우저를 열고 http://portal.azure.com/에서 [Azure 포털](https://portal.azure.com/)에 연결합니다.

1. http://portal.azure.com/에서 [Azure 포털](https://portal.azure.com/)에 로그인합니다.

1. 로그인 페이지가 표시되면 구독에 대한 자격 증명을 제공합니다.
