#### <a name="to-delete-a-cloud-appliance"></a>클라우드 어플라이언스를 삭제하려면

1. Azure 포털에 로그인합니다.
2. 데이터가 없는 비활성화된 디바이스만 삭제할 수 있습니다. 디바이스에서 데이터를 먼저 삭제하거나 [볼륨 컨테이너의 데이터를 다른 디바이스로 장애 조치할 수 있습니다](../articles/storsimple/storsimple-8000-device-failover-cloud-appliance.md). 데이터가 삭제되면 디바이스를 비활성화할 준비가 되었습니다.
3. StorSimple 디바이스 관리자 서비스 페이지에서 **디바이스**를 클릭한 다음, 해당 디바이스를 선택합니다. 마우스 오른쪽 단추로 클릭하고 **비활성화**를 선택합니다.
4. 디바이스가 비활성화되면 디바이스를 마우스 오른쪽 단추로 클릭하고 **삭제**를 선택합니다.

    ![비활성화된 디바이스를 선택하고 삭제를 클릭합니다.](./media/storsimple-8000-delete-cloud-appliance/delete-cloud-appliance1.png)

5. 디바이스 이름을 입력하여 삭제를 확인합니다. 디바이스를 삭제한 후에는 디바이스 목록이 업데이트됩니다.

    ![삭제 확인](./media/storsimple-8000-delete-cloud-appliance/delete-cloud-appliance2.png)

6. 디바이스가 삭제되면 알림이 표시됩니다.

    ![성공적인 디바이스 삭제 알림](./media/storsimple-8000-delete-cloud-appliance/delete-cloud-appliance4.png)

7. 디바이스 목록을 업데이트하여 삭제된 디바이스가 표시됩니다.

    ![업데이트된 디바이스 목록](./media/storsimple-8000-delete-cloud-appliance/delete-cloud-appliance5.png)
