> [!TIP]
> 기본적으로 함수 앱에 필요한 Azure 리소스는 사용자가 제공하는 함수 앱 이름을 기반으로 만들어집니다. 기본적으로 함수 앱을 사용하여 동일한 새 리소스 그룹에도 만들어집니다. 이러한 리소스의 이름을 사용자 지정하거나 기존 리소스를 다시 사용하려면 대신 [고급 만들기 옵션을 사용하여 프로젝트를 게시](../articles/azure-functions/functions-develop-vs-code.md#enable-publishing-with-advanced-create-options)해야 합니다.
